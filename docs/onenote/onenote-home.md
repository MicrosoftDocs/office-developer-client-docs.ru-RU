Здесь вы найдете инструкции, технические статьи, справочные материалы и записи блога, посвященные разработке решений и настройке классических приложений OneNote.
  
> [!NOTE]
> API-интерфейсы, описанные в этой документации предназначены только для решений для настольных компьютеров клиента OneNote Win32 отключенный сценариев. Для подключенных сценариев использования рекомендуемые API службы OneNote. Для получения дополнительных сведений посетите [dev.onenote.com](http://dev.onenote.com/).
