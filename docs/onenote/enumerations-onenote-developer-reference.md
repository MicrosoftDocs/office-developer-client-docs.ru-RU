# <a name="enumerations-onenote-developer-reference"></a>Перечисления (справочные материалы для разработчиков OneNote)

В этом разделе описываются перечисления в объектной модели OneNote 2013.
  
## <a name="createfiletype"></a>креатефилетипе
<a name="odc_CreateFileType"> </a>

При передаче в метод **OpenHierarchy** указывает тип создаваемого объекта (если путь, переданный в метод, еще не существует). 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**кфтноне** <br/> |нуль  <br/> |Новый объект не создается.  <br/> |
|**кфтнотебук** <br/> |1,1  <br/> |Создает записную книжку с использованием указанного имени и местоположения.  <br/> |
|**кфтфолдер** <br/> |2  <br/> |Создает группу разделов, используя заданные имя и расположение.  <br/> |
|**кфтсектион** <br/> |4  <br/> |Создает раздел, используя заданные имя и расположение.  <br/> |
   
## <a name="docklocation"></a>докклокатион

Указывает закрепленное расположение окна OneNote 2013 с помощью интерфейса [окна](window-interfaces-onenote.md) . Если задано свойство **доккедлокатион** , указывает расположение, в которое закрепляется окно OneNote. Это перечисление является новым в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**длдефаулт** <br/> |–1  <br/> |Окно OneNote закреплено в расположении по умолчанию на рабочем столе.  <br/> |
|**дллефт** <br/> |1,1  <br/> |Окно OneNote закреплено в левой части рабочего стола.  <br/> |
|**длригхт** <br/> |2  <br/> |Окно OneNote закреплено в правой части рабочего стола.  <br/> |
|**длтоп** <br/> |4  <br/> |Окно OneNote закреплено в верхней части рабочего стола.  <br/> |
|**длботтом** <br/> |4   <br/> |Окно OneNote закреплено в нижней части рабочего стола.  <br/> |

|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|

|**хсчилдрен** <br/> |1,1  <br/> |Получает непосредственные дочерние узлы начального узла и не содержит потомков в более высоких и нижних группах подразделов.  <br/> |

|**хссектионс** <br/> |4  <br/> |Получает все разделы под узлом "начало", включая разделы в группах разделов и подразделах.  <br/> |
|**хспажес** <br/> |4   <br/> |Получает все страницы под начальным узлом, включая все страницы в группах разделов и подразделах.  <br/> |

|:-----|:-----|:-----|
|**рртноне** <br/> |нуль  <br/> |Задает отсутствие списка недавних результатов для отображения.  <br/> |
|**рртфилинг** <br/> |1,1  <br/> |Задает отрисовывается список недавних результатов "Архивация".  <br/> |
|**рртсеарч** <br/> |2  <br/> |Задает отображаемый список последних результатов поиска.  <br/> |
|**рртлинкс** <br/> |4  <br/> |Задает для отображения список "ссылки".  <br/> |
