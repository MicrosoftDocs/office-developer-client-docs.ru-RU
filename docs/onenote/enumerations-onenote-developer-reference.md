<a name="odc_CreateFileType"> </a>

При передаче в метод **OpenHierarchy** указывает тип создаваемого объекта (если путь, переданный в метод, еще не существует). 
  
|**Member**|**Value**|**Описание**|
|:-----|:-----|:-----|
|**Кфтноне** <br/> |нуль  <br/> |Новый объект не создается.  <br/> |
|**Кфтнотебук** <br/> |1,1  <br/> |Создает записную книжку с использованием указанного имени и местоположения.  <br/> |
|**Кфтфолдер** <br/> |2  <br/> |Создает группу разделов, используя заданные имя и расположение.  <br/> |
|**Кфтсектион** <br/> |4  <br/> |Создает раздел, используя заданные имя и расположение.  <br/> |

|**Member**|**Value**|**Описание**|
|:-----|:-----|:-----|
|**Длдефаулт** <br/> |–1  <br/> |Окно OneNote закреплено в расположении по умолчанию на рабочем столе.  <br/> |
|**Дллефт** <br/> |1,1  <br/> |Окно OneNote закреплено в левой части рабочего стола.  <br/> |
|**Длригхт** <br/> |2  <br/> |Окно OneNote закреплено в правой части рабочего стола.  <br/> |
|**Длтоп** <br/> |4  <br/> |Окно OneNote закреплено в верхней части рабочего стола.  <br/> |
|**Длботтом** <br/> |SP4  <br/> |Окно OneNote закреплено в нижней части рабочего стола.  <br/> |

|**Рртфилинг** <br/> |1,1  <br/> |Задает отрисовывается список недавних результатов "Архивация".  <br/> |
|**Рртсеарч** <br/> |2  <br/> |Задает отображаемый список последних результатов поиска.  <br/> |
|**Рртлинкс** <br/> |4  <br/> |Задает для отображения список "ссылки".  <br/> |
