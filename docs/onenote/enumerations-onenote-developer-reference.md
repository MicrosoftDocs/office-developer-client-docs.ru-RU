# <a name="enumerations-onenote-developer-reference"></a>Enumerations (справка для разработчиков OneNote)

В этом разделе описываются enumerations в объектной модели OneNote 2013.
  
## <a name="createfiletype"></a>CreateFileType
<a name="odc_CreateFileType"> </a>

При передании **методу OpenHierarchy** указывает тип создаемого объекта, если путь, переданный методу, еще не существует. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**cftNone** <br/> |0  <br/> |Не создает новый объект.  <br/> |
|**cftNotebook** <br/> |1   <br/> |Создает записную книжку с использованием указанного имени и расположения.  <br/> |
|**cftFolder** <br/> |2   <br/> |Создает группу разделов с использованием указанного имени и расположения.  <br/> |
|**cftSection** <br/> |3   <br/> |Создает раздел с использованием указанного имени и расположения.  <br/> |
   
## <a name="docklocation"></a>DockLocation
<a name="odc_CreateFileType"> </a>

Указывает закрепленное расположение окна OneNote 2013 с помощью [интерфейса Window.](window-interfaces-onenote.md) Если задано свойство **DockedLocation,** указывает расположение, в котором необходимо пристыковать окно OneNote. Это новое enumeration в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**dlDefault** <br/> |–1  <br/> |Окно OneNote закреплено в расположении по умолчанию на рабочем столе.  <br/> |
|**dlLeft** <br/> |1   <br/> |Окно OneNote закреплено в левой части рабочего стола.  <br/> |
|**dlRight** <br/> |2   <br/> |Окно OneNote закреплено в правой части рабочего стола.  <br/> |
|**dlTop** <br/> |3   <br/> |Окно OneNote закреплено в верхней части рабочего стола.  <br/> |
|**dlBottom** <br/> |4   <br/> |Окно OneNote закреплено в нижней части рабочего стола.  <br/> |
   
## <a name="filinglocation"></a>FilingLocation
<a name="odc_CreateFileType"> </a>

При передается **методу SetFilingLocation,** указывает, для какого типа контента задано расположение для хранения, когда тип контента отправляется в OneNote. Это новое enumeration в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**flEMail** <br/> |0  <br/> |Задает место, в которое будут отправляться сообщения электронной почты Outlook.  <br/> |
|**flContacts** <br/> |1   <br/> |Задает место, в котором будут поданы контакты Outlook.  <br/> |
|**flTasks** <br/> |2   <br/> |Задает место, в котором будут выполняться задачи Outlook.  <br/> |
|**flMeetings** <br/> |3   <br/> |Задает место, в котором будут проходить собрания Outlook.  <br/> |
|**flWebContent** <br/> |4   <br/> |Задает место, в которое будет подано содержимое Internet Explorer.  <br/> |
|**flPrintOuts** <br/> |5   <br/> |Задает место, в котором будут печататься распечатки с принтера OneNote.  <br/> |
   
## <a name="filinglocationtype"></a>FilingLocationType
<a name="odc_CreateFileType"> </a>

При передается **методу SetFilingLocation,** указывает, куда отправляется содержимое OneNote. Это новое enumeration в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**fltNamedSectionNewPage** <br/> |0  <br/> |Задает содержимое, которое будет подано на новой странице в указанном разделе.  <br/> |
|**fltCurrentSectionNewPage** <br/> |1   <br/> |Задает содержимое, которое будет подано на новой странице в текущем разделе.  <br/> |
|**fltCurrentPage** <br/> |2   <br/> |Задает содержимое, которое должно быть подано на текущей странице.  <br/> |
|**fltNamedPage** <br/> |4   <br/> |Задает содержимое, которое должно быть подано на указанной странице.  <br/> |
   
## <a name="hierarchyelement"></a>HierarchyElement
<a name="odc_CreateFileType"> </a>

При присвоении свойству **TreeDepth** интерфейса [IQuickFilingDialog](quick-filing-dialog-box-interfaces-onenote.md) указывает глубину дерева OneNote, отображаемого при отображении диалоговое окно быстрой подачи. При передании в метод **AddButton** объекта **IQuickFilingDialog** ссылается на определенные элементы в иерархии OneNote. Это новое enumeration в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**heNone** <br/> |0  <br/> |Ссылается на элемент без элемента.  <br/> |
|**heNotebooks** <br/> |1   <br/> |Относится к элементам записной книжки.  <br/> |
|**heSectionGroups** <br/> |2   <br/> |Относится к элементам группы разделов.  <br/> |
|**heSections** <br/> |4   <br/> |Относится к элементам Section.  <br/> |
|**hePages** <br/> |8   <br/> |Относится к элементам Page.  <br/> |
   
## <a name="hierarchyscope"></a>HierarchyScope
<a name="odc_HierarchyScope"> </a>

При передается **методу GetHierarchy,** указывает самый низкий уровень, который необходимо получить в иерархии узла записной книжки. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**hsSelf** <br/> |0  <br/> |Получает только указанный узел запуска без потомков.  <br/> |
|**hsChildren** <br/> |1   <br/> |Возвращает непосредственные узлы-потомки в узле запуска, а потомки в группах подмещений более высокого или нижнего уровня не имеются.  <br/> |
|**hsNotebooks** <br/> |2   <br/> |Получает все записные книжки под узлом начала или корневым.  <br/> |
|**hsSections** <br/> |3   <br/> |Возвращает все разделы под узлом начала, включая разделы в группах разделов и группах подмещений.  <br/> |
|**hsPages** <br/> |4   <br/> |Возвращает все страницы под узлом начала, включая все страницы в группах разделов и группах подмещений.  <br/> |
   
## <a name="newpagestyle"></a>NewPageStyle

При передается **методу CreateNewPage,** задав стиль новой страницы. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**npsDefault** <br/> |0  <br/> |Создает страницу со стилем страницы по умолчанию.  <br/> |
|**npsBlankPageWithTitle** <br/> |1   <br/> |Создает пустую страницу с заголовком.  <br/> |
|**npsBlankPageNoTitle** <br/> |2   <br/> |Создает пустую страницу без заголовка.  <br/> |
   
## <a name="notebookfilterouttype"></a>NotebookFilterOutType
<a name="odc_HierarchyScope"> </a>

При передании **методу NotebookFilterOut** объекта **QFD** указывает, какие записные книжки будут отображаться при отображке окна QFD. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**nfoLocal** <br/> |1   <br/> |Разрешить только локальные записные книжки.  <br/> |
|**nfoNetwork** <br/> |2   <br/> |Разрешает UNC-записные книжки или записные книжки SharePoint.  <br/> |
|**nfoWeb** <br/> |4   <br/> |Разрешает записные книжки OneDrive.  <br/> |
|**nfoNoWacUrl** <br/> |8   <br/> |Все записные книжки в расположениях, где нет веб-клиента.  <br/> |

|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**piBasic** <br/> |0  <br/> |Возвращает только базовое содержимое страницы без разметки выделения, типы файлов для объектов двоичных данных и двоичных объектов данных. Это стандартное значение, передав его.  <br/> |
|**piBinaryData** <br/> |1   <br/> |Возвращает содержимое страницы без разметки выбора, но со всеми двоичными данными.  <br/> |
|**piSelection** <br/> |2   <br/> |Возвращает содержимое страницы с разметки выбора, но без двоичных данных.  <br/> |
|**piBinaryDataSelection** <br/> |3   <br/> |Возвращает содержимое страницы с разметки выбора и всеми двоичными данными.  <br/> |
|**piFileType** <br/> |4   <br/> |Возвращает содержимое страницы с информацией о типе файла для объектов двоичных данных.  <br/> |
|**piBinaryDataFileType** <br/> |5   <br/> |Возвращает содержимое страницы с информацией о типе файла для объектов двоичных данных и объектов двоичных данных  <br/> |
|**piSelectionFileType** <br/> |6   <br/> |Возвращает содержимое страницы с разметки выбора и сведениями о типе файла для двоичных данных.  <br/> |
|**piAll** <br/> |7   <br/> |Возвращает все содержимое страницы.  <br/> |

## <a name="recentresulttype"></a>RecentResultType
<a name="odc_RecentResultType"> </a>

При передается **методу SetRecentResults** объекта **IQuickFilingDialog,** указывает список последних результатов, который будет отображаться при отображении диалоговых окна "Быстрая подача данных". Последние списки результатов используются для отслеживания набора местоположений OneNote, выбранных пользователем в диалоговом окне "Быстрая подача данных". В OneNote 2013 есть три списка последних результатов, которые отслеживают действия по регистрации, поиску и связываниям. Это новое enumeration в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**rrtNone** <br/> |0  <br/> |Не задает список последних результатов для отображения.  <br/> |
|**rrtFiling** <br/> |1   <br/> |Задает список последних результатов "Подача" для отображения.  <br/> |
|**rrtSearch** <br/> |2   <br/> |Задает список последних результатов поиска для отображения.  <br/> |
|**rrtLinks** <br/> |3   <br/> |Задает список последних результатов "Ссылки" для отображения.  <br/> |

## <a name="xmlschema-updated-for-onenote-2013"></a>XMLSchema (обновлено для OneNote 2013)
<a name="odc_SpecialLocation"> </a>

При передается в один из следующих методов, указывает версию схемы XML OneNote для использования:
  
- **OneNote15.Application.GetPageContent**
    
- **OneNote15.Application.FindMeta**
    
- **OneNote15.Application.FindPages**
    
- **OneNote15.Application.GetHierarchy**
    
- **OneNote15.Application.GetPageContent**
    
- **OneNote15.Application.UpdateHierarchy**
    
- **OneNote15.Application.UpdatePageContent**
    
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**xs2007** <br/> |0  <br/> |Ссылается на схему OneNote 2007.  <br/> |
|**xs2010** <br/> |1   <br/> |Ссылается на схему OneNote 2010.  <br/> |
|**xs2013** <br/> |2   <br/> |Ссылки на схему OneNote 2013.  <br/> |
|**xsCurrent** <br/> |2   <br/> |Ссылается на схему текущей версии OneNote.  <br/> <br/>**ПРИМЕЧАНИЕ.** В большинстве случаев не рекомендуется использовать **xsCurrent,** так как это может вызвать проблемы совместимости с будущими версиями OneNote. Вместо этого укажите версию схемы, для обработки которую было построено приложение, например xs2013.           |
