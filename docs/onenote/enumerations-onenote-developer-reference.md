# <a name="enumerations-onenote-developer-reference"></a>Переумерия (OneNote ссылки разработчика)

В этом разделе описываются перемеры в объектной OneNote 2013 года.
  
## <a name="createfiletype"></a>CreateFileType
<a name="odc_CreateFileType"> </a>

Если путь передан **методу OpenHierarchy,** указывает тип объекта для создания, если он имеется, если путь, переданный методу, еще не существует. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**cftNone** <br/> |0  <br/> |Не создает новый объект.  <br/> |
|**cftNotebook** <br/> |1  <br/> |Создает записную книжку с помощью указанного имени и расположения.  <br/> |
|**cftFolder** <br/> |2  <br/> |Создает группу разделов с помощью указанного имени и расположения.  <br/> |
|**cftSection** <br/> |3  <br/> |Создает раздел с помощью указанного имени и расположения.  <br/> |
   
## <a name="docklocation"></a>DockLocation
<a name="odc_CreateFileType"> </a>

Указывает пристыкованное расположение окна 2013 OneNote 2013 с помощью [интерфейса Window.](window-interfaces-onenote.md) При задании **свойству DockedLocation** указывается расположение, на котором можно пристыковать OneNote окно. В 2013 году это OneNote новое. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**dlDefault** <br/> |–1  <br/> |Окно OneNote пристыковано к расположению по умолчанию на рабочем столе.  <br/> |
|**dlLeft** <br/> |1  <br/> |Окно OneNote пристыковано к левой стороне рабочего стола.  <br/> |
|**dlRight** <br/> |2  <br/> |Окно OneNote пристыковано к правой стороне рабочего стола.  <br/> |
|**dlTop** <br/> |3  <br/> |Окно OneNote состыковано в верхней части рабочего стола.  <br/> |
|**dlBottom** <br/> |4   <br/> |Окно OneNote состыковано в нижней части рабочего стола.  <br/> |
   
## <a name="filinglocation"></a>FilingLocation
<a name="odc_CreateFileType"> </a>

При перенаправле в метод **SetFilingLocation** указывается, какой тип контента задан для расположения подачи, когда тип контента отправляется в OneNote. В 2013 году это OneNote новое. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**flEMail** <br/> |0  <br/> |Задает, Outlook сообщения электронной почты будут отправлены.  <br/> |
|**flContacts** <br/> |1  <br/> |Задает, Outlook контакты будут поданы.  <br/> |
|**flTasks** <br/> |2  <br/> |Задает, Outlook задачи будут поданы.  <br/> |
|**flMeetings** <br/> |3  <br/> |Задает, Outlook собрания будут поданы.  <br/> |
|**flWebContent** <br/> |4   <br/> |Задает, где будет подано содержимое Internet Explorer.  <br/> |
|**flPrintOuts** <br/> |5   <br/> |Задает, где будут поданы распечатки OneNote принтера.  <br/> |
   
## <a name="filinglocationtype"></a>FilingLocationType
<a name="odc_CreateFileType"> </a>

После его перенаправки в **метод SetFilingLocation** указывается, куда отправляется содержимое, OneNote подано. В 2013 году это OneNote новое. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**fltNamedSectionNewPage** <br/> |0  <br/> |Задает содержимое, которое будет подано на новую страницу в указанном разделе.  <br/> |
|**fltCurrentSectionNewPage** <br/> |1  <br/> |Задает содержимое, которое будет подано на новую страницу в текущем разделе.  <br/> |
|**fltCurrentPage** <br/> |2  <br/> |Задает содержимое, которое будет подано на текущей странице.  <br/> |
|**fltNamedPage** <br/> |4   <br/> |Задает содержимое, которое будет подано на указанной странице.  <br/> |
   
## <a name="hierarchyelement"></a>HierarchyElement
<a name="odc_CreateFileType"> </a>

При присвоении **свойству TreeDepth** [интерфейса IQuickFilingDialog](quick-filing-dialog-box-interfaces-onenote.md) указывается глубина дерева OneNote для отображения при отрисовке быстрой подачи диалогов. При перенастройке в метод **AddButton** объекта **IQuickFilingDialog** ссылаются на некоторые элементы OneNote иерархии. В 2013 году это OneNote новое. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**heNone** <br/> |0  <br/> |Ссылается на отсутствие элемента.  <br/> |
|**heNotebooks** <br/> |1  <br/> |Относится к элементам Notebook.  <br/> |
|**heSectionGroups** <br/> |2  <br/> |Относится к элементам Section Group.  <br/> |
|**heSections** <br/> |4   <br/> |Относится к элементам Section.  <br/> |
|**hePages** <br/> |8   <br/> |Относится к элементам Page.  <br/> |
   
## <a name="hierarchyscope"></a>HierarchyScope
<a name="odc_HierarchyScope"> </a>

При перенашив **метод GetHierarchy,** указывается самый низкий уровень для получения в иерархии узлов записной книжки. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**hsSelf** <br/> |0  <br/> |Получает только указанный узел запуска, а потомков нет.  <br/> |
|**hsChildren** <br/> |1  <br/> |Получает ближайшие детские узлы узла запуска, а потомки в более высоких или нижних группах подгруппы не являются потомками.  <br/> |
|**hsNotebooks** <br/> |2  <br/> |Получает все записные книжки ниже узла запуска или корневого узла.  <br/> |
|**hsSections** <br/> |3  <br/> |Получает все разделы ниже точки запуска, в том числе разделы в группах разделов и подгруппах.  <br/> |
|**hsPages** <br/> |4   <br/> |Получает все страницы ниже узла запуска, в том числе все страницы в группах разделов и подгруппах.  <br/> |

|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**npsDefault** <br/> |0  <br/> |Создает страницу со стилем страницы по умолчанию.  <br/> |
|**npsBlankPageWithTitle** <br/> |1  <br/> |Создает пустую страницу с заголовком.  <br/> |
|**npsBlankPageNoTitle** <br/> |2  <br/> |Создает пустую страницу без заголовка.  <br/> |
   
## <a name="notebookfilterouttype"></a>NotebookFilterOutType
<a name="odc_HierarchyScope"> </a>

Когда объект **QFD** передается **методу NotebookFilterOut,** указывается, какие записные книжки должны отображаться при отрисовке окна QFD. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**nfoLocal** <br/> |1  <br/> |Разрешить только локальные записные книжки.  <br/> |
|**nfoNetwork** <br/> |2  <br/> |Позволяет unC или SharePoint записные книжки.  <br/> |
|**nfoWeb** <br/> |4   <br/> |Позволяет OneDrive записные книжки.  <br/> |
|**nfoNoWacUrl** <br/> |8   <br/> |Любые записные книжки в местах, где нет веб-клиента.  <br/> |

|**piBinaryData** <br/> |1  <br/> |Возвращает содержимое страницы без разметки выбора, но со всеми двоичными данными.  <br/> |
|**piSelection** <br/> |2  <br/> |Возвращает содержимое страницы с разметки выбора, но без двоичных данных.  <br/> |

|**piFileType** <br/> |4   <br/> |Возвращает содержимое страницы с данными типа файла для двоичных объектов данных.  <br/> |
|**piBinaryDataFileType** <br/> |5   <br/> |Возвращает содержимое страницы с данными типа файлов для двоичных объектов данных и двоичных объектов данных  <br/> |

|**piAll** <br/> |7   <br/> |Возвращает все содержимое страницы.  <br/> |

## <a name="recentresulttype"></a>RecentResultType
<a name="odc_RecentResultType"> </a>

При переадресовке в **метод SetRecentResults** объекта **IQuickFilingDialog** указывается, какой последний список результатов будет отображаться при отрисовке диалогового окна быстрой подачи. Последние списки результатов используются для отслеживания набора OneNote, выбранных пользователем в диалоговом окне Быстрая подача. В 2013 г. существует три списка последних результатов, отслеживающих подачу, поиск и связывание действий OneNote 2013 г. В 2013 году это OneNote новое. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**rrtNone** <br/> |0  <br/> |Не устанавливает список результатов последнего времени, который будет отрисовыт.  <br/> |
|**rrtFiling** <br/> |1  <br/> |Задает отрисовка списка "Подача" последних результатов.  <br/> |
|**rrtSearch** <br/> |2  <br/> |Задает список "Поиск" для отображения списка результатов последнего результата.  <br/> |
|**rrtLinks** <br/> |3  <br/> |Задает список "Ссылки" для отображения списка результатов последнего результата.  <br/> |
