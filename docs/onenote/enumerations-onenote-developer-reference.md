# <a name="enumerations-onenote-developer-reference"></a>Перечисления (Справочник разработчика по OneNote)

В этом разделе описываются перечисления в объектной модели OneNote 2013.
  
## <a name="createfiletype"></a>CreateFileType
<a name="odc_CreateFileType"> </a>

При передаче в метод **OpenHierarchy** , указывает тип объекта, чтобы создать, если какие-либо, если путь, передаваемый в метод еще не существует. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**cftNone** <br/> |0  <br/> |Создает новый объект.  <br/> |
|**cftNotebook** <br/> |1  <br/> |Создает записной книжке с использованием указанного имени и расположения.  <br/> |
|**cftFolder** <br/> |2  <br/> |Создает группу разделов с помощью указанного имени и расположения.  <br/> |
|**cftSection** <br/> |3  <br/> |Создает раздел с использованием указанного имени и расположения.  <br/> |
   
## <a name="docklocation"></a>DockLocation
<a name="odc_CreateFileType"> </a>

Указывает расположение закрепленного окна OneNote 2013 с помощью интерфейса [окна](window-interfaces-onenote.md) . Если параметр имеет значение **DockedLocation** свойства, указывает место, в котором для стыковки окна OneNote. Это перечисление является новым в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**dlDefault** <br/> |-1  <br/> |Окно OneNote закреплено в расположении по умолчанию на рабочем столе.  <br/> |
|**dlLeft** <br/> |1  <br/> |Окно OneNote закреплено в левой части рабочего стола.  <br/> |
|**dlRight** <br/> |2  <br/> |Окно OneNote, расположенной в правой части рабочего стола.  <br/> |
|**dlTop** <br/> |3  <br/> |Окно OneNote закрепить в верхней части рабочего стола.  <br/> |
|**dlBottom** <br/> |4  <br/> |Окно OneNote закреплено в нижней части рабочего стола.  <br/> |
   
## <a name="filinglocation"></a>FilingLocation
<a name="odc_CreateFileType"> </a>

При передаче в метод **SetFilingLocation** , указывает тип контента расположение хранения для при отправке тип содержимого в OneNote. Это перечисление является новым в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**flEMail** <br/> |0  <br/> |Наборы, где будут храниться сообщения электронной почты Outlook.  <br/> |
|**flContacts** <br/> |1  <br/> |Наборы, где будут храниться контактов Outlook.  <br/> |
|**flTasks** <br/> |2  <br/> |Наборы, где будут храниться задачи Outlook.  <br/> |
|**flMeetings** <br/> |3  <br/> |Наборы, где будут храниться встреч Outlook.  <br/> |
|**flWebContent** <br/> |4  <br/> |Наборы, где будут храниться содержимое Internet Explorer.  <br/> |
|**flPrintOuts** <br/> |5  <br/> |Наборы, где будут храниться распечаток из OneNote принтера.  <br/> |
   
## <a name="filinglocationtype"></a>FilingLocationType
<a name="odc_CreateFileType"> </a>

При передаче в метод **SetFilingLocation** указывает, где хранения контента, который отправляется в OneNote. Это перечисление является новым в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**fltNamedSectionNewPage** <br/> |0  <br/> |Задает содержимое, чтобы быть оформлена на новой странице указанного раздела.  <br/> |
|**fltCurrentSectionNewPage** <br/> |1  <br/> |Задает содержимое, чтобы быть оформлена на новой странице текущего раздела.  <br/> |
|**fltCurrentPage** <br/> |2  <br/> |Задает содержимое, чтобы быть оформлена на текущей странице.  <br/> |
|**fltNamedPage** <br/> |4  <br/> |Задает содержимое, чтобы быть оформлена на указанной странице.  <br/> |
   
## <a name="hierarchyelement"></a>HierarchyElement
<a name="odc_CreateFileType"> </a>

При присвоении свойству **TreeDepth** интерфейса [IQuickFilingDialog](quick-filing-dialog-box-interfaces-onenote.md) указывает число уровней дерева OneNote будет отображаться, если отображается диалоговое окно быстрого представления документов. При передаче метод **AddButton** объекта **IQuickFilingDialog** , ссылается на некоторые элементы в иерархии OneNote. Это перечисление является новым в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**heNone** <br/> |0  <br/> |Ссылается на элемент не.  <br/> |
|**heNotebooks** <br/> |1  <br/> |Относится к элементам записной книжки.  <br/> |
|**heSectionGroups** <br/> |2  <br/> |Относится к элементам группы раздела.  <br/> |
|**heSections** <br/> |4  <br/> |Относится к элементам раздела.  <br/> |
|**hePages** <br/> |8  <br/> |Относится к элементам страницы.  <br/> |
   
## <a name="hierarchyscope"></a>HierarchyScope
<a name="odc_HierarchyScope"> </a>

При передаче в метод **GetHierarchy** указывает нижнего уровня для получения в иерархии узлов записной книжки. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**hsSelf** <br/> |0  <br/> |Получает начальный узел указан и никакие потомки.  <br/> |
|**hsChildren** <br/> |1  <br/> |Получает непосредственные дочерние узлы начальный узел и никакие потомки в подразделе выше или ниже группах.  <br/> |
|**hsNotebooks** <br/> |2  <br/> |Получает все записные книжки под начальный узел или корневой.  <br/> |
|**hsSections** <br/> |3  <br/> |Получает все разделы под узлом Пуск, включая разделы в подразделе группы и групп разделов.  <br/> |
|**hsPages** <br/> |4  <br/> |Получает все страницы под начальный узел, включая все страницы в подразделе группы и групп разделов.  <br/> |

|:-----|:-----|:-----|
|**npsDefault** <br/> |0  <br/> |Создает страницу, которая содержит стиль страницы по умолчанию.  <br/> |
|**npsBlankPageWithTitle** <br/> |1  <br/> |Создается пустая страница с заголовком.  <br/> |
|**npsBlankPageNoTitle** <br/> |2  <br/> |Создает пустую страницу, которая не имеет заголовка.  <br/> |

|**nfoNetwork** <br/> |2  <br/> |Позволяет UNC или записных книжек SharePoint.  <br/> |
|**nfoWeb** <br/> |4  <br/> |Позволяет OneDrive записных книжек.  <br/> |
|**nfoNoWacUrl** <br/> |8  <br/> |Любой записных книжек в расположениях, не связанные с веб-клиента.  <br/> |

## <a name="recentresulttype"></a>RecentResultType
<a name="odc_RecentResultType"> </a>

При передаче в метод **SetRecentResults** объекта **IQuickFilingDialog** указывает, какие список последних результатов будет отображаться, если отображается диалоговое окно быстрого контроля над заполнением записей. Последние списки результатов используются для отслеживания набора OneNote расположения, в которых пользователь выбирает в диалоговом окне быстрого контроля над заполнением записей. Существует три списка последних результатов в OneNote 2013, используемые для отслеживания хранения, поиска и связывания действия. Это перечисление является новым в OneNote 2013. 
  
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**rrtNone** <br/> |0  <br/> |Задает список не последних результатов для отображения.  <br/> |
|**rrtFiling** <br/> |1  <br/> |Задает список последних результатов «Хранение» для отображения.  <br/> |
|**rrtSearch** <br/> |2  <br/> |Задает список последних результатов «Поиск» для отображения.  <br/> |
|**rrtLinks** <br/> |3  <br/> |Задает список последних результатов «Ссылки» для отображения.  <br/> |

- **OneNote15.Application.FindMeta**
    
- **OneNote15.Application.FindPages**
    
- **OneNote15.Application.GetHierarchy**
    
- **OneNote15.Application.GetPageContent**
    
- **OneNote15.Application.UpdateHierarchy**
    
- **OneNote15.Application.UpdatePageContent**
    
|**Элемент**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|**xs2007** <br/> |0  <br/> |Указывает схему OneNote 2007.  <br/> |
|**xs2010** <br/> |1  <br/> |Указывает схему OneNote 2010.  <br/> |
|**xs2013** <br/> |2  <br/> |Указывает схему OneNote 2013.  <br/> |
|**xsCurrent** <br/> |2  <br/> |Указывает схему текущей версии OneNote.  <br/> <br/>**Примечание**: не рекомендуется использовать **xsCurrent** в большинстве случаев, как он может вызвать проблемы совместимости с будущими версиями OneNote.
