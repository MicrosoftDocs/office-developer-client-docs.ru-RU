# <a name="sum-function-access-custom-web-app"></a>Функция СУММ (приложение настраиваемых web Access)

Возвращает сумму всех значений в выражении.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

 **Сумма** (*NumericExpression*) 
  
Функция **сумм** содержит следующий аргумент. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *NumericExpression*  <br/> |Выражение, определяющее поле, содержащее числовые данные, которые необходимо добавить или выражение, которое выполняет вычисления с использованием данных для этого поля. Операнды в *NumericExpression* может включать имя поля в таблице, константа или функция (это может быть встроенным или определяемым пользователем, но не в одной из других статистические функции SQL).  <br/> |
   
## <a name="remarks"></a>Замечания

Функция **сумм** игнорирует записи, содержащие значения Null. 
  
Функция **сумм** может использоваться только для числовых столбцов.
