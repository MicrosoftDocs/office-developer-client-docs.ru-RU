# <a name="len-function-access-custom-web-app"></a>Функция LEN (приложение настраиваемых web Access)

Возвращает число символов из указанного строкового выражения, исключая конечные пробелы.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

 **Функция Len** (*TextExpression*) 
  
Функция **Len** содержит следующий аргумент. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *TextExpression*  <br/> |Строковое выражение.  <br/> |
