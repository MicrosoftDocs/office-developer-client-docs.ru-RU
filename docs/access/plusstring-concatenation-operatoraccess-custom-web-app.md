# <a name="-string-concatenation-operator-access-custom-web-app"></a>+ (Оператор конкатуации строки) (Доступ к настраиваемой веб-приложению)

Знак плюс (+) — это оператор конкатеционных строк, который включает конкатеционные строки. Все остальные манипуляции со строками обрабатываются с помощью функций строки, таких как **Substring.** 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 *TextExpression* **+** *TextExpression* 
  
 *TextExpression*  Любое допустимый текстовое выражение.
