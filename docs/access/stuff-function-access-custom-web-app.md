# <a name="stuff-function-access-custom-web-app"></a>Функция "Прочее" (пользовательское веб-приложение для Access)

Вставляет текстовую строку в другую текстовую строку. Он удаляет указанную длину символов в первой строке в начальной позиции и вставляет вторую строку в первую строку в начальной позиции.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Материалы** (*интотекстекспрессион*, *Start*, *length*, *систекстекспрессион*) 
  
Функция **сведения содержит указанные ниже** аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *интотекстекспрессион*  <br/> |Текстовое выражение, задающее текст, в который будет вставлен текст, заданный параметром *систекстекспрессион* .  <br/> |
| *Начало*  <br/> |Целое значение, задающее расположение для начала удаления и вставки. Если значение Start или length отрицательно, возвращается пустая строка. Если Start длиннее первого *интотекстекспрессион* , возвращается пустая строка.  <br/> |
| *Length*  <br/> |Целое число, определяющее количество удаляемых символов. Если длина превышает первый *интотекстекспрессион* , удаление происходит до последнего символа в последнем *интотекстекспрессион* .  <br/> |
| *систекстекспрессион*  <br/> |Текстовое выражение Hat указывает текст, который необходимо вставить в *интотекстекспрессион* . Это выражение заменит длину символов *интотекстекспрессион* , начиная с *Start* .  <br/> |
   
## <a name="remarks"></a>Примечания

Если аргументы *начала* или *длины* отрицательны или если начальная позиция превышает длину первой строки, возвращается пустая строка. Если начальная позиция равна 0, возвращается значение null. Если длина, которую требуется удалить, больше, чем первая строка, она будет удалена до первого символа в первой строке.
