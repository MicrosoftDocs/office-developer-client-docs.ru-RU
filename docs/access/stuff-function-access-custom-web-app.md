# <a name="stuff-function-access-custom-web-app"></a>Функция Stuff (пользовательское веб-приложение Access)

Вставляет текстовую строку в другую текстовую строку. Он удаляет указанную длину символов в первой строке в начале, а затем вставляет вторую строку в первую строку в начале.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Stuff** (*IntoTextExpression,* *Start*, *Length*, *ThisTextExpression*) 
  
Функция **Stuff** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *IntoTextExpression*  <br/> |Текстовое выражение, которое указывает текст, в который будет вставлен текст, указанный *в ThisTextExpression.*  <br/> |
| *Начало*  <br/> |Это значение указывает расположение для начала удаления и вставки. Если начало или длина отрицательные, возвращается строка null. Если начало больше, чем первый  *IntoTextExpression,*  возвращается строка null.  <br/> |
| *Length*  <br/> |Число, которое указывает количество удаляемого символа. Если длина превышает длину первого *intoTextExpression,* удаление происходит до последнего символа последнего *intoTextExpression.*  <br/> |
| *ThisTextExpression*  <br/> |Текстовое выражение определяет текст, вставляемый в *IntoTextExpression.* Это выражение заменит символы длины *IntoTextExpression, начиная* с *начала.*  <br/> |
   
## <a name="remarks"></a>Примечания

Если  *аргументы Start*  или  *Length*  отрицательные или если начальная позиция больше длины первой строки, возвращается нулевая строка. Если в начале 0, возвращается значение null. Если длина, удаляемая, превышает длину первой строки, она удаляется для первого символа в первой строке.
