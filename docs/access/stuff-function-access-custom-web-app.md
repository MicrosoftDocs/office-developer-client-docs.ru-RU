# <a name="stuff-function-access-custom-web-app"></a>Функция (en) (приложение настраиваемых web Access)

Вставляет строку текста в другую текстовую строку. Его удаляет указанное количество символов в первой строке в положении начала и вставляет вторую строку в первой строки в начальной позиции.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **(En)** (*IntoTextExpression*, *запустите*, *длины*, *ThisTextExpression*) 
  
Функция **(en)** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *IntoTextExpression*  <br/> |Выражение текст, текст, в котором будет вставлен текст, заданный параметром *ThisTextExpression* .  <br/> |
| *Start*  <br/> |Целое число, указывающее место начала удаления и вставки. Если запустить или длина отрицательно, возвращается пустая строка. Если запустить длиннее первого *IntoTextExpression* , возвращается пустая строка.  <br/> |
| *Длина*  <br/> |Целое число, указывающее число символов для удаления. Если длина длиннее первого *IntoTextExpression* , удаление выполняется до последнего символа в последней *IntoTextExpression* .  <br/> |
| *ThisTextExpression*  <br/> |Шапка выражение текст текст для вставки в *IntoTextExpression* . Это выражение заменит длина символов *IntoTextExpression* , начиная с *запуска* .  <br/> |
   
## <a name="remarks"></a>Замечания

Если *запустить* или *Длина* аргументов отрицательные или позицию больше, чем длина первой строки, возвращается пустая строка. Если начальное положение равно 0, возвращается значение null. Если длина удаление содержит более первой строки, она удаляется первый символ в первой строке.
