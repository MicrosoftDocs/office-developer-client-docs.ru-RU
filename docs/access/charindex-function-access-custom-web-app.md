# <a name="charindex-function-access-custom-web-app"></a>Функция CharIndex (пользовательское веб-приложение для Access)

Выполняет поиск в текстовом выражении другого текстового выражения и возвращает его начальную позицию, если она найдена.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

**CharIndex** (*Текстекспрессион*, *Висинтекст*, [*Start*]) 
  
|**Имя аргумента**|**Обязательный**|**Описание**|
|:-----|:-----|:-----|
| *Текстекспрессион*  <br/> |Да  <br/> |Текстовое выражение, содержащее искомый текст.  <br/> |
| *Висинтекст*  <br/> |Да  <br/> |Текстовое выражение, в котором выполняется поиск.  <br/> |
| *Start*  <br/> |Нет  <br/> |Целое число, определяющее расположение в *висинтекст* для начала поиска. Если параметр *Start* не указан, это отрицательное число или 0, поиск начинается с начала *висинтекст* .  <br/> |
   
## <a name="remarks"></a>Примечания

Если значение *текстекспрессион* или *висинтекст* равно NULL, *charIndex* возвращает null. 
  
Если *текстекспрессион* не найдено в *висинтекст*, то *charIndex* возвращает 0. 
  
Возвращаемая позиция начинается с 1, а не от 0.
