# <a name="round-function-access-custom-web-app"></a>Функция Round (приложение настраиваемых web Access)

Возвращает числовое значение, округленное или сокращается до указанной длины или точности.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Round** (*Номер*, *точности* [ *TruncateInsteadOfRound* ]) 
  
Функция **Round** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Число*  <br/> |Числовое выражение.  <br/> |
| *Точность*  <br/> |Точность, к которым будет округлено *номер* .  *Точность* должна быть числовое выражение. Когда *точности* — это положительное число, *число* округляется до число десятичных знаков заданной длины. *Когда *точности* отрицательное значение, оно округляется слева от десятичной запятой, как указано в длину.*  <br/> |
| *TruncateInsteadOfRound*  <br/> |Тип операции для выполнения. Когда опускается, или значение 0, *число* округляется. *Если указано значение, отличное от 0, оно округляется.* Значение по умолчанию равно 0.  <br/> |
   
## <a name="remarks"></a>Замечания

 **Round** всегда возвращает значение. Если длина отрицательные и больше, чем количество цифр перед десятичной запятой, **Round** возвращает 0.
