# <a name="rundatamacro-macro-action-access-custom-web-app"></a>Макрос ЗапускМакросаДанных (приложение настраиваемых web Access)

**ЗапускМакросаДанных** можно использовать для запуска автономного макроса данных. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="setting"></a>Параметр

**ЗапускМакросаДанных** использует следующий аргумент. 
  
|**Аргумент действия**|**Описание**|
|:-----|:-----|
| _Имя макроса_ <br/> |Имя макроса данных для запуска.  <br/> |
   
## <a name="remarks"></a>Замечания

При выборе макрос данных, которая должна работать в конструктор макросов Access определяет необходимость в макрос данных параметров. Если макрос данных требуются параметры, текстовых полей отображаются для ввода в аргументах.
  
При запуске макроса, содержащего **ЗапускМакросаДанных** и достигает **ЗапускМакросаДанных** , Access запускает макрос называемое данных. По завершении макрос называемое данных Access возвращает исходное макрос и выполняется следующее действие.
