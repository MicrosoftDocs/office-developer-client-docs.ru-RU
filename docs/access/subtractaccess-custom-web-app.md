# <a name="--subtract-access-custom-web-app"></a>- (Вычитать) (Доступ к настраиваемого веб-приложения)

Вычитает два номера. Этот оператор арифметической вычитания также может вычитать номер, в течение нескольких дней, с даты.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 *выражение*   -   *выражение* 
  
 *выражение*  Любое допустимые выражения любого из типов данных в числовой категории, за исключением **типа битовых** данных. Нельзя использовать для типов данных о дате или времени. 
  
## <a name="return-value"></a>Возвращаемое значение

Возвращает тип данных аргумента с более высоким приоритетом.
