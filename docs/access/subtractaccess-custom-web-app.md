# <a name="--subtract-access-custom-web-app"></a>-(Вычитание) (пользовательское веб-приложение для Access)

Вычитает два числа. Этот арифметический оператор вычитания также может вычесть число в днях от даты.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 *expression*  -  *выражение* выражения 
  
 *Expression (выражение* )  Любое допустимое выражение любого из типов данных в числовой категории, кроме типа данных **bit** . Не может использоваться с типами данных Date и Time. 
  
## <a name="return-value"></a>Возвращаемое значение

Возвращает тип данных аргумента с более высоким приоритетом.
