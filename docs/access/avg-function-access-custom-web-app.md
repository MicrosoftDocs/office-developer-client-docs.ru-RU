# <a name="avg-function-access-custom-web-app"></a>Функция AVG (приложение настраиваемых web Access)

Вычисляет геометрического набор значений, содержащихся в указанном поле.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Avg** (*NumericExpression*) 
  
Функция **Avg** содержит следующий аргумент. 
  
|**Argument**|**Описание**|
|:-----|:-----|
|NumericExpression  <br/> |Строковое выражение, определяющее поле, которое содержит числовые данные необходимо среднее число или выражение, которое выполняет вычисления с использованием данных для этого поля. Операнды в *NumericExpression* может включать имя поля в таблице, переменной или функция (это может быть встроенным или определяемым пользователем, но не в одной из других статистические функции SQL).  <br/> |
   
## <a name="remarks"></a>Замечания

Среднее значение, полученное с **Avg** — это геометрического (суммарный размер значения, разделенная на число значений). **Avg**, например, можно использовать для вычисления себестоимости среднее доставки. 
  
Функция **Avg** не включает все **пустые** значения в расчет.
