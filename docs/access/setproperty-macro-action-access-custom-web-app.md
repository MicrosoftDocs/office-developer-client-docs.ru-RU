# <a name="setproperty-macro-action-access-custom-web-app"></a>Действия макроса SetProperty (приложение настраиваемых web Access)

Действия **SetProperty** можно использовать для задания свойства для элемента управления в представлении. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="setting"></a>Параметр

Действия **SetProperty** содержит аргументы, перечисленные в следующей таблице. 
  
|**Аргумент действия**|**Описание**|
|:-----|:-----|
| _Имя элемента управления_ <br/> |Введите имя в поле или элемент управления, для которого требуется задать значение свойства. Оставьте данный аргумент пустым, чтобы задать свойства для представления.  <br/> |
| _Свойство_ <br/> |Выберите свойство, которое требуется установить. **В этой статье представлен список свойств, которые можно настроить с помощью этого действия см.**  <br/> |
| _Значение_ <br/> |Введите значение, является ли данное свойство должно быть задано. Для свойства, значения которых являются Да или нет, используйте **значение -1** для Да и **0** для нет.  <br/> |
   
## <a name="remarks"></a>Замечания

Действия **SetProperty** можно использовать для настройки следующих свойств элемента управления: 
  
- Caption
    
- Включена
    
- Цвет текста
    
- Значение
    
- Visible
    
Если ввести недопустимое значение ** *значение* ** аргумент, ошибка не происходит, но доступа могут измените свойство на разные значения, в зависимости от того, как оно интерпретирует аргумента.
