# <a name="gotorecord-macro-action-access-custom-web-app"></a>Действие макрокоманды GoToRecord (пользовательское веб-приложение для Access)

Вы можете использовать действие **GoToRecord** , чтобы сделать указанную запись текущей записью в представлении. 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="setting"></a>Параметр

Макрокоманда **GoToRecord** имеет следующие аргументы. 
  
|**Аргумент макрокоманды**|**Описание**|
|:-----|:-----|
|**Record** <br/> |Запись для создания текущей записи. Выберите **предыдущий**, **следующий**, **первый**или **последний**. Значение по умолчанию — **Next (далее**).  <br/> |
