# <a name="gotorecord-macro-action-access-custom-web-app"></a>Макрос НаЗапись (приложение настраиваемых web Access)

Чтобы сделать указанную запись текущей в представлении можно использовать действие **НаЗапись** . 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="setting"></a>Параметр

Действие **НаЗапись** имеет следующие аргументы. 
  
|**Аргумент действия**|**Описание**|
|:-----|:-----|
|**Запись** <br/> |Сделать текущей запись. Выберите **предыдущие**, **Далее**, **первого**или **последнего**. Значение по умолчанию — **Далее**.  <br/> |
