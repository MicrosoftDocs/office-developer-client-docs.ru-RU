# <a name="lookuprecord-data-block-access-custom-web-app"></a>Блок данных LookupRecord (Пользовательский доступ к веб-приложению)

Блок **данных LookupRecord** выполняет набор действий на определенной записи. 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
> [!NOTE]
> Блок **данных LookupRecord** доступен только в макросах данных. 
  
## <a name="setting"></a>Setting

Действие **SetField** имеет следующие аргументы. 
  
|**Аргумент**|**Обязательный**|**Описание**|
|:-----|:-----|:-----|
| _В_ <br/> |Да  <br/> |Строка, определяемая для работы записи. Аргумент *In* может содержать имя таблицы, выбранный запрос или SQL.  <br/> |
| _Где состояние_ <br/> |Нет  <br/> |Строковая экспрессия, используемая для ограничения диапазона данных, на которых выполняется блок данных **LookupRecord.** Например, критерии часто эквивалентны пункту WHERE в SQL, без слова WHERE. Если не указаны критерии, блок **данных LookupRecord** работает на всем домене, указанном аргументом *In.* Любое поле, включенное в критерии, также должно быть полем  *в In*  .  <br/> |
| _Alias_ <br/> |Нет  <br/> |Строка, которая предоставляет альтернативное имя для записи, указанной аргументом *In.* Часто используется для сокращения имени таблицы для последующих ссылок, чтобы предотвратить возможные неоднозначные ссылки. Если  *псевдоним не*  указан, в качестве псевдонима будет использоваться таблица или имя запроса.  <br/> |
   
## <a name="remarks"></a>Примечания

Если критерии, указанные аргументами In  *и*  *Where Condition,*  задают несколько записей, блок данных **LookupRecord** будет работать только на первой записи. 
  
Если запись не удовлетворяет положение *"Где* состояние" или "In" не содержит записей, **то LookupRecord** создает пустую запись, в которой все поля содержат значение **Null.**
