# <a name="replace-function-access-custom-web-app"></a>Замените функцию (приложение настраиваемых web Access)

Заменяет все вхождения указанного строкового значения другим строковое значение.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

 **Замена** (*TextExpression*, *шаблон* *замены*) 
  
Функция **Replace** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *TextExpression*  <br/> |Строковое выражение для поиска.  <br/> |
| *Шаблон*  <br/> |Подстрока для поиска.  *Шаблон* не может быть пустой строкой (»»).  <br/> |
| *Замена*  <br/> |Строка замены.  <br/> |
