# <a name="changeview-macro-action-access-custom-web-app"></a>Действия макроса ChangeView (приложение настраиваемых web Access)

Действие **ChangeView** можно использовать для перехода между режимами на месте. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="setting"></a>Параметр

Действие **ChangeView** состоит из следующих аргументов. 
  
|**Аргумент действия**|**Обязательное**|**Описание**|
|:-----|:-----|:-----|
|Таблица  <br/> |Да  <br/> |Имя таблицы.  <br/> |
|Представление  <br/> |Да  <br/> |Имя представления, чтобы открыть.  <br/> |
|Где  <br/> |Нет  <br/> |Если указано, заменяет условие источника записей объекта.  <br/> |
|Order By  <br/> |Нет  <br/> |Строковое выражение, которое включает в себя имя поля или полей, по которому выполняется сортировка записей и необязательные ASC или DESC ключевых слов. По умолчанию этот аргумент не задан.  <br/> |
   
## <a name="remarks"></a>Замечания

При вызове действие **ChangeView** сортировки или фильтрации применяется пользователем снят. 
  
Аргумент *OrderBy* — это имя в поле или поля, в которых используется для сортировки записей. При использовании более одного имени поля, разделяйте имена запятыми (,). 
  
Если значение аргумента *OrderBy* , записи сортируются в порядке возрастания по умолчанию.
