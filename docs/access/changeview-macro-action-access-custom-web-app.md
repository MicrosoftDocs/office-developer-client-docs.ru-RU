# <a name="changeview-macro-action-access-custom-web-app"></a>Действие Макрос ChangeView (Доступ к настраиваемой веб-приложению)

Вы можете использовать действие **ChangeView** для перемещения между представлениями на месте. 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="setting"></a>Параметр

Действие **ChangeView** имеет следующие аргументы. 
  
|**Аргумент макрокоманды**|**Обязательный**|**Описание**|
|:-----|:-----|:-----|
|Table  <br/> |Да  <br/> |Имя открытой таблицы.  <br/> |
|Просмотреть  <br/> |Да  <br/> |Имя открываемого представления.  <br/> |
|Где  <br/> |Нет  <br/> |Если задано, заменит состояние Where источника записи объекта.  <br/> |
|Order By  <br/> |Нет  <br/> |Строковая фраза, включаемая имя поля или полей для сортировки записей, а также необязательные ключевые слова ASC или DESC. По умолчанию этот аргумент является пустым.  <br/> |
   
## <a name="remarks"></a>Примечания

Любая сортировка или фильтрация, применяемая пользователем, очищается при призыве **действия ChangeView.** 
  
Аргумент  *OrderBy*  — это имя поля или полей, на которых необходимо сортировать записи. При использовании более чем одного имени поля разделять имена с запятой (,). 
  
При наборе  *аргумента OrderBy*  записи сортироваться по умолчанию в порядке восходящей.
