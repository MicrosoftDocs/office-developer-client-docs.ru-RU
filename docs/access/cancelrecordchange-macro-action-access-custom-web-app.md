# <a name="cancelrecordchange-macro-action-access-custom-web-app"></a>CancelRecordChange Macro Action (Access custom web app)

Действие **CancelRecordChange** можно использовать для отмены изменений, примененных к записи в блоке данных **[CreateRecord](createrecord-data-block-access-custom-web-app.md)** или **[EditRecord](editrecord-data-block-access-custom-web-app.md)** до совершения изменений. 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
> [!NOTE]
> Действие **CancelRecordChange** доступно только в макросах данных. 
  
## <a name="remarks"></a>Примечания

При вызове **действия CancelRecordChange** блок **данных CreateRecord** или **EditRecord** немедленно выходит.
