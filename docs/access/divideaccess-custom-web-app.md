# <a name="-divide-access-custom-web-app"></a>/ (Разделить) (приложение настраиваемых web Access)

Делит одно число на другое.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 *дивидендов*  /  *делителя* 
  
 *дивидендов*  — Числовое выражение для деления. Может быть любым допустимым выражением одного типа данных из категории числовых типов данных, за исключением тип данных даты и времени. 
  
 *Делителя*  — Числовое выражение для деления дивидендов. Может быть любым допустимым выражением одного типа данных из категории числовых типов данных, за исключением тип данных даты и времени. 
  
## <a name="return-type"></a>Возвращаемый тип

Возвращает тип данных аргумента с более высокий приоритет. 
  
Если целое число *дивидендов* на целый *делителя* , результатом является целое число, имеет дробная часть результат усечено. 
  
## <a name="remarks"></a>Замечания

Фактическое значение, возвращаемое аудио- и оператор — частное от деления первого выражения на второе выражение.
