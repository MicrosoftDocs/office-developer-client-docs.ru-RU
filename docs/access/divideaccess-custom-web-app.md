# <a name="-divide-access-custom-web-app"></a>/ (Разделяй) (Доступ к настраиваемой веб-приложению)

Делит один номер на другой.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 *дивиденды*   /   *divisor* 
  
 *дивиденды*  Является ли числовая выражение разделяемой. Может быть любым допустимым выражением любого из типов данных категории числовых типов данных, за исключением типа данных даты. 
  
 *Divisor*  Это числовая выражение, с помощью которого можно разделить дивиденды. Может быть любым допустимым выражением любого из типов данных категории числовых типов данных, за исключением типа данных даты. 
  
## <a name="return-type"></a>Возвращаемый тип

Возвращает тип данных аргумента с более высоким приоритетом. 
  
Если дивиденды  на несколько этапов делятся  на разделителем, то в результате будет отсечена часть результата. 
  
## <a name="remarks"></a>Примечания

Фактическое значение, возвращаемое оператором, — это коэффициент первого выражения, разделенного вторым выражением.
