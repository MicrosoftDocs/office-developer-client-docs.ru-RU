# <a name="power-function-access-custom-web-app"></a>Функции Power (приложение настраиваемых web Access)

Возвращает значение указанного выражения в указанную степень.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

 **Power** (*NumericExpression*, *Power*) 
  
Функция **Power** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *NumericExpression*  <br/> |Числовое выражение.  <br/> |
| *Power*  <br/> |Питания, к которой необходимо вызвать *NumericExpression* .  <br/> |
