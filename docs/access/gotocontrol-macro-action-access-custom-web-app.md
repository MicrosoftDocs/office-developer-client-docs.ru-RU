# <a name="gotocontrol-macro-action-access-custom-web-app"></a>Макрос КЭлементуУправления (приложение настраиваемых web Access)

**КЭлементуУправления** можно использовать для перемещения фокуса на указанный элемент управления в текущей записи открыть представление. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="setting"></a>Параметр

**КЭлементуУправления** использует следующий аргумент. 
  
|**Аргумент действия**|**Описание**|
|:-----|:-----|
|**Имя элемента управления** <br/> |Имя поля или элемента управления, где будут фокус. Этот параметр является обязательным.  <br/> |
   
## <a name="remarks"></a>Замечания

Это действие можно использовать отдельного поля или элемент управления имеет фокус. Это действие также можно использовать для перехода в форме при выполнении определенных условий. Например, если пользователь вводит нет в элемента управления в форме медицинского страхования, фокус автоматически можно пропустить управления имя супруга/партнера и перемещение к следующему элементу управления.
