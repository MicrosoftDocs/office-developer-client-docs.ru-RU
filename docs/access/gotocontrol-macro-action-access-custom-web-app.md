# <a name="gotocontrol-macro-action-access-custom-web-app"></a>Макрокоманда GoToControl (пользовательское веб-приложение для Access)

Макрокоманда **GoToControl** позволяет переместить фокус на указанный элемент управления в текущей записи открытого представления. 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="setting"></a>Параметр

Макрокоманда **GoToControl** имеет указанный ниже аргумент. 
  
|**Аргумент макрокоманды**|**Описание**|
|:-----|:-----|
|**Имя элемента управления** <br/> |Имя поля или элемента управления, куда необходимо переместить фокус. Обязательный аргумент.  <br/> |
   
## <a name="remarks"></a>Замечания

Эта макрокоманда позволяет переместить фокус на определенное поле или на другой элемент управления. Кроме того, можно использовать эту макрокоманду для навигации в форме в соответствии с определенными условиями. Например, если пользователь введет "Нет" в элементе управления "Брак" формы медицинского страхования, фокус может сразу автоматически переместиться на элемент управления, следующий за элементом управления "Имя партнера".
