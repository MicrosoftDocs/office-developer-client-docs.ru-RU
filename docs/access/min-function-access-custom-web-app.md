# <a name="min-function-access-custom-web-app"></a>Min Function (Доступ к настраиваемой веб-приложению)

Возвращает минимальное значение выражения в запросе или таблице.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Min** *(Выражение)* 
  
Функция **Min** содержит следующий аргумент. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Expression*  <br/> |Строковая экспрессия, определяя поле, содержаще данные, которые необходимо оценить, или выражение, которое выполняет вычисление с помощью данных в этом поле. Operands в *Expression* могут включать имя настольного поля, константы или функции (которая может быть внутренней или пользовательской, но не одной из других SQL совокупных функций).  <br/> |
