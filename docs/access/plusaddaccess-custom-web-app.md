# <a name="-add-access-custom-web-app"></a>+ (Добавить) (пользовательское веб-приложение для Access)

Складывает два числа. Этот арифметический оператор сложения также может добавлять число (в днях) к дате. 
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **  +  *выражение* выражения 
  
 *Expression (выражение* )  Любое допустимое выражение любого из типов данных в числовой категории, кроме типа данных **bit** . Не может использоваться с типами данных Date и Time. 
  
## <a name="return-value"></a>Возвращаемое значение

Возвращает тип данных аргумента с более высоким приоритетом.
