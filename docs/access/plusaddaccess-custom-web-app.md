# <a name="-add-access-custom-web-app"></a>+ (Добавить) (приложение настраиваемых web Access)

Добавление двух чисел. Данный арифметический оператор сложения также можно добавить номера, в днях, к дате. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

 *выражение*  +  *выражение* 
  
 *выражение*  Любое допустимое выражение любого типа данных числовой категории, кроме типа данных **bit** . Не может использоваться с типами данных даты и времени.
