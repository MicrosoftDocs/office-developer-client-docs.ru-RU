# <a name="copyto-method-ado"></a>Метод CopyTo (ADO)

**Область применения**: Access 2013, Office 2013

Копирует указанное количество символов или bytes (в зависимости от [типа)](type-property-ado-stream.md)в [потоке](stream-object-ado.md) на другой **объект Stream.**

## <a name="syntax"></a>Синтаксис

*Stream*. CopyTo *DestStream*, *NumChars*

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*DestStream* |Переменная значение объекта, содержа которое содержит ссылку на объект open **Stream.** Текущий **поток** копируется в поток **назначения,** указанный *DestStream.* Поток **назначения уже** должен быть открыт. Если нет, возникает ошибка времени запуска.<br/><br/>**ПРИМЕЧАНИЕ.** Параметр *DestStream* не может быть прокси-сервером объекта **Stream,** так как для этого требуется доступ к частному интерфейсу объекта **Stream,** который не может быть удален для клиента.|
|*NumChars* |Необязательный параметр. Значение **Integer,** которое указывает количество bytes или символов, которые будут скопированы из текущего положения в **источнике Stream** в поток **назначения**. Значение по умолчанию —1, которое указывает, что все символы или bytes копируется из текущей позиции в [EOS.](eos-property-ado.md)|

## <a name="remarks"></a>Примечания

Этот метод копирует указанное количество символов или bytes, начиная с текущей позиции, указанной [свойством Position.](position-property-ado.md) Если указанное число превышает доступное количество bytes до **EOS,** то копируется только символы или bytes из текущего положения в **EOS.** Если значение *NumChars* —1 или пропущено, все символы или bytes, начиная с текущего положения, копируется.

Если в потоке назначения есть существующие символы или bytes, все содержимое за пределами точки, где заканчивается копия, остается и не усечено. **Позиция** становится byte сразу после последнего скопированного byte. Если вы хотите усечение этих bytes, позвоните [SetEOS](seteos-method-ado.md).

**CopyTo** следует использовать для копирования  данных в поток назначения того  же типа, что и исходный поток **(параметры** свойств type являются как **adTypeText,** так и **adTypeBinary).** Для объектов **text Stream** можно изменить параметр свойства  [Charset](charset-property-ado.md) в потоке назначения для перевода из одного набора символов в другой. Кроме того, **объекты text Stream** можно успешно скопировать в двоичные объекты **Stream,** но объекты **двоичного** потока не могут быть скопированы в объекты text **Stream.**
