# <a name="printout-macro-action"></a>Макрокоманда PrintOut

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **Печать** для печати активного объекта в открытой базе данных. Можно распечатать таблицы, отчеты, формы, страницы доступа к данным и модули.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Примечания

Это действие аналогично выбору объекта, щелчком на вкладке **файл** и нажатием кнопки **Печать**. Однако это действие не появление диалогового окна " **Печать** ".

> [!TIP]
> Если вы часто используете конкретные параметры печати, создайте макрос, содержащий действие **распечатки** , с помощью этих параметров в аргументах.

Аргументы этого действия соответствуют параметрам в диалоговом окне " **Печать** ". Однако в отличие от диалогового **окна "Макрокоманда"** **найти и заменить** "Параметры аргумента не являются общими с параметрами диалогового окна" **Печать** ".

Чтобы выполнить действие по **распечатке** в модуле Visual Basic для приложений (VBA), используйте метод **PrintOut** объекта **DoCmd** .
