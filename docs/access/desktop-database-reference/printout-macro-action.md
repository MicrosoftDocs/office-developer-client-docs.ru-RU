# <a name="printout-macro-action"></a>Макрокоманда PrintOut

**Применимо к**: Access 2013, Office 2013

**Макрокоманду** можно использовать для печати активного объекта в открытой базы данных. Можно распечатать таблицы данных, отчеты, формы, страницы доступа к данным и модули.

> [!NOTE]
> Это действие не разрешено, если база данных не является доверенной.

## <a name="remarks"></a>Замечания

Это действие аналогично при выборе объекта, щелкнув вкладку **файл** и затем выбрав команду **Печать**. С помощью этого действия тем не менее, не **Print** откроется диалоговое окно.

> [!TIP]
> При наличии определенного параметры печати часто используемых создайте макрос, содержащий **макрокоманду с этими параметрами аргументов** .

Аргументы для этого действия соответствующие параметры в диалоговом окне " **Печать** ". Тем не менее в отличие от **НайтиЗапись** и диалоговое окно **Найти и заменить** параметры аргумент не предоставлен параметры диалогового окна **Печать** .

Чтобы запустить **макрокоманду** в Visual Basic для приложений (VBA) модуль, используйте метод **PrintOut** **объекта** .
