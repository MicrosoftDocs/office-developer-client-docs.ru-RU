# <a name="command-property-adox"></a>Свойство Command (ADOX)


**Область применения**: Access 2013, Office 2013

Указывает объект ADO [Command,](command-object-ado.md) который можно использовать для создания или выполнения процедуры.

## <a name="settings-and-return-values"></a>Параметры и значения возврата

Задает или возвращает допустимый объект ADO **Command.**

## <a name="remarks"></a>Примечания

Ошибка возникает при получении и настройке этого свойства, если поставщик не поддерживает сохраняющиеся команды.
