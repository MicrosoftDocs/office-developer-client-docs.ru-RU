# <a name="updating-data"></a>Обновление данных


**Область применения**: Access 2013, Office 2013

Поведение и функции обновления во многом зависят от режима обновления (тип блокировки), типа курсора и положения курсора.

Используйте метод **Update** , чтобы сохранить изменения, внесенные в текущую запись объекта **Recordset** , с момента вызова метода **AddNew** или с помощью изменения любых значений полей в существующей записи. Объект **Recordset** должен поддерживать обновления.

Если объект **Recordset** поддерживает пакетное обновление, можно кэшировать несколько изменений в одну или несколько записей локально, пока не будет вызван метод **UpdateBatch** . Если вы редактируете текущую запись или добавляете новую запись при вызове метода **UpdateBatch** , ADO автоматически вызывает метод **Update** для сохранения всех ожидающих изменений в текущей записи, прежде чем передавать пакетные изменения в поставщики.

Текущая запись остается текущей, когда вы вызываете метод **Update** или **UpdateBatch** .
