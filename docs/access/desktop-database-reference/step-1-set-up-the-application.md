## <a name="step-1-set-up-the-application"></a>Этап 1: Настройка приложения

Создайте виртуальный каталог IIS, с именем «XMLPersist» с разрешениями скрипта. Создайте две новые текстовые файлы в папке, на который виртуальный каталог указывает, один именованный «XMLResponse.asp,» другие именованные «Default.htm».
