# <a name="connectionstring-property-ado"></a>Свойство ConnectionString (ADO)


**Область применения**: Access 2013, Office 2013

Указывает сведения, используемые для установления подключения к источнику данных.

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает или возвращает **строку.**

## <a name="remarks"></a>Заметки

Используйте свойство **ConnectionString,** чтобы указать источник данных, передав подробную строку подключения, содержащую серию аргументов  *=* значений, разделенных с заточки.

ADO поддерживает пять аргументов для свойства **ConnectionString;** любые другие аргументы передаются непосредственно поставщику без обработки ADO. Аргументы, поддерживаемые ADO, следуют следующим образом.

После того как вы настроили свойство **ConnectionString** и открыли объект [Connection,](connection-object-ado.md) поставщик может изменить содержимое свойства, например, соединая имена аргументов, определенных aDO, с их эквивалентами поставщика.

Свойство **ConnectionString** автоматически наследует значение, используемого для аргумента *ConnectionString* метода [Open,](open-method-ado-connection.md) поэтому вы можете переопредить текущее свойство **ConnectionString** во время вызова метода **Open.**

Так как *аргумент file Name* приводит к загрузке ADO связанного поставщика, невозможно передать аргументы *Provider* и *File Name.*

Свойство **ConnectionString** считывать и записывать при закрытии подключения и только для чтения, когда оно открыто.

Дубликаты аргумента в **свойстве ConnectionString** игнорируются. Используется последний экземпляр любого аргумента.

**Использование удаленной службы данных** При этом свойство **ConnectionString** может включать только параметры  удаленного поставщика и  *удаленного* сервера.
