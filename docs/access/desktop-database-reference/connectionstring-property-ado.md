# <a name="connectionstring-property-ado"></a>Свойство ConnectionString (ADO)


**Область применения**: Access 2013, Office 2013

Указывает сведения, используемые для установления подключения к источнику данных.

## <a name="settings-and-return-values"></a>Параметры и значения возврата

Задает или возвращает значение **String.**

## <a name="remarks"></a>Примечания

Используйте **свойство ConnectionString,** чтобы указать источник данных, передав подробную строку подключения, содержащую серию аргументов  *=* утверждения значений, разделенных зайколонами.

ADO поддерживает пять аргументов для свойства **ConnectionString;** любые другие аргументы передаются непосредственно поставщику без обработки ADO. Аргументы, поддерживаемые ADO, следуют следующим образом.

После задания свойства **ConnectionString** и открытия объекта [Подключения](connection-object-ado.md) поставщик может изменить содержимое свойства, например, с помощью сопоставления имен аргументов, определенных ADO, с эквивалентами поставщика.

Свойство **ConnectionString** автоматически наследует значение, используемого для аргумента *ConnectionString* метода Open, поэтому можно переопрепредить текущее свойство **ConnectionString** во время вызова метода **Open.** [](open-method-ado-connection.md)

Так как *аргумент Имя файла* вызывает загрузку ADO связанного поставщика, вы не можете передать аргументы *поставщика* и *имени* файлов.

Свойство **ConnectionString** считыв или записывалось при закрытии подключения и только для чтения, когда оно открыто.

Дубликаты аргумента в **свойстве ConnectionString** игнорируются. Используется последний экземпляр любого аргумента.

**Удаленное использование службы данных** При этом свойство **ConnectionString** может включать только параметры  удаленного поставщика и  *удаленного* сервера.
