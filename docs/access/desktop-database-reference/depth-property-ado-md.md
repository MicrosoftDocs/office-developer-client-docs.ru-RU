# <a name="depth-property-ado-md"></a>Depth Property (ADO MD)


**Применимо к**: Access 2013 | Office 2013

Показывает, сколько уровней между [уровень](level-object-ado-md.md) и корень иерархии.

## <a name="return-values"></a>Return Values

Возвращает значение типа integer и доступен только для чтения.

## <a name="remarks"></a>Замечания

**Уровень** в корне иерархии имеет значение **глубины** нуль (0).
