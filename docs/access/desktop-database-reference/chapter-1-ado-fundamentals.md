# <a name="chapter-1-ado-fundamentals"></a>Глава 1. Основы ADO

**Область применения**: Access 2013, Office 2013

В этой главе представлено введение в библиотеку ADO. В нем рассказывается, что можно делать с помощью ADO, рассматриваются объекты в иерархии ADO и представлено простое приложение ADO, которое использует многие объекты ADO для получения, редактирования и обновления данных из источника данных. Наконец, в этой главе рассматриваются две проблемы, которые важны для написания приложений ADO: поставщики OLE DB и ошибки.

В этой главе рассматриваются следующие темы:

- [Что можно делать с помощью ADO](what-you-can-do-with-ado.md)
- [сведения об ошибках](errors.md);
- [HelloData: простое приложение ADO (ADO)](hellodata-a-simple-ado-application.md)
- [Поставщики OLE DB (ADO)](ole-db-providers.md)
