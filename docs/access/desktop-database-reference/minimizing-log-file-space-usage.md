# <a name="minimizing-log-file-space-usage"></a>Минимизация места, занимаемого файлом журнала

**Область применения**: Access 2013, Office 2013

Файл журнала может быстро заполниться (остановив сервер), если в базе данных SQL Server большой объем активности. Вы можете установить для файла журнала значение **Truncate в** контрольной точке, чтобы значительно продлить срок службы файла журнала для базы данных.

**Чтобы включить усечение контрольной точки в Microsoft SQL Server 6.5**

1.  Запустите Microsoft SQL Server Enterprise Manager, откройте дерево сервера, а затем откройте дерево "Устройства базы данных".

2.  Дважды щелкните имя базы данных, в которой будет включена эта функция.

3.  На **вкладке "База** данных" выберите **"Усеченное".**

4.  На **вкладке "Параметры"** выберите пункт **"Усечение входа** в контрольную точку" и нажмите кнопку **"ОК".**

**Чтобы включить усечение контрольной точки в Microsoft SQL Server 7.0**

1.  Запустите Microsoft SQL Server Enterprise Manager, откройте дерево сервера, а затем откройте дерево баз данных.

2.  Щелкните правой кнопкой мыши имя базы данных, в которой будет включена эта функция, и выберите **"Свойства".**

3.  На **вкладке "Параметры"** выберите пункт **"Усечение входа** в контрольную точку" и нажмите кнопку **"ОК".**

Дополнительные сведения о функции **"Усечение в** контрольной точке" см. в Microsoft SQL Server документации.
