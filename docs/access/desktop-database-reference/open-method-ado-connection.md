# <a name="open-method-ado-connection"></a>Open Method (ADO Connection)


**Применимо к**: Access 2013 | Office 2013
 

Открывает подключение к источнику данных.

## <a name="syntax"></a>Синтаксис

*подключение*. Откройте*ConnectionString*, *идентификатор пользователя*, *пароль*, *Параметры*

## <a name="parameters"></a>Параметры

  - *ConnectionString*

  - Необязательный параметр. **Строковое** значение, содержащее сведения о подключении. Свойство [ConnectionString](connectionstring-property-ado.md) для получения дополнительных сведений см на допустимые значения.

  - *Идентификатор пользователя*

  - Необязательный параметр. **Строковое** значение, которое содержит имя пользователя для использования при установке подключения.

  - *Password*

  - Необязательный параметр. **Строковое** значение, содержащее пароль для использования при установке подключения.

  - *Варианты*

  - Необязательный параметр. [ConnectOptionEnum](connectoptionenum.md) значение, определяющее, будет ли этот метод должен возвращать после (синхронно) или до (асинхронно) подключения.

## <a name="remarks"></a>Замечания

С помощью метода **Open** объекта [подключения](connection-object-ado.md) устанавливает физических подключение к источнику данных. После успешного завершения этого метода подключения live можно и выполнение команд его обработки результатов.

Используйте необязательный аргумент *ConnectionString* , чтобы указать либо строку подключения, содержащую последовательность операторов *= значение* *аргумента* , разделенные точкой с запятой или файла ресурсов каталогов с помощью URL-адреса. Свойство **ConnectionString** автоматически наследует значение, используемое для аргумента *ConnectionString* . Таким образом можно использовать свойство **ConnectionString** объекта **подключения** перед открытием, или используйте аргумент *ConnectionString* задать или переопределить параметры подключения к текущей во время вызова метода **Open** .

Если пользователя и пароль, передается в аргументе *ConnectionString* и необязательные аргументы *идентификатор пользователя* и *пароль* , *идентификатор пользователя* и *пароль* аргументы переопределяет значения, заданные в * ConnectionString*.

Когда заключения операции вами через открытое **подключение**, используйте метод [Close](close-method-ado.md) чтобы освободить место на все связанные системные ресурсы. Закрытие объект не удаляется из памяти; можно изменить параметры его свойства и использовать метод **Open** для открытия попытку позже. Чтобы полностью удалить объект из памяти, задайте объектной переменной значение *Nothing*.

**Службы удаленных данных об использовании** При использовании на объект **подключения** со стороны клиента, метод **Open** не устанавливает фактически подключения к серверу пока не открывается [набор записей](recordset-object-ado.md) в объекте **подключений** .
