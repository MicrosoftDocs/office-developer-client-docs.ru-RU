# <a name="quitaccess-macro-action"></a>Макрокоманда QuitAccess

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **куитакцесс** для выхода из Microsoft Access. Действие **куитакцесс** также может указывать один из нескольких вариантов сохранения объектов базы данных перед выходом из Access.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Примечания

Access не выполняет никакие действия, которые следуют за действием **куитакцесс** в макросе.

Это действие можно использовать для закрытия Access без запросов на **Сохранение** диалоговых окон с помощью настраиваемых команд меню или кнопки в форме. Например, у вас может быть Главная форма, используемая для отображения объектов в настраиваемой рабочей области. Эта форма может содержать кнопку **Quit (выйти** ), которая запускает макрос, содержащий действие **куитакцесс** с набором аргументов **Options** , равным **Save All**.

Это действие эквивалентно переходу на вкладку **файл** и нажатие кнопки **выход**. Если при нажатии этой команды имеются несохраненные объекты, то отображаются те же диалоговые окна, что и при использовании параметра **Prompt** в качестве аргумента **Options** действия **куитакцесс** .

Макрокоманду **савеобжект** можно использовать в макросе для сохранения указанного объекта без закрытия доступа или закрытия объекта.

Чтобы запустить действие **куитакцесс** в модуле Visual Basic для приложений (VBA), используйте метод **Quit** объекта **DoCmd** .
