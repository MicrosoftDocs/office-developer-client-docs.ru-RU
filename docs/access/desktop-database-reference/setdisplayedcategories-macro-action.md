# <a name="setdisplayedcategories-macro-action"></a>Макрокоманда SetDisplayedCategories


**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **сетдисплайедкатегориес** для указания категорий, отображаемых в разделе **Перейти к категории** в строке заголовка области навигации. Например, если необходимо запретить пользователям переключать область навигации, чтобы она отображала объекты, отсортированные по **дате создания**, можно использовать это действие для скрытия этого параметра в раскрывающемся списке строки заголовка.

## <a name="remarks"></a>Примечания

  - Заголовок в строке заголовка области навигации указывает, какой фильтр в данный момент активен. Щелкните в любом месте панели, чтобы отобразить раскрывающийся список. Элементы управления действиями макроса отображаются в разделе **Перейти к категории**.

  - Это действие включает или отключает отображение указанных категорий или категорий. Он не выполняет переключения в область навигации. Например, если вы отображаете объекты, отсортированные по **дате создания** , и используете действие **сетдисплайедкатегориес** для отключения параметра **Дата создания** , Access не переключует область навигации в другую категорию.

  - Чтобы выполнить действие **сетдисплайедкатегориес** в модуле VBA, используйте метод **сетдисплайедкатегориес** объекта **DoCmd** .
