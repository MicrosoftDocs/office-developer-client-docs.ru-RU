# <a name="sql-expressions"></a><span data-ttu-id="a2bb6-102">Выражения SQL</span><span class="sxs-lookup"><span data-stu-id="a2bb6-102">SQL expressions</span></span>

<span data-ttu-id="a2bb6-103">**Область применения**: Access 2013, Office 2013</span><span class="sxs-lookup"><span data-stu-id="a2bb6-103">**Applies to**: Access 2013, Office 2013</span></span>

<span data-ttu-id="a2bb6-104">Выражение SQL — это строка, из которой частично или полностью состоит инструкция SQL.</span><span class="sxs-lookup"><span data-stu-id="a2bb6-104">An SQL expression is a string that makes up all or part of an SQL statement.</span></span> <span data-ttu-id="a2bb6-105">Например, метод **FindFirst**, примененный к объекту **Recordset**, использует выражение SQL, состоящее из условий отбора в [предложении WHERE](https://docs.microsoft.com/office/vba/access/Concepts/Structured-Query-Language/where-clause-microsoft-access-sql).</span><span class="sxs-lookup"><span data-stu-id="a2bb6-105">For example, the **FindFirst** method on a **Recordset** object uses an SQL expression consisting of the selection criteria found in an SQL [WHERE clause](https://docs.microsoft.com/office/vba/access/Concepts/Structured-Query-Language/where-clause-microsoft-access-sql).</span></span>

<span data-ttu-id="a2bb6-106">Для выполнения простых арифметических действий и вычисления функций ядро СУБД Microsoft Access использует Microsoft Visual Basic для приложений (VBA).</span><span class="sxs-lookup"><span data-stu-id="a2bb6-106">The Microsoft Access database engine uses the Microsoft Visual Basic for Applications (or VBA) expression service to perform simple arithmetic and function evaluation.</span></span> <span data-ttu-id="a2bb6-107">При этом все операторы, используемые в выражениях SQL ядра СУБД Microsoft Access (за исключением **[Between](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/between-and-operator)**, **[In](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/in-operator-microsoft-access-sql)** и **[Like](https://docs.microsoft.com/office/vba/access/Concepts/Structured-Query-Language/like-operator-microsoft-access-sql)**), определяются службой выражений VBA.</span><span class="sxs-lookup"><span data-stu-id="a2bb6-107">All of the operators used in Microsoft Access database engine SQL expressions (except **[Between](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/between-and-operator)**, **[In](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/in-operator-microsoft-access-sql)**, and **[Like](https://docs.microsoft.com/office/vba/access/Concepts/Structured-Query-Language/like-operator-microsoft-access-sql)**) are defined by the VBA expression service.</span></span> 

<span data-ttu-id="a2bb6-108">Кроме того, VBA позволяет использовать для выражений SQL более 100 функций VBA.</span><span class="sxs-lookup"><span data-stu-id="a2bb6-108">In addition, the VBA expression service offers over 100 VBA functions that you can use in SQL expressions.</span></span> <span data-ttu-id="a2bb6-109">Например, функции VBA можно использовать для создания запроса SQL в режиме конструктора запросов Microsoft Access, а также в режиме DAO с использованием метода **OpenRecordset** в коде Microsoft Visual C++, Microsoft Visual Basic и Microsoft Excel.</span><span class="sxs-lookup"><span data-stu-id="a2bb6-109">For example, you can use these VBA functions to compose an SQL query in the Microsoft Access query Design view, and you can also use these functions in an SQL query in the DAO **OpenRecordset** method in Microsoft Visual C++, Microsoft Visual Basic, and Microsoft Excel code.</span></span>
