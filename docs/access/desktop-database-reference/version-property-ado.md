# <a name="version-property-ado"></a>Свойство Version (ADO)


**Область применения**: Access 2013, Office 2013

Указывает номер версии ADO.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **String,** которое указывает версию.

## <a name="remarks"></a>Примечания

Чтобы **вернуть** номер версии реализации ADO, используйте свойство Version.

Версия поставщика будет доступна в качестве динамического свойства в коллекции [Properties.](properties-collection-ado.md)
