# <a name="command-object-overview"></a>Command Object Overview


**Применимо к**: Access 2013 | Office 2013

С помощью семейства сайтов, методы и свойства объекта **команды** сделайте следующее:

  - Определение исполняемый текст команды (например, инструкции SQL или хранимую процедуру) с помощью свойства **CommandText** .

  - Определение запросы с параметрами и аргументами хранимую процедуру с помощью **параметра** объекты и коллекции **параметров** .

  - Выполнение команды и возврата объекта **набора записей** , если это необходимо, с помощью метода **Execute** .

  - Укажите тип команды с помощью свойства **CommandType** , прежде чем начать выполнение для оптимизации производительности.

  - Определяют, является ли поставщик сохраняет готовую (или скомпилированную) версию команды, прежде чем начать выполнение с помощью свойства **подготовленных** .

  - Задайте число секунд, поставщик будет ожидать команду, чтобы выполнить с помощью свойства **CommandTimeout** .

  - Связывание открытое подключение с помощью объекта **команды** , присвоив свойству **ActiveConnection** .

  - **Имя** свойства для идентификации объекта **команду** как метод на связанный объект **подключения** .

  - Передайте объект **команды** свойства **источника** **записей** для получения данных.
