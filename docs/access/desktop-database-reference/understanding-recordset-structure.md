# <a name="understanding-recordset-structure"></a>Understanding Recordset Structure


**Применимо к**: Access 2013 | Office 2013

Каждый **набор записей** содержит коллекцию **полей** , состоящую из одного или нескольких объектов **поля** . Объект **поля** обычно представляет столбец таблицы. Следующие разделы объясняется, как перемещаться по коллекции **полей** и получить сведения о каждом поле. Затем обсуждаются какие сведения доступны через объект **поля** и способы его использования.

  - [The Field Object](the-field-object.md)

  - [Working with Recordsets](working-with-recordsets.md)
