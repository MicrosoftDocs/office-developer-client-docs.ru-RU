# <a name="understanding-recordset-structure"></a>Структура набора записей

**Область применения**: Access 2013, Office 2013

Каждый **объект Recordset** имеет коллекцию **Fields,** состоящую из одного или нескольких **объектов Field.** Объект **Field** обычно представляет столбец таблицы. 

В следующих темах объясняется, как перейти по коллекции **Fields** и получить сведения о каждом поле. Затем они обсуждают, какие сведения доступны через объект **Field** и как их использовать.

- [Коллекция Fields](the-fields-collection.md)
- [Объект Field](the-field-object.md)
- [Working with Recordsets](working-with-recordsets.md)
