# <a name="understanding-recordset-structure"></a>Структура набора записей

**Область применения**: Access 2013, Office 2013

Каждый **набор записей** содержит коллекцию **Fields** , состоящую из одного или нескольких объектов **field** . Объект **field** обычно представляет столбец таблицы. 

В следующих разделах объясняется, как перемещаться по коллекции **Fields** и получать сведения о каждом поле. Затем мы обсудим, какие сведения доступны вам через объект **field** , и как его использовать.

- [Коллекция Fields](the-fields-collection.md)
- [Объект Field](the-field-object.md)
- [Working with Recordsets](working-with-recordsets.md)
