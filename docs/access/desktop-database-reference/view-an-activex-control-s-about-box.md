# <a name="view-an-activex-controls-about-box"></a>Просмотр элемента управления ActiveX "о программе"

**Применимо к**: Access 2013 | Office 2013

Чтобы просмотреть диалоговое окно **о** версии и сведения об авторских правах для элемента управления ActiveX, щелкните поле **о** свойства в окне свойств Microsoft Access. Справа от поля свойства, нажмите кнопку **Создать** .
