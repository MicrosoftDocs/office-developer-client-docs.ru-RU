# <a name="view-an-activex-control-about-box"></a>Просмотр окна сведений об элементе ActiveX

**Область применения**: Access 2013, Office 2013

Чтобы просмотреть сведения **** о версии и авторских правах для элемента управления ActiveX, введите:

1. Выберите поле свойства **о** свойстве в окне Свойства Microsoft Access.

2. Нажмите кнопку **построить** справа от поля свойство.
