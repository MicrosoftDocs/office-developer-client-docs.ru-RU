# <a name="view-an-activex-control-about-box"></a>Просмотр окна сведений об элементе ActiveX

**Применимо к**: Access 2013, Office 2013

Чтобы просмотреть диалоговое окно **о** вывод сведений о версии и об авторских правах для элемента управления ActiveX:

1. Выберите свойство **о** поле в окне свойств Microsoft Access.

2. Справа от поля свойства, нажмите кнопку **Создать** .
