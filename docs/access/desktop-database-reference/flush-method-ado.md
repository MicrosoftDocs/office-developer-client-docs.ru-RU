# <a name="flush-method-ado"></a>Метод флеша (ADO)


**Область применения**: Access 2013, Office 2013

Привнося содержимое [потока,](stream-object-ado.md) оставшегося в буфере ADO, к основному объекту, с которым связан **поток.**

## <a name="syntax"></a>Синтаксис

*Stream*. Флеш

## <a name="remarks"></a>Примечания

Этот метод может использоваться для отправки содержимого буфера потока в исходный объект (например, узел или файл, представленный URL-адресом, который является источником объекта **Stream).** Этот метод должен быть вызван, если необходимо убедиться, что все изменения, внесенные в содержимое **потока,** были написаны. Однако с помощью ADO обычно не требуется вызывать **флеш,** так как ADO непрерывно сбрасывает буфер как можно больше в фоновом режиме. Изменения в содержимом **Потока** вносят автоматически, а не кэшируются до тех пор, пока не будет вызван **Флеш.**

Закрытие **потока** методом [Close](close-method-ado.md) автоматически смывает содержимое **потока;** нет необходимости явно вызывать флеш непосредственно **перед** **закрыть**.
