<span data-ttu-id="fc54b-105">Методы **сохранения** и **открытия** ADO в объекте **Recordset** не считаются безопасными операциями для запуска в Internet Explorer.</span><span class="sxs-lookup"><span data-stu-id="fc54b-105">The ADO **Save** and **Open** methods on the **Recordset** object are not considered safe operations to run in Internet Explorer.</span></span> <span data-ttu-id="fc54b-106">Таким образом, если эти методы используются в коде скрипта, который выполняется в приложении или элементе управления, размещенном в браузере, конфигурация безопасности браузера будет влиять на его поведение.</span><span class="sxs-lookup"><span data-stu-id="fc54b-106">Thus, if these methods are used in a script code that is running in an application or control that is hosted in a browser, the security configuration of the browser will have an effect on its behavior.</span></span>
