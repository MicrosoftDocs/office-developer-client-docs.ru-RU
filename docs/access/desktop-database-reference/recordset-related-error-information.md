# <a name="recordset-related-error-information"></a>Сведения об ошибках, связанных с наборами записей

**Применимо к**: Access 2013, Office 2013

Во время пакетной обработки свойство **Status** объекта **набора записей** сведения об отдельных записей в **набора записей**. Перед обновлением пакета, свойство **Status** **записей** отражает сведения о записях для добавления, изменения и удаления. 

После вызова **UpdateBatch** свойство **Status** указывает успешное или неудачное выполнение операции. При перемещении по записям в **набор записей,** значение изменения свойств **состояния** для описания состояние текущей записи.
