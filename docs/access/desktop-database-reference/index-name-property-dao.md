# <a name="indexname-property-dao"></a>Index.Name Property (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает или задает имя указанного объекта. Чтение и запись **строки** Если объект не были добавлены в коллекцию. Только для чтения **строка** Если объект были добавлены в коллекцию.

## <a name="syntax"></a>Синтаксис

*выражение* . Имя

*выражение* Переменная, которая представляет объект **индекса** .

## <a name="remarks"></a>Замечания

Максимальная длина имени объекта **индекса** — 64 символов.
