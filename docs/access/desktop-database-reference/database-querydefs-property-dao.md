# <a name="databasequerydefs-property-dao"></a>Свойство Database.QueryDefs (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **QueryDefs** , который содержит все объекты **QueryDef** с указанной базы данных. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . QueryDefs

*выражение* Переменная, которая представляет собой объект **базы данных** .
