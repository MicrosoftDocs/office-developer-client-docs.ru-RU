# <a name="creating-the-connection-string"></a>Создание строки подключения

**Область применения**: Access 2013, Office 2013

ADO напрямую поддерживает пять аргументов в строке подключения. Другие аргументы передаются поставщику, указанному в аргументе *Поставщик*, без какой-либо обработки ADO.

> [!NOTE]
> В следующих примерах и в руководстве программиста ADO для проверки подлинности на сервере используется пользовательский id "MyId" с паролем "123aBc". Эти значения следует заменить действительными учетными данными входа на сервер. Кроме того, замените имя сервера на "MySqlServer".

Приложение HelloData в главе 1 использовало следующую строку подключения:

```vb 
 
m_sConnStr = "Provider='SQLOLEDB';Data Source='MySqlServer';" & _ 
 "Initial Catalog='Northwind';Integrated Security='SSPI';" 
```

Единственным параметром ADO, указанным в этой строке подключения, был "Provider=SQLOLEDB", который указал поставщик DB Microsoft OLE для SQL Server. Другие допустимые параметры, которые можно передать в строке подключения, могут определяться с помощью ссылки на документацию отдельных поставщиков. По словам поставщика OLE DB для SQL Server документации, можно заменить параметр "Server" для параметра *Источник* данных и "База данных" для параметра *Начального каталога.* Таким образом, следующая строка подключения будет производить результаты, идентичные первому:

```vb 
 
m_sConnStr = "Provider='SQLOLEDB';Server='MySqlServer';" & _ 
 "Database='Northwind';Integrated Security='SSPI';" 
```

Чтобы открыть подключение, просто передай строку подключения в качестве первого аргумента в **методе Open** объекта **Подключения:**

```vb 
 
objConn.Open m_sConnStr 
```

Также можно указать большую часть подобных сведений, задав свойства объекта **Connection** перед открытием подключения. Например, вы можете добиться того же эффекта, что и строка подключения выше, используя следующий код:

```vb 
 
With objConn 
 .Provider = "SQLOLEDB" 
 .DefaultDatabase = "Northwind" 
 .Properties("Data Source") = "MySqlServer" 
 .Properties("Integrated Security") = "SSPI" 
 .Open 
End With 
```
