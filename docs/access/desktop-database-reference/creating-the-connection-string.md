> <span data-ttu-id="91d9c-120">В следующих примерах и в руководстве программиста по ADO идентификатор пользователя "MyId" с паролем "123aBc" используется для проверки подлинности на сервере.</span><span class="sxs-lookup"><span data-stu-id="91d9c-120">In the following examples and throughout the ADO programmer's guide, the user id "MyId" with a password of "123aBc" is used to authenticate against the server.</span></span> <span data-ttu-id="91d9c-121">Необходимо заменить эти значения действительными учетными данными для входа на сервер.</span><span class="sxs-lookup"><span data-stu-id="91d9c-121">You should substitute these values with valid login credentials for your server.</span></span> <span data-ttu-id="91d9c-122">Кроме того, замените имя вашего сервера на "Мисклсервер".</span><span class="sxs-lookup"><span data-stu-id="91d9c-122">Also, substitute the name of your server for "MySqlServer".</span></span>

<span data-ttu-id="91d9c-123">Приложение HelloData в главе 1 использовало следующую строку подключения:</span><span class="sxs-lookup"><span data-stu-id="91d9c-123">The HelloData application in Chapter 1 used the following connection string:</span></span>

```vb 
 
m_sConnStr = "Provider='SQLOLEDB';Data Source='MySqlServer';" & _ 
 "Initial Catalog='Northwind';Integrated Security='SSPI';" 
```

<span data-ttu-id="91d9c-124">Единственным параметром ADO, указанным в этой строке подключения, является "Provider = SQLOLEDB", который указал поставщика Microsoft OLE DB для SQL Server.</span><span class="sxs-lookup"><span data-stu-id="91d9c-124">The only ADO parameter supplied in this connection string was "Provider=SQLOLEDB", which indicated the Microsoft OLE DB Provider for SQL Server.</span></span> <span data-ttu-id="91d9c-125">Другие допустимые параметры, которые можно передать в строке подключения, могут определяться с помощью ссылки на документацию отдельных поставщиков.</span><span class="sxs-lookup"><span data-stu-id="91d9c-125">Other valid parameters that can be passed in the connection string can be determined by referring to individual providers' documentation.</span></span> <span data-ttu-id="91d9c-126">В соответствии с документацией поставщика OLE DB для SQL Server можно заменить значение "Server" для параметра *источника данных* и "Database" для начального параметра *каталога* .</span><span class="sxs-lookup"><span data-stu-id="91d9c-126">According to the OLE DB Provider for SQL Server documentation, you can substitute "Server" for the *Data Source* parameter and "Database" for the *Initial Catalog* parameter.</span></span> <span data-ttu-id="91d9c-127">Таким образом, следующая строка подключения будет давать результаты, идентичные первому:</span><span class="sxs-lookup"><span data-stu-id="91d9c-127">Thus, the following connection string would produce results identical to the first:</span></span>

 "Database='Northwind';Integrated Security='SSPI';" 
```

<span data-ttu-id="91d9c-128">Чтобы открыть подключение, просто передайте строку подключения в качестве первого аргумента метода **Open** объекта **Connection** :</span><span class="sxs-lookup"><span data-stu-id="91d9c-128">To open the connection, simply pass the connection string as the first argument in the **Connection** object **Open** method:</span></span>

objConn.Open m_sConnStr 
```

<span data-ttu-id="91d9c-129">Также можно указать большую часть подобных сведений, задав свойства объекта **Connection** перед открытием подключения.</span><span class="sxs-lookup"><span data-stu-id="91d9c-129">It is also possible to supply much of this information by setting properties of the **Connection** object before opening the connection.</span></span> <span data-ttu-id="91d9c-130">Например, вы можете добиться того же результата, что и в строке подключения, с помощью следующего кода:</span><span class="sxs-lookup"><span data-stu-id="91d9c-130">For example, you could achieve the same effect as the connection string above by using the following code:</span></span>

```vb 
 
With objConn 
 .Provider = "SQLOLEDB" 
 .DefaultDatabase = "Northwind" 
 .Properties("Data Source") = "MySqlServer" 
 .Properties("Integrated Security") = "SSPI" 
 .Open 
End With 
```
