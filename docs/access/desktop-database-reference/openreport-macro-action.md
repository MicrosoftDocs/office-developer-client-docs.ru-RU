# <a name="openreport-macro-action"></a>Макрокоманда OpenReport

**Область применения**: Access 2013, Office 2013

С помощью действия **OpenReport** можно открыть отчет в режиме конструктора или предварительного просмотра или отправить отчет непосредственно на принтер. Кроме того, вы можете ограничить записи, которые будут печататься в отчете.

## <a name="remarks"></a>Заметки

Параметр **"Печать"** для аргумента **View** немедленно печатает отчет с использованием  текущих параметров принтера, не выведя диалоговое окно "Печать". Вы также можете использовать действие **OpenReport,** чтобы открыть и настроить отчет, а затем с помощью действия PrintOut распечатать его. Например, может потребоваться изменить отчет или использовать действие **PrintOut** для изменения параметров принтера перед печатью.

Применяемое условие фильтра и where становится параметром свойства **Filter отчета.**

Действие **OpenReport** аналогично двойному щелчку отчета в области навигации или щелчку правой кнопкой мыши в области навигации и выбору представления или команды **"Печать".**

> [!TIP] 
> - Чтобы распечатать похожие отчеты для различных наборов данных, используйте фильтр или предложение WHERE, чтобы ограничить записи, печатаные в отчете. Затем измените макрос, чтобы применить другой фильтр, или измените аргумент Условия where.
> 
> - Вы можете перетащить отчет из области навигации в строку макроса. При этом автоматически создается действие **OpenReport,** которое открывает отчет в представлении "Отчет".

## <a name="example"></a>Пример

В следующем примере показано, как использовать действие OpenReport для передачи параметра, который фильтрует отчет по мере его открытия. Отчет **rptChapters** отображает записи для указанного автора, передавая элемент, выбранный в поле со списоком **cboAuthors,** в параметр SelectedAuthor.

**Пример кода из** [справочника программиста Microsoft Access 2010](https://www.amazon.com/Microsoft-Access-2010-Programmers-Reference/dp/8126528125).

```vb
    OpenReport
        Report Name rptChapters
        View Report
        Filter Name
        Where Condition
        Window Mode Normal
    
    Parameters
        SelectedAuthor =[cboAuthor]
```
