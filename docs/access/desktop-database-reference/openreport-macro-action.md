# <a name="openreport-macro-action"></a>Макрокоманда OpenReport

**Область применения**: Access 2013, Office 2013

С помощью макрокоманды **ОткрытьОтчет** можно открыть отчет в режиме конструктора или предварительный просмотр, а также отправить отчет непосредственно на принтер. Кроме того, вы можете ограничить записи, которые будут печататься в отчете.

## <a name="remarks"></a>Замечания

Параметр " **Печать** " для аргумента " **Просмотр** " отправляет отчет сразу с использованием текущих параметров принтера, не открывая диалоговое окно " **Печать** ". Вы также можете использовать макрокоманду **ОткрытьОтчет (OpenReport** ), чтобы открыть и настроить отчет, а затем использовать действие печать для его печати. Например, может потребоваться изменить отчет или использовать действие **Печать** , чтобы изменить параметры принтера перед печатью.

Фильтр и условие, где вы хотите применить, становятся значением свойства **Filter** отчета.

Действие **ОткрытьОтчет** аналогично двойному щелчку отчета в области навигации или щелчку правой кнопкой мыши отчета в области навигации и выбору представления или команды " **Печать** ".

> [!TIP] 
> - Чтобы напечатать похожие отчеты для разных наборов данных, используйте фильтр или предложение WHERE, чтобы ограничить записи, печатаемые в отчете. Затем измените макрос, чтобы применить другой фильтр или изменить аргумент условия отбора.
> 
> - Отчет можно перетащить из области навигации в строку макрокоманды. При этом автоматически создается действие **ОткрытьОтчет** , которое открывает отчет в режиме отчета.

## <a name="example"></a>Пример

В приведенном ниже примере показано, как с помощью макрокоманды ОткрытьОтчет передать параметр, который фильтрует отчет при его открытии. В отчете **рптчаптерс** отображаются записи указанного автора путем передачи элемента, выбранного в поле со списком **кбоаусорс** , в параметр селектедаусор.

**Пример кода из** [справочника программиста Microsoft Access 2010](https://www.amazon.com/Microsoft-Access-2010-Programmers-Reference/dp/8126528125).

```vb
    OpenReport
        Report Name rptChapters
        View Report
        Filter Name
        Where Condition
        Window Mode Normal
    
    Parameters
        SelectedAuthor =[cboAuthor]
```
