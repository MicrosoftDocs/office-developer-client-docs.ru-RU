# <a name="openreport-macro-action"></a>Макрокоманда OpenReport

**Область применения**: Access 2013, Office 2013

Вы можете использовать **действие OpenReport,** чтобы открыть отчет в представлении Design или Print Preview или отправить отчет непосредственно на принтер. Кроме того, вы можете ограничить записи, которые будут печататься в отчете.

## <a name="remarks"></a>Примечания

Параметр **Print** для аргумента **View** немедленно печатает отчет с помощью текущих параметров принтера, не выведя диалоговое окно **Print.** Вы также можете использовать **действие OpenReport** для открытия и настроить отчет, а затем использовать действие PrintOut для его печати. Например, перед печатью может потребоваться изменить отчет или использовать действие **PrintOut** для изменения параметров принтера.

Фильтр и условия WHERE, которые вы применяли, становятся параметром свойства **Фильтр отчетов.**

Действие **OpenReport** аналогично дважды щелкнуть отчет в области навигации или щелкнуть правой кнопкой мыши отчет в области навигации и выбрать представление или команду **Печать.**

> [!TIP] 
> - Чтобы напечатать аналогичные отчеты для различных наборов данных, используйте фильтр или пункт WHERE, чтобы ограничить записи, напечатанные в отчете. Затем измените макрос, чтобы применить другой фильтр или изменить аргумент Where Condition.
> 
> - Отчет можно перетаскивать из области навигации в строку действий макроса. Это автоматически создает действие **OpenReport,** которое открывает отчет в представлении Report.

## <a name="example"></a>Пример

В следующем примере показано, как использовать действие OpenReport для передачи параметра, который фильтрует отчет при его открытиях. Отчет **rptChapters** отображает записи для указанного автора, передав элемент, выбранный в комбо-окне **cboAuthors,** параметру SelectedAuthor.

**Пример кода из** [справочника программиста Microsoft Access 2010](https://www.amazon.com/Microsoft-Access-2010-Programmers-Reference/dp/8126528125).

```vb
    OpenReport
        Report Name rptChapters
        View Report
        Filter Name
        Where Condition
        Window Mode Normal
    
    Parameters
        SelectedAuthor =[cboAuthor]
```
