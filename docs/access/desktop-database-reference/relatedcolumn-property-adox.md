# <a name="relatedcolumn-property-adox"></a>Свойство RelatedColumn (ADOX)


**Область применения**: Access 2013, Office 2013

Указывает имя связанного столбца в связанной таблице (только для ключевых столбцов).

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает и возвращает **строку,** которая является именем связанного столбца в связанной таблице.

## <a name="remarks"></a>Заметки

Значение по умолчанию — пустая строка ("").

Это свойство доступно только для чтения для объектов [Column,](column-object-adox.md) которые уже были appended в коллекцию.
