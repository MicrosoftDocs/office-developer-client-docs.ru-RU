# <a name="parameterproperties-property-dao"></a>Свойство Parameter.Properties (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **[свойств](properties-collection-dao.md)** для указанного объекта. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . Свойства

*выражение* Переменная, которая представляет собой объект- **параметр** .
