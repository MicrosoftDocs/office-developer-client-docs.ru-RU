# <a name="minimizewindow-macro-action"></a>Макрокоманда MinimizeWindow

**Область применения**: Access 2013, Office 2013

Если Access настроен на использование перекрывающихся окон вместо документов с вкладками, с помощью действия **MinimizeWindow** можно уменьшить активное окно до небольшой строки заголовка в нижней части окна Access.

> [!NOTE]
> Это действие нельзя применить к окнам кода в редакторе Visual Basic. Сведения о том, как повлиять на окна кода, см. в разделе о **свойстве WindowState.**

## <a name="setting"></a>Setting

Действие **MinimizeWindow** не имеет аргументов.

## <a name="remarks"></a>Заметки

Это действие можно использовать для удаления окна с экрана, оставив объект открытым. Это действие также можно использовать для открытия объекта без отображения его окна. Чтобы отобразить объект, используйте **действие SelectObject** с действием **MaximizeWindow** или **RestoreWindow.** Действие **RestoreWindow** восстанавливает свернутое окно до предыдущего размера.

Действие **MinimizeWindow** имеет тот же эффект, что и нажатие кнопки  **"Свернуть"** в правом верхнем углу окна или "Свернуть" в меню "Управление" окна. 

**Советы**

- Сначала может потребоваться использовать действие **SelectObject,** если нужно свести к минимуму окно, не является активным окном.

- Чтобы скрыть области навигации, используйте действие **SelectObject** с аргументом  "Да" в области навигации, а затем используйте действие **MinimizeWindow.** Объект, выбранный в **действии SelectObject,** может быть любым объектом в базе данных.

- Вы можете скрыть активное окно, щелкнув   "Управление этим окном" в меню "Вид", а затем щелкнув **"Скрыть".** Окно становится невидимым, а не уменьшается до значка. Используйте команду **"Открепить"** в том же меню, чтобы снова появиться окно. Вы можете использовать действие **RunMenuCommand** для выполнения любой из этих команд из макроса.

- Вы также можете использовать **действие SetValue,** чтобы установить свойство **Visible** формы, чтобы скрыть или показать окно формы.

Чтобы выполнить действие **MinimizeWindow** в модуле Visual Basic для приложений VBA, используйте метод **Minimize** объекта **DoCmd.**
