# <a name="minimizewindow-macro-action"></a>Макрокоманда MinimizeWindow

**Область применения**: Access 2013, Office 2013

Если в Access настроено использование перекрывающихся окон вместо документов с вкладками, можно использовать действие **минимизевиндов** , чтобы уменьшить активное окно до маленькой строки заголовка в нижней части окна Access.

> [!NOTE]
> Это действие невозможно применить к окнам кода в редакторе Visual Basic. Сведения о том, как повлиять на окна кода, представлены в разделе свойство **WindowState** .

## <a name="setting"></a>Параметр

Макрокоманда **минимизевиндов** не содержит аргументов.

## <a name="remarks"></a>Замечания

С помощью этого действия можно удалить окно с экрана, оставив объект открытым. Вы также можете использовать это действие, чтобы открыть объект без отображения его окна. Чтобы отобразить объект, используйте макрокоманду **ВыделитьОбъект** с действием **максимизевиндов** или **ресторевиндов** . Действие **ресторевиндов** Восстанавливает свернутое окно до его предыдущего размера.

Действие **минимизевиндов** имеет тот же результат, что и нажатие кнопки **свертывания** в правом верхнем углу окна, или при нажатии кнопки **Свернуть** в **** оконном меню окна.

**Советы**

- Если окно, которое нужно свернуть, не является активным, может потребоваться использовать макрокоманду "ВыделитьОбъект" ( **SelectObject** ).

- Чтобы скрыть область навигации, используйте макрокоманду "ВыделитьОбъект" ( **SelectObject** ) с параметром в разделе в области навигации, равным **Да** , а затем используйте действие **минимизевиндов** . Объект, выбранный в действии **ВыделитьОбъект** , может быть любым объектом в базе данных.

- Чтобы скрыть активное окно, щелкните **Управление этим окном** в меню **вид** , а затем выберите команду **Скрыть**. Вместо того чтобы уменьшаться до значка, окно становится невидимым. Чтобы вновь отобразить окно, используйте команду **отобразить** в том же меню. Вы можете использовать действие **Макрокоманда runmenucommand** для выполнения любой из этих команд из макроса.

- Вы также можете использовать действие **SetValue** , чтобы задать свойство **Visible** формы для скрытия или отображения окна формы.

Чтобы запустить действие **минимизевиндов** в модуле Visual Basic для приложений (VBA), используйте метод **сворачивания** объекта **DoCmd** .
