# <a name="minimizewindow-macro-action"></a>Макрокоманда MinimizeWindow

**Область применения**: Access 2013, Office 2013

Если Access настроен на использование перекрывающихся окон вместо вкладки документов, можно использовать действие **MinimizeWindow,** чтобы уменьшить активное окно до небольшой панели заголовков в нижней части окна Access.

> [!NOTE]
> Это действие нельзя применить к окнам кода в редакторе Visual Basic. Сведения о том, как повлиять на окна кода, см. в разделе **Свойство WindowState.**

## <a name="setting"></a>Параметр

Действие **MinimizeWindow** не имеет аргументов.

## <a name="remarks"></a>Примечания

Вы можете использовать это действие, чтобы удалить окно с экрана, оставив объект открытым. Это действие также можно использовать для открытия объекта без отображения его окна. Чтобы отобразить объект, используйте **действие SelectObject** с действием **MaximizeWindow** или **RestoreWindow.** Действие **RestoreWindow** восстанавливает свести к минимуму окно до предыдущего размера.

Действие **MinimizeWindow** имеет тот же эффект, что щелкнуть кнопку **"Свести** к минимуму" в верхнем правом углу окна или нажать **Кнопку Свести** к минимуму в меню управления **окном.**

**Советы**

- Сначала может потребоваться использовать **действие SelectObject,** если необходимо свести к минимуму окно, не является активным окном.

- Чтобы скрыть области навигации, используйте **действие SelectObject** с аргументом В области навигации, за набором **да,** а затем используйте действие **MinimizeWindow.** Выбранный в **действии SelectObject** объект может быть любым объектом в базе данных.

- Активное окно можно скрыть, нажав кнопку **Управление этим** окном в меню **Просмотр,** а затем нажав **кнопку Скрыть**. Вместо того, чтобы быть уменьшенным до значка, окно становится невидимым. Используйте команду **Unhide** в том же меню, чтобы снова появиться окно. Вы можете использовать **действие RunMenuCommand** для выполнения любой из этих команд с макроса.

- Вы также можете использовать **действие SetValue,** чтобы установить свойство **Видимый** вид формы, чтобы скрыть или показать окно формы.

Чтобы выполнить действие **MinimizeWindow** в Visual Basic для приложений (VBA), используйте метод **Минимизация** объекта **DoCmd.**
