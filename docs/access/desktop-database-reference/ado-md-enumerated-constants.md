# <a name="ado-md-enumerated-constants"></a>Перечислимые константы ADO MD

**Область применения**: Access 2013, Office 2013

Чтобы помочь отладке, В ADO MD перечисляются константы, перечисляющие значение для каждой константы. Однако это значение является чисто консультативным и может изменяться с одного выпуска ADO MD на другой. Код должен зависеть только от имени, а не от фактического значения перенамеренных констант.

Определено следующее переопределение:

- [MemberTypeEnum](membertypeenum.md)
