# <a name="open-method-ado-record"></a>Метод Open (объект Record в ADO)

**Область применения**: Access 2013, Office 2013

Открывает существующий [объект Record](record-object-ado.md) или создает новый элемент, представленный **записью** (например, файл или каталог).

## <a name="syntax"></a>Синтаксис

Open *Source*, *ActiveConnection*, *Mode*, *CreateOptions*, *Options*, *UserName*, *Password*

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*Source* |Необязательный параметр. Вариант, который может представлять URL-адрес объекта, который будет представлен этим объектом **Record,** **Command,** открытым набором записей или другим объектом  **Record,** строкой, содержащей SQL SELECT или имя таблицы. [](recordset-object-ado.md)|
|*ActiveConnection* | Необязательный параметр. **Вариант,** который представляет строку подключения или [открытый объект Connection.](connection-object-ado.md)|
|*Mode* |Необязательный параметр. Значение [ConnectModeEnum,](connectmodeenum.md) значением по умолчанию которого является **adModeUnknown,** которое указывает режим доступа для итоговых **объектов Record.**|
|*CreateOptions* |Необязательный параметр. Значение [RecordCreateOptionsEnum,](recordcreateoptionsenum.md) значение по умолчанию — **adFailIfNotExists,** которое указывает, следует ли открыть существующий файл или каталог, или создать новый файл или каталог. Если установлено значение по умолчанию, режим доступа получается из [свойства Mode.](mode-property-ado.md) Этот параметр игнорируется, если параметр *Source* не содержит URL-адрес.|
|*Options* |Необязательный параметр. Значение [RecordOpenOptionsEnum,](recordopenoptionsenum.md) значение по умолчанию — **adOpenRecordUnspecified,** которое указывает параметры открытия **записи.** Эти значения могут быть объединены.|
|*UserName* |Необязательный параметр. **Строковая** строка, содержаная ИД пользователя, который при необходимости разрешает доступ к *источнику.*|
|*Password* |Необязательный параметр. Строковое значение, которое содержит пароль, который при необходимости проверяет *имя пользователя.* |

## <a name="remarks"></a>Заметки

*Источником* может быть:

- URL-адрес. Если для URL-адреса указан протокол HTTP, по умолчанию будет вызван поставщик услуг Интернета. Если URL-адрес указывает на узел, содержащий исполняемый сценарий (например, . СТРАНИЦА ASP),  после чего по умолчанию открывается запись, содержащая источник, а не выполненное содержимое. Используйте аргумент *Options* для изменения этого поведения.

- Объект **Record.** Объект **Record,** открытый из другой **записи,** клонируют **исходный объект Record.**

- Объект **Command.** Открытый **объект Record** представляет одну строку, возвращаемую при выполнении **команды.** Если результаты содержат несколько строк, содержимое первой строки помещается в запись, а в коллекцию Ошибок может быть **добавлена ошибка.**

- Инструкция SQL SELECT. Открытый **объект Record** представляет одну строку, возвращаемую путем выполнения содержимого строки. Если результаты содержат несколько строк, содержимое первой строки помещается в запись, а в коллекцию Ошибок может быть **добавлена ошибка.**

- Имя таблицы.

Если объект **Record** представляет сущность, доступ к которой невозможно получить с помощью URL-адреса (например, строка объекта **Recordset,** производная от базы данных), то значения свойства [ParentURL](parenturl-property-ado.md) и поля, к которой получен доступ с помощью константы **adRecordURL,** будут иметь значение null.

> [!NOTE]
> URL-адреса, использующие схему HTTP, автоматически вызывают поставщика [Microsoft OLE DB для интернет-публикации.](microsoft-ole-db-provider-for-internet-publishing.md) Дополнительные сведения см. в [сведениях об абсолютных и относительных URL-адресах.](absolute-and-relative-urls.md)
