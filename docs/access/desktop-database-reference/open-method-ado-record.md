# <a name="open-method-ado-record"></a>Метод Open (объект Record в ADO)

**Область применения**: Access 2013, Office 2013

Открывает существующий объект [Record](record-object-ado.md) или создает новый элемент, представленный **записью** (например, файлом или каталогом).

## <a name="syntax"></a>Синтаксис

Open *Source*, *ActiveConnection*, *mode*, *креатеоптионс*, *Options*, *username*, *Password*

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*Source* |Необязательное. **Variant** , который может представлять URL-адрес объекта, который должен быть представлен этим объектом **Record** , **команда**, Открытый [набор записей](recordset-object-ado.md) или другой объект **Record** — строка, содержащая инструкцию SQL SELECT или имя таблицы.|
|*ActiveConnection* | Необязательное. **Переменная Variant** , представляющая строку подключения или открытый объект [Connection](connection-object-ado.md) .|
|*Mode* |Необязательное. Значение [коннектмодинум](connectmodeenum.md) , для которого по умолчанию задано значение **адмодеункновн**, указывающее режим доступа для объекта результирующей **записи** .|
|*креатеоптионс* |Необязательное. Значение [рекордкреатеоптионсенум](recordcreateoptionsenum.md) , для которого по умолчанию задано значение **адфаилифнотексистс**, которое указывает, следует ли открыть существующий файл или каталог или создать новый файл или каталог. Если задано значение по умолчанию, режим доступа будет получен из свойства [mode](mode-property-ado.md) . Этот параметр игнорируется, если параметр *Source* не содержит URL-адрес.|
|*Параметры* |Необязательное. Значение [рекордопеноптионсенум](recordopenoptionsenum.md) , для которого по умолчанию задано значение **адопенрекордунспеЦифиед**, которое определяет параметры открытия **записи**. Эти значения можно объединять.|
|*UserName* |Необязательное. **Строковое** значение, содержащее идентификатор пользователя, который при необходимости авторизует доступ к *источнику*.|
|*Password* |Необязательное. **Строковое** значение, которое содержит пароль, при необходимости проверяющий *имя пользователя*.|

## <a name="remarks"></a>Примечания

*Источник* может быть:

- URL-адрес. Если для URL-адреса задан протокол HTTP, поставщик услуг Интернета будет вызываться по умолчанию. Если URL-адрес указывает на узел, содержащий исполняемый скрипт (например,. Страница ASP), то по умолчанию открывается **запись** , содержащая источник, а не выполняемое содержимое. Чтобы изменить это поведение, используйте аргумент *Options* .

- Объект **Record** . Объект **Record** , Открытый из другой **записи** , будет клонировать исходный объект **Record** .

- Объект **Command** . Открытый объект **Record** представляет одну строку, возвращенную при выполнении **команды**. Если результаты содержат более одной строки, содержимое первой строки размещается в записи, а в коллекцию **Errors** может быть добавлено сообщение об ошибке.

- Инструкция SQL SELECT. Открытый объект **Record** представляет одну строку, возвращенную при выполнении содержимого строки. Если результаты содержат более одной строки, содержимое первой строки размещается в записи, а в коллекцию **Errors** может быть добавлено сообщение об ошибке.

- Имя таблицы.

Если объект **Record** представляет сущность, доступ к которой невозможен с помощью URL-адреса (например, строки объекта **Recordset** , производного от базы данных), то значения свойства [ParentURL](parenturl-property-ado.md) и поля, доступ к которым осуществляется с помощью константы **адрекордурл** , равны NULL.

> [!NOTE]
> URL-адреса, использующие схему HTTP, автоматически будут вызывать [поставщик Microsoft OLE DB для публикации в Интернете](microsoft-ole-db-provider-for-internet-publishing.md). Для получения дополнительных сведений см [абсолютные и относительные URL-адреса](absolute-and-relative-urls.md).
