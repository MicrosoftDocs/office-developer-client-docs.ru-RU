# <a name="open-method-ado-record"></a>Метод Open (объект Record в ADO)

**Область применения**: Access 2013, Office 2013

Открывает [существующий объект Record](record-object-ado.md) или создает новый элемент, представленный **записью** (например, файл или каталог).

## <a name="syntax"></a>Синтаксис

С *открытым исходным* кодом , *ActiveConnection*, *Режим*, *CreateOptions*, *Параметры*, *Имя пользователя*, *Пароль*

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*Source* |Необязательный параметр. Вариант, который может представлять URL-адрес объекта, который будет представлен этим объектом **Record,** командой, открытым набором записей или другим объектом **Record,** строкой, содержащей SQL SELECT или имя таблицы.   [](recordset-object-ado.md)|
|*ActiveConnection* | Необязательный параметр. **Вариант,** который представляет соединител строку или открытый объект [Подключения.](connection-object-ado.md)|
|*Mode* |Необязательный параметр. Значение [ConnectModeEnum,](connectmodeenum.md) значение которого по умолчанию **— adModeUnknown,** которое указывает режим доступа к объекту **Record.**|
|*CreateOptions* |Необязательный параметр. Значение [RecordCreateOptionsEnum,](recordcreateoptionsenum.md) значение которого по умолчанию **— adFailIfNotExists,** которое указывает, следует ли открывать существующий файл или каталог или создавать новый файл или каталог. Если установлено значение по умолчанию, режим доступа получается из свойства [Mode.](mode-property-ado.md) Этот параметр игнорируется, если параметр *Source* не содержит URL-адрес.|
|*Options* |Необязательный параметр. Значение [RecordOpenOptionsEnum,](recordopenoptionsenum.md) значение которого по умолчанию **adOpenRecordUnspecified,** указывает параметры открытия **записи.** Эти значения могут быть объединены.|
|*UserName* |Необязательный параметр. Значение **String,** содержаее код пользователя, который при необходимости разрешает доступ к *Источнику.*|
|*Password* |Необязательный параметр. Строковое значение, содержаее пароль, который при необходимости проверяет *имя пользователя.* |

## <a name="remarks"></a>Примечания

*Источник* может быть:

- URL-адрес. Если протокол для URL-адреса http, то по умолчанию будет вызываться интернет-провайдер. Если URL-адрес указывает на узел, содержащий исполняемый сценарий (например, . Страница ASP),  затем по умолчанию открывается запись, содержащая источник, а не выполненное содержимое. Чтобы изменить это поведение, используйте аргумент *Options.*

- Объект **Запись.** Объект **Record,** открытый из другой **записи,** клонит исходный **объект Record.**

- Объект **Command.** Открытый **объект Record** представляет одну строку, возвращаемую при выполнении **команды**. Если результаты содержат более одной строки, содержимое первой строки помещается в запись и ошибка может быть добавлена в коллекцию **Ошибок.**

- Инструкция SQL SELECT. Открытый **объект Record** представляет одну строку, возвращаемую путем выполнения содержимого строки. Если результаты содержат более одной строки, содержимое первой строки помещается в запись и ошибка может быть добавлена в коллекцию **Ошибок.**

- Имя таблицы.

Если объект **Record** представляет сущность, доступ к которой невозможно получить с URL-адресом (например, строка **recordset,** полученная из базы данных), то значения свойства [ParentURL](parenturl-property-ado.md) и поля, доступ к нему с константой **adRecordURL,** являются нулевными.

> [!NOTE]
> URL-адреса, использующие схему http, автоматически вызывают поставщика DB Microsoft [OLE для публикации в Интернете.](microsoft-ole-db-provider-for-internet-publishing.md) Дополнительные сведения см. в [url-адресах Absolute и relative.](absolute-and-relative-urls.md)
