# <a name="querydeffields-property-dao"></a>Свойство QueryDef.Fields (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **[полей](fields-collection-dao.md)** , представляющую все хранятся объекты **[поля](field-object-dao.md)** для указанного объекта. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . Поля

*выражение* Переменная, которая представляет собой объект- **QueryDef** .
