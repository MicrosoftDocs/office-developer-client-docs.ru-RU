# <a name="definedsize-property-adox"></a>Свойство DefinedSize (ADOX)


**Применимо к**: Access 2013, Office 2013

Указывает сказано максимальный размер столбца.

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает и возвращает значение типа **Long** , максимальная длина в символах значений данных.

## <a name="remarks"></a>Примечания

Значение по умолчанию — нуль (0).

Это свойство доступно только для чтения для объектов [столбца](column-object-adox.md) уже добавляется в конец коллекции.
