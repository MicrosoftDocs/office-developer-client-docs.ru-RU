# <a name="comment-macro-statement"></a>Оператор макроса Comment


**Область применения**: Access 2013, Office 2013

Заявление **Comment** можно использовать для предоставления пояснительных комментариев в макросах.

## <a name="remarks"></a>Заметки

Длина **комментария** может быть не более 1000 символов. Однако по умолчанию отображается только первая строка комментария.
