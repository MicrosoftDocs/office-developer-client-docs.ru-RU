## <a name="example"></a><span data-ttu-id="2912f-157">Пример</span><span class="sxs-lookup"><span data-stu-id="2912f-157">Example</span></span>

<span data-ttu-id="2912f-158">В этом примере выбирает все записи в таблице предполагаемую новых клиентов и добавляет их в таблице Customers.</span><span class="sxs-lookup"><span data-stu-id="2912f-158">This example selects all records in a hypothetical New Customers table and adds them to the Customers table.</span></span> <span data-ttu-id="2912f-159">Когда отдельных столбцов не определен, имена столбцов в таблице ВЫБЕРИТЕ должны совпадать точно в вставки в таблице.</span><span class="sxs-lookup"><span data-stu-id="2912f-159">When individual columns are not designated, the SELECT table column names must match exactly those in the INSERT INTO table.</span></span>

```vb
    Sub InsertIntoX1() 
     
        Dim dbs As Database 
     
        ' Modify this line to include the path to Northwind 
        ' on your computer. 
        Set dbs = OpenDatabase("Northwind.mdb") 
         
        ' Select all records in the New Customers table  
        ' and add them to the Customers table. 
        dbs.Execute " INSERT INTO Customers " _ 
            & "SELECT * " _ 
            & "FROM [New Customers];" 
             
        dbs.Close 
     
    End Sub
```
