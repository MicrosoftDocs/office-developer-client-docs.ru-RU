# <a name="ado-collections"></a>ADO Collections


**Применимо к**: Access 2013 | Office 2013

Каждое семейство сайтов может содержать его соответствующего объекта. Например объект [Error](error-object-ado.md) может содержаться в семейство [Errors](errors-collection-ado.md) . Дополнительные сведения об объектах можно [Объекты ADO](ado-objects-and-interfaces.md) или раздела указанного объекта.
