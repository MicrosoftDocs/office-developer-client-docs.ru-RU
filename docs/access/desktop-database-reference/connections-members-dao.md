# <a name="connections-members-dao"></a>Connections Members (DAO)


**Применимо к**: Access 2013 | Office 2013


> [!NOTE]
> <P>Рабочие области технология ODBCDirect не поддерживаются в Microsoft Access 2013. Использование ADO, если вы хотите получить доступ к внешним источникам данных без использования ядро базы данных Microsoft Access. Подключения к коллекции содержит текущий объекты подключения объекта рабочей области. (Технология ODBCDirect рабочие области только).</P>
