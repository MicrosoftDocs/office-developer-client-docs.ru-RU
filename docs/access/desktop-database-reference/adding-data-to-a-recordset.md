# <a name="adding-data-to-a-recordset"></a>Добавление данных в объект Recordset

**Область применения**: Access 2013, Office 2013

Объект **Recordset** , вероятно, является наиболее часто используемым для объектов ADO. В ADO объект **Recordset** лучше всего представить в виде сочетания набора результатов из источника данных и связанного с ним поведения курсора. Таким образом, вы можете поместить данные в **набор записей** , а затем использовать методы и свойства **набора записей** для перемещения по строкам данных, просмотра значений в строках и других манипуляций с набором результатов.

В этом разделе приводится основное внимание на добавлении данных в **набор записей**. Сведения о переходе по данным или обновлении данных содержатся в [главе 4: Редактирование данных](chapter-4-editing-data.md) и [глава 5: обновление и постоянное использование данных](chapter-5-updating-and-persisting-data.md). Не всегда требуются расширенные возможности объекта **Command** для добавления набора результатов в **набор записей**. Часто можно выполнить команду, задав свойство **Source** в **наборе записей** или передав командНую строку методу **Open** объекта **Recordset** .

Существует несколько способов добавления данных из источника данных в **набор записей**. Используемый способ зависит от потребностей вашего приложения и возможностей поставщика.
