# <a name="adding-data-to-a-recordset"></a>Adding Data to a Recordset


**Применимо к**: Access 2013 | Office 2013

**Набор записей** представляет, вероятно, наиболее часто используемые объекты ADO. В ADO **набор записей** — лучше всего считать как сочетание результирующий набор из источника данных и связанных с ним поведений. Таким образом можно размещения данных в **набор записей** и затем использовать для перемещения по строк данных, просмотр значений в строках и управлять результирующий набор **записей** методы и свойства.

В этом разделе нацеливаются на добавление данных в **набор записей**. Сведения о навигации по данных или обновления данных в разделе [раздел 4: редактирование данных](chapter-4-editing-data.md) и [Глава 5: обновления и сохранение данных](chapter-5-updating-and-persisting-data.md). Расширенные возможности объекта **команды** для добавления результирующий набор **записей**всегда нет необходимости. Часто следует выполнить команду путем установки свойства **источника** **записей** или передачи командной строки в метод **Open** объекта **набора записей** .

Существует несколько способов добавления данных из источника данных для **набора записей**. Методики, которую вы используете зависит от требований приложения и возможности ваш поставщик.
