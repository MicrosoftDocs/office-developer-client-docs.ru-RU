# <a name="opentable-macro-action"></a>Макрокоманда OpenTable

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **опентабле** для открытия таблицы в режиме таблицы, конструктора или предварительного просмотра. Вы также можете выбрать режим ввода данных для таблицы.

## <a name="setting"></a>Параметр

Макрокоманда **опентабле** имеет следующие аргументы.

## <a name="remarks"></a>Примечания

Это действие аналогично двойному щелчку таблицы в области навигации или щелчке правой кнопкой мыши таблицы в области навигации и выбора представления.

> [!TIP]
> Вы можете перетащить таблицу из области навигации в строку макрокоманды. При этом автоматически создается действие **опентабле** , которое открывает таблицу в режиме таблицы.

Чтобы запустить действие **опентабле** в модуле Visual Basic для приложений (VBA), используйте метод **опентабле** объекта **DoCmd** .
