# <a name="setorderby-macro-action"></a>Действия макроса ЗадатьПорядокСортировки


**Применимо к**: Access 2013, Office 2013

Действие **ЗадатьПорядокСортировки** можно использовать для указания способа сортировки записей формы, отчета, таблицы или результата запроса.

## <a name="setting"></a>Параметр

Действие **ЗадатьПорядокСортировки** состоит из следующих аргументов.

<col style="width: 50%" />

<thead>
<tr class="header">

<td><p>Строковое выражение, которое включает в себя имя поля или полей, по которому выполняется сортировка записей и необязательные ASC или DESC ключевых слов.</p></td>

## <a name="remarks"></a>Замечания

При выполнении этого действия макроса сортировки применяется к таблице, форме, отчете или таблицы данных (результат запроса), который является активным и фокус находится на.

Аргумент Order By — это имя в поле или поля, в которых используется для сортировки записей. При использовании более одного имени поля, разделяйте имена запятыми (,). Свойство **OrderBy** активного объекта используется чтобы сохранить порядок сортировки и применять в дальнейшем. Значения OrderBy сохраняются с объектами, в которых они созданы. Автоматически загружаются при открытии объекта, но при этом не применяются автоматически.

Если значение аргумента Order By, указав один или несколько имен полей и запустите макрос, записи сортируются в порядке возрастания по умолчанию.

Чтобы отсортировать записи в убывающем порядке, введите DESC в конце выражения аргумента Order By. Например для сортировки записей клиентов в порядке убывания имя контакта, задайте Order By для «ContactName DESC». Чтобы отсортировать имена, LastName по убыванию и FirstName по возрастанию, значение аргумента Order By «Фамилия DESC, имя ASC».
