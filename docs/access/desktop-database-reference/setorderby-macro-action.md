# <a name="setorderby-macro-action"></a>Макрокоманда SetOrderBy


**Область применения**: Access 2013, Office 2013

Вы можете использовать **действие SetOrderBy,** чтобы указать, как сортировать записи в форме, отчете, таблице или результате запроса.

## <a name="setting"></a>Параметр

Действие **SetOrderBy** имеет следующие аргументы.

## <a name="remarks"></a>Примечания

При запуске этого макросного действия сорт применяется к таблице, форме, отчету или таблице данных (результат запроса), которая активна и имеет фокус.

Аргумент Order By — это имя поля или полей, на которых необходимо сортировать записи. При использовании более чем одного имени поля разделять имена с запятой (,). Свойство **OrderBy** активного объекта используется для сохранения значения заказа и его применения в более позднее время. Значения OrderBy сохраняются с объектами, в которых они создаются. Они автоматически загружаются при открываемом объекте, но не применяются автоматически.

Когда вы установите аргумент Order By, введите одно или несколько имен полей и запустите макрос, записи сортироваться по умолчанию в порядке восходящей.

Чтобы сортировать записи в порядке убывления, введите DESC в конце выражения Order By argument. Например, чтобы сортировать записи клиентов в порядке убывания по имени контакта, установите аргумент Order By на "ContactName DESC". Чтобы отсортировать имена по убыванию lastName и по возрастанию FirstName, установите Порядок аргументом "LastName DESC, FirstName ASC".
