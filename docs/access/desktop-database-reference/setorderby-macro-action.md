# <a name="setorderby-macro-action"></a>Макрокоманда SetOrderBy


**Область применения**: Access 2013, Office 2013

С помощью действия **SetOrderBy** можно указать, как нужно сортировать записи в форме, отчете, таблице или результатах запроса.

## <a name="setting"></a>Setting

## <a name="remarks"></a>Заметки

При запуске этого макроса сортировка применяется к таблице, форме, отчету или таблице (результат запроса), которая активна и находится в фокусе.

Аргумент Order By — это имя поля или полей, по которым необходимо сортировать записи. При использовании более одного имени поля разделять их запятой (,). Свойство **OrderBy** активного объекта используется для сохранения упорядоченного значения и его применения позже. Значения OrderBy сохраняются вместе с объектами, в которых они создаются. Они автоматически загружаются при открытом объекте, но не применяются автоматически.

Когда вы установите аргумент Order By, введите одно или несколько имен полей, а затем запустите макрос, записи сортироваться по умолчанию в порядке возрастания.

Чтобы отсортировать записи в порядке убывления, введите DESC в конце выражения аргумента Order By. Например, чтобы отсортировать записи клиентов по убываемой порядку по имени контакта, установите для аргумента Order By "ContactName DESC". Чтобы отсортировать имена по убыванию LastName и FirstName по возрастанию, установите для аргумента Order By "LastName DESC, FirstName ASC".
