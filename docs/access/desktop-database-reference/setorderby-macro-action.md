# <a name="setorderby-macro-action"></a>Макрокоманда SetOrderBy


**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **сетордерби** , чтобы указать, как следует сортировать записи в форме, отчете, таблице или результатах запроса.

## <a name="remarks"></a>Примечания

При выполнении этого макроса сортировка применяется к таблице, форме, отчету или таблице (результат запроса), которая активна и имеет фокус.

Аргумент ORDER BY — это имя поля или полей, для которых требуется отсортировать записи. При использовании нескольких имен полей разделяйте их запятыми (,). Свойство **OrderBy** активного объекта используется для сохранения значения упорядочения и его применения в дальнейшем. Значения OrderBy сохраняются вместе с объектами, в которых они созданы. Они автоматически загружаются при открытии объекта, но не применяются автоматически.

При задании аргумента ORDER BY путем ввода одного или нескольких имен полей и запуска макроса записи сортируются по умолчанию в порядке возрастания.

Чтобы отсортировать записи в убывающем порядке, введите DESC в конце выражения ORDER BY аргумента ORDER BY. Например, чтобы отсортировать записи клиентов по убыванию по имени контакта, задайте для аргумента ORDER BY значение "ContactName DESC". Для сортировки имен по убыванию и по возрастанию задайте для аргумента ORDER BY значение "фамилия DESC, имя ASC".
