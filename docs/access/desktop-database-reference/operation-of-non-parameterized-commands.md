<span data-ttu-id="6c0a5-104">Для команды без параметров выполняются все команды поставщика и **наборов записей** , созданных во время выполнения команды.</span><span class="sxs-lookup"><span data-stu-id="6c0a5-104">For non-parameterized commands, all of the provider commands are executed and the **Recordsets** are created during command execution.</span></span> <span data-ttu-id="6c0a5-105">Если команда выполняется синхронно, всех наборов **записей** будет полностью заполнена.</span><span class="sxs-lookup"><span data-stu-id="6c0a5-105">If the command is executed synchronously, all of the **Recordsets** will be fully populated.</span></span> <span data-ttu-id="6c0a5-106">Выделенный режим асинхронное заполнение, заполненный состояние **наборов записей** будут зависеть от режима заполнения и размера **набора записей**.</span><span class="sxs-lookup"><span data-stu-id="6c0a5-106">If an asynchronous population mode was selected, the populated state of the **Recordsets** will depend on the population mode and the size of the **Recordsets**.</span></span>

```vb

 APPEND ({SELECT * FROM Orders} AS chapOrders 
 RELATE customerID TO customerID) 
```
