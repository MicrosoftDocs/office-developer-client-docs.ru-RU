# <a name="primarykey-property-adox"></a>PrimaryKey Property (ADOX)


**Применимо к**: Access 2013 | Office 2013

Указывает, представляет ли индекс первичный ключ в таблице.

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает и возвращает значение **типа Boolean** .

## <a name="remarks"></a>Замечания

Значение по умолчанию — **False**.

Это свойство доступно только для чтения объектов [индекса](index-object-adox.md) уже добавляется в конец коллекции.
