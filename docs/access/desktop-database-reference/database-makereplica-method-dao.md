## <a name="example"></a><span data-ttu-id="52947-128">Пример</span><span class="sxs-lookup"><span data-stu-id="52947-128">Example</span></span>

<span data-ttu-id="52947-129">Эта функция использует метод **макереплика** для создания дополнительной реплики существующей основной реплики.</span><span class="sxs-lookup"><span data-stu-id="52947-129">This function uses the **MakeReplica** method to create an additional replica of an existing Design Master.</span></span> <span data-ttu-id="52947-130">Аргумент Интоптионс может быть сочетанием констант **дбрепмакереадонли** и **дбрепмакепартиал**, а также может быть равно 0.</span><span class="sxs-lookup"><span data-stu-id="52947-130">The intOptions argument can be a combination of the constants **dbRepMakeReadOnly** and **dbRepMakePartial**, or it can be 0.</span></span> <span data-ttu-id="52947-131">Например, чтобы создать частичную реплику только для чтения, необходимо передать значение **дбрепмакереадонли** + **дбрепмакепартиал** в качестве значения интоптионс.</span><span class="sxs-lookup"><span data-stu-id="52947-131">For example, to create a read-only partial replica, you should pass the value **dbRepMakeReadOnly** + **dbRepMakePartial** as the value of intOptions.</span></span>

```vb 
Function MakeAdditionalReplica(strReplicableDB As _ 
 String, strNewReplica As String, intOptions As _ 
 Integer) As Integer 
 
 Dim dbsTemp As Database 
 On Error GoTo ErrorHandler 
 
 Set dbsTemp = OpenDatabase(strReplicableDB) 
 
 ' If no options are passed to 
 ' MakeAdditionalReplica, omit the 
 ' options argument, which defaults to 
 ' a full, read/write replica. Otherwise, 
 ' use the value of intOptions. 
 
 If intOptions = 0 Then 
 dbsTemp.MakeReplica strNewReplica, _ 
 "Replica of " & strReplicableDB 
 Else 
 dbsTemp.MakeReplica strNewReplica, _ 
 "Replica of " & strReplicableDB, _ 
 intOptions 
 End If 
 
 dbsTemp.Close 
 
ErrorHandler: 
 Select Case Err 
 Case 0: 
 MakeAdditionalReplica = 0 
 Exit Function 
 Case Else: 
 MsgBox "Error " & Err & " : " & Error 
 MakeAdditionalReplica = Err 
 Exit Function 
 End Select 
 
End Function 
 
```
