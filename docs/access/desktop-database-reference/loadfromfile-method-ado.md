# <a name="loadfromfile-method-ado"></a>Метод LoadFromFile (ADO)

**Область применения**: Access 2013, Office 2013

ЗаГружает содержимое существующего файла в [поток](stream-object-ado.md).

## <a name="syntax"></a>Синтаксис

*Stream*. *Имя файла* LoadFromFile

## <a name="parameters"></a>Параметры

|Имя |Описание|
|:----|:----------|
|*FileName* |**Строковое** значение, содержащее имя файла, который необходимо загрузить в **поток**. *Filename* может содержать любой допустимый путь и имя в формате UNC. Если указанный файл не существует, возникает ошибка времени выполнения.|

## <a name="remarks"></a>Замечания

Этот метод можно использовать для загрузки содержимого локального файла в объект **Stream** . Это можно использовать для отправки содержимого локального файла на сервер.

Перед вызовом **LoadFromFile**объект **Stream** должен быть уже открыт. Этот метод не изменяет привязку объекта **Stream** ; Он по-прежнему будет привязан к объекту, указанному в URL-АДРЕСе или **записи** , с которой был изначально открыт **поток** .

**LoadFromFile** перезаписывает текущее содержимое объекта **Stream** , используя данные, считанные из файла. Все существующие в потоке **** байты перезаписываются содержимым файла. Все ранее существующие и оставшиеся байты после [EOS](eos-property-ado.md) , созданного с помощью **LoadFromFile**, усекаются.

После вызова **LoadFromFile**текущее положение устанавливается равным началу **потока** ([позиция](position-property-ado.md) равна 0).

Так как в начало потока можно добавить 2 байта для кодирования, размер потока может не совпадать с размером файла, из которого он был загружен.
