# <a name="item-property-ado-md-cellset"></a>Свойство Item (Cellset в ADO MD)

**Область применения**: Access 2013, Office 2013

Извлекает ячейку из элемента Cell с помощью ее координат.

## <a name="syntax"></a>Синтаксис

Задайте набор*ячеек*для ячейки.** =  Элемент (*Positions*)

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*Positions* |**Массив вариантов** значений, который уникально задает ячейку. *Положения* могут быть одним из следующих вариантов:<br/><br/>— Массив номеров расположений<br/>— Массив имен элементов<br/>— Порядковый номер |

## <a name="remarks"></a>Замечания

Используйте свойство **Item** , чтобы возвратить объект [Cell](cell-object-ado-md.md) в объекте [Cell](cellset-object-ado-md.md) . Если свойству **Item** не удается найти ячейку, соответствующую аргументу *Positions* , возникает ошибка.

Свойство **Item** является свойством по умолчанию для объекта набором **ячеек** . Следующие синтаксические формы являются взаимозаменяемыми:

```vb
    Cellset.Item ( Positions )
    Cellset ( Positions )
```

Аргумент *Positions* указывает, какую ячейку нужно вернуть. Вы можете указать ячейку по порядковому номеру или по позиции на каждой оси. Указывая положение ячейки вдоль каждой оси, вы можете указать числовое значение позиции или имена элементов для каждой позиции.

Порядковый номер — это число, уникально идентифицирующее одну ячейку в наборе **ячеек**. Как концептуально, ячейки пронумерованы в наборе **ячеек** , как если бы набор **ячеек** был ** одномерным массивом, где *p* — это количество осей. Ячейки направлены в основной порядок по строкам.

Если имена членов передаются в виде строк в **Item**, они должны быть указаны в порядке возрастания числовых идентификаторов осей. В пределах оси элементы должны быть перечислены в порядке возрастания вложенности измерений, то есть первым элементом измерения является первый элемент, а затем элементы внутренних измерений. Каждое измерение должно быть представлено отдельной строкой, а список строк членов должен разделяться запятыми.

> Получение ячеек по имени участника может не поддерживаться поставщиком данных. Для получения дополнительных сведений обратитесь к документации по вашему поставщику.
