# <a name="showtoolbar-macro-action"></a>Макрокоманда ShowToolbar

**Область применения**: Access 2013, Office 2013

С помощью действия **ShowToolbar** можно отобразить или скрыть группу команд на вкладке **"Надстройки".**

> [!NOTE]
> - Действие **ShowToolbar** не влияет на сочетания клавиш.
> - Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Заметки

Это действие можно использовать в макросах с условными выражениями для отображения или скрытие группы в зависимости от определенных условий.

Если вы хотите показать определенную группу только в одной форме или отчете, можно установить для свойства **OnActivate** формы или отчета имя макроса, который содержит действие **ShowToolbar** для показа группы. Затем укажите для свойства **OnDeactivate** формы или отчета имя макроса, который содержит действие **ShowToolbar,** чтобы скрыть группу.

Встроенные панели инструментов недоступны для отображения или скрытие с помощью этого действия, если для  свойства **AllowBuiltInToolbars** в модуле Visual Basic для приложений (VBA) установлено свойство False (0) или если для параметра **"Разрешить** встроенные панели инструментов" установлено свойство **False** в VBA с помощью метода **SetOption.**

Чтобы запустить действие **ShowToolbar** в модуле VBA, используйте метод **ShowToolbar** объекта **DoCmd.**
