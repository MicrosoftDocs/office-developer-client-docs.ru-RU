# <a name="position-property-ado"></a>Свойство Position (ADO)

**Область применения**: Access 2013, Office 2013

Указывает текущее положение в [объекте Stream.](stream-object-ado.md)

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает или возвращает **длинное** значение, которое указывает смещение текущей позиции от начала потока (в количестве в количествах). Значение по умолчанию — 0, которое представляет первый byte в потоке.

## <a name="remarks"></a>Заметки

Текущую позицию можно сдвигать в точку после окончания потока. Если указать текущее положение за пределами [](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/size-property-ado-stream) потока, размер объекта **Stream** будет увеличен соответствующим образом. Любые новые добавляемые таким образом bytes будут иметь null.

> [!NOTE]
> **Положение** всегда измеряется вбайтах. Для текстовых потоков с использованием многобайтового набора символов умножьте положение на размер символа, чтобы определить номер символа. Например, для двух byte character set первый символ находится в позиции 0, второй знак в позиции 2, третий символ в позиции 4 и так далее.

Отрицательные значения нельзя использовать для изменения текущей позиции в **потоке.** Для позиции можно использовать только положительные **числа.**

Для объектов **Stream** только для чтения ADO не возвращает ошибку, если **для Position** установлено значение больше, чем **размер** **потока.** Это не изменяет размер потока или не  изменяет его содержимое. Однако этого следует избегать, так как это приводит к бессмысленным **значениям Position.**
