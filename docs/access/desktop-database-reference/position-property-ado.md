# <a name="position-property-ado"></a>Свойство Position (ADO)

**Область применения**: Access 2013, Office 2013

Указывает текущее положение в [объекте Stream.](stream-object-ado.md)

## <a name="settings-and-return-values"></a>Параметры и значения возврата

Задает или возвращает значение **Long,** которое указывает смещение в количестве bytes текущего положения с начала потока. По умолчанию — 0, что представляет собой первый byte в потоке.

## <a name="remarks"></a>Примечания

Текущая позиция может быть перемещена в точку после окончания потока. Если указать текущее положение за пределами [](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/size-property-ado-stream) потока, размер объекта **Stream** будет соответственно увеличен. Любые новые bytes добавленные таким образом будут null.

> [!NOTE]
> **Позиция** всегда измеряет bytes. Для текстовых потоков с использованием многобайтных наборов символов умножьте положение на размер символа, чтобы определить число символов. Например, для набора символов с двумя баллами первый символ находится на позиции 0, второй — на позиции 2, третий — на позиции 4 и так далее.

Отрицательные значения нельзя использовать для изменения текущего положения в **потоке.** Только положительные номера можно использовать для **позиции**.

Для объектов  потока, только для чтения, ADO не возвращает ошибку,  если значение **Position** задается больше, чем размер **потока.** Это не изменяет размер потока **или** не изменяет содержимое **потока.** Однако этого следует избегать, так как это приводит к бессмысленному значению **Position.**
