# <a name="rds-returns-stream-not-read-error"></a>RDS возвращает \"ошибку "поток\" не прочитан"


**Область применения**: Access 2013, Office 2013

"Не удается прочитать объект Stream, так как он пуст или текущая позиция находится в конце потока. Для непустых потоков задайте текущее положение с помощью свойства Position. Чтобы определить, является ли поток пустым, проверьте свойство Size.

Если выдается сообщение об ошибке, оно может быть результатом использования параметризованного иерархического запроса по протоколу HTTP. RDS не позволяет выполнять удаленные параметризованные иерархии.
