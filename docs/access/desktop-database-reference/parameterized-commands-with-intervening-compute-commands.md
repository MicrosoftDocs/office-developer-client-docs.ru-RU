# <a name="parameterized-commands-with-intervening-compute-commands"></a>Параметризованные команды с промежуточными командами COMPUTE


**Область применения**: Access 2013, Office 2013

Типичная параметризованная команда APPEND фигуры  содержит предложение, которое создает родительский набор  записей с командой запроса и другое предложение, которое создает набор записей с параметризованной командой запроса, то есть командой, содержащей замещая параметр (вопросичная метка "?"). Итоговая фигура **recordset** имеет два уровня, на которых родительский уровень занимает верхний уровень, а потомок занимает нижний уровень.

Предложение, которое создает  набор записей, теперь может быть произвольным числом вложенных команд COMPUTE фигуры, где самая глубоко вложенная команда содержит параметризованный запрос. Полученный набор  записей имеет несколько уровней, на которых родительский уровень занимает верхний уровень, потомок занимает самый нижний уровень, а произвольное число наборов записей, созданных командами COMPUTE фигуры, занимают пересекающиеся уровни.

Обычно эта функция используется для вызова сводных возможностей функции и группировки команд SHAPE COMPUTE для создания пересекающихся объектов **Recordset** с аналитической информацией о наборе **записей.** Кроме того, поскольку это команда параметровизированной фигуры, при каждом доступе  к столбце главы родительского раздела может быть извлечен новый набор записей. Поскольку пересекающиеся уровни являются производными от детей, они также будут перекомпьютены.
