# <a name="parameterized-commands-with-intervening-compute-commands"></a>Parameterized Commands with Intervening COMPUTE Commands


**Применимо к**: Access 2013 | Office 2013

Фигурой типичного параметризованный команда APPEND имеет, который создает родительский объект **набора записей** с командой, запросов и другой предложения, который создает дочерний **набора записей** с помощью команды параметризованный запрос, то есть, содержащий параметр команды Заполнитель (вопросительный знак, «?»). Полученный форме **записей** имеет два уровня, в которых родительский занимает верхнего уровня и дочерние занимает низкого уровня.

Предложение, который создает дочерних **записей** теперь возможно произвольное число выполненных команд COMPUTE вложенные фигуры, которых наиболее глубоко вложенных команда содержит параметризованный запрос. Полученный форме **записей** имеет несколько уровней, в которых родительский занимает верхнем уровне, дочерние занимает цвет нижней уровень и произвольное число выполненных s **набора записей**, созданных с помощью команд COMPUTE фигуры занимают промежуточных уровни.

Эта функция обычно используется для вызова статистические функции и возможности группировки фигуры COMPUTE команды для создания промежуточных объектов **набора записей** с помощью аналитической сведения о дочерних **записей**. Кроме того так как это команда параметризованный фигуры при каждом обращении к ним столбец родительского новый дочерний **записей** могут быть получены. Так как промежуточных уровней являются производными от дочернего, они также будут пересчитывать.
