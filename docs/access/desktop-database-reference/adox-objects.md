# <a name="adox-objects"></a>Объекты ADOX

**Область применения**: Access 2013, Office 2013

Каждый объект может содержаться в соответствующей коллекции. Например, объект **Table** может содержаться в коллекции [Таблицы.](tables-collection-adox.md) Дополнительные сведения см. в [разделе ADOX collections](adox-collections.md) or a specific collection topic.
