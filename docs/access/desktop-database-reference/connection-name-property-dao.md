# <a name="connectionname-property-dao"></a>Connection.Name Property (DAO)


**Применимо к**: Access 2013 | Office 2013

Возвращает имя **[подключения](connection-object-dao.md)**.

## <a name="syntax"></a>Синтаксис

*выражение* . Имя

*выражение* Переменная, которая содержит объект **подключения** .

## <a name="remarks"></a>Замечания

Можно использовать свойство **Name** объекта с помощью Visual Basic для приложений **Dim** оператор в коде для создания других экземпляров объекта.
