# <a name="addmenu-macro-action"></a>Макрокоманда AddMenu


**Область применения**: Access 2013, Office 2013

В этой статье описывается базовая операция макрокоманды **ДобавитьМеню** .

Вы можете использовать макрокоманду **ДобавитьМеню** для создания:

- Настраиваемые меню на вкладке **** "надстройки" для конкретной формы или отчета.

- Пользовательское контекстное меню для формы, отчета или элемента управления. Пользовательское контекстное меню заменяет встроенное контекстное меню для формы, отчета или элемента управления.

- Глобальное контекстное меню. Глобальное контекстное меню заменяет встроенное контекстное меню для полей в таблицах и таблицах, формах и отчетах, за исключением случаев, когда вы добавили пользовательское контекстное меню для формы, отчета или элемента управления.

## <a name="remarks"></a>Замечания

Чтобы выполнить макрокоманду **ДобавитьМеню** в модуле Visual Basic для приложений (VBA), используйте метод **ДобавитьМеню** объекта DoCmd. **** Кроме того, можно задать свойство **MenuBar** или **ShortcutMenuBar** в VBA, чтобы создать настраиваемое меню на вкладке " **надстройки** " или присоединить пользовательское контекстное меню к форме, отчету или элементу управления. Вы можете задать свойство **ShortcutMenuBar** объекта **Application** , чтобы создать глобальное контекстное меню.
