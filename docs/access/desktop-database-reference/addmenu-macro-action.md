# <a name="addmenu-macro-action"></a>Макрокоманда AddMenu


**Область применения**: Access 2013, Office 2013

В этой статье описывается основная операция макрос **действия AddMenu.**

Вы можете использовать **действие AddMenu** для создания:

- Настраиваемые меню на **вкладке Надстройки** для определенной формы или отчета.

- Настраиваемое меню ярлыка для формы, отчета или управления. Настраиваемое меню ярлыков заменяет встроенное меню ярлыка для формы, отчета или управления.

- Глобальное меню ярлыков. Глобальное меню ярлыков заменяет встроенное меню ярлыков для полей в таблицах и таблицах данных запросов, форм и отчетов, за исключением тех случаев, когда добавлено настраиваемое меню ярлыка для формы, отчета или управления.

## <a name="remarks"></a>Примечания

Чтобы выполнить **действие AddMenu** в Visual Basic для приложений (VBA), используйте метод **AddMenu** объекта **DoCmd.** Вы также можете настроить свойство **MenuBar** или **ShortcutMenuBar** в VBA для создания настраиваемого меню на вкладке **Надстройки** или для добавления настраиваемого меню ярлыка в форму, отчет или управление. Вы можете настроить свойство **ShortcutMenuBar** объекта **Application** для создания глобального меню ярлыков.
