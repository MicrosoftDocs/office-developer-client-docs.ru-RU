# <a name="understanding-the-customization-file"></a>Understanding the Customization File


**Область применения**: Access 2013, Office 2013

Каждый заглавный раздел в файле настройки состоит из квадратных скобок **\[\]** () с типом и параметром. Четыре типа разделов указываются литеральными строками **подключения,** **sql,** **userlist** или **журналов**. Параметр — буквальная строка, по умолчанию, идентификатор, указанный пользователем, или ничего.

Поэтому каждый раздел отмечен одним из следующих заглавных разделов:

```text 
 
[ connect   default     ]
[ connect   identifier  ]
[ sql       default     ]
[ sql       identifier  ]
[ userlist  identifier  ]
[ logs                  ]
```

**DataFactory вызывает** обработчик, передавая параметры клиента. Обработчик ищет целые строки в параметрах клиента, соответствующих идентификаторам в соответствующих заголовках разделов. Если совпадение найдено, содержимое этого раздела применяется к параметру клиента.

Определенный раздел используется при следующих обстоятельствах:

  - Раздел **подключения** используется, если значение части клиента соединит ключевое слово строки "**Источник данных=****значение",* совпадает с идентификатором **раздела подключения** *.*

  - Раздел **sql** используется, если строка команды клиента содержит строку, которая соответствует идентификатору раздела **sql.**

  - Если **нет** совпадающих идентификаторов, используется раздел connect или **sql** с параметром по умолчанию.

  - Если идентификатор раздела **userlist** совпадает с идентификатором **раздела connect,** используется раздел **userlist.** Если имеется совпадение, содержимое раздела **userlist** применяется к подключению, управляемому разделом **connect.**

  - Если строка в строке подключения или командной  строке не соответствует идентификатору в любом  заголовке раздела connect или **sql,** и нет заголовника раздела подключения или **sql** с параметром по умолчанию, то клиентская строка используется без изменений.

  - Раздел **журналов** используется при работе **DataFactory.**
