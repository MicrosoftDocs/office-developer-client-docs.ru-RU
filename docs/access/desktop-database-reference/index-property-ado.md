<span data-ttu-id="11b09-115">Индекс с именем, свойство **Index** необходимо ранее объявленные в базовой таблице объекта **набора записей** .</span><span class="sxs-lookup"><span data-stu-id="11b09-115">The index named by the **Index** property must have previously been declared on the base table underlying the **Recordset** object.</span></span> <span data-ttu-id="11b09-116">То есть, индекса необходимо объявлены программными средствами как объект ADOX [индекса](index-object-adox.md) или создания базовой таблицы.</span><span class="sxs-lookup"><span data-stu-id="11b09-116">That is, the index must have been declared programmatically either as an ADOX [Index](index-object-adox.md) object, or when the base table was created.</span></span>

  - <span data-ttu-id="11b09-120">Если **записей** по-прежнему выполняет операцию (который устанавливается с помощью свойства [State](state-property-ado.md) ).</span><span class="sxs-lookup"><span data-stu-id="11b09-120">If the **Recordset** is still executing an operation (which can be determined by the [State](state-property-ado.md) property).</span></span>

  - <span data-ttu-id="11b09-121">Если установлен фильтр **набора записей** с помощью свойства [фильтра](filter-property-ado.md) .</span><span class="sxs-lookup"><span data-stu-id="11b09-121">If a filter has been set on the **Recordset** with the [Filter](filter-property-ado.md) property.</span></span>

<span data-ttu-id="11b09-125">Если индекс можно задать, является ли данное свойство [LockType для](locktype-property-ado.md) **adLockPessimistic** или **adLockOptimistic**неявных [UpdateBatch](updatebatch-method-ado.md) операция выполняется.</span><span class="sxs-lookup"><span data-stu-id="11b09-125">If the index can be set and the [LockType](locktype-property-ado.md) property is **adLockPessimistic** or **adLockOptimistic**, then an implicit [UpdateBatch](updatebatch-method-ado.md) operation is performed.</span></span> <span data-ttu-id="11b09-126">Это освобождает текущий и затронутых группы.</span><span class="sxs-lookup"><span data-stu-id="11b09-126">This releases the current and affected groups.</span></span> <span data-ttu-id="11b09-127">Выпущен любой существующий фильтр и текущей позиции строки изменяется в первую строку упорядоченный **набор записей**.</span><span class="sxs-lookup"><span data-stu-id="11b09-127">Any existing filter is released, and the current row position is changed to the first row of the reordered **Recordset**.</span></span>
