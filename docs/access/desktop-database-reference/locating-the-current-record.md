# <a name="locating-the-current-record"></a>Поиск текущей записи

**Область применения**: Access 2013, Office 2013

Текущее положение курсора в **наборе записей** разграничит текущее положение записи. При условии, что команда возвращает результаты, курсор автоматически помещается в первую запись, когда вызван метод **Recordset** **Open.** Таким образом, в примере **recordset** курсор будет в первой записи, "Угодный боб".
