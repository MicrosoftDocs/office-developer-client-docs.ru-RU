# <a name="position-object-ado-md"></a>Объект Position (ADO MD)


**Область применения**: Access 2013, Office 2013

Представляет набор из одного или нескольких элементов разных измерений, определяющих точку вдоль оси.

## <a name="remarks"></a>Замечания

С помощью свойств и коллекций объекта **position** можно выполнить следующие действия:

  - Свойство **Ordinal** используется для возврата порядкового номера **позиции** вдоль [оси](axis-object-ado-md.md).

  - Коллекция [Members](members-collection-ado-md.md) используется для возврата элементов, составляющих позицию на **оси**.
