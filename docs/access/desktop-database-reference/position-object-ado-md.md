# <a name="position-object-ado-md"></a>Position Object (ADO MD)


**Применимо к**: Access 2013 | Office 2013

Представляет набор одного или нескольких элементов различных измерений, который определяет точку оси.

## <a name="remarks"></a>Замечания

С помощью свойств и коллекций **позиции** объекта выполните следующее:

  - Воспользуйтесь свойством **порядковый номер** для возврата порядковый номер **позиции** по [оси](axis-object-ado-md.md).

  - Используйте коллекцию [элементов](members-collection-ado-md.md) для возврата элементов, которые составляют положение по **оси**.
