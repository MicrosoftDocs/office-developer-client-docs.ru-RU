# <a name="logevent-macro-action"></a>Макрокоманда LogEvent

**Область применения**: Access 2013, Office 2013

Действие **ложевент** записывает сведения в системную таблицу **усисаппликатионлог** .

> [!NOTE]
> Действие **ложевент** доступно только в макросах данных.

## <a name="remarks"></a>Примечания

Действие **ложевент** можно использовать для записи сведений о состоянии в системную таблицу **усисаппликатионлог** , которые не могут использовать действие **[раисиррор](raiseerror-macro-action.md)** для генерации ошибки. Например, вы можете записать изменения в определенном поле или использовать элементы, записанные в **усисаппликатионлог** , чтобы помочь вам в отладке макроса.

При использовании действия **ложевент** для записи в таблицу **Усисаппликатионлог** для столбца **Category** автоматически устанавливается значение **User**.

Чтобы просмотреть таблицу **усисаппликатионлог** , выполните указанные ниже действия.

1.  Откройте меню **файл** и выберите пункт **Параметры**.

2.  В диалоговом окне **Параметры Access** перейдите на вкладку **Текущая база данных** .

3.  В разделе **Навигация** щелкните **Параметры навигации**.

4.  В диалоговом окне **Параметры навигации** нажмите кнопку **Показать системные объекты**, а затем нажмите кнопку **ОК**.

5.  Нажмите кнопку **ОК** , чтобы закрыть диалоговое окно **Параметры Access** .
