# <a name="echo-macro-action"></a>Макрокоманда Echo

**Область применения**: Access 2013, Office 2013

Вы можете использовать макрокоманду **echo (Echo** ), чтобы указать, включена ли функция Echo. Например, это действие можно использовать для скрытия или отображения результатов макроса во время его выполнения.

## <a name="setting"></a>Параметр

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

Макрокоманда **echo** имеет следующие аргументы.

При запуске макроса часто отображает сведения, не необходимые для работы макроса. Если для аргумента **echo on** задано значение **нет**, макрос запускается без обновления экрана. Когда макрос закончится, Access автоматически включит Echo обратно и перерисовывает окно. Параметр **No** для аргумента **echo on** не влияет на функциональность макроса или его результаты.

Действие **echo** не отключает отображение модальных диалоговых окон, например сообщений об ошибках или всплывающих форм, таких как страницы свойств. Вы можете использовать диалоговые окна и всплывающие формы для сбора и отображения данных, даже если режим эхо отключен. Чтобы отключить все сообщения и диалоговые окна, за исключением окон сообщений об ошибках и диалоговых окон, в которых пользователь должен ввести сведения, используйте действие **сетварнингс** .

В макросе можно выполнить макрокоманду **echo** несколько раз. Это позволяет изменить текст строки состояния во время выполнения макроса.

Если отключить режим Echo, вы можете использовать действие **дисплайхаургласспоинтер** , чтобы изменить указатель мыши на значок песочных часов (или любой значок указателя мыши, заданный для "занято"), чтобы предоставить визуальную индикацию запуска макроса.

Чтобы выполнить макрокоманду **echo** в модуле Visual Basic для приложений (VBA), используйте метод **echo** объекта **DoCmd** .
