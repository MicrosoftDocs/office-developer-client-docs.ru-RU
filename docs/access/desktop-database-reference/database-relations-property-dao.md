# <a name="databaserelations-property-dao"></a>Свойство Database.Relations (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **отношений** , который содержит все хранимые объекты **отношения** для указанной базы данных. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . Отношения

*выражение* Переменная, которая представляет собой объект **базы данных** .
