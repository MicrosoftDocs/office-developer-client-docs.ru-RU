# <a name="granting-guest-privileges-to-a-web-server-computer-rds-guest-privileges"></a>Предоставление гостевых привилегий компьютеру веб-сервера; Привилегии гостей RDS

**Область применения**: Access 2013, Office 2013

Учетная запись анонимного веб-сервера (IUSR ComputerName) должна быть добавлена в локализованную группу Гостей на компьютере веб-сервера для \_ использования RDS.

**Предоставление гостевых привилегий компьютеру веб-сервера**

1.  На компьютере Microsoft Windows 2000 Server нажмите кнопку Начните **,** указать на **программы,** указать на административные **средства,** а затем нажмите **кнопку Управление компьютером**.

2.  В дереве консоли в **локальных пользователях и группах** щелкните **Группы**.

3.  Выберите **локализованную группу** "Гости". Из меню **Действия** выберите **свойства**.

4.  В **диалоговом окне Свойства гостей** нажмите кнопку **Добавить**.

5.  Если учетная запись анонимного веб-сервера не появится в списке в диалоговом окне **Выбор** пользователей или групп, введите ее имя (IUSR \_ *ComputerName)* в нижнее пустое поле и нажмите кнопку **Добавить**.

6.  Нажмите кнопку **ОК**.
