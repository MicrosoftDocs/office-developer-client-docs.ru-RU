# <a name="granting-guest-privileges-to-a-web-server-computer-rds-guest-privileges"></a>Предоставление гостевых привилегий компьютеру веб-сервера; Гостевые привилегии RDS

**Область применения**: Access 2013, Office 2013

Учетная запись анонимного веб-\_сервера (IUSR*ComputerName*) должна быть добавлена в локальную группу гостей на компьютере веб-сервера для использования RDS.

**Предоставление гостевых привилегий компьютеру веб-сервера**

1.  На компьютере под управлением Microsoft Windows 2000 Server нажмите кнопку **Пуск**, последовательно выберите пункты **программы**, **Администрирование**и **Управление компьютером**.

2.  В дереве консоли в разделе **Локальные пользователи и группы**щелкните **группы**.

3.  Выберите локальную группу " **гости** ". В меню **действие** выберите пункт **свойства**.

4.  В диалоговом окне **Свойства: гости** нажмите кнопку **Добавить**.

5.  Если анонимная учетная запись веб-сервера не отображается в списке в диалоговом окне **Выбор пользователей или групп** , введите его имя (\_IUSR*ComputerName*) в нижнем пустом поле и нажмите кнопку **Добавить**.

6.  Нажмите кнопку **ОК**.
