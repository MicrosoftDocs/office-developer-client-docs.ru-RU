# <a name="renameobject-macro-action"></a>Макрокоманда RenameObject

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **ренамеобжект** для переименования указанного объекта базы данных.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Примечания

Новое имя объекта базы данных должно соответствовать стандартным соглашениям об именовании для объектов Access.

Невозможно переименовать открытый объект.

Если оставить аргументы **тип объекта** и **старый имя** пустыми, Access переименует объект, выбранный в области навигации. Чтобы выбрать объект в области навигации, можно использовать макрокоманду **"ВыделитьОбъект"**( **ВыделитьОбъект** ) с параметром **в разделе в области навигации** со значением Да.

Вы также можете переименовать объект, щелкнув его правой кнопкой мыши в области навигации, выбрав команду **Переименовать**и введя новое имя. С помощью действия **ренамеобжект** не нужно сначала выбирать объект в области навигации, а также останавливать макрос для ввода нового имени.

Это действие отличается от команды **КопироватьОбъект** , которая создает копию объекта под новым именем.

Чтобы запустить действие **ренамеобжект** в модуле Visual Basic для приложений (VBA), используйте метод **Rename** объекта **DoCmd** .
