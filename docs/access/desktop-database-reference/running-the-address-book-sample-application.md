# <a name="running-the-address-book-sample-application"></a>Запуск примера приложения адресной книги

**Область применения**: Access 2013, Office 2013

Чтобы запустить приложение адресной книги, выполните указанные ниже действия.

**Чтобы запустить это приложение, выполните следующие командлеты.**

1.  Убедитесь, что запущен Microsoft SQL Server. Нажмите кнопку **Пуск**, последовательно выберите пункты **программы**, **Microsoft SQL Server 7,0**и **Диспетчер служб**. Если в белом кружке есть зеленая стрелка, то выполняется SQL Server. Если это не так (в белом кружке окажется красным квадратиком), нажмите кнопку **начать/продолжить**.

2.  В Microsoft Internet Explorer 4,0 или более поздней версии введите следующий адрес: **https:///РДС/аддрессбук/аддрбук.АСП***webserver***** веб-сервер, где веб- *сервер* — это имя веб-сервера, на котором установлены компоненты RDS Server.

3.  Затем вы можете попробовать различные сценарии в примере приложения адресной книги, например поиск человека на основе его имени электронной почты, список всех людей с названием "Диспетчер программ" или редактирование существующих записей. Нажмите кнопку **найти** , чтобы заполнить сетку данных все доступные имена.
