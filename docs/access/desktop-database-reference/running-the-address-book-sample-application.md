# <a name="running-the-address-book-sample-application"></a>Запуск примера приложения адресной книги

**Область применения**: Access 2013, Office 2013

Чтобы запустить приложение Адресная книга, выполните эту процедуру.

**Для запуска этого приложения**

1.  Убедитесь, что Microsoft SQL Server запущен. Нажмите **кнопку Начните,** указать на **программы,** указать на Microsoft SQL Server **7.0,** а затем нажмите **диспетчер службы**. Если в белом круге есть зеленая стрелка, SQL Server запущена. Если это не так (в белом круге будет красный квадрат), нажмите **кнопку Начните/Продолжить**.

2.  В Microsoft Internet Explorer 4.0 или более поздней странице введите следующий адрес: **https://***webserver***/RDS/AddressBook/AddrBook.asp,** где веб-сервер является именем веб-сервера, на котором установлены компоненты сервера RDS. 

3.  Затем можно попробовать различные сценарии в примере приложения адресной книги, например поиск человека на основе его имени электронной почты, перечисление всех людей с заголовком "Менеджер программы" или редактирование существующих записей. Щелкните **Найти,** чтобы заполнить сетку данных всеми доступными именами.
