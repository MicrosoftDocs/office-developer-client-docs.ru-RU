# <a name="running-the-address-book-sample-application"></a>Запуск примера приложения адресной книги

**Область применения**: Access 2013, Office 2013

Чтобы запустить приложение адресной книги, выполните эту процедуру.

**Запуск этого приложения**

1.  Убедитесь, что Microsoft SQL Server запущена. Нажмите **кнопку**"Начните", **найдите** **"Программы", Microsoft SQL Server 7.0,** а затем выберите **"Диспетчер служб".** Если в белом круге есть зеленая стрелка, SQL Server запущена. Если это не так (в белом круге будет красный квадрат), нажмите кнопку **"Начните/продолжить".**

2.  В Microsoft Internet Explorer 4.0 или более поздней введите следующий адрес: **https://***webserver***/RDS/AddressBook/AddrBook.asp,** где *webserver* — это имя веб-сервера, на котором установлены компоненты сервера RDS.

3.  Затем можно попробовать различные сценарии в примере приложения адресной книги, такие как поиск человека на основе его имени электронной почты, перечисление всех людей с заголовком "Program Manager" или редактирование существующих записей. Нажмите **кнопку** "Найти", чтобы заполнить сетку данных всеми доступными именами.
