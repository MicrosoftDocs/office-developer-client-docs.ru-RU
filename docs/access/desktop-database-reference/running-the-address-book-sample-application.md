# <a name="running-the-address-book-sample-application"></a>Запуск примера приложения адресной книги


**Применимо к**: Access 2013, Office 2013

Чтобы запустить приложение адресной книги, выполните следующую процедуру.

**Для запуска этого приложения**

1.  Убедитесь в том, что запущен Microsoft SQL Server. Нажмите кнопку **Пуск**, выберите пункты **программы**, выберите пункт **Microsoft SQL Server 7.0**и затем **Диспетчер служб**. Если белый кружок зеленую стрелку, затем выполняется SQL Server. Если он не установлен (будет красный квадрат белый кружок), нажмите кнопку **Пуск и продолжить**.

2.  В Microsoft Internet Explorer 4.0 или более поздней версии, введите следующий адрес: **https:// /RDS/AddressBook/AddrBook.asp***веб-сервера***** , где *веб-сервер* — это имя веб-сервере, где установлены серверные компоненты служб удаленных рабочих СТОЛОВ.

3.  Затем можно попробовать различных сценариев в примере приложения адресной книги, такие как поиске сотрудника на основании его имя электронной почты, список всех людей с заголовком «Руководитель программы», или изменения существующих записей. Нажмите кнопку **Найти** для заполнения сетки данных со всех доступных имен.
