# <a name="selectobject-macro-action"></a>Макрокоманда SelectObject

**Область применения**: Access 2013, Office 2013

## <a name="remarks"></a>Замечания

Макрокоманда **ВыделитьОбъект** работает с любым объектом Access, который может получить фокус. Это действие приводит к указанному объекту фокуса и отображает объект, если он скрыт. Если объект является формой, то Макрокоманда **ВыделитьОбъект** устанавливает для свойства **Visible** формы значение **Да** и возвращает форму в режим, заданный свойствами формы (например, в виде модальной или всплывающей формы).

Если объект не открыт в одном из других окон Access, можно выбрать его в области навигации, указав для аргумента **в области навигации** значение **Да**. Если для аргумента **в области навигации** задано значение **нет**, при попытке выбрать объект, который не открыт, появится сообщение об ошибке.

Это действие часто можно использовать для выбора объекта, для которого требуется выполнить дополнительные действия. Например, если доступ настроен на использование перекрывающихся окон вместо документов с вкладками, может потребоваться восстановить объект, свернутый (с помощью действия **ресторевиндов** ), или развернуть окно, содержащее объект, с которым вы хотите работать. (с помощью действия **максимизевиндов** ).

Если выбрана форма, вы можете использовать действия **КЭлементуУправления**, **GoToRecord**и **GoToPage** , чтобы перейти к определенным областям формы. Действие **GoToRecord** также работает для таблиц с таблицами.

Чтобы выполнить макрокоманду **ВыделитьОбъект** в модуле Visual Basic для приложений (VBA), используйте метод **SelectObject** объекта DoCmd. ****
