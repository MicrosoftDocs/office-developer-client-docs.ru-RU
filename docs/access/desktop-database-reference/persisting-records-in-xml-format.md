<span data-ttu-id="7175c-113">Это самый простой способ сохранения данных в формат XML и загрузить его обратно еще раз через ADO — с помощью методов **Сохранить** и **Открыть** соответственно.</span><span class="sxs-lookup"><span data-stu-id="7175c-113">The easiest way to persist data into XML and load it back again through ADO is with the **Save** and **Open** methods, respectively.</span></span> <span data-ttu-id="7175c-114">В следующем примере кода ADO демонстрируется сохранение данных в таблице заголовки в файл с именем titles.sav.</span><span class="sxs-lookup"><span data-stu-id="7175c-114">The following ADO code example demonstrates saving the data in the Titles table to a file named titles.sav.</span></span>

```vb 
 
Dim rs as new Recordset 
Dim rs2 as new Recordset 
Dim c as new Connection 
Dim s as new Stream 
 
' Query the Titles table. 
c.Open "provider='sqloledb';data source='mydb';initial catalog='pubs';Integrated Security='SSPI'" 
rs.cursorlocation = adUseClient 
rs.Open "select * from titles", c, adOpenStatic 
 
' Save to the file in the XML format. Note that if you don't specify 
' adPersistXML, a binary format (ADTG) will be used by default. 
rs.Save "titles.sav", adPersistXML 
 
' Save the Recordset into the ADO Stream object. 
rs.save s, adPersistXML 
rs.Close 
c.Close 
 
set rs = nothing 
 
Reopen the file. 
rs.Open "titles.sav",,,,adCmdFile 
'Open the Stream back into a Recordset. 
rs2.open s 
```

<span data-ttu-id="7175c-115">ADO всегда сохраняется весь объект **набора записей** .</span><span class="sxs-lookup"><span data-stu-id="7175c-115">ADO always persists the entire **Recordset** object.</span></span> <span data-ttu-id="7175c-116">Если вы хотите сохраняться только подмножество строк объекта **набора записей** , используйте метод **фильтра** сузить строки или измените предложение вашего выбора.</span><span class="sxs-lookup"><span data-stu-id="7175c-116">If you wish to only persist a subset of rows of the **Recordset** object, use the **Filter** method to narrow down the rows or change your selection clause.</span></span> <span data-ttu-id="7175c-117">Тем не менее, необходимо открыть объект **набора записей** с курсором со стороны клиента (**CursorLocation** = **adUseClient**) использовать метод **фильтра** для сохранения подмножество строк.</span><span class="sxs-lookup"><span data-stu-id="7175c-117">However, you must open a **Recordset** object with a client-side cursor (**CursorLocation** = **adUseClient**) to use the **Filter** method for saving a subset of rows.</span></span> <span data-ttu-id="7175c-118">К примеру для получения заголовков, которые начинаются с буквы «б», можно применить фильтр open объект **набора записей** :</span><span class="sxs-lookup"><span data-stu-id="7175c-118">For example, to retrieve titles that start with the letter "b," you can apply a filter to an open **Recordset** object:</span></span>

```vb 
 
rs.Filter "title_id like 'B*'" 
rs.Save "btitles.sav", adPersistXML 
```
