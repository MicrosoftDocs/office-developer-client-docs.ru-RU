# <a name="singlestep-macro-action"></a>Макрокоманда SingleStep

**Применимо к**: Access 2013, Office 2013

Действие **SingleStep** можно использовать для приостановки выполнения макроса и открыть диалоговое окно **Макроса одним действием** .

## <a name="setting"></a>Параметр

Действие **SingleStep** не имеет каких-либо аргументов.

## <a name="remarks"></a>Примечания

- Действие **SingleStep** используется для устранения неполадок макрос, не работает должным образом. Можно добавить действие **SingleStep** макрос перед действие, который может быть причина проблемы. Действие приостанавливает макрос и открывает диалоговое окно **Макроса одним действием** . Это диалоговое окно сведения о получена, такие как имя макроса, любого указанного условия, действие имя, аргументы и номер ошибки, если это возможно. В диалоговом окне нажмите кнопку **Шаг** , чтобы перейти к следующей действия макроса, **Остановите все макросы** для остановки текущего макроса и другие макросы, работающих под управлением или **Продолжить** , чтобы остановить пошаговый и возобновления нормальной работы макроса.

- Действие **SingleStep** имеет следующий эффект, щелкнув **Один шаг** в группе **Работа** на вкладке **Конструктор** построения макросов. Различие между это и выполнение действия **SingleStep** — это что, выполнив действия, вы можете разместить действие в макросе, точно место одного пошаговое выполнение, чтобы начать. Таким образом, у вас нет к действию через все предыдущие действия для перехода к вам нужно проверить. С другой стороны Если щелкнуть **Один шаг** в инструменте построения макросов, вам необходимо сделать перед выполнение макроса. В этом случае пошаговый начинается с первого действия в макрос.

> [!NOTE]
> Если вы пошагово до конца макрос без нажав кнопку **Продолжить**, единого пошаговое выполнение будет по-прежнему быть фактически окончания макрос. Все последующие макросов, при запуске будет запускаться в режиме одним действием. Чтобы отключить режим пошагового нажмите кнопку **Продолжить** в диалоговом окне **Макрос одним действием** или, с помощью макроса в режиме конструктора, на вкладке **Конструктор** в группе **Сервис** нажмите кнопку **Один шаг** , чтобы он больше не выбран.
