# <a name="databasetabledefs-property-dao"></a>Свойство Database.TableDefs (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **TableDefs** , который содержит все объекты **TableDef** , хранящиеся в указанной базе данных. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . TableDefs

*выражение* Переменная, которая представляет собой объект **базы данных** .
