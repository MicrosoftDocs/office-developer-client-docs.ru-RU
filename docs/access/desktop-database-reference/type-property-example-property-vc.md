# <a name="type-property-example-property-vc"></a>Пример использования свойства Type (Property) (VC++)


**Область применения**: Access 2013, Office 2013

В этом примере [демонстрируется свойство Type.](type-property-ado.md) Это модель утилиты для перечисления имен и типов коллекции, таких как [свойства,](properties-collection-ado.md) [поля](fields-collection-ado.md)и т.д.

Нам не нужно открывать набор [Recordset для](recordset-object-ado.md) доступа к его коллекции **свойств;** они в жизнь, когда **объект Recordset** мгновенно. Однако настройка свойства [CursorLocation](cursorlocation-property-ado.md) в **adUseClient** добавляет несколько динамических  свойств в коллекцию свойств объекта **Recordset,** что делает пример немного более интересным. Для иллюстрации мы явно используем свойство [Item](item-property-ado.md) для доступа к каждому [объекту Property.](property-object-ado.md)

void PrintComError(_com_error &e);

// Main Function // 
// // 
//////////////////////////////////////////////////////////
