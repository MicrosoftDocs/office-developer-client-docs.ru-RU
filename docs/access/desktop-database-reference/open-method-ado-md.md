# <a name="open-method-ado-md"></a>Метод Open (ADO MD)

**Область применения**: Access 2013, Office 2013

Получает результаты многомерного запроса и возвращает результаты в набор ячеек.

## <a name="syntax"></a>Синтаксис

Набор *ячеек*. Открытый*источник*, *ActiveConnection*

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*Source* |Необязательное. Значение **Variant** , которое оценивается как допустимый многомерный запрос, например многомерный запрос многомерных выражений (MDX). *Исходный* аргумент соответствует свойству [Source](source-property-ado-md.md) . Для получения дополнительных сведений об многомерных выражениях обратитесь к документации по OLE DB для OLAP в пакете SDK компонентов MDAC Data Components.|
|*ActiveConnection* |Необязательное. Значение **Variant** , которое оценивается как строка, указывающая допустимое имя переменной объекта [подключения](connection-object-ado.md) ADO или определение подключения. Аргумент *ActiveConnection* указывает подключение, в котором будет открыт объект "набор [ячеек](cellset-object-ado-md.md) ". Если вы передаете определение подключения для этого аргумента, ADO открывает новое подключение, используя указанные параметры. Аргумент *ActiveConnection* соответствует свойству [ActiveConnection](activeconnection-property-ado-md.md) .|

## <a name="remarks"></a>Примечания

Метод **Open** создает ошибку, если какой-либо из его параметров опущен, а соответствующее значение свойства не было задано, прежде чем пытаться открыть набор **ячеек**.
