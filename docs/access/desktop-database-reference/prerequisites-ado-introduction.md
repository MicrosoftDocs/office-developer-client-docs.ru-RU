# <a name="prerequisites-ado-introduction"></a>Prerequisites (ADO Introduction)

**Область применения**: Access 2013, Office 2013

*Руководство программиста по ADO* будет удобным для разработчиков с широким спектром фоновых рисунков. У читателей должен быть как минимум промежуточный уровень взаимодействия при разработке приложений с помощью Microsoft Visual Basic, так как большинство примеров в руководстве написаны на этом языке. Другие примеры написаны на Microsoft Visual C++, Java, Visual Basic, Scripting Edition (VBScript) и Microsoft JScript.

Так как ADO используется для доступа к данным из различных источников, читателям также может потребоваться понимание основ основных концепций системы управления реляционными базами данных, основных понятий аналитической обработки (OLAP) и основных понятий, связанных с Интернетом и Протоколы Интернета.

ADO является частью стратегии Microsoft Data Access (УДА). (Дополнительные сведения о уда можно найти в [статье роль ADO в доступе к данным Microsoft](the-role-of-ado-in-microsoft-data-access.md), приведенной далее в этой главе.) Таким образом, он взаимодействует с технологией OLE DB. OLE DB основано на модели COM. Таким образом, знакомство с COM также может быть полезен для понимания некоторых более сложных концепций ADO.
