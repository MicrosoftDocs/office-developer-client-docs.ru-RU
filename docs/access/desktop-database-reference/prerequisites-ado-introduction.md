# <a name="prerequisites-ado-introduction"></a>Prerequisites (ADO Introduction)

**Область применения**: Access 2013, Office 2013

Руководство *программиста ADO* окажется полезным для разработчиков с самыми разнообразными фонами. Как минимум, читатели должны иметь промежуточный уровень опыта разработки приложений с microsoft Visual Basic, так как большинство примеров в руководстве написаны на этом языке. Другие примеры написаны в Microsoft Visual C++, Java, Visual Basic, Scripting Edition (VBScript) и Microsoft JScript.

Поскольку ADO используется для доступа к данным из различных источников, читателям также может потребоваться некоторое понимание фундаментальных концепций системы управления базами данных, концепций интерактивной аналитической обработки (OLAP) и базового ознакомления с протоколами Интернета и Интернета.

ADO является частью стратегии Microsoft Data Access (UDA). (Дополнительные сведения о UDA см. в [разделах Роль ADO в Microsoft Data Access](the-role-of-ado-in-microsoft-data-access.md)( далее в этой главе).) Таким образом, он совме-ется с технологией OLE DB. OLE DB основан на объектной модели компонентов Майкрософт (COM). Таким образом, знакомство с COM также может быть полезно для понимания некоторых более продвинутых концепций в ADO.
