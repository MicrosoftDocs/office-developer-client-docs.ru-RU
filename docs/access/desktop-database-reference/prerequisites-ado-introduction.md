# <a name="prerequisites-ado-introduction"></a>Prerequisites (ADO Introduction)

**Применимо к**: Access 2013, Office 2013

*Руководство программиста ADO* будет оказаться полезным для разработчиков с различными фона. Как минимум читатели должен промежуточный уровень взаимодействия при разработке приложений с помощью Microsoft Visual Basic, поскольку большая часть примеров, описанных в руководстве по записываются на этом языке. Другие примеры написаны в Microsoft Visual C++, Java, Visual Basic, Scripting Edition (VBScript) и Microsoft JScript.

Так как ADO используется для доступа к данным из различных источников, читатели может потребоваться некоторые общие сведения о основные понятия системы управления основной реляционной базе данных, концепции интерактивной аналитической обработки (OLAP) и знакомство с Интернетом и Протоколы Интернета.

ADO является частью стратегии доступа к данным Microsoft (Функциональности). (Дополнительные сведения о Функциональности см. Далее в этой главе [Роль ADO, в данных Microsoft Access](the-role-of-ado-in-microsoft-data-access.md).) Таким образом взаимодействует с технологией OLE DB. OLE DB основано на Microsoft модели компонентных объектов (COM). Таким образом Знакомство с COM также может быть полезен для понимания некоторые дополнительные концепции в ADO.
