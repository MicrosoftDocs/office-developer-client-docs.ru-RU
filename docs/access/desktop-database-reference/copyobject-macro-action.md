# <a name="copyobject-macro-action"></a>Макрокоманда CopyObject

**Область применения**: Access 2013, Office 2013

Вы можете использовать **действие CopyObject** для копирования указанного объекта базы данных в другую базу данных доступа или в ту же базу данных или проект Access под новым именем. Например, можно скопировать или создать копию существующего объекта в другой базе данных или быстро создать аналогичный объект с несколькими изменениями.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Примечания

Для этого действия необходимо ввести значение для  одного или обоих аргументов **базы** данных назначения и нового имени.

Если аргументы **типа исходных** объектов и имени исходных объектов будут пустыми, Access копирует объект, выбранный в области навигации.  Чтобы выбрать объект в области навигации, можно использовать действие **SelectObject** с аргументом В области навигации, за набором **да.**

Действие **CopyObject** аналогично выполнению следующих действий вручную:

1. Выберите объект в области навигации.

2. На вкладке Главная в группе Буфер обмена щелкните Копировать.

3. На той же вкладке щелкните **вклейку**. Диалоговое **окно Paste As** отображается таким образом, чтобы можно было дать объекту новое имя. Действие **CopyObject** выполняет все эти действия автоматически.

> [!NOTE]
> При копировании страниц доступа к данным действие **CopyObject** копирует только ссылку на связанный файл .htm, а не сам файл.

Путь и имя файла базы данных назначения должны существовать до того, как макрос запускает **действие CopyObject.** Если их не существует, Access отображает сообщение об ошибке.

Чтобы выполнить **действие CopyObject** в Visual Basic для приложений (VBA), используйте метод **CopyObject** объекта **DoCmd.**

Вы также можете вручную скопировать объект, выбранный в области навигации, или объект, открытый в настоящее время, щелкнув вкладку **Файл** и нажав кнопку **Сохранить Как**. Эта команда будет делать копию объекта только в текущей базе данных. В **диалоговом** окне Сохранить Как введите имя копии и выберите тип объекта, который необходимо сохранить. Если исходный объект уже сохранен и вы сохраните его в текущей базе данных с новым именем, оригинальная версия по-прежнему существует со старым именем.

Вручную скопировать объект в другую базу данных Access:

1. На **вкладке Внешние данные** в группе **Экспорт** нажмите кнопку **Больше,** а затем нажмите **кнопку Access Database**.

2. В диалоговом окне Экспорт **—** Доступ к базе данных введите имя  файла базы данных назначения.-или-Щелкните Обзор, чтобы отобразить диалоговое окно Сохранить файл, найдите базу данных назначения и нажмите кнопку **Сохранить**. 

3. В **диалоговом окне Экспорт — Доступ** к базе данных нажмите **кнопку ОК**. Отображается **диалоговое** окно Экспорт.

4. В **диалоговом** окне Экспорт введите имя объекта в базе данных назначения. Выберите все применимые параметры, такие как **экспортное определение и данные** или **определение только для** таблиц. По завершении нажмите кнопку **ОК**.
