# <a name="copyobject-macro-action"></a>Макрокоманда CopyObject

**Область применения**: Access 2013, Office 2013

С помощью действия **CopyObject** можно скопировать указанный объект базы данных в другую базу данных Access или в ту же базу данных или проект Access под новым именем. Например, можно скопировать или создать копию существующего объекта в другой базе данных или быстро создать похожий объект с несколькими изменениями.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Заметки

Для этого действия необходимо ввести значение как  для одного, так и для обоих аргументов **"База** данных назначения" и "Новое имя".

Если оставить **аргументы Source Object Type** и **Source Object Name** пустыми, Access копирует объект, выбранный в области навигации. Чтобы выбрать объект в области навигации, можно использовать действие **SelectObject** с аргументом "Да" в области **навигации.**

Действие **CopyObject** аналогично выполнению следующих действий вручную:

1. Выберите объект в области навигации.

2. На вкладке Главная в группе Буфер обмена щелкните Копировать.

3. На той же вкладке щелкните **"Ветвь".** **Появится** диалоговое окно "Ветвь как", в результате которого объекту можно дать новое имя. Действие **CopyObject** выполняет все эти действия автоматически.

> [!NOTE]
> При копировании страниц доступа к данным действие **CopyObject** копирует только ссылку на связанный HTM-файл, а не сам файл.

Путь и имя файла конечной базы данных должны существовать перед запуском макроса **действия CopyObject.** Если они не существуют, Access отображает сообщение об ошибке.

Чтобы запустить **действие CopyObject** в модуле Visual Basic для приложений (VBA), используйте метод **CopyObject** объекта **DoCmd.**

Вы также можете вручную скопировать объект, выбранный в области навигации, или  открытый объект, щелкнув вкладку "Файл" и выбрав "Сохранить **как".** Эта команда будет делать копию объекта только в текущей базе данных. В **диалоговом** окне "Сохранить как" введите имя копии и выберите тип объекта, который вы хотите сохранить. Если исходный объект уже сохранен и вы сохраняете его в текущей базе данных с новым именем, исходная версия по-прежнему существует со старым именем.

Чтобы вручную скопировать объект в другую базу данных Access:

1. На **вкладке "Внешние данные"**  в группе **"Экспорт"** щелкните "Еще" и выберите "База данных **Access".**

2. В диалоговом окне "Экспорт — база данных **Access"** введите имя  файла конечной базы данных.-или нажмите кнопку **"Обзор",** чтобы отобразить диалоговое окно "Сохранение файлов", найдите пунктов назначения базы данных и нажмите кнопку "Сохранить". 

3. В **диалоговом окне** "Экспорт — база данных Access" нажмите кнопку **"ОК".** **Появится** диалоговое окно "Экспорт".

4. В **диалоговом** окне "Экспорт" введите имя объекта в базе данных назначения. Выберите все применимые параметры, например "Экспорт **определения" и "Данные"** или **"Определение только для** таблиц". По завершении нажмите кнопку **ОК**.
