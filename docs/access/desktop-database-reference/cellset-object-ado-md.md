# <a name="cellset-object-ado-md"></a>Объект Cellset (ADO MD)

**Применимо к**: Access 2013, Office 2013

Представляет результаты запроса многомерных. Это коллекция ячеек, выбранные из кубов или других наборов ячеек.

## <a name="remarks"></a>Примечания

Данные в рамках набора **ячеек** извлекаются с помощью прямого доступа как массив. Вы можете «выполнить» к определенному элементу, для получения данных о этого элемента. Например следующий код возвращает заголовок первого элемента в начале на оси первого набора ячеек, с именем cst:

`cst.Axes(0).Positions(0).Members(0).Caption`

Нет отсутствует понятие текущей ячейки в рамках набора ячеек. Вместо этого свойство [Item](item-property-ado-md-cellset.md) получает объект определенные [ячейки](cell-object-ado-md.md) из ячеек. Аргументы свойство **Item** определение извлекается ячейки. Можно указать уникальный порядковый номер ячейки. Вы также можете получить ячейки с помощью их положение номера каждой оси ячеек. Дополнительные сведения о получении ячеек [элемента](item-property-ado-md-cellset.md) см.

С помощью семейства сайтов, методы и свойства объекта **ячеек** сделайте следующее:

  - Свяжите открытое подключение с объектом **ячеек** , задав его свойство [ActiveConnection](activeconnection-property-ado-md.md) .

  - Выполнение и получение результатов многомерные запроса с помощью метода [Open](open-method-ado-md.md) .

  - Получение **ячейки** из **ячеек** с помощью свойства [элемента](item-property-ado-md-cellset.md) .

  - Возвращает объекты [ось](axis-object-ado-md.md) , определяющие **ячеек** с коллекцией [осей](axes-collection-ado-md.md) .

  - Получение сведений о измерений, используемых для фильтрации данных в наборе **ячеек** с помощью свойства [FilterAxis](filteraxis-property-ado-md.md) .

  - Укажите запрос, используемый для определения **ячеек** с помощью свойства [источника](source-property-ado-md.md) или возврата.

  - Возвращает текущее состояние **ячеек** (открытым, закрытым, выполнение, или подключения) со свойством [состояния](state-property-ado-md.md) .

  - Закрытие открытых **ячеек** с помощью метода [Close](close-method-ado-md.md) .

  - Получение сведений от поставщика об **ячеек** в стандартный ADO [Свойства](properties-collection-ado.md) коллекции.
