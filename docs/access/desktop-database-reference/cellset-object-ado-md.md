# <a name="cellset-object-ado-md"></a>Объект набора ячеек (ADO MD)

**Область применения**: Access 2013, Office 2013

Представляет результаты многомерного запроса. Это коллекция ячеек, выбранных из кубов или других целлсетс.

## <a name="remarks"></a>Примечания

Данные в наборе **ячеек** извлекаются с помощью прямого, подобного массиву доступа. Можно "детализировать" до определенного члена, чтобы получить данные об этом члене. Например, следующий код возвращает подпись первого элемента в первой оси в наборе ячеек с именем CST:

`cst.Axes(0).Positions(0).Members(0).Caption`

В наборе ячеек отсутствует понятие текущей ячейки. Вместо этого свойство [Item](item-property-ado-md-cellset.md) получает определенный объект [Cell](cell-object-ado-md.md) из коллекции Cells. Аргументы свойства **Item** определяют, какая ячейка извлекается. Вы можете указать уникальное порядковое значение ячейки. Вы также можете извлекать ячейки, используя их номера позиции на каждой оси набора ячеек. Дополнительные сведения об извлечении ячеек содержатся в разделе свойство [Item](item-property-ado-md-cellset.md) .

С помощью коллекций, методов и свойств объекта набора **ячеек** можно выполнить следующие действия:

  - Свяжите открытое подключение с объектом **Cell** , задав его свойство [ActiveConnection](activeconnection-property-ado-md.md) .

  - Выполнение и получение результатов многомерного запроса с помощью метода [Open](open-method-ado-md.md) .

  - Получение **ячейки** из **коллекции с помощью** свойства [Item](item-property-ado-md-cellset.md) .

  - Возвращает объекты [осей](axis-object-ado-md.md) , которые определяют набор **ячеек** с помощью коллекции [осей](axes-collection-ado-md.md) .

  - Получение сведений об измерениях, используемых для фильтрации данных в наборе **ячеек** , с помощью свойства [FilterAxis](filteraxis-property-ado-md.md) .

  - Возвращает или задает запрос, используемый для определения набором **ячеек** с помощью свойства [Source](source-property-ado-md.md) .

  - Возвращает текущее состояние набора **ячеек** (открытие, закрытие, выполнение или подключение) со свойством [State](state-property-ado-md.md) .

  - Закройте открытый набор **ячеек** с помощью метода [Close](close-method-ado-md.md) .

  - Получение сведений о наборе **ячеек** с помощью стандартной коллекции [свойств](properties-collection-ado.md) ADO.
