# <a name="chapter-8-understanding-cursors-and-locks"></a>Глава 8. Курсоры и блокировки

**Область применения**: Access 2013, Office 2013

Важно понимать, как работают курсоры, чтобы выбрать оптимальный и эффективный тип курсора для требований к доступу к данным приложения. Менее оптимальная конфигурация курсора может значительно замедлить операции доступа к данным.

Многие возможности объекта ADO **Recordset** определяются типом и расположением курсора, а также типом блокировки.
