# <a name="documentdatecreated-property-dao"></a>Document.DateCreated Property (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает дату и время создания объекта. Только для чтения **Variant**.

## <a name="syntax"></a>Синтаксис

*выражение* . DateCreated

*выражение* Переменная, которая представляет собой объект- **документов** .
