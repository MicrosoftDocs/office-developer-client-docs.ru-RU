# <a name="querydefname-property-dao"></a>QueryDef.Name (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает или задает имя указанного объекта. Для чтения и записи, **String**.

## <a name="syntax"></a>Синтаксис

*выражения* . Имя

*выражение*: переменная, представляющая объект **QueryDef**.

## <a name="remarks"></a>Примечания

Максимальная длина имени объекта **QueryDef** — 64 символа.
