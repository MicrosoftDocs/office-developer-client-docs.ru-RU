# <a name="deleterule-property-adox"></a>Свойство DeleteRule (ADOX)


**Применимо к**: Access 2013, Office 2013

Указывает, действие выполняется при удалении первичный ключ.

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает и возвращает значение типа **Long** , может быть одной из констант [RuleEnum](ruleenum.md) . Значение по умолчанию — **adRINone**.

## <a name="remarks"></a>Замечания

Это свойство доступно только для чтения в объектах [ключ](key-object-adox.md) уже добавляется в конец коллекции.
