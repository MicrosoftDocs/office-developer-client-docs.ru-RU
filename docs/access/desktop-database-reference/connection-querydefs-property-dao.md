# <a name="connectionquerydefs-property-dao"></a>Connection.QueryDefs Property (DAO)


**Применимо к**: Access 2013 | Office 2013

Возвращает коллекцию **QueryDefs** , который содержит все объекты **QueryDef** указанного подключения. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . QueryDefs

*выражение* Переменная, которая содержит объект **подключения** .
