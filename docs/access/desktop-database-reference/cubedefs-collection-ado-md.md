# <a name="cubedefs-collection-ado-md"></a>Коллекция Коллекция cubedefs (ADO MD)


**Область применения**: Access 2013, Office 2013

Содержит объекты [CubeDef](cubedef-object-ado-md.md) , представляющие куб из многомерного каталога.

## <a name="remarks"></a>Замечания

**Коллекция cubedefs** — это стандартная коллекция ADO. С помощью свойств и методов коллекции можно выполнить следующие действия:

- Получите число объектов в коллекции со свойством [Count](count-property-ado.md) .

- Возвращает объект из коллекции со свойством [Item](item-property-ado.md) по умолчанию.

- Обновление объектов в коллекции от поставщика с помощью метода [Refresh](refresh-method-ado.md) .
