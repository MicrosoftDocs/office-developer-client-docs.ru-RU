# <a name="system-requirements-for-the-address-book-application"></a>Требования к системе для приложения адресной книги


**Область применения**: Access 2013, Office 2013

Чтобы настроить пример приложения адресной книги, необходимо выполнить следующие требования к программному обеспечению и базе данных:

**Требования к программному обеспечению**

Требования к программному обеспечению серверного компьютера для работы с этим веб-приложением включают:

  - Microsoft Windows NT Server 4.0 с Пакет обновления или более поздней Windows 2000 Server.

  - Microsoft IIS версии 3.0 или более поздней версии с ASP.

Требования к программному обеспечению для клиентского компьютера для работы с этим веб-приложением включают:

  - Microsoft Internet Explorer 4.0 или более поздний.

  - Microsoft Windows NT 4.0 workstation или Server, Windows 2000 или Microsoft Windows 98.

## <a name="database-requirements"></a>Требования к базе данных

Чтобы использовать этот пример, необходимо иметь:

  - Операционный Microsoft SQL Server версии 6.5 или более позднего сервера базы данных.

  - Привилегии для создания базы данных и ее заполнения примерными данными.

  - Проверка заполненных данных через Enterprise manager или утилиты ISQL (называемый анализатор запросов в SQL Server 7.0).

Если у вас нет привилегий, администратору базы данных может потребоваться настроить систему и предоставить вам разрешение на доступ к базе данных или настроить базу данных для вас.
