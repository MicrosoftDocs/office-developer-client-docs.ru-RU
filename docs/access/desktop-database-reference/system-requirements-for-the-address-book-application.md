# <a name="system-requirements-for-the-address-book-application"></a>Требования к системе для приложения адресной книги


**Область применения**: Access 2013, Office 2013

Чтобы настроить пример приложения адресной книги, необходимо выполнить следующие требования к программному обеспечению и базе данных:

**Требования к программному обеспечению**

Требования к программному обеспечению компьютера сервера для запуска этого веб-приложения включают:

  - Microsoft Windows NT Server 4.0 с Пакет обновления 3 или более поздней версии или Microsoft Windows 2000 Server.

  - Microsoft IIS (IIS) версии 3.0 или более поздней, с ASP.

Требования к программному обеспечению клиентского компьютера для запуска этого веб-приложения:

  - Microsoft Internet Explorer 4.0 или более поздней.

  - Microsoft Windows NT 4.0 Workstation или Server, Windows 2000 или Microsoft Windows 98.

## <a name="database-requirements"></a>Требования к базе данных

Чтобы использовать этот пример, необходимо:

  - Операционный Microsoft SQL Server версии 6.5 или более поздней.

  - Привилегии для создания базы данных и заполнения ее примерами данных.

  - Проверка заполненных данных через Enterprise Manager или программы ISQL (называемые анализатором запросов в SQL Server 7.0).

Если у вас нет привилегий, администратору базы данных может потребоваться настроить систему и предоставить доступ к базе данных или настроить базу данных за вас.
