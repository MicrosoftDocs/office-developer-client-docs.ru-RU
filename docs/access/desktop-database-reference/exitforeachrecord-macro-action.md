# <a name="exitforeachrecord-macro-action"></a>ExitForEachRecord Macro Action


**Применимо к**: Access 2013 | Office 2013

Действие **ExitForEachRecord** используется для немедленного выхода из блока **[ДляКаждойЗаписи](foreachrecord-data-block.md)** данных.


> [!NOTE]
> <P>Действие <STRONG>ExitForEachRecord</STRONG> доступна только в макросов данных.</P>
