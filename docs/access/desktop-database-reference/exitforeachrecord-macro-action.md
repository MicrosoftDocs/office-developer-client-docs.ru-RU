# <a name="exitforeachrecord-macro-action"></a>Макрокоманда ExitForEachRecord


**Область применения**: Access 2013, Office 2013

Чтобы немедленно выйти из блока данных **[ForEachRecord, используйте действие ExitForEachRecord.](foreachrecord-data-block.md)** 


> [!NOTE]
> Действие **ExitForEachRecord** доступно только в макросах данных.
