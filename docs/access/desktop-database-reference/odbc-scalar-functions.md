# <a name="odbc-scalar-functions"></a>Скалярные функции ODBC

**Область применения**: Access 2013, Office 2013

Microsoft Access SQL поддерживает использование определенного ODBC синтаксиса для скалярных функций. 

Например, запрос возвращает все строки, в которых абсолютное значение изменения цены акции больше `SELECT DAILYCLOSE, DAILYCHANGE FROM DAILYQUOTE WHERE {fn ABS(DAILYCHANGE)} > 5` пяти.

Поддерживается подмножество определенных скалярных функций ODBC. В следующей таблице перечислены поддерживаемые функции.

Описание аргументов и полное описание escape-синтаксиса для включаемой функции в инструкцию SQL см. в документации ODBC.
