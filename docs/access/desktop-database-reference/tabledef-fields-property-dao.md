# <a name="tabledeffields-property-dao"></a>Свойство TableDef.Fields (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает коллекцию **Fields**, которая представляет все объекты **Field** для указанного объекта. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение .* Поля

*выражение*: переменная, представляющая объект **TableDef**.
