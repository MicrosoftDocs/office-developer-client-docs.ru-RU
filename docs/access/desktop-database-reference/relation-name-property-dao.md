# <a name="relationname-property-dao"></a>Свойство Relation.Name (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает или задает имя указанного объекта. Чтение и запись **строки** Если объект не были добавлены в коллекцию. Только для чтения **строка** Если объект были добавлены в коллекцию.

## <a name="syntax"></a>Синтаксис

*выражение* . Имя

*выражение* Переменная, которая представляет собой объект- **связи** .

## <a name="remarks"></a>Примечания

Максимальная длина имени объекта **связи** — 64 символов.
