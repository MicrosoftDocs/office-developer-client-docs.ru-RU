# <a name="relationname-property-dao"></a>Relation.Name (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает или задает имя указанного объекта. Строка **read/write,** если объект не был придан коллекции. Строка **только для** чтения, если объект был придан коллекции.

## <a name="syntax"></a>Синтаксис

*выражения* . Имя

*выражение* Переменная, представляюная объект **Relation.**

## <a name="remarks"></a>Примечания

Максимальная длина имени объекта **Relation** — 64 символа.
