# <a name="clearmacroerror-macro-action"></a><span data-ttu-id="b9999-102">Макрокоманда ClearMacroError</span><span class="sxs-lookup"><span data-stu-id="b9999-102">ClearMacroError macro action</span></span>

<span data-ttu-id="b9999-104">С помощью действия **клеармакроеррор** можно очистить сведения об ошибке, которая хранится в объекте **MacroError** .</span><span class="sxs-lookup"><span data-stu-id="b9999-104">You can use the **ClearMacroError** action to clear information about an error that is stored in the **MacroError** object.</span></span>

## <a name="remarks"></a><span data-ttu-id="b9999-107">Примечания</span><span class="sxs-lookup"><span data-stu-id="b9999-107">Remarks</span></span>

- <span data-ttu-id="b9999-108">При возникновении ошибки в макросе сведения об ошибке хранятся в объекте **MacroError** .</span><span class="sxs-lookup"><span data-stu-id="b9999-108">When an error occurs in a macro, information about the error is stored in the **MacroError** object.</span></span> <span data-ttu-id="b9999-109">Если вы не использовали действие **[OnError](onerror-macro-action.md)** для подавления сообщений об ошибках, макрос перестает работать, а сведения об ошибке отображаются в стандартном сообщении об ошибке.</span><span class="sxs-lookup"><span data-stu-id="b9999-109">If you have not used the **[OnError](onerror-macro-action.md)** action to suppress error messages, the macro stops and the error information is displayed in a standard error message.</span></span> <span data-ttu-id="b9999-110">Тем не менее, если вы использовали действие **OnError** для подавления сообщений об ошибках, может потребоваться использовать сведения, хранящиеся в объекте **MacroError** в условии или в настраиваемом сообщении об ошибке.</span><span class="sxs-lookup"><span data-stu-id="b9999-110">However, if you have used the **OnError** action to suppress error messages, you might want to use the information stored in the **MacroError** object in a condition or in a custom error message.</span></span>
    
  <span data-ttu-id="b9999-111">После того как ошибка будет обработана, сведения в объекте **MacroError** устарели, поэтому рекомендуется очистить объект с помощью действия **клеармакроеррор** .</span><span class="sxs-lookup"><span data-stu-id="b9999-111">After an error has been handled, the information in the **MacroError** object is out of date, so it is a good idea to clear the object by using the **ClearMacroError** action.</span></span> <span data-ttu-id="b9999-112">При этом номер ошибки в объекте **MacroError** сбрасывается на 0 и удаляется любая другая информация об ошибке, которая хранится в объекте, например описание ошибки, имя макроса, имя действия, условие и аргументы.</span><span class="sxs-lookup"><span data-stu-id="b9999-112">Doing so resets the error number in the **MacroError** object to 0 and clears any other information about the error that is stored in the object, such as the error description, macro name, action name, condition, and arguments.</span></span> <span data-ttu-id="b9999-113">Таким образом, вы можете снова проверить объект **MacroError** , чтобы проверить, произошла ли другая ошибка.</span><span class="sxs-lookup"><span data-stu-id="b9999-113">This way, you can inspect the **MacroError** object again later to see if another error has occurred.</span></span>

- <span data-ttu-id="b9999-114">Объект **MacroError** автоматически очищается при завершении любого макроса, поэтому не нужно использовать действие **клеармакроеррор** в конце макроса.</span><span class="sxs-lookup"><span data-stu-id="b9999-114">The **MacroError** object is automatically cleared when any macro ends, so you do not need to use the **ClearMacroError** action at the end of a macro.</span></span>

- <span data-ttu-id="b9999-115">Объект **MacroError** содержит сведения только об одной ошибке за раз.</span><span class="sxs-lookup"><span data-stu-id="b9999-115">The **MacroError** object contains information about only one error at a time.</span></span> <span data-ttu-id="b9999-116">Если в макросе произошла более одной ошибки, объект **MacroError** содержит сведения только о последней ошибке.</span><span class="sxs-lookup"><span data-stu-id="b9999-116">If more than one error has occurred in a macro, the **MacroError** object contains information only about the last error.</span></span>

- <span data-ttu-id="b9999-117">Чтобы выполнить действие **клеармакроеррор** в модуле VBA, используйте метод **клеармакроеррор** объекта **DoCmd** .</span><span class="sxs-lookup"><span data-stu-id="b9999-117">To run the **ClearMacroError** action in a VBA module, use the **ClearMacroError** method of the **DoCmd** object.</span></span>
