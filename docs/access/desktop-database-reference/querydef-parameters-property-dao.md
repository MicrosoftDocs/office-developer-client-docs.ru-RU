# <a name="querydefparameters-property-dao"></a>QueryDef.Parameters Property (DAO)


**Применимо к**: Access 2013 | Office 2013

Возвращает коллекцию **[параметров](parameters-collection-dao.md)** , который содержит все объекты **[параметров](parameter-object-dao.md)** из указанного **QueryDef**. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . Параметры

*выражение* Переменная, которая представляет собой объект- **QueryDef** .
