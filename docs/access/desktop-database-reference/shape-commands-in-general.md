<span data-ttu-id="979bd-124">Команда Shape может содержать предложение, задающее команду запроса для базового поставщика данных, который будет возвращать объект **Recordset** .</span><span class="sxs-lookup"><span data-stu-id="979bd-124">A shape command may contain a clause specifying a query command to an underlying data provider that will return a **Recordset** object.</span></span> <span data-ttu-id="979bd-125">Синтаксис запроса зависит от требований базового поставщика данных.</span><span class="sxs-lookup"><span data-stu-id="979bd-125">The query's syntax depends on the requirements of the underlying data provider.</span></span> <span data-ttu-id="979bd-126">Как правило, это будет язык структурированных запросов (SQL), хотя для ADO не требуется использование какого-либо конкретного языка запросов.</span><span class="sxs-lookup"><span data-stu-id="979bd-126">This will usually be Structured Query Language (SQL), although ADO does not require the use of any particular query language.</span></span>

<span data-ttu-id="979bd-127">Вы можете использовать предложение JOIN SQL, чтобы связать две таблицы; Однако иерархический **набор записей** может представлять данные более эффективно.</span><span class="sxs-lookup"><span data-stu-id="979bd-127">You could use a SQL JOIN clause to relate two tables; however, a hierarchical **Recordset** may represent the information more efficiently.</span></span> <span data-ttu-id="979bd-128">Каждая строка **набора записей** , созданная при объединении, повторяет данные из одной из таблиц.</span><span class="sxs-lookup"><span data-stu-id="979bd-128">Each row of a **Recordset** created by a JOIN repeats information redundantly from one of the tables.</span></span> <span data-ttu-id="979bd-129">Иерархический **набор** записей имеет только один родительский **набор записей** для каждого из нескольких дочерних объектов **Recordset** .</span><span class="sxs-lookup"><span data-stu-id="979bd-129">A hierarchical **Recordset** has only one parent **Recordset** for each of multiple child **Recordset** objects.</span></span>
