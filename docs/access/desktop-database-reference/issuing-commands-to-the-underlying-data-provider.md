# <a name="issuing-commands-to-the-underlying-data-provider"></a>Отправка команд базовому поставщику данных

**Область применения**: Access 2013, Office 2013

Любая команда, которая не начинается с SHAPE, передается поставщику данных. Это эквивалентно выдаче команды фигуры в форме "SHAPE {provider command}". Эти команды *не должны* создавать **набор записей.** Например, "SHAPE {DROP TABLE MyTable} является вполне допустимой командой фигур, если поставщик данных поддерживает DROP TABLE.

Эта возможность позволяет как обычным командам поставщиков, так и командам формы совместно использовать одно и то же подключение и транзакцию.
