# <a name="getstring-method-ado"></a>Метод GetString (ADO)

**Область применения**: Access 2013, Office 2013

Возвращает объект [Recordset](recordset-object-ado.md) в виде строки.

## <a name="syntax"></a>Синтаксис

** = *Набор записей*Variant. GetString (*стрингформат*, *нумровс*, *колумнделимитер*, *ровделимитер*, *нуллекспр*)

## <a name="return-value"></a>Возвращаемое значение

Возвращает объект **Recordset** как **Variant** со строковым значением (BSTR).

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*Стрингформат* |Значение [стрингформатенум](stringformatenum.md) , задающее способ преобразования **набора записей** в строку. Параметры *ровделимитер*, *колумнделимитер*и *Нуллекспр* используются только в *стрингформат* **адклипстринг**.|
|*Нумровс* |Необязательный атрибут. Число строк, преобразуемых в **набор записей**. Если *нумровс* не указан или превышает общее количество строк в **наборе записей**, все строки в **наборе записей** преобразуются.|
|*Колумнделимитер* |Необязательный атрибут. Разделитель, используемый между столбцами, если он указан, в противном случае — символ ТАБУЛЯЦИи.|
|*Ровделимитер* |Необязательный атрибут. Разделитель, используемый между строками, если он указан, в противном случае — символ возврата КАРЕТки.|
|*Нуллекспр* |Необязательный атрибут. Выражение, используемое вместо значения NULL, если оно указано, в противном случае — пустая строка.|

## <a name="remarks"></a>Замечания

Данные строки, но не данные схемы, сохраняются в строке. Таким образом, невозможно повторно открыть объект **Recordset** с помощью этой строки.

Этот метод эквивалентен методу RDO **жетклипстринг** .
