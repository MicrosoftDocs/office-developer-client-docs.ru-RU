# <a name="unique-table-unique-schema-unique-catalog-dynamic-properties-ado"></a>Уникальная таблица, уникальная схема, динамические свойства каталога Unique (ADO)


**Область применения**: Access 2013, Office 2013

Позволяет тесно контролировать изменения в определенной базовой таблице в [наборе записей](recordset-object-ado.md) , созданном ОПЕРАЦИей JOIN с несколькими базовыми таблицами.

  - **Уникальная таблица** указывает имя базовой таблицы, для которой разрешены обновления, вставки и удаления.

  - **Уникальная схема** указывает *схему*или имя владельца таблицы.

  - **Уникальный каталог** указывает *Каталог*или имя базы данных, содержащей таблицу.

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает или возвращает **строковое** значение, которое представляет собой имя таблицы, схемы или каталога.

## <a name="remarks"></a>Примечания

Нужная базовая таблица уникально идентифицируется по имени каталога, схемы и таблиц. Если задано свойство **UNIQUE Table** , то для поиска базовой таблицы используются значения свойств **UNIQUE Schema** или **UNIQUE Catalog** . Он предназначен (но не обязательно), который задается как для уникальной **схемы** , так и для **уникальных свойств каталога** , прежде чем будет задано свойство **UNIQUE Table** .

Первичный ключ **уникальной таблицы** рассматривается как первичный ключ всего **набора записей**. Это ключ, используемый для любого метода, для которого требуется первичный ключ.

Если задана **уникальная таблица** , метод [Delete](delete-method-ado-recordset.md) влияет только на именованную таблицу. Методы [AddNew](addnew-method-ado.md), [Resync](resync-method-ado.md), [Update](update-method-ado.md)и [UpdateBatch](updatebatch-method-ado.md) влияют на соответствующие базовые таблицы **набора записей**.

Перед выполнением любых пользовательских повторных синхронизаций необходимо указать **уникальную таблицу** . Если не указана **уникальная таблица** , свойство [Command Resync](resync-command-property-dynamic-ado.md) не будет иметь никакого действия.

Если не удается найти уникальную базовую таблицу, возникает ошибка времени выполнения.

Эти динамические свойства добавляются в коллекцию [свойств](properties-collection-ado.md) объекта **Recordset** , если для свойства [CursorLocation](cursorlocation-property-ado.md) задано значение **адусеклиент**.
