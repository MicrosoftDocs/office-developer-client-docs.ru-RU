# <a name="appendix-d-ado-samples"></a>Приложение Г. Примеры ADO

**Область применения**: Access 2013, Office 2013

В Windows SDK содержится несколько примерных приложений, которые демонстрируют использование кода ADO и RDS. Эти примеры предназначены для того, чтобы помочь вам узнать, как использовать ADO и RDS, но не предназначены для развертывания на производственных серверах.

> [!NOTE]
> Компании, имена и данные, используемые в примерах, являются фиктивными, если иное не указано.

Чтобы найти образцы ADO, перейдите в корневую папку Windows SDK, например C: Program \\ Files \\ Microsoft SDKs \\ Windows \\ v6.0. Затем перейдите в подмостки \\ Samples dataaccess \\ ADO. Чтобы начать работу с примерами приложений ADO и RDS, откройте default.htm из папок образцов ADO и RDS соответственно. Эти файлы описывают каждое доступное примеровое приложение и объясняют, как настроить и запустить примеры, как просматривать исходный код образца.

Документация ADO также содержит примеры кода на нескольких языках, которые включены в качестве тем в этой онлайн-справке. Дополнительные сведения об этих примерах см. в [примере ADO Code Examples.](ado-code-examples.md)
