# <a name="appendix-d-ado-samples"></a>Приложение Г. Примеры ADO

**Область применения**: Access 2013, Office 2013

Пакет SDK для Windows содержит несколько примеров приложений, демонстрирующих использование ADO и кода RDS. Эти примеры предназначены для того, чтобы научиться использовать ADO и RDS, но они не предназначены для развертывания на рабочих серверах.

> [!NOTE]
> Компании, имена и данные, используемые в примерах, являются фиктивными, если не указано иное.

Чтобы найти\\примеры ADO, перейдите к корневой папке пакета Windows SDK, например C: Program Files\\Microsoft SDK\\Windows\\версии 6.0. Затем перейдите к подпапке\\ADO Access\\Samples. Чтобы начать работу с примерами приложений ADO и RDS, откройте Default. htm в папках примеров ADO и RDS соответственно. В этих файлах описываются все доступные примеры приложений, а также объясняется, как настраивать и запускать примеры, как просмотреть исходный код примера.

В документации по ADO также содержатся примеры кода на нескольких языках, которые включены в качестве разделов в этой интерактивной справке. Дополнительные сведения об этих примерах приведены в статье [примеры кода ADO](ado-code-examples.md).
