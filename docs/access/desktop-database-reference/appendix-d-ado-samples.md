# <a name="appendix-d-ado-samples"></a>Приложение Г. Примеры ADO

**Область применения**: Access 2013, Office 2013

Windows SDK содержит несколько примеров приложений, демонстрируя использование кода ADO и RDS. Эти примеры призваны помочь вам узнать, как использовать ADO и RDS, но не предназначены для развертывания на производственных серверах.

> [!NOTE]
> Компании, имена и данные, используемые в примерах, являются вымышленными, если не указано иное.

Чтобы найти примеры ADO, перейдите в корневую папку Windows SDK, например C: \\ Program Files \\ Microsoft SDKs \\ Windows \\ 6.0. Затем перейдите в подпапку \\ Samples dataaccess \\ ADO. To get started with the ADO and RDS sample applications, open default.htm from the ADO and RDS samples folders, respectively. Эти файлы описывают все доступные примеры приложений и поясняют, как настраивать и запускать примеры, как просматривать пример исходный код.

Документация по ADO также содержит примеры кода на нескольких языках, которые включены в качестве разделов в этой справке в Интернете. Дополнительные сведения об этих примерах см. в примерах кода [ADO.](ado-code-examples.md)
