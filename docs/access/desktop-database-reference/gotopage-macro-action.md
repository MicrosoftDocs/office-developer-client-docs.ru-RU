# <a name="gotopage-macro-action"></a>Макрокоманда GoToPage

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **GoToPage** для перемещения фокуса активной формы на первый элемент управления на указанной странице. Вы можете использовать это действие, если вы создали форму с разрывами страниц, которые содержат группы связанных сведений. Например, у вас может быть форма Employees с личными данными на одной странице, сведения о Office на другой странице и информация о продажах на третьей странице. Чтобы переместиться на нужную страницу, можно использовать действие **GoToPage** . Вы также можете представить несколько страниц информации в одной форме, используя элементы управления Tab.

## <a name="remarks"></a>Примечания

С помощью этого действия можно выбрать первый элемент управления (как определено в последовательности табуляции формы) на указанной странице. Используйте макрокоманду **КЭлементуУправления (КЭлементуУправления** ), чтобы перейти к определенному элементу управления в форме.

Вы можете использовать аргументы **right** и **Down** для форм с страницами больше, чем окно Access. Используйте аргумент **номер страницы** для перемещения на нужную страницу, а затем используйте аргументы **right** и **Down** для отображения части страницы, которую нужно просмотреть. Access отображает часть страницы, верхний левый угол которой смещается на заданное расстояние от верхнего левого угла страницы.

Вы не можете использовать действие **GoToPage** в следующих случаях:

- Перемещение фокуса на страницу скрытой формы.

- Для перемещения фокуса с одной страницы на другую в пределах элемента управления "Вкладка".

Чтобы запустить действие **GoToPage** в модуле Visual Basic для приложений (VBA), используйте метод **GoToPage** объекта **DoCmd** .
