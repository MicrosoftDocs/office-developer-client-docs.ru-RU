# <a name="gotopage-macro-action"></a>Макрокоманда GoToPage

**Область применения**: Access 2013, Office 2013

Вы можете использовать **действие GoToPage,** чтобы переместить фокус в активной форме на первый контроль на указанной странице. Вы можете использовать это действие, если вы создали форму с разрывами страниц, которая содержит группы связанных сведений. Например, на одной странице может быть форма Сотрудников с персональными данными, сведения об офисе на другой странице и сведения о продажах на третьей странице. Вы можете использовать **действие GoToPage** для перемещения на нужную страницу. Вы также можете представить несколько страниц информации в одной форме с помощью элементов управления вкладками.

## <a name="remarks"></a>Примечания

Это действие можно использовать для выбора первого управления (как определено порядком вкладки формы) на указанной странице. Чтобы перейти к определенному контролю формы, используйте действие **GoToControl.**

Аргументы **"Право** и **вниз"** можно использовать для форм со страницами, размером больше, чем окно Access. Используйте аргумент **Номер страницы,** чтобы перейти на нужную  страницу, а затем используйте аргументы **"Право"** и "Вниз", чтобы отобразить нужную часть страницы. Доступ отображает часть страницы, верхний левый угол которой смещен заданным расстоянием от верхнего левого угла страницы.

Действие **GoToPage** нельзя использовать в следующих случаях:

- Перемещение фокуса на страницу в скрытой форме.

- Перемещение фокуса с одной страницы на другую в области управления вкладками.

Чтобы выполнить **действие GoToPage** в Visual Basic для приложений (VBA), используйте метод **GoToPage** объекта **DoCmd.**
