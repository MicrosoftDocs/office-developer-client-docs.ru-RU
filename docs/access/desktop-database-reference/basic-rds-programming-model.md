- <span data-ttu-id="a44dc-119">Любые изменения объекта **Recordset** отправляются обратно в серверную программу, которая использует их для обновления источника данных.</span><span class="sxs-lookup"><span data-stu-id="a44dc-119">Any modifications to the **Recordset** object are sent back to the server program, which uses them to update the data source.</span></span>
