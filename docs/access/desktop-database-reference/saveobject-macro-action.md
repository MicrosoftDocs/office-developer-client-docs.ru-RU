# <a name="saveobject-macro-action"></a>Макрокоманда SaveObject

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **савеобжект** для сохранения указанного объекта Access или активного объекта, если он не указан. В некоторых случаях можно также сохранить активный объект с новым именем (Эта функция аналогична команде **Сохранить как** на **панели быстрого доступа**).

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Замечания

Действие **савеобжект** работает со всеми объектами базы данных, которые пользователь может открывать и сохранять в явном виде. Указанный объект должен быть открыт для того, чтобы действие **савеобжект** действовало на объект. Это действие эквивалентно выбору объекта и его сохранению с помощью кнопки **сохранить** на **панели быстрого доступа**. 

Если оставить аргумент " **тип объекта** " пустым и ввести новое имя в аргументе **имя объекта** , то то же самое делает то же самое, что и нажатие кнопки **Сохранить как** на **панели быстрого доступа**и ввода нового имени для активного объекта. С помощью действия **савеобжект** можно указать объект, который необходимо сохранить, и выполнить команду **Сохранить как** в макросе.

> [!NOTE]
> Вы не можете использовать действие **савеобжект** для сохранения следующего имени с новым именем:
> - Форма в представлении формы или таблицы
> - Отчет в режиме предварительного просмотра
> - Модуль
> - Представление "сервер" в режиме таблицы или предварительный просмотр
> - Страница доступа к данным в режиме страницы
> - Таблица в режиме таблицы или в режиме предварительного просмотра
> - Запрос в режиме таблицы или предварительный просмотр
> - Хранимая процедура в представлении таблицы или предварительный просмотр

Действие **савеобжект** , выполняемое при запуске макроса в текущей базе данных или в базе данных библиотеки, всегда сохраняет указанный объект или активный объект в базе данных, в которой был создан объект.

Если вы сохраните активный объект с новым именем, но это имя совпадает с именем существующего объекта этого типа, появится диалоговое окно с запросом на перезапись существующего объекта. Если вы установили для **** аргумента **Сетварнингс** действия " **нет**", диалоговое окно не отображается, а старый объект автоматически перезаписывается.

Чтобы запустить действие **савеобжект** в модуле Visual Basic для приложений (VBA), используйте метод **Save** объекта **DoCmd** .
