# <a name="saveobject-macro-action"></a>SaveObject Macro Action


**Применимо к**: Access 2013 | Office 2013

Действие **SaveObject** можно использовать для сохранения указанный объект доступа или активного объекта, если он не задан. Можно также сохранить активного объекта под новым именем, в некоторых случаях (это работает так же, как команды **Сохранить как** в панели **Быстрого доступа**).


> [!NOTE]
> <P>
		Эта действие не разрешено, если база данных не является доверенной. Дополнительные сведения о включении макросов см. по ссылкам в разделе See Also этой статьи.
</P>

## <a name="remarks"></a>Замечания

Действие **SaveObject** работает на все объекты базы данных, которые пользователь может явно открывать и сохранять. Указанный объект должен быть открыт для выполнения действия **SaveObject** оказывает никакого влияния на объекте. Это действие имеет тот же эффект, как при выборе объекта и сохранения его, нажав кнопку **Сохранить** на панели **Быстрого доступа**. Оставить пустым аргумент **Тип объекта** и введите новое имя в качестве аргумента **Имя объекта** имеет тот же эффект, что и нажмите кнопку **Сохранить как** на панели **Быстрого доступа**и введите новое имя для активного объекта. Действие **SaveObject** позволяет указать объект для сохранения и выполнения команды **Сохранить как** из макроса.


> [!NOTE]
> <P>Чтобы сохранить любую из следующих действий с новым именем нельзя использовать действие <STRONG>SaveObject</STRONG> :</P>



  - Форма в режиме формы или таблицы.

  - Отчет в режиме предварительного просмотра.

  - Модуль.

  - Представление в режиме или режиме предварительного просмотра.

  - Страницы в представлении Страница доступа к данным.

  - Таблица в режиме таблицы или в режиме предварительного просмотра.

  - Запрос в режиме таблицы или в режиме предварительного просмотра.

  - Хранимую процедуру в режиме таблицы или в режиме предварительного просмотра.

**SaveObject** действия, выполняемые в макросе, в текущей базе данных или в базе данных библиотеки, всегда сохраняет ли указанный объект или активный объект в базе данных, в котором был создан объект.

Сохранение активного объекта под новым именем, но имя совпадает с именем существующего объекта этого типа, диалоговое окно с запросом на перезапись существующего объекта. Если выбрать **В** аргументе действие **УстановитьСообщения** **Нет**не отображается диалоговое окно и старый объект автоматически перезаписан.

Чтобы выполнить действие **SaveObject** в Visual Basic для приложений (VBA) модуль, используйте метод **сохранения** **объекта** .
