# <a name="runsql-macro-action"></a>Макрокоманда RunSQL

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **рунскл** для выполнения запроса на изменение с помощью соответствующего оператора SQL. Кроме того, можно выполнить запрос определения данных.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных. 

## <a name="setting"></a>Параметр

Макрокоманда **рунскл** имеет следующие аргументы.

<tr class="odd">
<td><p><strong>Оператор SQL</strong></p></td>
<td><p>Инструкция SQL для запроса на изменение или запроса на определение данных, которые необходимо выполнить. Максимальная длина этого оператора составляет 255 символов. Это обязательный аргумент.</p></td>

## <a name="remarks"></a>Примечания

Запросы на изменение можно использовать для добавления, удаления и обновления записей, а также для сохранения набора результатов запроса в виде новой таблицы. С помощью запросов на определение данных можно создавать, изменять и удалять таблицы, а также создавать и удалять индексы. Вы можете использовать действие **рунскл** для выполнения этих операций непосредственно из макроса без необходимости использовать сохраненные запросы.

Если необходимо ввести инструкцию SQL длиннее 255 символов, используйте метод **рунскл** объекта **DoCmd** в модуле Visual Basic для приложений (VBA). В VBA можно вводить инструкции SQL длиной до 32 768 символов.
