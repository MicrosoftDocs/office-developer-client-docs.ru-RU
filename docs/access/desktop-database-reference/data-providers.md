# <a name="data-providers"></a>Поставщики данных


**Область применения**: Access 2013, Office 2013

Поставщики данных представляют различные источники данных, такие как SQL базы данных, индексные-последовательное файлы, электронные таблицы, хранилища документов и почтовые файлы. Поставщики подвергают данные равномерно, используя общую абстракцию под названием rowset.

ADO является мощным и гибким, поскольку он может подключаться к любому из нескольких различных поставщиков данных и по-прежнему выставлять ту же модель программирования, независимо от конкретных функций того или иного поставщика. Однако, поскольку каждый поставщик данных уникален, взаимодействие приложения с ADO зависит от поставщика данных.

Например, возможности и возможности поставщика OLE DB для SQL Server, который используется для доступа к базам данных Microsoft SQL Server, значительно отличаются от возможностей поставщика DB Microsoft OLE для интернет-публикации, который используется для доступа к хранилищам файлов на веб-сервере.
