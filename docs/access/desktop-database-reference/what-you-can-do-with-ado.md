# <a name="what-you-can-do-with-ado"></a>Что можно делать с помощью ADO


**Область применения**: Access 2013, Office 2013

ADO предоставляет разработчикам мощную логическую объектную модель для программного доступа, редактирования и обновления различных источников данных с помощью системных интерфейсов OLE DB. Чаще всего ADO запрашивает таблицы или таблицы в реляционной базе данных, извлекает и отображает результаты в приложении, а также, возможно, позволяет пользователям вносить и сохранять изменения в данных. Кроме того, с помощью ADO можно программным путем:

- Запрос базы данных с помощью SQL и отображение результатов.

- Доступ к сведениям в хранилище файлов через Интернет.

- Управление сообщениями и папками в почтовой системе.

- Сохранение данных из базы данных в XML-файл.

- Позволяет пользователю просмотреть и внести изменения в данные в таблицах баз данных.

- Создание и повторное повторное создание параметровизованных команд базы данных.

- Выполнение хранимой процедуры.

- Динамическое создание гибкой структуры, называемой **набором записей,** для хранения, навигации и управления данными.

- Выполнение транзакционной операции с базой данных.

- Фильтрация и сортировка локальных копий данных базы данных на основе критериев времени работы.

- Создание и управление иерархическими результатами из баз данных.

- Привязка полей базы данных к компонентам с данными.

- Создание удаленных, отключенных **записей.**

ADO должен предоставлять широкий выбор параметров и параметров, чтобы обеспечить такую гибкость. Поэтому важно использовать методический подход к использованию ADO в приложении, разбивая каждую из ваших целей на управляемые части.

В большинстве программ ADO участвуют четыре основные операции: получение данных, изучение данных, изменение данных и обновление данных. В следующих четырех главах каждая из этих операций рассматривается более подробно.

Прежде чем перезапываться, ознакомьтесь с объектами в объектной модели ADO. Затем [просмотрите HelloData: простое приложение ADO.](hellodata-a-simple-ado-application.md) Это приложение написано на Visual Basic и выполняет каждую из четырех основных операций ADO.
