# <a name="what-you-can-do-with-ado"></a>Что можно сделать с помощью ADO


**Применимо к**: Access 2013, Office 2013

ADO предназначен для предоставления разработчикам мощные, логических объектной модели для программного доступа к, изменение и обновление широкий спектр источники данных через интерфейсы OLE DB системы. Наиболее распространенные использования ADO является запрос таблицы или таблицы в реляционной базе данных, извлечения и отображения результатов в приложении и возможно разрешить пользователям одновременное внесение и сохранение изменений в данные. Другие действия, которые выполняются программным путем с помощью ADO включают:

- Запросы к базе данных с помощью SQL и отображения результатов.

- Доступ к данным в хранилище файлов через Интернет.

- Управление файлами сообщений и папок в системе электронной почты.

- Сохранение данных из базы данных в XML-файл.

- Позволяет пользователю для просмотра и изменения данных в таблицах базы данных.

- Создание и повторное использование параметризованные команды базы данных.

- Выполнение хранимых процедур.

- Динамическое создание гибкие структуры, называется набора **записей**, для хранения, перейдите и работы с данными.

- Для выполнения операций транзакций базы данных.

- Фильтрация и сортировка локальные копии сведения о базе данных на основе критериев во время выполнения.

- Создание и управление ими иерархических результаты из базы данных.

- Привязка полей базы данных для данных компонентов.

- Создание удаленных, прервана **наборов записей**.

ADO должен предоставлять широкий набор параметров и настроек, чтобы обеспечить такую гибкость. Поэтому важно выполнить тщательно, чтобы научиться использовать ADO в приложении, неудачного завершения каждого из выполнения задач на контролируемый части.

Четыре основных операций участвуют в большинстве программ ADO: получение данных, анализ данных, изменение данных и обновление данных. Следующие четыре главы проверьте каждой из этих операций, более подробно.

Прежде чем продолжить, ознакомьтесь с объектами в объектной модели ADO. Просмотрите [HelloData: простое приложение ADO](hellodata-a-simple-ado-application.md). В этом приложении написаны в Visual Basic и выполняет каждого из четырех основных операций ADO.
