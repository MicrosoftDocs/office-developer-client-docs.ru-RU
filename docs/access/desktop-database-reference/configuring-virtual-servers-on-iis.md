# <a name="configuring-virtual-servers-on-iis"></a>Настройка виртуальных серверов в IIS

**Область применения**: Access 2013, Office 2013

При создании виртуальных серверов в службы IIS 4.0 необходимы следующие два дополнительных шага, чтобы настроить виртуальный сервер для работы с RDS:

1.  При настройке сервера проверьте "Разрешить выполнение доступа".

2.  Переместите msadcs.dll *в vroot* \\ msadc, где *vroot* является домашним каталогом виртуального сервера.
