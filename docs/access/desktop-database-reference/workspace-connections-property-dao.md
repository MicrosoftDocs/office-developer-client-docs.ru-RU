# <a name="workspaceconnections-property-dao"></a>Свойство Workspace.Connections (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает коллекцию **Подключений,** которая представляет текущие подключения в указанном рабочем **пространстве.** Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражения* . Подключения

*expression*: переменная, представляющая объект **Workspace**.
