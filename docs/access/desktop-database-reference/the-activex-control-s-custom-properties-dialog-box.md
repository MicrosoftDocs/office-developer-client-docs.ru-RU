# <a name="activex-control-custom-properties-dialog-box"></a>Диалоговое окно настраиваемых свойств элемента ActiveX

**Область применения**: Access 2013, Office 2013

При задании свойств элемента управления ActiveX может потребоваться или предпочтительнее использовать диалоговое окно Настраиваемые свойства элемента управления. Это диалоговое окно настраиваемых свойств предоставляет альтернативу списку свойств на странице свойств Microsoft Access для настройки свойств элемента управления ActiveX в представлении конструктора.

> [!NOTE]
> Эти сведения относятся только к элементам управления ActiveX в среде базы данных Microsoft Access.

## <a name="two-ways-to-set-properties"></a>Два способа задания свойств

Причина для диалогового окна настраиваемые свойства состоит в том, что не все приложения, использующие элементы управления ActiveX, предоставляют страницу свойств, такую как Microsoft Access. Диалоговое окно Настраиваемые свойства предоставляет интерфейс для настройки свойств элемента управления независимо от интерфейса, предоставляемого ведущим приложением.

Для некоторых свойств элементов управления ActiveX можно выбрать одно из этих двух расположений, чтобы задать свойство:

- Страница свойств Microsoft Access.
- Диалоговое окно "настраиваемые свойства" элемента управления ActiveX.

В некоторых случаях диалоговое окно "настраиваемые свойства" — единственный способ задать свойство в представлении конструктора. Обычно это ситуация, когда интерфейс, необходимый для установки свойства, не работает в окне свойств Microsoft Access. Например, свойство **гридфонт** элемента управления Calendar имеет ряд аргументов; в каждом свойстве на странице свойств Microsoft Access нельзя установить более одного аргумента для каждого свойства.

## <a name="finding-the-custom-properties-dialog-box"></a>Поиск диалогового окна "настраиваемые свойства"

Не все элементы управления ActiveX предоставляют диалоговое окно настраиваемых свойств. Чтобы узнать, предоставляет ли элемент управления диалоговое окно настраиваемых свойств, найдите **настраиваемое** свойство в окне свойств Microsoft Access для этого элемента управления. Если список свойств содержит имя **Custom**, элемент управления предоставляет диалоговое окно Настраиваемые свойства.

## <a name="using-the-custom-properties-dialog-box"></a>Использование диалогового окна "настраиваемые свойства"

После выбора настраиваемого **** поля свойства на странице свойств Microsoft Access нажмите кнопку **построить** справа от поля свойство, чтобы открыть диалоговое окно Настраиваемые свойства элемента управления, которое часто представляется как диалоговое окно с вкладками. Перейдите на вкладку, содержащую интерфейс для настройки свойств, которые требуется задать.

После внесения изменений на одной вкладке часто можно применить эти изменения сразу, нажав кнопку **Применить** (если она указана). При необходимости можно выбрать другие вкладки, чтобы задать другие свойства. Чтобы утвердить все изменения, внесенные в диалоговом окне Настраиваемые свойства, нажмите кнопку **ОК** . Чтобы вернуться на страницу свойств Microsoft Access, не изменяя параметры свойств, нажмите кнопку **Отмена** .

Вы также можете просмотреть диалоговое окно Настраиваемые свойства, выбрав подкоманду свойства для команды **Свойства** элемента ActiveX **** (например, **объект элемента управления Calendar**) в меню **Правка** или выбрав эту же подкоманду на контекстное меню для элемента управления ActiveX. Кроме того, некоторые свойства на странице свойств Microsoft Access элемента управления ActiveX, например свойство **гридфонтколор** элемента управления Calendar, имеют кнопку **построения** справа от поля свойства. При нажатии кнопки **построить** отображается диалоговое окно Настраиваемые свойства, в котором выбрана соответствующая вкладка (например, **цвета**).
