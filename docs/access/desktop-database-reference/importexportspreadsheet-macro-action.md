# <a name="importexportspreadsheet-macro-action"></a>ImportExportSpreadsheet Macro Action


**Применимо к**: Access 2013 | Office 2013

Действие **ImportExportSpreadsheet** можно использовать для импорта или экспорта данных между текущей базы данных Access (MDB- или .accdb) или проект Microsoft Access (файлы с расширением ADP) и файла электронной таблицы. Также можно связать данные в электронную таблицу Microsoft Excel для текущей базы данных Microsoft Access. Связывания электронной таблицы можно просматривать и изменять данные электронной таблицы с доступом по-прежнему предоставляя полный доступ к данным из программы электронной таблицы Excel. Также можно связать с данным в файл электронной таблицы Lotus 1-2-3, но эти данные в доступа только для чтения.


> [!NOTE]
> <P>
		Эта действие не разрешено, если база данных не является доверенной. Дополнительные сведения о включении макросов см. по ссылкам в разделе See Also этой статьи.
</P>

## <a name="remarks"></a>Замечания

Можно экспортировать данные в запросах доступа электронных таблиц. Microsoft Access экспортирует набор результатов запроса в виде таблицы.

Данные электронной таблицы, добавляемые в существующую таблицу Access должны быть совместимы со структурой таблицы.

  - Каждое поле в таблице должен быть типу данных, то соответствующее поле в таблице.

  - Поля должны быть в том же порядке (Если аргумент **Содержит имена полей** задано значение **Да**, в этом случае поле имена в электронной таблице должны совпадать имена полей в таблице).

Это действие аналогично выбрав вкладку **Внешних данных** и и выбрав пункты **Excel** в **Импорт** или **Экспорт** группу или **несколько** в группе **Импорт** или **Экспорт** и щелкнув **Файл Lotus 1-2-3**. Эти команды можно использовать для выбора источника данных, например Access или тип базы данных, электронную таблицу или текстовый файл. При выборе электронной таблице отображается ряд диалоговых окон или запускается мастер Microsoft Access, в котором выберите имя электронной таблицы и другие параметры. Аргументы действие **ImportExportSpreadsheet** соответствуют параметрам этих диалоговых окон или мастера.


> [!NOTE]
> <P>Запроса и фильтрации связанной электронной таблице запроса или фильтра с учетом регистра.</P>



При связывании таблицы Excel, которая открыта в режиме редактирования Access будет ждать в электронной таблице Excel — режим редактирования перед завершением ссылок; нет тайм-аут.

Чтобы выполнить действие **ImportExportSpreadsheet** в Visual Basic для приложений (VBA) модуль, используйте метод **ПреобразоватьЭлектроннуюТаблицу** **объекта** .
