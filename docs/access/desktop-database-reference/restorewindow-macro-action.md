# <a name="restorewindow-macro-action"></a>Макрокоманда RestoreWindow

**Область применения**: Access 2013, Office 2013

Вы можете использовать действие **ресторевиндов** для восстановления предыдущего размера развернутого или свернутого окна.

> [!NOTE]
> Это действие невозможно применить к окнам кода в редакторе Visual Basic. Сведения о том, как повлиять на окна кода, представлены в разделе свойство **WindowState** .

## <a name="setting"></a>Параметр

Макрокоманда **ресторевиндов** не содержит аргументов.

## <a name="remarks"></a>Замечания

Это действие выполняется для выбранного объекта. Если объект свернут, его можно сначала выбрать с помощью макрокоманды **ВыделитьОбъект (SelectObject** ), а затем восстановить до его предыдущего размера с помощью действия **ресторевиндов** .

Вы можете использовать действие **мовеандсизевиндов** для перемещения или изменения размера восстановленного окна.

Действие **ресторевиндов** имеет тот же результат, что и нажатие кнопки " **восстановить** " в правом верхнем углу окна или нажатие команды " **восстановить** " в оконном меню окна. ****

Чтобы запустить действие **ресторевиндов** в модуле Visual Basic для приложений (VBA), используйте метод Restore **** объекта DoCmd. ****
