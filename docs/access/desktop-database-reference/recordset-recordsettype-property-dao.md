# <a name="recordsetrecordsettype-property-dao"></a>Recordset.RecordsetType Property (DAO)

**Применимо к**: Access 2013, Office 2013

Свойство **Тип набора записей** можно использовать для указания типа набора записей можно сделать доступной для формы. Чтение и запись **байтов**.

## <a name="syntax"></a>Синтаксис

*выражение* . Тип набора записей

*выражение* Переменная, которая представляет собой объект- **форму** .

> [!NOTE]
> Изменение свойства **Тип набора записей** открытая форма или отчет вызывает автоматическое воссоздание набора записей.



Можно создавать формы, основанные на нескольких таблицах с помощью полей, привязанных к элементам управления в формах. В зависимости от настройки свойства **Тип набора записей** , можно ограничить, какой из этих связанных элементов управления можно изменить их невозможно.

Помимо управления редактирования предоставлено **Тип набора записей**каждого элемента управления в форме имеет свойство **Locked** , можно задать для определения, является ли элемент управления и базовые данные можно изменить. Если свойство **Locked** задано значение Да, не могут изменять данные.

## <a name="example"></a>Пример

В следующем примере только в том случае, если пользователю присваивается идентификатор АДМИНИСТРАТОР может записи обновляться. Этот пример кода задает свойство **Тип набора записей** к снимку Если значение общей переменной gstrUserID не администратор.

```vb
    Sub Form_Open(Cancel As Integer) 
     Const conSnapshot = 2 
     If gstrUserID <> "ADMIN" Then 
     Forms!Employees.RecordsetType = conSnapshot 
     End If 
    End Sub
```
