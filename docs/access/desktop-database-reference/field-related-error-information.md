# <a name="field-related-error-information"></a>Сведения об ошибках, связанных с полями


**Область применения**: Access 2013, Office 2013

Если сообщение об ошибке напрямую связано с полем (например, если данные отсутствуют или для поля задан недопустимый тип), можно получить дополнительные сведения о причине проблемы, изучив свойство **Status** объекта **field** . Это свойство было расширено для предоставления определенных сведений о проблеме. Например, если при вызове **UpdateBatch** возникает ошибка, причина проблемы может быть определена путем проверки свойства **Status** **полей** в каждой из действующих записей.
