# <a name="beep-macro-action"></a>Макрокоманда Beep


**Применимо к**: Access 2013, Office 2013

Действие **Звуковые** можно использовать для получения сигнала звуковой сигнал через динамик компьютера.

## <a name="setting"></a>Параметр

Действие **звуковые сигналы** не требует аргументов.

## <a name="remarks"></a>Примечания

Можно использовать действие **Beep** для сигнала в следующих случаях:

  - На экране изменений.

  - Неверный тип данных был введен в элементе управления. Например пользователь ввел числовые данные в элемент управления текстового поля.

  - Макрос достиг указанной точки или завершился.

Частота и продолжительность сигнала зависеть от оборудования, могут отличаться от указанных между компьютерами.

Чтобы выполнить действие **звуковые сигналы** в Visual Basic для приложений (VBA) модуль, используйте метод **звуковые сигналы** **объекта** .
