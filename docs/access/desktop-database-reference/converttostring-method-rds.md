# <a name="converttostring-method-rds"></a>Метод ConvertToString (RDS)

**Область применения**: Access 2013, Office 2013 

Преобразует объект [Recordset](recordset-object-ado.md) в строку MIME, представляющую данные набора записей.

## <a name="syntax"></a>Синтаксис

*Фактическое*. ConvertToString (*Recordset*)

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*DataFactory* |Объектная переменная, представляющая объект [фактОв рдссервер.](datafactory-object-rdsserver.md) DataObject.|
|*Recordset* |Объектная переменная, представляющая объект **Recordset** .|

## <a name="remarks"></a>Замечания

С помощью файлов. ASP используйте **ConvertToString** , чтобы внедрить **набор записей** на HTML-страницу, созданную на сервере, чтобы передать его на клиентский компьютер.

**ConvertToString** сначала загружает **набор записей** в таблицы службы курсоров, а затем создает поток в формате MIME.

На клиенте служба удаленных данных может преобразовать строку MIME обратно в полностью работоспособный **набор записей**. Он подходит для обработки менее 400 строк данных, размер которых не превышает 1024 байт в каждой строке. Его не следует использовать для потоковой передачи больших ДВОИЧных данных и больших наборов результатов по ПРОТОКОЛу HTTP. В строке не выполняется сжатие по каналу, поэтому очень большие наборы данных потребуют значительного времени для транспорта по ПРОТОКОЛу HTTP, по сравнению с форматом таблеграм, определенным и развернутым удаленной службой данных в качестве собственного формата транспортного протокола.

> [!NOTE]
> Если вы используете страницы Active Server для внедрения полученной строки MIME на клиентской HTML-странице, имейте в виду, что более ранние версии VBScript, чем версия 2,0, ограничивают размер строки до 32 КБ. Если это ограничение превышено, возвращается ошибка. При внедрении MIME с помощью файлов ASP отДерживайте область запроса относительно небольшой. Чтобы устранить эту проблему, скачайте последнюю версию VBScript из [центра загрузки Майкрософт](https://www.microsoft.com/download/default.aspx).
