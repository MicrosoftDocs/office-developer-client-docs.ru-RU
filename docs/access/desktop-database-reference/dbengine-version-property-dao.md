# <a name="dbengineversion-property-dao"></a>Свойство DBEngine. Version (DAO)


**Область применения**: Access 2013, Office 2013

Рретурнс используемая версия объекта DAO. Только для чтения, **String**.

## <a name="syntax"></a>Синтаксис

*Expression* . Версия

*expression*: переменная, представляющая объект **DBEngine**.

## <a name="remarks"></a>Примечания

Возвращаемое значение — это строка, которая оценивает номер версии в виде "Major. Minor". Например, "3,0". Номер версии продукта состоит из номера версии (3), точки и номера выпуска (0).
