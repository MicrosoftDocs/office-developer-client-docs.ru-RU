# <a name="dbengineerrors-property-dao"></a>Свойство DBEngine.Errors (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **ошибок** , который содержит все хранимые объекты **об ошибках** для указанного объекта. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . Ошибки

*выражение* Переменная, которая представляет собой объект- **DBEngine** .
