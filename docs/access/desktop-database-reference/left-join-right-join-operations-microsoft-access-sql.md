<span data-ttu-id="4314d-135">В этом примере:</span><span class="sxs-lookup"><span data-stu-id="4314d-135">This example:</span></span>
- <span data-ttu-id="4314d-136">Предполагается наличие предполагаемую имя отдела и отдела идентификатор поля в таблице «Сотрудники».</span><span class="sxs-lookup"><span data-stu-id="4314d-136">Assumes the existence of hypothetical Department Name and Department ID fields in an Employees table.</span></span> <span data-ttu-id="4314d-137">Обратите внимание на то, что фактически эти поля не существуют в таблице Employees базы данных Northwind.</span><span class="sxs-lookup"><span data-stu-id="4314d-137">Note that these fields do not actually exist in the Northwind database Employees table.</span></span>

- <span data-ttu-id="4314d-138">Выбирает все отделы, в том числе без сотрудников.</span><span class="sxs-lookup"><span data-stu-id="4314d-138">Selects all departments, including those without employees.</span></span>

- <span data-ttu-id="4314d-139">Вызывает процедуру EnumFields, которые можно найти в примере инструкции SELECT.</span><span class="sxs-lookup"><span data-stu-id="4314d-139">Calls the EnumFields procedure, which you can find in the SELECT statement example.</span></span>


```vb
    Sub LeftRightJoinX() 
     
        Dim dbs As Database, rst As Recordset 
     
        ' Modify this line to include the path to Northwind 
        ' on your computer. 
        Set dbs = OpenDatabase("Northwind.mdb") 
         
        ' Select all departments, including those  
        ' without employees. 
        Set rst = dbs.OpenRecordset _ 
            ("SELECT [Department Name], " _ 
            & "FirstName & Chr(32) & LastName AS Name " _ 
            & "FROM Departments LEFT JOIN Employees " _ 
            & "ON Departments.[Department ID] = " _ 
            & "Employees.[Department ID] " _ 
            & "ORDER BY [Department Name];") 
         
        ' Populate the Recordset. 
        rst.MoveLast 
         
        ' Call EnumFields to print the contents of the  
        ' Recordset. Pass the Recordset object and desired 
        ' field width. 
        EnumFields rst, 20 
     
        dbs.Close 
     
    End Sub
```
