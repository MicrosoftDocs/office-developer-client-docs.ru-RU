# <a name="append-method-adox-tables"></a>Метод Append (коллекция Tables в ADOX)

**Область применения**: Access 2013, Office 2013

Добавляет новый объект [Table](table-object-adox.md) в коллекцию [Tables](tables-collection-adox.md) .

## <a name="syntax"></a>Синтаксис

*Таблицы*. Добавление*таблицы*

## <a name="parameters"></a>Параметры

|Параметр|Описание|
|:--------|:----------|
|*Table* | Значение **Variant** , которое содержит ссылку на **таблицу** для добавления или имя таблицы для создания и добавления.|

## <a name="remarks"></a>Примечания

Если поставщик не поддерживает создание таблиц, произойдет ошибка.
