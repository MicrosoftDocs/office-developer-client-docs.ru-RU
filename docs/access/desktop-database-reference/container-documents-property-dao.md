# <a name="containerdocuments-property-dao"></a>Container.Documents Property (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **документов** , который содержит все документы, хранящиеся в указанном объекте. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . Документы

*выражение* Переменная, которая представляет собой объект- **контейнер** .
