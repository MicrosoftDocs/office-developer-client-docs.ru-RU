# <a name="importexporttext-macro-action"></a>Макрокоманда ImportExportText

**Применимо к**: Access 2013, Office 2013

Действие **ImportExportText** можно использовать для импорта или экспорта текста между текущей базы данных Microsoft Access (MDB- или .accdb) или проект Microsoft Access (файлы с расширением ADP) и в текстовый файл. Также можно связать данные в текстовый файл текущей базы данных Access. С помощью связанного текстового файла можно просмотреть текстовых данных с доступом по-прежнему предоставляя полный доступ к данным из программы обработки текста. Также импортировать из, экспортировать и ссылка на таблицы или списка в файле HTML (\*.html).

> [!NOTE]
> При связывании данных в текстовый файл или HTML-файл данных доступен только для чтения в клиенте. Это действие не разрешено, если база данных не является доверенной.

## <a name="remarks"></a>Примечания

Текстовые файлы можно экспортировать данные в запросах доступа. Microsoft Access экспортирует набор результатов запроса в виде таблицы.

Текстовые данные, добавляемые в существующую таблицу Access должны быть совместимы со структурой таблицы.

- Каждое поле в текст должен быть типу данных, то соответствующее поле в таблице.

- Поля должны быть в том же порядке (Если аргумент **Содержит имена полей** задано значение **Да**, в этом случае поле имена в тексте должны совпадать имена полей в таблице).

Это действие аналогично выберите **Текстовый файл** в **Импорт** или **Экспорт** группы на вкладке **Внешних данных** . Аргументы действие **ImportExportText** соответствуют параметрам в мастере работы с помощью команды **Текстовый файл** .

> [!TIP]
> Спецификации импорта/экспорта содержит сведения, необходимые Microsoft Access для импорта и экспорта текстового файла. Можно использовать сохраненные спецификаций на импорт, экспорт и связывание данных текст для следующего текстовые файлы. Например вы можете получить еженедельно объемы продаж в текстовый файл с большого компьютера. Можно создавать и сохранять спецификацию для этого типа данных и затем использовать спецификации при каждом добавлении эти данные в базу данных Access.

> [!NOTE]
> Запроса и фильтрации связанный текстовый файл запроса или фильтра с учетом регистра.

Чтобы выполнить действие **ImportExportText** в Visual Basic для приложений (VBA) модуль, используйте метод **ПреобразоватьТекст** **объекта** .
