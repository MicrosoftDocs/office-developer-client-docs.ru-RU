# <a name="connectionrecordsets-property-dao"></a>Свойство Connection.Recordsets (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает **коллекцию Recordsets,** которая содержит все открытые записи в указанном соединении. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражения* . Записи

*выражение*: переменная, представляющая объект **Connection**.
