# <a name="onreadystatechange-event-rds"></a>onReadyStateChange event (RDS)

**Область применения**: Access 2013, Office 2013

Событие **onReadyStateChange** называется всякий раз, когда меняется значение свойства [ReadyState.](readystate-property-rds.md)

## <a name="syntax"></a>Синтаксис

onReadyStateChange

## <a name="parameters"></a>Параметры

Нет.

## <a name="remarks"></a>Примечания

Свойство **ReadyState** отражает ход выполнения [RDS. Объект DataControl](datacontrol-object-rds.md) асинхронно извлекает данные в объект [Recordset.](recordset-object-ado.md) Используйте **событие onReadyStateChange** для отслеживания изменений в свойстве **ReadyState** при их возникновения. Это эффективнее, чем периодически проверять значение свойства.
