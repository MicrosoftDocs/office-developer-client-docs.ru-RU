# <a name="errors"></a>Ошибки

**Область применения**: Access 2013, Office 2013

Любая операция, включающая объекты ADO, может создавать одну или несколько ошибок поставщика. При возникновении каждой ошибки в коллекцию **Errors** объекта **Connection** помещаются один или несколько объектов **Error** . Подробные сведения об обработке предупреждений и ошибок в приложении ADO содержатся в [главе 6: обработка ошибок](chapter-6-error-handling.md).

Ошибки приложений могут быть вызваны отдельным механизмом. Например, в Visual Basic объект **Err** будет содержать ошибки на уровне приложения.
