# <a name="levelname-property-ado-md"></a>Свойство LevelName (ADO MD)


**Область применения**: Access 2013, Office 2013

Указывает имя уровня элемента.

## <a name="return-values"></a>Возвращаемые значения

Возвращает **строку** и доступна только для чтения.

## <a name="remarks"></a>Примечания

Используйте свойство **LevelName** для получения имени уровня, к которому принадлежит элемент. Это соответствует свойству [Name](name-property-ado-md.md) объекта [Level](level-object-ado-md.md) .
