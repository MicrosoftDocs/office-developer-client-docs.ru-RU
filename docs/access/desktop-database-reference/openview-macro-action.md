# <a name="openview-macro-action"></a>Макрокоманда OpenView

**Область применения**: Access 2013, Office 2013

В проекте Access можно использовать действие **опенвиев** для открытия представления в режиме таблицы, конструктора или предварительного просмотра. Это действие запускает именованное представление при открытии в представлении таблицы данных. Вы можете выбрать запись данных для представления и ограничить записи, отображаемые в представлении.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Примечания

Это действие аналогично двойному щелчку представления в области навигации или щелчком правой кнопкой мыши представления в области навигации и выбора нужной команды.

> [!TIP]
> - Вы можете перетащить представление из области навигации в строку макрокоманды. При этом автоматически создается действие **опенвиев** , которое открывает представление в режиме таблицы.
> - Если вы не хотите отображать системные сообщения, которые обычно отображаются при запуске представления (указывает, что это представление и показывает, сколько записей будет затронуто), можно использовать действие **сетварнинг** , чтобы отключить отображение этих сообщений.

Чтобы запустить действие **опенвиев** в модуле Visual Basic для приложений (VBA), используйте метод **опенвиев** объекта **DoCmd** .
