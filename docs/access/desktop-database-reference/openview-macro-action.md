# <a name="openview-macro-action"></a>Макрокоманда OpenView

**Область применения**: Access 2013, Office 2013

В проекте Access можно использовать действие **OpenView** для открытия представления в представлении таблицы данных, представлении дизайна или предварительном просмотре печати. Это действие выполняет именуемую точку представления при открываемом представлении Таблицы данных. Вы можете выбрать запись данных для представления и ограничить записи, отображаемые в представлении.

> [!NOTE]
> Эта макрокоманда доступна только для доверенных баз данных.

## <a name="remarks"></a>Примечания

Это действие аналогично дважды щелкнув представление в области навигации или щелкнув правой кнопкой мыши представление в области навигации, а затем щелкнув команду, которая вам нужна.

> [!TIP]
> - Можно перетаскивать представление из области навигации в строку действий макроса. Это автоматически создает действие **OpenView,** которое открывает представление в представлении Datasheet.
> - Если вы не хотите отображать системные сообщения, которые обычно отображаются при запуске представления (указывающее представление и показывая, сколько записей будет затронуто), вы можете использовать действие **SetWarning** для подавления отображения этих сообщений.

Чтобы выполнить **действие OpenView** в Visual Basic для приложений (VBA), используйте метод **OpenView** объекта **DoCmd.**
