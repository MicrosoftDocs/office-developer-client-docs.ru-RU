# <a name="relationfields-property-dao"></a>Свойство Relation.Fields (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает коллекцию **Fields**, которая представляет все объекты **Field** для указанного объекта. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражения* . Поля

*выражение* Переменная, представляюная объект **Relation.**
