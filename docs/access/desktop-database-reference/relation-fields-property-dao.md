# <a name="relationfields-property-dao"></a>Свойство Relation.Fields (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает коллекцию **полей** , представляющую все хранятся объекты **поля** для указанного объекта. Только для чтения.

## <a name="syntax"></a>Синтаксис

*выражение* . Поля

*выражение* Переменная, которая представляет собой объект- **связи** .
