# <a name="reshape-name-dynamic-property-ado"></a>Динамическое свойство Name Shape (ADO)


**Область применения**: Access 2013, Office 2013

Задает имя объекта [Recordset](recordset-object-ado.md) .

## <a name="return-values"></a>Возвращаемые значения

Возвращает **строковое** значение, которое является именем объекта **Recordset**.

## <a name="remarks"></a>Примечания

Имена хранятся в течение всего подключения или до закрытия **набора записей** .

Свойство **изменить имя фигуры** в основном предназначено для использования с функцией изменения формы [службы формирования данных Microsoft для поставщика услуг OLE DB](microsoft-data-shaping-service-for-ole-db-ado-service-provider.md) . Имена должны быть уникальными для участия в изменения формы.

Это свойство доступно только для чтения, но оно может быть задано как неявное при создании набора записей. Например, если предложение команды SHAPE создает объект Recordset и присваивает ему имя псевдонима с ключевым словом "AS", то псевдоним назначается свойству "изменить имя фигуры". Если псевдоним не объявлен, свойство RESHAPE Name присваивает ему уникальное имя, созданное службой формирования данных. Если псевдоним совпадает с именем существующего **набора записей**, то ни один из них не должен быть обновлен до тех пор, пока **не будет выпущен** один из них. Поведение по умолчанию можно изменить, задав для свойства "уникальные имена перерисовок" (см. раздел "Служба формирования данных Майкрософт для OLE DB") для подключения ADO значение TRUE. Это дает службе формирования данных разрешение на изменение имен пользователей ассиженд, если это необходимо, чтобы обеспечить уникальность.

Используйте свойство **RESHAPE Name** , если необходимо сослаться на объект **RECORDSET** в команде Shape, или если имя не известно, так как оно было создано службой формирования данных. В этом случае вы можете создать команду SHAPE, присоединяемую команду к строке, возвращаемой свойством **RESHAPE Shape Name** .

**Имя изменения фигуры** — это динамическое свойство, добавленное в коллекцию [свойств](properties-collection-ado.md) объекта **Recordset** , если для свойства [CursorLocation](cursorlocation-property-ado.md) задано значение **адусеклиент**.
