# <a name="reshape-name-dynamic-property-ado"></a>Изменить форму динамических свойство Name (ADO)


**Применимо к**: Access 2013, Office 2013

Указывает имя для объекта [набора записей](recordset-object-ado.md) .

## <a name="return-values"></a>Возвращаемые значения

Возвращает значение типа **String** , — это имя **набора записей**.

## <a name="remarks"></a>Примечания

Сохранение имен во время выполнения подключения или до закрытия **набора записей** .

Свойство **Name изменения формы** в основном предназначены для использования с повторно формирования функции поставщика услуг [Служба формирования Microsoft данных для OLE DB](microsoft-data-shaping-service-for-ole-db-ado-service-provider.md) . Имена должны быть уникальными для участия в формировании повторно.

Это свойство доступно только для чтения, но можно задать косвенно, при создании набора записей. Например если предложение ФИГУРУ, команда создает набор записей и присваивает ему имя псевдонима с ключевым словом «AS», псевдоним назначается свойство Name изменить форму. Если псевдоним не объявляется свойство name изменения формы присваивается уникальное имя, созданных функцией службы формирования данных. Если имя псевдонима совпадает с именем существующего **набора записей**, ни один из **набора записей** быть изменили форму до их выпуска. Поведение по умолчанию можно изменить, задав «Уникальный изменить форму имена» свойства (см. «Данных Microsoft формирования служба для OLE DB») на ADO-подключение имеет значение TRUE. Это получить формирования службе разрешения на изменение имен assigend пользователей, если это необходимо для обеспечения уникальности данных.

Для ссылки на **набор записей** в команде ФИГУРЫ, или если вы не знаете его имени, так как он был создан, служба формирования данных, воспользуйтесь свойством **Имя формы** . В этом случае удалось создать команду ФИГУРЫ путем объединения команду, применимую к строки, возвращаемой свойством **Name изменить форму** .

**Изменить форму имя** является динамическим добавляется в конец коллекции [свойств](properties-collection-ado.md) объекта **набора записей** при [CursorLocation](cursorlocation-property-ado.md) задано значение **adUseClient**.
