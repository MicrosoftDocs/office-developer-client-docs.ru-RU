# <a name="indexfields-property-dao"></a>Свойство Index.Fields (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает коллекцию **Fields**, которая представляет все объекты **Field** для указанного объекта. Для чтения и записи.

## <a name="syntax"></a>Синтаксис

*выражения* . Поля

*выражение* Переменная, представляюная объект **Index.**
