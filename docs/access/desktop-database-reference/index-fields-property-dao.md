# <a name="indexfields-property-dao"></a>Index.Fields Property (DAO)


**Применимо к**: Access 2013 | Office 2013

Возвращает коллекцию **полей** , представляющую все хранятся объекты **поля** для указанного объекта. Для чтения и записи.

## <a name="syntax"></a>Синтаксис

*выражение* . Поля

*выражение* Переменная, которая представляет объект **индекса** .
