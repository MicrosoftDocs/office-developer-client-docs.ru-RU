## <a name="example"></a><span data-ttu-id="dde25-152">Пример</span><span class="sxs-lookup"><span data-stu-id="dde25-152">Example</span></span>

<span data-ttu-id="dde25-153">В этом примере добавляется поле Salary (Заработная плата) с типом данных **Money** в таблицу Employees (Сотрудники).</span><span class="sxs-lookup"><span data-stu-id="dde25-153">This example adds a Salary field with the data type **Money** to the Employees table.</span></span>

```vb
    Sub AlterTableX1() 
     
        Dim dbs As Database 
     
        ' Modify this line to include the path to Northwind 
        ' on your computer. 
        Set dbs = OpenDatabase("Northwind.mdb") 
     
        ' Add the Salary field to the Employees table  
        ' and make it a Money data type. 
        dbs.Execute "ALTER TABLE Employees " _ 
            & "ADD COLUMN Salary MONEY;" 
     
        dbs.Close 
     
    End Sub 
```

<br/>

<span data-ttu-id="dde25-154">В этом примере тип данных поля Salary (Заработная плата) изменяется с **Money** на **Char**.</span><span class="sxs-lookup"><span data-stu-id="dde25-154">This example changes the Salary field from the data type **Money** to the data type **Char**.</span></span>

```vb
    Sub AlterTableX2() 
     
        Dim dbs As Database 
     
        ' Modify this line to include the path to Northwind 
        ' on your computer. 
        Set dbs = OpenDatabase("Northwind.mdb") 
     
        ' Modify the existing Salary field of the Employees table  
        ' by changing it to a CHAR data type. 
        dbs.Execute "ALTER TABLE Employees " _ 
            & "ALTER COLUMN Salary CHAR(20);" 
     
        dbs.Close 
     
    End Sub 
```

<br/>

<span data-ttu-id="dde25-155">В этом примере удаляется поле Salary (Заработная плата) из таблицы Employees (Сотрудники).</span><span class="sxs-lookup"><span data-stu-id="dde25-155">This example removes the Salary field from the Employees table.</span></span>

```vb
    Sub AlterTableX3() 
     
        Dim dbs As Database 
     
        ' Modify this line to include the path to Northwind 
        ' on your computer. 
        Set dbs = OpenDatabase("Northwind.mdb") 
     
        ' Delete the Salary field from the Employees table.

    End Sub
```

<br/>

<span data-ttu-id="dde25-156">В этом примере добавляется внешний ключ для таблицы Orders (Заказы).</span><span class="sxs-lookup"><span data-stu-id="dde25-156">This example adds a foreign key to the Orders table.</span></span> <span data-ttu-id="dde25-157">Внешний ключ основан на поле EmployeeID (Код сотрудника) и ссылается на поле EmployeeID (Код сотрудника) таблицы Employees (Сотрудники).</span><span class="sxs-lookup"><span data-stu-id="dde25-157">The foreign key is based on the EmployeeID field and refers to the EmployeeID field of the Employees table.</span></span> <span data-ttu-id="dde25-158">В этом примере не требуется перечислять поле EmployeeID (Код сотрудника) после таблицы Employees (Сотрудники) в предложении REFERENCES, так как EmployeeID — это первичный ключ таблицы Employees (Сотрудники).</span><span class="sxs-lookup"><span data-stu-id="dde25-158">In this example, you do not have to list the EmployeeID field after the Employees table in the REFERENCES clause because EmployeeID is the primary key of the Employees table.</span></span>

```vb
    Sub AlterTableX4() 
     
        Dim dbs As Database 
     
        ' Modify this line to include the path to Northwind 
        ' on your computer. 
        Set dbs = OpenDatabase("Northwind.mdb") 
     
        ' Add a foreign key to the Orders table. 
        dbs.Execute "ALTER TABLE Orders " _ 
            & "ADD CONSTRAINT OrdersRelationship " _ 
            & "FOREIGN KEY (EmployeeID) " _ 
            & "REFERENCES Employees (EmployeeID);" 
     
        dbs.Close 
     
    End Sub 
```

<br/>

```vb
        ' Modify this line to include the path to Northwind 
        ' on your computer. 
        Set dbs = OpenDatabase("Northwind.mdb") 
     
        ' Remove the OrdersRelationship foreign key from 
        ' the Orders table. 
        dbs.Execute "ALTER TABLE Orders " _ 
            & "DROP CONSTRAINT OrdersRelationship;" 
     
        dbs.Close 
     
    End Sub
```
