# <a name="indexes-collection-adox"></a>Indexes Collection (ADOX)


**Применимо к**: Access 2013 | Office 2013

Содержит все объекты [индекса](index-object-adox.md) таблицы.

## <a name="remarks"></a>Замечания

Метод [Append](append-method-adox-indexes.md) для коллекции **индексов** является уникальным для ADOX. Вы можете выполнить указанные ниже действия.

  - Добавьте новый индекс коллекции с помощью метода **Append** .

Остальные свойства и методы являются стандартными в семействах сайтов ADO. Вы можете выполнить указанные ниже действия.

  - Доступ к индексу в коллекции с помощью свойства [элемента](item-property-ado.md) .

  - Возвращает число индексов, содержащихся в коллекции со свойством [Count](count-property-ado.md) .

  - Удаление индекса из коллекции с помощью метода [Delete](delete-method-adox-collections.md) .

  - Обновление объектов в коллекции в соответствии с помощью метода [обновления на](refresh-method-ado.md) схему текущей базы данных.
