# <a name="recordset2parentrecordset-property-dao"></a>Recordset2.ParentRecordset Property (DAO)


**Применимо к**: Access 2013, Office 2013 

Возвращает родительский **записей** из указанного набора записей. Только для чтения.

## <a name="version-information"></a>Сведения о версии

Добавлена версия: Access 2007


## <a name="syntax"></a>Синтаксис

*выражение* . ParentRecordset

*выражение* Переменная, которая представляет собой объект- **Recordset2** .

## <a name="remarks"></a>Замечания

Свойство **ParentRecordset** возвращает **значение Null** , если указанный набор записей представляет многозначное поле.
