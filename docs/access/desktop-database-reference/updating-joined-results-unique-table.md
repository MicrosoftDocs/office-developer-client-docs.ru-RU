# <a name="updating-joined-results-unique-table"></a>Обновление результатов JOINed: уникальная таблица


**Применимо к**: Access 2013, Office 2013

## <a name="updating-joined-results-unique-table"></a>Обновление результатов JOINed: уникальная таблица

ADO позволяет тесно управления изменения определенного базовая таблица в **набор записей** , которая была создана с помощью операции ОБЪЕДИНЕНИЯ на нескольких базовых таблиц с помощью свойства динамического **Уникальной таблицы** . Для получения дополнительных сведений об использовании **Уникальной таблицы**обратитесь к программиста ADO справочные разделы, посвященные **Уникальной таблицы** и **Установки выполнить повторную синхронизацию обновление** динамических свойств.
