# <a name="updating-joined-results-unique-table"></a>Updating JOINed Results: Unique Table


**Область применения**: Access 2013, Office 2013

## <a name="updating-joined-results-unique-table"></a>Обновление результатов JOINed: уникальная таблица

ADO позволяет тесно контролировать изменения в определенной базовой таблице в **наборе записей** , созданном ОПЕРАЦИей JOIN с несколькими базовыми таблицами с помощью динамического свойства **UNIQUE Table** . Для получения подробных сведений об использовании **уникальной таблицы**обратитесь к справочным разделам ПРОГРАММИСТа ADO, поСвященным **уникальной таблице** , и обновите динамические свойства **ресинхронизации** .
