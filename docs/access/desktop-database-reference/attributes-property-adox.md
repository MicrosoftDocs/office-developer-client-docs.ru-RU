# <a name="attributes-property-adox"></a>Свойство Attributes (ADOX)


**Применимо к**: Access 2013, Office 2013

Описывает характеристики столбца.

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает или возвращает значение типа **Long** . Значение указывает характеристики таблицы, представленного объектом [столбца](column-object-adox.md) и может быть комбинацией [ColumnAttributesEnum](columnattributesenum.md) констант. Значение по умолчанию — нуль (0), который не является ни **adColFixed** , ни **adColNullable**.
