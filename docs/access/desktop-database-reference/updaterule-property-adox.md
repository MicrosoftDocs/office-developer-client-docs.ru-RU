# <a name="updaterule-property-adox"></a>Свойство UpdateRule (ADOX)


**Область применения**: Access 2013, Office 2013

Указывает действие, выполняемое при обновлении первичного ключа.

## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения

Задает и возвращает **длинное** значение, которое может быть одной из констант [рулинум](ruleenum.md) . Значение по умолчанию — **адриноне**.

## <a name="remarks"></a>Примечания

Это свойство доступно только для чтения для объектов [Key](key-object-adox.md) , уже добавленных в коллекцию.
