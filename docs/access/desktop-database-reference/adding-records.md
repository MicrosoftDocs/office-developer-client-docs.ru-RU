# <a name="adding-records"></a>Добавление записей

**Область применения**: Access 2013, Office 2013

Используйте метод **AddNew** для создания и инициализации новой записи в существующем **наборе записей.** Вы можете использовать метод **Supports** со значением **CursorOptionEnum** **объекта adAddNew,** чтобы проверить, можно ли добавлять записи к текущему объекту **Recordset.**

После вызова метода **AddNew** новая запись становится текущей и остается текущей после вызова метода **Update.** Если объект **Recordset** не поддерживает закладки, возможно, вы не сможете получить доступ к новой записи после перемещения в другую запись. Поэтому в зависимости от типа курсора может потребоваться вызвать метод **Requery,** чтобы сделать новую запись доступной.

Если вы вызываете **AddNew** при редактировании текущей записи или при добавлении новой записи, ADO вызывает метод **Update,** чтобы сохранить все изменения, а затем создает новую запись.
