# <a name="fieldvalidationrule-property-dao"></a>Field.ValidationRule Property (DAO)


**Применимо к**: Access 2013 | Office 2013

Задает или возвращает значение, которое проверяет данные в поля, как она изменен или добавлен в таблицу (только для рабочих областей Microsoft Access). Для чтения и записи, **String**.

## <a name="syntax"></a>Синтаксис

*выражение* . Значение

*выражение* Выражение, возвращающее объект **поля** .

## <a name="remarks"></a>Замечания

Параметры или возвращаемые значения — это строка, описывающая сравнения в виде SQL, ГДЕ предложения WHERE без зарезервированным словом. Для объекта еще не добавляется в конец коллекции **[полей](fields-collection-dao.md)** это свойство соответствует чтения и записи.

**Значение** определяет, содержит ли поле допустимых данных. Если данные не является допустимым, возникает перехватываемые ошибки времени выполнения. Сообщения об ошибке — текст свойства **[сообщение об ошибке](field-validationtext-property-dao.md)** , если указан или текст заданное **значение**.

Для объекта **[поля](field-object-dao.md)** используйте **значение** зависит от объекта, который содержит коллекцию **полей** , к которому добавляется объект **поля** .

Выражение не может ссылаться на пользовательские функции, статистические функции SQL и запросы. Чтобы задать **значение объекта **поля** ** при его **Проверка набора** свойство имеет значение **True**, выражение необходимо успешно проанализировать (с именем подразумеваемых операнда) и иметь значение **True**.
