# <a name="recordsetdatecreated-property-dao"></a>Свойство Recordset.DateCreated (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает дату и время создания базовой таблицы (только для рабочих областей Microsoft Access). Только для чтения, **Variant**.

## <a name="syntax"></a>Синтаксис

*Expression* . DateCreated

*expression*: переменная, представляющая объект **Recordset**.

## <a name="remarks"></a>Примечания

Параметры даты и времени производятся на компьютере, на котором была создана или обновлена базовая таблица.
