# <a name="moveandsizewindow-macro-action"></a>Макрокоманда MoveAndSizeWindow

**Применимо к**: Access 2013, Office 2013

Если документ окно Параметры настроены для использования перекрывающиеся windows вместо вкладками, можно использовать действие **MoveAndSizeWindow** переместить или изменить размер активного окна. Сведения о том, как задать параметры окна документа см.

## <a name="setting"></a>Параметр

Действие **MoveAndSizeWindow** состоит из следующих аргументов.

Необходимо ввести значение для по крайней мере один из аргументов.

> [!NOTE]
> Измерения — в дюймах или сантиметрах, в зависимости от региональных параметров в панели управления Windows.

## <a name="remarks"></a>Примечания

Чтобы настроить приложение для использования перекрывающиеся windows вместо вкладками, используйте следующую процедуру:

1.  Нажмите кнопку **Параметры**

2.  Нажмите кнопку **текущей базы данных**.

3.  В разделе **Параметры приложения** в разделе **Параметры окна документа**нажмите кнопку **Перекрывающиеся Windows**.

4.  Нажмите кнопку **ОК**и закройте и снова откройте базу данных.

Это действие аналогично меню **переместить** или **размер** окна **элемента управления** . С помощью команды меню используйте клавиши со стрелками для перемещения или изменить размеры этого окна. С действием **MoveAndSizeWindow** введите его положение и размер значения напрямую. Можно также использовать мышь в перемещение и размер windows.

Это действие можно использовать для любого окна в любом представлении.

> [!TIP]
> - Переместить окно без его размера, введите значения для **вправо** и **вниз** аргументов, но оставить пустым аргументы **ширины** и **высоты** .
> - Чтобы изменить размер окна без перемещения, введите значения для **ширины** и **высоты** аргументов, но оставить пустым аргументы **вправо** и **вниз** .

Чтобы выполнить действие **MoveAndSizeWindow** в Visual Basic для приложений (VBA) модуль, используйте метод **СдвигРазмер** **объекта** .
