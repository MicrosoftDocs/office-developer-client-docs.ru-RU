# <a name="formattedvalue-property-ado-md"></a>Свойство FormattedValue (ADO MD)


**Область применения**: Access 2013, Office 2013

Указывает форматированный дисплей значения ячейки.

## <a name="return-values"></a>Возвращаемые значения

Возвращает **строку и** только для чтения.

## <a name="remarks"></a>Примечания

Используйте свойство **FormattedValue** для получения отформатированных значений отображения свойства [Value](value-property-ado-md.md) объекта [Cell.](cell-object-ado-md.md) Например, если значение ячейки было 1056.87, а это значение представляло сумму доллара, **FormattedValue** будет $1,056.87.
