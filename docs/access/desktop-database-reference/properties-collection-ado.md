# <a name="properties-collection-ado"></a>Коллекция Properties (ADO)

**Область применения**: Access 2013, Office 2013

Содержит все [объекты Property](property-object-ado.md) для определенного экземпляра объекта.

## <a name="remarks"></a>Заметки

Некоторые объекты ADO имеют коллекцию **свойств,** которая состоит из **объектов Property.** **Каждый** объект Property соответствует характеристике объекта ADO, характерного для поставщика.

> [!NOTE]
> Более [подробное описание](property-object-ado.md) использования объектов Property см. в разделе "Объект **свойства".**

**Динамические свойства** объекта **Recordset** выходить за пределы области (становятся недоступными) при закрытии **объекта Recordset.**
