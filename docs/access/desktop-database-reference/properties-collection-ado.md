# <a name="properties-collection-ado"></a>Коллекция свойств (ADO)

**Область применения**: Access 2013, Office 2013

Содержит все [объекты Свойства](property-object-ado.md) для определенного экземпляра объекта.

## <a name="remarks"></a>Примечания

Некоторые объекты ADO имеют коллекцию **свойств,** которая состоит из **объектов Property.** Каждый объект **Property** соответствует характеристике объекта ADO, определенного поставщику.

> [!NOTE]
> Дополнительные [сведения](property-object-ado.md) об использовании объектов Свойства см. в разделе Объект **Свойства.**

**Динамические свойства** объекта **Recordset** выйдут из области (становятся недоступными) при закрытии **наборов** записей.
