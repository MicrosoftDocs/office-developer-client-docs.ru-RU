# <a name="setmenuitem-macro-action"></a>Макрокоманда SetMenuItem

**Область применения**: Access 2013, Office 2013

Действие **SetMenuItem** можно использовать для настройки состояния элементов меню (включено или отключено, выбрано или невыбрано) в настраиваемом или глобальном меню на вкладке Надстройки. 

> [!NOTE]
> Действие **SetMenuItem** работает только с пользовательскими и глобальными меню, созданными с помощью макрос меню. Действие **SetMenuItem** включено в Microsoft Access только для совместимости с предыдущими версиями. Он не работает с функцией панели команд. Однако в модуле  Visual Basic для приложений  (VBA) можно использовать свойства включено и состояние для отключения или включения и выбора элементов в меню ярлыка или настраиваемом или глобальном меню.

## <a name="remarks"></a>Примечания

Действие **SetMenuItem** работает только в настраиваемом или глобальном меню. Если в активном окне нет настраиваемого или глобального меню, запуск макроса, содержащего действие **SetMenuItem,** вызывает ошибку во времени запуска.

Это действие можно использовать для настройки состояния команд меню и подкоммандатов, но не подкомкомитетов подкомитетов.

Чтобы выполнить **действие SetMenuItem** в модуле Visual Basic для приложений (VBA), используйте метод **SetMenuItem** объекта **DoCmd.**
