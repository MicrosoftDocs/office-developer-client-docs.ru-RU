# <a name="setmenuitem-macro-action"></a>Макрокоманда SetMenuItem


**Применимо к**: Access 2013, Office 2013

**Макрокоманды** можно использовать для задания состояния пунктов меню (включены или отключены, выбранные или не выбран) на пользовательских и глобальных меню на вкладке **Надстройки** .


> [!NOTE]
> <P><STRONG>Макрокоманды</STRONG> работает только с пользовательских и глобальных меню, созданные с помощью макросов меню. <STRONG>Макрокоманды</STRONG> включен в Microsoft Access только для совместимости с предыдущими версиями. Он не работает с функциональных возможностях панели команд. Тем не менее можно использовать свойства <STRONG>Enabled</STRONG> и <STRONG>состояния</STRONG> в Visual Basic для приложений (VBA) модуль отключение или включение и выберите или отмените выбор настройки элементов контекстных меню или настраиваемые или глобального меню.</P>

## <a name="remarks"></a>Примечания

**Макрокоманды** работает только в пользовательских или глобального меню. Если активного окна не имеет настраиваемых или глобального меню, макрос, содержащий **макрокоманды** вызывает ошибку времени выполнения.

Это действие можно использовать для установки состояния команды меню и команд, но не команд подменю следующих уровней.

Для запуска **макрокоманды** в Visual Basic для приложений (VBA) модуль, используйте метод **ЗадатьКомандуМеню** **объекта** .
