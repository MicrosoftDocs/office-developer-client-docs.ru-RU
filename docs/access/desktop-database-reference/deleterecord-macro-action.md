# <a name="deleterecord-macro-action"></a>Макрокоманда DeleteRecord

**Область применения**: Access 2013, Office 2013

Для удаления записи можно использовать действие **DeleteRecord.**

## <a name="setting"></a>Setting

Блок **данных CreateRecord** имеет следующие аргументы.

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><p>Аргументация</p></th>
<th><p>Описание</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p><strong>Запись псевдонима</strong></p></td>
<td><p>Строка, идентифицирует запись, которую необходимо удалить. Если <em>аргумент Alias</em> не указан, текущая запись удаляется.</p></td>
</tr>
</tbody>
</table>

## <a name="remarks"></a>Заметки

Локализованную переменную **LastCreateRecordIdentity** можно использовать для работы с последней записью, созданной в блоке данных **CreateRecord.** Например, используйте следующий синтаксис для ссылки на недавно созданную запись:

`[LastCreateRecordIdentity]`
