# <a name="positions-collection-ado-md"></a>Коллекция Positions (ADO MD)


**Область применения**: Access 2013, Office 2013

Содержит объекты [position](position-object-ado-md.md) , определяющие точку на оси.

## <a name="remarks"></a>Примечания

**Positions** — это стандартная коллекция ADO. С помощью свойств и методов коллекции можно выполнить следующие действия:

  - Получите число объектов в коллекции со свойством [Count](count-property-ado.md) .

  - Возвращает объект из коллекции со свойством [Item](item-property-ado.md) по умолчанию.

  - Обновление объектов в коллекции от поставщика с помощью метода [Refresh](refresh-method-ado.md) .
