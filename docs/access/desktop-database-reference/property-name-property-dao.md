# <a name="propertyname-property-dao"></a>Свойство Property.Name (DAO)


**Применимо к**: Access 2013, Office 2013

Возвращает или задает имя указанного объекта. Чтение и запись **строки** Если объект не были добавлены в коллекцию. Только для чтения **строка** Если объект были добавлены в коллекцию.

## <a name="syntax"></a>Синтаксис

*выражение* . Имя

*выражение* Переменная, которая представляет собой объект- **свойство** .

## <a name="remarks"></a>Примечания

Свойство **Name** встроенных свойств всегда является только для чтения.

Максимальная длина имени объекта **Property** — 64 символов.
