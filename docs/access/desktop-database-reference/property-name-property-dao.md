# <a name="propertyname-property-dao"></a>Property.Name (DAO)


**Область применения**: Access 2013, Office 2013

Возвращает или задает имя указанного объекта. Строка **read/write,** если объект не был придан коллекции. Строка **только для** чтения, если объект был придан коллекции.

## <a name="syntax"></a>Синтаксис

*выражения* . Имя

*выражение* Переменная, представляюная объект **Property.**

## <a name="remarks"></a>Примечания

Свойство **Name** встроенного свойства всегда только для чтения.

Максимальная длина имени объекта **Property** — 64 символа.
