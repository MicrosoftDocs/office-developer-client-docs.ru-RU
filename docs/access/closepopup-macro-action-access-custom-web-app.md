# <a name="closepopup-macro-action-access-custom-web-app"></a>Макро-действие ClosePopup (Доступ к настраиваемой веб-приложению)

Для закрытия активного всплывающих представлений документов можно использовать действие **ClosePopup.** 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="remarks"></a>Примечания

Если текущая запись в всплывающее представление находится в режиме редактирования, изменения в записи автоматически сохраняются.
