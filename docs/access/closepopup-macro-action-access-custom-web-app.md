# <a name="closepopup-macro-action-access-custom-web-app"></a>ClosePopup Macro Action (пользовательское веб-приложение Access)

С помощью действия **ClosePopup** можно закрыть всплывающее представление активного документа. 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="remarks"></a>Примечания

Если текущая запись во всплывающее представление находится в режиме редактирования, изменения в записи сохраняются автоматически.
