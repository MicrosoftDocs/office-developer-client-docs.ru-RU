# <a name="datediff-function-access-custom-web-app"></a>Функция DateDiff (приложение настраиваемых web Access)

Возвращает количество границ часть определенной даты, пересекающихся указанную начальную и конечную даты.
  
> [!NOTE]
> Компонент хранилища облаке, описанных в этой статье в Office 2013 и Office 2016 больше не поддерживается и может привести следующее сообщение об ошибке: > *к сожалению, мы возникают проблемы с сервера, поэтому мы не удается добавить \<службы\> на данный момент. Повторите попытку позже.* > Для облачного хранилища для Microsoft Office Online, Office для операций ввода-вывода и Office для Android можно найти в нашем [Партнерской программы Office облачных хранилища](https://dev.office.com/programs/officecloudstorage). 
  
## <a name="syntax"></a>Синтаксис

**Функция DateDiff** (*DatePart*, *Дата начала*, *Дата окончания*) 
  
Функция **DateDiff** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *DatePart*  <br/> |Является частью *StartDate* и *EndDate* , которое указывает тип границы пересечение. Обратитесь к "Примечания" для списка допустимых значений.  <br/> |
| *StartDate*  <br/> |Выражение, которое разрешается в значение даты и времени. Аргумент выражение *даты* , выражение столбца, переменной, определенной пользователем или строковый литерал.  <br/> |
| *Дата окончания*  <br/> |Выражение, которое разрешается в значение даты и времени. Аргумент выражение *даты* , выражение столбца, переменной, определенной пользователем или строковый литерал.  <br/> |
