# <a name="max-function-access-custom-web-app"></a>Функция МАКС (приложение настраиваемых web Access)

Возвращает максимальное значение в выражении запроса или в таблице.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

 **Максимальное число** (*Выражение*) 
  
Функция **Max** содержит следующий аргумент. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Выражение*  <br/> |Строковое выражение, определяющее поле, содержащее данные, которые требуется вычислить или выражение, которое выполняет вычисления с использованием данных для этого поля. Операнды в *выражении* может включать имя поля в таблице, константа или функция (это может быть встроенным или определяемым пользователем, но не в одной из других статистические функции SQL).  <br/> |
