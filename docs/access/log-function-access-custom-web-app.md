# <a name="log-function-access-custom-web-app"></a>Функция log (пользовательское веб-приложение для Access)

Возвращает натуральный логарифм или логарифм заданной базы данных Base указанного выражения.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Log** (*Нумерицекспрессион* , [ *base* ]) 
  
Функция **log** содержит указанные ниже аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Нумерицекспрессион*  <br/> |Положительное число, для которого вычисляется логарифм.  <br/> |
| *Base*  <br/> |Основание логарифма. Если этот параметр опущен, функция **log** Возвращает натуральный логарифм.  <br/> |
   
## <a name="remarks"></a>Замечания

Функция **LOG10** похожа, но всегда возвращает общий логарифм, то есть логарифм для базового 10. 
  
Натуральный логарифм — это логарифм базового e, где e — константа ирратионал приблизительно равной 2,718281828.
