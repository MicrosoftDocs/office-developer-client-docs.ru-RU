# <a name="log-function-access-custom-web-app"></a>Log Function (Access custom web app)

Возвращает естественный логарифм или логарифм указанного выражения.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Log** (*NumericExpression*  , [  *Base*  ]) 
  
Функция **Log** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *NumericExpression*  <br/> |Положительное число, для которого нужно логарифм.  <br/> |
| *Base*  <br/> |Основа логарифма. Если этот опущен, функция **Log** возвращает естественный логарифм.  <br/> |
   
## <a name="remarks"></a>Примечания

Функция **Log10** аналогична, но всегда возвращает общий логарифм, то есть логарифм для базовой 10. 
  
Естественный логарифм — это логарифм в базовой e, где e — это константа-заметь, приблизительно равная 2,718281828.
