# <a name="log-function-access-custom-web-app"></a>Функция log (приложение настраиваемых web Access)

Возвращает натуральный логарифм или логарифм базовой, указанного выражения.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Журнал** (*NumericExpression* , [ *Base* ]) 
  
Функция **Log** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *NumericExpression*  <br/> |Положительное число, для которого вы хотите логарифм.  <br/> |
| *Base*  <br/> |Основание логарифма. Если этот параметр опущен, функция **Log** Возвращает натуральный логарифм.  <br/> |
   
## <a name="remarks"></a>Замечания

Функция **Log10** аналогичен, но всегда возвращает десятичного логарифма, то есть логарифма для основанию 10. 
  
Натуральный логарифм — это логарифм с основанием e, где e является константой irrational примерно равно 2.718281828.
