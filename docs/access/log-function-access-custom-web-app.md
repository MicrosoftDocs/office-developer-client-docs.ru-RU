# <a name="log-function-access-custom-web-app"></a>Функция журнала (Пользовательский доступ к веб-приложению)

Возвращает естественный логарифм или логарифм указанного выражения.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Журнал** *(NumericExpression*  , [  *База*  ]) 
  
Функция **Журнала** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *NumericExpression*  <br/> |Положительное число, для которого нужен логарифм.  <br/> |
| *Base*  <br/> |База логарифма. При опущении функция **Log** возвращает естественный логарифм.  <br/> |
   
## <a name="remarks"></a>Примечания

Функция **Log10** аналогична, но всегда возвращает общий логарифм, то есть логарифм для базы 10. 
  
Естественный логарифм — это логарифм базового e, где e — иррациональная константа, примерно равная 2.718281828.
