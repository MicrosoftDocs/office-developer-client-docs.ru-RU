# <a name="custom-date-and-time-formats-for-the-format-function-access-custom-web-app"></a>Форматы настраиваемые даты и времени для функции Format (приложение настраиваемых web Access)

Узнайте, как управлять отображением даты или времени путем создания пользовательского форматирования.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="format-specifications"></a>Спецификации формата

В следующей таблице приведены знаки, которые можно использовать с помощью функции [Функция Format (приложение настраиваемых web Access)](format-function-access-custom-web-app.md) для создания настраиваемых даты и времени. 
  
|**Спецификации формата**|**Описание**|
|:-----|:-----|
|(:)  <br/> |Разделитель времени. В некоторых языковых стандартов для представления разделителя времени могут использоваться другие символы. Временной разделитель отделяет часы, минуты и секунды при форматировании значений времени. Фактический знак, используемый в качестве разделителя времени в выходных данных форматирования, определяется значение текущего языка и региональных параметров приложения.  <br/> |
|(/)  <br/> |Разделитель даты. В некоторых языковых стандартов могут использоваться другие символы разделителя даты. Разделитель даты отделяет день, месяц и год при форматировании значений даты. Фактический знак, используемый в качестве разделителя даты в выходных данных форматирования, определяется текущий язык и региональные параметры приложения.  <br/> |
|(%)  <br/> |Служит для указания, что следующие знак должен читаться как формат одной буквы безотносительно все конечные буквы. Также используется для указания, что формат одной буквы считываются как определяемый пользователем формат. В разделе ниже для получения дополнительной информации.  <br/> |
|d  <br/> |Отображает день в виде числа без нуля в начале (например, 1). Используйте %d, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|dd  <br/> |Отображает день в виде числа с нулем в начале (например, 01).  <br/> |
|ddd  <br/> |Отображение дня с использованием сокращений (например, Sun).  <br/> |
|dddd  <br/> |Отображает день, как полное имя (например, воскресенье).  <br/> |
|M  <br/> |Отображает месяц в виде числа без нуля в начале (например, январь отображается как 1). Используйте %M, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|ММ  <br/> |Отображает месяц в виде числа с нулем в начале (например, 01/12/01).  <br/> |
|МММ  <br/> |Отображает месяц с использованием сокращений (например, янв).  <br/> |
|ММММ  <br/> |Отображает полное название месяца (например, январь).  <br/> |
|GG  <br/> |Отображает строку эры (например, A.D.).  <br/> |
|h  <br/> |Отображает час в виде числа без нуля в начале в 12-часовом формате (например, 1:15:15 PM). Используйте %h, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|hh  <br/> |Отображает час в виде числа с нулем в 12-часовом формате (например, 01:15:15 PM).  <br/> |
|H  <br/> |Отображает час в виде числа без нуля в начале 24-часовом формате (например, 1:15:15). Используйте %H, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|HH  <br/> |Отображает час в виде числа с нулем в 24-часовом формате (например, 01:15:15).  <br/> |
|m  <br/> |Отображает минуты в виде числа без нуля в начале (например, 12:1:15). Используйте %m, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|mm  <br/> |Отображает минуты в виде числа с нулем (например, 12:01:15).  <br/> |
|s  <br/> |Отображает секунды в виде числа без нуля в начале (например, 12:15:5). Используйте %s, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|ss  <br/> |Отображает секунды в виде числа с нулем (например, 12:15:05).  <br/> |
|f  <br/> |Отображает доли секунды. Например ff отображает сотые доли секунды, а ffff — десятитысячные времени в секундах. В формате пользовательских может использовать до семи символов f. Используйте %f, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|t  <br/> |Используется 12-часовом формат и отображается A в верхнем регистре для любого часа до полудня; Отображает P в верхнем регистре для любым часом между полуднем и 11:59 P.M. Используйте %t, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|tt  <br/> |Для региональных стандартов, использующих 12-часовом формате отображает AM в верхнем регистре с любым часом до полудня; Отображает PM в верхнем регистре с любым часом между полуднем и 11:59 P.M.  <br/> Для языков, использующих 24-часовом формате, ничего не выводит.  <br/> |
|y  <br/> |Отображает число года (0-9) без нуля в начале. Используйте %y, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|yy  <br/> |Отображает год в двух цифр числовой формат с нулем в начале, если это возможно.  <br/> |
|yyy  <br/> |Отображение года в четырех цифр числовой формат.  <br/> |
|yyyy  <br/> |Отображение года в четырех цифр числовой формат.  <br/> |
||Отображает смещение часового пояса без нуля в начале (например, -8). Используйте %z, если это единственный знак в определяемый пользователем числовой формат.  <br/> |
|z  <br/> |Отображает смещение часового пояса с нулем в начале (например, -08).  <br/> |
|Яя  <br/> |Отображает смещение часового пояса с нулем в начале (например, -08)  <br/> |
|zzz  <br/> |Отображает полное часовыми поясами (например, -08:00)  <br/> |
   
## <a name="remarks"></a>Замечания

Форматирования строк зависят от регистра символов. Различные форматирование можно получить с помощью различных дела. Например при форматирования значение даты со строки «D» Вы получаете даты в длинный формат (в соответствии с языком). Тем не менее если изменить регистр на «d» вы получаете короткий формат даты. Кроме того непредсказуемым последствиям или ошибка может возникнуть при предполагаемая форматирование не соответствует case любую строку определенного формата.
