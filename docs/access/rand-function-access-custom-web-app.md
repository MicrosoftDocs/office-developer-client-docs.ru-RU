# <a name="rand-function-access-custom-web-app"></a>Функция RAND (пользовательское веб-приложение для Access)

Возвращает псевдо — случайное число от 0 до 1.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **СЛЧИС** ([ *SEED* ]) 
  
Функция **Rand** содержит следующий аргумент. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Seed*  <br/> |Целочисленное выражение, задающее начальное значение. Если параметр *SEED* не указан, начальное значение присваивается случайным образом.  <br/> |
   
## <a name="remarks"></a>Замечания

Повторные вызовы функции **СЛЧИС** с одинаковым начальным значением возвращают одни и те же результаты.
