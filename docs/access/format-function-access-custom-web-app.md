# <a name="format-function-access-custom-web-app"></a>Функция Format (приложение настраиваемых web Access)

Возвращает значение, отформатированное в соответствии с указанному шаблону.
  
> [!NOTE]
> Компонент хранилища облаке, описанных в этой статье в Office 2013 и Office 2016 больше не поддерживается и может привести следующее сообщение об ошибке: > *к сожалению, мы возникают проблемы с сервера, поэтому мы не удается добавить \<службы\> на данный момент. Повторите попытку позже.* > Для облачного хранилища для Microsoft Office Online, Office для операций ввода-вывода и Office для Android можно найти в нашем [Партнерской программы Office облачных хранилища](https://dev.office.com/programs/officecloudstorage). 
  
## <a name="syntax"></a>Синтаксис

 **Формат** (*Выражение*, *Формат*) 
  
Функция **Format** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Выражение*  <br/> |Выражение поддерживаемый тип данных для форматирования.  <br/> |
| *Формат*  <br/> | Шаблон формата. Аргумент format должен содержать строку допустимый формат, как строка стандартного формата (например, «C» или «D») или образец настраиваемых символов для дат и числовых значений (например, «мммм дд, гггг (dddd)»). Для получения дополнительных сведений см.  <br/> |
   
## <a name="remarks"></a>Замечания

Для параметра *Format* вы можете передать строк, соответствующих любой из следующих шаблонов: 
  
- [Строки стандартного числового формата](https://msdn.microsoft.com/library/dwhawy9k%28v=vs.110%29.aspx)
    
- [Строки настраиваемых числовых форматов](https://msdn.microsoft.com/library/0c899ak8%28v=vs.110%29.aspx)
    
- [Стандартный формат даты и времени форматирования строк](https://msdn.microsoft.com/library/az4se3k1%28v=vs.110%29.aspx)
    
- [Настраиваемые даты и времени форматирования строк](https://msdn.microsoft.com/library/8kb3ddd4%28v=vs.110%29.aspx)
    
Функция **Format** нельзя использовать в следующих областях веб-приложений Access 2013: 
  
- Вычисляемых столбцов на уровне таблицы
    
- Макросы пользовательского интерфейса
    
- Выражения в представлениях (например, в формах)
