# <a name="and-access-custom-web-app"></a>И (пользовательское веб-приложение для Access)

Объединяет два логических выражения и возвращает значение TRUE, если оба выражения имеют значение TRUE. 
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 *булеан_експрессион* **А** *булеан_експрессион* 
  
## <a name="return-value"></a>Возвращаемое значение

Возвращает значение TRUE, если оба выражения имеют значение TRUE.
  
## <a name="remarks"></a>Замечания

Если в операторе используется несколько логических операторов, операторы AND оцениваются первыми. Порядок оценки можно изменить с помощью круглых скобок.
