# <a name="group-macro-block-access-custom-web-app"></a>Блок группы макросов (приложение настраиваемых web Access)

Оператор **группы** позволяет указать блок действия макроса, который можно развернуть или свернуть. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="setting"></a>Параметр

Действие **группы** имеет следующие аргументы. 
  
|**Аргумент**|**Обязательное**|**Описание**|
|:-----|:-----|:-----|
|**Описание** <br/> |НЕТ  <br/> |Строка, которая отображается как заголовок группы, когда она свернута.  <br/> |
   
## <a name="remarks"></a>Замечания

Оператор **группы** не определяет область макрос, который может выполняться отдельно.
