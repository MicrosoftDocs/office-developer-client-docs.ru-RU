# <a name="datewithtimefromparts-function-access-custom-web-app"></a>Функция DateWithTimeFromParts (приложение настраиваемых web Access)

Возвращает дату и время в зависимости от указанного год, месяц, день и время.
  
> [!NOTE]
> Компонент хранилища облаке, описанных в этой статье в Office 2013 и Office 2016 больше не поддерживается и может привести следующее сообщение об ошибке: > *к сожалению, мы возникают проблемы с сервера, поэтому мы не удается добавить \<службы\> на данный момент. Повторите попытку позже.* > Для облачного хранилища для Microsoft Office Online, Office для операций ввода-вывода и Office для Android можно найти в нашем [Партнерской программы Office облачных хранилища](https://dev.office.com/programs/officecloudstorage). 
  
## <a name="syntax"></a>Синтаксис

**DateWithTimeFromParts** (*Год*, *месяц*, *день*, *час*, *минута*, *второй*) 
  
Функция **DateWithTimeFromParts** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Год*  <br/> |Выражение целое число, указывающее года.  <br/> |
| *Месяц*  <br/> |Выражение целое число, указывающее месяц.  <br/> |
| *День*  <br/> |Выражение целое число, указывающее день.  <br/> |
| *Час*  <br/> |Выражение целое число, указывающее часов.  <br/> |
| *Минуты*  <br/> |Выражение целое число, указывающее минут.  <br/> |
| *Секунды*  <br/> |Выражение целое число, указывающее секунд.  <br/> |
   
## <a name="remarks"></a>Замечания

**DateWithTimeFromParts** возвращает полностью инициализации значение даты и времени. Если аргументы не допускаются, возникает ошибка. При необходимости, что аргументы имеют значение Null, возвращается значение Null.
