# <a name="custom-numeric-formats-for-the-format-function-access-custom-web-app"></a>Настраиваемые числовые форматы для функции Format (приложение настраиваемых web Access)

Узнайте, как для управления отображением номер, создав определяемый пользователем числовой формат.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 

Можно изменить способ отображения числа, создав определяемый пользователем числовой формат. Определяемый пользователем числовой формат может иметь от одного до трех разделов, разделенных точкой с запятой (;). Если аргумент стиля функции [Функция Format (приложение настраиваемых web Access)](format-function-access-custom-web-app.md) содержит один из предварительно определенные числовые форматы, разрешен только один раздел. 
  
## <a name="format-specifications"></a>Спецификации формата
<a name="bk_addresources"> </a>

В следующей таблице приведены знаки, которые можно использовать для создания пользовательских числовых форматов.
  
|**Спецификации формата**|**Описание**|
|:-----|:-----|
|Нет  <br/> |Отображает номер без форматирования.  <br/> |
|**0** (ноль знаков)  <br/> |Шаблон цифры. Отображает цифру или ноль. Если выражение цифры в том месте, где отображается ноль в строку формата, отображается цифры; в противном случае отображается ноль в этой позиции.  <br/> Если количество цифр меньше, чем нулей (по обеим сторонам от запятой) в выражении формата отображает начальных и конечных нулей. Если число имеет больше цифр справа от разделителя целой и дробной части, чем нулей справа от десятичного разделителя в выражении формата округление до количества десятичных разрядов, существует являются нулям. Если число имеет больше цифр слева от десятичного разделителя, чем нулей слева от десятичного разделителя в выражение формата отображает дополнительные цифры без изменений.  <br/> |
|#  <br/> |Шаблон цифры. Отображает цифру или ничего. Если выражение цифры в том месте, где отображается символ # в строку формата, отображается цифры; в противном случае ничего не выводит в этой позиции.  <br/> Этот символ действует так же, как заменитель цифры 0, за исключением того, что начальные и конечные нули не отображаются, если количество цифр меньше, чем знаков # по обеим сторонам от десятичного разделителя в выражение формата.  <br/> |
|. (символа точки)  <br/> |Десятичный разделитель. Десятичный разделитель определяет количество цифр, отображаемых слева и справа от разделителя целой и дробной части. Если выражение формата содержит только знаки # слева от данного символа; числа меньше 1 начинаются с десятичного разделителя. Чтобы отобразить нуля в начале перед дробными числами, используйте ноль как первый заменитель цифры слева от разделителя целой и дробной части. В некоторых языковых стандартов запятой используется в качестве десятичного разделителя. Символ, который используется в качестве десятичного заполнителя в выходных данных форматирования, зависит от числовой формат, обнаруженные в системе. Таким образом вам следует использовать период как десятичный разделитель в собственных форматах даже в том случае, если вы используете языкового стандарта, использующий запятую в качестве десятичного заполнителя. Отформатированная строка будет отображаться в формате для языкового стандарта.  <br/> |
|%  <br/> |Заменитель процентов. Умножает выражение на 100. Знак процента (%) вставляется в том месте, где он находится в строку формата.  <br/> |
|, (запятой)  <br/> |Знака разделителя групп разрядов. Разделитель тысяч отделяет тысячи от сотен в числе с четырьмя или более разрядами слева от разделителя целой и дробной части. Указан стандартного использования знака разделителя групп разрядов, если формат содержит разделитель заключается в цифровые заполнители (0 или #).  <br/> Разделитель непосредственно слева от десятичного разделителя (ли указан тип decimal) или правым символов в строке, это означает «масштабировать номер, разделив 1000, округления при необходимости.» Меньше 1 000, но больше или равно 500 числа отображаются как 1, а числа меньше 500 отображаются как 0. Два смежных разделителя групп разрядов в этой позиции масштабирования с коэффициентом 1 миллиона и еще раз 1 000 для каждого последующего разделителя.  <br/> Несколько разделителей слева от разделителя целой и дробной части или не крайней правой позиции в строке обрабатываются только как разделитель разрядов. В некоторых языковых стандартов за период используется как десятичный разделитель. Символ, который используется в качестве разделителя групп разрядов в выходных данных форматирования, зависит от числовой формат, определяется системой. Таким образом вам следует использовать запятую как знака разделителя групп разрядов в собственных форматах даже в том случае, если вы используете языкового стандарта, который используется точка как десятичный разделитель. Отформатированная строка будет отображаться в формате для языкового стандарта.  <br/> Например рассмотрим следующие три строки форматирования:  <br/> «#, 0.», которые использует разделители тысяч для форматирования числа 100 миллионов в строку «100 000 000».  <br/> «#0.», которая использует деление на тысячу для форматирования числа 100 миллионов в строку «100000».  <br/> «#, 0,.», которые использует разделитель тысяч и деление на тысячу для форматирования числа 100 миллионов в строку «100000».  <br/> |
|: (двоеточие)  <br/> |Разделитель времени. В некоторых языковых стандартов для представления разделителя времени могут использоваться другие символы. Временной разделитель отделяет часы, минуты и секунды при форматировании значений времени. Фактический знак, используемый в качестве разделителя времени в выходных данных форматирования, определяется параметрами системы.  <br/> |
|/ (символ косой черты)  <br/> |Разделитель даты. В некоторых языковых стандартов могут использоваться другие символы разделителя даты. Разделитель даты отделяет день, месяц и год при форматировании значений даты. Фактический знак, используемый в качестве разделителя даты в выходных данных форматирования, определяется параметрами системы.  <br/> |
|**E-E +, e-, e +** <br/> |Инженерный формат. Если выражение формата содержит по крайней мере один шаблон цифры (0 или #) слева от E-, E +, e- или e + номер отображается в форматов и E или e вставляется между числом и его степенью. Количество цифр слева определяет число цифр в степени. Чтобы поместить знак минуса рядом с отрицательными показателями используйте E - или e. Чтобы поместить знак минуса рядом с отрицательным степени и плюс положительными используйте E + или e +. Необходимо также включить цифр справа от данного символа для достижения правильное форматирование.  <br/> |
|**- + $ ( )** <br/> |Буквенные символы. Эти символы отображаются так же, как в строку формата. Чтобы отобразить символ, отличающегося от перечисленных, поставьте перед ним обратную косую черту (\) или заключите его в двойные кавычки (» «).  <br/> |
|\ (обратной косой черты)  <br/> |Отображает следующий символ в строку формата. Чтобы отобразить символ, который имеет особое значение как знак, поставьте перед ним обратную косую черту (\). Обратная косая черта не отображается. С помощью обратной косой черты совпадает с заключения следующего символа в двойные кавычки. Чтобы отобразить обратную косую черту, используйте два обратную косую черту (\\).  <br/> Примеры символов, которые не могут отображаться как литералы и символы форматирование даты и времени (a, c, d, h, m, n, p, q, s, t, w, y, / / и:), символы форматирования чисел (# 0, %, E, e, разделители и период) и форматирования строк символы (@, &amp;, \<, \>, и!).  <br/> |
|«ABC»  <br/> |Отображает строку в двойные кавычки (» «). Чтобы включить строку в аргумент стиля кода, необходимо использовать Chr(34) для заключения текста (34 — код знака для кавычки ("«)).  <br/> |
