# <a name="less-than-access-custom-web-app"></a>Less Than (Access custom web app)

Сравнивает два выражения. При сравнении ненулних выражений результат имеет значение TRUE, если значение левого операнда меньше, чем у правого операнда; в противном случае результатом будет FALSE.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

`< (Less Than)`

*expression*  \<  *expression* 
  
*expression* — любое допустимое выражение. Оба выражения должны содержать неявно преобразуемые типы данных. Преобразование зависит от правил приоритета для типов данных.
