# <a name="concat-function-access-custom-web-app"></a>Функция Concat (пользовательское веб-приложение для Access)

Возвращает строку, которая является результатом объединения двух или более строковых значений.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

**Concat** (*Значение1*, *Значение2*,... [*Валуен*]) 
  
Функция **concat** содержит следующие аргументы: 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
|Значение  <br/> |Строковое значение для сцепления с другими значениями.  <br/> |
   
## <a name="remarks"></a>Примечания

**Сцепление** принимает переменное число строковых аргументов и объединяет их в одну строку. Необходимо указать не менее двух строковых аргументов; в противном случае возникает ошибка. 
  
Все аргументы неявно преобразуются в типы данных String, а затем сцепляются.
  
## <a name="example"></a>Пример

Следующее выражение можно использовать для отображения полного имени пользователя, где в таблице содержатся поля FirstName, Миддлеинитиал и LastName. Если поле Миддлеинитиал не заполнено, только поля FirstName и LastName будут объединены для отображения полного имени.
  
```vb
IIf([MiddleInitial] Is Null,Concat([FirstName]," ",[LastName]),Concat([FirstName]," ",[MiddleInitial]," ",[LastName]))
```
