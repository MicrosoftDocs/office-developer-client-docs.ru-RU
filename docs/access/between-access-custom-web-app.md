# <a name="between-access-custom-web-app"></a>МЕЖДУ (пользовательское веб-приложение для Access)

Указывает диапазон для проверки.
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 *тест_експрессион*  НЕ **Между** *бегин_експрессион* **А** *енд_експрессион* 
  
Оператор **between** содержит следующие аргументы. 
  
|**Аргумент**|**Обязательный**|**Описание**|
|:-----|:-----|:-----|
| *тест_експрессион*  <br/> |Да  <br/> |Выражение для проверки в диапазоне, определенном с помощью *бегин_експрессион* и *енд_експрессион* . Должны иметь тот же тип данных, что и *бегин_експрессион* , и *енд_експрессион* .  <br/> |
| *NOT*  <br/> |Нет  <br/> |Указывает, что результат предиката должен быть инвертирован.  <br/> |
| *бегин_експрессион*  <br/> |Да  <br/> |Допустимое выражение. Должны иметь тот же тип данных, что и *тест_експрессион* , и *енд_експрессион* .  <br/> |
| *енд_експрессион*  <br/> |Да  <br/> |Допустимое выражение. Должны иметь тот же тип данных, что и *тест_експрессион* , и *бегин_експрессион* .  <br/> |
| *AND*  <br/> |Да  <br/> |Указывает, что *тест_експрессион* должен находиться в диапазоне, указанном в *бегин_експрессион* и *енд_експрессион* .  <br/> |
   
## <a name="result-type"></a>Тип результата

 **Boolean**
  
## <a name="remarks"></a>Примечания

 **** Возвращает **значение true** , если значение *тест_експрессион* больше или равно значению *бегин_експрессион* и меньше или равно значению *енд_експрессион* . 
  
 **NOT BETWEEN** возвращает **true** , если значение *тест_експрессион* меньше, чем значение *бегин_експрессион* , или больше, чем значение *енд_експрессион* . 
  
Чтобы указать монопольный диапазон, используйте операторы больше (\>) и меньше, чем (\<).
