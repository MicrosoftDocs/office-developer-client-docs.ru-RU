# <a name="foreachrecord-data-block-access-custom-web-app"></a>Блок данных ДляКаждойЗаписи (пользовательское веб-приложение для Access)

Блок данных **ДляКаждойЗаписи** повторяет набор операторов для каждой записи в домене. 
  
> [!IMPORTANT]
> Корпорация Майкрософт больше не рекомендует создавать и использовать веб-приложения для Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/), чтобы создавать бизнес-решения без кода для Интернета и мобильных устройств. 
  
> [!NOTE]
> Блок данных **ДляКаждойЗаписи** доступен только в макросах данных. 
  
## <a name="setting"></a>Параметр

Макрокоманда **ДляКаждойЗаписи** имеет следующие аргументы. 
  
|**Имя аргумента**|**Обязательный**|**Описание**|
|:-----|:-----|:-----|
|**Возврата** <br/> |Да  <br/> |Строка, определяющая домен записей, с которыми выполняется работа. Аргумент *in* может содержать имя таблицы, запрос на выборку или инструкцию SQL.  <br/> **Note**: указанный домен не может содержать данные, хранящиеся в связанной таблице или источнике данных ODBC.           |
|**Условие отбора** <br/> |Нет  <br/> |Строковое выражение, используемое для ограничения диапазона данных, в котором выполняется блок данных блока **ДляКаждойЗаписи** . Например, критерии часто эквивалентны предложению WHERE в выражении SQL без слова WHERE. Если критерии опущены, блок данных **ДляКаждойЗаписи** работает на всем домене, указанном аргументом *in* . Все поля, включенные в критерии, также должны быть полями в ** .  <br/> |
|**Alias** <br/> |Нет  <br/> |Строка, предоставляющая альтернативное имя для домена, указанного аргументом *in* . Часто используется для сокращения имени таблицы для последующих ссылок, чтобы предотвратить возможные неоднозначные ссылки. Если параметр *Alias* не указан, имя таблицы или запроса будет использоваться в качестве псевдонима.  <br/> |
   
## <a name="remarks"></a>Замечания

Используйте действие **[екситфореачрекорд](exitforeachrecord-macro-action-access-custom-web-app.md)** для немедленного выхода из блока данных **ДляКаждойЗаписи** .
