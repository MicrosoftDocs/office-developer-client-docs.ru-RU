# <a name="setvariable-macro-action-access-custom-web-app"></a>SetVariable Macro Action (Access custom web app)

Действие **SetVariable можно использовать** для создания временной переменной и ее значения. Затем переменная может использоваться в качестве условия или аргумента в последующих действиях, или вы можете использовать переменную в другом макрос пользовательского интерфейса (пользовательского интерфейса). 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="setting"></a>Параметр

Действие **SetVariable** имеет следующие аргументы. 
  
|**Аргумент макрокоманды**|**Описание**|
|:-----|:-----|
|**Переменная** <br/> |Введите имя временной переменной.  <br/> |
|**Значение =** <br/> |Введите выражение, которое будет использоваться для заданной временной переменной. Не предшествуй выражению с равным **=** () знаком. Для этого аргумента ![](media/buildbut_ZA06047218.gif "") можно нажать кнопку **Формула** сборки, чтобы использовать конструктор выражений.  <br/> |
