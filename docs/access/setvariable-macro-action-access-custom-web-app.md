# <a name="setvariable-macro-action-access-custom-web-app"></a>Действия макроса SetVariable (приложение настраиваемых web Access)

Чтобы создать временную переменную и присвойте ей значение определенного значения можно использовать действие **SetVariable** . Переменная затем использовать как условие или аргумент в последующие действия или в другой макрос пользовательского интерфейса пользователя можно использовать переменную. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="setting"></a>Параметр

Действие **SetVariable** состоит из следующих аргументов. 
  
|**Аргумент действия**|**Описание**|
|:-----|:-----|
|**Переменная** <br/> |Введите имя временную переменную.  <br/> |
|**Значение =** <br/> |Введите выражение, которое будет использоваться для задания значения для этой временной переменной. Перед выражением равенства (**=** ) входа. Можно нажмите кнопку **Построить** ![формулы](media/buildbut_ZA06047218.gif "формулы") для задания аргумента с помощью построителя выражений.  <br/> |
