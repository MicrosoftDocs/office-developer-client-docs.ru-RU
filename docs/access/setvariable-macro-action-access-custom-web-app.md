# <a name="setvariable-macro-action-access-custom-web-app"></a>Макрокоманда Сетвариабле (пользовательское веб-приложение для Access)

Вы можете использовать действие **сетвариабле** , чтобы создать временную переменную и присвоить ей определенное значение. Переменную можно использовать в качестве условия или аргумента в последующих действиях, или можно использовать переменную в другом макросе пользовательского интерфейса. 
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="setting"></a>Параметр

Макрокоманда **сетвариабле** имеет следующие аргументы. 
  
|**Аргумент макрокоманды**|**Описание**|
|:-----|:-----|
|**Переменная** <br/> |Введите имя временной переменной.  <br/> |
|**Значение =** <br/> |Введите выражение, которое будет использоваться для задания значения для этой временной переменной. Не ставьте перед выражением знак равенства (**=** ). Чтобы использовать построитель выражений для задания этого аргумента, можно нажать(media/buildbut_ZA06047218.gif "формулу") ![с формулой для кнопки] **построить** .  <br/> |
