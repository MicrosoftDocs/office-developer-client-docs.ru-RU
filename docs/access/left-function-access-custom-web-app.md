# <a name="left-function-access-custom-web-app"></a>Левая функция (доступ к пользовательскому веб-приложению)

Возвращает левую часть строки символов с заданным количеством символов.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

 **Слева** *(TextExpression*, *Длина*) 
  
**Левая** функция содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *TextExpression*  <br/> |Текстовое выражение.  <br/> |
| *Length*  <br/> |Это положительный integer, который указывает, сколько символов  *TextExpression*  возвращаются.  <br/> |
