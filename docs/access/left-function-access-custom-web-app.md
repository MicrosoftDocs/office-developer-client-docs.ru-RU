# <a name="left-function-access-custom-web-app"></a>Функции LEFT (приложение настраиваемых web Access)

Возвращает строку знаков с указанное число символов в левой части.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

 **Слева** (*TextExpression*, *Длина*) 
  
Функции **Left** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *TextExpression*  <br/> |Выражение текста.  <br/> |
| *Длина*  <br/> |— Это положительное целое число, указывающее количество знаков *TextExpression* возвращаются.  <br/> |
