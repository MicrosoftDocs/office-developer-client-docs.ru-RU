# <a name="count-function-access-custom-web-app"></a>Функция Count (приложение настраиваемых web Access)

Возвращает число записей в запросе или в таблице.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="syntax"></a>Синтаксис

**Count** (*Выражение*) 
  
Функция **Count** содержит следующий аргумент. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *Выражение*  <br/> |Строковое выражение, определяющее поле, содержащее данные необходимо число или выражение, которое выполняет вычисления с использованием данных в поле. Операнды в *выражении* может включать имя поля в таблице или функция (это может быть встроенным или определяемым пользователем, но не другие SQL статистические функции). Можно подсчитать любой тип данных, включая текст.  <br/> |
   
## <a name="remarks"></a>Замечания

Count можно использовать для определения числа записей в запросе. Count можно использовать, например, чтобы подсчитать количество заказов, отправленных в определенной стране или регионе.
  
**Count** (\*) возвращает число элементов в группе. Этот компонент включает значения NULL и повторяющиеся.
