# <a name="not-access-custom-web-app"></a>NOT (Access custom web app)

Отменяет входные данные boolean.
  
> [!IMPORTANT]
> Корпорация Майкрософт в настоящее время не рекомендует создавать и использовать веб-приложения Access в SharePoint. В качестве альтернативы можно использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для создания бизнес-решений без кода для Интернета и мобильных устройств. 
  
## <a name="syntax"></a>Синтаксис

[ **Not** ]  *BooleanExpression* 
  
Оператор **Not** содержит следующие аргументы. 
  
|**Имя аргумента**|**Описание**|
|:-----|:-----|
| *BooleanExpression*  <br/> |Допустимые выражения boolean.  <br/> |
   
## <a name="remarks"></a>Примечания

В следующей таблице показаны результаты сравнения значений TRUE и FALSE с помощью оператора **Not.** 
  
||**Not**|
|:-----|:-----|
|**TRUE** <br/> |Неверно  <br/> |
|**FALSE** <br/> |Верно  <br/> |
