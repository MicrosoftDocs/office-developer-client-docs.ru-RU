# <a name="newrecord-macro-action-access-custom-web-app"></a>Действия макроса NewRecord (приложение настраиваемых web Access)

Создает новую запись в текущем представлении.
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
## <a name="remarks"></a>Замечания

Действие **NewRecord** можно использовать для добавления пустую запись в конце представления, которые можно ввести новые данные. 
  
Действие **NewRecord** эквивалентно в области действия, используя кнопку **Добавить** для добавления записи.
