# <a name="setlocalvar-macro-action-access-custom-web-app"></a>Макрос ЗадатьЛокПеременную (приложение настраиваемых web Access)

**ЗадатьЛокПеременную** создает временную переменную и присвойте ей значение определенного значения. 
  
> [!IMPORTANT]
> Корпорация Майкрософт рекомендует больше не Создание и использование веб-приложениях Access в SharePoint. Кроме того рекомендуется использовать [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) для построения без написания кода бизнес-решений для мобильных устройств и веб. 
  
> [!NOTE]
> **ЗадатьЛокПеременную** доступна только в макросов данных. 
  
## <a name="setting"></a>Параметр

**ЗадатьЛокПеременную** имеет следующие аргументы. 
  
|**Имя аргумента**|**Обязательное**|**Описание**|
|:-----|:-----|:-----|
|**Name** <br/> |Да  <br/> |Строка, задающая имя переменной.  <br/> |
|**Выражение** <br/> |Да  <br/> |Выражение, которое будет использоваться для задания значения для этой временной переменной. Перед выражением знак равенства (=). Можно нажмите кнопку **Построить** для задания аргумента с помощью **Построителя выражений** .  <br/> |
