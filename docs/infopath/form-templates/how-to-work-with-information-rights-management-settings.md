### <a name="example"></a><span data-ttu-id="845c9-214">Пример</span><span class="sxs-lookup"><span data-stu-id="845c9-214">Example</span></span>

<span data-ttu-id="845c9-215">В следующем примере нажатие элемента управления **кнопка** возвращает **объект UserPermissionsCollection** для текущей формы, добавляет и назначает пользователь изменить уровень доступа и задает дату окончания срока действия двух дней с текущей даты.</span><span class="sxs-lookup"><span data-stu-id="845c9-215">In the following example, clicking the **Button** control gets the **UserPermissionsCollection** for the current form, adds and assigns a user to the Change access level, and sets an expiration date of two days from the current date.</span></span> 
  
```cs
public void CTRL1_Clicked(object sender, ClickedEventArgs e)
{
   string strExpirationDate = DateTime.Today.AddDays(2).ToString();
   DateTime dtExpirationDate = DateTime.Parse(strExpirationDate);
   this.Permission.UserPermissions.Add("someone@example.com", 
      PermissionType.Change, dtExpirationDate);
}
```
