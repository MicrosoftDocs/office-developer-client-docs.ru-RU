### <a name="domain"></a><span data-ttu-id="286e0-129">Домен</span><span class="sxs-lookup"><span data-stu-id="286e0-129">Domain</span></span>

### <a name="full-trust"></a><span data-ttu-id="286e0-133">Полное доверие</span><span class="sxs-lookup"><span data-stu-id="286e0-133">Full Trust</span></span>

Этот параметр позволяет форме обращаться к вызовам объектной модели, например к функции сохранения файлов; кроме того, отключаются некоторые запросы безопасности, которые выводятся при запуске с более ограниченным уровнем безопасности.</span><span class="sxs-lookup"><span data-stu-id="286e0-p104">The Full Trust security level allows you to run a form with full trust on the computer where the form will be used. Both methods require setting the **requireFullTrust** attribute to "yes".
