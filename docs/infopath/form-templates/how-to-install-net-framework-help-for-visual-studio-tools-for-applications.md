# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a>Установка локальной справки .NET Framework для набора средств Visual Studio Tools для работы с приложениями

Если компьютер, на котором вы работаете Microsoft InfoPath и Visual Studio 2012, подключен к Интернету, Visual Studio 2012 г. может получить доступ к документации платформа .NET Framework и ссылки на Visual Basic и visual C# из интернет-источников. Однако, если компьютер, с который вы работаете, не подключен к Интернету, вы не сможете получить доступ к этим источникам документации, пока не загрузите и не установите локализованную копию документации платформа .NET Framework 2.0 SDK и не выполните действия, необходимые для интеграции этой документации с средой разработки Visual Studio 2012 г.
  
Чтобы просмотреть раздел справки, который описывает,  как это сделать, откройте области содержимого справки, расширьте **Visual Studio** для приложений, расширьте начало работы с интегрированной средой разработки, а затем нажмите кнопку Интеграция документации платформа .NET Framework **SDK** с интегрированной средой разработки .
  
> [!IMPORTANT]
> Если вы работаете в Windows Vista с управлением учетной записью пользователя, перед выполнением вышеуказанной процедуры необходимо запустить InfoPath в качестве администратора. Чтобы выполнить эту операцию, перейдите в папку C:\Program Files\Microsoft Office\Office14, щелкните правой кнопкой мыши файл INFOPATH.EXE, а затем щелкните **Запуск от имени администратора**. Если запуск InfoPath осуществляется не от имени администратора, эту процедуру выполнить не удастся.
