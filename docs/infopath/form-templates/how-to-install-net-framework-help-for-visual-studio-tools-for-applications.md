# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a>Установка локальной справки .NET Framework для набора средств Visual Studio Tools для работы с приложениями

Когда компьютер, на котором запущены Microsoft InfoPath и Visual Studio 2012, подключен к Интернету, программа просмотра справки Visual Studio 2012 может получить доступ к документации .NET Framework и ссылкам на язык Visual Basic и Visual C# из веб-источников. Однако если компьютер, с который вы работаете, не подключен к Интернету, вы не сможете получить доступ к этим источникам документации, пока не загрузите и не установите локализованную копию документации SDK .NET Framework 2.0 и не выполните действия, необходимые для интеграции этой документации со средой разработки Visual Studio 2012.
  
Чтобы просмотреть раздел справки, в которой описано, как это сделать, откройте раздел справки "Содержимое", разойдите **Visual Studio** для приложений, разойдите раздел "Начало работы со интегрированной средой разработки" и выберите "Интеграция документации  **SDK .NET Framework** со встроенной средой разработки". 
  
> [!IMPORTANT]
> Если вы работаете под управлением Windows Vista с управлением учетной записью пользователя, необходимо запустить InfoPath от качестве администратора, прежде чем выполнять процедуру, на которую ссылается выше. Чтобы выполнить эту операцию, перейдите в папку C:\Program Files\Microsoft Office\Office14, щелкните правой кнопкой мыши файл INFOPATH.EXE, а затем щелкните **Запуск от имени администратора**. Если запуск InfoPath осуществляется не от имени администратора, эту процедуру выполнить не удастся.
