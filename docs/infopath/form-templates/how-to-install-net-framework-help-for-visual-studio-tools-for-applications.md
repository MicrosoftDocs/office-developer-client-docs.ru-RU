# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a>Установка локального справки .NET Framework для Visual Studio Tools для приложений

Когда компьютер, на котором запущены приложения Microsoft InfoPath и набор Visual Studio 2012, подключен к Интернету, средство просмотра справки набора Visual Studio 2012 может получать доступ к интерактивным источникам документации .NET Framework и справочных материалов по языкам Visual Basic и Visual C#. Однако когда рабочий компьютер не подключен к Интернету, доступ к этим источникам документации отсутствует. Чтобы использовать их автономно, следует загрузить и установить локальную копию документации .NET Framework 2.0 SDK и выполнить действия, необходимые для интеграции этой документации с набором Visual Studio 2012.
  
Чтобы прочесть справочную тему, описывающую данную процедуру, откройте вкладку справки **Содержимое**, разверните последовательно пункты **Visual Studio для приложений** и **Введение в интегрированную среду разработки**, а затем щелкните **Практическое руководство. Интеграция документации .NET Framework SDK с интегрированной средой разработки**.
  
> [!IMPORTANT]
> [!Важно!] Если работа осуществляется в системе Windows Vista с контролем учетных записей пользователей, необходимо запустить InfoPath от имени администратора, прежде чем выполнять указанную выше процедуру. Чтобы выполнить эту операцию, перейдите в папку C:\Program Files\Microsoft Office\Office14, щелкните правой кнопкой мыши файл INFOPATH.EXE, а затем щелкните **Запуск от имени администратора**. Если запуск InfoPath осуществляется не от имени администратора, эту процедуру выполнить не удастся.
