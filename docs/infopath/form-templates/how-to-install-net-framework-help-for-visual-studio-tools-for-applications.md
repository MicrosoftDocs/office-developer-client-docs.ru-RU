# <a name="install-local-net-framework-help-for-visual-studio-tools-for-applications"></a><span data-ttu-id="b48f2-104">Установка локального справки .NET Framework для Visual Studio Tools для приложений</span><span class="sxs-lookup"><span data-stu-id="b48f2-104">Install local .NET Framework Help for Visual Studio Tools for Applications</span></span>

<span data-ttu-id="b48f2-p101">Когда компьютер, на котором запущены приложения Microsoft InfoPath и набор Visual Studio 2012, подключен к Интернету, средство просмотра справки набора Visual Studio 2012 может получать доступ к интерактивным источникам документации .NET Framework и справочных материалов по языкам Visual Basic и Visual C#. Однако когда рабочий компьютер не подключен к Интернету, доступ к этим источникам документации отсутствует. Чтобы использовать их автономно, следует загрузить и установить локальную копию документации .NET Framework 2.0 SDK и выполнить действия, необходимые для интеграции этой документации с набором Visual Studio 2012.</span><span class="sxs-lookup"><span data-stu-id="b48f2-p101">When the computer on which you are running Microsoft InfoPath and Visual Studio 2012 is connected to the Internet, the Visual Studio 2012 help viewer can access .NET Framework documentation and the Visual Basic and Visual C# language references from online sources. However, when the computer you are working with is not connected to the Internet, you cannot access these sources of documentation until you download and install a local copy of .NET Framework 2.0 SDK documentation and perform the steps required to integrate that documentation with the Visual Studio 2012 development environment.</span></span>
  
<span data-ttu-id="b48f2-107">Чтобы прочесть справочную тему, описывающую данную процедуру, откройте вкладку справки **Содержимое**, разверните последовательно пункты **Visual Studio для приложений** и **Введение в интегрированную среду разработки**, а затем щелкните **Практическое руководство. Интеграция документации .NET Framework SDK с интегрированной средой разработки**.</span><span class="sxs-lookup"><span data-stu-id="b48f2-107">To view a help topic that describes how to do this, open the help **Contents** pane, expand **Visual Studio for Applications**, expand **Getting Started with the Integrated Development Environment**, and then click **How to: Integrate the .NET Framework SDK Documentation with the Integrated Development Environment**.</span></span>
  
> [!IMPORTANT]
> <span data-ttu-id="b48f2-p102">[!Важно!] Если работа осуществляется в системе Windows Vista с контролем учетных записей пользователей, необходимо запустить InfoPath от имени администратора, прежде чем выполнять указанную выше процедуру. Чтобы выполнить эту операцию, перейдите в папку C:\Program Files\Microsoft Office\Office14, щелкните правой кнопкой мыши файл INFOPATH.EXE, а затем щелкните **Запуск от имени администратора**. Если запуск InfoPath осуществляется не от имени администратора, эту процедуру выполнить не удастся.</span><span class="sxs-lookup"><span data-stu-id="b48f2-p102">If you are running under Windows Vista with user account control, you must start InfoPath as an administrator before performing the procedure referenced above. If you are not running InfoPath as an administrator, you will be unable to complete the procedure.</span></span>
