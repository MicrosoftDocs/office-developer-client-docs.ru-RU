<span data-ttu-id="81914-110">Сведения о создании обработчиков событий InfoPath см. в [добавлении обработчик событий.](how-to-add-an-event-handler.md)</span><span class="sxs-lookup"><span data-stu-id="81914-110">For information about creating the InfoPath event handlers, see [Add an Event Handler](how-to-add-an-event-handler.md).</span></span>

<span data-ttu-id="81914-p106">При создании обработчика событий приложение InfoPath создает объявление обработчика событий в коде формы проекта. <span data-ttu-id="81914-229">Например, при создании обработчика  событий для события Загрузка в режиме разработки  (щелкнув меню **Loading Event** на вкладке Разработчик), InfoPath добавляет объявление обработчику событий, получаюшего объект [LoadingEventArgs,](https://msdn.microsoft.com/library/Microsoft.Office.InfoPath.LoadingEventArgs.aspx) в файл кода формы, а затем открывает редактор кода, чтобы можно было добавить код в следующее объявление обработчик событий.</span><span class="sxs-lookup"><span data-stu-id="81914-229">For example, when you create an event handler for the **Loading** event in design mode (by clicking **Loading Event** menu on the **Developer** tab), InfoPath adds the declaration for the event handler that receives the [LoadingEventArgs](https://msdn.microsoft.com/library/Microsoft.Office.InfoPath.LoadingEventArgs.aspx) object to the form code file, and then opens the code editor so that you can add your code to the following event handler declaration.</span></span>
