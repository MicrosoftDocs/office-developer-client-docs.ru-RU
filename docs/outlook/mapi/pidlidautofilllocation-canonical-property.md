# <a name="pidlidautofilllocation-canonical-property"></a>Каноническое свойство PidLidAutoFillLocation

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, что значение свойства **dispidLocation** ([PidLidLocation)](pidlidlocation-canonical-property.md)занося в свойство **PR_DISPLAY_NAME** ([PidTagDisplayName)](pidtagdisplayname-canonical-property.md)объекта RecipientRow, представляюного ресурс.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidAutoFillLocation  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x0000823A  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Дополнительные сведения о RecipientRow см. в протоколе Message и Attachment Object, как указано [в [MS-OXCMSG]](https://msdn.microsoft.com/library/7fd7ec40-deec-4c06-9493-1bc06b349682%28Office.15%29.aspx).
