# <a name="custom-processing-with-tnef"></a>Настраиваемая обработка с помощью TNEF

**Область применения**: Outlook 2013 | Outlook 2016 
  
Поставщики транспорта могут использовать настраиваемую обработку для обработки свойств самого вложения, передачи вложений по отдельности или их передачи через модель вложения системы обмена сообщениями. В формате TNEF используется механизм, который позволяет поставщику транспорта отправлять вложения отдельно от сообщения и повторно подключаться к ним на принимающей стороне.
