# <a name="custom-processing-with-tnef"></a>Настраиваемая обработка с помощью TNEF

**Область применения**: Outlook 2013 | Outlook 2016 
  
Поставщики транспорта могут использовать настраиваемую обработку для обработки свойств на самом вложении, отдельной передачи вложений или передачи их через модель вложений системы обмена сообщениями. TNEF использует механизм, который позволяет поставщику транспорта отправлять вложения отдельно от сообщения и повторно подключить их на стороне приема.
