# <a name="pidtagoriginatorcertificate-canonical-property"></a>Каноническое свойство PidTagOriginatorCertificate

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит сертификат ASN.1 для источника сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINATOR_CERTIFICATE  <br/> |
|Идентификатор:  <br/> |0x0022  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является копией свойства  PR_USER_CERTIFICATE[(PidTagUserCertificate).](pidtagusercertificate-canonical-property.md)
