# <a name="pidtagassistant-canonical-property"></a>Каноническое свойство PidTagAssistant

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит имя помощника по администрированию получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_АССИСТАНТ, ПР_АССИСТАНТ_А, ПР_АССИСТАНТ_В  <br/> |
|Идентификатор:  <br/> |0x3A30  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Комментарии

Эти свойства предоставляют сведения о идентификации и доступе для получателя. Они определяются получателем и их организацией.
