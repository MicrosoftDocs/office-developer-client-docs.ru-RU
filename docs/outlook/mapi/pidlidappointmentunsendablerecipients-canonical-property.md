# <a name="pidlidappointmentunsendablerecipients-canonical-property"></a>PidLidAppointmentUnsendableRecipients Canonical Property

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит список неиссякаемых участников.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptUnsendableRecips  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x0000823D  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство не требуется, но должно быть заданной. Его формат подробно описан [в [MS-OXOCAL].](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx)
