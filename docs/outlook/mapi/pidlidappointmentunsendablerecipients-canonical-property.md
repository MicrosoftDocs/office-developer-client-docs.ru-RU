# <a name="pidlidappointmentunsendablerecipients-canonical-property"></a>Каноническое свойство PidLidAppointmentUnsendableRecipients

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит список участников, отправить.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptUnsendableRecips  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000823D  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Meetings (собрания);  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство не требуется, но необходимо задать. Формат описано в [[MS-OXOCAL]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
