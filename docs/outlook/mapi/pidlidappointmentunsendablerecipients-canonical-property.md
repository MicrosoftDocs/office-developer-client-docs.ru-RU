# <a name="pidlidappointmentunsendablerecipients-canonical-property"></a>Каноническое свойство PidLidAppointmentUnsendableRecipients

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит список неотправленных участников.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ДиспидапптунсендаблереЦипс  <br/> |
|Набор свойств:  <br/> |Псетид_аппоинтмент  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000823D  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Комментарии

Это свойство не является обязательным, но должно быть задано. Его формат подробно описывается в [[MS-оксокал]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
