# <a name="pidtagttytddphonenumber-canonical-property"></a>Каноническое свойство PidTagTtyTddPhoneNumber

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит номер телефона для контакта телетайпа (TTY) или телекоммуникаций устройство для глухих (TDD).
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A4B  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
