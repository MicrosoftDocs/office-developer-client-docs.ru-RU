# <a name="pidtagdeferreddeliverytime-canonical-property"></a>Каноническое свойство PidTagDeferredDeliveryTime

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит дату и время, когда отправитель сообщения хочет доставить сообщение. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DEFERRED_DELIVERY_TIME  <br/> |
|Идентификатор:  <br/> |0x000F  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

MAPI не выполняет отложенную доставку; это возможность для обработки отложенной доставки в рамках системы обмена сообщениями.
