# <a name="pidnamelastauthor-canonical-property"></a>Каноническое свойство PidNameLastAuthor

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает последнего автора присоединенного файла.
  
|||
|:-----|:-----|
|Дружественные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Имя свойства:  <br/> |LastAuthor  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Распространенная  <br/> |
