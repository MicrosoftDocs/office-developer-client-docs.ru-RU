# <a name="pidtagcallbacktelephonenumber-canonical-property"></a>Каноническое свойство PidTagCallbackTelephoneNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит номер телефона, который получатель сообщения может использовать для доступа к отправителю. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КАЛЛБАКК_ТЕЛЕФОНЕ_НУМБЕР, ПР_КАЛЛБАКК_ТЕЛЕФОНЕ_НУМБЕР_А, ПР_КАЛЛБАКК_ТЕЛЕФОНЕ_НУМБЕР_В  <br/> |
|Идентификатор:  <br/> |0x3A02  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Комментарии

Эти свойства являются примерами свойств, обеспечивающих идентификацию и доступ к сведениям о получателе. Они определяются получателем и Организацией получателя.
