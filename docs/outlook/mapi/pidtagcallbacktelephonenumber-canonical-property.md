# <a name="pidtagcallbacktelephonenumber-canonical-property"></a>Каноническое свойство PidTagCallbackTelephoneNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит телефонный номер, который получатель сообщения может использовать для достижения отправитель. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CALLBACK_TELEPHONE_NUMBER, PR_CALLBACK_TELEPHONE_NUMBER_A, PR_CALLBACK_TELEPHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A02  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства являются примерами свойств, которые предоставляют идентификацию и доступ к данным о получателе. Они определяются получателем и организацией получателя.
