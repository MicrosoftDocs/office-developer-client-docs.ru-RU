# <a name="ixpprovider--iunknown"></a>IXPProvider : IUnknown

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Инициализирует объект поставщика транспорта и закрывает объект, когда он больше не требуется.

|Вызывающая сторона:  <br/> |Шпалер MAPI  <br/> |
|Идентификатор интерфейса:  <br/> |IID_IXPProvider  <br/> |
