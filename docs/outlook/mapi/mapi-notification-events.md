# <a name="mapi-notification-events"></a>События уведомлений MAPI

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
При регистрации клиентских приложений для уведомлений о событиях, их необходимо указать один или несколько событий. События, которые они могут указать зависят от набор событий, которые поддерживает источника предполагаемая уведомлений. Существует десять типа уведомлений, клиентов и поставщиков услуг можно регистрировать для каждой представленной константа. Уведомление о доставке объект является исключением. Уведомление о доставке объект является внутренних уведомлений MAPI; клиенты не удалось зарегистрировать его и его не удается создать поставщиков услуг. В следующей таблице описаны типы событий и объектов источника уведомлений, которые может поддерживать их. Константа события входит в состав типа события.
  
|**Тип события**|**Описание**|**Уведомить объектов источника**|
|:-----|:-----|:-----|
|Критическая ошибка ( _fnevCriticalError_)  <br/> |Глобальные сообщение об ошибке или событие произошло, такие как завершение сеанса в процессе выполнения.  <br/> |Сеанс хранилища сообщений и адресной книги объекты, таблицы, состояние всех типов  <br/> |
|Изменения объекта ( _fnevObjectModified_)  <br/> |Объект MAPI был изменен.  <br/> |Папки, сообщения, все типы объектов адресной книги  <br/> |
|Объект, созданный ( _fnevObjectCreated_)  <br/> |Объект MAPI был создан.  <br/> |Папки, сообщения, все типы объектов адресной книги  <br/> |
|Объект перемещен ( _fnevObjectMoved_)  <br/> |Объект MAPI был перемещен.  <br/> |Папки, сообщения, все типы объектов адресной книги  <br/> |
|Удаление объекта ( _fnevObjectDeleted_)  <br/> |Объект MAPI был удален.  <br/> |Папки, сообщения, все типы объектов адресной книги  <br/> |
|Скопирован объект ( _fnevObjectCopied_)  <br/> |Объект MAPI был скопирован.  <br/> |Папки, сообщения, все типы объектов адресной книги  <br/> |
|Расширенные события ( _fnevExtended_)  <br/> |Внутренний определенные поставщиком услуг конкретного события.  <br/> |Любой объект источника уведомлений  <br/> |
|Поиск завершен ( _fnevSearchComplete_)  <br/> |Результаты поиска доступны и завершения операции поиска.  <br/> |Folders  <br/> |
|Таблица изменена ( _fnevTableModified_)  <br/> |Сведения в объекте таблицы MAPI, была изменена.  <br/> |Таблицы  <br/> |
|Новые сообщения ( _fnevNewMail_)  <br/> |Сообщение было доставлено и ожидает обработки.  <br/> |Хранилище сообщений, папок  <br/> |
   
Расширенные события определяется поставщиком услуг для представления событие, которое не может быть рассмотрен с любой из предварительно определенных событий. Для этого события можно регистрировать только клиентов, которые знать, прежде чем они зарегистрировать, что поставщик услуг поддерживает расширенные события. Не поддерживается для клиентов определить без базы знаний предварительное ли поставщик услуг поддерживает расширенные события, и если это так, как обрабатывать события при получении.
