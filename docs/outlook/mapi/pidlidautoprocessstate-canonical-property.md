# <a name="pidlidautoprocessstate-canonical-property"></a>Каноническое свойство PidLidAutoProcessState

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает параметры, используемые при автоматической обработке сообщений электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSniffState  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x0000851A  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Свойство может отсутствовать, в этом случае используется значение "0x00000000". Если заданной, это свойство должно быть заданной для одного из значений в следующей таблице.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0x00000000  <br/> |Не обработать сообщение автоматически.  <br/> |
|0x00000001  <br/> |Обработать сообщение автоматически или после открытия сообщения.  <br/> |
|0x00000002  <br/> |Обработка сообщения только после открытия сообщения.  <br/> |
