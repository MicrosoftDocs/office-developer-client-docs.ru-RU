# <a name="pidtagassociatedsharingprovider-canonical-property"></a>Каноническое свойство PidTagAssociatedSharingProvider

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Определяет, настроен ли поставщик хранения (PST) файл личных папок, как файл личных ПАПОК Microsoft SharePoint 2010.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ASSOCIATED_SHARING_PROVIDER  <br/> |
|Идентификатор:  <br/> |0x0EA00048  <br/> |
|Тип данных:  <br/> |PT_CLSID  <br/> |
|Область:  <br/> |Таблица личных папок (.pst) внутренний  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство показывает, настроен ли файл личных ПАПОК Microsoft SharePoint 2010. Если свойство не указан, PST не был настроен как файл личных ПАПОК Microsoft SharePoint 2010.
