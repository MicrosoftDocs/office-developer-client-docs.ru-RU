# <a name="pidtagtelecommunicationsdevicefordeaftelephonenumber-canonical-property"></a>Каноническое свойство PidTagTelecommunicationsDeviceForDeafTelephoneNumber

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит телекоммуникационное устройство пользователя обмена сообщениями для телефонного номера "TTYTDD" (TTYTDD).
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A4B  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения об идентификации и доступе к пользователю системы обмена сообщениями. Содержимое определяется пользователем системы обмена сообщениями и его организацией.
