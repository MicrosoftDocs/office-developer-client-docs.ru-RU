# <a name="pidlidverbstream-canonical-property"></a>Каноническое свойство PidLidVerbStream

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, какие ответы на голосование пользователь может выполнять в ответ на сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидвербстреам  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008520  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Настройка времени выполнения  <br/> |
