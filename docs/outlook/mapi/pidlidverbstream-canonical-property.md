# <a name="pidlidverbstream-canonical-property"></a>Каноническое свойство PidLidVerbStream

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, какие ответы на голосование пользователь может сделать в ответ на сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidVerbStream  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008520  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конфигурация времени запуска  <br/> |
