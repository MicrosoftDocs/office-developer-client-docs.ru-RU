# <a name="mapi-features-and-architecture"></a>Функции и архитектура MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
API обмена сообщениями (MAPI) состоит из набора общих интерфейсов программирования приложений и компонента библиотеки динамических ссылок (DLL). Интерфейсы используются для создания и доступа к разнообразным приложениям и системам обмена сообщениями, предлагая единую среду для разработки и использования и обеспечивая настоящую независимость для обоих. DLL содержит подсистему MAPI, которая управляет взаимодействием между интерфейсными приложениями для обмена сообщениями и системами обмена сообщениями и предоставляет общий пользовательский интерфейс для частых задач. Подсистема MAPI выступает в качестве центрального клиинга, который унифицирует различные системы обмена сообщениями и защищает клиентов от их различий.
