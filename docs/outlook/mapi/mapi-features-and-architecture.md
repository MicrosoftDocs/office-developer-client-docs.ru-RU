# <a name="mapi-features-and-architecture"></a>Функции MAPI и архитектура

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Система обмена сообщениями API (MAPI) состоит из набора общих программных интерфейсов и компонент библиотеки динамической компоновки (DLL). Интерфейсы используются для создания и доступа к различных приложений для обмена сообщениями и систем обмена сообщениями, предлагая универсальный среды для разработки и и предоставления независимости значение true для обоих. Библиотека DLL содержит подсистемы MAPI, который управляет взаимодействие между интерфейсных приложений для обмена сообщениями и серверными системами обмена сообщениями и предоставляет общий интерфейс пользователя для частых задач. Подсистема MAPI действует как центра расчетной объединить различных систем обмена сообщениями, а также защитить клиентов от их отличия.
