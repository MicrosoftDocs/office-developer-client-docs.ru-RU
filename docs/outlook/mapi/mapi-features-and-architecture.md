# <a name="mapi-features-and-architecture"></a>Функции и архитектура MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
API обмена сообщениями (MAPI) состоит из набора общих программных интерфейсов приложений и компонента библиотеки динамической ссылки (DLL). Интерфейсы используются для создания и доступа к разнообразным приложениям для обмена сообщениями и системам обмена сообщениями, обеспечивая унифицированную среду для разработки и использования, а также обеспечивая подлинное независимость для обоих этих приложений. DLL содержит подсистему MAPI, которая управляет взаимодействием между интерфейсными приложениями обмена сообщениями и системами обмена сообщениями на тыловом компьютере и предоставляет общий пользовательский интерфейс для выполнения частых задач. Подсистема MAPI выступает в качестве центральной кливеринг-хранилища, чтобы объединить различные системы обмена сообщениями и защитить клиенты от их различий.
