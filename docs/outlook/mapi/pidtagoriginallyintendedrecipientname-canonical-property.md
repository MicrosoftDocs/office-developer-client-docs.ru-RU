# <a name="pidtagoriginallyintendedrecipientname-canonical-property"></a>Каноническое свойство PidTagOriginallyIntendedRecipientName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит закодированное имя исходного получателя автоматически переназначаемого сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINALLY_INTENDED_RECIPIENT_NAME  <br/> |
|Идентификатор:  <br/> |0x0020  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Server  <br/> |
   
## <a name="remarks"></a>Примечания

Свойство **PR_ORIGINALLY_INTENDED_RECIPIENT_NAME** должно быть установлено автоматическим агентом, который переадил сообщение.
