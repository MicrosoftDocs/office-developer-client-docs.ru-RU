# <a name="pidtaghometelephonenumber-canonical-property"></a>Каноническое свойство PidTagHomeTelephoneNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит основной телефонный номер дома получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_TELEPHONE_NUMBER, PR_HOME_TELEPHONE_NUMBER_A, PR_HOME_TELEPHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A09  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства обеспечивают идентификацию и доступ для получателя. Они определяются получателем и организацией.
