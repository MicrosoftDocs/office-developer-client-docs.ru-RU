# <a name="pidlidverbresponse-canonical-property"></a>Каноническое свойство PidLidVerbResponse

  
  
**Относится к**: Outlook 
  
Задает параметр голосования выбранного респондентов.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidVerbResponse  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008524  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общие системы обмена сообщениями  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство обычно задается с разделителями значения, содержащиеся в свойстве **dispidVerbStream** ([PidLidVerbStream](pidlidverbstream-canonical-property.md)), на котором выдвигается респондентов.
