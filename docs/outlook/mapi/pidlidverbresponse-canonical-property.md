# <a name="pidlidverbresponse-canonical-property"></a>Каноническое свойство PidLidVerbResponse

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает параметр голосования, который выбран респондентом.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидвербреспонсе  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008524  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
   
## <a name="remarks"></a>Примечания

Для этого свойства обычно задается одно из значений с разделителями, которые хранятся в свойстве **диспидвербстреам** ([PidLidVerbStream](pidlidverbstream-canonical-property.md)), голосовом респондента.
