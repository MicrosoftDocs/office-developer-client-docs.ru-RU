# <a name="pidtagprofession-canonical-property"></a>Каноническое свойство PidTagProfession

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит род занятий пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PROFESSION, PR_PROFESSION_A, PR_PROFESSION_W  <br/> |
|Идентификатор:  <br/> |0x3A46  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства определяются пользователем или организации пользователя.
