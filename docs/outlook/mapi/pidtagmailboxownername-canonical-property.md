# <a name="pidtagmailboxownername-canonical-property"></a>Каноническое свойство PidTagMailboxOwnerName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Отображает имя владельца почтового ящика.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_MAILBOX_OWNER_NAME, PR_MAILBOX_OWNER_NAME_A, PR_MAILBOX_OWNER_NAME_W  <br/> |
|Идентификатор:  <br/> |0x001F  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_UNICODE  <br/> |
|Область:  <br/> |Хранилище сообщений  <br/> |
