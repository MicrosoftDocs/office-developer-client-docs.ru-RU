# <a name="pidlidtaskstatusoncomplete-canonical-property"></a>Каноническое свойство PidLidTaskStatusOnComplete

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, был ли получен запрос на отправку сообщения электронной почты после обновления по завершении назначенной задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидтасксок  <br/> |
|Набор свойств:  <br/> |Псетид_таск  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008119  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Задача  <br/> |
