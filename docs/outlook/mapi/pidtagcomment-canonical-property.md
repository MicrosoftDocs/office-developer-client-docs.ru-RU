# <a name="pidtagcomment-canonical-property"></a>Каноническое свойство PidTagComment

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит комментарий о назначении или содержимом объекта. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_COMMENT, PR_COMMENT_A, PR_COMMENT_W  <br/> |
|Идентификатор:  <br/> |0x3004  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Распространенная  <br/> |
   
## <a name="remarks"></a>Примечания

Содержимое строки определяется пользователем системы обмена сообщениями.
