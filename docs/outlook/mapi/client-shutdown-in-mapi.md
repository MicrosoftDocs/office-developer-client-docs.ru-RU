# <a name="client-shutdown-in-mapi"></a>Завершение работы клиента в MAPI 
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Начиная с Microsoft Outlook 2010, русская версия и Microsoft Outlook 2013, клиенты MAPI могут закрываться так же, как и раньше, или использовать быстрое отключение. Чтобы быстрое отключение было успешно завершено, клиент MAPI, поставщик MAPI и администратор клиентского компьютера должны поддерживать быстрое отключение. 
  
В разделах описана поддержка MAPI для клиента для быстрого отключения.
  
[Обзор быстрого завершения работы](fast-shutdown-overview.md)
  
> В этом разделе вводится базовый механизм быстрого отключения.
    
[Параметры быстрого отключения пользователей](fast-shutdown-user-options.md)
  
> В этом разделе описаны варианты, доступные администраторам для быстрого отключения на уровне пользователя для клиентов MAPI пользователя.
    
[Рекомендации по быстрому завершению работы](best-practices-for-fast-shutdown.md)
  
> В этом разделе рекомендуется использовать интерфейсы быстрого отключения для предотвращения потери данных во время остановки клиента MAPI.
