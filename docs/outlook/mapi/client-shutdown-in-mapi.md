# <a name="client-shutdown-in-mapi"></a>Завершение работы клиента в MAPI 
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Начиная с Microsoft Outlook 2010, русская версия и теперь, включая Microsoft Outlook 2013, клиенты MAPI могут отключиться так же, как и раньше, или использовать быстрое завершение работы. Чтобы быстрое завершение работы было успешно завершено, клиент MAPI, поставщик MAPI и администратор клиентского компьютера должны поддерживать быстрое завершение работы. 
  
В темах этого раздела описывается поддержка MAPI для быстрого завершения работы клиента.
  
[Обзор быстрого завершения работы](fast-shutdown-overview.md)
  
> В этом разделе вводится базовый механизм быстрого завершения работы.
    
[Параметры пользователей быстрого завершения работы](fast-shutdown-user-options.md)
  
> В этом разделе описаны варианты, доступные администраторам для быстрого завершения работы на уровне пользователя для клиентов MAPI пользователя.
    
[Рекомендации по быстрому завершению работы](best-practices-for-fast-shutdown.md)
  
> В этом разделе рекомендуется использовать интерфейсы быстрого завершения работы, чтобы предотвратить потерю данных во время завершения работы клиента MAPI.
