# <a name="client-shutdown-in-mapi"></a>Завершение работы клиента в MAPI 
  
**Относится к**: Outlook 
  
Приступая к работе в Microsoft Outlook 2010, теперь включительно, Microsoft Outlook 2013, клиенты MAPI могут завершение работы так же, как перед или они могут использовать быстрое завершение работы. Быстрое завершение работы будет выполнена успешно клиент MAPI, поставщика MAPI и администратора на клиентском компьютере необходимо поддерживает быстрое завершение работы. 
  
В этом разделе описываются поддержки MAPI для клиента выполнить быстрое завершение работы.
  
[Обзор быстрого завершения работы](fast-shutdown-overview.md)
  
> В этом разделе рассматриваются базовый механизм быстрое завершение работы.
    
[Пользовательские параметры быстрого завершения работы](fast-shutdown-user-options.md)
  
> В этом разделе описываются варианты администраторы могут принять быстрое завершение работы на уровне пользователя для клиентов MAPI пользователя.
    
[Рекомендации по быстрому завершению работы](best-practices-for-fast-shutdown.md)
  
> В данном разделе содержатся советы и рекомендации для использования интерфейсов быстрое завершение работы для предотвращения потери данных во время завершения работы клиента MAPI.
