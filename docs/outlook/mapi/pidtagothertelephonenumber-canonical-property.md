# <a name="pidtagothertelephonenumber-canonical-property"></a>Каноническое свойство PidTagOtherTelephoneNumber

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит альтернативного телефонный номер для получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OTHER_TELEPHONE_NUMBER, PR_OTHER_TELEPHONE_NUMBER_A, PR_OTHER_TELEPHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A1F  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства предоставляют идентификации и доступа к сведениям для получателя. Они определены получателя и их организации. 
  
Эти свойства, используемые для телефонный номер, отличных от получателя место рабочий, домашний, или office.
