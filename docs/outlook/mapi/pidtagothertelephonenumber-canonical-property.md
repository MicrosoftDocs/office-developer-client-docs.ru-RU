# <a name="pidtagothertelephonenumber-canonical-property"></a>Каноническое свойство PidTagOtherTelephoneNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит альтернативный телефонный номер получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OTHER_TELEPHONE_NUMBER, PR_OTHER_TELEPHONE_NUMBER_A, PR_OTHER_TELEPHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A1F  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Адрес  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют идентификацию и сведения о доступе для получателя. Они определяются получателем и организацией. 
  
Эти свойства используются для номера телефона, не на месте бизнеса, дома или офиса получателя.
