# <a name="pidtagsearchfoldertag-canonical-property"></a>Каноническое свойство PidTagSearchFolderTag

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит значение, используемое для синхронизации этого сообщения определения с подходящим контейнером папок поиска.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_WB_SF_TAG  <br/> |
|Идентификатор:  <br/> |0x6847  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Поиск  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство изменяется при изменении сообщения определения. Он должен изменить каждую итерацию, но может быть не уникальным.
