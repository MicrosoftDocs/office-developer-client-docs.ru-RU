## <a name="search-keys"></a><span data-ttu-id="028a4-118">Клавиши поиска</span><span class="sxs-lookup"><span data-stu-id="028a4-118">Search Keys</span></span>

<span data-ttu-id="028a4-123">У пользователей сообщений и сообщений есть ключи поиска.</span><span class="sxs-lookup"><span data-stu-id="028a4-123">Messages and messaging users have search keys.</span></span> <span data-ttu-id="028a4-124">Ключ поиска сообщения — уникальный идентификатор данных сообщения.</span><span class="sxs-lookup"><span data-stu-id="028a4-124">The search key of a <span data-ttu-id="028a4-127">Если запись адресной книги является записаемой, ее ключ поиска может быть не доступен до тех пор, пока не будет установлен тип и адрес адреса с помощью метода
