# <a name="imapimessagesitegetfolder"></a>IMAPIMessageSite::GetFolder

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Возвращает папку, в которой было создано или открыто текущее сообщение, если такая папка существует. Этот метод возвращает NULL в  _параметре ppFolder_ для встроенных сообщений, которые не хранятся непосредственно в папке. 
  
```cpp
HRESULT GetFolder(
  LPMAPIFOLDER FAR * ppFolder
);
```

## <a name="parameters"></a>Parameters

 _ppFolder_
  
> [вышел] Указатель на указатель на возвращенную папку.
