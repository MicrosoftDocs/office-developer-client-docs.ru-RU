# <a name="pidtagphysicalforwardingprohibited-canonical-property"></a>Каноническое свойство PidTagPhysicalForwardingProhibited

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение TRUE, если отправитель сообщения запрещает физическое перенаправление сообщений для определенного получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ФИСИКАЛ_ФОРВАРДИНГ_ПРОХИБИТЕД  <br/> |
|Идентификатор:  <br/> |0x0C0F  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Получатель MAPI  <br/> |
