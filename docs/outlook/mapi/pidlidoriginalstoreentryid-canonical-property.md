# <a name="pidlidoriginalstoreentryid-canonical-property"></a>Каноническое свойство PidLidOriginalStoreEntryId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает идентификатор записи хранилища представителя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидоригсториид  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008237  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть задано для объектов собраний, которые были созданы или обновлены представителем.
