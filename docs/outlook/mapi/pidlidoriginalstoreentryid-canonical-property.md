# <a name="pidlidoriginalstoreentryid-canonical-property"></a>Каноническое свойство PidLidOriginalStoreEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает ID записи магазина делегатора.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidOrigStoreEid  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008237  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть установлено для объектов собраний, которые были созданы или обновлены делегатом.
