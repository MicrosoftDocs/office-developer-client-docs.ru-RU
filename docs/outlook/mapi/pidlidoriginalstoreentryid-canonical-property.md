# <a name="pidlidoriginalstoreentryid-canonical-property"></a>Каноническое свойство PidLidOriginalStoreEntryId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает идентификатор записи хранилища сотрудника.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidOrigStoreEid  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008237  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Meetings (собрания);  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство необходимо задать на собрание объекты, которые были созданы или изменены с делегатом.
