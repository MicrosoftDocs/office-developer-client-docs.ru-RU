# <a name="pidtaglanguages-canonical-property"></a>Каноническое свойство PidTagLanguages

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит список языков, включенные в сообщение ASCII. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_LANGUAGES, PR_LANGUAGES_A, PR_LANGUAGES_W  <br/> |
|Идентификатор:  <br/> |0x002F  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Общие системы обмена сообщениями  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства содержат последовательность кодов двухсимвольных страны или региона, разделенных запятыми.
