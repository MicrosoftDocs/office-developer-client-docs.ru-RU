# <a name="pidtaglanguages-canonical-property"></a>Каноническое свойство PidTagLanguages

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит список asCII языков, включенных в сообщение. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_LANGUAGES, PR_LANGUAGES_A, PR_LANGUAGES_W  <br/> |
|Идентификатор:  <br/> |0x002F  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства содержат последовательность двух символов кодов страны и региона, разделенных запятой.
