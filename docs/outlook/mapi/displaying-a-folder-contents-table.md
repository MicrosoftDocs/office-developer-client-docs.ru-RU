    3. <span data-ttu-id="2f9e1-116">Создание ограничения в структуре [сандрестриктион](sandrestriction.md) , которая присоединяет ограничения свойств и битовой маски.</span><span class="sxs-lookup"><span data-stu-id="2f9e1-116">Create a restriction in an [SAndRestriction](sandrestriction.md) structure that joins the property and bitmask restrictions.</span></span>
