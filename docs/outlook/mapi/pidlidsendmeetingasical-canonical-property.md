# <a name="pidlidsendmeetingasical-canonical-property"></a>Каноническое свойство PidLidSendMeetingAsIcal

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, следует ли отправить приглашение на собрание в виде сообщения в формате iCal.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидсендмтгасикал  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008200  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Конференции  <br/> |
