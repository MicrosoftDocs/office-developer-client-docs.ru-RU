# <a name="pidlidsendmeetingasical-canonical-property"></a>Каноническое свойство PidLidSendMeetingAsical

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, следует ли отправить запрос на собрание в качестве сообщения iCal.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSendMtgAsICAL  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008200  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Конференц-связь  <br/> |
