# <a name="pidtagnonipmsubtreeentryid-canonical-property"></a>Каноническое свойство PidTagNonIpmSubtreeEntryId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Отображает ид записи **NON_IPM_SUBTREE** в хранилище общедоступных папок Exchange. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_NON_IPM_SUBTREE_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x6620  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Хранилище сообщений  <br/> |
