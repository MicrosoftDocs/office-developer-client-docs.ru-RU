# <a name="pidtaghomeaddresscountry-canonical-property"></a>Каноническое свойство PidTagHomeAddressCountry

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит округ в адресе контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_ADDRESS_COUNTRY, PR_HOME_ADDRESS_COUNTRY_A, PR_HOME_ADDRESS_COUNTRY_W  <br/> |
|Идентификатор:  <br/> |0x3A5A  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
