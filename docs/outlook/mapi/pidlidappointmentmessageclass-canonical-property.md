# <a name="pidlidappointmentmessageclass-canonical-property"></a>Каноническое свойство PidLidAppointmentMessageClass

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает свойство **пр_мессаже_класс** ([PidTagMessageClass](pidtagmessageclass-canonical-property.md)) собрания, которое будет создано из приглашения на собрание.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидапптмессажекласс  <br/> |
|Набор свойств:  <br/> |Псетид_митинг  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00000024  <br/> |
|Тип данных:  <br/> |ПТ_ТСТРИНГ  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть "IPM. Встреча "или иметь префикс IPM. Встреча. ". Это свойство не является обязательным.
