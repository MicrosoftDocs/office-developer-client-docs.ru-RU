# <a name="pidlidappointmentmessageclass-canonical-property"></a>Каноническое свойство PidLidAppointmentMessageClass

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает **PR_MESSAGE_CLASS** ([PidTagMessageClass)](pidtagmessageclass-canonical-property.md)собрания, которое должно быть сформировано из запроса на собрание.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptMessageClass  <br/> |
|Набор свойств:  <br/> |PSETID_Meeting  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00000024  <br/> |
|Тип данных:  <br/> |PT_TSTRING  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть либо "IPM. Встреча" или с префиксом "IPM. Встреча.". Это свойство не требуется.
