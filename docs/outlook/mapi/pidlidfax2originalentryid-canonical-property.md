# <a name="pidlidfax2originalentryid-canonical-property"></a>Каноническое свойство PidLidFax2OriginalEntryId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает исходный идентификатор EntryID для домашнего Факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax2OriginalEntryID  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000080C5  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

При наличии этого свойства необходимо указать одноразовый EntryId, соответствующий этому адресу факса.
