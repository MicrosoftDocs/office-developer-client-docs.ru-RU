# <a name="pidtagkeyword-canonical-property"></a>Каноническое свойство PidTagKeyword

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит ключевое слово, которое определяет получателя для системного администратора получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КЭЙВОРД, ПР_КЭЙВОРД_А, ПР_КЭЙВОРД_В  <br/> |
|Идентификатор:  <br/> |0x3A0B  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства предоставляют сведения о идентификации и доступе для получателя. Они определяются получателем и их организацией.
