# <a name="address-book-provider-sample"></a>Пример поставщика адресных книг

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
В этом примере поддерживается один контейнер только для чтения для отображаемого имени и адресов электронной почты, которые читают из плоского двоичного файла. В примере поддерживаются одновековые шаблоны и все параметры конфигурации, кроме мастера профиля.
  
Этот пример можно скачать Outlook API обмена сообщениями [(MAPI).](https://go.microsoft.com/fwlink/?LinkId=129740
)

## <a name="supported-features"></a>Поддерживаемые функции

В этом примере поддерживаются следующие функции:
  
- Ограничения таблицы. В примере реализовано разрешение префикса-совпадения и двусмысленных имен. Он не реализует полный язык ограничений MAPI, и ограничения поддерживаются только на имени отображения.
    
- Таблица сведений для пользователей обмена сообщениями. 
    
- Разовая адресов.
    
- Расширенный диалоговое окно поиска.
    
- Интерфейс [IMAPIStatus : IMAPIProp.](imapistatusimapiprop.md) Этот интерфейс частично поддерживается; его **методы IMAPIProp** делегируют **интерфейсу IPropData.** Дополнительные сведения см. в [интерфейсе IPropData : IMAPIProp.](ipropdataimapiprop.md) 
    
- Интерактивная и программная конфигурация.
    
## <a name="unsupported-features"></a>Неподтверченные функции

В этом примере не поддерживаются следующие функции:
  
- Сортировка.
    
- Списки рассылки.
    
- Создание, удаление и изменение записей.
    
- Свойства с несколькими значениями.
    
- Свойства с именем.
    
- Различие между именами и фамилиями в отображающихся именах.
    
 **Установка поставщика образцов адресной книги**
  
1. Чтобы скачать пример поставщика адресной книги, см. в книге [Загрузка Outlook MAPI Samples](downloading-the-outlook-mapi-samples.md).
    
2. Найдите папку, в которой сохранены Outlook MAPI Samples. Щелкните правой кнопкой **мыши папку OutlookMAPISamples-version \< \> number** zip и нажмите **кнопку Извлечь все**.
    
3. Щелкните **Обзор,** выберите расположение, в котором необходимо сохранить образец, и нажмите кнопку **Извлечение**.
    
4. Запуск Visual Studio 2008.
    
5. В Visual Studio 2008 нажмите **кнопку Файл**, выберите **Открыть**, а затем **нажмите кнопку Project/Решение**.
    
6. Просмотрите расположение, в котором сохранен пример, щелкните **SABP.vcproj** и нажмите кнопку **Открыть**.
    
7. On the **Build** menu, click **Build Solution**.
    
8. В **диалоговом окне Сохранить файл как** диалоговое окно нажмите **кнопку Сохранить**.
    
9. В папке, в которой сохранен пример,  щелкните правой кнопкой мыши файлinstall.batи нажмите **кнопку Выполнить в качестве администратора**.
    
10. В диалоговом окне **Контроль учетных записей пользователей** нажмите кнопку **Продолжить**.
    
    > [!NOTE]
    > **Install.bat** копирует .dll в папку установки Microsoft Office по умолчанию C:\Program Files\Microsoft Office\Office12\. Если вы установили Office в другом расположении, щелкните правой кнопкой **мыши** Install.batнажмите **кнопку Изменить**. Файл открывается в Блокнот. Замените путь установки по умолчанию на путь установки, используемый на компьютере.
