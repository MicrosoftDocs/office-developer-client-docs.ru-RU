# <a name="pidlidyomicompanyname-canonical-property"></a>Каноническое свойство PidLidYomiCompanyName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает фонетическую транскрипцию названия компании контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидйомикомпанинаме  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000802E  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
