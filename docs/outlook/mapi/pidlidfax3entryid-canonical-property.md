# <a name="pidlidfax3entryid-canonical-property"></a>Каноническое свойство PidLidFax3EntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает идентификатор для другого Факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax3EntryID  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000080D1  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Контакт  <br/> |
