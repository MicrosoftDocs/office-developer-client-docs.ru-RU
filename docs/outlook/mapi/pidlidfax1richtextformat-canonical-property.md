# <a name="pidlidfax1richtextformat-canonical-property"></a>Каноническое свойство PidLidFax1RichTextFormat

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает формат формата формата бизнес-факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax1RTF  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x000080B6  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Контакт  <br/> |
