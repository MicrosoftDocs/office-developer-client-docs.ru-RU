# <a name="pidtagredirectionhistory-canonical-property"></a>Каноническое свойство PidTagRedirectionHistory

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит сведения о маршруте, покрываемом доставленным сообщением.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РЕДИРЕКТИОН_ХИСТОРИ  <br/> |
|Идентификатор:  <br/> |0x002C  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство используется, когда сообщение автоматически переадресовано автоматическим агентом.
