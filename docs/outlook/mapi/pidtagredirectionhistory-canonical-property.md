# <a name="pidtagredirectionhistory-canonical-property"></a>Каноническое свойство PidTagRedirectionHistory

  
  
**Относится к**: Outlook 
  
Содержит сведения о маршруте, к которому применяется доставке сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_REDIRECTION_HISTORY  <br/> |
|Идентификатор:  <br/> |0x002C  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство используется при сообщение будет автоматически переадресовано автоматического агентом.
