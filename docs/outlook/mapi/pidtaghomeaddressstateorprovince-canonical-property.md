# <a name="pidtaghomeaddressstateorprovince-canonical-property"></a>Каноническое свойство PidTagHomeAddressStateOrProvince

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит область области или штата в адресе пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_ADDRESS_STATE_OR_PROVINCE, PR_HOME_ADDRESS_STATE_OR_PROVINCE_A PR_HOME_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Идентификатор:  <br/> |0x3A5C  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
