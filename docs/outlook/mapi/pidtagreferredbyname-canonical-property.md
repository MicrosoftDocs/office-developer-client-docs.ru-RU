# <a name="pidtagreferredbyname-canonical-property"></a>Каноническое свойство PidTagReferredByName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит имя ссылки пользователя почтового ящика.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РЕФЕРРЕД_БИ_НАМЕ, ПР_РЕФЕРРЕД_БИ_НАМЕ_А, ПР_РЕФЕРРЕД_БИ_НАМЕ_В  <br/> |
|Идентификатор:  <br/> |0x3A47  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
