# <a name="pidtagreferredbyname-canonical-property"></a>Каноническое свойство PidTagReferredByName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя пользователя почты ссылок.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_REFERRED_BY_NAME, PR_REFERRED_BY_NAME_A, PR_REFERRED_BY_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A47  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
