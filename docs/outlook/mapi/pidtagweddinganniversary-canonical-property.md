# <a name="pidtagweddinganniversary-canonical-property"></a>Каноническое свойство PidTagWeddingAnniversary

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит дату Годовщина свадьбы пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_WEDDING_ANNIVERSARY  <br/> |
|Идентификатор:  <br/> |0x3A41  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство определяется пользователем или организации пользователя.
