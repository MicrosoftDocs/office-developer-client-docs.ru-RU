# <a name="pidlidsharingconfigurationurl-canonical-property"></a>Каноническое свойство PidLidSharingConfigurationUrl

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Определяет, как свойство общего доступа сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSharingConfigUrl  <br/> |
|Набор свойств:  <br/> |PSETID_Sharing  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008A24  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Sharing  <br/> |
   
## <a name="remarks"></a>Замечания

Значение этого свойства должна быть строка нулевой длины.
