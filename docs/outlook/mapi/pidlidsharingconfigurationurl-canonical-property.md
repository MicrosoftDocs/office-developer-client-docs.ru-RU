# <a name="pidlidsharingconfigurationurl-canonical-property"></a>Каноническое свойство PidLidSharingConfigurationUrl

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Обозначается как свойство общего сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSharingConfigUrl  <br/> |
|Набор свойств:  <br/> |PSETID_Sharing  <br/> |
|Long ID (LID):  <br/> |0x00008A24  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Доступ  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть строкой нулевой длины.
