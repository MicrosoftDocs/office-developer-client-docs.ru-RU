# <a name="pidtagreportdispositiontoemailaddresses-canonical-property"></a>Каноническое свойство PidTagReportDispositionToEmailAddresses

  
  
**Относится к**: Outlook 
  
Указывает адреса электронной почты получателей отчетов для сообщений, запрашивающих поступлений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES, PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_A, PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_W  <br/> |
|Идентификатор:  <br/> |0x0084  <br/> |
|Тип данных:  <br/> |PT_MV_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
