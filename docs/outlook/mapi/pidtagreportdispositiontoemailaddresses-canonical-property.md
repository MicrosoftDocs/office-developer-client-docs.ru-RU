# <a name="pidtagreportdispositiontoemailaddresses-canonical-property"></a>Каноническое свойство PidTagReportDispositionToEmailAddresses

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает адреса электронной почты получателей отчетов для сообщений, запрашивающих уведомления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РЕПОРТ_ДИСПОСИТИОН_ТО_ЕМАИЛ_АДДРЕССЕС, ПР_РЕПОРТ_ДИСПОСИТИОН_ТО_ЕМАИЛ_АДДРЕССЕС_А, PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_W  <br/> |
|Идентификатор:  <br/> |0x0084  <br/> |
|Тип данных:  <br/> |PT_MV_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
