# <a name="personal-form-libraries"></a>Личные библиотеки форм

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Как можно предположить из названия, личные библиотеки форм содержат формы, интересные конкретному пользователю. Личная библиотека форм пользователя — это библиотека форм, связанная с хранилищем сообщений по умолчанию, которое определено в профиле пользователя; Каждый профиль, установленный на рабочей станции, может использовать отдельное хранилище по умолчанию и, следовательно, отдельную личную библиотеку форм. Личная библиотека форм может содержать копии форм, которые также содержатся в других библиотеках форм в дополнение к другим формам.
  
Личная библиотека форм реализована в связанной с содержимым таблице корневой папки в хранилище сообщений пользователя по умолчанию , независимо от того, находится ли она на сервере или локально на рабочей станции пользователя. Если хранилище сообщений пользователя по умолчанию хранится на рабочей станции пользователя, библиотеки личных форм обеспечивают повышенную производительность, позволяя приложениям получать локальный доступ к формам, а не по сети. Он также делает формы доступными пользователям, работающим в автономном режиме, что может произойти, если пользователи хотят использовать формы с ними на переносных компьютерах и не имеют доступа к сети.
  
Свойства и основные реализации личных записей библиотеки форм включают свойство "Идентификатор контейнера", которое определяет контейнер master, с котором должна синхронизироваться локализованная запись. Это может быть идентификатор произвольной папки, которая содержит формы. Это полезно при использовании настраиваемого диспетчера форм, который поддерживает какую-либо библиотеку форм для всей организации; диспетчер форм должен будет синхронизировать формы, хранимые в личной библиотеке форм и библиотеке форм для всей организации. Возможно, это произойдет при загрузке диспетчера форм, но теоретически это может произойти в любое время.
