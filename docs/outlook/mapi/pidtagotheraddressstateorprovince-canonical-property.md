# <a name="pidtagotheraddressstateorprovince-canonical-property"></a>Каноническое свойство PidTagOtherAddressStateOrProvince

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит название области или области, используемой в другом адресе.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OTHER_ADDRESS_STATE_OR_PROVINCE, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_A, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Идентификатор:  <br/> |0x3A62  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
