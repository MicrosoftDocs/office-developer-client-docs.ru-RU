# <a name="pidtagscheduleinfofreebusy-canonical-property"></a>Каноническое свойство PidTagScheduleInfoFreeBusy

  
  
**Относится к**: Outlook 
  
Устаревшие сведения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SCHDINFO_FREEBUSY  <br/> |
|Идентификатор:  <br/> |0x686C  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Обмен сведениями о доступности  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство не следует устанавливать и должны игнорироваться.
