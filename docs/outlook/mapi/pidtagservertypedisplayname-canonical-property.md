# <a name="pidtagservertypedisplayname-canonical-property"></a>Каноническое свойство PidTagServerTypeDisplayName

  
  
**Относится к**: Outlook 
  
Указывает строку Юникод для отображения в строке состояния с помощью этого свойства.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SERVER_TYPE_DISPLAY_NAME  <br/> |
|Идентификатор:  <br/> |0x341D  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Common  <br/> |
