# <a name="mapi-transport-provider-objects"></a>Объекты поставщика транспорта MAPI
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Помимо стандартных поставщиков и объектов для работы со службами, реализованных всеми поставщиками услуг, для реализации объекта состояния требуются поставщики транспорта. Для других типов поставщиков услуг реализация объекта состояния необязательна. Однако mapI требует его для поставщиков транспорта. Поставщики транспорта, поддерживаюющие загрузку сообщений с удаленного сервера, также реализуют папку и таблицу. 
  
На следующем рисунке показан каждый из объектов, которые поставщики транспорта могут реализовать с помощью соответствующих интерфейсов. На рисунке также по указано, является ли MAPI или клиент пользователем объекта.
