# <a name="mapi-transport-provider-objects"></a>Объекты поставщика транспорта MAPI
  
**Относится к**: Outlook 
  
В дополнение к стандартный поставщик и объектов входа в систему, реализованных всех поставщиков услуг поставщиками транспорта необходимы для реализации объекта состояния. Для других типов поставщика службы реализация объекта состояние является необязательным. Тем не менее MAPI он необходим для поставщиков транспорта. Поставщики транспорта, которые поддерживают загрузку заголовков сообщений с удаленного сервера также реализовать папки и таблицы. 
  
На следующем рисунке показана объектам, которые можно реализовать транспорта поставщиков с их соответствующие интерфейсы. Иллюстрация также указывает, является ли MAPI или клиента объекта пользователя.
