# <a name="cbsproptagarray"></a>CbSPropTagArray

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество байтов в существующей структуре [спроптагаррай](sproptagarray.md) . 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
|Связанная структура:  <br/> |**SPropTagArray** <br/> |
   
```cpp
CbSPropTagArray (_lparray)
```

## <a name="parameters"></a>Параметры

 __лпаррай_
  
> Указатель на существующую структуру **спроптагаррай** .
