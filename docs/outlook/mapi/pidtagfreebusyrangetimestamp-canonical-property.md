# <a name="pidtagfreebusyrangetimestamp-canonical-property"></a>Каноническое свойство PidTagFreeBusyRangeTimestamp

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит время, когда была опубликована данные.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_FREEBUSY_RANGE_TIMESTAMP  <br/> |
|Идентификатор:  <br/> |0x6868  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Обмен сведениями о доступности  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство соответствует количество минут, начиная с полуночи 1 января 1601 в формате UTC.
