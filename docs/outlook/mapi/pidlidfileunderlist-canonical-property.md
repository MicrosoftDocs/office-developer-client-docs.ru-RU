# <a name="pidlidfileunderlist-canonical-property"></a>Каноническое свойство PidLidFileUnderList

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает список возможных значений для свойства **диспидфилеундерид** ([PidLidFileUnderId](pidlidfileunderid-canonical-property.md)).
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидфилеундерлист  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008026  <br/> |
|Тип данных:  <br/> |PT_MV_LONG  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Каждое значение в многозначном свойстве должно быть одним из допустимых значений для **диспидфилеундерид** , указанных в [[MS – оксокнтк]](https://msdn.microsoft.com/library/9b636532-9150-4836-9635-9c9b756c9ccf%28Office.15%29.aspx).
