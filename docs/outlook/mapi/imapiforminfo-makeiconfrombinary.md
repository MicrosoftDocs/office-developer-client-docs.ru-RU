# <a name="imapiforminfomakeiconfrombinary"></a>IMAPIFormInfo::MakeIconFromBinary

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Создает значок из одного из свойств значка формы.
  
```cpp
HRESULT MakeIconFromBinary(
  ULONG nPropID,
  HICON FAR * phicon
);
```

## <a name="parameters"></a>Parameters

 _nPropID_
  
> [in] Идентификатор свойства для свойства значок.
    
 _phicon_
  
> [вышел] Указатель на возвращенный значок.
    
## <a name="return-value"></a>Возвращаемое значение

S_OK

## <a name="remarks"></a>Примечания

Клиентские приложения называют **метод IMAPIFormInfo::MakeIconFromBinary** для создания значка из одного из свойств значка формы. В  _параметре nPropID_ **MakeIconFromBinary** принимает идентификатор свойства одного из свойств значка формы. С помощью этого идентификатора свойства создается значок, который может отображаться в представлениях таблиц, включая столбцы свойств для значков.
