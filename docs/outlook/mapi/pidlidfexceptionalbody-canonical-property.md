# <a name="pidlidfexceptionalbody-canonical-property"></a>Каноническое свойство PidLidFExceptionalBody

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, что объект внедренного сообщения исключения имеет текст, который отличается от повторяющегося объекта календаря.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFExceptionalBody  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008206  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Если значение этого свойства — TRUE, то объект внедренного сообщения исключения должен иметь текст. Если значение этого свойства — FALSE или свойство не существует, клиент или сервер должны получить тело из повторяющегося объекта календаря.
