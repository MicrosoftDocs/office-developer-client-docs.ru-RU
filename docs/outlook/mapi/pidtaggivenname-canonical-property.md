# <a name="pidtaggivenname-canonical-property"></a>Каноническое свойство PidTagGivenName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя или имя получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_GIVEN_NAME, PR_GIVEN_NAME_A, PR_GIVEN_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A06  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют идентификационные данные и доступ к сведениям о получателе. Они определяются получателем и его организацией.
