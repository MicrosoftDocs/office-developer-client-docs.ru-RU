# <a name="pidtaggivenname-canonical-property"></a>Каноническое свойство PidTagGivenName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит имя получателя или имя получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ГИВЕН_НАМЕ, ПР_ГИВЕН_НАМЕ_А, ПР_ГИВЕН_НАМЕ_В  <br/> |
|Идентификатор:  <br/> |0x3A06  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства обеспечивают идентификацию и доступ к сведениям о получателе. Они определяются получателем и их организацией.
