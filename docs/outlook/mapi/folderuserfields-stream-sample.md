# <a name="folderuserfields-stream-sample"></a>Пример потока FolderUserFields

**Относится к**: Outlook 2013 | Outlook 2016 
  
В этом разделе описывается пример потока FolderUserFields. Поток содержит определение определяемого пользователем поля `TextField1`. Типом является **Text**, а поток FolderUserFields содержит как части фолдерусерфиелдсанси, так и фолдерусерфиелдсуникоде. Дополнительные [сведения см.](folder-fields-stream-structures.md)

Ниже приведен пример синтаксического анализа данных для потока **FolderUserFields** :
  
- Фолдерусерфиелдсанси: смещение 0x0.
    
  - Фиелддефинитионкаунт: offset 0x0, 4 байта: 0x00000002 (2).
    
  - Фиелддефинитионс: offset 0x4, массив из 2 Фолдерфиелддефинитиона потоков.
    
    **Первый элемент массива**:
    
    - FieldType: offset 0x4, 4 байта: 0x00000001 (Фтстринг).
      
    - Фиелднамеленгс: offset 0x8, 2 байта: 0x000A (10)
      
    - FieldName: offset 0xA, массив из 10 знаков. Строковое значение ANSI: "TextField1".
      
    - Common: offset 0x14.
    
      - Пропсетгуид: offset 0x14, 16 байт: {00020329-0000-0000-C000-000000000046} (PS_PUBLIC_STRINGS).
        
      - фкапм: offset 0x24, 4 байта: 0x80000007 (FCAPM_CAN_EDIT | FCAPM_CAN_SORT | FCAPM_CAN_GROUP | FCAPM_CAN_EDIT_IN_ITEM).
        
      - Двстринг: offset 0x28, 4 байта: 0x00000000.
        
      - Двбитмап: offset 0x2C, 4 байта: 0x00000000.
        
      - Двдисплай: offset 0x30, 4 байта: 0x00000000.
        
      - Ифмт: offset 0x34, 4 байта: 0x00000000.
        
      - Всзформулаленгс: offset 0x38, 2 байта: 0x0000 (0).
        
      - Всзформула: offset 0x3A, массив 0 Вчарс. Пустое строковое значение.
    
    **Второй элемент массива**:
    
    - FieldType: offset 0x3A, 4 байта: 0x00000000 (Фтноне).
      
    - Фиелднамеленгс: offset 0x3E, 2 байта: 0x0000 (0).
      
    - FieldName: смещение 0x40, массив 0 знаков. Пустое строковое значение.
      
    - Общие: смещение 0x40.
    
      - Пропсетгуид: смещение 0x40, 16 байт: {00000000-0000-0000-0000-000000000000} (GUID_NULL).
        
      - фкапм: offset 0x50, 4 байта: 0x00000000 (0).
        
      - Двстринг: offset 0x54, 4 байта: 0x00000000.
        
      - Двбитмап: offset 0x58, 4 байта: 0x00000000.
        
      - Двдисплай: offset 0x5C, 4 байта: 0x00000000.
        
      - Ифмт: offset 0x60, 4 байта: 0x00000000.
        
      - Всзформулаленгс: offset 0x64, 2 байта: 0x0000 (0).
        
      - Всзформула: offset 0x66, массив 0 Вчарс. Пустое строковое значение.
    
- Фолдерусерфиелдсуникоде: offset 0x66.
    
  - Фиелддефинитионкаунт: offset 0x66, 4 байта: 0x00000002 (2).
    
  - Фиелддефинитионс: offset 0x6A, массив из 2 Фолдерфиелддефинитионв потоков.
    
    **Первый элемент массива**:
    
    - FieldType: offset 0x6A, 4 байта: 0x00000001 (Фтстринг).
      
    - Фиелднамеленгс: offset 0x6E, 2 байта: 0x000A (10).
      
    - FieldName: смещение 0x70, массив 10 Вчарс. Строковое значение Юникода: "TextField1".
      
    - Common: offset 0x84.
    
      - Пропсетгуид: offset 0x84, 16 байт: {00020329-0000-0000-C000-000000000046} (PS_PUBLIC_STRINGS).
        
      - фкапм: offset 0x94, 4 байта: 0x80000007 (FCAPM_CAN_EDIT | FCAPM_CAN_SORT | FCAPM_CAN_GROUP | FCAPM_CAN_EDIT_IN_ITEM).
        
      - Двстринг: offset 0x98, 4 байта: 0x00000000.
        
      - Двбитмап: offset 0x9C, 4 байта: 0x00000000.
        
      - Двдисплай: offset 0xA0, 4 байта: 0x00000000.
        
      - Ифмт: offset 0xA4, 4 байта: 0x00000000.
        
      - Всзформулаленгс: offset 0xA8, 2 байта: 0x0000 (0).
        
      - Всзформула: offset 0xAA, массив 0 Вчарс. Пустое строковое значение.
    
    **Второй элемент массива**:
    
    - FieldType: offset 0xAA, 4 байта: 0x00000000 (Фтноне).
      
    - Фиелднамеленгс: offset 0xAE, 2 байта: 0x0000 (0).
      
    - FieldName: смещение 0xB0, массив 0 Вчарс. Пустое строковое значение.
      
    - Common: offset 0xB0.
    
      - Пропсетгуид: offset 0xB0, 16 байт: {00000000-0000-0000-0000-000000000000} (GUID_NULL).
        
      - фкапм: offset 0xC0, 4 байта: 0x00000000 (0).
        
      - Двстринг: offset 0xC4, 4 байта: 0x00000000.
        
      - Двбитмап: offset 0xC8, 4 байта: 0x00000000.
        
      - Двдисплай: offset 0xCC, 4 байта: 0x00000000.
        
      - Ифмт: offset 0xD0, 4 байта: 0x00000000.
        
      - Всзформулаленгс: offset 0xD4, 2 байта: 0x0000 (0).
        
      - Всзформула: offset 0xD6, массив 0 Вчарс. Пустое строковое значение.
