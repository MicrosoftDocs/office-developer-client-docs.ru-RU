# <a name="naming-folders-by-using-character-strings"></a>Именование папок с помощью символьных строк

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
При частом доступе к одной или нескольким папкам в ходе сеанса рекомендуется назначать имена папкам с помощью метода [IMsgStore:: сетрецеивефолдер](imsgstore-setreceivefolder.md) . Несмотря на то, что **IMsgStore:: сетрецеивефолдер** используется преимущественно для создания специальных папок для получения входящих сообщений для определенных классов сообщений, он также может использоваться для сопоставления любой папки с именем. Имя может совпадать с классом Message или может представлять собой любую строку символов, настроенную для использования клиента. Связывание имени с папкой сокращает время, необходимое для поиска и открытия папки.
