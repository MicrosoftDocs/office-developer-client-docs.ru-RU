# <a name="pidtagrulemsgname-canonical-property"></a>Каноническое свойство PidTagRuleMsgName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит имя расширенного правила.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RULE_MSG_NAME  <br/> |
|Идентификатор:  <br/> |0x65EC  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Правила  <br/> |
