# <a name="retrieving-data-from-table-rows"></a>Получение данных из строк таблицы

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Получение строк из таблицы состоит из следующих компонентов:
  
- Получение значений свойств для всех столбцов.
    
- Изменение текущей позиции.
    
Один из обязательных столбцов в большинстве таблиц — это идентификатор записи — свойство **PR_ENTRYID** ([PidTagEntryId](pidtagentryid-canonical-property.md)), которое можно использовать для открытия объекта, представляющего строку. Этот идентификатор записи обычно является коротким идентификатором записи, который не сохраняется в течение всего времени существования таблицы. Тем не менее, это может быть долгосрочный идентификатор, если поставщик услуг, реализующий таблицу, поддерживает только один тип идентификатора записи.
  
Клиенты и поставщики услуг могут совершать один из следующих вызовов для получения строк:
  
|||
|:-----|:-----|
|[IMAPITable::QueryRows](imapitable-queryrows.md) <br/> |Извлекает заданное число строк, начиная с текущей строки, в направлении вперед или назад.  <br/> |
|[HrQueryAllRows](hrqueryallrows.md) <br/> |Получает все строки таблицы.  <br/> |
|[ITableData::HrQueryRow](itabledata-hrqueryrow.md) <br/> |Извлекает строку в таблице в соответствии со значением его столбца индекса. **PR_INSTANCE_KEY** ([PidTagInstanceKey](pidtaginstancekey-canonical-property.md)) обычно является столбцом индекса для таблицы.  <br/> |
   
Если необязательное свойство включено в один из столбцов таблицы, некоторые строки могут иметь допустимые значения для столбца, в то время как другие могут не иметь. Указывает, существует ли допустимое значение для столбца, зависит от того, задает ли объект, предоставляющий информацию для строки, свойство. В зависимости от реализации объекта несуществующее свойство может быть представлено в таблице как **PR_NULL** ([PidTagNull](pidtagnull-canonical-property.md)) или произвольное значение. Пользователи таблиц должны следить за тем, чтобы различать свойства, которые не существуют и имеют бессмысленные значения и свойства, которые имеют допустимые значения.
