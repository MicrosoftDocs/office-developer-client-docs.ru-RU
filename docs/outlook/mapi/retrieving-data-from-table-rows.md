# <a name="retrieving-data-from-table-rows"></a>Сбор данных из строк таблицы

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Ирисовка строк из таблицы включает в себя:
  
- Получение значений свойств для всех столбцов.
    
- Изменение текущей позиции.
    
Одним из необходимых столбцов в большинстве таблиц является идентификатор записи **— свойство PR_ENTRYID** [(PidTagEntryId),](pidtagentryid-canonical-property.md)которое можно использовать для открытия объекта, который представляет строку. Этот идентификатор записи обычно является краткосрочным идентификатором записи, который не сохраняется в течение всего срока службы таблицы. Однако это может быть долгосрочный идентификатор, если поставщик услуг, реализующий таблицу, поддерживает только один тип идентификатора записи.
  
Клиенты и поставщики услуг могут сделать один из следующих вызовов для получения строк:
  
|||
|:-----|:-----|
|[IMAPITable::QueryRows](imapitable-queryrows.md) <br/> |Извлекает определенное количество строк, начиная с текущей строки в направлении вперед или назад.  <br/> |
|[HrQueryAllRows](hrqueryallrows.md) <br/> |Извлекает все строки в таблице.  <br/> |
|[ITableData::HrQueryRow](itabledata-hrqueryrow.md) <br/> |Извлекает строку в таблице в соответствии со значением столбца индекса. **PR_INSTANCE_KEY** [(PidTagInstanceKey)](pidtaginstancekey-canonical-property.md)обычно является столбцом индекса для таблицы.  <br/> |
   
Если необязательное свойство включено в качестве одного из столбцов в таблице, некоторые строки могут иметь допустимые значения для столбца, а другие — нет. Существует ли допустимые значения для столбца, зависит от того, задает ли свойство объект, предоставляющий сведения для строки. В зависимости от реализации объекта в таблице может быть представлено несуществующее свойство **PR_NULL** [(PidTagNull)](pidtagnull-canonical-property.md)или произвольное значение. Пользователи таблиц должны быть осторожны, чтобы различать свойства, которые не существуют и имеют бессмысленными значениями и свойствами, которые существуют и имеют допустимые значения.
