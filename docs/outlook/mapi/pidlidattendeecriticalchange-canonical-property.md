# <a name="pidlidattendeecriticalchange-canonical-property"></a>Каноническое свойство PidLidAttendeeCriticalChange

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату и время отправки объекта, связанного с собранием.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ЛИД_АТТЕНДИ_КРИТИКАЛ_ЧАНЖЕ  <br/> |
|Набор свойств:  <br/> |Псетид_митинг  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00000001  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Комментарии

Значение должно быть указано в формате UTC.
