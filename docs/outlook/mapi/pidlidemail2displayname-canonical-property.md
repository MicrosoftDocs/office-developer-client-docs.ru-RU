# <a name="pidlidemail2displayname-canonical-property"></a>Каноническое свойство PidLidEmail2DisplayName

  
  
**Относится к**: Outlook 
  
Задает для чтения отображаемое имя для второго адреса электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail2DisplayName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008090  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
