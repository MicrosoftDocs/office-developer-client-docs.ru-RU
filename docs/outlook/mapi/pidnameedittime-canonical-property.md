# <a name="pidnameedittime-canonical-property"></a>Каноническое свойство PidNameEditTime

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает время последнего изменения файла.
  
|||
|:-----|:-----|
|Понятные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |ПС_ПУБЛИК_СТРИНГС  <br/> |
|Имя свойства:  <br/> |Едиттиме  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Распространенная  <br/> |
