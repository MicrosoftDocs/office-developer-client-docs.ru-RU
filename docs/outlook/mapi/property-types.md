# <a name="property-types"></a>Типы свойств

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
MAPI поддерживает одно значение и несколько значений свойств. С одним значением свойства базового типа, для свойства одно значение. Свойство несколько значений существует несколько значений базового типа. 
  
В следующей таблице описаны типы одно значение и несколько значений свойств, которые поддерживает MAPI. Для каждого типа одно значение, которое имеет соответствующий тип несколько значений тип несколько значений отображается в скобках после типа одно значение.
  
|**Тип свойства**|**Шестнадцатеричное значение**|**Описание**|
|:-----|:-----|:-----|
|PT_UNSPECIFIED  <br/> |0000  <br/> |Указывает, что неизвестный тип свойства. Этот тип свойства зарезервирован для использования с помощью методов интерфейса.  <br/> |
|PT_NULL  <br/> |0001  <br/> |Указывает на отсутствие значения свойства. Этот тип свойства зарезервирован для использования с помощью методов интерфейса и — то же, что OLE введите VT_NULL.  <br/> |
|PT_I2 (PT_MV_I2)  <br/> |0002  <br/> |16-разрядной (2-байтовое) целого числа со знаком. Этот тип свойства — то же, как PT_SHORT (PT_MV_SHORT) и OLE введите VT_I2.  <br/> |
|PT_I4 (PT_MV_I4)  <br/> |0003  <br/> |Подписанные или неподписанные 32-разрядная версия (4-байтовое) целое число. Этот тип свойства — то же, как PT_LONG (PT_MV_LONG) и OLE введите VT_I4.  <br/> |
|PT_FLOAT (PT_MV_FLOAT)  <br/> |0004  <br/> |32-разрядная версия (8-байтовое) число с плавающей запятой. Этот тип свойства — это то же, что PT_R4 (PT_MV_R4) и тип OLE VT_R4.  <br/> |
|PT_DOUBLE (PT_MV_DOUBLE)  <br/> |0005  <br/> |64-разрядная версия (8-байтовое) число с плавающей запятой. Этот тип свойства совпадает с PT_R8 и типы OLE VT_R8 и VT_DOUBLE.  <br/> |
|PT_CURRENCY (PT_MV_CURRENCY)  <br/> |0006  <br/> |64-разрядное целое число (8-байтовых) считается, что тип decimal. Этот тип свойства совместим с типом CURRENCY Microsoft Visual Basic и — то же, что OLE введите VT_CY.  <br/> |
|PT_APPTIME (PT_MV_APPTIME)  <br/> |0007  <br/> |Значение типа Double, интерпретируется как дата и время. Целая часть — это дата, а дробная часть — время. Этот тип свойства совпадает с типом OLE VT_DATE и совместим с представление времени Microsoft Visual Basic.  <br/> |
|PT_ERROR  <br/> |000A  <br/> |Значение SCODE; 32-разрядное целое число без знака (4-байтовых). Этот тип свойства совпадает с типом OLE VT_ERROR.  <br/> |
|PT_BOOLEAN (PT_MV_12)  <br/> |000B  <br/> |16-разрядной (2-байтовое) логическое значение, где равна нулю, равно нулю, **значение true**и **false** . Этот тип свойства совпадает с типом OLE VT_BOOL.  <br/> |
|PT_OBJECT  <br/> |000D  <br/> |Указатель на объект, реализующий интерфейс **IUnknown** . Этот тип свойства похож на несколько типов OLE, таких как VT_UNKNOWN.  <br/> |
|PT_I8 (PT_MV_I8)  <br/> |0014  <br/> |Знаком или без 64-разрядная версия (8-байтовое) целое число, которое использует структуру **LARGE_INTEGER** . Этот тип свойства — то же, как PT_I8 и OLE введите VT_I8.  <br/> |
|PT_STRING8 (PT_MV_STRING8)  <br/> |001E  <br/> |Строка символов (2-байтовое) 8-разрядных символом NULL. Этот тип свойства совпадает с типом OLE VT_LPSTR.  <br/> |
|PT_TSTRING (PT_MV_TSTRING)  <br/> |001F  <br/> |Строка символов (2-байтовое) 16-разрядный символом NULL. Свойства с этим типом имеют тип свойства, сброс PT_UNICODE при компиляции с символа ЮНИКОДА и PT_STRING8 при компиляции не с символа ЮНИКОДА. Этот тип свойства совпадает с OLE тип VT_LPSTR итоговый свойства PT_STRING8 и VT_LPWSTR для свойств PT_UNICODE  <br/> |
|PT_SYSTIME (PT_MV_SYSTIME)  <br/> |0040  <br/> |64-разрядная версия (8-байтовых) дату и время целое число в виде структуру **FILETIME** . Этот тип свойства совпадает с типом OLE VT_FILETIME.  <br/> |
|PT_CLSID (PT_MV_CLSID)  <br/> |0048  <br/> |Значение **CLSID** структуры. Этот тип свойства совпадает с типом OLE VT_CLSID.  <br/> |
|PT_SVREID  <br/> |00FB  <br/> |Размер переменной (2-байтовое) 16-разрядное **число** следуют структуру.  <br/> |
|PT_SRESTRICT  <br/> |00FD  <br/> |Размер переменной массив байтов, представляющий одну или несколько структур ограничение.  <br/> |
|PT_ACTIONS  <br/> |00FE  <br/> |Переменного размера (2-байтовое) 16-разрядное **число** действий (не в байтах) за несколько структур действие правила.  <br/> |
|PT_BINARY (PT_MV_BINARY)  <br/> |0102  <br/> |Значение структуры **SBinary** , массив байтов инвентаризации.  <br/> |
   
> [!NOTE]
> Чтобы определить шестнадцатеричное значение для типа многозначные свойства или PT_MV введите флаг (0x00001000) шестнадцатеричное значение для свойства. Например шестнадцатеричное значение для PT_MV_UNICODE является 0x101F и шестнадцатеричное значение для PT_MV_BINARY является 0x1102.
