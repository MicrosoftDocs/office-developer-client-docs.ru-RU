# <a name="property-types"></a>Типы свойств

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
MAPI поддерживает свойства с одним значением и с несколькими значениями. Для свойства с одним значением существует одно значение базового типа для свойства. При наличии свойства с несколькими значениями существует несколько значений базового типа. 
  
Типы свойств с одним значением и с несколькими значениями, поддерживаемые MAPI, описаны в приведенной ниже таблице. Для каждого типа с одним значением, который имеет соответствующий многозначный тип, тип с несколькими значениями отображается в круглых скобках после типа с одним значением.
  
|**Тип свойства**|**Шестнадцатеричное значение**|**Описание**|
|:-----|:-----|:-----|
|PT_UNSPECIFIED  <br/> |0000  <br/> |Указывает, что тип свойства неизвестен. Этот тип свойства зарезервирован для использования с методами интерфейса.  <br/> |
|PT_NULL  <br/> |0001  <br/> |Указывает, что значение свойства отсутствует. Этот тип свойства зарезервирован для использования с методами интерфейса и совпадает с типом OLE VT_NULL.  <br/> |
|PT_I2 (PT_MV_I2)  <br/> |0002  <br/> |16-разрядное (2-байтовое) целое число. Этот тип свойства такой же, как PT_SHORT (PT_MV_SHORT) и VT_I2 типа OLE.  <br/> |
|PT_I4 (PT_MV_I4)  <br/> |0003  <br/> |32-разрядное целое число со знаком и без знака (4 байта). Этот тип свойства такой же, как PT_LONG (PT_MV_LONG) и VT_I4 типа OLE.  <br/> |
|PT_FLOAT (PT_MV_FLOAT)  <br/> |0004  <br/> |32-разрядный (8-байтовый) значение с плавающей запятой. Этот тип свойства такой же, как PT_R4 (PT_MV_R4) и VT_R4 типа OLE.  <br/> |
|PT_DOUBLE (PT_MV_DOUBLE)  <br/> |0005  <br/> |64-разрядный (8-байтовый) значение с плавающей запятой. Этот тип свойства такой же, как PT_R8, а VT_R8 OLE Types и VT_DOUBLE.  <br/> |
|PT_CURRENCY (PT_MV_CURRENCY)  <br/> |0006  <br/> |64-разрядное (8-байтовое) целое число интерпретируется как десятичное. Этот тип свойства совместим с типом валюты Microsoft Visual Basic и совпадает с типом OLE VT_CY.  <br/> |
|PT_APPTIME (PT_MV_APPTIME)  <br/> |0007  <br/> |Значение Double, которое интерпретируется как Дата и время. Целая часть — это дата, а дробная часть — время. Этот тип свойства совпадает с типом OLE VT_DATE и совместим с представлением времени Microsoft Visual Basic.  <br/> |
|PT_ERROR  <br/> |000A  <br/> |Значение SCODE; 32-разрядное целое число без знака (4 байта). Этот тип свойства совпадает с типом OLE VT_ERROR.  <br/> |
|PT_BOOLEAN (PT_MV_12)  <br/> |000B  <br/> |16-разрядное (2-байтовое) логическое значение, где ноль равняется **false** , а не ноль равно **true**. Этот тип свойства совпадает с типом OLE VT_BOOL.  <br/> |
|PT_OBJECT  <br/> |000D  <br/> |Указатель на объект, который реализует интерфейс **IUnknown** . Этот тип свойства аналогичен нескольким типам OLE, например VT_UNKNOWN.  <br/> |
|PT_I8 (PT_MV_I8)  <br/> |0014  <br/> |64-разрядное целое число со знаком и без знака (8 байт), использующее структуру **LARGE_INTEGER** . Этот тип свойства такой же, как PT_I8, и VT_I8 типа OLE.  <br/> |
|PT_STRING8 (PT_MV_STRING8)  <br/> |001E  <br/> |8-разрядная (2-байтовая) строка символов, заканчивающаяся нулем. Этот тип свойства совпадает с типом OLE VT_LPSTR.  <br/> |
|PT_TSTRING (PT_MV_TSTRING)  <br/> |001F  <br/> |16-разрядная (2-байтовая) строка символов, заканчивающаяся нулем. Свойства с этим типом заменяют тип свойства на PT_UNICODE при компиляции с символом Юникода и PT_STRING8 при отсутствии компиляции с символом Юникода. Этот тип свойства совпадает с типом OLE VT_LPSTR, что PT_STRING8 свойства и VT_LPWSTR для свойств PT_UNICODE  <br/> |
|PT_SYSTIME (PT_MV_SYSTIME)  <br/> |0040  <br/> |64-разрядные (8-байтовые) целочисленные данные и значение времени в виде структуры **fileTime** . Этот тип свойства совпадает с типом OLE VT_FILETIME.  <br/> |
|PT_CLSID (PT_MV_CLSID)  <br/> |0048  <br/> |Значение структуры **CLSID** . Этот тип свойства совпадает с типом OLE VT_CLSID.  <br/> |
|PT_SVREID  <br/> |00FB  <br/> |Значение переменной Size, 16-разрядное (2-байтовое) **число** , за которым следует структура.  <br/> |
|PT_SRESTRICT  <br/> |00FD  <br/> |Размер переменной, массив байтов, представляющий одну или несколько структур ограничений.  <br/> |
|PT_ACTIONS  <br/> |00FE  <br/> |Размер переменной, 16-разрядное (2-байтовое) **количество** действий (не байт), за которыми следует большое количество структур действий правил.  <br/> |
|PT_BINARY (PT_MV_BINARY)  <br/> |0102  <br/> |Значение структуры **сбинари** , пересчитанный массив байтов.  <br/> |
   
> [!NOTE]
> Для определения шестнадцатеричного значения для типа многозначного свойства или флага PT_MV (0x00001000) в шестнадцатеричное значение для типа свойства. Например, шестнадцатеричным значением для PT_MV_UNICODE является 0x101F, а шестнадцатеричным значением PT_MV_BINARY является 0x1102.
