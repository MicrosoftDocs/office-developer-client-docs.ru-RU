# <a name="property-types"></a>Типы свойств

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
MAPI поддерживает свойства с одним значением и с несколькими значениями. Для свойства с одним значением существует одно значение базового типа для свойства. При наличии свойства с несколькими значениями существует несколько значений базового типа. 
  
Типы свойств с одним значением и с несколькими значениями, поддерживаемые MAPI, описаны в приведенной ниже таблице. Для каждого типа с одним значением, который имеет соответствующий многозначный тип, тип с несколькими значениями отображается в круглых скобках после типа с одним значением.
  
|**Тип свойства**|**Шестнадцатеричное значение**|**Описание**|
|:-----|:-----|:-----|
|ПТ_УНСПЕЦИФИЕД  <br/> |0000  <br/> |Указывает, что тип свойства неизвестен. Этот тип свойства зарезервирован для использования с методами интерфейса.  <br/> |
|ПТ_НУЛЛ  <br/> |0001  <br/> |Указывает, что значение свойства отсутствует. Этот тип свойства зарезервирован для использования с методами интерфейса и совпадает с типом OLE ВТ_НУЛЛ.  <br/> |
|PT_I2 (PT_MV_I2)  <br/> |0002  <br/> |16-разрядное (2-байтовое) целое число. Этот тип свойства такой же, как ПТ_ШОРТ (ПТ_МВ_ШОРТ) и OLE Type VT_I2.  <br/> |
|PT_I4 (PT_MV_I4)  <br/> |0003  <br/> |32-разрядное целое число со знаком и без знака (4 байта). Этот тип свойства такой же, как ПТ_ЛОНГ (ПТ_МВ_ЛОНГ) и OLE Type VT_I4.  <br/> |
|ПТ_ФЛОАТ (ПТ_МВ_ФЛОАТ)  <br/> |0004  <br/> |32-разрядный (8-байтовый) значение с плавающей запятой. Этот тип свойства такой же, как PT_R4 (PT_MV_R4) и OLE Type VT_R4.  <br/> |
|ПТ_ДАУБЛЕ (ПТ_МВ_ДАУБЛЕ)  <br/> |0005  <br/> |64-разрядный (8-байтовый) значение с плавающей запятой. Этот тип свойства такой же, как PT_R8, а OLE Types VT_R8 и ВТ_ДАУБЛЕ.  <br/> |
|ПТ_КУРРЕНЦИ (ПТ_МВ_КУРРЕНЦИ)  <br/> |0006  <br/> |64-разрядное (8-байтовое) целое число интерпретируется как десятичное. Этот тип свойства совместим с типом валюты Microsoft Visual Basic и совпадает с типом OLE ВТ_ЦИ.  <br/> |
|ПТ_АППТИМЕ (ПТ_МВ_АППТИМЕ)  <br/> |0007  <br/> |Значение Double, которое интерпретируется как Дата и время. Целая часть — это дата, а дробная часть — время. Этот тип свойства совпадает с типом OLE ВТ_ДАТЕ и является совместимым с представлением времени Microsoft Visual Basic.  <br/> |
|PT_ERROR  <br/> |000A  <br/> |Значение SCODE; 32-разрядное целое число без знака (4 байта). Этот тип свойства совпадает с типом OLE ВТ_ЕРРОР.  <br/> |
|ПТ_БУЛЕАН (PT_MV_12)  <br/> |000B  <br/> |16-разрядное (2-байтовое) логическое значение, где ноль равняется **false** , а не ноль равно **true**. Этот тип свойства совпадает с типом OLE ВТ_БУЛ.  <br/> |
|ПТ_ОБЖЕКТ  <br/> |000D  <br/> |Указатель на объект, который реализует интерфейс **IUnknown** . Этот тип свойства аналогичен нескольким типам OLE, например ВТ_УНКНОВН.  <br/> |
|PT_I8 (PT_MV_I8)  <br/> |0014  <br/> |64-разрядное целое число со знаком или без знака (8 байт), использующее структуру **ларже_интежер** . Этот тип свойства такой же, как PT_I8, и тип OLE VT_I8.  <br/> |
|PT_STRING8 (PT_MV_STRING8)  <br/> |001E  <br/> |8-разрядная (2-байтовая) строка символов, заканчивающаяся нулем. Этот тип свойства совпадает с типом OLE ВТ_ЛПСТР.  <br/> |
|ПТ_ТСТРИНГ (ПТ_МВ_ТСТРИНГ)  <br/> |001F  <br/> |16-разрядная (2-байтовая) строка символов, заканчивающаяся нулем. Свойства с этим типом имеют тип свойства reseted to ПТ_УНИКОДЕ при компиляции с символом Юникода и PT_STRING8 при отсутствии компиляции с символом Юникода. Этот тип свойства совпадает с типом OLE ВТ_ЛПСТР для результирующих свойств PT_STRING8 и ВТ_ЛПВСТР для свойств ПТ_УНИКОДЕ  <br/> |
|ПТ_СИСТИМЕ (ПТ_МВ_СИСТИМЕ)  <br/> |0040  <br/> |64-разрядные (8-байтовые) целочисленные данные и значение времени в виде структуры **fileTime** . Этот тип свойства совпадает с типом OLE ВТ_ФИЛЕТИМЕ.  <br/> |
|ПТ_КЛСИД (ПТ_МВ_КЛСИД)  <br/> |0048  <br/> |Значение структуры **CLSID** . Этот тип свойства совпадает с типом OLE ВТ_КЛСИД.  <br/> |
|ПТ_СВРЕИД  <br/> |00FB  <br/> |Значение переменной Size, 16-разрядное (2-байтовое) **число** , за которым следует структура.  <br/> |
|ПТ_СРЕСТРИКТ  <br/> |00FD  <br/> |Размер переменной, массив байтов, представляющий одну или несколько структур ограничений.  <br/> |
|ПТ_АКТИОНС  <br/> |00FE  <br/> |Размер переменной, 16-разрядное (2-байтовое) **количество** действий (не байт), за которыми следует большое количество структур действий правил.  <br/> |
|ПТ_БИНАРИ (ПТ_МВ_БИНАРИ)  <br/> |0102  <br/> |Значение структуры **сбинари** , пересчитанный массив байтов.  <br/> |
   
> [!NOTE]
> Для определения шестнадцатеричного значения для многозначного типа свойства или флага ПТ_МВ (0x00001000) в шестнадцатеричное значение для типа свойства. Например, шестнадцатеричное значение для ПТ_МВ_УНИКОДЕ — 0x101F, а шестнадцатеричное значение для ПТ_МВ_БИНАРИ — 0x1102.
