<span data-ttu-id="a13e5-105">Уведомления в таблице отображения отправляются поставщиком услуг, отвечающим за создание таблицы отображения в MAPI.</span><span class="sxs-lookup"><span data-stu-id="a13e5-105">Notifications on a display table are sent by the service provider responsible for creating the display table to MAPI.</span></span> <span data-ttu-id="a13e5-106">MAPI регистрирует эти уведомления, вызывая метод [IMAPITable::Advise](imapitable-advise.md) таблицы отображения и указав событие изменения таблицы.</span><span class="sxs-lookup"><span data-stu-id="a13e5-106">MAPI registers for these notifications by calling a display table's [IMAPITable::Advise](imapitable-advise.md) method and specifying the table modified event.</span></span>
