<span data-ttu-id="05b13-105">Уведомления для отображаемой таблицы отправляются поставщиком услуг, который отвечает за создание таблицы отображения для MAPI.</span><span class="sxs-lookup"><span data-stu-id="05b13-105">Notifications on a display table are sent by the service provider responsible for creating the display table to MAPI.</span></span> <span data-ttu-id="05b13-106">Регистрирует регистры MAPI для этих уведомлений, вызывая метод [IMAPITable:: Advise](imapitable-advise.md) и указывая событие Modified таблицы.</span><span class="sxs-lookup"><span data-stu-id="05b13-106">MAPI registers for these notifications by calling a display table's [IMAPITable::Advise](imapitable-advise.md) method and specifying the table modified event.</span></span>
