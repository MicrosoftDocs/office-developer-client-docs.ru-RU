 <span data-ttu-id="005cb-106">У **IUnknown** есть три метода: [IUnknown:: AddRef](https://msdn.microsoft.com/library/ms691379%28v=VS.85%29.aspx), [IUnknown:: **QueryInterface**два объекта, не имеющие сведений о функциях друг друга, могут взаимодействовать друг с другом.</span><span class="sxs-lookup"><span data-stu-id="005cb-108">With **QueryInterface**, <span data-ttu-id="005cb-111">Когда **QueryInterface** возвращает запрошенный указатель интерфейса, он также должен увеличить число ссылок нового объекта.</span><span class="sxs-lookup"><span data-stu-id="005cb-111">When **QueryInterface** returns a requested interface pointer, it must also increase the new object's reference count.</span></span> <span data-ttu-id="005cb-112">Число ссылок на объект — это числовое значение, используемое для управления сроком действия объекта.</span><span class="sxs-lookup"><span data-stu-id="005cb-112">An object's reference count is a numeric value used to manage the object's lifespan.</span></span> <span data-ttu-id="005cb-113">Если значение счетчика ссылок больше 1, память объекта не может быть освобождена, так как она используется в активном состоянии.</span><span class="sxs-lookup"><span data-stu-id="005cb-113">When the reference count is greater than 1, the object's memory cannot be freed because it is actively being used.</span></span> <span data-ttu-id="005cb-114">Он используется только в том случае, когда число ссылок сбрасывается до 0, что объект может безопасно освобождаться.</span><span class="sxs-lookup"><span data-stu-id="005cb-114">It is only when the reference count drops to 0 that the object can be released safely.</span></span>
