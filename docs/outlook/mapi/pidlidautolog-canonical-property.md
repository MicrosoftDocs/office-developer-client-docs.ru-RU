# <a name="pidlidautolog-canonical-property"></a>Каноническое свойство PidLidAutoLog

  
  
**Относится к**: Outlook 
  
Указывает, следует ли создать журнал для каждого действия, связанные с этим контактом.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidAutoLog  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008025  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Контакт  <br/> |
