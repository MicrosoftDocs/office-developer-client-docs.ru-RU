# <a name="pidtagoriginatingmtacertificate-canonical-property"></a>Каноническое свойство PidTagOriginatingMtaCertificate

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор сообщения передачи агента (), с которого поступило сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINATING_MTA_CERTIFICATE  <br/> |
|Идентификатор:  <br/> |0x0E25  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Сервер  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство, если задано, доступные на отправляются сообщения в папке «Отправленные».
  
Это свойство соответствует атрибуту X.400 отчета-message.
