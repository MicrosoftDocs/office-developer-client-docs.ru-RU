# <a name="pidtagoriginatingmtacertificate-canonical-property"></a>Каноническое свойство PidTagOriginatingMtaCertificate

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор агента передачи сообщений (MTA), отправившего сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ОРИГИНАТИНГ_МТА_ЦЕРТИФИКАТЕ  <br/> |
|Идентификатор:  <br/> |0x0E25  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Сервер  <br/> |
   
## <a name="remarks"></a>Комментарии

Это свойство, если задано, доступно для отправленных сообщений в папке "Отправленные".
  
Это свойство соответствует атрибуту отчета X. 400 для каждого сообщения.
