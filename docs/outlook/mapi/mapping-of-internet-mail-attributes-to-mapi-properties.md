# <a name="mapping-of-internet-mail-attributes-to-mapi-properties"></a>Сопоставление атрибутов интернет-почты с свойствами MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
В этом приложении описывается, как поставщик транспорта MAPI или шлюз MAPI, который подключается к Интернету, должен переводить между свойствами сообщений MAPI и атрибутами сообщения Simple Mail Transport Protocol (SMTP). SMTP — это протокол обмена сообщениями, используемый на большей части Интернета. SMTP определяет набор заглавных заглавных сообщений — конверт сообщений — и формат контента сообщений. SMTP полностью задокументирован в наборе двух документов RFC 821 и RFC 822, которые можно найти на нескольких сайтах FTP и WWW в Интернете.
  
Сведения о протоколе SMTP, используемом для связи с почтовыми агентами на основе SMTP, см. в рубрике RFC 821, "Простой протокол передачи почты". [https://www.rfc-editor.org](https://www.rfc-editor.org)
  
Для адресов и стандартных заглавных заглавных сообщений см. в тексте RFC 822", "Стандарт для формата текстовых сообщений в Интернете [https://www.rfc-editor.org](https://www.rfc-editor.org) ARPA".
  
Для MIME см. в рубрике RFC 1521", "MIME (многоцелевые расширения интернет-почты) Часть 1: механизмы указания и описания формата органов интернет-сообщений". [https://www.rfc-editor.org](https://www.rfc-editor.org)
  
Цель сопоставления атрибутов SMTP-сообщений со свойствами MAPI (и наоборот) заключается в том, чтобы обеспечить надежное обмен полным контентом сообщений MAPI, которые могут быть закодированы с использованием атрибутов сообщений SMTP, между различными компонентами MAPI, которые должны общаться через Интернет. Этот документ основан на уже проделаной работе по таким компонентам в Корпорации Майкрософт. 
  
В этом документе предполагается знакомство с транспортами MAPI, TNEF и SMTP-почтой. Он стремится быть кратким, а не предельно понятным.
  
Как конвенция, "исходящие" относится к почте, перемещающейся из UA или MTA, совместимого с MAPI, в Интернет, а "входящие" — к почте, перемещающейся из Интернета в компонент MAPI.
