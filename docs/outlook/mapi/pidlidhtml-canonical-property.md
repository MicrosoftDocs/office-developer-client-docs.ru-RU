# <a name="pidlidhtml-canonical-property"></a>Каноническое свойство PidLidHtml

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает URL-адрес бизнес-страницы контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidHTML  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000802B  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

При этом значение этого свойства должно быть таким же, как и значение **свойства PR_BUSINESS_HOME_PAGE** [(PidTagBusinessHomePage).](pidtagbusinesshomepage-canonical-property.md)
