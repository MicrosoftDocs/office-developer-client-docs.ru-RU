# <a name="local-form-libraries"></a>Локальные библиотеки форм

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Локальные библиотеки форм хранятся непосредственно на компьютере пользователя, как правило, в файле FRMCACHE.DAT. 
  
Клиентские приложения могут получать доступ к формам в локальной библиотеке форм без доступа к сетевым ресурсам. Локальные библиотеки форм являются исключением из правила, которое хранится в связанных таблицах контента, так как локализованная библиотека форм — это просто файл на диске, который не входит в иерархию папок MAPI.
