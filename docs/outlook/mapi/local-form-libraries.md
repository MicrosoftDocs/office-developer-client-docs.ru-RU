# <a name="local-form-libraries"></a>Локальные библиотеки форм

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Локальные библиотеки форм хранятся непосредственно на компьютере пользователя, обычно в файле с именем ФРМКАЧЕ. DAT. 
  
Клиентские приложения могут получать доступ к формам в локальной библиотеке форм, не обращаясь к сетевым ресурсам. Локальные библиотеки форм — это исключение из правила, которое формы хранятся в связанных таблицах содержимого, так как локальная Библиотека форм — это просто файл на диске, который не является частью иерархии папок MAPI.
