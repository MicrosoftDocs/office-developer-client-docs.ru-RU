# <a name="local-form-libraries"></a>Локальные библиотеки форм

  
  
**Относится к**: Outlook 
  
Библиотеки форм локального хранятся непосредственно на компьютере пользователя, обычно в файл с именем FRMCACHE.DAT. 
  
Клиентские приложения можно получить доступ к формы в библиотеке форм локального без доступа к ресурсам сети. Библиотеки форм локального являются исключение в правило, что формы хранятся в таблицах связанное содержимое с момента локальной библиотеки — это просто файл на диске, который не является частью любой иерархии папок MAPI.
