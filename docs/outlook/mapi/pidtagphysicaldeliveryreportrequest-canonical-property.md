# <a name="pidtagphysicaldeliveryreportrequest-canonical-property"></a>Каноническое свойство PidTagPhysicalDeliveryReportRequest

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит режим отчета, который будет доставлен определенному получателю сообщения после завершения физической доставки сообщения системой обработки сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PHYSICAL_DELIVERY_REPORT_REQUEST  <br/> |
|Идентификатор:  <br/> |0x0C0C  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Получатель MAPI  <br/> |
