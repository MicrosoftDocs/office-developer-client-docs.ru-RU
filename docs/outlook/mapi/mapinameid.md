# <a name="mapinameid"></a>MAPINAMEID

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Описывает именованное свойство. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
   
```cpp
typedef struct _MAPINAMEID
{
  LPGUID lpguid;
  ULONG ulKind;
  union
  {
    LONG lID;
    LPWSTR lpwstrName;
  } Kind;
} MAPINAMEID, FAR *LPMAPINAMEID;

```

## <a name="members"></a>Members

 **лпгуид**
  
> Указатель на структуру [GUID](guid.md) , определяющую определенный набор свойств; Этот элемент не может иметь значение NULL. Допустимы следующие значения: 
    
ПС_ПУБЛИК_СТРИНГС
  
> 
    
ПС_МАПИ
  
> 
    
Значение, определяемое клиентом
  
> 
    
 **Улкинд**
  
> Значение, описывающее тип значения в члене **типа** . Допустимы следующие значения: 
    
MNID_ID 
  
> Элемент **Kind** содержит целое значение, представляющее имя свойства. 
    
MNID_STRING 
  
> Элемент **Kind** содержит строку символов Юникода, представляющую имя свойства. 
    
 **Kind**
  
> Union, описывающий имя именованного свойства. Имя может быть целым числом, хранящимся в **крышки**, или строкой символов Юникода, хранящимся в **лпвстрнаме**.
    
## <a name="remarks"></a>Примечания

Структура **мапинамеид** используется для описания свойств именованных свойств, идентификаторы которых превыше 0x8000. Набор свойств является важной частью именованного свойства. Например, ПС_ПУБЛИК_СТРИНГС или ПС_РАУТИНГ_АДДРТИПЕ — это наборы свойств, определенные MAPI. 
  
Именованные свойства позволяют клиентам определять настраиваемые свойства в пространстве имен, превышающем доступное в диапазоне идентификаторов свойств, заданных MAPI. Имена свойств нельзя использовать для непосредственного получения значений свойств; Сначала они должны быть сопоставлены с идентификаторами свойств с помощью метода [IMAPIProp:: жетидсфромнамес](imapiprop-getidsfromnames.md) . Для определенных объектов, например сообщений, в MAPI резервируется диапазон идентификаторов свойств для настраиваемых свойств. Таким образом, для этих объектов клиентам не нужно использовать именованные свойства и могут сохранять связанные издержки. 
  
Более подробную информацию об именованных свойствах можно узнать в статье [именованНые свойства](mapi-named-properties.md).
