# <a name="pidlidworkaddressstate-canonical-property"></a>Каноническое свойство PidLidWorkAddressState

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает область штата или района для почтового адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидворкаддрессстате  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008047  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
