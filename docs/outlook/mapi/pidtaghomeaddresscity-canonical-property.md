# <a name="pidtaghomeaddresscity-canonical-property"></a>Каноническое свойство PidTagHomeAddressCity

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит город для домашнего адреса получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_ADDRESS_CITY, PR_HOME_ADDRESS_CITY_A, PR_HOME_ADDRESS_CITY_W  <br/> |
|Идентификатор:  <br/> |0x3A59  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
