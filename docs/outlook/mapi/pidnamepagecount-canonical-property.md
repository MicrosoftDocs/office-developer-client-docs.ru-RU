# <a name="pidnamepagecount-canonical-property"></a>Каноническое свойство PidNamePageCount

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает количество страниц файла, вложенного в документ.
  
|||
|:-----|:-----|
|Дружелюбные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Имя свойства:  <br/> |PageCount  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Распространенная  <br/> |
