# <a name="pidlidcontactuserfield1-canonical-property"></a>Каноническое свойство PidLidContactUserField1

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает первое поле контакта, предназначенное для использования в других контактах.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidContactUserField1  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000804F  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство содержит текст, не связанный с каким-либо другим свойством, зависящим от контакта.
