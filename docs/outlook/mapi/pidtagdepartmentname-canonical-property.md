# <a name="pidtagdepartmentname-canonical-property"></a>Каноническое свойство PidTagDepartmentName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя отдела, в котором работает получатель. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DEPARTMENT_NAME, PR_DEPARTMENT_NAME_A PR_DEPARTMENT_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A18  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства обеспечивают идентификацию и доступ к сведениям о получателе. Они определяются получателем и их организацией.
