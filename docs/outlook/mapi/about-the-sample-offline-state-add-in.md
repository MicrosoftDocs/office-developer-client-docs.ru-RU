# <a name="about-the-sample-offline-state-add-in"></a>Пример надстройки состояния в автономном режиме

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
API состояния автономного доступа поддерживает вызовы, указывающие на изменения в состоянии подключения пользователя в Outlook, например, от подключения к сети в Outlook до отключения. Пример надстройки состояния в автономном режиме — это надстройка COM, написанная в C++ и демонстрирующая получение уведомлений об изменениях состояния подключения и изменение текущего состояния с помощью API состояния автономного доступа. Дополнительные сведения об API автономного режима см. в статье [Об API автономного режима](about-the-offline-state-api.md).
