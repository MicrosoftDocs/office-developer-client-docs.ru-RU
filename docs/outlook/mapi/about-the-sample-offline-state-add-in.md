# <a name="about-the-sample-offline-state-add-in"></a>Сведения о примере надстройки с автономным состоянием

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
API автономного состояния поддерживает обратные вызовы, указывающие на изменения состояния подключения пользователя в Outlook (например, из сети Outlook в автономный режим). Надстройка "автономное состояние" — это надстройка COM, написанная на C++, которая показывает, как получать уведомления об изменениях состояния подключения и как изменять текущее состояние с помощью API автономного состояния. Дополнительные сведения об API автономного режима см. в статье [Об API автономного режима](about-the-offline-state-api.md).
