<span data-ttu-id="a29e9-105">Некоторым пользователям требуются службы нескольких систем обмена сообщениями, каждая из которых имеет одного или нескольких поставщиков услуг.</span><span class="sxs-lookup"><span data-stu-id="a29e9-105">Some users require the services of several messaging systems, each with one or more service providers.</span></span> <span data-ttu-id="a29e9-106">Поскольку устанавливать и настраивать каждого из этих поставщиков услуг по отдельности, а также потому, что серверу обмена сообщениями обычно требуется группа связанных поставщиков для представления всех его функций, MAPI включает концепцию службы сообщений.</span><span class="sxs-lookup"><span data-stu-id="a29e9-106">Because it is cumbersome to install and configure each of these service providers individually, and because a messaging server usually requires a group of related providers to expose all its functionality, MAPI includes the concept of a message service.</span></span> <span data-ttu-id="a29e9-107">Службы сообщений помогают пользователям устанавливать и настраивать поставщиков услуг.</span><span class="sxs-lookup"><span data-stu-id="a29e9-107">Message services help users install and configure their service providers.</span></span>
  
<span data-ttu-id="a29e9-108">Чтобы создать службу сообщений, разработчик записывает программу точки входа службы сообщений для обработки конфигурации каждого поставщика в службе и программу установки, чтобы сделать следующее:</span><span class="sxs-lookup"><span data-stu-id="a29e9-108">To create a message service, a developer writes a message-service entry point program to handle the configuration of each provider in the service and a setup program to do the following:</span></span>

<span data-ttu-id="a29e9-112">Файл Mapisvc.inf содержит сведения, которые связаны с конфигурацией всех служб сообщений и поставщиков служб, установленных на компьютере.</span><span class="sxs-lookup"><span data-stu-id="a29e9-112">The Mapisvc.inf file contains information that relates to the configuration of all message services and service providers installed on the computer.</span></span> <span data-ttu-id="a29e9-113">Она организована в иерархические разделы, каждый уровень которых связан со следующим.</span><span class="sxs-lookup"><span data-stu-id="a29e9-113">It is organized in hierarchical sections, with each level linked to the next.</span></span> <span data-ttu-id="a29e9-114">В верхней части три раздела содержат следующие разделы:</span><span class="sxs-lookup"><span data-stu-id="a29e9-114">At the top are three sections that contain the following:</span></span>

- <span data-ttu-id="a29e9-116">Список наиболее важных (или стандартных) служб сообщений.</span><span class="sxs-lookup"><span data-stu-id="a29e9-116">A list of the most important, or default, message services.</span></span>
    
- <span data-ttu-id="a29e9-117">Список всех служб на компьютере.</span><span class="sxs-lookup"><span data-stu-id="a29e9-117">A list of all the services on the computer.</span></span>
    
<span data-ttu-id="a29e9-118">Следующий уровень содержит разделы для каждой службы сообщений, а последний уровень содержит разделы для каждого поставщика службы в службе.</span><span class="sxs-lookup"><span data-stu-id="a29e9-118">The next level contains sections for each message service, and the last level contains sections for each service provider in a service.</span></span> <span data-ttu-id="a29e9-119">MAPI требует, чтобы разработчики поставщиков услуг и служб сообщений добавляли определенные записи в mapisvc.inf; <span data-ttu-id="a29e9-124">Профили создаются во время установки службы сообщений или во время их логотипа клиентского приложения, которое обеспечивает поддержку конфигурации.</span><span  
<span data-ttu-id="a29e9-131">В то время как элемент панели управления всегда вызывается с помощью панели управления, существует множество сценариев, которые могут привести к вызову мастера
