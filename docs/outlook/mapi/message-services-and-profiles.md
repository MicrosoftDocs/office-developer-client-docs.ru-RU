<span data-ttu-id="db995-105">Некоторым пользователям требуются службы нескольких систем обмена сообщениями, каждая из которых имеет одного или нескольких поставщиков услуг.</span><span class="sxs-lookup"><span data-stu-id="db995-105">Some users require the services of several messaging systems, each with one or more service providers.</span></span> <span data-ttu-id="db995-106">Поскольку установка и настройка каждого из этих поставщиков услуг по отдельности является громоздкой задачей, а поскольку серверу обмена сообщениями обычно требуется группа связанных поставщиков, чтобы раскрыть все его функции, MAPI включает концепцию службы сообщений.</span><span class="sxs-lookup"><span data-stu-id="db995-106">Because it is cumbersome to install and configure each of these service providers individually, and because a messaging server usually requires a group of related providers to expose all its functionality, MAPI includes the concept of a message service.</span></span> <span data-ttu-id="db995-107">Службы сообщений помогают пользователям устанавливать и настраивать поставщиков услуг.</span><span class="sxs-lookup"><span data-stu-id="db995-107">Message services help users install and configure their service providers.</span></span>

- <span data-ttu-id="db995-116">Список наиболее важных или по умолчанию служб сообщений.</span><span class="sxs-lookup"><span data-stu-id="db995-116">A list of the most important, or default, message services.</span></span>

<span data-ttu-id="db995-118">Следующий уровень содержит разделы для каждой службы сообщений, а последний уровень содержит разделы для каждого поставщика услуг в службе.</span><span class="sxs-lookup"><span data-stu-id="db995-118">The next level contains sections for each message service, and the last level contains sections for each service provider in a service.</span></span> <span data-ttu-id="db995-119">MAPI требует, чтобы разработчики поставщиков услуг и служб сообщений добавляли определенные записи в Mapisvc.inf; <span data-ttu-id="db995-127">Мастер профилей — это простое приложение, предназначенное для максимально простого добавления службы сообщений в профиль.</span><span class="sxs-lookup"><span <span data-ttu-id="db995-129">Пользователь получает запрос только для значений для наиболее важных параметров; все остальные параметры наследуют значения по умолчанию.</span><span
