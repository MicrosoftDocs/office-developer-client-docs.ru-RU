# <a name="pidtagnull-canonical-property"></a>Каноническое свойство PidTagNull

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Представляет значение null или параметр свойства или резервировать пространство массива.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_NULL  <br/> |
|Идентификатор:  <br/> |0x0000  <br/> |
|Тип данных:  <br/> |PT_NULL  <br/> |
|Область:  <br/> |Распространенная  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство используется для резерва пространства в массивах [структур SPropValue.](spropvalue.md) Он используется в массиве структур [SPropTagArray,](sproptagarray.md) чтобы сообщить методу о резервировке пространства в возвращаемом массиве **структур SPropValue.** Это позволяет заполнять вычисляемую емким способом. 
  
Дополнительные сведения см. в обзоре типов [свойств MAPI.](mapi-property-type-overview.md)
