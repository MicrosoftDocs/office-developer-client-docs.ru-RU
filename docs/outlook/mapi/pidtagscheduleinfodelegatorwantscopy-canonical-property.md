# <a name="pidtagscheduleinfodelegatorwantscopy-canonical-property"></a>Каноническое свойство PidTagScheduleInfoDelegatorWantsCopy

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение TRUE, если представитель хочет получать копии связанных с собранием объектов, отправленных представителю.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_СЧДИНФО_БОСС_ВАНТС_КОПИ  <br/> |
|Идентификатор:  <br/> |0x6842  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Сведения о доступности  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство должно быть задано в объекте сведений о делегате.
