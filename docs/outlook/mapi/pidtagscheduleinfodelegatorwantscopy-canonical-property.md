# <a name="pidtagscheduleinfodelegatorwantscopy-canonical-property"></a>Каноническое свойство PidTagScheduleInfoDelegatorWantsCopy

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит true, если делегатор хочет получить копии объектов, связанных с собранием, которые отправляются делегату.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SCHDINFO_BOSS_WANTS_COPY  <br/> |
|Идентификатор:  <br/> |0x6842  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Free/Busy  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть установлено в объекте делегирования информации.
