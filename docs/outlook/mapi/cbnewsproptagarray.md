# <a name="cbnewsproptagarray"></a>CbNewSPropTagArray

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество вбайтов, выделяемого для новой [структуры SPropTagArray,](sproptagarray.md) которая содержит указанное число тегов свойств. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |**SPropTagArray** <br/> |
   
```cpp
CbNewSPropTagArray (_ctag)
```

## <a name="parameters"></a>Параметры

 _ _ctag_
  
> Количество тегов свойств, которые необходимо включить в новую **структуру SPropTagArray.**
