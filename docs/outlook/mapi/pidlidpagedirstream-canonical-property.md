# <a name="pidlidpagedirstream-canonical-property"></a>Каноническое свойство PidLidPageDirStream

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Представляет сведения о страницах настраиваемой формы, связанной с сообщением.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидпажедирстреам  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008513  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Настройка времени выполнения  <br/> |
