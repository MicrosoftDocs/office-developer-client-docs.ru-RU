# <a name="pidlidpagedirstream-canonical-property"></a>Каноническое свойство PidLidPageDirStream

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Представляет сведения о страницах настраиваемой формы, связанных с сообщением.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidPageDirStream  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008513  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конфигурация времени запуска  <br/> |
