# <a name="pidtagotheraddresscountry-canonical-property"></a>Каноническое свойство PidTagOtherAddressCountry

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит другую страну или регион пользователя почтового ящика.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OTHER_ADDRESS_COUNTRY, PR_OTHER_ADDRESS_COUNTRY_A PR_OTHER_ADDRESS_COUNTRY_W  <br/> |
|Идентификатор:  <br/> |0x3A60  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Адресная книга  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
