# <a name="install-mapi-header-files"></a>Установить файлы заголовков MAPI

**Применимо к**: Outlook 2013 | Outlook 2016 
  
В этом разделе описывается, как получить и установить файлы заголовков MAPI и файлов библиотеки.

Файлы заголовков MAPI, доступны для загрузки на [Outlook 2010: файлы заголовков MAPI](http://www.microsoft.com/downloads/details.aspx?FamilyID=f8d01fc8-f7b5-4228-baa3-817488a66db1). По умолчанию, файлы заголовков MAPI устанавливаются в следующую папку: C:\Program Files\Microsoft SDKs\Office\14.0\Include.
