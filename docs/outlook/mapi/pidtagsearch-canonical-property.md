# <a name="pidtagsearch-canonical-property"></a>Каноническое свойство PidTagSearch

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит объект Container, который используется для расширенного поиска.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_СЕАРЧ  <br/> |
|Идентификатор:  <br/> |0x3607  <br/> |
|Тип данных:  <br/> |ПТ_ОБЖЕКТ  <br/> |
|Область:  <br/> |Контейнер адресной книги  <br/> |
   
## <a name="remarks"></a>Комментарии

В контейнере, не поддерживающем расширенные возможности поиска, не требуется указывать свойство.
