# <a name="pidtagsearch-canonical-property"></a>Каноническое свойство PidTagSearch

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит объект контейнера, используемый для расширенных поисков.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SEARCH  <br/> |
|Идентификатор:  <br/> |0x3607  <br/> |
|Тип данных:  <br/> |PT_OBJECT  <br/> |
|Область:  <br/> |Контейнер адресной книги  <br/> |
   
## <a name="remarks"></a>Примечания

Контейнер, который не поддерживает расширенные возможности поиска, не должен предоставить свойство.
