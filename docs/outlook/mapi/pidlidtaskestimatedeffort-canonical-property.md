# <a name="pidlidtaskestimatedeffort-canonical-property"></a>Каноническое свойство PidLidTaskEstimatedEffort

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Указывает количество времени, в минутах, что необходима для выполнения задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskEstimatedEffort  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008111  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Замечания

Значение должно быть больше или равно 0 и меньше, чем 0x5AE980DF (1,525,252,319), где 480 минут равно один день и 2400 минут равно 1 неделя (8 часов рабочего дня и пять дней в неделе).
