# <a name="pidtagcontactversion-canonical-property"></a>Каноническое свойство PidTagContactVersion

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит версию контактных данных получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONTACT_VERSION  <br/> |
|Идентификатор:  <br/> |0x3A52  <br/> |
|Тип данных:  <br/> |PT_CLSID  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является одним из свойств, которое предоставляет идентификацию и доступ к данным о получателе. Эти свойства определяются получателем и организацией получателя.
