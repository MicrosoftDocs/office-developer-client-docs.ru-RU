# <a name="pidtagconflictitems-canonical-property"></a>Каноническое свойство PidTagConflictItems

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит один или несколько идентификаторов записей элементов, которые участвовали в автоматическом разрешении конфликтов.
  
## 

|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КОНФЛИКТ_ИТЕМС  <br/> |
|Идентификатор:  <br/> |0x1098  <br/> |
|Тип свойства:  <br/> |PT_MV_BINARY  <br/> |
|Область:  <br/> |ICS  <br/> |
   
## <a name="remarks"></a>Комментарии

Типы стандартных элементов Microsoft Outlook, поддерживающих автоматическое разрешение конфликтов, включают следующие стандартные типы элементов: элементы встреч, элементы контактов, элементы дневника, элементы почты, элементы собраний, элементы клейких заметок и элементы задач. Элемент, принадлежащий классу сообщения, который является производным от одного из этих стандартных типов элементов, также поддерживает автоматическое разрешение конфликтов. В Microsoft Outlook 2003 и Microsoft Office Outlook 2007, когда Outlook синхронизирует элементы и считает, что итоговая копия может не содержать все важные данные, Outlook сохраняет конфликтующие копии в конфликте **** папку в папке " **ошибки синхронизации** ". 
  
> [!NOTE]
> **Проблемы синхронизации** и вложенные папки скрываются до тех пор, пока не будет выбран пункт **список папок** в меню **Перейти** . 
  
Элемент предоставляет свойство **пр_конфликт_итемс** , если это один из типов элементов, поддерживающих автоматическое разрешение конфликтов, выиграл разрешение конфликтов или размещается в папке **конфликтов** из-за разрешения конфликтов. Папка, в которой размещается элемент, определяет содержимое **пр_конфликт_итемс**. Если элемент находится в папке, отличной от папки **конфликтов** , а элемент предоставляет свойство **пр_конфликт_итемс** , элемент должен иметь разрешение конфликтов, а **Пр_конфликт_итемс** содержит один или несколько идентификаторов записей элементы, потерянные при разрешении конфликтов. Если элемент находится в папке **конфликтов** , а элемент предоставляет свойство **пр_конфликт_итемс** , этот элемент должен потерял разрешение конфликтов, а **пр_конфликт_итемс** будет содержать идентификатор элемента, который выиграл конфликт. решение.
