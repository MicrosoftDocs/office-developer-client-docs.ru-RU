# <a name="pidlidtaskglobalid-canonical-property"></a>Каноническое свойство PidLidTaskGlobalId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Находит существующую задачу, используя GUID, после получения ответа на задачу или обновления задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskGlobalObjId  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008519  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство не замечено для ненаписаных задач.
