# <a name="pidlidtaskglobalid-canonical-property"></a>Каноническое свойство PidLidTaskGlobalId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Ищет существующую задачу с помощью идентификатора GUID при получении ответа задачи или обновления задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидтаскглобалобжид  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008519  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство оставлено неопределенным для неназначенных задач.
