# <a name="pidlidsharingproviderguid-canonical-property"></a>Каноническое свойство PidLidSharingProviderGuid

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает глобально уникальный идентификатор поставщика общего доступа (GUID). Это свойство общего сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSharingProviderGuid  <br/> |
|Набор свойств:  <br/> |PSETID_Sharing  <br/> |
|Long ID (LID):  <br/> |0x00008A01  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Доступ  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть задавано "%xAE.F0.06.00.00.00.00.00.C0.00.00.00.46".
