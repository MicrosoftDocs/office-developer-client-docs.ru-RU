# <a name="pidlidsharingproviderguid-canonical-property"></a>Каноническое свойство PidLidSharingProviderGuid

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает глобальный уникальный идентификатор (GUID) поставщика общего доступа. Это свойство сообщения о совместном доступе.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидшарингпровидергуид  <br/> |
|Набор свойств:  <br/> |Псетид_шаринг  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008A01  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Общий доступ  <br/> |
   
## <a name="remarks"></a>Комментарии

Значение этого свойства должно быть равно "% Ксае. F 0.06.00.00.00.00.00. C 0.00.00.00.00.00.00.46".
