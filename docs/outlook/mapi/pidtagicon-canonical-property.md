# <a name="pidtagicon-canonical-property"></a>Каноническое свойство PidTagIcon

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит толограмму значка полного размера формы. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ICON  <br/> |
|Идентификатор:  <br/> |0x0FFD  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |MAPI, не передаваемый  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство содержит изображение значка размером 32 × 32 пикселя, то же самое, что и содержимое значка. ICO-файл. Обычно это свойство копируется из . ICO-файл, указанный в строке LargeIcon соответствующего раздела [Описание] файла конфигурации формы.
