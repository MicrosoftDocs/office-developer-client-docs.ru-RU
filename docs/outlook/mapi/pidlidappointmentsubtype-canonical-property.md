# <a name="pidlidappointmentsubtype-canonical-property"></a>Каноническое свойство PidLidAppointmentSubType

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, является ли событие событием весь день.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptSubType  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008215  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Календарь  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство указывает, является ли событие событием на весь день, как указано пользователем. Значение TRUE указывает, что событие является событием на весь день, в этом случае время начала и окончания должно быть полночью, чтобы продолжительность была кратной 24 часам и не менее 24 часов. Значение FALSE или отсутствие этого свойства указывает, что событие не является событием на весь день. Клиент или сервер не должны высмеять значение TRUE, когда пользователь создает событие, которое составляет 24 часа, даже если событие начинается и заканчивается в полночь.
