# <a name="cbnewspropattrarray"></a>CbNewSPropAttrArray

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество bytes, которые должны быть выделены для новой [структуры SPropAttrArray,](spropattrarray.md) которая содержит определенное количество атрибутов свойств. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Imessage.h  <br/> |
|Связанная структура:  <br/> |**SPropAttrArray** <br/> |
   
```cpp
CbNewSPropAttrArray (_cattr)
```

## <a name="parameters"></a>Parameters

 _ _cattr_
  
> Количество атрибутов свойств, которые будут включены в новую **структуру SPropAttrArray.**
