# <a name="cbnewspropattrarray"></a>CbNewSPropAttrArray

  
  
**Относится к**: Outlook 
  
Вычисляет число байтов, должен быть размещен для новой структуры [SPropAttrArray](spropattrarray.md) , содержащий указанное число атрибуты свойства. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |IMessage.h  <br/> |
|Связанные структуры:  <br/> |**SPropAttrArray** <br/> |
   
```cpp
CbNewSPropAttrArray (_cattr)
```

## <a name="parameters"></a>Параметры

 __cattr_
  
> Количество атрибутов свойств, должны быть включены в новой структуры **SPropAttrArray** .
