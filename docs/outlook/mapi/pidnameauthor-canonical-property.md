# <a name="pidnameauthor-canonical-property"></a>Каноническое свойство PidNameAuthor

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Представляет автора файла, присоединенного к документу.
  
|||
|:-----|:-----|
|Дружественные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Имя свойства:  <br/> |Автор  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Распространенная  <br/> |
