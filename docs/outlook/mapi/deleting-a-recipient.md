# <a name="deleting-a-recipient"></a>Удаление получателя

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
 **Удаление одной или нескольких записей адресной книги из изменяемого контейнера**
  
- ВыЗовите метод [иабконтаинер::D елетинтриес](iabcontainer-deleteentries.md) , передав массив идентификаторов записей, представляющих удаляемые записи адресной книги. **Делетинтриес** может возвращать предупреждение, мапи_в_партиал_комплетион, чтобы показать, что не удалось удалить одну или несколько записей. Проверьте возвращаемое значение с помощью макроса **хр_фаилед** и вызовите метод Container [IMAPIProp:: GetLastError](imapiprop-getlasterror.md) , если требуется больше сведений о проблеме. 
    
Когда вы наводите указатель на структуру [адрентри](adrentry.md) удаленного элемента в кэше, вы по-прежнему сможете получать свойства с помощью идентификатора записи. Это вызвано тем, что запись отмечена только для удаления. MAPI поддерживает уровень доступа к этим помеченным записям по проектированию.
