# <a name="dtblcombobox"></a>DTBLCOMBOBOX

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Описание управления полем со combo, который будет использоваться в диалоговом окне, построенного из таблицы отображения.
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанный макрос:  <br/> |[SizedDtblComboBox](sizeddtblcombobox.md) <br/> |
   
```cpp
typedef struct _DTBLCOMBOBOX
{
  ULONG ulbLpszCharsAllowed;
  ULONG ulFlags;
  ULONG ulNumCharsAllowed;
  ULONG ulPRPropertyName;
  ULONG ulPRTableName;
} DTBLCOMBOBOX, FAR *LPDTBLCOMBOBOX;

```

## <a name="members"></a>"Участники"

 **ulbLpszCharsAllowed**
  
> Смещение от начала структуры **DTBLCOMBOBOX** к фильтру строк символов, который описывает ограничения (если таковые есть) на символы, которые можно ввести в поле со combo. Фильтр не интерпретируется как регулярное выражение, и к каждому введенному символу применяется один и тот же фильтр. Фильтр имеет такой формат: 
    
|**Символ**|**Описание**|
|:-----|:-----|
| `*` <br/> |Любой символ разрешен (например,  `"*"` ).  <br/> |
| `[ ]` <br/> |Определяет набор символов (например,  `"[0123456789]"` ).  <br/> |
| `-` <br/> |Указывает диапазон символов (например,  `"[a-z]"` ).  <br/> |
| `~` <br/> |Указывает, что эти символы не разрешены. (например,  `"[~0-9]"` ).  <br/> |
| `\` <br/> |Используется для кавычка любых из предыдущих символов (например,  `"[\-\\\[\]]"` знаки -, \, [и ] разрешены).  <br/> |
   
 **ulFlags**
  
> Битоваяmas флагов, используемых для обозначения формата фильтра строки символов. Можно установить следующий флаг:
    
MAPI_UNICODE 
  
> Фильтр имеет формат Юникод. Если флаг MAPI_UNICODE не установлен, фильтр будет в формате ANSI.
    
 **ulNumCharsAllowed**
  
> Максимальное количество символов, которые могут быть введены в текстовом поле со полем со полем со значением со значением.
    
 **ulPRPropertyName**
  
> Тег свойства для свойства типа PT_TSTRING. 
    
 **ulPRTableName**
  
> Тег свойства для свойства типа PT_OBJECT, в котором интерфейс **IMAPITable** можно открыть с помощью вызова **OpenProperty.** В таблице должен быть один столбец со свойством того же типа, что и свойство, определенное членом **ulPRPropertyName.** Строки таблицы используются для заполнения списка. 
    
## <a name="remarks"></a>Примечания

Структура **DTBLCOMBOBOX** описывает поле со списком, которое состоит из списка и поля выбора. В списке представлены сведения, из которых пользователь может выбрать, а в поле выбора отображается текущий выбор. Поле выбора — это редактируемая область, которую также можно использовать для ввода текста, который еще не находится в списке. 
  
Два члена тега свойства работают вместе для координации отображения списка с помощью управления редактированием. При первом отображении окна со combo MAPI вызывает метод **OpenProperty** реализации **IMAPIProp,** связанный с таблицей отображения, чтобы получить таблицу, представленную членом **ulPRTableName.** В этой таблице имеется один столбец, содержащий значения свойства, представленного членом **ulPRPropertyName.** Поэтому этот столбец должен иметь тот же тип, что и свойство **ulPRPropertyName,** а оба столбца должны быть строками символов. 
  
Значения столбца отображаются в разделе списка в поле со списком. Таким образом, **PR_NULL** ([PidTagNull](pidtagnull-canonical-property.md)) не является допустимым тегом свойства **для ulPRPropertyName**. Когда пользователь выбирает одну из строк или вводит новые данные в текстовое поле, для свойства **ulPRPropertyName** устанавливается выбранное или введенное значение. 
  
Чтобы отобразить начальное значение для управления редактированием, MAPI вызывает [IMAPIProp::GetProps](imapiprop-getprops.md) для получения значений свойств для таблицы отображения. Если одно из извлеченных свойств соответствует свойству, представленного членом **ulPRPropertyName,** его значение становится исходным значением. 
  
Обзор таблиц отображения см. в [таблице отображения.](display-tables.md) Сведения о реализации таблицы отображения см. в таблице ["Реализация таблицы отображения".](display-table-implementation.md)
