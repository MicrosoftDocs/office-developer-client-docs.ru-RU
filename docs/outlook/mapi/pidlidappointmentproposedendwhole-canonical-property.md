# <a name="pidlidappointmentproposedendwhole-canonical-property"></a>Каноническое свойство PidLidAppointmentProposedEndWhole

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает предлагаемое значение для свойства **диспидапптендвхоле** ([PidLidAppointmentEndWhole](pidlidappointmentendwhole-canonical-property.md)) для предложения Counter.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидапптпропоседендвхоле  <br/> |
|Набор свойств:  <br/> |Псетид_аппоинтмент  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008251  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Комментарии

Это значение должно быть указано в формате UTC.
