# <a name="pidlidsharingremotename-canonical-property"></a>Каноническое свойство PidLidSharingRemoteName

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Указывает имя удаленного общей папки. Это свойство общего доступа сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSharingRemoteName  <br/> |
|Набор свойств:  <br/> |PSETID_Sharing  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008A05  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Sharing  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство должно быть присвоено значение свойства **PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) в общей папке.
