# <a name="pidtagcontrolflags-canonical-property"></a>Каноническое свойство PidTagControlFlags

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит битовую маску флагов, определяющих поведение элемента управления, используемого в диалоговом окне, созданном из таблицы отображения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONTROL_FLAGS  <br/> |
|Идентификатор:  <br/> |0x3F00  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Таблица отображения MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Для этого свойства можно задать один или несколько из следующих флагов:
  
DT_ACCEPT_DBCS 
  
> Элемент управления может иметь в нем символы набора двухбайтовых символов (DBCS). Этот флаг используется с элементами управления редактированием. Допускается множество многобайтовых кодировок.
    
DT_EDITABLE 
  
> Элемент управления можно редактировать; можно изменить значение, связанное с элементом управления. Если этот флаг не установлен, элемент управления доступен только для чтения. Это значение игнорируется в элементах управления "подпись", "Группа", "Стандартная кнопка", "многозначный" раскрывающийся список и элемент управления "список".
    
DT_MULTILINE 
  
> Элемент управления "поле ввода" может содержать несколько строк. Это означает, что возвращаемый символ можно ввести в элементе управления. Этот флаг действителен только для элементов управления редактированием.
    
DT_PASSWORD_EDIT 
  
> Применяется к элементам управления для редактирования. Элемент управления редактированием рассматривается как пароль. Значение отображается с использованием звездочек вместо того, чтобы отображать введенные фактические символы.
    
DT_REQUIRED 
  
> Если элемент управления разрешает изменения (DT_EDITABLE), он должен иметь значение до [IMAPIProp:: SaveChanges](imapiprop-savechanges.md) . 
    
DT_SET_IMMEDIATE 
  
> Включает немедленный параметр значения; как только значение в элементе управления изменится, MAPI вызывает метод **SetProps** для свойства, связанного с этим элементом управления. Если этот флаг не задан, значения задаются при закрытии диалогового окна. 
    
DT_SET_SELECTION 
  
> Если в списке сделан выбор, в качестве значения свойства используется столбец индекс этого списка. Всегда используется с DT_SET_IMMEDIATE.
    
Это свойство хранится в элементе Улктлфлагс структуры [дтктл](dtctl.md) элемента управления. Большинство флагов элемента управления применяются ко всем элементам управления, которые позволяют вводить данные пользователем; несколько относятся только к элементу управления редактирования. Элементы управления, которые не допускают ввод пользователем (например, кнопку или метку), задают значение 0 для своих флагов управления. 
  
Многие из значений флагов понятны ясно. Например, если для элемента управления задано значение DT_REQUIRED, оно должно содержать значение, прежде чем разрешить отправку этого диалогового окна. Поставщик услуг может предоставить значение с помощью своей реализации **IMAPIProp** или ввести один из пользователей. DT_EDITABLE указывает, что значение для элемента управления можно изменить. DT_MULTILINE позволяет значению элемента управления "поле ввода" занимать несколько строк. 
  
Некоторые флаги элементов управления не настолько очевидны. Когда элемент управления устанавливает флаг DT_SET_IMMEDIATE, любые изменения его значения вступают в силу сразу после того, как пользователь перемещается на новый элемент управления. MAPI создает один вызов метода свойства [IMAPIProp:: SetProps](imapiprop-setprops.md) для свойства элемента управления. Это отличается от поведения по умолчанию, которое откладывается на то, что изменения значений элементов управления вступают в силу до тех пор, пока пользователь не нажимает кнопку **ОК** или не закрывает диалоговое окно. Флаг DT_SET_IMMEDIATE часто используется в сочетании с уведомлениями в таблице отображения. 
  
В следующей таблице перечислены типы элементов управления и все значения флагов, которые можно задать для каждого типа.
  
|**Элемент управления**|**Допустимые значения для этого свойства**|
|:-----|:-----|
|Кнопка  <br/> |Должно быть равно нулю  <br/> |
|Флажок  <br/> |DT_EDITABLE DT_SET_IMMEDIATE  <br/> |
|Поле со списком  <br/> |DT_EDITABLE, DT_REQUIRED DT_SET_IMMEDIATE  <br/> |
|Раскрывающийся список  <br/> |DT_EDITABLE DT_SET_IMMEDIATE  <br/> |
|Edit  <br/> |DT_ACCEPT_DBCS, DT_MULTILINE, DT_EDITABLE, DT_PASSWORD_EDIT, DT_REQUIRED, DT_SET_IMMEDIATE  <br/> |
|Поле группы  <br/> |Должно быть равно нулю  <br/> |
|Label  <br/> |Должно быть равно нулю  <br/> |
|Поле со списком  <br/> |Должно быть равно нулю  <br/> |
|Раскрывающийся список с несколькими значениями  <br/> |Должно быть равно нулю  <br/> |
|Многозначный список  <br/> |Должно быть равно нулю  <br/> |
|Страница с вкладками  <br/> |Должно быть равно нулю  <br/> |
|Переключатель  <br/> |Должно быть равно нулю  <br/> |
