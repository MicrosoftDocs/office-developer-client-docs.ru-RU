# <a name="pidnamexsharingcapabilities-canonical-property"></a>Каноническое свойство PidNameXSharingCapabilities

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение, которое указывает возможности совместного использования элемента.
  
|||
|:-----|:-----|
|Дружественные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Имя свойства:  <br/> |Возможности X-Sharing  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Доступ  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство содержит значение, которое указывает возможности общего доступа объекта сообщения.
