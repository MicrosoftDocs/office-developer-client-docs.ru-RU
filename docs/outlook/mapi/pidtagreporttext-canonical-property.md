# <a name="pidtagreporttext-canonical-property"></a>Каноническое свойство PidTagReportText

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит необязательный текст для отчета, созданного системой обмена сообщениями.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РЕПОРТ_ТЕКСТ, ПР_РЕПОРТ_ТЕКСТ_А, ПР_РЕПОРТ_ТЕКСТ_В  <br/> |
|Идентификатор:  <br/> |0x1001  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Сообщение MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Как правило, текст, содержащийся в этих свойствах, создается в ответ на отчет о доставке или недоставке или отчет о прочтении или непрочтении, полученный из базовой системы обмена сообщениями, но не является текстом, передаваемым через эту систему.
