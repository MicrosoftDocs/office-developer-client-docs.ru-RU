# <a name="pidtaghobbies-canonical-property"></a>Каноническое свойство PidTagHobbies

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имена хобби пользователя обмена сообщениями.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOBBIES, PR_HOBBIES_A PR_HOBBIES_W  <br/> |
|Идентификатор:  <br/> |0x3A43  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства обеспечивают идентификацию и доступ к информации о пользователе для обмена сообщениями, а также о содержимом, определенном организацией пользователя.
