# <a name="pidtaghobbies-canonical-property"></a>Каноническое свойство PidTagHobbies

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит имена занятиях обмена сообщениями пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOBBIES, PR_HOBBIES_A, PR_HOBBIES_W  <br/> |
|Идентификатор:  <br/> |0x3A43  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства предоставляют идентификации и доступа к сведениям о обмена сообщениями пользователя и содержимое определяется организации пользователя.
