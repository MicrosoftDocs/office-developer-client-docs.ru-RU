# <a name="cbnewentryid"></a>CbNewENTRYID

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество ветвей, которые должны быть выделены для новой [структуры ENTRYID](entryid.md) указанного размера. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |**ENTRYID** <br/> |
   
```cpp
CbNewENTRYID (_cb)
```

## <a name="parameters"></a>Параметры

 __cb_
  
> Количество (в том числе) в идентификаторе записи, который будет храниться в новой **структуре ENTRYID.**
