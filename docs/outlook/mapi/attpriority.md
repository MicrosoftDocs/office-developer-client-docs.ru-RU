**Применимо к**: Outlook 2013 | Outlook 2016

|**Priority**|**Значение MAPI**|**Значение TNEF**|
|:-----|:-----|:-----|
|обычный  <br/> |-1  <br/> |3  <br/> |
|Высокая  <br/> |0  <br/> |2  <br/> |
|Низкая  <br/> |1  <br/> |1  <br/> |
