**Область применения**: Outlook 2013 | Outlook 2016

|**Приоритет**|**Значение MAPI**|**Значение TNEF**|
|:-----|:-----|:-----|
|нормальный  <br/> |–1  <br/> |3  <br/> |
|высокая  <br/> |0  <br/> |2  <br/> |
|низкий  <br/> |1  <br/> |1  <br/> |
