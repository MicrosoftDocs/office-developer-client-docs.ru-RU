# <a name="mapi-client-application-overview"></a>Обзор клиентских приложений MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Клиентская программа MAPI — это любое приложение, использующее интерфейс программирования MAPI. Клиентские приложения реализуют задачи обмена сообщениями в качестве основного или вторичного фокуса. Клиентские приложения для обмена сообщениями, такие как приложения, отправив и получаюющие электронную почту, реализуют обмен сообщениями в качестве основного фокуса. Для клиентских приложений без обмена сообщениями, таких как приложения инвентаризации или конфигурации, обмен сообщениями является второстепенной функцией.
  
Действия по обмену сообщениями могут включать, например, приложение обработки слов, в меню файловое меню есть команда **Отправка,** которая позволяет отправлять документы, Microsoft Office Outlook электронную почту, программы автоматизации потока работы и службы доски объявлений.  
  
Клиентские приложения могут включать пользователя для создания интерактивной среды или работать без пользователя в автоматизированной среде. Хотя MAPI поставляет набор общих диалогов со стандартным пользовательским интерфейсом, клиентские приложения не обязаны представлять пользовательский интерфейс. По сути, все обработки можно обрабатывать в приложении, если хотите. Примером автоматизированного клиентского приложения может быть приложение для управления запасами, которое запрограммировано на регулярной основе для маршрутизации элементов определенного типа к стандартным получателям.
