# <a name="mapi-client-application-overview"></a>Обзор клиентского приложения MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Клиентские приложения MAPI — это любое приложение, использующее программный интерфейс MAPI. Клиентские приложения реализуют задачи обмена сообщениями в качестве основного или дополнительного фокуса. Клиентские приложения для обмена сообщениями, такие как приложения, которые отправляют и получают электронную почту, реализуют обмен сообщениями в качестве основного. Для клиентских приложений, не относяющихся к обмену сообщениями, таких как приложения инвентаризации или конфигурации, обмен сообщениями является дополнительным.
  
Действия с сообщениями могут включать, например, приложение  word, в  меню "Файл" с командой "Отправить", которая позволяет отправлять документы, Microsoft Office Outlook электронную почту, программы автоматизации потока работы и службы доски объявлений. 
  
Клиентские приложения могут включать пользователя для создания интерактивной среды или работать без пользователя в автоматизированной среде. Хотя MAPI обеспечивает набор общих диалогов со стандартным пользовательским интерфейсом, клиентские приложения не обязаны представлять пользовательский интерфейс. По сути, вся обработка может быть обработана в приложении, если это необходимо. Примером автоматизированного клиентского приложения может быть приложение для управления инвентаризацией, которое запрограммировано маршрутизировать элементы определенного типа стандартным получателям на регулярной основе.
