# <a name="mapi-client-application-overview"></a>Общие сведения о клиентских приложений MAPI

  
  
**Относится к**: Outlook 
  
Клиентского приложения MAPI — это любое приложение, использующее программный интерфейс MAPI. Клиентские приложения реализовать сообщениями либо как их основного или дополнительного фокус. Обмен сообщениями клиентских приложений, например приложений, отправлять и получать электронную почту, реализации системы обмена сообщениями, как их основного фокус. Не системы обмена сообщениями клиентских приложений, таких как запасов или настройки приложений системы обмена сообщениями — это дополнительный компонент.
  
Операции обмена сообщениями может включать, например, текстовый редактор, имеющей команды **Отправить** в меню **файл** , чтобы включить документы отправку, электронной почты Microsoft Office Outlook, рабочего потока автоматизации программы и доски объявлений. 
  
Клиентские приложения можно либо включить пользователю создавать интерактивные среды или работать без пользователей в среде автоматической. Хотя MAPI предоставляет набор общие диалоговые окна с его стандартный пользовательский интерфейс, чтобы представить пользовательский интерфейс не требуются клиентские приложения. На самом деле все обработки можно обрабатывать в приложении, если вы хотите, чтобы. Пример автоматизированных клиентское приложение будет приложения управления запасов, запрограммирован маршрут элементов определенного типа стандартный получателям на регулярной основе.
