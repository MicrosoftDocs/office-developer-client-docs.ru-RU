# <a name="pidlidlogdocumentrouted-canonical-property"></a>Каноническое свойство PidLidLogDocumentRouted

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, был ли документ отправлен получателю маршрутизации во время ведения журнала.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидлогдокраутед  <br/> |
|Набор свойств:  <br/> |Псетид_лог  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008710  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Журнал  <br/> |
