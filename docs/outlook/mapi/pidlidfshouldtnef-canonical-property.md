# <a name="pidlidfshouldtnef-canonical-property"></a>Каноническое свойство PidLidFShouldTNEF

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, следует ли кодировать элемент с помощью формата инкапсуляции с нейтральным транспортом (TNEF). 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFShouldTNEF  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x000085A5  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Конфигурация времени запуска  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство заданной, Microsoft Word заданной в качестве редактора электронной почты, и отправляет объект OLE, встроенный в поток Rich Text Format (RTF).
