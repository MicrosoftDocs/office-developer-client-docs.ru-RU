# <a name="pidtaginternetarticlenumber-canonical-property"></a>Каноническое свойство PidTagInternetArticleNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Число, связанное с элементом в банке сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ИНТЕРНЕТ_АРТИКЛЕ_НУМБЕР  <br/> |
|Идентификатор:  <br/> |0x0E23  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Примечания

В хранилищах сообщений IMAPI это свойство соответствует ИДЕНТИФИКАТОРу UID IMAPI.
