# <a name="pidtaginternetarticlenumber-canonical-property"></a>Каноническое свойство PidTagInternetArticleNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Номер, связанный с элементом в хранилище сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_INTERNET_ARTICLE_NUMBER  <br/> |
|Идентификатор:  <br/> |0x0E23  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Примечания

В хранилищах сообщений IMAPI это свойство соответствует UID IMAPI.
