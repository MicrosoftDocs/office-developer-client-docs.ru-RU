# <a name="message-store-features"></a>Функции магазина сообщений

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Поставщики store сообщений сложнее, чем другие поставщики служб MAPI, в том, что поставщики хранения сообщений имеют более широкий спектр дополнительных функций, которые они могут реализовать. Список необходимых функций для поставщика store сообщений довольно короткий. Однако типичный поставщик банка сообщений поддерживает ряд дополнительных функций, так как многие дополнительные функции очень полезны или требуются большинству клиентов MAPI. В следующей таблице перечислены основные функции, которые могут реализовать поставщики store сообщений, а также их обязательное или необязательное значение для всех поставщиков и поставщиков по умолчанию.
