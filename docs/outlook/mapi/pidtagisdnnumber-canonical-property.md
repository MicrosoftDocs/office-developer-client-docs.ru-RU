# <a name="pidtagisdnnumber-canonical-property"></a>Каноническое свойство PidTagIsdnNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит номер телефона для получателя с поддержкой ISDN. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ИСДН_НУМБЕР, ПР_ИСДН_НУМБЕР_А, ПР_ИСДН_НУМБЕР_В  <br/> |
|Идентификатор:  <br/> |0x3A2D  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Комментарии

Эти свойства предоставляют сведения о идентификации и доступе для получателя. Они определяются получателем и их организацией.
