# <a name="pidtagassistanttelephonenumber-canonical-property"></a>Каноническое свойство PidTagAssistantTelephoneNumber

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит номер телефона помощника получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ASSISTANT_TELEPHONE_NUMBER, PR_ASSISTANT_TELEPHONE_NUMBER_A, PR_ASSISTANT_TELEPHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A2E  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства предоставляют идентификации и доступа к сведениям для получателя. Они определены получателя и их организации. 
  
— Номер телефона помощника, указанных в свойстве **PR_ASSISTANT** ([PidTagAssistant](pidtagassistant-canonical-property.md)).
