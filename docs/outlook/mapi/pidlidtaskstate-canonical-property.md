# <a name="pidlidtaskstate-canonical-property"></a>Каноническое свойство PidLidTaskState

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает текущее состояние назначения задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskState  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x00008113  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть одним из следующих.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0x00000000  <br/> |Эта задача была создана для того, чтобы соответствовать задаче, встроенной в отторжение задачи, но не удалось найти локально.  <br/> |
|0x00000001  <br/> |Задача не назначена.  <br/> |
|0x00000002  <br/> |Задача — это копия назначенной задачи назначенного назначения.  <br/> |
|0x00000003  <br/> |Задача — это копия назначенной задачи назначит назначимый назначимый назначение.  <br/> |
|0x00000004  <br/> |Задача — это копия отклоненной задачи назначитель задачи.  <br/> |
