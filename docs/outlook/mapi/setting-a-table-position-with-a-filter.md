# <a name="setting-a-table-position-with-a-filter"></a>Установка положения таблицы с использованием фильтра

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Пользователи могут переместите курсор к строку, которая соответствует набор условий фильтра. Фильтры может быть основана на ряд рекомендаций, например, значения свойств столбца, битовой маски или вложенных объектов. Фильтры указаны в MAPI, с помощью структуры [SRestriction](srestriction.md) . 
  
 **Для размещения таблицы в первую строку, которая соответствует критериям, приведенным в качестве ограничения**
  
- Вызовите метод [IMAPITable::FindRow](imapitable-findrow.md) . Начиная с строки, представленной определенную закладку **FindRow** осуществляет поиск в прямом или обратном направлении найдите строку, которая соответствует условиям, указанного в ограничении. Можно использовать для реализации полосу прокрутки, основанный на строк символов, а не дробной **FindRow** . Например клиент может вызвать реализацию MAPI's **FindRow** при поиске по интегрированной адресной книги, чтобы включить для пользователя, указав один или несколько знаков, чтобы найти имя, которое начинается с указанных символов.
