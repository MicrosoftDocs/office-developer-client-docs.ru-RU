# <a name="setting-a-table-position-with-a-filter"></a>Настройка позиции таблицы с помощью фильтра

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Пользователи таблицы могут переместить курсор в строку, которая соответствует набору критериев фильтрации. Фильтры могут быть основаны на различных рекомендациях, таких как значения свойств столбцов, битмаски или подобекты. Фильтры указаны в MAPI с помощью [структуры SRestriction.](srestriction.md) 
  
 **Расположение таблицы к первой строке, которая соответствует критериям, установленным в ограничении**
  
- Вызов метода [IMAPITable::FindRow.](imapitable-findrow.md) Начиная со строки, представленной определенной закладкой, **FindRow** выполняет поиск в направлении вперед или назад, чтобы найти строку, которая соответствует критериям, указанным в ограничении. **FindRow** может быть полезен для реализации панели прокрутки, основанной на строках символов, а не на дробных значениях. Например, клиент может вызвать реализацию **MapI FindRow** при поиске через интегрированную адресную книгу, чтобы пользователь, введите один или несколько символов, найти имя, которое начинается с указанных символов.
