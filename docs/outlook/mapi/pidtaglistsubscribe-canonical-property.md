# <a name="pidtaglistsubscribe-canonical-property"></a>Каноническое свойство PidTagListSubscribe

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение поля многоцелевых расширений интернет-почты (MIME) List-Subscribe.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_LIST_SUBSCRIBE, PR_LIST_SUBSCRIBE_A, PR_LIST_SUBSCRIBE_W  <br/> |
|Идентификатор:  <br/> |0x1044  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Разное  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы создать поле List-Subscribe, клиенты должны задать значение этих свойств нужному значению. Авторы MIME должны скопировать значение этих свойств в поле List-Subscribe.
  
Чтобы установить значение таких свойств, связанных с сервером, клиенты MIME должны написать поля загона, указанные в следующей таблице.
  
|**Property**|**Предпочтительное имя поля загона**|**Альтернативное имя поля загона**|
|:-----|:-----|:-----|
|**PidTagListSubscribe** <br/> |List-Subscribe  <br/> |X-List-Subscribe  <br/> |
