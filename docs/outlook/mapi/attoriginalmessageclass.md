# <a name="attoriginalmessageclass"></a>attOriginalMessageClass

**Относится к**: Outlook 2013 | Outlook 2016 
  
Класс сообщения хранится в строке. Закодированная строка обычно содержит имя класса сообщения, указанное в MAPI.
