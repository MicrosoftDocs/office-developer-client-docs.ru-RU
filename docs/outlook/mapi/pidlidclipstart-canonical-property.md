# <a name="pidlidclipstart-canonical-property"></a>Каноническое свойство PidLidClipStart

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату и время начала события в формате UTC для объектов календаря с одним экземпляром и значение полуночи на дату первого экземпляра в формате UTC для повторяющегося ряда.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидклипстарт  <br/> |
|Набор свойств:  <br/> |Псетид_аппоинтмент  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008235  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Календарь  <br/> |
