# <a name="pidlidclipstart-canonical-property"></a>Каноническое свойство PidLidClipStart

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает дату начала и время события в формате универсального времени (UTC) для одного экземпляра объектов календаря и указывает полночь день первую строку в формате UTC для серии повторяющихся.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidClipStart  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008235  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Календарь  <br/> |
