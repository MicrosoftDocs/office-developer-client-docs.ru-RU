# <a name="pidtagbirthday-canonical-property"></a>Каноническое свойство PidTagBirthday

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит день рождения контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_BIRTHDAY  <br/> |
|Идентификатор:  <br/> |0x3A42  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является одним из свойств, которое предоставляет идентификацию и доступ к сведениям о контакте. Эти свойства определяются пользователем и организацией пользователя.
