# <a name="pidtagbirthday-canonical-property"></a>Каноническое свойство PidTagBirthday

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит день рождения контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_BIRTHDAY  <br/> |
|Идентификатор:  <br/> |0x3A42  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство является одним из свойств, которые предоставляет идентификации и доступа к информации о контакте. Эти свойства определены пользователей и организаций пользователя.
