# <a name="mapi-message-store-provider-objects"></a>Объекты поставщика хранилища сообщений MAPI
  
**Относится к**: Outlook 
  
Поставщики хранилища сообщений реализации поставщика и объекты входа в систему как всех поставщиков услуг. Они также реализовать объект хранилища сообщений, папок, сообщения, вложения и таблиц. Как вариант некоторые поставщики хранилища сообщений реализации состояния объектов.
  
На следующем рисунке показана каждого объекта хранилища сообщений с его соответствующий интерфейс и компонент MAPI, который использует его.
