# <a name="mapi-message-store-provider-objects"></a>Объекты поставщика сообщений MAPI
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Поставщики магазинов сообщений реализуют объекты поставщика и логотипы, как и все поставщики услуг. Они также реализуют объект магазина сообщений, папки, сообщения, вложения и таблицы. В качестве параметра некоторые поставщики магазинов сообщений реализуют объекты состояния.
  
На следующем рисунке показан каждый объект магазина сообщений с соответствующим интерфейсом и компонент MAPI, который его использует.
