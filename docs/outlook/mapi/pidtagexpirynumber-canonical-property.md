# <a name="pidtagexpirynumber-canonical-property"></a>Каноническое свойство PidTagExpiryNumber

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Определяет время отправки с истечением срока действия в сочетании со **свойством PR_EXPIRY_UNITS** ([PidTagExpiryUnits).](pidtagexpiryunits-canonical-property.md)
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_EXPIRY_NUMBER  <br/> |
|Идентификатор:  <br/> |0x3FED  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Состояние MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть установлено от 0 до 999 включительно, если оно присутствует.
