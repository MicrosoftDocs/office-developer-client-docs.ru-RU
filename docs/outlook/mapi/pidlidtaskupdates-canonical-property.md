# <a name="pidlidtaskupdates-canonical-property"></a>Каноническое свойство PidLidTaskUpdates

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, был ли поручена поручена на отправку обновления задачи при изменении назначенной задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидтаскупдатес  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000811B  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Задача  <br/> |
