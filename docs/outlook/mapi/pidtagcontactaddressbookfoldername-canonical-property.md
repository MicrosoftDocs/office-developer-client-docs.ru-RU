# <a name="pidtagcontactaddressbookfoldername-canonical-property"></a>Каноническое свойство PidTagContactAddressBookFolderName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит имя папки, используемой для записей адресной книги.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КОНТАБ_ФОЛДЕР_НАМЕ, ПР_КОНТАБ_ФОЛДЕР_НАМЕ_В  <br/> |
|Идентификатор:  <br/> |0x6613  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Адресная книга контактов  <br/> |
   
## <a name="remarks"></a>Примечания

Следующие символы нельзя использовать в именах папок:
  
[ ] / \ &amp; ~ ? \* | \<\> " ; : +
