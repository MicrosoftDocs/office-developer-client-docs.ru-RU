# <a name="index_search_pusher_process"></a>INDEX_SEARCH_PUSHER_PROCESS

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает процесс, который отправляет уведомление обработчиве протокола MAPI о готовности объекта в этом хранилище к индексации.
  
## <a name="quick-info"></a>Краткие сведения

```cpp
typedef struct _INDEX_SEARCH_PUSHER_PROCESS {  
    DWORD dwPID;  
} INDEX_SEARCH_PUSHER_PROCESS; 
```

## <a name="members"></a>Элементы

 *dwPID* 
  
>  ИД процесса, отправляющий уведомление об индексации индексалю обработщика протокола MAPI.
