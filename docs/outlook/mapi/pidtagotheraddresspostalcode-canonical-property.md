# <a name="pidtagotheraddresspostalcode-canonical-property"></a>Каноническое свойство PidTagOtherAddressPostalCode

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит почтовый индекс для другого почтового адреса пользователя почтового ящика.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OTHER_ADDRESS_POSTAL_CODE, PR_OTHER_ADDRESS_POSTAL_CODE_A PR_OTHER_ADDRESS_POSTAL_CODE_W  <br/> |
|Идентификатор:  <br/> |0x3A61  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
