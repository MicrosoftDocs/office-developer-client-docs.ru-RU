# <a name="ulfromszhex"></a>UlFromSzHex

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Преобразует строку шестнадцатеричных цифр, заканчивающуюся нулем, в длинное целое число без знака. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
|Реализовано в:  <br/> |MAPI  <br/> |
|Вызывающая сторона:  <br/> |Клиентские приложения и поставщики услуг  <br/> |
   
```cpp
ULONG UlFromSzHex(
LPCSTR lpsz
);
```

## <a name="parameters"></a>Параметры

 _лпсз_
  
> возврата Указатель на преобразуемую строку, заканчивающуюся нулем. Значение параметра _лпсз_ не должно превышать 65536 символов. 
    
## <a name="return-value"></a>Возвращаемое значение

 **Улфромсзекс** возвращает неподписанное длинное целое число. Если строка не начинается по крайней мере с одной шестнадцатеричной цифры, возвращается ноль. 
  
## <a name="remarks"></a>Примечания

Функция **улфромсзекс** завершает преобразование при достижении первого символа в строке, не являющийся шестнадцатеричной цифрой. Например, если задается строка "5A", **улфромсзекс** возвращает целое значение 90. При наличии строки "5g5h" функция возвращает целое значение 5. При наличии строки "g5h5" **улфромсзекс** возвращает ноль. 
  
 **Улфромсзекс** конфиденциально задиакритические различия, но допускает как "a" до "f", так И "a" до "f" для шестнадцатеричных цифр. Поддерживаются строки в форматах Юникод и DBCS. Предельная длина в _лпсз_ — символы, не обязательно — байты.
