# <a name="pidlidcontactlinkentry-canonical-property"></a>Каноническое свойство PidLidContactLinkEntry

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит список адресных книг **идентификаторами EntryID** , связанных с этим объектом Message. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидконтактлинкентри  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008585  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Контакт  <br/> |
