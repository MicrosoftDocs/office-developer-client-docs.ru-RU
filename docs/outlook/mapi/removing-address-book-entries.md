# <a name="removing-address-book-entries"></a>Удаление записей адресной книги
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Для удаления одного или нескольких получателей вызывается метод [иабконтаинер::D елетинтриес](iabcontainer-deleteentries.md) . **Делетинтриес** имеет два параметра: массив идентификаторов записей, представляющих удаляемых получателей и зарезервированное значение флагов. Удаление получателя влияет на таблицу содержимого контейнера; в дополнение к удалению получателя, в контейнере должна быть удалена строка таблицы содержимого, представляющая получателя. Когда строка удалена из таблицы, ваш контейнер должен выдать табличное уведомление каждому зарегистрированному клиенту. 
  
### <a name="to-implement-iabcontainerdeleteentries"></a>Реализация Иабконтаинер::D Елетинтриес
  
1. Удалите из контейнера каждого получателя, представленного идентификатором записи.
    
2. Если таблица содержимое контейнера открыта:
    
   - Отправьте уведомление _фневтаблемодифиед_ с набором элементов **ултабливент** в табле_ров_делетед для зарегистрированных клиентов для каждой строки удаленной таблицы содержимого. Если поставщик использует программу уведомления, вызовите [имаписуппорт:: notify](imapisupport-notify.md) для отправки этих уведомлений. 
    
   - Если поставщик поддерживает уведомления объектов, также отправляет уведомление _фневобжектделетед_ .
