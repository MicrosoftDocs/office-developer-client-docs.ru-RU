# <a name="pidlidcontactuserfield2-canonical-property"></a>Каноническое свойство PidLidContactUserField2

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Второе поле указывает на контакт, который предназначен для использования Прочее для этого контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidContactUserField2  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008050  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство содержит текст, который связан с других свойств конкретного контакта.
