# <a name="pidlidcontactuserfield2-canonical-property"></a>Каноническое свойство PidLidContactUserField2

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает второе поле контакта, предназначенное для разного использования контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidContactUserField2  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008050  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство содержит текст, не связанный с другим свойством, связанным с контактом.
