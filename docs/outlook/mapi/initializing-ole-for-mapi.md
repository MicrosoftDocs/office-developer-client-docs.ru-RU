# <a name="initializing-ole-for-mapi"></a>Инициализация OLE для MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
При использовании OLE, вызовите функцию OLE [OleInitialize](https://msdn.microsoft.com/library/ms690134%28v=VS.85%29.aspx) для инициализации библиотеки OLE. **OleInitialize** инициализирует глобальные данные для сеанса и подготавливает библиотек OLE для приема звонков. Сведения о вызовах **OleInitialize**просмотрите пакет SDK для Windows.
