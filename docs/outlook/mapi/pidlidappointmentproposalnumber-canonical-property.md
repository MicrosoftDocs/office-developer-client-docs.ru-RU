# <a name="pidlidappointmentproposalnumber-canonical-property"></a>Каноническое свойство PidLidAppointmentProposalNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает число участников, отправивших встречные предложения, которые не были приняты или отклонены организатором.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptProposalNum  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008259  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Собрания  <br/> |
