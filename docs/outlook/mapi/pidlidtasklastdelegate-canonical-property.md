# <a name="pidlidtasklastdelegate-canonical-property"></a>Каноническое свойство PidLidTaskLastDelegate

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
 Имя пользователя, который последним назначен задаче или назначен ему. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидтаскластделегате  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008125  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Примечания

Перед отправкой запроса задачи клиент задает для этого свойства имя назначения задачи. Перед отправкой ответа задачи клиент задает для этого свойства имя уполномоченного задачи.
