# <a name="pidtagbusinesshomepage-canonical-property"></a>Каноническое свойство PidTagBusinessHomePage

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит URL-адрес домашней страницы для предприятия.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_БУСИНЕСС_ХОМЕ_ПАЖЕ, ПР_БУСИНЕСС_ХОМЕ_ПАЖЕ_А, ПР_БУСИНЕСС_ХОМЕ_ПАЖЕ_В  <br/> |
|Идентификатор:  <br/> |0x3A51  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Комментарии

Эти свойства используются в качестве ссылки на домашнюю страницу Организации контакта.
