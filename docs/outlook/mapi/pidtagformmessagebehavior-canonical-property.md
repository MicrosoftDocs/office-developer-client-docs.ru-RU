# <a name="pidtagformmessagebehavior-canonical-property"></a>Каноническое свойство PidTagFormMessageBehavior

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит TRUE, если сообщение должно быть составлено в текущей папке. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_FORM_MESSAGE_BEHAVIOR  <br/> |
|Идентификатор:  <br/> |0x330A  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |MAPI общие  <br/> |
   
## <a name="remarks"></a>Примечания

Значение FALSE указывает, что сообщение должно быть составлено как любое другое межличностное сообщение, то есть в папке Outbox.
