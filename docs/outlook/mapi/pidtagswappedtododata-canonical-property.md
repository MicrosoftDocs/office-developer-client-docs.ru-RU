# <a name="pidtagswappedtododata-canonical-property"></a>Каноническое свойство PidTagSwappedToDoData

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Поддерживает второй набор значений свойств, которые не влияют на состояние помеченного объекта Message.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SWAPPED_TODO_DATA  <br/> |
|Идентификатор:  <br/> |0x0E2D  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |MAPI, не передаваемый  <br/> |
   
## <a name="remarks"></a>Примечания

В качестве дополнительного места хранения флагов, если поддерживаются флаги отправитель или напоминания отправитель, эта структура предоставляет расположение, в котором хранятся все свойства, относящиеся к протоколу информационных пометок, которые поддерживаются в флагах отправитель, и все свойства, связанные с протоколом параметров напоминания, которые поддерживаются в напоминаниях отправитель без отправки флага отправитель или сведения напоминания отправитель получателям сообщения.
  
Аналогичным образом эта структура предоставляет расположение, в котором хранятся все свойства, относящиеся к протоколу информационного пометки, которые поддерживаются в флагах получателей и свойствах, связанных с протоколом параметров напоминания, которые поддерживаются в напоминаниях получателей в ранее отправленном сообщении.
  
Дополнительные сведения об этом свойстве см. [в [MS-OXOFLAG]](https://msdn.microsoft.com/library/f1e50be4-ed30-4c2a-b5cb-8ff3aaaf9b91%28Office.15%29.aspx).
