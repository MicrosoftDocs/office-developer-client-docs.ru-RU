# <a name="contab_entryid"></a>CONTAB_ENTRYID

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор папки "Контакты".
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |мсомапиутил. h  <br/> |
   
```cpp
#pragma pack(4) 
typedef struct _contab_entryid
{
    BYTE abFlags[4];
    MAPIUID muid;
    ULONG ulVersion;
    ULONG ulType;
    ULONG ulIndex;
    ULONG cbeid;
    BYTE abeid[1];
}   CONTAB_ENTRYID, *LPCONTAB_ENTRYID;
#pragma pack() 
```

## <a name="members"></a>"Участники"

 **абфлагс**
  
> Битовая маска флагов, предоставляющая информацию, описывающую объект. Дополнительные сведения можно найти в описании поля **абфлагс** структуры [EntryID](entryid.md) . 
    
 **Muid**
  
> GUID, определяющий поставщика хранилища.
    
 **улверсион**
  
> Номер версии структуры **CONTAB_ENTRYID** . Необходимо задать значение CONTAB_VERSION. 
    
 **ултипе**
  
> Целое число, представляющее тип идентификатора записи контакта. Он должен иметь одно из следующих значений:
    
|**Название**|**Описание**|
|:-----|:-----|
|CONTAB_USER  <br/> |Объект пользователя, отправляющего сообщение.  <br/> |
|CONTAB_DISTLIST  <br/> |Объект списка рассылки.  <br/> |
   
 **улиндекс**
  
> Индекс в подмножестве свойств электронной почты.
    
 **кбеид**
  
> Размер идентификатора записи контактного сообщения, связанного с данной записью в адресной книге контактов.
    
 **абеид**
  
> Идентификатор записи контактного сообщения, связанного с этой записью в адресной книге Contacts.
    
## <a name="remarks"></a>Примечания

Адресная книга контактов — это адресная книга, содержащая все элементы контактов в папке "Контакты", у которых есть адрес электронной почты или номер факса. Каждая запись в адресной книге контактов связана либо с адресом электронной почты, либо с номером факса. Так как элемент контакта может иметь до трех адресов электронной почты и три номера факса, элемент контакта можно представить в виде шести записей в адресной книге контактов.
  
Адресная книга контактов предназначена для поддержки пользователей, обращающихся к сообщениям электронной почты с контактами в папке "Контакты". Поставщик адресной книги Contacts, поддерживаемый Microsoft Outlook 2010 и Microsoft Outlook 2013 — contab32. dll.
  
Структура **CONTAB_ENTRYID** поддерживает подмножество сведений, присутствующих в базовом сообщении MAPI. Он определяет Контактное сообщение, с которым связана запись адресной книги для определенных контактов. 
  
Поля **кбеид** и **абеид** являются допустимыми только в том случае, если для поля **ултипе** задано значение CONTAB_DISTLIST или CONTAB_USER. Если для поля **ултипе** задано значение CONTAB_ROOT, CONTAB_SUBROOT или CONTAB_CONTAINER, вместо этого следует использовать структуру [DIR_ENTRYID](dir_entryid.md) .
