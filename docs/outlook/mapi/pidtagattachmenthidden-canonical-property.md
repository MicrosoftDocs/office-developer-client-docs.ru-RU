# <a name="pidtagattachmenthidden-canonical-property"></a>Каноническое свойство PidTagAttachmentHidden

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, скрыто ли вложение от конечного пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ATTACHMENT_HIDDEN  <br/> |
|Идентификатор:  <br/> |0x7FFE  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Вложение в сообщение  <br/> |
