# <a name="pidlidclassification-canonical-property"></a>Каноническое свойство PidLidClassification

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает имя классификации сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидклассификатион  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000085B6  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
