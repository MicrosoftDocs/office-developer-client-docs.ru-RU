# <a name="implementing-standard-form-verbs"></a>Реализация стандартных глаголов форм

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
MAPI определяет набор стандартных глаголов или действий, которые должны поддерживаться всеми пользователями, которые должны поддерживаться при выборе меню или нажатии кнопки. С каждым глаголом связана константа для идентификации, определяемая в EXCHFORM. Файл h-загона. В следующей таблице перечислены стандартные команды формы и связанные с ними константы:
  
|**Verb**|**Значение**|
|:-----|:-----|
|Открыть  <br/> |EXCHIVERB_OPEN  <br/> |
|Ответить  <br/> |EXCHIVERB_REPLYTOSENDER  <br/> |
|Ответ всем  <br/> |EXCHIVERB_REPLYTOALL  <br/> |
|Перенаправление  <br/> |EXCHIVERB_FORWARD  <br/> |
|Печать  <br/> |EXCHIVERB_PRINT  <br/> |
|Сохранить как  <br/> |EXCHIVERB_SAVEAS  <br/> |
|Ответ папке  <br/> |EXCHIVERB_REPLYTOFOLDER  <br/> |
   
Когда пользователь выбирает глагол, передав его константы в вызове метода [IMAPIForm::D oVerb](imapiform-doverb.md) формы, чтобы выполнить соответствующее действие. 
  
Помимо доступа к глаголам с помощью просмотра формы, пользователи иногда могут получать доступ к глаголам непосредственно из формы. Например, некоторые объекты формы позволяют  пользователю вызвать глагол "Печать", щелкнув правой кнопкой мыши форму и выбрав пункт **"Печать"** в контекстном меню.
