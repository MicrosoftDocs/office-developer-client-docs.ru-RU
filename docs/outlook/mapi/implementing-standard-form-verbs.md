# <a name="implementing-standard-form-verbs"></a>Реализация стандартных глаголов формы

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
MAPI определяет набор стандартных глаголов или действий, принятых, когда пользователь делает выбор меню или нажимает кнопку, которую должны поддерживать все зрители форм. Каждый глагол имеет констант, связанный с ним для идентификации, определенный в EXCHFORM. Файл заглавной папки H. В следующей таблице перечислены стандартные глаголы формы и связанные с ними константы:
  
|**Verb**|**Значение**|
|:-----|:-----|
|Открыть  <br/> |EXCHIVERB_OPEN  <br/> |
|Ответить  <br/> |EXCHIVERB_REPLYTOSENDER  <br/> |
|Ответ на все  <br/> |EXCHIVERB_REPLYTOALL  <br/> |
|Перенаправление  <br/> |EXCHIVERB_FORWARD  <br/> |
|Печать  <br/> |EXCHIVERB_PRINT  <br/> |
|Сохранить как  <br/> |EXCHIVERB_SAVEAS  <br/> |
|Ответ на папку  <br/> |EXCHIVERB_REPLYTOFOLDER  <br/> |
   
Когда пользователь выбирает глагол, передай его константа в вызове методу [IMAPIForm::D oVerb](imapiform-doverb.md) для выполнения соответствующего действия. 
  
В дополнение к доступу к глаголам через вашего просмотра формы, пользователи иногда могут получить доступ к глаголам непосредственно из формы. Например, некоторые объекты формы позволяют пользователю вызывать глагол **Print,** щелкнув правой кнопкой мыши по форме и выбрав print **из** контекстного меню.
