# <a name="mapi-form-objects"></a>Объекты формы MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Объекты form создаются динамически с помощью серверов форм, чтобы отображать определенные сообщения и позволять пользователям взаимодействовать с ними. Таким образом, объект формы — это мгновенный вывод класса из [IMAPIForm,](imapiformiunknown.md) реализуемого сервером форм. Когда клиентская заявка открывает сообщение, сервер формы для этого класса сообщений создает объект формы для обработки сообщения. Объект формы создает интерфейс и отображает свойства сообщения в нем. Объект формы и его интерфейс сохраняются до тех пор, пока пользователь не закроет его. Объект формы обрабатывает любые изменения значений свойств сообщения. 
  
Кроме того, интерфейсы форм MAPI определяют механизм, с помощью которого один объект формы может загружать и отображать ряд сообщений. Это механизм эффективности, который позволяет избежать необходимости уничтожения и создания объектов сообщений и их интерфейсов. Если клиент обмена сообщениями запрашивает загрузку другого сообщения, объект формы должен сохранить все изменения в свойствах текущего сообщения.
  
Сведения о перспективах объектов форм в клиентском приложении см. в приложении [MAPI Custom Form Objects.](mapi-custom-form-objects.md)
