# <a name="mapi-form-objects"></a>Объекты формы MAPI

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Объекты формы создаются динамически серверами формы для правильного отображения конкретные сообщения и пользователи могли взаимодействовать с ними. Объект формы является, таким образом, экземпляр класса, производного от [IMAPIForm](imapiformiunknown.md) , которая реализована на сервере формы. При открытии сообщения в клиентское приложение сервера форм для этого класса сообщений создает объект формы для обработки сообщения. Объект формы затем создает его интерфейс и отображает свойства сообщения в нем. Объект формы и интерфейс повторяется, пока пользователь не закроет его. Объект формы обрабатывает любые изменения значений свойств сообщений. 
  
Кроме того интерфейсы формы MAPI определяет механизм, с помощью которого можно загружать и отображать несколько сообщений один объект формы. Это механизм эффективности, как она позволяет избежать ненужного уничтожения и создания объектов сообщений и их интерфейсы. При запросе клиентом обмена сообщениями для загрузки другое сообщение, объект формы должен сохранить все изменения для свойства текущего сообщения.
  
Подробную информацию об Перспектива клиентское приложение объектов формы см [MAPI настраиваемые формы](mapi-custom-form-objects.md).
