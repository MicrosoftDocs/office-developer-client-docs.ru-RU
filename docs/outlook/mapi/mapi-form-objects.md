# <a name="mapi-form-objects"></a>Объекты форм MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Объекты форм динамически создаются серверами форм для отображения определенных сообщений и обеспечения взаимодействия пользователей с ними. Таким образом, объект формы является копией класса, производного от [IMAPIForm,](imapiformiunknown.md) который реализуется сервером форм. Когда клиентская приложение открывает сообщение, сервер форм для этого класса сообщений создает объект формы для обработки сообщения. Затем объект формы создает свой интерфейс и отображает свойства сообщения в нем. Объект формы и его интерфейс сохраняются, пока пользователь не закроет его. Объект формы обрабатывает любые изменения значений свойств сообщения. 
  
Кроме того, интерфейсы форм MAPI определяют механизм, с помощью которого один объект формы может загружать и отображать серию сообщений. Это механизм эффективности, который позволяет избежать необходимости уничтожения и создания объектов сообщений и их интерфейсов. Когда клиент обмена сообщениями запрашивает загрузку другого сообщения, объект формы должен сохранить все изменения в свойствах текущего сообщения.
  
Сведения о перспективе объектов форм в клиентском приложении см. в поддоме "Объекты [настраиваемой формы MAPI".](mapi-custom-form-objects.md)
