# <a name="pidtagcontactaddressbookfolderentryid-canonical-property"></a>Каноническое свойство PidTagContactAddressBookFolderEntryId

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор для этой папки в хранилище сообщений, места хранения почтовых пользователей. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONTAB_FOLDER_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x6610  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Адресная книга контактов  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство в контейнере контактов адресной книги и на столбец в таблице контейнеров контактов адресной книги.
