# <a name="pidtagoriginalsendername-canonical-property"></a>Каноническое свойство PidTagOriginalSenderName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит отображаемую именем отправитель первой версии сообщения, то есть сообщение перед отправкой или ответом на него.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINAL_SENDER_NAME, PR_ORIGINAL_SENDER_NAME_A, PR_ORIGINAL_SENDER_NAME_W  <br/> |
|Идентификатор:  <br/> |0x005A  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства являются примерами свойств адресов для исходного отправитель сообщения. При первом представлении сообщения клиентская заявка должна установить **эти свойства** к значению свойства [PR_SENDER_NAME (PidTagSenderName).](pidtagsendername-canonical-property.md) Он никогда не меняется, когда сообщение переададовыв или отвечает на него.
