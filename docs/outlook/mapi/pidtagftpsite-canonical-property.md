# <a name="pidtagftpsite-canonical-property"></a>Каноническое свойство PidTagFtpSite

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит URL-адрес контакта. FTP — это протокол, который используется для передачи данных, как указано в [RFC959].
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ФТП_СИТЕ, ПР_ФТП_СИТЕ_А, ПР_ФТП_СИТЕ_В  <br/> |
|Идентификатор:  <br/> |0x3A4C  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
