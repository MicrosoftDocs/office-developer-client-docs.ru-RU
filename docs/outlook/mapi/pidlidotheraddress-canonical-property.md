# <a name="pidlidotheraddress-canonical-property"></a>Каноническое свойство PidLidOtherAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает полный адрес другого адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidOtherAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x0000801C  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть комбинацией других свойств физического адреса и основано на региональных региональных данных клиента.
