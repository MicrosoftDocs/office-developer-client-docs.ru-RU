# <a name="pidlidclassificationguid-canonical-property"></a>Каноническое свойство PidLidClassificationGuid

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает глобальный уникальный идентификатор (GUID) существующей классификации сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidClassGuid  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x000085B8  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Дополнительные сведения см. в тексте Как развернуть классификацию сообщений [для Outlook 2007 г.](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)
