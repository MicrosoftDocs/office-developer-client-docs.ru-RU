# <a name="pidlidclassificationguid-canonical-property"></a>Каноническое свойство PidLidClassificationGuid

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает глобальный уникальный идентификатор (GUID) существующей классификации сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidClassGuid  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x000085B8  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общие системы обмена сообщениями  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения дополнительных сведений см [порядок развертывания классификации сообщений для Outlook 2007](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)
