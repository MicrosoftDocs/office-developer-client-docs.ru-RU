# <a name="pidlidappointmentsequence-canonical-property"></a>Каноническое свойство PidLidAppointmentSequence

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает порядковый номер объекта собрания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидапптсекуенце  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008201  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Собрания  <br/> |
