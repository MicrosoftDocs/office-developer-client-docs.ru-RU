# <a name="forcing-a-notification"></a>Принудительное уведомление

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Когда поставщики услуг используют методы [имаписуппорт: IUnknown](imapisupportiunknown.md) для уведомлений, MAPI доставляет уведомления с помощью скрытого окна и соответствующей процедуры окна. Для каждого процесса, который должен получать уведомление, MAPI отправляет специальное сообщение в скрытое окно. Этому сообщению присвоено значение константы **сзмапинотификатионмсг** , которое определено в MAPIDEFS. Высоты. 
  
Эти уведомления появляются, когда в окне скрытого окна обрабатывается сообщение **сзмапинотификатионмсг** . Чтобы гарантировать доставку уведомлений, необходимо дождаться и отправить это сообщение **сзмапинотификатионмсг** . Реализация логики для достижения этого можно выполнить довольно просто, но MAPI предоставляет точку входа в БИБЛИОТЕКУ DLL MAPI под названием [хрдиспатчнотификатионс](hrdispatchnotifications.md) , чтобы сделать обработку еще проще. Для получения уведомлений в клиенте выЗовите **хрдиспатчнотификатионс** следующим образом: 
  
```cpp
HRESULT hr = HrDispatchNotifications(0);
 
```
