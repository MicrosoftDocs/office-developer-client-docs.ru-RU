# <a name="sshortarray"></a>SShortArray

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит массив неподписаенных целый значений, используемых для описания свойства типа PT_MV_SHORT.
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
   
```cpp
typedef struct _SShortArray
{
  ULONG cValues;
  short int FAR *lpi;
} SShortArray;

```

## <a name="members"></a>"Участники"

 **cValues**
  
> Количество значений в массиве, на который указывает член **LPI.** 
    
 **lpi**
  
> Указатель на массив неподписаных значений.
