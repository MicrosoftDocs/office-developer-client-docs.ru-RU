# <a name="displaying-table-rows-and-columns"></a>Отображение строк и столбцов таблиц

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
 Страница свойства может быть использована поставщиком адресных книг, чтобы пользователи могли определять новых получателей электронной почты. 
  
Соответствующая таблица отображения содержит четыре строки, по одному для каждого управления. Значения столбцов, которые указывают положение, являются следующими.
  
|**Control**|**XPOS**|**YPOS**|**DELTAX**|**DELTAY**|
|:-----|:-----|:-----|:-----|:-----|
|Отображение метки имени  <br/> |14   <br/> |18   <br/> |49  <br/> |8   <br/> |
|Отображение окна редактирования имени  <br/> |76  <br/> |16   <br/> |89  <br/> |12   <br/> |
|Метка адресов электронной почты  <br/> |14   <br/> |42  <br/> |50  <br/> |8   <br/> |
|Окно редактирования адресов электронной почты  <br/> |76  <br/> |40  <br/> |89  <br/> |12   <br/> |
|Флажок  <br/> |14   <br/> |64  <br/> |90  <br/> |12   <br/> |
   
В следующей таблице предлагаются соответствующие значения для типа управления, свойства **PR_CONTROL_TYPE** [(PidTagControlType)](pidtagcontroltype-canonical-property.md)и битмаски флагов, его **PR_CONTROL_FLAGS** [(PidTagControlFlags).](pidtagcontrolflags-canonical-property.md)
  
|**Control**|**Тип**|**Flags**|
|:-----|:-----|:-----|
|Отображение метки имени  <br/> |DTCT_LABEL  <br/> |0  <br/> |
|Отображение окна редактирования имени  <br/> |DTCT_EDIT  <br/> |DT_EDITABLE | DT_REQUIRED  <br/> |
|Метка адресов электронной почты  <br/> |DTCT_LABEL  <br/> |0  <br/> |
|Окно редактирования адресов электронной почты  <br/> |DTCT_EDIT  <br/> |DT_EDITABLE | DT_REQUIRED  <br/> |
|Флажок  <br/> |DTCT_CHECKBOX  <br/> |DT_EDITABLE  <br/> |
   
В итоговой таблице перечислены все управления содержимым связанной структуры управления. Обратите внимание, что значение для каждого из элементов управления меткой отображается в памяти непосредственно после структуры.
  
|**Control**|**Структура**|
|:-----|:-----|
|Отображение метки имени  <br/> |{sizeof (DTBLLABEL), 0} "Отображение имени:"  <br/> |
|Отображение окна редактирования имени  <br/> |{sizeof (DTBLEDIT), 0, 80, PR_DISPLAY_NAME}  <br/> |
|Метка адресов электронной почты  <br/> |{sizeof (DTBLLABEL), 0} "Адрес электронной почты:"  <br/> |
|Окно редактирования адресов электронной почты  <br/> |{sizeof (DTBLEDIT), 0, 80, PR_EMAIL_ADDRESS}  <br/> |
|Флажок  <br/> |PR_SEND_RICH_INFO  <br/> |
   
> [!NOTE]
> Кнопки **OK,** **Cancel** и **Help** не включены в таблицу отображения. Пользовательский интерфейс может добавлять контекст в диалоговое окно, добавляя элементы управления не в таблице отображения.
