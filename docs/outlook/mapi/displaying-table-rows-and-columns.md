# <a name="displaying-table-rows-and-columns"></a>Отображение строк и столбцов таблицы

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
 Поставщик адресной книги может использовать страницу свойств, чтобы пользователи могли определять новых получателей электронной почты. 
  
Соответствующая таблица отображения содержит четыре строки, по одной для каждого из них. Значения столбцов, которые указывают позицию, указаны следующим образом.
  
|**Control**|**XPOS**|**YPOS**|**DELTAX**|**DELTAY**|
|:-----|:-----|:-----|:-----|:-----|
|Метка отображаемой имени  <br/> |14   <br/> |18   <br/> |49  <br/> |8   <br/> |
|Окно редактирования отображаемой имени  <br/> |76  <br/> |16   <br/> |89  <br/> |12   <br/> |
|Метка адреса электронной почты  <br/> |14   <br/> |42  <br/> |50  <br/> |8   <br/> |
|Поле редактирования адреса электронной почты  <br/> |76  <br/> |40  <br/> |89  <br/> |12   <br/> |
|Флажок  <br/> |14   <br/> |64  <br/> |90  <br/> |12   <br/> |
   
В следующей таблице предлагаются соответствующие значения для типа, свойства **PR_CONTROL_TYPE** [(PidTagControlType)](pidtagcontroltype-canonical-property.md)и битовой маской флагов, его свойства **PR_CONTROL_FLAGS** ([PidTagControlFlags).](pidtagcontrolflags-canonical-property.md)
  
|**Control**|**Тип**|**Flags**|
|:-----|:-----|:-----|
|Метка отображаемой имени  <br/> |DTCT_LABEL  <br/> |0  <br/> |
|Окно редактирования отображаемой имени  <br/> |DTCT_EDIT  <br/> |DT_EDITABLE | DT_REQUIRED  <br/> |
|Метка адреса электронной почты  <br/> |DTCT_LABEL  <br/> |0  <br/> |
|Поле редактирования адреса электронной почты  <br/> |DTCT_EDIT  <br/> |DT_EDITABLE | DT_REQUIRED  <br/> |
|Флажок  <br/> |DTCT_CHECKBOX  <br/> |DT_EDITABLE  <br/> |
   
В последней таблице перечисляется каждый из них с содержимым связанной с ним структуры. Обратите внимание, что значение для каждого из элементов управления меткой отображается в памяти непосредственно после структуры.
  
|**Control**|**Структура**|
|:-----|:-----|
|Метка отображаемой имени  <br/> |{sizeof(DTBLLABEL), 0} "Display name:"  <br/> |
|Окно редактирования отображаемой имени  <br/> |{sizeof(DTBLEDIT), 0, 80, PR_DISPLAY_NAME}  <br/> |
|Метка адреса электронной почты  <br/> |{sizeof(DTBLLABEL), 0} "Адрес электронной почты:"  <br/> |
|Поле редактирования адреса электронной почты  <br/> |{sizeof(DTBLEDIT), 0, 80, PR_EMAIL_ADDRESS}  <br/> |
|Флажок  <br/> |PR_SEND_RICH_INFO  <br/> |
   
> [!NOTE]
> Кнопки **"ОК",** **"Отмена"** и "Справка" не включены в таблицу отображения.  Пользовательский интерфейс может добавлять контекст в диалоговое окно, добавляя элементы управления не в отображаемую таблицу.
