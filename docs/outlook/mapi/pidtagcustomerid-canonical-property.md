# <a name="pidtagcustomerid-canonical-property"></a>Каноническое свойство PidTagCustomerId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит ИДЕНТИФИКАЦИОНный номер клиента контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КУСТОМЕР_ИД, ПР_КУСТОМЕР_ИД_А, ПР_КУСТОМЕР_ИД_В  <br/> |
|Идентификатор:  <br/> |0x3A4A  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Замечания

Содержимое этих свойств определяется пользователем или организацией пользователя.
