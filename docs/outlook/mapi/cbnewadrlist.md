# <a name="cbnewadrlist"></a>CbNewADRLIST

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество bytes, которые должны быть выделены для новой [структуры ADRLIST,](adrlist.md) которая содержит определенное число получателей, представленных [структурами ADRENTRY.](adrentry.md) 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |**ADRLIST** <br/> |
   
```cpp
CbNewADRLIST (_centries)
```

## <a name="parameters"></a>Parameters

 _ _центрис_
  
> Количество **структур ADRENTRY,** которые будут включены в новую **структуру ADRLIST.**
