# <a name="pidtagstreetaddress-canonical-property"></a>Каноническое свойство PidTagStreetAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит адрес электронной почты получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_STREET_ADDRESS, PR_STREET_ADDRESS_A, PR_STREET_ADDRESS_W, PR_BUSINESS_ADDRESS_STREET, PR_BUSINESS_ADDRESS_STREET_A, PR_BUSINESS_ADDRESS_STREET_W  <br/> |
|Идентификатор:  <br/> |0x3A29  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения для идентификации и доступа получателя. Они определяются получателем и организацией получателя.
