# <a name="hrdecomposemsgid"></a>HrDecomposeMsgID

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Отделяет представление ASCII для идентификатора составной записи объекта, как правило, к сообщению в хранилище сообщений, к идентификатору записи этого объекта в хранилище и идентификатору записи хранилища. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Мапиутил. h  <br/> |
|Реализовано в:  <br/> |MAPI  <br/> |
|Вызывающая сторона:  <br/> |Клиентские приложения  <br/> |
   
```cpp
HrDecomposeMsgID(
  LPMAPISESSION psession,
  LPSTR szMsgID,
  ULONG FAR * pcbStoreEID,
  LPENTRYID FAR * ppStoreEID,
  ULONG FAR * pcbMsgEID,
  LPENTRYID FAR * ppMsgEID
);
```

## <a name="parameters"></a>Параметры

 _псессион_
  
> возврата Указатель на сеанс, используемый клиентским приложением. 
    
 _сзмсгид_
  
> возврата Строка, представляющая идентификатор записи объекта. 
    
 _пкбсториид_
  
> вышли Указатель на возвращенный размер (в байтах) идентификатора записи для хранилища сообщений, содержащего объект. Если параметр _сзмсгид_ указывает на строку несоставного идентификатора записи, параметр _пкбсториид_ указывает на ноль. 
    
 _ппсториид_
  
> вышли Указатель на указатель возвращаемого идентификатора записи хранилища сообщений, содержащего объект. Если параметр _сзмсгид_ указывает на несоставный идентификатор записи, в параметре _ппсториид_ возвращается значение null. 
    
 _пкбмсжеид_
  
> вышли Указатель на возвращенный размер (в байтах) идентификатора записи объекта в его хранилище. Если параметр _сзмсгид_ указывает на строку несоставного идентификатора записи, параметр _пкбмсжеид_ равен значению параметра _кбеид_ . 
    
 _ппмсжеид_
  
> вышли Указатель на указатель на возвращаемую строку идентификатора записи объекта в его хранилище. Если параметр _сзмсгид_ указывает на несоставной идентификатор записи, _ппмсжеид_ указывает на указатель на преобразованную копию несоставного идентификатора записи. 
    
## <a name="return-value"></a>Возвращаемое значение

Нет.
  
## <a name="remarks"></a>Примечания

Если идентификатор, указанный в параметре _сзмсгид_ , является составным, он преобразуется из ASCII и разбивается на идентификатор записи объекта в его банке сообщений и идентификаторе записи хранилища. Строки несоставных идентификаторов записей просто преобразуются и копируются. Строка составного идентификатора, которая должна быть разделена, обычно создана с помощью функции [хркомпосемсгид](hrcomposemsgid.md) . 
  
Вызов функции **хрдекомпосемсгид** эквивалентен вызову функции [хрентридфромсз](hrentryidfromsz.md) , а затем функции [хрдекомпосиид](hrdecomposeeid.md) .
