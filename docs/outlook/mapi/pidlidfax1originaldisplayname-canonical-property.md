# <a name="pidlidfax1originaldisplayname-canonical-property"></a>Каноническое свойство PidLidFax1OriginalDisplayName

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Задает исходное отображаемое имя контакта рабочий Факс адрес.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax1OriginalDisplayName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x000080B4  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство, если этот параметр указан, должен иметь значение совпадает со значением свойства **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).
