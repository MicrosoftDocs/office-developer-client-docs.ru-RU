# <a name="pidtagmessagelocaleid-canonical-property"></a>Каноническое свойство PidTagMessageLocaleId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит код языка Windows для конечного пользователя, создавшего сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_MESSAGE_LOCALE_ID  <br/> |
|Идентификатор:  <br/> |0x3FF1  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Разное  <br/> |
