# <a name="pidtagmessagelocaleid-canonical-property"></a>Каноническое свойство PidTagMessageLocaleId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор Windows конечного пользователя, создавшего это сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_МЕССАЖЕ_ЛОКАЛЕ_ИД  <br/> |
|Идентификатор:  <br/> |0x3FF1  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Разное  <br/> |
