# <a name="developing-a-mapi-transport-provider"></a>Разработка поставщика транспорта MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вы должны быть знакомы с архитектурой подсистем MAPI, языком C++ и с написанием библиотек динамической компоновки (DLL) для различных платформ Windows, прежде чем разрабатывать поставщик транспорта.
