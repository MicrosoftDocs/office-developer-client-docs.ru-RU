# <a name="pidtagabdefaultpab-canonical-property"></a>Каноническое свойство PidTagAbDefaultPab

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор контейнера адресной книги, который будет использоваться в качестве личной адресной книги (PAB). 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_АБ_ДЕФАУЛТ_ПАБ  <br/> |
|Идентификатор:  <br/> |0x3D07  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Адресная книга  <br/> |
   
## <a name="remarks"></a>Примечания

Не используйте это свойство. Оно зарезервировано для использования MAPI.
