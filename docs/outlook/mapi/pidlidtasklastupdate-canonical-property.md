# <a name="pidlidtasklastupdate-canonical-property"></a>Каноническое свойство PidLidTaskLastUpdate

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает скоординированное универсальное время (UTC) и дату последнего изменения, которое было сделано в задачу и указано **свойством dispidTaskHistory** [(PidLidTaskHistory).](pidlidtasklastupdate-canonical-property.md)
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskLastUpdate  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x00008115  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Задача  <br/> |
