# <a name="pidlidsharingprovidername-canonical-property"></a>Каноническое свойство PidLidSharingProviderName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает отображаемое пользователем имя поставщика общего доступа, которое определяется **dispidSharingProviderGuid** ([PidLidSharingProviderGuid).](pidlidsharingproviderguid-canonical-property.md) Это свойство сообщения общего доступа.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSharingProviderName  <br/> |
|Набор свойств:  <br/> |PSETID_Sharing  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008A02  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общий доступ  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство следует игнорировать.
