# <a name="displaying-table-controls"></a>Отображение элементов управления таблицы

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Существует множество различных типов элементов управления, нет уникальных для MAPI. Однако MAPI определяет собственные структуры, которые используются вместе с [буилддисплайтабле](builddisplaytable.md) , для описания уникального набора данных, связанных с каждым элементом управления. 
  
В следующей таблице перечислены структуры, описывающие каждый тип элемента управления. 
  
|**Структура элемента управления**|**Описание**|
|:-----|:-----|
|[DTBLBUTTON](dtblbutton.md) <br/> |Описывает элемент управления "Кнопка".  <br/> |
|[DTBLCHECKBOX](dtblcheckbox.md) <br/> |Описывает элемент управления "флажок".  <br/> |
|[DTBLCOMBOBOX](dtblcombobox.md) <br/> |Описывает элемент управления "поле со списком".  <br/> |
|[DTBLDDLBX](dtblddlbx.md) <br/> |Описывает элемент управления "раскрывающийся список".  <br/> |
|[DTBLEDIT](dtbledit.md) <br/> |Описывает элемент управления редактированием.  <br/> |
|[DTBLGROUPBOX](dtblgroupbox.md) <br/> |Описывает элемент управления "Группа".  <br/> |
|[DTBLLABEL](dtbllabel.md) <br/> |Описывает элемент управления "метка".  <br/> |
|[DTBLLBX](dtbllbx.md) <br/> |Описывает элемент управления "список".  <br/> |
|[DTBLMVDDLBOX](dtblmvddlbox.md) <br/> |Описывает элемент управления раскрывающегося списка с несколькими значениями.  <br/> |
|[DTBLMVLISTBOX](dtblmvlistbox.md) <br/> |Описывает элемент управления "список с несколькими значениями".  <br/> |
|[DTBLPAGE](dtblpage.md) <br/> |Описывает элемент управления "Вкладка".  <br/> |
|[DTBLRADIOBUTTON](dtblradiobutton.md) <br/> |Описывает элемент управления "переключатель".  <br/> |
