# <a name="pidtagformversion-canonical-property"></a>Каноническое свойство PidTagFormVersion

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит версию формы. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_FORM_VERSION, PR_FORM_VERSION_A, PR_FORM_VERSION_W  <br/> |
|Идентификатор:  <br/> |0x3301  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Общие mapI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства указывают, какая версия разработки в настоящее время действует для формы. Версия определяется и поддерживается конструктором формы и не обязательно связана с какой-либо версией компонента MAPI.
