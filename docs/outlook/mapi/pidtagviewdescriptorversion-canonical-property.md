# <a name="pidtagviewdescriptorversion-canonical-property"></a>Каноническое свойство PidTagViewDescriptorVersion

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает версию дескриптора представления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_VD_VERSION  <br/> |
|Идентификатор:  <br/> |0x7007  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Разное  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть равно 8.
