# <a name="ipm-subtree"></a>Подtree IPM

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
MAPI создает дерево папок под корневой папкой банка сообщений для всех клиентов, которые отправляют и получают сообщения от человека, а не от компьютера, получателей. Сообщения, которые обмениваются между людьми, называются межличностными сообщениями, а это дерево называется межличностным сообщением или поддеревом IPM. 
  
Поддереть IPM для хранения доставки состоит по крайней мере из следующих папок:
  
- Inbox;
    
- Исходящие
    
- Отправленные
    
- Удаленные
    
Это имена и роли по умолчанию для каждой из этих папок; Если имена по умолчанию не подходят, клиент может указать собственные имена. MAPI назначает имена и связи по умолчанию для этих папок, чтобы сообщения не исчезали случайно, если клиент не хочет устанавливать принимающие папки для сообщений. 
  
В контексте Microsoft Office Outlook поддерегов IPM состоит из дополнительных папок по умолчанию для календаря, контактов, задач, заметок и журнала.
  
Как правило, почтовый ящик содержит входящие сообщения, а "Исходящие" — исходящие сообщения (то есть сообщения, ожидающие их передачи). Папка "Отправленные" содержит копию каждого отправленного **сообщения,** если клиент установил для свойства PR_SENTMAIL_ENTRYID [(PidTagSentMailEntryId)](pidtagsentmailentryid-canonical-property.md)идентификатор записи этой папки. Папка "Удаленные" содержит сообщения, помеченные для удаления.
