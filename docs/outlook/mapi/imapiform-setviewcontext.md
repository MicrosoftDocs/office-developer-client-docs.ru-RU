# <a name="imapiformsetviewcontext"></a>IMAPIForm::SetViewContext

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Создает контекст представления для формы. 
  
```cpp
HRESULT SetViewContext(
  LPMAPIVIEWCONTEXT pViewContext
);
```

## <a name="parameters"></a>Параметры

 _Пвиевконтекст_
  
> возврата Указатель на новый контекст представления для формы.
    
## <a name="return-value"></a>Возвращаемое значение

S_OK 
  
> Контекст представления успешно задан.
    
## <a name="remarks"></a>Примечания

Средства просмотра форм вызывают метод **имапиформ:: сетвиевконтекст** для установки определенного контекста представления формы в качестве текущего. В форме одновременно может быть только один контекст представления. 
  
## <a name="notes-to-implementers"></a>Примечания для исполнителей

Большинство серверов форм реализуют **сетвиевконтекст** с помощью следующего алгоритма: 
  
- Если для формы уже существует контекст представления, отмените регистрацию формы, вызвав метод [имапивиевконтекст:: сетадвисесинк](imapiviewcontext-setadvisesink.md) с **нулевым значением** в параметре _пмнвс_ , а затем вызовите метод [IUnknown:: Release](https://msdn.microsoft.com/library/ms682317%28v=VS.85%29.aspx) контекста представления. , чтобы уменьшить счетчик ссылок. 
    
- Если новый контекст представления не **равен null**, вызовите метод **Имапивиевконтекст:: сетадвисесинк** с помощью параметра _пвиевконтекст_ , чтобы настроить новый приемник уведомлений о просмотре. 
    
- Если новый контекст представления не **равен null**, вызовите метод [Имапивиевконтекст:: жетвиевстатус](imapiviewcontext-getviewstatus.md) , чтобы определить, какие флаги состояния заданы. 
    
- Если новый контекст представления не равен **null**, сохраните его и вызовите его метод [IUnknown:: AddRef](https://msdn.microsoft.com/library/ms691379%28VS.85%29.aspx) , чтобы увеличить значение счетчика ссылок. 
    
- Обновите все элементы пользовательского интерфейса, зависящие от контекста представления. 
    
В зависимости от флагов состояния, возвращаемых из **имапивиевконтекст:: жетвиевстатус**, **сетвиевконтекст** также может выполнять другие действия. Например, если возвращаются флаги ВКСТАТУС_НЕКСТ и ВКСТАТУС_ПРЕВ, **сетвиевконтекст** может включить кнопки **Next** и **Previous** для нового контекста представления.
