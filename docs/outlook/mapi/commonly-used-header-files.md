# <a name="commonly-used-header-files"></a>Часто используемые файлы заголовки

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Каждая эталонная тема содержит таблицу, в которой именуется файл заголовки MAPI, в котором определен элемент API. В следующей таблице описываются часто используемые разработчиками файлы заголовок MAPI.
  
|**Файл header**|**Описание**|
|:-----|:-----|
|Mapi.h  <br/> |Определяет структуры и константы.  <br/> |
|Mapiaux.h  <br/> |Определяет дополнительные интерфейсы, структуры и константы, используемые интерфейсом программирования приложений обмена сообщениями.  <br/> |
|Mapicode.h  <br/> |Определяет коды состояния для значений ошибок.  <br/> |
|Mapidbg.h  <br/> |Определяет макрос для отладки.  <br/> |
|Mapidefs.h  <br/> |Определяет часто используемые типы событий, типы свойств, флаги, структуры и интерфейсы.  <br/> |
|Mapiform.h  <br/> |Определяет элементы форм.  <br/> |
|Mapiguid.h  <br/> |Определяет GUID-интерфейсы.  <br/> |
|Mapinis.h  <br/> |Определяет утилиты, поддерживают интернационализацию.  <br/> |
|Mapioid.h  <br/> |Определяет идентификаторы объектов MAPI.  <br/> |
|Mapispi.h  <br/> |Определяет флаги и интерфейсы, которые MAPI реализует для поставщиков услуг и служб сообщений.  <br/> |
|Mapitags.h  <br/> |Определяет теги свойств.  <br/> |
|Mapiutil.h  <br/> |Определяет полезные интерфейсы и функции.  <br/> |
|Mapival.h  <br/> |Определяет макрос проверки.  <br/> |
|Mapiwin.h  <br/> |Определяет типы, константы, флаги, функции и макрос для разработки поставщиков услуг из одного источника.  <br/> |
|Mapiwz.h  <br/> |Определяет прототипы и константы мастера поставщика.  <br/> |
|Mapix.h  <br/> |Определяет флаги и интерфейсы, которые mapI реализует для клиентов.  <br/> |
|Mspst.h  <br/> |Определяет свойства магазина персональных данных (PST-файла).  <br/> |
   
Сведения о получении файлов заголовок MAPI см. в [материалах Install MAPI Header Files.](how-to-install-mapi-header-files.md)
