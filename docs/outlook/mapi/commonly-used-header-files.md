# <a name="commonly-used-header-files"></a>Часто используемые файлы заголовков

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Каждый раздел справочника включает в себя таблицу, имя файла заголовка MAPI, где определен элемент API. В следующей таблице описываются файлы заголовков MAPI, которые разработчики часто используют.
  
|**Файл заголовка**|**Описание**|
|:-----|:-----|
|MAPI.h  <br/> |Определяет структур и констант.  <br/> |
|Mapiaux.h  <br/> |Определяет дополнительные интерфейсы, структуры и константы, используемые интерфейс программирования приложений системы обмена сообщениями.  <br/> |
|Mapicode.h  <br/> |Определяет коды состояния для ошибок.  <br/> |
|Mapidbg.h  <br/> |Определяет макросы для отладки.  <br/> |
|Mapidefs.h  <br/> |Определяет типы часто используемые событий, типы свойств, флаги, структуры и интерфейсы.  <br/> |
|Mapiform.h  <br/> |Определяет элементы форм.  <br/> |
|Mapiguid.h  <br/> |Определяет идентификаторы GUID.  <br/> |
|Mapinis.h  <br/> |Определяет служебных программ, которые поддерживают многоязыковой поддержки.  <br/> |
|Mapioid.h  <br/> |Определяет идентификаторы объектов MAPI.  <br/> |
|Mapispi.h  <br/> |Определяет флаги и интерфейсы, реализуемые программным интерфейсом MAPI для поставщиков услуг и службы сообщений.  <br/> |
|Mapitags.h  <br/> |Определяет свойство тегов.  <br/> |
|Mapiutil.h  <br/> |Определяет интерфейсы служебной программы и функции.  <br/> |
|Mapival.h  <br/> |Определяет проверки макросов.  <br/> |
|Mapiwin.h  <br/> |Определяет типы, константы, флаги, функции и макросы для разработки поставщиков услуг одного источника.  <br/> |
|Mapiwz.h  <br/> |Определяет прототипов и константы для мастера поставщика.  <br/> |
|Mapix.h  <br/> |Определяет флаги и интерфейсы, реализуемые программным интерфейсом MAPI для клиентов.  <br/> |
|MSPST.h  <br/> |Определяет свойства для хранения личных данных (PST-файл).  <br/> |
   
Сведения о том, как получить файлы заголовков MAPI, можно [Установить файлы заголовков MAPI](how-to-install-mapi-header-files.md).
