# <a name="commonly-used-header-files"></a>Часто используемые файлы заголовок

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Каждый справочный раздел содержит таблицу с именами файла заголовок MAPI, в котором определен элемент API. В следующей таблице описываются файлы заголовок MAPI, часто используемые разработчиками.
  
|**Файл загона**|**Описание**|
|:-----|:-----|
|Mapi.h  <br/> |Определяет структуры и константы.  <br/> |
|Mapiaux.h  <br/> |Определяет дополнительные интерфейсы, структуры и константы, используемые программным интерфейсом приложений для обмена сообщениями.  <br/> |
|Mapicode.h  <br/> |Определяет коды состояния для значений ошибок.  <br/> |
|Mapidbg.h  <br/> |Определяет макрос для отладки.  <br/> |
|Mapidefs.h  <br/> |Определяет часто используемые типы событий, типы свойств, флаги, структуры и интерфейсы.  <br/> |
|Mapiform.h  <br/> |Определяет элементы форм.  <br/> |
|Mapiguid.h  <br/> |Определяет GUID.  <br/> |
|Mapinis.h  <br/> |Определяет utilities that support internationalization.  <br/> |
|Mapioid.h  <br/> |Определяет идентификаторы объектов MAPI.  <br/> |
|Mapispi.h  <br/> |Определяет флаги и интерфейсы, реализованные MAPI для поставщиков услуг и служб сообщений.  <br/> |
|Mapitags.h  <br/> |Определяет теги свойств.  <br/> |
|Mapiutil.h  <br/> |Определяет сканные интерфейсы и функции.  <br/> |
|Mapival.h  <br/> |Определяет макрос проверки.  <br/> |
|Mapiwin.h  <br/> |Определяет типы, константы, флаги, функции и макрос для разработки поставщиков услуг с одним источником.  <br/> |
|Mapiwz.h  <br/> |Определяет прототипы и константы для мастера поставщика.  <br/> |
|Mapix.h  <br/> |Определяет флаги и интерфейсы, реализованные MAPI для клиентов.  <br/> |
|Mspst.h  <br/> |Определяет свойства личных данных (PST-файл).  <br/> |
   
Сведения о том, как получить файлы заголовок MAPI, см. в поддоме "Установка файлов [заголовок MAPI".](how-to-install-mapi-header-files.md)
