# <a name="pidtagalternaterecipientallowed-canonical-property"></a>Каноническое свойство PidTagAlternateRecipientAllowed

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит true, если отправитель разрешает автоматическую перенаправку этого сообщения. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ALTERNATE_RECIPIENT_ALLOWED  <br/> |
|Идентификатор:  <br/> |0x0002  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Примечания

Если автопереадка запрещена или альтернативный получатель не назначен, должен быть создан отчет о ненастройке.
