# <a name="pidtagalternaterecipientallowed-canonical-property"></a>Каноническое свойство PidTagAlternateRecipientAllowed

  
  
**Относится к**: Outlook 
  
Содержит значение TRUE, если отправитель разрешает автоматическое перенаправление этого сообщения. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ALTERNATE_RECIPIENT_ALLOWED  <br/> |
|Идентификатор:  <br/> |0x0002  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Если автоматическая переадресация не разрешается или обозначенный альтернативного получатели, будет создан отчет о недоставке.
