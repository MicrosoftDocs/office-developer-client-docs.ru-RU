# <a name="pidtagcomputernetworkname-canonical-property"></a>Каноническое свойство PidTagComputerNetworkName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя сети, используемой для передачи сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_COMPUTER_NETWORK_NAME, PR_COMPUTER_NETWORK_NAME_A, PR_COMPUTER_NETWORK_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A49  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Контакт  <br/> |
