# <a name="pidtagmiddlename-canonical-property"></a>Каноническое свойство PidTagMiddleName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит второе имя контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_MIDDLE_NAME, PR_MIDDLE_NAME_A, PR_MIDDLE_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A44  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Адрес  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем.
