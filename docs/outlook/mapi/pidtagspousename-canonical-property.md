# <a name="pidtagspousename-canonical-property"></a>Каноническое свойство PidTagSpouseName

  
  
**Относится к**: Outlook 
  
Содержит имя супруга пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SPOUSE_NAME, PR_SPOUSE_NAME_A, PR_SPOUSE_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A48  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства определяются пользователем или организации пользователя.
