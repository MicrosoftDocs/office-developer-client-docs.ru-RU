# <a name="optional-transport-provider-features"></a>НеОбязательные функции поставщика транспорта

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
НеОбязательные функции поставщики транспорта транспорта могут реализовать следующие функции:
  
- Регистрация параметров сообщений и получателей, относящихся к поставщику транспорта.
    
- Поддержание профиля (при необходимости) для хранения сведений о конфигурации и учетных данных в системе обмена сообщениями.
    
- Выполнение проверки учетных данных, необходимых для системы обмена сообщениями.
    
- Поддержка уведомлений о событиях для заинтересованных клиентских приложений с помощью метода [имаписуппорт:: notify](imapisupport-notify.md) . 
    
- Отображение страниц свойств конфигурации и диалоговых окон мастера, чтобы позволить пользователям настраивать параметры поставщика транспорта.
    
- Предоставление отчетов о доставке сообщений клиентским приложениям.
