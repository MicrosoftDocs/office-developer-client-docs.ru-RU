# <a name="optional-transport-provider-features"></a>Необязательные функции поставщика транспорта

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Следующие дополнительные возможности, которые можно реализовать поставщиками транспорта:
  
- Регистрация сообщений и получателей параметры, характерные для поставщика транспорта.
    
- Обслуживание профилей, если это необходимо для хранения конфигурационных данных и учетные данные в систему обмена сообщениями.
    
- Для выполнения проверки любого учетные данные, необходимые для системы обмена сообщениями.
    
- Поддержка уведомлений о событиях для заинтересованных клиентских приложений с помощью метода [IMAPISupport::Notify](imapisupport-notify.md) . 
    
- Отображение свойств конфигурации и мастер диалоговых окон, чтобы пользователи могли настроить параметры поставщика транспорта.
    
- Предоставление отчеты о доставке сообщения для клиентских приложений.
