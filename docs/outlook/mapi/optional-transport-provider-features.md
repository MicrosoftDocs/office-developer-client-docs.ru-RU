# <a name="optional-transport-provider-features"></a>Дополнительные функции поставщика транспорта

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Дополнительные функции, которые могут реализовать поставщики транспорта:
  
- Регистрация сообщений и параметров получателей, характерных для поставщика транспорта.
    
- Сохранение профиля при необходимости для хранения сведений о конфигурации и учетных данных в системе обмена сообщениями.
    
- Выполнение проверки учетных данных, необходимых системе обмена сообщениями.
    
- Поддержка уведомлений о событиях для заинтересованных клиентских приложений с помощью метода [IMAPISupport::Notify.](imapisupport-notify.md) 
    
- Отображение таблиц свойств конфигурации и диалоговых окно мастера, позволяющих пользователям настраивать параметры поставщика транспорта.
    
- Предоставление отчетов о доставке сообщений в клиентские приложения.
