# <a name="pidlidcurrentversion-canonical-property"></a>Каноническое свойство PidLidCurrentVersion

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит длинный номер сборки приложения Outlook для элемента.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidCurrentVersion  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008552  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
