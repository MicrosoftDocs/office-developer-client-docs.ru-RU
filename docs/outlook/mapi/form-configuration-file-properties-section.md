# <a name="form-configuration-file-properties-section"></a>Раздел Конфигурация формы [Свойства]

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
В **разделе [Свойства]** перечислены полные свойства, которые форма использует и публикует; то есть свойства, которые он создает в пользовательских сообщениях, которые клиентские приложения MAPI могут использовать при отображке столбцов, фильтрации таблиц контента, настройке папок результатов поиска и т. д. Каждая запись в этом списке свойств ссылается на **последующее [Свойство.** _строка_ **]** раздела, как показано ниже. 
  
 **[Свойства]**
  
 **Свойство.** _строка_  =   _строка_
  
Формат **[Свойства.** _строка_] Раздел: 
  
 **[Свойство.** _строка_ **]**
  
 **Тип**  =   _integer_
  
 **NmidPropset**  =   _guid_
  
 **NmidString**  =   _строка_
  
 **NmidInteger**  =   _integer_
  
 **DisplayName**  =   _строка_
  
 **Флаги**  =   _integer_
  
 **SpecialType** = 0|1 
  
 **Enum1**  =   _строка_
  
Каждое **[Свойство.** _в_ **разделе string ]** описывается одно свойство. В **записи** Type указывается тип свойства MAPI, например 3 (PT_I4) свойства. Запись **NmidPropset** необязательна; Вместе с **записью NmidString** или **записью NmidInteger** запись **NmidPropset** дает имя свойства. **NmidString** дает имя свойства, а **NmidInteger** — идентификатор свойства. **NmidString** и **NmidInteger** являются взаимоисключающими. 
  
Если установлено, **NmidPropset** должен содержать имя набора свойств; При отсутствии **NmidPropset** устанавливается значение по умолчанию, основанное на следующем правиле: если **NmidInteger** присутствует и его значение меньше 0x8000, **NmidPropset** задан для PS_MAPI. Если значение **NmidInteger** задано свыше 0x8000 или если оно отсутствует, **NmidPropset** задан для PS_PUBLIC_STRINGS. 
  
Запись **DisplayName** содержит метку свойства. Запись **SpecialType,** если она присутствует и не является, указывает на то, что это свойство является специальным свойством. В настоящее время единственным определенным типом специального свойства является **SpecialType** = 1, который указывает на свойства строки, указанные в строке. Если **в SpecialType** установлено 1, запись **Enum1 ссылается** на **[Enum1).** _строка_ **]** раздела. 
  
Ниже приводится пример **раздела [Свойства]** и **[Свойства.** _строка_ **]** раздела. 
  
```cpp
[Properties]
Property.1 = Fire Hazard
Property.2 = Safe
[Property.Fire Hazard]
Type = 1
NmidPropSet = {E47F4480-8400-101B-934D-04021C007002]
NmidString = FireHazard
DisplayName = Fire Hazard
SpecialType = 1
Enum1 = HazardEnum

```

Запись **Enum1** в предварительном примере ссылается на последующий **[Enum1.** _строка_ **]** раздел, описывающий описание определенного типа. Такое переумежение связывает первое свойство в **[Property.** _строка_ **]** раздела с свойством integer, называемым индексом. Такое перечисляние также содержит список возможных значений, которые может предполагать пара display-index. Указывать тип свойства для переумерия необязательно, так как по определению запись **Enum1** всегда имеет PT_I4 тип. Формат **[Enum1.** _строка_ **]** раздел: 
  
 **[Enum1.** _строка_ **]**
  
 **NmidPropset**  =   _guid_
  
 **NmidString**  =   _строка_
  
 **NmidInteger**  =   _integer_
  
 **EnumCount**  =   _integer_
  
 **Вал.** _integer_ **. Отображение**  =   _строки_
  
 **Вал.** _integer_ **. Индексация**  =   
  
Ниже приводится пример определения свойства для засвещенного свойства с именем Fire Hazard с возможными значениями Low, Medium и High.
  
```cpp
[Properties]
Property1 = Fire Hazard
[Enum1.HazardEnum]
IdxNmidPropset={E47F4480-8400-101B-934D-04021C007002]
IdxNmidString=FireHazardEnum
EnumCount = 3
Val.1.Display = Low
Val.1.Index = 1
Val.2.Display = Medium
Val.2.Index = 2
Val.3.Display = High
Val.3.Index = 3

```

 **[Enum1.** _строка_ **]** разделы могут использоваться приложениями для двух целей: для ускорения фильтрации свойств с помощью индекса вместо строки и сортировки по другому порядку, чем альфа-порядку значений строки. Например, сортировка может быть сделана на основе порядка с низким уровнем средней и высокой высоты, а не по заказу high-Medium-Low.
