# <a name="pidtagcorrelate-canonical-property"></a>Каноническое свойство PidTagCorrelate

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит значение TRUE, если отправитель сообщения запрашивает функцию корреляции системы обмена сообщениями.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КОРРЕЛАТЕ  <br/> |
|Идентификатор:  <br/> |0x0E0C  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Exchange;  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство используется для запроса корреляции входящих отчетов с исходным отправленным сообщением. Когда поставщик транспорта обнаруживает отправленное сообщение с **пр_коррелате** , для которого ЗАДАНО значение true, он задает для свойства **пр_коррелате_мтсид** ([PidTagCorrelateMtsid](pidtagcorrelatemtsid-canonical-property.md)) идентификатор системы передачи сообщений (MTS) для этого сообщения.
  
 **Пр_коррелате** следует использовать с системами обмена сообщениями, поддерживающими корреляцию по идентификатору MTS, например X. 400.
