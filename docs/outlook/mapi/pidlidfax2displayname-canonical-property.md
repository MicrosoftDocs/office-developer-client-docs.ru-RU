# <a name="pidlidfax2displayname-canonical-property"></a>Каноническое свойство PidLidFax2DisplayName

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Задает отображаемое имя адрес домашнего факса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax2DisplayName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x000080C0  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
