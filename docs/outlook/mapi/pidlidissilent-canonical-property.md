# <a name="pidlidissilent-canonical-property"></a>Каноническое свойство PidLidIsSilent

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, что пользователь не включил текст в текст ответа на собрание.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |LID_IS_SILENT  <br/> |
|Набор свойств:  <br/> |PSETID_Meeting  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00000004  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Собрания  <br/> |
