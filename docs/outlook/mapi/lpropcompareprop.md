# <a name="lpropcompareprop"></a>LPropCompareProp

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Сравнивает два значения свойства, чтобы определить, равны ли они. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Мапиутил. h  <br/> |
|Реализовано в:  <br/> |MAPI  <br/> |
|Вызывающая сторона:  <br/> |Клиентские приложения и поставщики услуг  <br/> |
   
```cpp
LONG LPropCompareProp(
  LPSPropValue lpSPropValueA,
  LPSPropValue lpSPropValueB
);
```

## <a name="parameters"></a>Параметры

 _лпспропвалуеа_
  
> возврата Указатель на структуру [спропвалуе](spropvalue.md) , определяющую значение первого свойства, которое требуется сравнить. 
    
 _лпспропвалуеб_
  
> возврата Указатель на структуру **спропвалуе** , определяющую второе значение свойства, которое требуется сравнить. 
    
## <a name="return-value"></a>Возвращаемое значение

 **Лпропкомпарепроп** возвращает одно из следующих значений для большинства типов свойств: 
  
- Меньше нуля, если значение, указанное параметром _лпспропвалуеа_ , меньше, чем указано в параметре _лпспропвалуеб_ . 
    
- Больше нуля, если значение, указанное в _лпспропвалуеа_ , больше, чем указано в _лпспропвалуеб_.
    
- Ноль, если значение, заданное параметром _лпспропвалуеа_ , равно значению, указанному в параметре _лпспропвалуеб_. 
    
Для типов свойств, не имеющих внутреннего упорядочения, таких как логические типы или типы ошибок, функция **лпропкомпарепроп** возвращает неопределенное значение, если два значения свойства не равны. Это неопределенное значение ненулевое и согласованное в вызовах. 
  
## <a name="remarks"></a>Примечания

Используйте функцию **лпропкомпарепроп** только в том случае, если типы двух сравниваемых свойств совпадают. 
  
Перед вызовом **лпропкомпарепроп**клиентское приложение или поставщик услуг сначала должны получить свойства для сравнения с вызовом метода [IMAPIProp::](imapiprop-getprops.md) . Когда клиент или поставщик вызывает **лпропкомпарепроп**, функция сначала проверяет теги свойств, чтобы убедиться, что сравнение значений свойств является допустимым. Затем функция сравнивает значения свойств, возвращая соответствующее значение. 
  
Если значения свойств не совпадают, **лпропкомпарепроп** определяет, какой из них больше. Свойства, сравниваемые **лпропкомпарепроп** , не обязательно должны принадлежать одному и тому же объекту.
