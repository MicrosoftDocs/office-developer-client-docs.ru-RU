# <a name="pidlidsharingresponsetime-canonical-property"></a>Каноническое свойство PidLidSharingResponseTime

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает время, когда получатель запроса на совместное использование отправил ответ на общий доступ. Это свойство сообщения о совместном доступе.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидшарингреспонсетиме  <br/> |
|Набор свойств:  <br/> |Псетид_шаринг  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008A28  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Общий доступ  <br/> |
