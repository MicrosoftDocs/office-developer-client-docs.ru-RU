# <a name="pidlidsharingresponsetime-canonical-property"></a>Каноническое свойство PidLidSharingResponseTime

  
  
**Относится к**: Outlook 
  
Задает время отправки общего доступа ответа получателя запрос общий доступ. Это свойство общего доступа сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSharingResponseTime  <br/> |
|Набор свойств:  <br/> |PSETID_Sharing  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008A28  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Sharing  <br/> |
