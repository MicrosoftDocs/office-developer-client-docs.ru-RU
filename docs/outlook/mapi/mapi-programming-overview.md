# <a name="mapi-programming-overview"></a>Общие сведения о программировании MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Эта справка по API сообщений Microsoft Outlook (MAPI) написана для разработчиков C и C++ с различными потребностями и опытом работы с сообщениями. Для тех разработчиков, которые хотят использовать MAPI для расширения своих приложений с функциями обмена сообщениями, никаких специальных знаний не требуется. Для использования MAPI для создания полномасштабных приложений рабочих групп или драйверов для специализированных систем обмена сообщениями необходим фон обмена сообщениями и модель COM.
  
Перед началом разработки следует учесть следующие сведения об использовании MAPI, процессе запуска и настройке профилей и служб сообщений.
  
MapI — это обширный набор функций, которые разработчики могут использовать для создания приложений с включенной поддержкой почты. Полная библиотека функций называется MAPI. MAPI обеспечивает полный контроль над системой обмена сообщениями на клиентском компьютере, создание сообщений и управление ими, управление почтовым ящиком клиента, поставщиками услуг и так далее.
  
> [!NOTE]
> Расширенный MAPI такой же, как MAPI, и просто называется MAPI в документации MAPI. 
  
 **Simple MAPI**
  
Simple MAPI предоставляет набор функций, позволяющих добавить базовый уровень функций обмена сообщениями в приложения на основе Microsoft Windows.
  
> [!IMPORTANT]
> Простая функция MAPI MAPISendMail поддерживается в Microsoft Outlook 2013 и Microsoft Outlook 2010, русская версия. Другие простые функции MAPI были неподготовлены в Windows.
