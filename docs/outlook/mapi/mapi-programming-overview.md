# <a name="mapi-programming-overview"></a>Общие сведения о программировании MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Эта ссылка Outlook microsoft Outlook (MAPI) написана для разработчиков C и C++ с различными потребностями и опытом работы с сообщениями. Для тех разработчиков, которые хотят использовать MAPI для расширения своих приложений с функциями обмена сообщениями, не требуется никаких специальных необходимых знаний. Чтобы использовать MAPI для создания полномасштабного приложения или драйверов для специализированных систем обмена сообщениями, вам потребуется фон в области обмена сообщениями и компонентная объектная модель (COM).
  
Перед началом работы над разработкой следует рассмотреть следующие сведения об использовании MAPI, процессе логоса и о том, как создаются и настраиваются профили и службы сообщений.
  
Интерфейс программы приложений для обмена сообщениями (MAPI) — это обширный набор функций, которые разработчики могут использовать для создания приложений с поддержкой почты. Полная библиотека функций называется MAPI. MAPI позволяет полностью контролировать систему обмена сообщениями на клиентском компьютере, создавать и управлять сообщениями, управлять почтовым ящиком клиента, поставщиками услуг и так далее.
  
> [!NOTE]
> Расширенный MAPI такой же, как MAPI, и просто упоминается как MAPI в документации MAPI. 
  
 **Simple MAPI**
  
Simple MAPI предоставляет набор функций, которые позволяют добавить базовый уровень функций обмена сообщениями в приложениях microsoft Windows на основе microsoft.
  
> [!IMPORTANT]
> Функция MAPI MAPI MAPISendMail поддерживается Microsoft Outlook 2013 и Microsoft Outlook 2010, русская версия. Другие простые функции MAPI были обесценивлены в Windows.
