# <a name="pidlidrecalltime-canonical-property"></a>Каноническое свойство PidLidRecallTime

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает время инициирования отзыва сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидрекаллтиме  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008549  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
