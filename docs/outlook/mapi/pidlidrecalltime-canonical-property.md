# <a name="pidlidrecalltime-canonical-property"></a>Каноническое свойство PidLidRecallTime

  
  
**Относится к**: Outlook 
  
Указывает время, в которой была инициирована отзыв сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidRecallTime  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008549  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Общие системы обмена сообщениями  <br/> |
