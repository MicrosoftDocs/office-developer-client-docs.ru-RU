# <a name="creating-a-profile-by-using-the-profile-wizard"></a>Создание профиля с помощью мастера профилей

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Мастер профилей — это функция MAPI, которая позволяет пользователю создавать профиль самым простым способом. Мастер профилей отображает ряд диалогов, в которых пользователю будет предложено выбрать службы сообщений и ввести значения для нескольких наиболее важных свойств конфигурации. Для большинства других необходимых свойств мастер профилей использует предоставленные значения по умолчанию. Чтобы вызвать мастер профилей, вызовите **LaunchWizard**, функцию, основанную на [прототипе LAUNCHWIZARDENTRY.](launchwizardentry.md) 
  
Пользователь может добавить только эти службы сообщений и поставщиков служб в новый профиль, поддерживаю который поддерживает мастер профилей. Так как для каждой службы сообщений может потребоваться настроить больше свойств, чем может обработать мастер профилей, следует помнить, что при использовании этого подхода можно неполно настроить одну или несколько выбранных служб или поставщиков.
