# <a name="creating-a-profile-by-using-the-profile-wizard"></a>Создание профиля с помощью мастера профилей

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Мастер профилей — это функция MAPI, которая позволяет пользователю создавать профиль самым простым способом. Мастер профилей отображает ряд диалогов, которые побуждают пользователя выбирать службы сообщений и вводить значения для нескольких наиболее важных свойств конфигурации. Для большинства других необходимых свойств мастер профилей использует предоставленные значения по умолчанию. Чтобы вызвать мастер профиля, позвоните **в LaunchWizard**, функцию, основанную на [прототипе LAUNCHWIZARDENTRY.](launchwizardentry.md) 
  
Пользователь может добавлять в новый профиль только те службы и поставщики сообщений, которые поддерживают мастер профилей. Поскольку для каждой службы сообщений может потребоваться больше свойств, чем может обрабатывать мастер профилей, следует помнить, что при использовании этого подхода может быть неполно настроена одна или несколько выбранных служб или поставщиков.
