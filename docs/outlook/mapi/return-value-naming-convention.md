# <a name="return-value-naming-convention"></a>Соглашение об именовании возвращаемых значений

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
MAPICODE. Файл заголовка содержит множество значений, что клиента или поставщика, может возвращать из реализацию метода интерфейса или может видеть возвращенных из звонка.
  
Коды для представления условия предупреждения и ошибки выполните различных соглашение об именовании, начинающийся с префикса MAPI, знака подчеркивания и W или E, чтобы указать тип кода. Оставшуюся часть кода является строкой краткое символов для описания условие. Каждого слова в строке, разделенных символом подчеркивания. Например значение ошибки MAPI_E_TOO_COMPLEX указывает, что реализации может не обрабатывать все, что был запрошен в вызове. Значение MAPI_W_PARTIAL_COMPLETION указывает, что вызов завершился успешно, но, что возникли проблемы. Часть операция успешно завершена.
