# <a name="pidtagphysicalforwardingaddressrequested-canonical-property"></a>Каноническое свойство PidTagPhysicalForwardingAddressRequested

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит значение TRUE, если отправитель сообщения запрашивает у агента передачи сообщений вложенный физический адрес для получателя сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PHYSICAL_FORWARDING_ADDRESS_REQUESTED  <br/> |
|Идентификатор:  <br/> |0x0C0E  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Получатель MAPI  <br/> |
