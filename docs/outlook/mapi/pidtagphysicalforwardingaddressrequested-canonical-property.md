# <a name="pidtagphysicalforwardingaddressrequested-canonical-property"></a>Каноническое свойство PidTagPhysicalForwardingAddressRequested

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение TRUE, если отправитель сообщения запрашивает у агента передачи сообщений физический адрес пересылки для получателя сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ФИСИКАЛ_ФОРВАРДИНГ_АДДРЕСС_РЕКУЕСТЕД  <br/> |
|Идентификатор:  <br/> |0x0C0E  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Получатель MAPI  <br/> |
