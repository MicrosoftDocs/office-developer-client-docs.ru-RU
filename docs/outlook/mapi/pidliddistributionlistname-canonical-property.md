# <a name="pidliddistributionlistname-canonical-property"></a>Каноническое свойство PidLidDistributionListName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает имя личного списка рассылки.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidDLName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008053  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть таким же, как и значение **свойства PR_DISPLAY_NAME** [(PidTagDisplayName).](pidtagdisplayname-canonical-property.md)
