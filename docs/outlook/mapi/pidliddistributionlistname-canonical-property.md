# <a name="pidliddistributionlistname-canonical-property"></a>Каноническое свойство PidLidDistributionListName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает имя личного списка рассылки.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспиддлнаме  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008053  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Комментарии

Значение этого свойства должно совпадать со значением свойства **пр_дисплай_наме** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)).
