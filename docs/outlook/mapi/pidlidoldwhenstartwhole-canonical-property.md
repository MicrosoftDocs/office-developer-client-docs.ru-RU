# <a name="pidlidoldwhenstartwhole-canonical-property"></a>Каноническое свойство PidLidOldWhenStartWhole

  
  
**Относится к**: Outlook 
  
Указывает исходное значение свойства **dispidApptStartWhole** ([PidLidAppointmentStartWhole](pidlidappointmentstartwhole-canonical-property.md)) до обновления собрания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidOldWhenStartWhole  <br/> |
|Набор свойств:  <br/> |PSETID_Meeting  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00000029  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Meetings (собрания);  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство не требуется.
