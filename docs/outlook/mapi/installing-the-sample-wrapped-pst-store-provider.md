1. <span data-ttu-id="02ee3-109">Чтобы загрузить оболочку PST поставщик хранилища, обратитесь к разделу [Outlook 2007 примеры Дополнительные справочные кода и распространяемого установщика](https://www.microsoft.com/en-us/download/details.aspx?id=24102).</span><span class="sxs-lookup"><span data-stu-id="02ee3-109">To download the Sample Wrapped PST Store Provider, see [Outlook 2007 Auxiliary Reference Code Samples and Redistributable Installer](https://www.microsoft.com/en-us/download/details.aspx?id=24102).</span></span>

5. <span data-ttu-id="02ee3-114">Откройте Visual Studio 2005 с правами администратора.</span><span class="sxs-lookup"><span data-stu-id="02ee3-114">Open Visual Studio 2005 as an administrator.</span></span>
    
    > [!NOTE]
    > <span data-ttu-id="02ee3-115">Если компьютер работает под управлением Windows XP, вы должны войти в систему с правами администратора.</span><span class="sxs-lookup"><span data-stu-id="02ee3-115">If your computer is running Windows XP, you must be logged in as an Administrator.</span></span> <span data-ttu-id="02ee3-116">Если компьютер работает под управлением ОС Windows Vista, вы должны войти в систему как администратор и щелкните правой кнопкой мыши значок Visual Studio 2005 и щелкните **Запуск от имени администратора**.</span><span class="sxs-lookup"><span data-stu-id="02ee3-116">If your computer is running Windows Vista, you must be logged in as an Administrator and you must right-click the Visual Studio 2005 icon and click **Run as administrator**.</span></span> 
  
6. <span data-ttu-id="02ee3-117">В Visual Studio 2005 откройте меню **файл**, выберите команду **Открыть**и щелкните **Проект/решение**.</span><span class="sxs-lookup"><span data-stu-id="02ee3-117">In Visual Studio 2005, click **File**, select **Open**, and then click **Project/Solution**.</span></span>
    
7. <span data-ttu-id="02ee3-118">Перейдите к папке, в которой вы сохранили примера **WrapPST**и нажмите кнопку **Открыть**.</span><span class="sxs-lookup"><span data-stu-id="02ee3-118">Browse to the location where you saved the sample, click **WrapPST**, and then click **Open**.</span></span>

9. <span data-ttu-id="02ee3-120">В диалоговом окне **Сохранение файла** нажмите кнопку **Сохранить**.</span><span class="sxs-lookup"><span data-stu-id="02ee3-120">In the **Save File As** dialog box, click **Save**.</span></span>
    
10. <span data-ttu-id="02ee3-121">В папке, где был сохранен примера щелкните правой кнопкой мыши файл **Install.bat** и выберите команду **Запуск от имени администратора**.</span><span class="sxs-lookup"><span data-stu-id="02ee3-121">In the folder where you saved the sample, right-click the **Install.bat** file and click **Run as administrator**.</span></span>
