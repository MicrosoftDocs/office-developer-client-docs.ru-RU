# <a name="outlook-mapi-reference"></a>Справочник по MAPI для Outlook

 
  
**Относится к**: Outlook 
  
Добро пожаловать на страницы справочника по MAPI для Microsoft Outlook 2013. MAPI предоставляет архитектуру обмена сообщениями для Microsoft Outlook 2013. MAPI для Outlook 2013 предоставляет набор интерфейсов, функций и других типов данных, помогающих при разработке приложений для обмена сообщениями в Outlook 2013. Приложения и службы транспорта используют MAPI для управления данными электронной почты, создания электронных сообщений и папок для их хранения, а также для поддержки уведомлений об изменениях имеющихся данных, связанных с MAPI.
  
> [!NOTE]
> Если у вас нет опыта разработки решений для Outlook, см. статью [Выбор API или технологии разработки решений для Outlook 2013](http://msdn.microsoft.com/ru-RU/library/jj900714.aspx), чтобы выбрать API-интерфейсы и технологии, наиболее соответствующие вашим потребностям.
