# <a name="outlook-mapi-reference"></a>Справочник по MAPI для Outlook
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Добро пожаловать на страницы справочника по API сообщений (MAPI) для Microsoft Outlook. MAPI предоставляет архитектуру обмена сообщениями для Microsoft Outlook 2013 и Outlook 2016. MAPI для Outlook предоставляет набор интерфейсов, функций и других типов данных, помогающих при разработке приложений для обмена сообщениями в Outlook. Приложения и службы транспорта используют MAPI для управления данными электронной почты, создания электронных сообщений и папок для их хранения, а также для поддержки уведомлений об изменениях имеющихся данных, связанных с MAPI.
  
> [!NOTE]
> Если у вас нет опыта разработки решений для Outlook, см. статью [Выбор API или технологии разработки решений для Outlook](../selecting-an-api-or-technology-for-developing-solutions-for-outlook.md), чтобы выбрать API-интерфейсы и технологии, наиболее соответствующие вашим потребностям.
