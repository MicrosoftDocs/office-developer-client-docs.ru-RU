# <a name="pidlidcommonstart-canonical-property"></a>Каноническое свойство PidLidCommonStart

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Представляет дату начала и время сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidCommonStart  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008516  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Общие системы обмена сообщениями  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство показывает время начала для элемента. Он должен быть меньше или равно значению свойства **dispidCommonEnd** ([PidLidCommonEnd](pidlidcommonend-canonical-property.md)).
  
Значение этого свойства должны быть эквивалент по Гринвичу (UTC) свойство **dispidTaskStartDate** ([PidLidTaskStartDate](pidlidtaskstartdate-canonical-property.md)).
