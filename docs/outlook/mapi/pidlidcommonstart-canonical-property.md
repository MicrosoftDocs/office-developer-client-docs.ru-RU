# <a name="pidlidcommonstart-canonical-property"></a>Каноническое свойство PidLidCommonStart

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Представляет дату и время начала сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidCommonStart  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008516  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство указывает время начала для элемента. Оно должно быть меньше или равно значению свойства **dispidCommonEnd** ([PidLidCommonEnd).](pidlidcommonend-canonical-property.md)
  
Значение этого свойства должно быть эквивалентом свойства **dispidTaskStartDate** [(PidLidTaskStartDate)](pidlidtaskstartdate-canonical-property.md)в UTC.
