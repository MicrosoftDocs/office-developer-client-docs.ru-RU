# <a name="pidtaglongtermentryidfromtable-canonical-property"></a>Каноническое свойство PidTagLongTermEntryIdFromTable

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Получает долгосрочный идентификатор записи элемента.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_LONGTERM_ENTRYID_FROM_TABLE  <br/> |
|Идентификатор:  <br/> |0x6670  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Свойства таблицы  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство можно использовать в таблице содержимого для получения идентификатора записи элемента в качестве долгосрочного идентификатора записи, а не краткого идентификатора записи. Сведения о долгосрочных и краткосрочных идентификаторах см. в PR_ENTRYID **(** [PidTagEntryId).](pidtagentryid-canonical-property.md)
