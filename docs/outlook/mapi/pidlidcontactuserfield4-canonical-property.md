# <a name="pidlidcontactuserfield4-canonical-property"></a>Каноническое свойство PidLidContactUserField4

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает четвертое поле контакта, которое предназначено для использования в различных целях контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidContactUserField4  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008052  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство содержит текст, не связанный с каким-либо другим свойством, зависящим от контакта.
