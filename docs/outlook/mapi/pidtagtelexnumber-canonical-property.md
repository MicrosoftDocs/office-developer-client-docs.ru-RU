# <a name="pidtagtelexnumber-canonical-property"></a>Каноническое свойство PidTagTelexNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит номер телекса получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ТЕЛЕКС_НУМБЕР, ПР_ТЕЛЕКС_НУМБЕР_А, ПР_ТЕЛЕКС_НУМБЕР_В  <br/> |
|Идентификатор:  <br/> |0x3A2C  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения о идентификации и доступе для получателя. Эти свойства определяются получателем и Организацией получателя.
