# <a name="ufromsz"></a>UFromSz

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Преобразует строку десятичных цифр с завершающим нулем в целое число без знака. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
|Реализовано в:  <br/> |MAPI  <br/> |
|Вызывающая сторона:  <br/> |Клиентские приложения и поставщики услуг  <br/> |
   
```cpp
UINT UFromSz(
  LPCSTR lpsz
);
```

## <a name="parameters"></a>Параметры

 _лпсз_
  
> возврата Указатель на преобразуемую строку, заканчивающуюся нулем. Значение параметра _лпсз_ не должно превышать 65536 символов. 
    
## <a name="return-value"></a>Возвращаемое значение

 **Уфромсз** возвращает целое число без знака. Если строка не начинается по крайней мере с одной десятичной цифрой, возвращается ноль. 
  
## <a name="remarks"></a>Примечания

Функция **уфромсз** завершает преобразование при достижении первого символа в строке, не являющийся десятичной цифрой. Например, если задается строка "55", **уфромсз** возвращает целое значение 55. При наличии строки "5a5b" функция возвращает целое значение 5. При наличии строки "a5b5" **уфромсз** возвращает ноль. 
  
 **Уфромсз** учитывает диакритические различия. Поддерживаются строки в форматах Юникод и DBCS. Предельная длина в _лпсз_ — символы, не обязательно — байты.
