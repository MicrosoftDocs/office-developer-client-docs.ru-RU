# <a name="pidtagobjecttype-canonical-property"></a>Каноническое свойство PidTagObjectType

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит тип объекта. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OBJECT_TYPE  <br/> |
|Идентификатор:  <br/> |0x0FFE  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Распространенная  <br/> |
   
## <a name="remarks"></a>Примечания

Тип объекта, содержащийся в этом свойстве, соответствует основному интерфейсу, доступного для объекта, доступного через интерфейс **OpenEntry.** Обычно он получается путем консультации с _параметром lpulObjType,_ возвращаемого соответствующим **методом OpenEntry.** При получении интерфейса другими способами вызовите [IMAPIProp::GetProps,](imapiprop-getprops.md) чтобы получить значение для этого свойства. 
  
Это свойство может иметь одно из следующих значений:
  
MAPI_ABCONT 
  
> Объект контейнера адресной книги 
    
MAPI_ADDRBOOK 
  
> Объект адресной книги 
    
MAPI_ATTACH 
  
> Объект вложения сообщения 
    
MAPI_DISTLIST 
  
> Объект списка рассылки 
    
MAPI_FOLDER 
  
> Объект папки 
    
MAPI_FORMINFO 
  
> Объект Form 
    
MAPI_MAILUSER 
  
> Объект пользователя системы обмена сообщениями 
    
MAPI_MESSAGE 
  
> Объект Message 
    
MAPI_PROFSECT 
  
> Объект раздела профиля 
    
MAPI_SESSION 
  
> Объект Session 
    
MAPI_STATUS 
  
> Объект Status 
    
MAPI_STORE 
  
> Объект хранилище сообщений
