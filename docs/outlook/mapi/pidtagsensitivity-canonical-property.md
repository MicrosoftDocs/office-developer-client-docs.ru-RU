# <a name="pidtagsensitivity-canonical-property"></a>Каноническое свойство PidTagSensitivity

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит значение, которое указывает на то, что отправитель сообщения придерживается конфиденциальности сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SENSITIVITY  <br/> |
|Идентификатор:  <br/> |0x0036  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Рекомендуется, чтобы объекты сообщений выдали это свойство.
  
Это свойство может иметь одно из следующих значений:
  
SENSITIVITY_NONE 
  
> Сообщение не имеет особой конфиденциальности.
    
SENSITIVITY_PERSONAL 
  
> Сообщение является личным.
    
SENSITIVITY_PRIVATE 
  
> Сообщение является частным.
    
SENSITIVITY_COMPANY_CONFIDENTIAL 
  
> Сообщение является конфиденциальным для компании.
