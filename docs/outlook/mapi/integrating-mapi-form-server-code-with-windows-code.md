# <a name="integrating-mapi-form-server-code-with-windows-code"></a>Интеграция кода сервера форм MAPI с кодом Windows

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Вспомним, что сервер формы является приложением Win32. Таким образом, существует ряд задач, связанных с загрузкой сервера форм в память и выходом из чистого режима. Как и для всех приложений Windows, точка входа для сервера формы является функцией **WinMain** . Эта функция является соответствующим местом для выполнения следующих задач: 
  
- Создание и регистрация класса окна, чтобы сервер форм мог взаимодействовать с другими компонентами OLE.
    
- Создание и регистрация класса или классов окна для пользовательских интерфейсов объектов формы.
    
- Вызов функции [мапиинитиализе](mapiinitialize.md) . Кроме того, **мапиинитиализе** обрабатывает необходимую инициализацию OLE. Это необходимо сделать один раз для каждого экземпляра сервера форм. 
    
- Регистрация глобального атома с строковым представлением идентификатора класса сервера форм (CLSID). Этот атом должен существовать в течение всего времени существования сервера форм.
    
- Вызов функции OLE [корегистерклассобжект](https://msdn.microsoft.com/library/ms693407.aspx) для регистрации фабрики классов сервера форм с помощью OLE. 
    
- Создание основного окна для получения сообщений. Это окно, вероятно, не обязательно должно быть видимым, так как пользователь будет взаимодействовать с определенными окнами, связанными с отдельными объектами формы. Однако во время разработки главное окно может быть удобным для отладки выходных данных или управления сервером формы.
    
- Создание цикла обработки сообщений, выполняемого в течение всего времени существования сервера форм, перевод и отправка сообщений Windows в активные объекты формы.
    
При завершении работы сервера форм необходимо выполнить следующие задачи:
  
- Вызовите функцию [КОРЕВОКЕКЛАССОБЖЕКТ](https://msdn.microsoft.com/library/ms688650%28VS.85%29.aspx) OLE, чтобы отозвать регистрацию OLE своего класса сообщений. 
    
- Вызов **мапиунинитиализе** для правильного закрытия подключения сервера форм к MAPI. 
    
- Удалите глобальный атом, содержащий строковое представление идентификатора класса.
