# <a name="pidlidfax1emailaddress-canonical-property"></a>Каноническое свойство PidLidFax1EmailAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает адрес электронной почты для факса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax1EmailAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x000080B3  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно содержать понятное отображаемое имя, за которым следует символ @, за которым следует номер факса.
