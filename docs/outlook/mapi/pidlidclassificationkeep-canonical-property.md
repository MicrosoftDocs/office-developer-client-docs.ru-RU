# <a name="pidlidclassificationkeep-canonical-property"></a>Каноническое свойство PidLidClassificationKeep

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, должна ли классификация сообщений сохраняться вместе с сообщением, если сообщение переададантовыно или отвечает.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidClassKeep  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x000085BA  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Дополнительные сведения см. в сведениях о развертывании [классификации сообщений для Outlook 2007.](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)
