# <a name="pidlidclassificationkeep-canonical-property"></a>Каноническое свойство PidLidClassificationKeep

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, следует ли классификации сообщений присвоенной этому сообщению, если сообщение будет переслано или него будет отправлен ответ.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidClassKeep  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x000085BA  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Общие системы обмена сообщениями  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения дополнительных сведений см [Развертывание классификации сообщений для Outlook 2007](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx).
