# <a name="pidlidclassificationkeep-canonical-property"></a>Каноническое свойство PidLidClassificationKeep

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, должна ли классификация сообщений храниться вместе с сообщением, если сообщение переадресовано или отвечено.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидкласскип  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000085BA  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
   
## <a name="remarks"></a>Примечания

Дополнительную информацию [можно узнать в статье как развернуть классификацию сообщений для Outlook 2007](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx).
