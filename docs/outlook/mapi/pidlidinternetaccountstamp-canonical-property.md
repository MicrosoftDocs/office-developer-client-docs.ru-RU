# <a name="pidlidinternetaccountstamp-canonical-property"></a>Каноническое свойство PidLidInternetAccountStamp

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает идентификатор учетной записи электронной почты, через которую отправляется сообщение электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидинетакктстамп  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008581  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
   
## <a name="remarks"></a>Замечания

Формат этой строки зависит от реализации. Это свойство можно использовать в клиенте, чтобы определить, какой сервер должен направить почту, но является необязательным и значение не имеет смысла для сервера.
