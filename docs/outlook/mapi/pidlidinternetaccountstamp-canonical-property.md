# <a name="pidlidinternetaccountstamp-canonical-property"></a>Каноническое свойство PidLidInternetAccountStamp

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает ИД учетной записи электронной почты, через которую отправляется сообщение электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidInetAcctStamp  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008581  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Формат этой строки зависит от реализации. Это свойство может использоваться клиентом для определения сервера, на который направляется почта, но является необязательным, а значение не имеет значения для сервера.
