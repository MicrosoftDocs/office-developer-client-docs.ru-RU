# <a name="setting-table-position-with-a-fractional-value"></a>Задание позиции таблицы с использованием дробного значения

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Пользователи таблиц могут перемещаться в позицию, представляющую Приблизительный процент строк относительно итогового значения. Вместо того чтобы перемещаться в точную строку, этот метод размещения разделяет таблицу на части на основе дробей. Пользователь таблицы может перемещаться, например, в двустороннюю точку таблицы или в строку, которая находится в 7/8 способе прокрутки таблицы. 
  
 **Перемещение курсора на приблизительное число строк**
  
- Вызов [IMAPITable:: сикроваппрокс](imapitable-seekrowapprox.md). **Сикроваппрокс** перемещается в строку, представляющую определенный процент строк относительно начала таблицы. Этот процент задается в параметрах _улнумератор_ и _улденоминатор_ . **Сикроваппрокс** часто используется для реализации полос прокрутки. 
    
 **Определение приблизительного положения таблицы**
  
- Вызов [IMAPITable:: куерипоситион](imapitable-queryposition.md). **Куерипоситион** можно использовать для информирования пользователя о текущей позиции. Он задает дробное значение на основе количества строк в таблице и номера текущей строки. Ожидается, что это значение представляет собой приближение. Разработчикам таблиц рекомендуется не вычислять точную позицию, так как точные реализации могут вызывать дорогостоящие затраты, особенно на классифицированных таблицах.
