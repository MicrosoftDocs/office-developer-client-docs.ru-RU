# <a name="setting-table-position-with-a-fractional-value"></a>Установка положения таблицы с использованием дробного значения

  
  
**Относится к**: Outlook 
  
В таблице пользователи могут перемещаться в позицию, представляющий приблизительно, например процент строк при использовании Общая. Вместо перемещения в точной строки, этот метод размещения разделяет таблицы в части на основе доли. В таблице пользователи могут переместить, например точка наполовину способ таблицы или строку, в которой — 7 и 8 способ из таблицы. 
  
 **Перемещение курсора приблизительное число строк**
  
- Вызов [IMAPITable::SeekRowApprox](imapitable-seekrowapprox.md). **SeekRowApprox** перемещает строку, представляющую определенный процент строк при использовании начала таблицы. Этот процент указано в параметрах _ulNumerator_ и _ulDenominator_ . **SeekRowApprox** часто используется для реализации полосы прокрутки. 
    
 **Чтобы определить, приблизительно, например положение таблицы**
  
- Вызов [IMAPITable::QueryPosition](imapitable-queryposition.md). **QueryPosition** можно использовать для оповещения пользователя о текущей позиции. Задает дробное значение на основе номер текущей строки и число строк в таблице. Ожидается, что это значение представляет приблизительным. В таблице специалистов по внедрению, рекомендуется не для вычисления точное положение, поскольку точных реализации может быть дорогим для вызова, особенно в таблицах по категориям.
