# <a name="setting-table-position-with-a-fractional-value"></a>Настройка положения таблицы с дробным значением

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Пользователи таблицы могут перейти к позиции, представляющую примерный процент строк по отношению к общему. Вместо перехода к точной строке этот метод позиционирования делит таблицу на части, основанные на фракциях. Пользователи таблицы могут перемещаться, например, в точку на полпути таблицы или в строку, которая находится в 7/8 пути через таблицу. 
  
 **Перемещение курсора приблизительное число строк**
  
- Вызов [IMAPITable::SeekRowApprox](imapitable-seekrowapprox.md). **SeekRowApprox** перемещается в строку, которая представляет определенный процент строк по отношению к началу таблицы. Этот процент указан в _параметрах ulNumerator_ и _ulDenominator._ **SeekRowApprox** часто используется для реализации свитков. 
    
 **Определение приблизительного положения таблицы**
  
- Вызов [IMAPITable::QueryPosition](imapitable-queryposition.md). **QueryPosition** можно использовать для информирования пользователя о текущей позиции. Он задает дробное значение в зависимости от количества строк в таблице и числа текущей строки. Ожидайте, что это значение представляет приближение. Реализации таблиц рекомендуется не вычислять точную позицию, так как точные реализации могут быть дорогостоящими для вызова, особенно на классифицируются таблицы.
