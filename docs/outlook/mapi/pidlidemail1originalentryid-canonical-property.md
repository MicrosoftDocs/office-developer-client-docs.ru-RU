# <a name="pidlidemail1originalentryid-canonical-property"></a>Каноническое свойство PidLidEmail1OriginalEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает **entryId объекта,** соответствующего первому адресу электронной почты. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail1OriginalEntryID  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008085  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть как разовый **EntryId** для этого электронного адреса, так и допустимый объект Address Book **EntryId**.
