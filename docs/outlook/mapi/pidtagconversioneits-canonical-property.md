# <a name="pidtagconversioneits-canonical-property"></a>Каноническое свойство PidTagConversionEits

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит закодированные типы сведений (EITs), которые применяются к сообщению в пути для описания преобразований.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONVERSION_EITS  <br/> |
|Идентификатор:  <br/> |0x000C  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Примечания

Среды X.400 используют это свойство как для отчетов о невывозе, так и для доставки.
