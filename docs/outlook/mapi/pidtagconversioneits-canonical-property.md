# <a name="pidtagconversioneits-canonical-property"></a>Каноническое свойство PidTagConversionEits

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит закодированные типы данных (ЕИТС), которые применяются к сообщению в транзитном транспорте для описания преобразований.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КОНВЕРСИОН_ЕИТС  <br/> |
|Идентификатор:  <br/> |0x000C  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Комментарии

В средах X. 400 это свойство используется как для отчетов о недоставке, так и для отчетов о доставке.
