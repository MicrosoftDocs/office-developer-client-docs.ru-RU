# <a name="pidlidtaskownership-canonical-property"></a>Каноническое свойство PidLidTaskOwnership

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает роль текущего пользователя относительно задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskOwnership  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x00008129  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть одним из следующих значений.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0x00000000  <br/> |Задача не назначена.  <br/> |
|0x00000001  <br/> |Задача — это копия задачи, которую назначит назначитель задачи.  <br/> |
|0x00000002  <br/> |Задача — это копия задачи, которую назначает назначить.  <br/> |
