# <a name="pidtagoriginatorreturnaddress-canonical-property"></a>Каноническое свойство PidTagOriginatorReturnAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит обратный адрес, закодированный в двоичном коде инициатора сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ОРИГИНАТОР_РЕТУРН_АДДРЕСС  <br/> |
|Идентификатор:  <br/> |0x0024  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |MIME  <br/> |
