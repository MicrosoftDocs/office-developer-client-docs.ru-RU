# <a name="pidtagaddresstype-canonical-property"></a>Каноническое свойство PidTagAddressType

**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит тип адреса электронной почты пользователя, например SMTP.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ADDRTYPE, PR_ADDRTYPE_A, PR_ADDRTYPE_W  <br/> |
|Идентификатор:  <br/> |0x3002  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства являются примерами свойств базового адреса, общих для всех пользователей системы обмена сообщениями. Он указывает, какую систему обмена сообщениями MAPI использует для обработки заданного сообщения.
  
Это свойство также определяет формат строки адреса в PR_EMAIL_ADRESS **(** [PidTagEmailAddress).](pidtagemailaddress-canonical-property.md) Строка, которая предоставляется, может содержать только буквы в верхнем регистре от A до Z и числа от 0 до 9.
  
Допустимые примеры для строки: 
  
```cpp
FAX
MHS
PROFS
SMTP
X400

```
