# <a name="pidtagreceivedbyentryid-canonical-property"></a>Каноническое свойство PidTagReceivedByEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор входа пользователя обмена сообщениями, который получает сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RECEIVED_BY_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x003F  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Адресная книга  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является одним из свойств адресов для пользователя обмена сообщениями, который получает сообщение. Он должен быть установлен поставщиком входящих транспортных средств.
  
Это свойство соответствует атрибуту доставки X.400 на MH_T_ сообщений.
