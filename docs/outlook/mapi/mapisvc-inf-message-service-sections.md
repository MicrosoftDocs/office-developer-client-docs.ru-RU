# <a name="mapisvcinf-message-service-sections"></a>Разделы службы сообщений MapiSvc.inf

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Mapisvc.inf включает в себя один раздел службы сообщений для каждой записи, указанные в разделе **[Services]** . Существует два типа записей в следующих разделах: один для задания определенных свойств, а другой для списка имена разделов, относящихся к службе сообщений выполняется настройка.
