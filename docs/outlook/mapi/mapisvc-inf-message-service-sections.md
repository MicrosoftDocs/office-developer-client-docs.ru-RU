# <a name="mapisvcinf-message-service-sections"></a>Разделы службы сообщений MapiSvc.inf

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
mapisvc.inf содержит один раздел службы сообщений для каждой записи, перечисленной в **разделе [Службы].** В этих разделах есть два типа записей: один для настройки определенных свойств, а другой для перечисления имен разделов, связанных с настраиваемой службой сообщений.
