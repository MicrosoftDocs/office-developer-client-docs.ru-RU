# <a name="mapisvcinf-message-service-sections"></a>Разделы службы сообщений MapiSvc. INF

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Mapisvc. inf содержит один раздел службы сообщений для каждой из записей, перечисленных в разделе **[службы]** . В этих разделах есть два типа записей: один для установки определенных свойств, а другой — для списка имен разделов, связанных с настраиваемой службой сообщений.
