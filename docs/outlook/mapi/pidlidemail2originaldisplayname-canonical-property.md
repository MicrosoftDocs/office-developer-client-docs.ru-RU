# <a name="pidlidemail2originaldisplayname-canonical-property"></a>Каноническое свойство PidLidEmail2OriginalDisplayName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает второе имя отображения, соответствующее адресу электронной почты, указанному для контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail2OriginalDisplayName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008094  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Если значение свойства **dispidEmail2AddrType** [(PidLidEmail2AddressType)](pidlidemail2addresstype-canonical-property.md)является "SMTP", значение соответствующего свойства **PidLidEmail2OriginalDisplayName** должно равняться значению соответствующего свойства **dispidEmail2EmailAddress** [(PidLidEmail2EmailAddress).](pidlidemail2emailaddress-canonical-property.md) Целью этого свойства является отображение альтернативного удобного для пользователя адреса, эквивалентного адресу **dispidEmail2EmailAddress.**
