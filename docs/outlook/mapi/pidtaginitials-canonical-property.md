# <a name="pidtaginitials-canonical-property"></a>Каноническое свойство PidTagInitials

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит инициалы для частей полного имени получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Идентификатор:  <br/> |0x3A0A  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения для идентификации и доступа получателя. Они определяются получателем и его организацией.
