# <a name="pidtaginitials-canonical-property"></a>Каноническое свойство PidTagInitials

  
  
**Относится к**: Outlook 
  
Содержит инициалы для части полного имени получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Идентификатор:  <br/> |0x3A0A  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства предоставляют идентификации и доступа к сведениям для получателя. Они определены получателя и их организации.
