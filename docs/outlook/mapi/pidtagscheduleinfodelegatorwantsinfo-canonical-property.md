# <a name="pidtagscheduleinfodelegatorwantsinfo-canonical-property"></a>Каноническое свойство PidTagScheduleInfoDelegatorWantsInfo

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит сведения о делегировании, которые хотят получать информационные обновления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_СЧДИНФО_БОСС_ВАНТС_ИНФО  <br/> |
|Идентификатор:  <br/> |0x684B  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Сведения о доступности  <br/> |
