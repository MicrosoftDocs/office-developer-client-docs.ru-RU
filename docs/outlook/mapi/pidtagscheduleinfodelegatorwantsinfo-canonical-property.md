# <a name="pidtagscheduleinfodelegatorwantsinfo-canonical-property"></a>Каноническое свойство PidTagScheduleInfoDelegatorWantsInfo

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит делегатор, который хочет получать информационные обновления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SCHDINFO_BOSS_WANTS_INFO  <br/> |
|Идентификатор:  <br/> |0x684B  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Бесплатный/занятый  <br/> |
