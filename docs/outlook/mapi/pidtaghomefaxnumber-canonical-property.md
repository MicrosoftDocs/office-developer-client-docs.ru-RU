# <a name="pidtaghomefaxnumber-canonical-property"></a>Каноническое свойство PidTagHomeFaxNumber

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит номер телефона домашнего факс-компьютера получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_FAX_NUMBER, PR_HOME_FAX_NUMBER_A, PR_HOME_FAX_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A25  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения об идентификации и доступе к получателю. Они определяются получателем и его организацией.
