# <a name="pidtagdisclosureofrecipients-canonical-property"></a>Каноническое свойство PidTagDisclosureOfRecipients

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит true, если разрешено разглашение получателей.
