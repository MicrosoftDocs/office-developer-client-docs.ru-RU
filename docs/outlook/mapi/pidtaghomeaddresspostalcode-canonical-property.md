# <a name="pidtaghomeaddresspostalcode-canonical-property"></a>Каноническое свойство PidTagHomeAddressPostalCode

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит почтовый индекс для домашнего адреса пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ХОМЕ_АДДРЕСС_ПОСТАЛ_КОДЕ, ПР_ХОМЕ_АДДРЕСС_ПОСТАЛ_КОДЕ_А, ПР_ХОМЕ_АДДРЕСС_ПОСТАЛ_КОДЕ_В  <br/> |
|Идентификатор:  <br/> |0x3A5B  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства определяются пользователем или организацией пользователя.
