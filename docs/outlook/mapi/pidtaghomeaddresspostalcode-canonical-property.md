# <a name="pidtaghomeaddresspostalcode-canonical-property"></a>Каноническое свойство PidTagHomeAddressPostalCode

  
  
**Относится к**: Outlook 
  
Содержит почтовый индекс для домашний адрес пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_ADDRESS_POSTAL_CODE, PR_HOME_ADDRESS_POSTAL_CODE_A, PR_HOME_ADDRESS_POSTAL_CODE_W  <br/> |
|Идентификатор:  <br/> |0x3A5B  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства определяются пользователем или организации пользователя.
