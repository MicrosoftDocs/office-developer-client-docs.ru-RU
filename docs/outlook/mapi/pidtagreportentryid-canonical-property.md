# <a name="pidtagreportentryid-canonical-property"></a>Каноническое свойство PidTagReportEntryId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор записи для получателя, который должен получать отчеты по этому сообщению.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_REPORT_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x0045  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является одним из свойств адреса получателя, которому отправитель делегировал получение отчетов, созданных для этого сообщения.
  
Клиентские приложения, которые должны маршрутить отчеты другому пользователю, должно установить это свойство во время отправки сообщения. Если он не установлен, отчеты отправляются отправителю сообщения.
