# <a name="pidlidemail2originalentryid-canonical-property"></a>Каноническое свойство PidLidEmail2OriginalEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает идентификатор **EntryID** объекта, соответствующий второму адресу электронной почты. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail2OriginalEntryID  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008095  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Комментарии

Значение этого свойства должно быть одноразовым **EntryID** для этого электронного адреса или допустимым идентификатором **EntryID**объекта адресной книги.
