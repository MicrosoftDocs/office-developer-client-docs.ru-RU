# <a name="pidlidemail2originalentryid-canonical-property"></a>Каноническое свойство PidLidEmail2OriginalEntryId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает **EntryId объекта,** который соответствует второму адресу электронной почты. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail2OriginalEntryID  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008095  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть одноразовым **EntryId** для этого электронного адреса или допустимым объектом адресной книги **EntryId.**
