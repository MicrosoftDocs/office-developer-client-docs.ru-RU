# <a name="bookmark"></a>BOOKMARK

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Определяет данные закладки для запоминания позиции в таблице. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанные методы:  <br/> |[IMAPITable::CreateBookmark](imapitable-createbookmark.md)[IMAPITable::FreeBookmark](imapitable-freebookmark.md) <br/> |
   
```cpp
typedef ULONG_PTR BOOKMARK;
```

## <a name="remarks"></a>Примечания

MAPI определяет три закладки, перечисленные следующим образом:
  
BOOKMARK_BEGINNING 
  
> Запоминает исходное положение таблицы. 
    
BOOKMARK_CURRENT 
  
> Запоминает текущее положение таблицы.
    
BOOKMARK_END 
  
> Запоминает конечную позицию таблицы.
    
Клиенты могут создавать другие закладки для запоминания других позиций таблицы. Закладки действительны только в том случае, если таблица открыта. Клиенты должны освободить все закладки, созданные ими, перед закрытием связанной таблицы.
