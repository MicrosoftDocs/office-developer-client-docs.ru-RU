# <a name="pidtagoriginallyintendedrecipentryid-canonical-property"></a>Каноническое свойство PidTagOriginallyIntendedRecipEntryId

  
  
**Относится к**: Outlook 
  
Содержит идентификатор записи изначально требуемого получателя сообщения автоматически переслано.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINALLY_INTENDED_RECIP_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x1012  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Сервер  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство является одним из свойств адреса для получателя, изначально предполагаемая сообщения. Он должен иметь значение агентом автоматическое перенаправление сообщения.
  
Это свойство соответствует атрибуту каждого получателя отчета X.400.
