# <a name="pidtagoriginallyintendedrecipentryid-canonical-property"></a>Каноническое свойство PidTagOriginallyIntendedRecipEntryId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор изначально предполагаемого получателя сообщения с автоматическим пересылкой.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINALLY_INTENDED_RECIP_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x1012  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Сервер  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является одним из свойств адреса изначально назначенного получателя сообщения. Он должен быть задан автоматическим агентом, который перенаправлял сообщение.
  
Это свойство соответствует атрибуту отчета X. 400 отчета "на получателя".
