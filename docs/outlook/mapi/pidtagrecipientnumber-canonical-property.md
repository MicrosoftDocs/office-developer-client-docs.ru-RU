# <a name="pidtagrecipientnumber-canonical-property"></a>Каноническое свойство PidTagRecipientNumber

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Представляет получателей количество получателей на сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RECIPIENT_NUMBER  <br/> |
|Идентификатор:  <br/> |0x6662  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Сообщение Exchange чтения/записи  <br/> |
