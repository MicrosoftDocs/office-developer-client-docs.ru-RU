# <a name="cbmapiverbarray"></a>CbMAPIVerbArray

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество байтов в существующей структуре [смапивербаррай](smapiverbarray.md) , которая содержит указанное число структур [смапиверб](smapiverb.md) . 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Мапиформ. h  <br/> |
|Связанная структура:  <br/> |**SMAPIVerbArray** <br/> |
   
```cpp
int CbMAPIVerbArray (_c)
```

## <a name="parameters"></a>Параметры

 __c_
  
> Количество структур **смапиверб** в существующей структуре **смапивербаррай** .
