# <a name="cbmapiverbarray"></a>CbMAPIVerbArray

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Вычисляет число байтов в структуру [SMAPIVerbArray](smapiverbarray.md) , содержащий указанное число [SMAPIVerb](smapiverb.md) структуры. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapiform.h  <br/> |
|Связанные структуры:  <br/> |**SMAPIVerbArray** <br/> |
   
```cpp
int CbMAPIVerbArray (_c)
```

## <a name="parameters"></a>Параметры

 __c_
  
> Число структур **SMAPIVerb** в имеющуюся структуру **SMAPIVerbArray** .
