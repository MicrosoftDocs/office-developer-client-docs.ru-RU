# <a name="pidtagprovideritemid-canonical-property"></a>Каноническое свойство PidTagProviderItemId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает идентификатор папки или элемента в магазине.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PROVIDER_ITEMID  <br/> |
|Идентификатор:  <br/> |0x0EA3  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |MapiNonTransmittable  <br/> |
   
## <a name="remarks"></a>Примечания

Поставщики магазинов могут указать значение для этого свойства для папки или элемента, но между сеансами значение должно быть одинаковым. Поставщики магазинов используют это свойство для определения результатов поиска, возвращаемой из поисковой системы.
