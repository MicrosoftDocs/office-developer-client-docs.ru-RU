# <a name="pidlidfax2addresstype-canonical-property"></a>Каноническое свойство PidLidFax2AddressType

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает тип адреса для домашнего факса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax2AddrType  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080C2  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство, если присутствует, должно быть задавано "ФАКС".
