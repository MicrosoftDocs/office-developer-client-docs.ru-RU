# <a name="pidtagrtfsynctrailingcount-canonical-property"></a>Каноническое свойство PidTagRtfSyncTrailingCount

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит количество игнорируемых символов, которые отображаются после значащих символов сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RTF_SYNC_TRAILING_COUNT  <br/> |
|Идентификатор:  <br/> |0x1011  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Сообщение MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является дополнительным свойством в формате текста в формате RTF (РФТ). Эти свойства используются функцией [ртфсинк](rtfsync.md) и не предназначены для непосредственного использования клиентскими приложениями.
