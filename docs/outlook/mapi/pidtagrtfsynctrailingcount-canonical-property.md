# <a name="pidtagrtfsynctrailingcount-canonical-property"></a>Каноническое свойство PidTagRtfSyncTrailingCount

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит количество игнорируемых символов, которые отображаются после важных символов сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RTF_SYNC_TRAILING_COUNT  <br/> |
|Идентификатор:  <br/> |0x1011  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Сообщение MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство является вспомогательным свойством Rich Text Format (RFT). Эти свойства используются функцией [RTFSync](rtfsync.md) и не предназначены для непосредственного использования клиентских приложений.
