# <a name="pidtagserviceinstallid-canonical-property"></a>Каноническое свойство PidTagServiceInstallId

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Идентификатор компонента поставщика.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SERVICE_INSTALL_ID, PR_SERVICE_INSTALL_ID_A, PR_SERVICE_INSTALL_ID_W  <br/> |
|Идентификатор:  <br/> |0x3D13  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Профиль MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства могут быть использованы в качестве компонента параметра вызова **MsiProvideQualifiedComponent** для установки поставщика.
