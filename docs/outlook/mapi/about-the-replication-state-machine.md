2. <span data-ttu-id="ea167-113">Outlook 2013 или Outlook 2010 г. выделяет структуру данных и инициализирует структуру данных с необходимой для клиента информацией.</span><span class="sxs-lookup"><span data-stu-id="ea167-113">Outlook 2013 or Outlook 2010 allocates the data structure and initializes the data structure with the necessary information for the client.</span></span>
