# <a name="pidtaggender-canonical-property"></a>Каноническое свойство PidTagGender

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит пол пользователя обмена сообщениями.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_GENDER  <br/> |
|Идентификатор:  <br/> |0x3A4D  <br/> |
|Тип данных:  <br/> |PT_I2  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство предоставляет идентификацию и доступ к данным о пользователе обмена сообщениями и контенте. Содержимое определяется пользователем обмена сообщениями и организацией пользователя обмена сообщениями. 
  
Возможные значения для этого свойства определяются в гендерном переумеществе. Они перечислены следующим образом:
  
|**Гендерное переумежение**|**Значение**|**Описание**|
|:-----|:-----|:-----|
|genderUnspecified  <br/> |0x0000  <br/> |Пол контакта не уточняется.  <br/> |
|genderFemale  <br/> |0x0001  <br/> |Контакт — женский.  <br/> |
|genderMale  <br/> |0x0002  <br/> |Контакт — мужской.  <br/> |
