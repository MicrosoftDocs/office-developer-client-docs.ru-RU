# <a name="pidtaghomeaddresspostofficebox-canonical-property"></a>Каноническое свойство PidTagHomeAddressPostOfficeBox

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит сведения о почтовый ящик для адреса пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_ADDRESS_POST_OFFICE_BOX, PR_HOME_ADDRESS_POST_OFFICE_BOX_A, PR_HOME_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Идентификатор:  <br/> |0x3A5E  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства определяются пользователем или организации пользователя.
