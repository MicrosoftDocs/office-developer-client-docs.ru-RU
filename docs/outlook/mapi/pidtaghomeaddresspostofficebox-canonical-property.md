# <a name="pidtaghomeaddresspostofficebox-canonical-property"></a>Каноническое свойство PidTagHomeAddressPostOfficeBox

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит сведения о почтовом ящике для адреса пользователя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ХОМЕ_АДДРЕСС_ПОСТ_ОФФИЦЕ_БОКС, ПР_ХОМЕ_АДДРЕСС_ПОСТ_ОФФИЦЕ_БОКС_А, ПР_ХОМЕ_АДДРЕСС_ПОСТ_ОФФИЦЕ_БОКС_В  <br/> |
|Идентификатор:  <br/> |0x3A5E  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
