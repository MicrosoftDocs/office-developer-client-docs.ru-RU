# <a name="updel"></a>UPDEL

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Сведения об удаленных в локальном хранилище элементов. Эти сведения используются во время состояния [отправки удаления.](upload-delete-status-state.md)
  
## <a name="quick-info"></a>Краткие сведения

```cpp
struct UPDEL 
{ 
    PUPDELE pupde; 
    UINT cEnt; 
};
```

## <a name="members"></a>Элементы

 _pupde_
  
>  [out] Вектор [записей UPDELE.](updele.md) 
    
 _cEnt_
  
> [out] Количество записей в *pupde.*
