# <a name="imapitablecreatebookmark"></a>IMAPITable::CreateBookmark

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Создает закладку в текущей позиции таблицы.
  
```cpp
HRESULT CreateBookmark(
BOOKMARK FAR * lpbkPosition
);
```

## <a name="parameters"></a>Параметры

 _Лпбкпоситион_
  
> вышли Указатель на возвращенное значение закладки 32/бит. Позже эту закладку можно передать при вызове метода [IMAPITable:: сикров](imapitable-seekrow.md) .

## <a name="remarks"></a>Комментарии

Метод **IMAPITable:: креатебукмарк** помечает положение таблицы, создавая значение, которое называется закладкой. С помощью закладки можно вернуться к положению, определенному закладкой. Позиция с закладками связана с объектом в этой строке таблицы. 
  
Закладки не поддерживаются в таблицах вложений, а реализации таблиц вложений **креатебукмарк** возвращают мапи_е_но_суппорт. 
  
## <a name="notes-to-implementers"></a>Примечания для исполнителей

Из-за затрат памяти на поддержание положения курсоров с помощью закладок Ограничьте число закладок, которые можно создать. При достижении этого номера возвращает МАПИ_Е_УНАБЛЕ_ТО_КОМПЛЕТЕ из всех последующих вызовов **креатебукмарк**.
  
Иногда закладка указывает на строку, которая больше не находится в представлении таблицы. Если абонент использует такую закладку, переместите курсор на следующую видимую строку и остановите ее. 
  
Когда вызывающий объект пытается использовать закладку, указывающую на невидимую строку, так как она была свернута, возвращайте МАПИ_В_ПОСИТИОН_ЧАНЖЕД после перемещения закладки. Вы можете поместить закладку в следующую видимую строку на данный момент или при свертывании в методе **сетколлапсестате** . Если вы перемещаете закладку на момент сворачивания строки, необходимо сохранить в закладе бит, указывающий на то, что была перемещена Закладка: с момента ее последнего использования или никогда не использовался с момента создания. 
  
## <a name="notes-to-callers"></a>Примечания для вызывающих методов

 **Креатебукмарк** выделяет память для создаваемой закладки. Освободите ресурсы для закладки, вызвав метод [IMAPITable:: фрибукмарк](imapitable-freebookmark.md) .
