<span data-ttu-id="da3ec-122">Клиенты должны использовать универсальное соглашение об именах (UNC) в большинстве случаев при файл является общей и следует использовать абсолютный путь, когда локальный файл.</span><span class="sxs-lookup"><span data-stu-id="da3ec-122">Clients should use a universal naming convention (UNC) in most cases when the file is shared, and should use an absolute path when the file is local.</span></span>
