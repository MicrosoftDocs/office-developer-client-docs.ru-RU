# <a name="pidtagoriginalsenderemailaddress-canonical-property"></a>Каноническое свойство PidTagOriginalSenderEmailAddress

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит адрес электронной почты отправитель первой версии сообщения, то есть сообщение перед отправкой или ответом на него.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINAL_SENDER_EMAIL_ADDRESS, PR_ORIGINAL_SENDER_EMAIL_ADDRESS_A, PR_ORIGINAL_SENDER_EMAIL_ADDRESS_W  <br/> |
|Идентификатор:  <br/> |0x0067  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства являются примерами свойств адресов для исходного отправитель сообщения. При первой отправке сообщения клиентская заявка должна установить это свойство к значению **PR_SENDER_EMAIL_ADDRESS** [(PidTagSenderEmailAddress).](pidtagsenderemailaddress-canonical-property.md) Он никогда не меняется, когда сообщение переададовыв или отвечает на него.
