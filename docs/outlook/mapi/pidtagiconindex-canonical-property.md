# <a name="pidtagiconindex-canonical-property"></a>Каноническое свойство PidTagIconIndex

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит число, которое указывает, какой значок следует использовать при отобраке группы объектов электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ICON_INDEX  <br/> |
|Идентификатор:  <br/> |0x1080  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания
