# <a name="pidtagiconindex-canonical-property"></a>Каноническое свойство PidTagIconIndex

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит номер, который указывает, какой значок следует использовать при отобраив группу объектов электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ICON_INDEX  <br/> |
|Идентификатор:  <br/> |0x1080  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство, если оно существует, является подсказкой для клиента. Клиент может игнорировать значение этого свойства.
