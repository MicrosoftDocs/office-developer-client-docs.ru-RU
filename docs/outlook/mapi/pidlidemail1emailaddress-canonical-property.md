# <a name="pidlidemail1emailaddress-canonical-property"></a>Каноническое свойство PidLidEmail1EmailAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает первый адрес электронной почты контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail1EmailAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008083  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть соответствующим типу адреса, указанному для этого адреса электронной почты.
