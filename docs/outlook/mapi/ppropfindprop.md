# <a name="ppropfindprop"></a>PpropFindProp

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Выполняет поиск указанного свойства в наборе свойств.
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Мапиутил. h  <br/> |
|Реализовано в:  <br/> |MAPI  <br/> |
|Вызывающая сторона:  <br/> |Клиентские приложения и поставщики услуг  <br/> |
   
```cpp
LPSPropValue PpropFindProp(
  LPSPropValue rgprop,
  ULONG cprop,
  ULONG ulPropTag
);
```

## <a name="parameters"></a>Параметры

 _ргпроп_
  
> возврата Массив структур [спропвалуе](spropvalue.md) , определяющих свойства для поиска. 
    
 _кпроп_
  
> возврата Количество свойств в наборе свойств, указанном с помощью параметра _ргпроп_ . 
    
 _Улпроптаг_
  
> возврата Тег Property для свойства, которое необходимо найти в наборе свойств, указанном с помощью параметра _ргпроп_ . 
    
## <a name="return-value"></a>Возвращаемое значение

 **Ппропфиндпроп** возвращает структуру [спропвалуе](spropvalue.md) , определяющую свойство, которое соответствует тегу входного свойства, или значение null, если совпадение отсутствует. 
  
## <a name="remarks"></a>Примечания

Если заданный тег свойства указывает свойство типа ПТ_УНСПЕЦИФИЕД, функция **ппропфиндпроп** находит соответствующее значение только для идентификатора свойства в теге. В противном случае обнаруживается сравнение для всего тега свойства, включая тип свойства, и возвращается указанное свойство.
