# <a name="mapi-forms"></a>Формы MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
После прочтения этого обзора архитектуры форм MAPI вы сможете получить представление о том, какие формы MAPI и как они взаимодействуют с другими компонентами подсистемы MAPI. Цель этого раздела — предоставить вам концептуальные знания, необходимые для реализации собственных серверов форм MAPI.
  
Перед чтением этого раздела необходимо ознакомиться с материалами в разделе [MapI Forms Overview.](mapi-forms-overview.md) 
  
> [!NOTE]
> Так как архитектура формы MAPI основана на компонентной объектной модели (COM), разработка приложений сервера форм требует знаний о программировании com. Дополнительные сведения о com см. в разделе COM и ActiveX object Services в Windows SDK.
