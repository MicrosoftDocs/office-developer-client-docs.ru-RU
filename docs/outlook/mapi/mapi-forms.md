# <a name="mapi-forms"></a>Формы MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
После прочтения этого обзора архитектуры форм MAPI вы сможете понять, что такое формы MAPI и как они взаимодействуют с другими компонентами подсистемы MAPI. Цель этого раздела — предоставить вам концептуальные знания, необходимые для реализации собственных серверов форм MAPI.
  
Прежде чем читать этот раздел, ознакомьтесь с материалами в разделе "Общие сведения о [формах MAPI".](mapi-forms-overview.md) 
  
> [!NOTE]
> Поскольку архитектура формы MAPI основана на модели com, для разработки серверных приложений форм требуется знание программирования COM. Дополнительные сведения о COM см. в разделе com и ActiveX Object Services в Windows SDK.
