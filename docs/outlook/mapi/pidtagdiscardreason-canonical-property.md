# <a name="pidtagdiscardreason-canonical-property"></a>Каноническое свойство PidTagDiscardReason

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит причину, по которой агент передачи сообщений (MTA) отклоил сообщение. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DISCARD_REASON  <br/> |
|Идентификатор:  <br/> |0x0011  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Причина, содержательная в этом свойстве, используется в отчете о ненастройстве для сообщения.
