# <a name="pidtagdiscardreason-canonical-property"></a>Каноническое свойство PidTagDiscardReason

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит причину, почему сообщение передачи (агента) отброшенного сообщения. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DISCARD_REASON  <br/> |
|Идентификатор:  <br/> |0x0011  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Сообщения причину, содержащихся в это свойство используется в отчете о недоставке.
