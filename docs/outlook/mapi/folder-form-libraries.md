# <a name="folder-form-libraries"></a>Библиотеки форм в папках

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
В некоторых случаях может потребоваться связать одну или несколько форм с указанной папки. Например сотрудников в организации может все иметь папку отчетов о ходе выполнения в хранилище личных сообщений для создания и хранения отчетов о ходе выполнения. Поскольку отчет о ходе выполнения определенных папку отчет о ходе выполнения каждого пользователя, он может быть неприемлема для хранения форма отчета о ходе выполнения в библиотеке форм широкий системы. Тем не менее копию форма отчета о ходе выполнения можно сохраняются в таблице связанное содержимое папки отчет о ходе выполнения каждого пользователя. Это запрещает пользователя с помощью форм отчет о ходе выполнения не из указанной папки.
  
Концептуально существует один библиотека форм папки для каждой папки в банке сообщений, даже в том случае, если нет формы серверов установлены в нем. Библиотеки форм папки реализуются как другие библиотеки форм, сохраняемые в виде таблицы связанного содержимого в альтернативных части папки. Так как библиотеки форм папки содержатся в папке, они копируются вместе с родительской папки в операции копирования.
