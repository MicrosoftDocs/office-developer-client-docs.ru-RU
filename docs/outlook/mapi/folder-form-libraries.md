# <a name="folder-form-libraries"></a>Библиотеки форм папок

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
В некоторых случаях может потребоваться связать одну или несколько форм с определенной папкой. Например, у сотрудников организации может быть папка Отчет о ходе выполнения в личном хранилище сообщений для создания и хранения отчетов о ходе работы. Поскольку отчет о ходе работы является специфическим для папки отчетов о ходе работы каждого пользователя, может оказаться не подходящим хранить форму отчета о ходе работы в библиотеке широкой формы системы. Однако копия формы отчета о ходе выполнения может храниться в таблице связанного контента папки отчетов о ходе работы каждого пользователя. Это ограничивает использование пользователями форм отчетов о ходе выполнения за пределами указанной папки.
  
Концептуально существует одна библиотека форм папок для каждой папки в хранилище сообщений, даже если в ней не установлены серверы форм. Библиотеки форм папок реализуются, как и другие библиотеки форм, они хранятся в качестве таблиц со связанным содержимым в альтернативной части папки. Так как библиотеки форм папок содержатся в папке, они копируются вместе с родительской папкой в операциях копирования.
