# <a name="folder-form-libraries"></a>Библиотеки форм папок

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
В некоторых случаях может потребоваться связать одну или несколько форм с определенной папкой. Например, сотрудники Организации могут иметь папку отчетов о ходе выполнения в своем личном хранилище сообщений для создания и хранения отчетов о ходе выполнения. Так как отчет о ходе выполнения относится к папке отчетов о ходе выполнения каждого пользователя, может оказаться нецелесообразным хранить форму отчета о ходе выполнения в системной библиотеке форм. Тем не менее копия формы отчета о ходе выполнения может храниться в таблице сопоставлений содержимого папки отчетов о ходе выполнения каждого пользователя. Он запрещает пользователю использовать формы отчетов о ходе выполнения вне указанной папки.
  
Концептуально, существует одна библиотека форм папки для каждой папки в хранилище сообщений, даже если в ней не установлены серверы форм. Библиотеки форм папки реализуются так же, как и другие библиотеки форм — они хранятся в виде таблиц с содержимым в альтернативной части папки. Так как библиотеки форм папки хранятся в папке, они копируются вместе с родительской папкой в операциях копирования.
