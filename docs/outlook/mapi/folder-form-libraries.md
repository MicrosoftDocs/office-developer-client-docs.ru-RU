# <a name="folder-form-libraries"></a>Библиотеки форм папок

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
В некоторых случаях может потребоваться связать одну или несколько форм с определенной папкой. Например, у сотрудников организации может быть папка "Отчет о ходе выполнения" в хранилище личных сообщений для создания и хранения отчетов о ходе выполнения. Поскольку отчет о ходе выполнения является специфическим для папки отчета о ходе выполнения каждого пользователя, может оказаться не подходящим хранить форму отчета о ходе выполнения в библиотеке форм на всей системе. Однако копию формы отчета о ходе выполнения можно сохранить в связанной таблице содержимого папки отчета о ходе выполнения каждого пользователя. Это ограничивает использование пользователем форм отчетов о ходе выполнения за пределами указанной папки.
  
Концептуально существует одна библиотека форм папки для каждой папки в хранилище сообщений, даже если в ней не установлены серверы форм. Библиотеки форм папок реализуются так же, как и другие библиотеки форм— они хранятся в виде связанных таблиц содержимого в альтернативной части папки. Так как библиотеки форм папок содержатся в папке, они копируется вместе с родительской папкой в операциях копирования.
