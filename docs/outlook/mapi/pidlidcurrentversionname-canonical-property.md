# <a name="pidlidcurrentversionname-canonical-property"></a>Каноническое свойство PidLidCurrentVersionName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит строку, указывающую основной и дополнительный номер версии приложения Outlook для элемента.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидкуррентверсионнаме  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008554  <br/> |
|Тип данных:  <br/> |Подлежит уточнению.  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
