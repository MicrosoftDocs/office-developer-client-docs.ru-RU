# <a name="pidlidemail2addresstype-canonical-property"></a>Каноническое свойство PidLidEmail2AddressType

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает тип адреса второго адреса электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail2AddrType  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008092  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Если задано значение, значение этого свойства должно быть допустимым типом адреса.
