# <a name="pidtagreadreceiptsearchkey-canonical-property"></a>Каноническое свойство PidTagReadReceiptSearchKey

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит ключ поиска для пользователя обмена сообщениями, которому система обмена сообщениями должна направить отчет о прочтении сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РЕАД_РЕЦЕИПТ_СЕАРЧ_КЭЙ  <br/> |
|Идентификатор:  <br/> |0x0053  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство игнорируется, если для свойства **пр_реад_рецеипт_рекуестед** ([PidTagReadReceiptRequested](pidtagreadreceiptrequested-canonical-property.md)) не задано значение true.
  
Если клиентское приложение хочет получать отчеты о прочтении, оно может оставить это свойство неопределенным или задать его в качестве ключа поиска, который хранится в свойстве **пр_сендер_сеарч_кэй** ([PidTagSenderSearchKey](pidtagsendersearchkey-canonical-property.md)) во время отправки сообщения.
