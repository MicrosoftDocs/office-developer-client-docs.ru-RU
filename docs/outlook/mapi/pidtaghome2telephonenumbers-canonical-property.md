# <a name="pidtaghome2telephonenumbers-canonical-property"></a>Каноническое свойство PidTagHome2TelephoneNumbers

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит список дополнительных телефонных номеров на домашней странице получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME2_TELEPHONE_NUMBER_A_MV  <br/> |
|Идентификатор:  <br/> |0x3A2F  <br/> |
|Тип данных:  <br/> |PT_MV_UNICODE8  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
