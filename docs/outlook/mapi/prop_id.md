# <a name="prop_id"></a>PROP_ID

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Возвращает идентификатор свойства указанного тега свойства.
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |[SPropValue](spropvalue.md) <br/> |
   
```cpp
PROP_ID (ulPropTag)
```

## <a name="parameters"></a>Parameters

 _ulPropTag_
  
> Тег свойства, содержащий возвращаемый идентификатор.
    
## <a name="remarks"></a>Примечания

Каждый тег свойства содержит тип свойства в слове низкого порядка (биты от 0 до 15) и идентификатор свойства в слове высокого порядка (биты от 16 до 31). Макрос **PROP_ID** извлекает идентификатор свойства и помещает его в биты от 0 до 15 возвращаемого integer. Остальные биты возвращаемого значения устанавливаются до нуля. 
  
Макрос **PROP_ID** можно использовать, например, для получения идентификатора, чтобы передать [IMAPIProp::GetNamesFromIDs](imapiprop-getnamesfromids.md). **GetNamesFromIDs** извлекает имя свойства, связанное с идентификатором имени свойства.
