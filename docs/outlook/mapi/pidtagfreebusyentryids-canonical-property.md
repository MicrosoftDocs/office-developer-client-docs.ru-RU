# <a name="pidtagfreebusyentryids-canonical-property"></a>Каноническое свойство PidTagFreeBusyEntryIds

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит коды **entryID** для сообщения о делегате, сведения о занятости во время входа пользователя и папку, чей PR_DISPLAY_NAME **(** [PidTagDisplayName)](pidtagdisplayname-canonical-property.md)равен "FreeBusy Data".
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_FREEBUSY_ENTRYIDS  <br/> |
|Идентификатор:  <br/> |0x36E4  <br/> |
|Тип данных:  <br/> |PT_MV_BINARY  <br/> |
|Область:  <br/> |Контейнер MAPI  <br/> |
