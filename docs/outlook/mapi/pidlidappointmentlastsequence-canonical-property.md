# <a name="pidlidappointmentlastsequence-canonical-property"></a>Каноническое свойство PidLidAppointmentLastSequence

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает организатору последний номер последовательности, отправленный любому участнику.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptLastSequence  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008203  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство не имеет значения для участника.
