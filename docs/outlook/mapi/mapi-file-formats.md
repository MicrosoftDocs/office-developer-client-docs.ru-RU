# <a name="mapi-file-formats"></a>Форматы файлов MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
MAPI публикует форматы различных файлов конфигурации, используемых клиентами и поставщиками услуг для описания служб сообщений и их поставщиков услуг и форм.
  
В следующей таблице приведено описание этих файлов по имени или типу:
  
|**Файл конфигурации**|**Описание**|
|:-----|:-----|
|MapiSvc. INF  <br/> |Содержит сведения о конфигурации для подсистем MAPI, служб сообщений и поставщиков услуг.  <br/> |
|Файлы с расширением. cfg  <br/> |Содержит сведения о конфигурации для формы.  <br/> |
   
Подробную информацию о формате файла MapiSvc. inf можно узнать в разделе [Формат файла MapiSvc. INF](file-format-of-mapisvc-inf.md). 
  
Подробную информацию о формате файла конфигурации формы можно узнать в разделе Формат файлов [конфигурации форм](file-format-of-form-configuration-files.md).
