# <a name="mapi-file-formats"></a>Форматы файлов MAPI

  
  
**Относится к**: Outlook 
  
MAPI публикует форматы различные файлы конфигурации, используемые клиентов и поставщиков услуг для описания службы сообщений и их поставщиков услуг и форм.
  
В следующей таблице описываются эти файлы по имени или типа:
  
|**Файл конфигурации**|**Описание**|
|:-----|:-----|
|MapiSvc.inf  <br/> |Содержит сведения о конфигурации для подсистемы MAPI, службы сообщений и поставщиков услуг.  <br/> |
|Файлы с расширением .cfg  <br/> |Содержит сведения о конфигурации для формы.  <br/> |
   
Подробные сведения о формате файла MapiSvc.inf см в [Файл формата MapiSvc.inf](file-format-of-mapisvc-inf.md). 
  
Подробные сведения о формате файла файл конфигурации формы см [файл формата из формы](file-format-of-form-configuration-files.md).
