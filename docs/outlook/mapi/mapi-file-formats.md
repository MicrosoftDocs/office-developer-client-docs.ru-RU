# <a name="mapi-file-formats"></a>Форматы файлов MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
MAPI публикует форматы различных файлов конфигурации, которые клиенты и поставщики услуг используют для описания служб сообщений и их поставщиков и форм.
  
В следующей таблице описываются эти файлы по имени или типу:
  
|**Файл конфигурации**|**Описание**|
|:-----|:-----|
|MapiSvc.inf  <br/> |Содержит сведения о конфигурации подсистемы MAPI, служб сообщений и поставщиков услуг.  <br/> |
|Файлы с расширением .cfg  <br/> |Содержит сведения о конфигурации для формы.  <br/> |
   
Подробные сведения о формате файлов MapiSvc.inf см. в файловом формате [MapiSvc.inf.](file-format-of-mapisvc-inf.md) 
  
Подробные сведения о формате файла файла конфигурации формы см. в материалах [File Format of Form Configuration Files.](file-format-of-form-configuration-files.md)
