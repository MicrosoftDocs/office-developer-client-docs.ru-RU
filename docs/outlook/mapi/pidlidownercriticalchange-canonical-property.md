# <a name="pidlidownercriticalchange-canonical-property"></a>Каноническое свойство PidLidOwnerCriticalChange

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату и время отправки приглашения на собрание организатором.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ЛИД_ОВНЕР_КРИТИКАЛ_ЧАНЖЕ  <br/> |
|Набор свойств:  <br/> |Псетид_митинг  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000001A  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Примечания

Значение должно быть указано в формате UTC.
