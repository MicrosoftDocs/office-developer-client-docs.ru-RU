# <a name="pidlidownercriticalchange-canonical-property"></a>Каноническое свойство PidLidOwnerCriticalChange

  
  
**Относится к**: Outlook 
  
Указывает дату и время отправки приглашения на собрание, организатор.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |LID_OWNER_CRITICAL_CHANGE  <br/> |
|Набор свойств:  <br/> |PSETID_Meeting  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000001A  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Meetings (собрания);  <br/> |
   
## <a name="remarks"></a>Замечания

Значением должен быть указан в формате UTC.
