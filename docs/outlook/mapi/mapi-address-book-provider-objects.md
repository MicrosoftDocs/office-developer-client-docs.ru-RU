# <a name="mapi-address-book-provider-objects"></a>Объекты поставщика адресной книги MAPI
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Помимо стандартных объектов поставщиков и объектов для логотипа, поставщики адресных книг реализуют контейнеры адресных книг, списки рассылки, пользователей сообщений, таблицы, объекты состояния и элементы управления.
  
На следующем рисунке показаны эти объекты, соответствующие им интерфейсы и компоненты MAPI, которые их используют.
