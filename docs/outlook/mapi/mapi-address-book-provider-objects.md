# <a name="mapi-address-book-provider-objects"></a>Объекты поставщика адресных книг MAPI
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Помимо стандартных объектов поставщика и логотипов, поставщики адресных книг реализуют контейнеры адресной книги, списки рассылки, пользователей сообщений, таблицы, объекты состояния и элементы управления.
  
На следующей иллюстрации показаны эти объекты, соответствующие им интерфейсы и компоненты MAPI, которые их используют.
  
![Объекты, обращаясь к поставщикам книг, реализуют](media/amapi_64.gif "объекты, реализуемые поставщиками адресных книг.")
