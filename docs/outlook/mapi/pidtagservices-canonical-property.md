# <a name="pidtagservices-canonical-property"></a>Каноническое свойство PidTagServices

  
  
**Относится к**: Outlook 
  
Содержит список идентификаторов для службы сообщений текущего профиля.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SERVICES  <br/> |
|Идентификатор:  <br/> |0x3D0E  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Профиль MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство не используется. Зарезервирован для использования с MAPI.
