# <a name="pidtagservices-canonical-property"></a>Каноническое свойство PidTagServices

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит список идентификаторов служб сообщений в текущем профиле.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SERVICES  <br/> |
|Идентификатор:  <br/> |0x3D0E  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Профиль MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Не используйте это свойство. Оно зарезервировано для использования MAPI.
