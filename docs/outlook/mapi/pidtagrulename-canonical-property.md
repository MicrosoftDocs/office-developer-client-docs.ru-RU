# <a name="pidtagrulename-canonical-property"></a>Каноническое свойство PidTagRuleName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает имя правила, читаемое пользователем.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РУЛЕ_НАМЕ  <br/> |
|Идентификатор:  <br/> |0x6682  <br/> |
|Тип данных:  <br/> |PT_STRING8  <br/> |
|Область:  <br/> |Правила на стороне сервера  <br/> |
