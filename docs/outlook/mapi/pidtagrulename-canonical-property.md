# <a name="pidtagrulename-canonical-property"></a>Каноническое свойство PidTagRuleName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает понятное для пользователя имя правила.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RULE_NAME  <br/> |
|Идентификатор:  <br/> |0x6682  <br/> |
|Тип данных:  <br/> |PT_STRING8  <br/> |
|Область:  <br/> |Правила со стороны сервера  <br/> |
