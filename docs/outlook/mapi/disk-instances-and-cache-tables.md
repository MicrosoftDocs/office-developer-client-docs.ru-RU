# <a name="disk-instances-and-cache-tables"></a>Экземпляры дисков и таблицы кэша

**Применимо к**: Outlook 2013 | Outlook 2016 
  
Чтобы активировать формы, его исполняемых файлов должен быть доступен на компьютере пользователя. Если не отображаются, их необходимо скопировать из библиотеки форм на локальный диск. Для этого диспетчер форм по умолчанию создает подкаталог в каталоге пользователя Windows для содержат исполняемые файлы формы (. Файлы exe. HLPs). Этот каталог называется диска экземпляр формы.
  
Диспетчер форм по умолчанию ведет таблицу все экземпляры диска, чтобы если экземпляр диска его можно использовать без необходимости для копирования файлов из библиотеки форм диск компьютера пользователя. Таблица диска экземпляры управляемых как минимальных часто используемых кэша. При необходимости новый экземпляр диска, она копируется на компьютере пользователя, заменив экземпляра минимальных часто используемых диска. В таблице кэша диска экземпляр обновляется в соответствии с последними конфигурации. Размер дисковый кэш это пользовательских, что позволит пользователям для балансировки скорость объемом свободного места.
  
В дополнение к экземпляру дисковый кэш диспетчер форм по умолчанию сохраняет работает экземпляр таблица, перечисляющая всех запущенных экземпляров серверов формы на компьютере пользователя. Возможность MAPI это используется для хранения экземпляров простоя форма в состоянии невидимой до форму, форма активируется класса сообщений сервера. Другими словами серверы формы можно кэширования в оперативной памяти для сведения к минимуму число попыток исполняемый файл формы должен находится в библиотеке форм и загружается в память с диска или по сети. Как экземпляр дисковый кэш работает экземпляр кэша работает в режиме минимальных часто используемых таким образом, выполняемым экземпляром формы могут быть удалены из кэша, чтобы освободить место для другого экземпляра формы. В этом кэше выполняется поиск выполняемый экземпляр сервера форм библиотеки форм для поиска для сервера формы.
  
> [!NOTE]
> Диспетчер форм по умолчанию отображает индикатор выполнения при установке формы на рабочей станции пользователя, позволяя пользователю отменить операцию. Это особенно полезна, если для подключения пользователей исполняемый файл сервера формы является низкой пропускной способности сети.
