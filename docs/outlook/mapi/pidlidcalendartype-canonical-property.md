# <a name="pidlidcalendartype-canonical-property"></a>Каноническое свойство PidLidCalendarType

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает значение поля Календартипе из свойства **диспидапптрекур** ([PidLidAppointmentRecur](pidlidappointmentrecur-canonical-property.md)).
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ЛИД_КАЛЕНДАР_ТИПЕ  <br/> |
|Набор свойств:  <br/> |Псетид_митинг  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000001C  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Комментарии

Если приглашение на собрание представляет повторяющийся ряд или исключение, это значение поля Календартипе из свойства **диспидапптрекур** . В противном случае это свойство не задается и считается равным 0.
