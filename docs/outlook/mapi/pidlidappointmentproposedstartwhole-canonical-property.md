# <a name="pidlidappointmentproposedstartwhole-canonical-property"></a>Каноническое свойство PidLidAppointmentProposedStartWhole

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает предлагаемое значение для **диспидапптстартвхоле** ([PidLidAppointmentStartWhole](pidlidappointmentstartwhole-canonical-property.md)) для предложения Counter.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидапптпропоседстартвхоле  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008250  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Это значение должно быть указано в формате UTC.
