# <a name="pidlidappointmentproposedstartwhole-canonical-property"></a>Каноническое свойство PidLidAppointmentProposedStartWhole

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает предлагаемое значение **для dispidApptStartWhole** [(PidLidAppointmentStartWhole)](pidlidappointmentstartwhole-canonical-property.md)для встречного предложения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptProposedStartWhole  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008250  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Это значение должно быть указано в координируется универсальное время (UTC).
