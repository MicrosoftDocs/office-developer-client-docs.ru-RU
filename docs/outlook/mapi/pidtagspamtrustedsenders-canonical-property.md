# <a name="pidtagspamtrustedsenders-canonical-property"></a>Каноническое свойство PidTagSpamTrustedSenders

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит список адресов электронной почты и доменов, в которые входят надежные отправители.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x0418  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Спам  <br/> |
