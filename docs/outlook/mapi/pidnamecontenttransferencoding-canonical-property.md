# <a name="pidnamecontenttransferencoding-canonical-property"></a>Каноническое свойство PidNameContentTransferEncoding

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение поля поля заглавного поля контента-transfer-Encoding [RFC3282].
  
|||
|:-----|:-----|
|Дружественные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Имя свойства:  <br/> |Content-Transfer-Encoding  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Электронная почта  <br/> |
