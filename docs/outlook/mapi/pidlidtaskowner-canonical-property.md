# <a name="pidlidtaskowner-canonical-property"></a>Каноническое свойство PidLidTaskOwner

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя владельца задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидтасковнер  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000811F  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Задача  <br/> |
