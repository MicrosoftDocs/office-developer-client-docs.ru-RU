# <a name="pidtagtransportkey-canonical-property"></a>Каноническое свойство PidTagTransportKey

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит значение, используемого пулом MAPI для отслеживания хода выполнения исходящие сообщения через поставщиков исходящие транспортные службы.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_TRANSPORT_KEY  <br/> |
|Идентификатор:  <br/> |0x0E16  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |MAPI, не передаваемый  <br/> |
   
## <a name="remarks"></a>Примечания

Не используйте это свойство. Он зарезервирован для использования в MAPI.
