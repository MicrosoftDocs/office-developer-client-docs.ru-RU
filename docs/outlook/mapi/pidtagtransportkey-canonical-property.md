# <a name="pidtagtransportkey-canonical-property"></a>Каноническое свойство PidTagTransportKey

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение, используемое диспетчером очереди MAPI для отслеживания хода исходящего сообщения через поставщиков транспорта исходящей почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ТРАНСПОРТ_КЭЙ  <br/> |
|Идентификатор:  <br/> |0x0E16  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Несъемный MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Не используйте это свойство. Оно зарезервировано для использования MAPI.
