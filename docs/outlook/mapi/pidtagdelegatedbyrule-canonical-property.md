# <a name="pidtagdelegatedbyrule-canonical-property"></a>Каноническое свойство PidTagDelegatedByRule

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, делегировано ли сообщение папки правилом.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DELEGATED_BY_RULE  <br/> |
|Идентификатор:  <br/> |0x3FE3  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Состояние MAPI  <br/> |
