# <a name="pidtagprimaryfaxnumber-canonical-property"></a>Каноническое свойство PidTagPrimaryFaxNumber

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит номер телефона основного факсимильного аппарата получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PRIMARY_FAX_NUMBER, PR_PRIMARY_FAX_NUMBER_A PR_PRIMARY_FAX_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A23  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения о идентификации и доступе для получателя. Они определяются получателем и их организацией.
