# <a name="iprofadmindeleteprofile"></a>IProfAdmin::DeleteProfile

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Удаляет профиль.
  
```cpp
HRESULT DeleteProfile(
  LPSTR lpszProfileName,
  ULONG ulFlags
);
```

## <a name="parameters"></a>Параметры

 _Лпсзпрофиленаме_
  
> возврата Указатель на имя профиля, который требуется удалить.
    
 _ulFlags_
  
> возврата Всегда имеет значение NULL. 
    
## <a name="return-value"></a>Возвращаемое значение

S_OK 
  
> Профиль успешно удален.
    
МАПИ_Е_НОТ_ФАУНД 
  
> Указанный профиль не существует.
    
## <a name="remarks"></a>Примечания

Метод **ипрофадмин::D елетепрофиле** удаляет профиль. Если профиль, который требуется удалить, используется при вызове **делетепрофиле** , **ДЕЛЕТЕПРОФИЛЕ** возвращает значение S_OK, но не удаляет профиль немедленно. Вместо этого **делетепрофиле** помечает профиль для удаления и удаляет его после того, как он больше не будет использоваться после завершения всех активных сеансов. 
  
Функция точки входа для каждой службы сообщений в профиле вызывается со значением МСГ_СЕРВИЦЕ_ДЕЛЕТЕ, установленным в параметре _улконтекст_ . Сначала функция удаляет службу, а затем удаляет раздел профиля службы. Функция точки входа службы сообщений не вызывается повторно после удаления службы. 
  
Для удаления профиля не требуется указывать пароль.
