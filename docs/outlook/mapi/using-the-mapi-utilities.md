# <a name="using-the-mapi-utilities"></a>Использование служебных программ MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Служебные программы MAPI состоят из объектов данных и данных свойств таблиц и различных функций для поддержки различных функций. Клиенту может потребоваться использовать только эти служебные программы и не нужно выполнять вход в подсистему MAPI для установления подключения к поставщикам услуг. Если ваш клиент соответствует этой категории, вызовите функцию API [сЦинитмапиутил](scinitmapiutil.md) , а не функцию [мапиинитиализе](mapiinitialize.md) во время инициализации. 
  
 **СЦинитмапиутил** позволяет клиентам использовать служебные функции, требующие распределителя MAPI, но не запрашивают явный механизм распределителяй. Когда закрывается время, вызовите [деинитмапиутил](deinitmapiutil.md) для освобождения ресурсов, а не [мапиунинитиализе](mapiuninitialize.md). Клиенты, которые никогда не вызывают **мапиинитиализе** , не должны вызывать **мапиунинитиализе**.
  
Если вы вызвали **сЦинитмапиутил** , а не **мапиинитиализе** и используете таблицы с помощью **** методов, а не через методы **IMAPITable** , имейте в виду, что уведомления таблицы не будут работать. Для уведомлений требуется использование библиотек MAPI и [IMAPITable: IUnknown](imapitableiunknown.md).
