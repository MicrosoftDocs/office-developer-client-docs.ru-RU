# <a name="olflagicon"></a>OlFlagIcon

  
  
**Относится к**: Outlook 
  
Указывает ли элемент имеет флаг, и если он установлен, цвет, отмеченного элемента.
  
## <a name="quick-info"></a>Краткие сведения

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>Members

 _olNoFlagIcon_
  
> Флаг не связанный с элементом.
    
 _olPurpleFlagIcon_
  
> Фиолетовый была связана с отмеченного элемента.
    
 _olOrangeFlagIcon_
  
> Оранжевый была связана с отмеченного элемента.
    
 _olGreenFlagIcon_
  
> Зеленый была связана с отмеченного элемента.
    
 _olYellowFlagIcon_
  
> Желтый была связана с отмеченного элемента.
    
 _olBlueFlagIcon_
  
> Синий была связана с отмеченного элемента.
    
 _olRedFlagIcon_
  
> Красный была связана с отмеченного элемента.
