# <a name="olflagicon"></a>OlFlagIcon

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, помечен ли элемент, а если это так, то есть цвет, связанный с отмеченным элементом.
  
## <a name="quick-info"></a>Краткие сведения

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>Элементы

 _Олнофлагикон_
  
> С элементом не связан ни один флаг.
    
 _Олпурплефлагикон_
  
> С отмеченным элементом связан фиолетовый.
    
 _Олоранжефлагикон_
  
> Оранжевый цвет связан с отмеченным элементом.
    
 _Олгринфлагикон_
  
> Зеленый цвет связан с отмеченным элементом.
    
 _Олелловфлагикон_
  
> С отмеченным элементом связан желтый цвет.
    
 _Олблуефлагикон_
  
> Синий цвет связан с отмеченным элементом.
    
 _Олредфлагикон_
  
> Красный цвет связан с отмеченным элементом.
