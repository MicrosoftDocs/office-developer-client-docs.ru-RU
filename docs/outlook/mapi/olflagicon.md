# <a name="olflagicon"></a>OlFlagIcon

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, помечен ли элемент и имеет ли он цвет, связанный с помеченным элементом.
  
## <a name="quick-info"></a>Краткие сведения

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>Элементы

 _olNoFlagIcon_
  
> Флаг не был связан с элементом.
    
 _olPurpleFlagIcon_
  
> Фиолетовый цвет связан с помеченным элементом.
    
 _olOrangeFlagIcon_
  
> Оранжевый был связан с помеченным элементом.
    
 _olGreenFlagIcon_
  
> Зеленый цвет связан с помеченным элементом.
    
 _olYellowFlagIcon_
  
> Желтый цвет связан с помеченным элементом.
    
 _olBlueFlagIcon_
  
> Синий цвет связан с помеченным элементом.
    
 _olRedFlagIcon_
  
> Красный цвет связан с помеченным элементом.
