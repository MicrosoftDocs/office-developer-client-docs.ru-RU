# <a name="pidtaghomeaddressstreet-canonical-property"></a>Каноническое свойство PidTagHomeAddressStreet

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит часть адреса пользователя, на которой размещен почтовый ящик.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_HOME_ADDRESS_STREET, PR_HOME_ADDRESS_STREET_A PR_HOME_ADDRESS_STREET_W  <br/> |
|Идентификатор:  <br/> |0x3A5D  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
