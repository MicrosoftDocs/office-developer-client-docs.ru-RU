# <a name="pidtagspamthreshold-canonical-property"></a>Каноническое свойство PidTagSpamThreshold

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Длинное значение, указывающее уровень фильтрации нежелательной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SPAM_THRESHOLD  <br/> |
|Длинный идентификатор (крышка):  <br/> | 0x041B  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Нежелательная почта  <br/> |
   
## <a name="values"></a>Values

Ниже приведены значения для фильтрации нежелательной почты.
  
|**Уровень нежелательной почты**|**Значение**|
|:-----|:-----|
|Нет  <br/> |Равен  <br/> |
|Низкий  <br/> |0x00000006  <br/> |
|Средние  <br/> |0x00000005  <br/> |
|Высокий  <br/> |0x00000003  <br/> |
