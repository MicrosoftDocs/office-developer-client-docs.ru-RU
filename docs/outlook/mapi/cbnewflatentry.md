# <a name="cbnewflatentry"></a>CbNewFLATENTRY

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество байтов, которое необходимо выделить для новой структуры [флатентри](flatentry.md) , которая содержит идентификатор записи указанного размера в байтах. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
|Связанная структура:  <br/> |**FLATENTRY** <br/> |
   
```cpp
CbNewFLATENTRY (_cb)
```

## <a name="parameters"></a>Параметры

 __CB_
  
> Количество байтов в идентификаторе записи, включаемых в новую структуру **флатентри** .
