# <a name="cbnewflatentry"></a>CbNewFLATENTRY

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество bytes, которые должны быть выделены для новой структуры [FLATENTRY,](flatentry.md) которая содержит идентификатор входа определенного размера byte. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |**FLATENTRY** <br/> |
   
```cpp
CbNewFLATENTRY (_cb)
```

## <a name="parameters"></a>Parameters

 _ _cb_
  
> Количество bytes в идентификаторе записи, который будет включен в новую структуру **FLATENTRY.**
