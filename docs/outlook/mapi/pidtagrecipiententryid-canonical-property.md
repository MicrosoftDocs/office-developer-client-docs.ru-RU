# <a name="pidtagrecipiententryid-canonical-property"></a>Каноническое свойство PidTagRecipientEntryId

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит **идентификатор записи** получателя 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RECIPIENT_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x5FF7  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Идентификатор свойства  <br/> |
