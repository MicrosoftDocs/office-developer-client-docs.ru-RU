# <a name="pidlidfax1entryid-canonical-property"></a>Каноническое свойство PidLidFax1EntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает вводный адрес бизнес-факса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax1EntryID  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080B1  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Contact  <br/> |
