# <a name="pidtagreceivedbysearchkey-canonical-property"></a>Каноническое свойство PidTagReceivedBySearchKey

  
  
**Относится к**: Outlook 
  
Содержит ключ поиска сообщений пользователя, который получает сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RECEIVED_BY_SEARCH_KEY  <br/> |
|Идентификатор:  <br/> |0x0051  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство является одним из свойств адреса для обмена мгновенными сообщениями пользователя, который получает сообщение. Он должен иметь значение входящих поставщика транспорта.
