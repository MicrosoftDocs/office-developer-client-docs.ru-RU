# <a name="cbnewmtsid"></a>CbNewMTSID

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество байтов, которое следует выделить для новой структуры [мтсид](mtsid.md) с идентификатором агента передачи сообщений указанного размера. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
|Связанная структура:  <br/> |**MTSID** <br/> |
   
```cpp
CbNewMTSID (_cb)
```

## <a name="parameters"></a>Параметры

 __CB_
  
> Число байтов для идентификатора агента передачи сообщений, включаемого в новую структуру **мтсид** .
