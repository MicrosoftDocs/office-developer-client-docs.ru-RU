# <a name="cbnewmtsid"></a>CbNewMTSID

  
  
**Относится к**: Outlook 
  
Вычисляет число байтов, должен быть размещен для новой структуры [MTSID](mtsid.md) с идентификатором агента передачи сообщений из указанного размера. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанные структуры:  <br/> |**MTSID** <br/> |
   
```cpp
CbNewMTSID (_cb)
```

## <a name="parameters"></a>Параметры

 __Сертификация_
  
> Число байт для идентификатора агента передачи сообщений должны быть включены в новой структуры **MTSID** .
