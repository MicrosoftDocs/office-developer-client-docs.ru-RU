# <a name="cbnewmtsid"></a>CbNewMTSID

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Вычисляет количествобайтов, которые должны быть выделены для новой структуры [MTSID](mtsid.md) с идентификатором агента передачи сообщений указанного размера. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |**MTSID** <br/> |
   
```cpp
CbNewMTSID (_cb)
```

## <a name="parameters"></a>Параметры

 __cb_
  
> Количество включаемого в новую структуру **MTSID** идентификатора агента передачи сообщений.
