# <a name="pidlidfax3originaldisplayname-canonical-property"></a>Каноническое свойство PidLidFax3OriginalDisplayName

  
  
**Относится к**: Outlook 
  
Указывает, что исходное отображаемое имя контакта в другой адрес факса.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax3OriginalDisplayName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x000080D4  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство, если этот параметр указан, должен иметь значение совпадает со значением свойства **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).
