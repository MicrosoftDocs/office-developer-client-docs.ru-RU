# <a name="pidlidfax3originaldisplayname-canonical-property"></a>Каноническое свойство PidLidFax3OriginalDisplayName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает исходное отображаемого имени другого факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax3OriginalDisplayName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x000080D4  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

При его значении этому свойству необходимо установить то же **значение, что** и для свойства PR_NORMALIZED_SUBJECT ([PidTagNormalizedSubject).](pidtagnormalizedsubject-canonical-property.md)
