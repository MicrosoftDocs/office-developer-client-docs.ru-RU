# <a name="pidnameaudionotes-canonical-property"></a>Каноническое свойство PidNameAudioNotes

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Указывает текстовые примечания, подключенные к голосовое сообщение.
  
|||
|:-----|:-----|
|Понятные имена:  <br/> |UMAudioNotes  <br/> |
|Набор свойств:  <br/> |PSETID_UnifiedMessaging  <br/> |
|Имя свойства:  <br/> |UMAudioNotes  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Единая система обмена сообщениями  <br/> |
   
## <a name="remarks"></a>Замечания

Чтобы включить пользователь может читать и изменять звуковых заметок непосредственно на голосовое сообщение, клиент обеспечивает текстовое поле, где пользователь может ввести набор примечания, которые добавлены с этим свойством объект сообщения голосовой связи.
