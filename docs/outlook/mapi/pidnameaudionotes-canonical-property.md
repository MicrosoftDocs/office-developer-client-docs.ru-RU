# <a name="pidnameaudionotes-canonical-property"></a>Каноническое свойство PidNameAudioNotes

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает текстовые заметки, вложенные в голосовое сообщение.
  
|||
|:-----|:-----|
|Понятные имена:  <br/> |Умаудионотес  <br/> |
|Набор свойств:  <br/> |Псетид_унифиедмессагинг  <br/> |
|Имя свойства:  <br/> |Умаудионотес  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Единая система обмена сообщениями  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы разрешить конечному пользователю считывать и редактировать звуковые заметки непосредственно в голосовом сообщении, клиент предоставляет поле редактирования, в котором пользователь может ввести набор заметок, добавляемых к этому свойству объекта голосового сообщения.
