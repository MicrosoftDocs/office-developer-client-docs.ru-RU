# <a name="downloading-the-outlook-mapi-samples"></a>Загрузка Outlook MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Примеры mapi Outlook Майкрософт включают поставщика адресной книги, поставщика магазина сообщений и поставщика транспорта.
  
 **Загрузка поставщика адресной книги примера**
  
1. Чтобы скачать Outlook MAPI, щелкните  вкладку Home на странице API API Outlook обмена сообщениями [(MAPI).](https://ol2010mapisamples.codeplex.com/) 
    
2. На **вкладке Главная** нажмите кнопку Нажмите здесь, чтобы скачать источник для Outlook **2010 MAPI Samples**.
    
3. Ознакомьтесь с лицензионным соглашением и, если вы согласны, нажмите **кнопку Я согласен**.
    
4. В **диалоговом окне Загрузка** файлов нажмите кнопку **Сохранить**. В **диалоговом** окне Сохранить как диалоговое окно найдите папку, в которой необходимо сохранить исходные файлы, а затем нажмите **кнопку Сохранить**.
    
5. В **диалоговом окне Загрузка полный** нажмите **кнопку Закрыть,** чтобы закрыть диалоговое окно.
