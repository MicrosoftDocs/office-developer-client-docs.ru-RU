# <a name="downloading-the-outlook-mapi-samples"></a>Загрузка примеров Outlook MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Примеры Microsoft Outlook MAPI включают поставщика адресных книг, поставщика хранилища сообщений и поставщика транспорта.
  
 **Загрузка образца поставщика адресных книг**
  
1. Чтобы скачать примеры Outlook MAPI, перейдите на вкладку **Главная** на странице [примеры кода для API обмена сообщениями Outlook (MAPI)](https://ol2010mapisamples.codeplex.com/) . 
    
2. На вкладке **Главная** щелкните **ссылку Щелкните здесь, чтобы скачать источник для образцов MAPI Outlook 2010**.
    
3. ПроЧтите лицензионное соглашение и, если вы согласны, **** нажмите кнопку Принимаю.
    
4. В диалоговом окне " **Загрузка файла** " нажмите кнопку **сохранить**. В диалоговом окне **Сохранить как** выберите папку, в которой нужно сохранить исходные файлы, а затем нажмите кнопку **сохранить**.
    
5. В диалоговом окне **Загрузка завершена** нажмите кнопку **Закрыть** , чтобы закрыть диалоговое окно.
