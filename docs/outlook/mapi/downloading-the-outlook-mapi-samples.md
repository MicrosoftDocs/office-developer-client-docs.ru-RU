# <a name="downloading-the-outlook-mapi-samples"></a>Загрузка примеров MAPI для Outlook

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Примеры MAPI Для Microsoft Outlook включают поставщик адресной книги, поставщик хранения сообщений и поставщик транспорта.
  
 **Загрузка примера поставщика адресной книги**
  
1. Чтобы скачать примеры MAPI для Outlook, перейдите на вкладку **"Главная"** на странице примеров кода для API сообщений [Outlook (MAPI).](https://ol2010mapisamples.codeplex.com/) 
    
2. На **вкладке "Главная"** щелкните здесь, чтобы скачать источник примеров **MAPI для Outlook 2010.**
    
3. Ознакомьтесь с лицензионным соглашением и, если вы согласны, нажмите кнопку **"Я соглашаюсь".**
    
4. В **диалоговом окне** "Загрузка файла" нажмите кнопку **"Сохранить".** В **диалоговом** окне "Сохранить как" найдите папку, в которой нужно сохранить исходные файлы, и нажмите кнопку **"Сохранить".**
    
5. В **диалоговом окне** "Загрузка завершена" нажмите **кнопку "Закрыть",** чтобы закрыть диалоговое окно.
