# <a name="pidtaggeneration-canonical-property"></a>Каноническое свойство PidTagGeneration

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит сокращенное название, которое соответствует полному имени получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ЖЕНЕРАТИОН, ПР_ЖЕНЕРАТИОН_А, ПР_ЖЕНЕРАТИОН_В  <br/> |
|Идентификатор:  <br/> |0x3A05  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства обеспечивают идентификацию и доступ к сведениям о получателе. Они определяются получателем и их организацией. 
  
Распространенные значения: мл., SR. и III.
