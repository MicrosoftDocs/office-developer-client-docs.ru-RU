# <a name="pidlidhomeaddress-canonical-property"></a>Каноническое свойство PidLidHomeAddress

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает полный адрес домашнего адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidHomeAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000801A  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть сочетанием других свойств физических адресов и основано на локальном расположении клиента.
