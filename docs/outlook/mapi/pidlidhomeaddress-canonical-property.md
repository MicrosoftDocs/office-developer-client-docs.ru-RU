# <a name="pidlidhomeaddress-canonical-property"></a>Каноническое свойство PidLidHomeAddress

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает полный адрес домашнего адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидхомеаддресс  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000801A  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть сочетанием других свойств физических адресов и зависит от языкового стандарта клиента.
