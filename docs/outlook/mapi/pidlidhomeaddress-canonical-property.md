# <a name="pidlidhomeaddress-canonical-property"></a>Каноническое свойство PidLidHomeAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает полный адрес домашний адрес контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidHomeAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000801A  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство должно быть комбинацией другие свойства физических адресов и основано на языковой стандарт клиентского компьютера.
