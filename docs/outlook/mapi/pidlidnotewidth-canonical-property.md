# <a name="pidlidnotewidth-canonical-property"></a>Каноническое свойство PidLidNoteWidth

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает ширину видимого окна сообщения в пикселях. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспиднотевидс  <br/> |
|Набор свойств:  <br/> |PSETID_Note  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008B02  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Заметка  <br/> |
   
## <a name="remarks"></a>Примечания

Это значение должно быть больше нуля.
