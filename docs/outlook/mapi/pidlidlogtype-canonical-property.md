# <a name="pidlidlogtype-canonical-property"></a>Каноническое свойство PidLidLogType

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит отображаемое имя приложения для ведения журнала (например, "Мсворд") и обычно является атрибутом произвольной формы сообщения журнала, как правило, строка.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидлогтипе  <br/> |
|Набор свойств:  <br/> |Псетид_лог  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008700  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Журнал  <br/> |
