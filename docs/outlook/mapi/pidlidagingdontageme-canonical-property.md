# <a name="pidlidagingdontageme-canonical-property"></a>Каноническое свойство PidLidAgingDontAgeMe

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает значение TRUE, если элемент не будет автоматически архивироваться.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидагингдонтажеме  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000850E  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Распространенная  <br/> |
