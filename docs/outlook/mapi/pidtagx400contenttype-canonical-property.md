# <a name="pidtagx400contenttype-canonical-property"></a>Каноническое свойство PidTagX400ContentType

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Это свойство содержит тип контента для отправленного сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_X400_CONTENT_TYPE  <br/> |
|Идентификатор:  <br/> |0x003C  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |X400  <br/> |
