# <a name="pidtagownstoreentryid-canonical-property"></a>Каноническое свойство PidTagOwnStoreEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор строго связанного хранилища сообщений транспорта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ОВН_СТОРЕ_ЕНТРИД  <br/> |
|Идентификатор:  <br/> |0x3E06  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Свойства хранилища сообщений  <br/> |
   
## <a name="remarks"></a>Комментарии

Это свойство указывает идентификатор для тесно связанного хранилища, если он существует. Например, поставщик транспорта может указать идентификатор записи хранилища личных папок, чтобы диспетчер очереди MAPI мог подключить поставщик транспорта к хранилищу.
