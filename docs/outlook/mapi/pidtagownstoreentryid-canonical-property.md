# <a name="pidtagownstoreentryid-canonical-property"></a>Каноническое свойство PidTagOwnStoreEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор записи в тесном хранилище сообщений транспортного средства.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OWN_STORE_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x3E06  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Свойства магазина сообщений  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство указывает идентификатор записи для тесно соедененного магазина, если он существует. Например, поставщик транспорта может указать идентификатор входа в частный магазин папок, чтобы шпалер MAPI подключил поставщика транспорта к магазину.
