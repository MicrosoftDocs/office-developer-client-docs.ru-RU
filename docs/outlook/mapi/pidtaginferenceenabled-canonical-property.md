# <a name="pidtaginferenceenabled-canonical-property"></a>Каноническое свойство PidTagInferenceEnabled

  
  
**Относится к**: Outlook 
  
Хранить свойство Управление вывод ведения журнала.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_INFERENCES_ENABLED  <br/> |
|Идентификатор:  <br/> |0x3439000  <br/> |
|Тип данных:  <br/> |Логический  <br/> |
|Область:  <br/> |Хранение сообщений  <br/> |
