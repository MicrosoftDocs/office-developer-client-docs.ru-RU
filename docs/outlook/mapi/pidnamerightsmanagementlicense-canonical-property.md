# <a name="pidnamerightsmanagementlicense-canonical-property"></a>Каноническое свойство PidNameRightsManagementLicense

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Кэшет лицензии на использование для управляемого правами сообщения электронной почты.
  
|||
|:-----|:-----|
|Дружественные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Имя свойства:  <br/> |DRMLicense  <br/> |
|Тип данных:  <br/> |PT_MV_BINARY  <br/> |
|Область:  <br/> |Безопасный обмен сообщениями  <br/> |
   
## <a name="remarks"></a>Примечания

Если свойство присутствует в сообщении электронной почты, управляемом правами, первое значение этого нескольких двоичных свойств должно содержать ZLIB (как указано в [RFC1950]) сжатую лицензию на использование для управляемого правами сообщения электронной почты.
