# <a name="pidlidremindertime-canonical-property"></a>Каноническое свойство PidLidReminderTime

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает начальное время сигнала для напоминания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидреминдертиме  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008502  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Напоминание  <br/> |
   
## <a name="remarks"></a>Примечания

Для объектов Calendar это свойство представляет время, когда пользователь будет заменяться временем начала встречи. Клиенты должны задать значение в формате UTC.
