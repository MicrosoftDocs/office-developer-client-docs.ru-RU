# <a name="pidlidlogduration-canonical-property"></a>Каноническое свойство PidLidLogDuration

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Представляет интервал в минутах сообщения журнала.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidLogDuration  <br/> |
|Набор свойств:  <br/> |PSETID_Log  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008707  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |������  <br/> |
   
## <a name="remarks"></a>Замечания

Интервал в минутах, действия, которые должны быть различие между свойствами **dispidLogStart** ([PidLidLogStart](pidlidlogstart-canonical-property.md)) и **dispidLogEnd** ([PidLidLogEnd](pidlidlogend-canonical-property.md)).
