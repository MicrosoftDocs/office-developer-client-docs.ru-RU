# <a name="pidtagtypeofx400user-canonical-property"></a>Каноническое свойство PidTagTypeOfX400User

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Это свойство содержит тип получателя сообщения для использования в отчете.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_TYPE_OF_MTS_USER  <br/> |
|Идентификатор:  <br/> |0x0C1C  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Получатель MAPI  <br/> |
