# <a name="pidtagspamtrustedrecipients-canonical-property"></a>Каноническое свойство PidTagSpamTrustedRecipients
 
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит список адресов электронной почты и доменов, которые представляют надежных получателей, разделенных точкой с запятой.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SPAM_TRUSTED_RECIPIENTS_W  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0419  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Нежелательной почты  <br/> |
