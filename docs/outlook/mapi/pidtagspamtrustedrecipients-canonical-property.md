# <a name="pidtagspamtrustedrecipients-canonical-property"></a>Каноническое свойство PidTagSpamTrustedRecipients
 
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит разделенный точками с запятой список адресов электронной почты и доменов, которые представляют доверенных получателей.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_СПАМ_ТРУСТЕД_РЕЦИПИЕНТС_В  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0419  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Нежелательная почта  <br/> |
