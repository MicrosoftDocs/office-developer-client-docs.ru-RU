# <a name="ipersistmessagesave"></a>IPersistMessage::Save

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Сохраняет измененную форму обратно в сообщение, из которого она была загружена или создана.
  
```cpp
HRESULT Save(
  LPMESSAGE pMessage,
  ULONG fSameAsLoad
);
```

## <a name="parameters"></a>Параметры

 _Пмессаже_
  
> возврата Указатель на сообщение.
    
 _Фсамеаслоад_
  
> возврата ЗНАЧЕНИЕ TRUE, чтобы указать, что сообщение, на которое указывает _пмессаже_ , является сообщением, из которого была загружена или создана форма; в противном случае — FALSE. 
    
## <a name="return-value"></a>Возвращаемое значение

S_OK 
  
> Форма успешно сохранена.
    
## <a name="remarks"></a>Замечания

Средства просмотра форм вызывают метод **иперсистмессаже:: Save** , чтобы сохранить измененную форму обратно в сообщение, из которого она была загружена или создана. 
  
 Метод **Save** следует вызывать только в том случае, если форма находится в [нормальном](normal-state.md) состоянии. 
  
## <a name="notes-to-implementers"></a>Примечания для исполнителей

Не зафиксируйте сохраненные изменения; Вы можете зафиксировать изменения, вызываемый абонентом. Никогда не вносите изменения в свойства, принадлежащие сообщению формы, за исключением во время вызова метода **Save** . 
  
Если для _фсамеаслоад_ ЗАДАНО значение true, вы можете сохранить изменения в существующем сообщении формы. Если для _фсамеаслоад_ ЗАДАНО значение false, необходимо скопировать все свойства из исходного сообщения в сообщение, на которое указывает _пмессаже_ , перед выполнением сохранения. Используйте метод [IMAPIProp:: CopyTo](imapiprop-copyto.md) исходного сообщения, чтобы скопировать свойства. 
  
Когда все свойства будут скопированы, введите состояние " [каракули](noscribble-state.md) ". Если ошибок не возникло, возвращается значение S_OK. В противном случае возвращается сообщение об ошибке из действия Failed. 
  
Если метод **Save** вызывается, когда форма находится в любом состоянии, кроме Normal, возвращается е_унекспектед. 
  
Для получения дополнительных сведений о сохранении объектов хранилища обратитесь к документации по методу [иперсистстораже](https://msdn.microsoft.com/library/1c1a20fc-c101-4cbc-a7a6-30613aa387d7%28Office.15%29.aspx) .
