# <a name="pidtagattachmimesequence-canonical-property"></a>Каноническое свойство PidTagAttachMimeSequence

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит номер последовательности MIME вложения сообщений MIME.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ATTACH_MIME_SEQUENCE  <br/> |
|Идентификатор:  <br/> |0x3710  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Свойства вложения сообщений  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство используется для поддержки MHTML. Он представляет номер последовательности вложения в родительской многопартийной части MIME в сообщении MIME.
