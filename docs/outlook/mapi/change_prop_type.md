# <a name="change_prop_type"></a>CHANGE_PROP_TYPE

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Обновляет тип свойства тега свойства до указанного значения. Идентификатор свойства не изменяется. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |[SPropValue](spropvalue.md) <br/> |
   
```cpp
CHANGE_PROP_TYPE (ulPropTag, ulPropType)
```

## <a name="parameters"></a>Параметры

 _ulPropTag_
  
> Тег свойства, который необходимо изменить.
    
 _ulPropType_
  
> Новое значение для типа свойства.
