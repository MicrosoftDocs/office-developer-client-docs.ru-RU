# <a name="form-verbs"></a>Форма команд

**Относится к**: Outlook 
  
Пользовательский интерфейс формы обычно предлагает пунктов меню или элементов управления, которые позволяют пользователям выполнить какое-либо действие с формой. Это задание сервера формы для обработки этих действий пользователя. Этот интерфейс реализуется с помощью стандартных интерфейсов API Win32; одну запись — так же, как записи других интерфейсов для регулярного Win32 программы.
  
Часто пользовательских действий связаны с команд. Команды — это имя для действия, характерные для определенного класса сообщения. Например **ответ** — это команда, реализуемый большое количество серверов формы, каждая из которых может иметь разные интерпретацию этой команды. Команды иногда называется команды. 
  
> [!NOTE]
> Не все элементы меню и элементов управления на форме соответствуют команды. Например кнопка **Отмена** не соответствует команда "Отмена" в пределах сервера формы. Как правило команды связаны с действиями, характерные для определенного сообщения класса или набора классов сообщений. Хотя для различных классов сообщений может поддерживать различные наборы команд, все по крайней мере поддерживает команду Open, отображает формы пользовательского интерфейса и загружает значения свойства сообщения. 
  
Команды может занять без параметров. Формы, которые Экспорт команд с помощью переменной параметры необходимо использовать механизмы автоматизации.
  
Клиенты можно определить, какие команды поддерживаются классом определенного сообщения через метод [IMAPIFormInfo::CalcVerbSet](imapiforminfo-calcverbset.md) , который реализован диспетчера формы MAPI. Диспетчер форм возвращает эту информацию из файла конфигурации формы. Команда set, возвращаемые данным методом используется клиентом для отображения пользователя, какие команды могут быть выполнены на сообщение. Например клиент может разрешить пользователям щелкните правой кнопкой мыши на сообщение, которое должно отображаться команд, которые применяются к этого сообщения.
