<span data-ttu-id="76df3-105">Пользовательский интерфейс формы обычно содержит элементы меню или элементы управления, позволяющие пользователям выполнять некоторые действия с формой.</span><span class="sxs-lookup"><span data-stu-id="76df3-105">A form's user interface typically offers menu items or controls that enable users to take some kind of action with the form.</span></span> <span data-ttu-id="76df3-106">Это задача сервера форм для обработки этих действий пользователя.</span><span class="sxs-lookup"><span data-stu-id="76df3-106">It is the form server's job to handle these user actions.</span></span> <span data-ttu-id="76df3-107">Этот интерфейс реализуется с помощью стандартных API Win32; Написание такового аналогично написанию других интерфейсов для обычных программ Win32.</span><span class="sxs-lookup"><span data-stu-id="76df3-107">This interface is implemented using standard Win32 APIs; writing one is just like writing other interfaces for regular Win32 programs.</span></span>
  
<span data-ttu-id="76df3-108">Часто действия пользователя связаны с глаголами.</span><span class="sxs-lookup"><span data-stu-id="76df3-108">Often, user actions are associated with verbs.</span></span> <span data-ttu-id="76df3-109">Глагол — это имя действия, которое относится к определенному классу сообщений.</span><span class="sxs-lookup"><span data-stu-id="76df3-109">A verb is the name for an action that is specific to a certain message class.</span></span> <span data-ttu-id="76df3-110">Например, **ответ** — это команда, реализованная многими серверами форм, каждая из которых может иметь различную интерпретацию этой команды.</span><span class="sxs-lookup"><span data-stu-id="76df3-110">For example, **Reply** is a verb that is implemented by many form servers, each of which may have a different interpretation of that verb.</span></span> <span data-ttu-id="76df3-111">Команды иногда называют командами.</span><span class="sxs-lookup"><span data-stu-id="76df3-111">Verbs are sometimes referred to as commands.</span></span> 
  
> [!NOTE]
> <span data-ttu-id="76df3-112">Не все элементы меню и элементы управления в форме соответствуют глаголу.</span><span class="sxs-lookup"><span data-stu-id="76df3-112">Not all menu items and controls on a form correspond to a verb.</span></span> <span data-ttu-id="76df3-113">Например, кнопка **Cancel** не соответствует команде отмены в форме сервера.</span><span class="sxs-lookup"><span data-stu-id="76df3-113">For example, a **Cancel** button does not correspond to a Cancel verb within the form server.</span></span> <span data-ttu-id="76df3-114">Обычно команды связаны с действиями, которые относятся к определенному классу сообщений или набору классов сообщений.</span><span class="sxs-lookup"><span data-stu-id="76df3-114">Usually, verbs are associated with actions that are specific to a particular message class or a set of message classes.</span></span> <span data-ttu-id="76df3-115">Несмотря на то что различные классы сообщений могут поддерживать различные наборы команд, все они поддерживают по крайней мере команду открытия, которая отображает пользовательский интерфейс формы и загружает их со значениями свойств сообщения.</span><span class="sxs-lookup"><span data-stu-id="76df3-115">Although different message classes can support different sets of verbs, all support at least the Open verb, which displays the form's user interface and loads it with the message's property values.</span></span> 
  
<span data-ttu-id="76df3-116">Команды могут не иметь параметров.</span><span class="sxs-lookup"><span data-stu-id="76df3-116">Verbs may take no parameters.</span></span> <span data-ttu-id="76df3-117">Формы, в которых экспортируются команды с параметрами переменных, должны использовать механизмы автоматизации.</span><span class="sxs-lookup"><span data-stu-id="76df3-117">Forms that export commands with variable parameters must use the Automation mechanisms.</span></span>
  
<span data-ttu-id="76df3-118">Клиенты могут определить, какие команды поддерживаются определенным классом сообщения с помощью метода [имапиформинфо:: калквербсет](imapiforminfo-calcverbset.md) , реализованного диспетчером форм MAPI.</span><span class="sxs-lookup"><span data-stu-id="76df3-118">Clients can determine which verbs are supported by a particular message class through the [IMAPIFormInfo::CalcVerbSet](imapiforminfo-calcverbset.md) method, which is implemented by the MAPI form manager.</span></span> <span data-ttu-id="76df3-119">Диспетчер форм получает эти сведения из файла конфигурации формы.</span><span class="sxs-lookup"><span data-stu-id="76df3-119">The form manager gets this information from the form's configuration file.</span></span> <span data-ttu-id="76df3-120">Набор команд, возвращаемый этим методом, используется клиентом, чтобы показать пользователю, какие команды можно выполнить над сообщением.</span><span class="sxs-lookup"><span data-stu-id="76df3-120">The verb set returned by this method is used by the client to show the user which commands can be executed on a message.</span></span> <span data-ttu-id="76df3-121">Например, клиент может позволить пользователям щелкнуть правой кнопкой мыши на сообщении, чтобы отобразить команды, которые относятся к этому сообщению.</span><span class="sxs-lookup"><span data-stu-id="76df3-121">For example, a client might enable users to click the right mouse button over a message to display verbs applicable to that message.</span></span>
