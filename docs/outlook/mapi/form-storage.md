<span data-ttu-id="caec7-110">Связанные часть включает объекты скрытого сообщения, связанные с папкой, включая определения формы, представления, шаблоны правил, ответ шаблонов и т. д.</span><span <span data-ttu-id="caec7-118">Кроме того форм могут храниться в файлах и расположение, которое поддерживает используется диспетчер форм.</span><span class="sxs-lookup"><span data-stu-id="caec7-118">Additionally, forms can be stored in any file or location that the form manager being used supports.</span></span> <span data-ttu-id="caec7-119">Диспетчер форм по умолчанию хранит серверы формы MAPI, но диспетчер настраиваемая форма может реализовать свой собственный хранилища для серверов формы.</span><span class="sxs-lookup"><span data-stu-id="caec7-119">The default form manager stores form servers in MAPI folders, but a custom form manager could implement its own storage for form servers.</span></span>
