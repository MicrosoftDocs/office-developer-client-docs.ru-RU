# <a name="copying-a-message-service"></a>Копирование службы сообщений

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
 **Копирование службы сообщений в профиль**
  
- Вызов [IMsgServiceAdmin::CopyMsgService](imsgserviceadmin-copymsgservice.md).
    
При копировании службы сообщений новый экземпляр службы настраивается точно так же, как и оригинал. Иногда **CopyMsgService** возвращает ошибку MAPI_E_ACCESS_DENIED. Наиболее распространенной причиной возврата ошибок является служба сообщений, которая не позволяет дублироваться.
