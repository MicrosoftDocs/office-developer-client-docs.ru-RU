# <a name="capabilities-not-supported-by-form-managers"></a>Возможности, не поддерживаемые управляющими формами

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Следующие функции не поддерживаются диспетчером форм по умолчанию в целях повышения производительности, но могут поддерживаться настраиваемыми управляющими формами.
  
- Иерархия, позволяющая группировать формы или упорядочивать их по всей библиотеке форм. Библиотека форм — это база данных с плоскими файлами форм.
    
- Управление доступом для категорий форм, соответствующих классам сообщений или подклассам.
    
- Поддержка версий одной формы для нескольких языков в одной библиотеке форм.
    
Это проблемы реализации. Не существует ничего, чтобы предотвратить реализацию этих функций настраиваемым диспетчером форм.
  
Архитектура форм MAPI не поддерживает несколько диспетчеров форм, выполняющихся одновременно. Несмотря на то, что MAPI поддерживает несколько одновременных поставщиков хранилищ сообщений, поставщиков транспорта и поставщиков адресных книг, поддерживается только один диспетчер форм.
  
Так как при реализации настраиваемого диспетчера форм может выполняться только один диспетчер форм, при реализации настраиваемого диспетчера форм потребуется повторно реализовать все функциональные возможности из диспетчера форм по умолчанию. Так как настраиваемый диспетчер форм будет полностью заменять Диспетчер форм по умолчанию, возможности диспетчера форм по умолчанию будут недоступны, если они не дублируются в настраиваемом диспетчере форм.
