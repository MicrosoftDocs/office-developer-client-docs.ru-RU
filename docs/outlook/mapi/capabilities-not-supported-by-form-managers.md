<span data-ttu-id="4fcfb-105">Следующие функции не поддерживаются диспетчером форм по умолчанию в целях повышения производительности, но могут поддерживаться настраиваемыми управляющими формами.</span><span class="sxs-lookup"><span data-stu-id="4fcfb-105">The following features are not supported by the default form manager for performance reasons but can be supported by custom form managers.</span></span>
