# <a name="capabilities-not-supported-by-form-managers"></a>Возможности, не поддерживаемые диспетчерами форм

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Следующие функции не поддерживаются диспетчером форм по умолчанию из соображений производительности, но могут поддерживаться настраиваемые диспетчеры форм.
  
- Иерархия, которая позволяет сгруппированы или классифицируются в библиотеке форм. Библиотека форм — это плоская база данных форм.
    
- Управление доступом для категорий форм, соответствующих классам сообщений или суперклассам.
    
- Поддержка нескольких языковых версий одной формы в одной библиотеке форм.
    
Это проблемы реализации. Нет ничего, чтобы предотвратить реализацию этих функций настраиваемой диспетчером форм.
  
Архитектура формы MAPI не поддерживает одновременное запуск нескольких диспетчеров форм. Хотя MAPI поддерживает несколько поставщиков одновременного хранения сообщений, поставщиков транспорта и поставщиков адресных книг, поддерживается только один диспетчер форм.
  
Так как одновременно может работать только один диспетчер форм, при реализации настраиваемого диспетчера форм потребуется повторно реализовать все необходимые функции из диспетчера форм по умолчанию. Так как настраиваемый диспетчер форм полностью заменит диспетчер форм по умолчанию, возможности диспетчера форм по умолчанию будут недоступны, если они не дублируются в настраиваемом диспетчере форм.
