# <a name="implementing-a-configuration-interface-for-message-store-providers"></a>Реализация интерфейса конфигурации для поставщиков магазинов сообщений

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Поставщики магазинов сообщений должны реализовать интерфейс, позволяющий пользователю настраивать поставщика магазина сообщений для работы на компьютере этого пользователя. Обычно поставщик магазина сообщений настраивается при добавлении поставщика магазина сообщений в профиль пользователя. Интерфейс конфигурации поставщика хранилища сообщений обычно обрабатывает такие задачи, как настройка имен пользователей и паролей для защищенных магазинов сообщений, выбор путей к необходимым файлам и создание механизма хранения, который он будет использовать при необходимости.
  
Реализованный интерфейс конфигурации имеет доступ к дополнительным точкам входа в DLL поставщика сообщений. Дополнительные сведения см. [в тексте Настройка службы сообщений.](configuring-a-message-service.md) Интерфейс конфигурации поставщика магазина сообщений — это единственный пользовательский интерфейс, который должен реализовать поставщик магазина сообщений.
