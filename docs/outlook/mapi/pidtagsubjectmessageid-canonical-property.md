# <a name="pidtagsubjectmessageid-canonical-property"></a>Каноническое свойство PidTagSubjectMessageId

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит двоичные значения, которые копируются из сообщения, для которого создается отчет. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SUBJECT_IPM  <br/> |
|Идентификатор:  <br/> |0x0038  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство как свойство **PR_REPORT_TAG** ([PidTagReportTag](pidtagreporttag-canonical-property.md)) можно использовать для соотнесения с отчетом с исходного сообщения.
