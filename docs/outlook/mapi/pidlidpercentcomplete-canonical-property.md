# <a name="pidlidpercentcomplete-canonical-property"></a>Каноническое свойство PidLidPercentComplete

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает прогресс, достигнутый пользователем в задаче. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidPercentComplete  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x00008102  <br/> |
|Тип данных:  <br/> |PT_R8  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть больше или равно 0.0 и меньше или меньше 1.0, где 1.0 указывает, что работа завершена, а 0.0 указывает, что работа еще не начата.
  
Для объекта сообщения с флагом времени значение этого свойства должно быть установлено до 1.0, если объект помечен завершен, а в противном случае должно быть задано значение 0.0.
