# <a name="pidlidsharingremotestoreuid-canonical-property"></a>Каноническое свойство PidLidSharingRemoteStoreUid

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает в строковом представлении значение свойства **PR_STORE_ENTRYID** ([PidTagStoreEntryId)](pidtagstoreentryid-canonical-property.md)в общей папке. Это свойство общего сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidSharingRemoteStoreUid  <br/> |
|Набор свойств:  <br/> |PSETID_Sharing  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008A48  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общий доступ  <br/> |
