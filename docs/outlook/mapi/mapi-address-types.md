# <a name="mapi-address-types"></a>Типы адресов MAPI

  
  
**Относится к**: Outlook 
  
Каждый пользователь с сообщениями связан с тип адреса символ строка, описывающая формат адрес пользователя, которая хранится в свойстве **PR_ADDRTYPE** ([PidTagAddressType](pidtagaddresstype-canonical-property.md)). Сопоставление типов адресов форматов адресов. То есть просмотрев тип адреса получателя, клиентские приложения можно определить форматирование адреса для получателя. 
  
Например `SMTP` указывает тип адреса стандартных веб-адрес: 
  
 `username@companyname.com.`
  
И `EX` тип адреса указывает адрес сервера Exchange. 
  
Все записи адресной книги должен иметь допустимый адрес тип. Клиентам требуется своим пользователям указать тип адреса при создании типа настраиваемого получателя не поддерживается поставщик адресной книги. Для записи, которые они поддерживают поставщиками адресной книги необходимо предоставить допустимый адрес типы. 
  
MAPI определяет тип только один адрес: MAPIPDL, соответствующий списку рассылки.
  
Чтобы получить список типов адресов, поддерживаемые всеми поставщиками транспорта в сеансе, клиентские приложения вызовите метод **IMAPISession::EnumAdrTypes** . Для получения дополнительных сведений см [IMAPISession::EnumAdrTypes](imapisession-enumadrtypes.md).
