# <a name="mapi-address-types"></a>Типы адресов MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Каждый пользователь обмена сообщениями связан с типом адреса, строкой символов, описывающих формат адреса пользователя, который хранится в **свойстве PR_ADDRTYPE** [(PidTagAddressType).](pidtagaddresstype-canonical-property.md) Карта типов адресов для адресных форматов. То есть, изучав тип адреса получателя, клиентские приложения могут определить, как форматовать адрес, соответствующий получателю. 
  
Например, тип  `SMTP` адреса указывает стандартный интернет-адрес: 
  
 `username@companyname.com.`
  
И тип `EX` адреса указывает Exchange Server адрес. 
  
Все записи адресной книги должны иметь допустимый тип адреса. Клиенты требуют от пользователей указать тип адреса при создании типа настраиваемого получателя, неподтверченного поставщиком адресной книги. Для поддерживаемой записи поставщики адресных книг должны предоставить допустимые типы адресов. 
  
MAPI определяет только один тип адреса: MAPIPDL, который обозначает личный список рассылки.
  
Чтобы получить список типов адресов, поддерживаемых всеми поставщиками транспорта в сеансе, клиентские приложения называют **метод IMAPISession::EnumAdrTypes.** Дополнительные сведения см. [в iMAPISession::EnumAdrTypes.](imapisession-enumadrtypes.md)
