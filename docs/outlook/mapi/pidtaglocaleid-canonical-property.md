# <a name="pidtaglocaleid-canonical-property"></a>Каноническое свойство PidTagLocaleId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Устанавливает языковой язык для перевода системных сообщений, таких как отчеты о доставке.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_LOCALE_ID  <br/> |
|Идентификатор:  <br/> |0x66A1  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Разное  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство доступно для частного почтового ящика.
