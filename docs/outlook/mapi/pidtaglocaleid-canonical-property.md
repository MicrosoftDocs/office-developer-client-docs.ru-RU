# <a name="pidtaglocaleid-canonical-property"></a>Каноническое свойство PidTagLocaleId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Устанавливает языковой стандарт для преобразования созданных системой сообщений, таких как отчеты о доставке.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ЛОКАЛЕ_ИД  <br/> |
|Идентификатор:  <br/> |0x66A1  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Разное  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство доступно для личного входа в почтовый ящик.
