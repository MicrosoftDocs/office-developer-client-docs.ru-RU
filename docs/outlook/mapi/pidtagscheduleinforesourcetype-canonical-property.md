# <a name="pidtagscheduleinforesourcetype-canonical-property"></a>Каноническое свойство PidTagScheduleInfoResourceType

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение, которое должно быть установлено до нуля (0).
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SCHDINFO_RESOURCE_TYPE  <br/> |
|Идентификатор:  <br/> |0x6841  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Бесплатный/занятый  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть установлено до нуля (0) при отправлении и игнорировании при получении.
