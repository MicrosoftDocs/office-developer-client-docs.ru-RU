# <a name="pidlidremotesearchkey-canonical-property"></a>Каноническое свойство PidLidRemoteSearchKey

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает ключ поиска удаленного элемента.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидремотесеарчкэй  <br/> |
|Набор свойств:  <br/> |Псетид_ремоте  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008F06  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Удаленное сообщение  <br/> |
