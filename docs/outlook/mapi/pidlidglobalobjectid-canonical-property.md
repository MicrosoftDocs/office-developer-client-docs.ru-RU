# <a name="pidlidglobalobjectid-canonical-property"></a>Каноническое свойство PidLidGlobalObjectId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает уникальный идентификатор объекта Calendar.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |LID_GLOBAL_OBJID  <br/> |
|Набор свойств:  <br/> |PSETID_Meeting  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00000003  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

После установки для объекта Calendar значение этого свойства не должно изменяться. Подробное описание формата можно найти в [[MS – оксокал]](https://msdn.microsoft.com/library/9b636532-9150-4836-9635-9c9b756c9ccf%28Office.15%29.aspx).
