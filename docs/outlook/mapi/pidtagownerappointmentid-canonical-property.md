# <a name="pidtagownerappointmentid-canonical-property"></a>Каноническое свойство PidTagOwnerAppointmentId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор для встречи в расписании владельца.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OWNER_APPT_ID  <br/> |
|Идентификатор:  <br/> |0x0062  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Appointment  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство используется в запросах на собрания. Это не идентификатор записи, а длинный ряд, который однозначно определяет назначение в расписании отправителем.
