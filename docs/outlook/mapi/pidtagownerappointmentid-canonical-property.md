# <a name="pidtagownerappointmentid-canonical-property"></a>Каноническое свойство PidTagOwnerAppointmentId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор встречи в расписании владельца.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ОВНЕР_АППТ_ИД  <br/> |
|Идентификатор:  <br/> |0x0062  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Appointment  <br/> |
   
## <a name="remarks"></a>Комментарии

Это свойство используется в приглашениях на собрания. Он не представляет идентификатор записи, но длинное целое число, однозначно идентифицирующее встречу в расписании отправителя.
