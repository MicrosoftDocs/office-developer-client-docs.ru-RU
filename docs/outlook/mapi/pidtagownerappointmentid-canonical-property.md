# <a name="pidtagownerappointmentid-canonical-property"></a>Каноническое свойство PidTagOwnerAppointmentId

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор для встречи в расписание владельца.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OWNER_APPT_ID  <br/> |
|Идентификатор:  <br/> |0x0062  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Appointment  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство используется в приглашении на собрание. Представляет идентификатор записи, но длинное целое число, однозначно идентифицирующее встречи в расписание отправителя.
