# <a name="pidlidworkaddress-canonical-property"></a>Каноническое свойство PidLidWorkAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает полный рабочий адрес контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидворкаддресс  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000801B  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство должно быть сочетанием других свойств физических адресов и зависит от языкового стандарта клиента.
