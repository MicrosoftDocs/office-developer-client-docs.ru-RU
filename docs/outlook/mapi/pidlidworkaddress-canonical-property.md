# <a name="pidlidworkaddress-canonical-property"></a>Каноническое свойство PidLidWorkAddress

  
  
**Относится к**: Outlook 
  
Задает полный рабочий адрес контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidWorkAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000801B  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство должно быть комбинацией другие свойства физических адресов и основано на языковой стандарт клиентского компьютера.
