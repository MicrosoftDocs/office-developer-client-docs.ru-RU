# <a name="pidtagbodycontentlocation-canonical-property"></a>Каноническое свойство PidTagBodyContentLocation

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит значение поля загона содержимого MIME.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_BODY_CONTENT_LOCATION, PR_BODY_CONTENT_LOCATION_A, PR_BODY_CONTENT_LOCATION_W  <br/> |
|Идентификатор:  <br/> |0x1014  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы установить значение этих свойств, клиенты MIME должны записать нужное значение в поле заголовщика content-Location в объекте MIME, соповедуемом с текстом сообщения.
  
Читатели MIME должны скопировать значение поля загоровка content-Location для такой сущности MIME в значение этих свойств.
