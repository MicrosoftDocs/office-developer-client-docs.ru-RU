# <a name="sizeddtblbutton"></a>SizedDtblButton

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Создает именованную структуру, которая включает структуру [DTBLBUTTON](dtblbutton.md) для описания кнопки и метки указанной длины. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |**DTBLBUTTON** <br/> |
   
```cpp
SizedDtblButton (n, u)
```

## <a name="parameters"></a>Параметры

 _n_
  
> Длина метки, включаемой в новую структуру.
    
 _u_
  
> Имя новой структуры.
    
## <a name="remarks"></a>Примечания

Новая структура создается с помощью следующих членов:
  
```
DTBLBUTTON dtblbutton;
TCHAR lpszLabel[n];

```
