# <a name="pidtagjunkphishingenablelinks-canonical-property"></a>Каноническое свойство PidTagJunkPhishingEnableLinks

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Означает, что в случае true фишинговая отметка в сообщении должна быть проигнорирована.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_JUNK_PHISHING_ENABLE_LINKS  <br/> |
|Идентификатор:  <br/> |0x6107  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Спам  <br/> |
