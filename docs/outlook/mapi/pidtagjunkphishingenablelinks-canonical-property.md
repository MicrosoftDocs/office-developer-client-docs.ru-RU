# <a name="pidtagjunkphishingenablelinks-canonical-property"></a>Каноническое свойство PidTagJunkPhishingEnableLinks

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Означает, что штамп фишинга в сообщении должен игнорироваться.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ЖУНК_ФИШИНГ_ЕНАБЛЕ_ЛИНКС  <br/> |
|Идентификатор:  <br/> |0x6107  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Нежелательная почта  <br/> |
