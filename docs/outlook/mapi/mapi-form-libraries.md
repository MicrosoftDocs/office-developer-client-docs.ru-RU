# <a name="mapi-form-libraries"></a>Библиотеки форм MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
С целью упорядочения серверов форм и сделать их легко доступными для клиентских приложений, архитектура формы MAPI включает в себя библиотеки форм, которые поддерживают установку, администрирования, репликации и использования серверов формы.
  
Библиотеки форм, репозиториев формы серверов и конфигурации сведения о них, которые использует диспетчер форм при загрузке формы серверов. 
  
> [!NOTE]
> На раннем этапе журнал MAPI библиотеки форм вызов «значений». Таким образом термин «реестра» сохраненные в некоторые связанные с формы MAPI интерфейсы, свойства, записей файла конфигурации и т. д. При интерфейса вашего приложения требуется использовать один из этих терминов, термин «библиотеки» следует использовать исключительно.
