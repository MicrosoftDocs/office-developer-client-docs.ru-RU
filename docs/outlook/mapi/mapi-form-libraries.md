# <a name="mapi-form-libraries"></a>Библиотеки форм MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Чтобы упростить организацию серверов форм и сделать их доступными для клиентских приложений, архитектура форм MAPI включает библиотеки форм, которые поддерживают установку, Администрирование, репликацию и использование серверов форм.
  
Библиотеки форм — это репозитории серверов форм и сведения о конфигурации, которые Диспетчер форм использует при загрузке серверов форм. 
  
> [!NOTE]
> На ранних стадиях истории MAPI библиотеки форм назывались "реестрами". Поэтому термин "реестр" в некоторых интерфейсах MAPI, связанных с формами, свойствами, записями файла конфигурации и т. д. Если интерфейс приложения требует использования одного из этих терминов, следует использовать исключительно термин "Library".
