# <a name="mapi-form-libraries"></a>Библиотеки форм MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Чтобы у упорядочить серверы форм и сделать их доступными для клиентских приложений, архитектура форм MAPI включает библиотеки форм, которые поддерживают установку, администрирование, репликацию и использование серверов форм.
  
Библиотеки форм — это репозитории серверов форм и сведения о конфигурации, которые диспетчер форм использует при загрузке серверов форм. 
  
> [!NOTE]
> В начале истории MAPI библиотеки форм назывались "реестрами". По этой причине термин "реестр" сохраняется в некоторых интерфейсах MAPI, свойствах, файлах конфигурации и т. д. Если в интерфейсе приложения требуется использовать один из этих терминов, следует использовать термин "библиотека" исключительно.
