# <a name="pidtagotheraddresspostofficebox-canonical-property"></a>Каноническое свойство PidTagOtherAddressPostOfficeBox

  
  
**Относится к**: Outlook 
  
Содержит почтовый ящик для контакта в другой адрес.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OTHER_ADDRESS_POST_OFFICE_BOX, PR_OTHER_ADDRESS_POST_OFFICE_BOX_A, PR_OTHER_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Идентификатор:  <br/> |0x3A64  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства определяются пользователем или организации пользователя.
