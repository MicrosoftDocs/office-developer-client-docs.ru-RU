# <a name="pidtagcompanymaintelephonenumber-canonical-property"></a>Каноническое свойство PidTagCompanyMainTelephoneNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит основной номер телефона компании
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КОМПАНИ_МАИН_ФОНЕ_НУМБЕР, ПР_КОМПАНИ_МАИН_ФОНЕ_НУМБЕР_А, ПР_КОМПАНИ_МАИН_ФОНЕ_НУМБЕР_В  <br/> |
|Идентификатор:  <br/> |0x3A57  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Комментарии

Эти свойства определяются пользователем или организацией пользователя.
