# <a name="pidtagcompanymaintelephonenumber-canonical-property"></a>Каноническое свойство PidTagCompanyMainTelephoneNumber

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит основной телефонный номер компании
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_COMPANY_MAIN_PHONE_NUMBER, PR_COMPANY_MAIN_PHONE_NUMBER_A, PR_COMPANY_MAIN_PHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A57  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства определяются пользователем или организацией пользователя.
