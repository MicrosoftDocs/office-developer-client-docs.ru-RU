# <a name="pidlidscriptstream-canonical-property"></a>Каноническое свойство PidLidScriptStream

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Представляет код, написанный в Visual Basic scripting Edition (VBScript) за настраиваемой формой, связанной с сообщением.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidScriptStream  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008541  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конфигурация времени запуска  <br/> |
