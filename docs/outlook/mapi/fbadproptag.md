# <a name="fbadproptag"></a>FBadPropTag

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Проверяет указанный тег свойства. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapival.h  <br/> |
|Реализовано в:  <br/> |MAPI  <br/> |
|Вызывающая сторона:  <br/> |Поставщики услуг  <br/> |
   
```cpp
ULONG FBadPropTag(
  ULONG ulPropTag
);
```

## <a name="parameters"></a>Параметры

 _ulPropTag_
  
> [in] Проверяемая метка свойства.
    
## <a name="return-value"></a>Возвращаемое значение

TRUE 
  
> Указанный тег свойства не является допустимым тегом свойства MAPI. 
    
FALSE 
  
> Указанный тег свойства является допустимым тегом свойства MAPI.
    
## <a name="remarks"></a>Примечания

Функция **FBadPropTag** проверяет указанный тег свойства на основе определений MAPI. Он позволяет убедиться, что тип свойства является одним из типов, определенных MAPI, и что идентификатор свойства определен как тип этого типа.
