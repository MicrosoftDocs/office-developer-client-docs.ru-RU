# <a name="pidnamexsharingremotetype-canonical-property"></a>Каноническое свойство PidNameXSharingRemoteType

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Представляет значение свойства **диспидшарингремотетипе** ([PidLidSharingRemoteType](pidlidsharingremotetype-canonical-property.md)).
  
|||
|:-----|:-----|
|Понятные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |ПС_ИНТЕРНЕТ_ХЕАДЕРС  <br/> |
|Имя свойства:  <br/> |X – совместное использование — удаленный тип  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общий доступ  <br/> |
