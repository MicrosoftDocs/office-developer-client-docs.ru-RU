# <a name="pidlidtaskfrecurring-canonical-property"></a>Каноническое свойство PidLidTaskFRecurring

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, является ли задача включает в себя шаблон повторения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskFRecur  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008126  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Замечания

Если оставить это свойство определено, предполагается значение по умолчанию FALSE. Если он имеет значение TRUE, **dispidTaskRecur** ([PidLidTaskRecurrence](pidlidtaskrecurrence-canonical-property.md)), а также необходимо задать свойства **dispidTaskDeadOccur** ([PidLidTaskDeadOccurrence](pidlidtaskdeadoccurrence-canonical-property.md)) как указано в [[MS-OXOTASK]](https://msdn.microsoft.com/library/55600ec0-6195-4730-8436-59c7931ef27e%28Office.15%29.aspx).
