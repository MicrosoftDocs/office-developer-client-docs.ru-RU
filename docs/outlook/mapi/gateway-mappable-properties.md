# <a name="gateway-mappable-properties"></a>Свойства, сопоставляемые со шлюзами

**Область применения**: Outlook 2013 | Outlook 2016 
  
Свойства, которые можно использовать для шлюза, — это свойства, которые могут требовать перевода при отправке из одного домена обмена сообщениями в другой. Свойства MAPI, которые можно использовать с помощью шлюза, позволяют сообщениям включать сведения, которые требуют шлюза для обеспечения правильного использования системы обмена сообщениями назначения. Несмотря на то, что разработчики шлюзов не обязаны предоставлять эту возможность перевода, они должны рассматривать свойства, которые можно использовать для шлюза, как возможность улучшения обработки контента сообщений.
  
MAPI указывает пять типов свойств, которые могут быть недоступны для шлюза:
  
- Различающееся имя (DN)
    
- Краткое имя
    
- Тип электронной почты
    
- Идентификатор входа
    
- Ключ поиска
    
Это набор адресных свойств, связанных с получателями, отправителями, получателями отчетов и делегированием отправителей и получателей. Чтобы помочь клиенту определить эти свойства, чтобы шлюз обрабатывал их специально, MAPI указывает конвенцию имен с использованием именных свойств и наборов свойств. Существует пять наборов свойств для удержания именных свойств, которые требуют сопоставления. Существует один набор свойств для каждого типа подавимого свойства. Наборы свойств, которые будут удерживать указанные свойства адресов, следуют следующим образом.
  
|**Набор свойств**|**Описание**|
|:-----|:-----|
|PS_ROUTING_DISPLAY_NAME  <br/> |Содержит свойства строки, используемые в качестве имен отображения.  <br/> |
|PS_ROUTING_EMAIL_ADDRESSES  <br/> |Содержит свойства строки, используемые в качестве адресов электронной почты.  <br/> |
|PS_ROUTING_ADDRTYPE  <br/> |Содержит свойства строки, используемые в качестве типов адресов электронной почты.  <br/> |
|PS_ROUTING_ENTRYID  <br/> |Содержит двоичные свойства, используемые в качестве долгосрочных идентификаторов записи.  <br/> |
|PS_ROUTING_SEARCH_KEY  <br/> |Содержит двоичные свойства, используемые в качестве ключей поиска.  <br/> |
