# <a name="pidlidfax2originaldisplayname-canonical-property"></a>Каноническое свойство PidLidFax2OriginalDisplayName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает исходное отображаемое имя домашнего Факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax2OriginalDisplayName  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000080C4  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство, если оно задано, должно иметь то же значение, что и свойство **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).
