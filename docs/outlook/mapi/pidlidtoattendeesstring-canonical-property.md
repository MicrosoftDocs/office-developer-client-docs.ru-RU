# <a name="pidlidtoattendeesstring-canonical-property"></a>Каноническое свойство PidLidToAttendeesString

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит список всех отправимых участников, которые также являются требуемыми участниками.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidToAttendeesString  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x0000823B  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Значение для каждого участника — **PR_DISPLAY_NAME** [(PidTagDisplayName)](pidtagdisplayname-canonical-property.md)адресной книги участника. Отдельные записи должны быть делимитированы полуколоном, за которым следует пробел. Это свойство не требуется.
