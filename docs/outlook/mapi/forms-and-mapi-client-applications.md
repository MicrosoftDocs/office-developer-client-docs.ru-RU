# <a name="forms-and-mapi-client-applications"></a>Формы и клиентские приложения MAPI

**Область применения**: Outlook 2013 | Outlook 2016 
  
Поскольку все сообщения MAPI являются экземплярами форм, клиентские приложения отображаются и взаимодействуют с объектами форм так же, как и с другими объектами сообщений.
  
Клиентские приложения, которые используют интерфейсы форм MAPI, могут обеспечить дополнительное взаимодействие с объектами форм. Клиентские приложения могут использовать специальные глаголы, определенные формой, чтобы предоставить пользователям полный диапазон взаимодействий, предназначенных конструктором формы. Клиенты также могут использовать любые настраиваемые свойства, которые предоставляет сервер форм, чтобы помочь пользователям сортировать, просматривать или иным образом управлять формами в интерфейсе клиента.
  
Дополнительные сведения см. в [переработчике форм MAPI.](handling-mapi-forms.md)
