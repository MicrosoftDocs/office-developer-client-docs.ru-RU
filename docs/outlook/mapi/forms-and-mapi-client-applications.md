# <a name="forms-and-mapi-client-applications"></a>Формы и клиентских приложений MAPI

**Применимо к**: Outlook 2013 | Outlook 2016 
  
Так как все сообщения MAPI экземпляры форм, клиентские приложения отображения и взаимодействия с объектам формы так же, как и с другими объектами сообщения.
  
Клиентские приложения, использующие интерфейсы формы MAPI может предоставить дополнительные взаимодействия с объектами формы. Клиентские приложения могут использовать любой особые действия, определенные в форме для обеспечения пользователей весь спектр возможностей взаимодействия предназначен конструктора форм. Клиенты могут также использовать все настраиваемые свойства, предоставляемые сервера форм, чтобы помочь пользователям сортировки, предварительного просмотра или в противном случае — Управление формами в интерфейсе клиента.
  
Для получения дополнительных сведений см. [Обработка формы MAPI](handling-mapi-forms.md).
