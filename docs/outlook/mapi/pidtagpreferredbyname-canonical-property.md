# <a name="pidtagpreferredbyname-canonical-property"></a>Каноническое свойство PidTagPreferredByName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя получателя предпочитаемом вызов.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PREFERRED_BY_NAME, PR_PREFERRED_BY_NAME_A, PR_PREFERRED_BY_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A47  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
