# <a name="pidlidtaskfcreator-canonical-property"></a>Каноническое свойство PidLidTaskFCreator

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, что задача была создана с помощью текущего пользователя или агент пользователя, а не при обработке запроса на задачу.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskFCreator  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000811E  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Замечания

Клиент этому свойству значение TRUE, когда пользователь создает задачу и значение FALSE при назначении задачи другим пользователем. Если оставить это свойство определено, предполагается значение TRUE.
