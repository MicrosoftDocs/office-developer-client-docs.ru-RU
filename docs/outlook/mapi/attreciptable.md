# <a name="attreciptable"></a>attRecipTable

**Относится к**: Outlook 2013 | Outlook 2016 
  
При кодировании таблицы получателей каждый получатель кодируется как строка свойств MAPI. Формат выглядит следующим образом: 
  
_Ров_сек:_
  
>  _число строк_ _Проперти_секуенце,_... 
    
Формат _проперти_сек_ такой же, как и в атрибуте [аттмапипропс](attmapiprops.md) .
