# <a name="pidtagadditionalrenentryidsex-canonical-property"></a>Каноническое свойство PidTagAdditionalRenEntryIdsEx

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификаторы записей специальных папок для объекта Store. Каждая запись в этом многозначном свойстве может быть сопоставлена с одним или несколькими идентификаторами записей, то есть отношение "один ко многим" между записью и ее идентификаторами записи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_АДДИТИОНАЛ_РЕН_ЕНТРИДС_ЕКС  <br/> |
|Идентификатор:  <br/> |0x36D9  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Приложение Outlook  <br/> |
   
## <a name="remarks"></a>Комментарии

Если используется это свойство, оно содержит массив блоков, указывающий идентификаторы записей для папок. Блоки соответствуют формату, указанному в следующих четырех таблицах.
  
**Блок Персистдата**

|**Name**|**Type**|**Size**|**Описание**|
|:-----|:-----|:-----|:-----|
|**Персистид** <br/> |WORD  <br/> |2  <br/> |Значение идентификатора типа для этой записи **персистдата** . Список допустимых значений представлен в таблице "значения Персистблокктипе".  <br/> |
|**Датаелементссизе** <br/> |WORD  <br/> |2  <br/> |Размер (в байтах) поля " **элементы** ".  <br/> |
|**Элементы с элементами** <br/> |массив блоков **персистелемент**  <br/> |переменная  <br/> |Указывает, сколько записей **персистелемент** существует для магазина. В таблице "блок Персистелемент" приведен формат этой структуры.  <br/> |
   
**Значения Персистблокктипе**

|**Имя**|**Value**|**Описание**|
|:-----|:-----|:-----|
|ПЕРСИСТ_СЕНТИНЕЛ  <br/> |0x0000  <br/> |Указывает, что больше не будут обрабатываться блоки **персистдата** .  <br/> |
|РСФ_ПИД_РСС_СУБСКРИПТИОН  <br/> |0x8001  <br/> |Указывает на то, что этот блок содержит данные для папки "подписки на RSS-канал".  <br/> |
|РСФ_ПИД_СЕНД_АНД_ТРАКК  <br/> |0x8002  <br/> |Указывает на то, что этот блок содержит данные для папки обработки отслеживаемой почты.  <br/> |
|РСФ_ПИД_ТОДО_СЕАРЧ  <br/> |0x8004  <br/> |Указывает на то, что этот блок содержит данные для папки поиска дел.  <br/> |
|РСФ_ПИД_КОНВ_АКТИОНС  <br/> |0x8006  <br/> |Указывает на то, что этот блок содержит данные для папки "Параметры действия беседы".  <br/> |
|РСФ_ПИД_КОМБИНЕД_АКТИОНС  <br/> |0x8007  <br/> |Это значение зарезервировано.  <br/> |
|РСФ_ПИД_СУГЖЕСТЕД_КОНТАКТС  <br/> |0x8008  <br/> |Указывает на то, что этот блок содержит данные для папки предложенных контактов.  <br/> |
|РСФ_ПИД_КОНТАКТ_СЕАРЧ  <br/> |0x8009  <br/> |Указывает на то, что этот блок содержит данные для папки поиска Contacts.  <br/> Используется только Outlook.  <br/> |
|РСФ_ПИД_БУДДИЛИСТ_ПДЛС  <br/> |0x800A  <br/> |Указывает на то, что этот блок содержит данные для папки контактов для обмена мгновенными сообщениями (IM). Указанная папка содержит персональные списки рассылки (Пдлс), представляющие каждую группу в списке контактов для обмена МГНОВЕНными СООБЩЕНИЯми.  <br/> Используется как в Outlook, так и в Exchange.  <br/> |
|РСФ_ПИД_БУДДИЛИСТ_КОНТАКТС  <br/> |0x800B  <br/> |Указывает на то, что этот блок содержит данные для папки "Контакты для обмена МГНОВЕНными СООБЩЕНИЯми". Указанная папка содержит отдельные контакты, на которые ссылаются группы списка контактов для обмена МГНОВЕНными СООБЩЕНИЯми.  <br/> Используется как в Outlook, так и в Exchange.  <br/> |
   
Если значение **персистблокктипе** не определено в этом разделе, то блок **персистдата** игнорируется и обработка продолжается до тех пор, пока не будет обработана персист_сентинел **персистид** или не будет достигнут конец потока. 
  
**Персистелементблокк**

|**Name**|**Type**|**Size**|**Описание**|
|:-----|:-----|:-----|:-----|
|**ElementID** <br/> |WORD  <br/> |2  <br/> |Задает значение идентификатора типа для этого блока **персистелемент** . Список допустимых значений представлен в таблице "значения Персистелементтипе".  <br/> |
|**Елементдатасизе** <br/> |WORD  <br/> |2  <br/> |Задает размер (в байтах) поля **елементдата** .  <br/> |
|**Елементдата** <br/> |массив двоичных данных  <br/> |переменная  <br/> |Содержит данные для этой **персистид** + **ElementID** .  <br/> |
   
**Значения Персистелементтипе**

|**Имя**|**Value**|**Значение Елементдатасизе**|**Описание**|
|:-----|:-----|:-----|:-----|
|РСФ_ЕЛИД_ХЕАДЕР  <br/> |0x0002  <br/> |0x0004  <br/> |Указывает, что поле **елементдата** этого блока содержит значение заголовка DWORD. Способ интерпретации этого значения зависит от типа **персистид** блока.  <br/> Для всех типов **персистид** , указанных в [[MS-оксосфлд]](https://msdn.microsoft.com/library/a60e9c16-2ba8-424b-b60c-385a8a2837cb.aspx), это значение равно нулю.  <br/> |
|РСФ_ЕЛИД_ЕНТРИД  <br/> |0x0001  <br/> |переменная  <br/> |Указывает на то, что этот блок содержит идентификатор **EntryID** для папки, заданной параметром **персистид**.  <br/> |
|ЕЛЕМЕНТ_СЕНТИНЕЛ  <br/> |0x0000  <br/> |0x0000  <br/> |Указывает, что больше не будут обрабатываться блоки **персистелемент** .  <br/> |
   
Если значение **персистелементтипе** не определено в этом разделе, то блок **персистелемент** игнорируется и обработка продолжается до тех пор, пока не будет обработана елемент_сентинел **ElementID** или не будет достигнут конец потока.
