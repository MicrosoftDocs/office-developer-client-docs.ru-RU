# <a name="saving-frequently-used-properties"></a>Сохранение часто используемых свойств

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Повышение производительности за счет хранения данных, для получения и частого доступа к ним требуется время и ресурсы. Использование дополнительной памяти иногда может быть более лучшим вариантом, когда повторялись повторы. При создании кэша для хранения этих данных следует соблюдать осторожность и осторожность. Помните, что плохо разработанный кэш может отрицательно сказаться на производительности.
  
Например, большинству клиентов межличностных сообщений (IPM) необходимо много раз отобразить и открыть поддереть IPM папок во время типичного сеанса. Для повышения производительности можно хранить идентификаторы записей для каждой из этих папок.
