# <a name="saving-frequently-used-properties"></a>Сохранение часто используемых свойств

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Повышение производительности путем хранения данных, которые требуется время и ресурсы для получения и частого доступа. Использование дополнительной памяти иногда может быть более лучшим вариантом, который повторяет повторяемую повторную выдачу. При создании кэша для хранения этих данных используйте осторожность и осторожность. Помните, что плохо разработанный кэш может отрицательно повлиять на производительность.
  
Например, большинству клиентов межличностного сообщения (IPM) во время типичного сеанса необходимо много раз отображать и открывать подтриму IPM-папки. Вы можете повысить производительность, храня идентификаторы записей для каждой из этих папок.
