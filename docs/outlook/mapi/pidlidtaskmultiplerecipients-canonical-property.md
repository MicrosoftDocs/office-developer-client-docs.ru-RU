# <a name="pidlidtaskmultiplerecipients-canonical-property"></a>Каноническое свойство PidLidTaskMultipleRecipients

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Предоставляет подсказки оптимизации о получателях задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskMultRecips  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008120  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Примечания

Если за установлено, этому свойству необходимо установить по битовую операцию **OR,** которая может иметь ноль или больше следующих значений. 
  
|**Name**|**Value**|**Описание**|
|:-----|:-----|:-----|
|Sent  <br/> |0x00000001  <br/> |Задача имеет несколько основных получателей.  <br/> |
|ПОЛУЧЕНО  <br/> |0x00000002  <br/> |Несмотря на то что подсказка "Отправлено" не была представлена, клиент обнаружил, что задача имеет несколько основных получателей.  <br/> |
|Reserved  <br/> |0x00000004  <br/> |Это значение зарезервировано.  <br/> |
