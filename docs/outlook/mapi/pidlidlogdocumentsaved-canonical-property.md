# <a name="pidlidlogdocumentsaved-canonical-property"></a>Каноническое свойство PidLidLogDocumentSaved

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, был ли документ сохранен во время ведения журнала.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидлогдоксавед  <br/> |
|Набор свойств:  <br/> |PSETID_Log  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000870F  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Журнал  <br/> |
