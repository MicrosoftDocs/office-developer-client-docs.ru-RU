# <a name="pidtagdistributionlistexpansionprohibited-canonical-property"></a>Каноническое свойство PidTagDistributionListExpansionProhibited

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение TRUE, если агент передачи сообщений (MTA) запрещает расширять списки рассылки.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ДЛ_ЕКСПАНСИОН_ПРОХИБИТЕД  <br/> |
|Идентификатор:  <br/> |0x0014  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
