# <a name="pidtagdistributionlistexpansionhistory-canonical-property"></a>Каноническое свойство PidTagDistributionListExpansionHistory

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит историю, показывающая расширение списка рассылки во время передачи сообщений. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DL_EXPANSION_HISTORY  <br/> |
|Идентификатор:  <br/> |0x0013  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство доступно для получения клиентских приложений, если поставщик транспорта установил его. Он также доступен для отправляемого клиента, если содержимое сообщения возвращается с отчетом.
