# <a name="pidtagdistributionlistexpansionhistory-canonical-property"></a>Каноническое свойство PidTagDistributionListExpansionHistory

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит журнал, в котором показано, как список рассылки был расширен во время передачи сообщения. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ДЛ_ЕКСПАНСИОН_ХИСТОРИ  <br/> |
|Идентификатор:  <br/> |0x0013  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство доступно для получения клиентских приложений, если оно задано поставщиком транспорта. Он также доступен для отправляющего клиента, если содержимое сообщения возвращается с отчетом.
