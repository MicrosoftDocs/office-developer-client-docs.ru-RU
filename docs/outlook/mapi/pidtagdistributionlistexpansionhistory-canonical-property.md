# <a name="pidtagdistributionlistexpansionhistory-canonical-property"></a>Каноническое свойство PidTagDistributionListExpansionHistory

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит журнал, в котором показано, как в список рассылки дополнены во время передачи сообщений. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DL_EXPANSION_HISTORY  <br/> |
|Идентификатор:  <br/> |0x0013  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство доступно для получения клиентских приложений, если установлен поставщика транспорта. Это также доступны для клиента, если содержимое сообщения, возвращаемых в отчет.
