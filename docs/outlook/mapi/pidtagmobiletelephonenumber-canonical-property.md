# <a name="pidtagmobiletelephonenumber-canonical-property"></a>Каноническое свойство PidTagMobileTelephoneNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит номер сотового телефона получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_MOBILE_TELEPHONE_NUMBER, PR_MOBILE_TELEPHONE_NUMBER_A, PR_MOBILE_TELEPHONE_NUMBER_W, PR_CELLULAR_TELEPHONE_NUMBER, PR_CELLULAR_TELEPHONE_NUMBER_A, PR_CELLULAR_TELEPHONE_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A1C  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Адресная книга  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют идентификацию и сведения о доступе для получателя. Они определяются получателем и организацией.
