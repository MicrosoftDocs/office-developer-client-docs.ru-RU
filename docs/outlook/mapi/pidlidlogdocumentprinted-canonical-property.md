# <a name="pidlidlogdocumentprinted-canonical-property"></a>Каноническое свойство PidLidLogDocumentPrinted

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, был ли документ напечатан во время журнала.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidLogDocPrinted  <br/> |
|Набор свойств:  <br/> |PSETID_Log  <br/> |
|Long ID (LID):  <br/> |0x0000870E  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Журнал  <br/> |
