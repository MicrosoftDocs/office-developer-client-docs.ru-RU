# <a name="pidnamexsharingproviderguid-canonical-property"></a>Каноническое свойство PidNameXSharingProviderGuid

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Представляет значение свойства **диспидшарингпровидергуид** ([PidLidSharingProviderGuid](pidlidsharingproviderguid-canonical-property.md)).
  
|||
|:-----|:-----|
|Понятные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Имя свойства:  <br/> |X – Sharing: provider — GUID  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общий доступ  <br/> |
