# <a name="pidtagotheraddresscity-canonical-property"></a>Каноническое свойство PidTagOtherAddressCity

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя электронной почты пользователя другие расположения, такие как город или города.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_OTHER_ADDRESS_CITY, PR_OTHER_ADDRESS_CITY_A, PR_OTHER_ADDRESS_CITY_W  <br/> |
|Идентификатор:  <br/> |0x3A5F  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства определяются пользователем или организации пользователя.
