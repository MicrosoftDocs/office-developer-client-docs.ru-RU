# <a name="pidlidreminderfileparameter-canonical-property"></a>Каноническое свойство PidLidReminderFileParameter

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает имя файла звукового сопровождения, которое должен воспринимать клиент, когда напоминание для этого объекта будет просрочено.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидреминдерфилепарам  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000851F  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Напоминание  <br/> |
   
## <a name="remarks"></a>Примечания

Если это свойство отсутствует, клиент может использовать значение по умолчанию.
