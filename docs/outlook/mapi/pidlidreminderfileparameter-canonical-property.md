# <a name="pidlidreminderfileparameter-canonical-property"></a>Каноническое свойство PidLidReminderFileParameter

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает имя файла звук, клиент должен воспроизводиться мере просроченные оповещение для этого объекта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidReminderFileParam  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000851F  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Reminder  <br/> |
   
## <a name="remarks"></a>Замечания

Если это свойство не задан, клиент может использовать значение по умолчанию.
