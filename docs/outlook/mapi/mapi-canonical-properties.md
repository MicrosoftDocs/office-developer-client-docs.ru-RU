и [PidNamePhishingStamp](pidnamephishingstamp-canonical-property.md) представляет "," именуемого свойства, которое имеет строковое имя и которое указывает строку пометки сообщений, которые могут быть http://schemas.microsoft.com/outlook/phishingstamp фишингом.</span><span class="sxs-lookup"><span data-stu-id="50178-111">For example, [PidTagAccount](pidtagaccount-canonical-property.md) represents the tagged properties, **PR_ACCOUNT** ([PidTagAccount](pidtagaccount-canonical-property.md)), **PR_ACCOUNT_A** ([PidTagAccount](pidtagaccount-canonical-property.md)), and **PR_ACCOUNT_W** ([PidTagAccount](pidtagaccount-canonical-property.md)), that specify a recipient's account name; [PidLidContacts](pidlidcontacts-canonical-property.md) represents the **dispidContacts** property, a named property that has a numerical identifier and that specifies the name of contacts associated with a message; and [PidNamePhishingStamp](pidnamephishingstamp-canonical-property.md) represents "http://schemas.microsoft.com/outlook/phishingstamp," a named property that has a string name, and that specifies the string marking messages that are likely to be phishing.</span></span>

### <a name="identifying-properties-in-mapi"></a><span data-ttu-id="50178-113">Определение свойств в MAPI</span><span class="sxs-lookup"><span data-stu-id="50178-113">Identifying Properties in MAPI</span></span>

<span data-ttu-id="50178-114">Все свойства в MAPI идентифицируются идентификатором свойства, который является неподписаным 16-битным значением.</span><span class="sxs-lookup"><span data-stu-id="50178-114">All properties in MAPI are identified by a property identifier that is an unsigned 16-bit value.</span></span> <span data-ttu-id="50178-115">Идентификатор свойства и тип свойства (другое неподписаное 16-битное значение) составляют тег свойства.</span><span class="sxs-lookup"><span data-stu-id="50178-115">The property identifier and the property type (another unsigned 16-bit value) constitute a property tag for the property.</span></span> 
  
<span data-ttu-id="50178-116">MAPI использует тег свойства для уникального определения свойств.</span><span class="sxs-lookup"><span data-stu-id="50178-116">MAPI uses a property tag to uniquely define properties.</span></span> <span data-ttu-id="50178-117">Свойства с одинаковыми тегами свойств, например **PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber)](pidtagbusiness2telephonenumber-canonical-property.md)и **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber),](pidtagbusiness2telephonenumber-canonical-property.md)считаются идентичными свойствами в MAPI.</span><span class="sxs-lookup"><span data-stu-id="50178-117">Properties that have the same property tag, like **PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)) and **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)), are considered identical properties in MAPI.</span></span> <span data-ttu-id="50178-118">Список тегов свойств, определенных MAPI для собственных свойств, см. в файле загона MAPI Mapitags.h.</span><span class="sxs-lookup"><span data-stu-id="50178-118">For a list of property tags that MAPI has defined for its own properties, see the MAPI header file, Mapitags.h.</span></span>
  
<span data-ttu-id="50178-119">Обратите внимание, что MAPI делит идентификаторы свойств на диапазоны.</span><span class="sxs-lookup"><span data-stu-id="50178-119">Note that MAPI divides property identifiers into ranges.</span></span> <span data-ttu-id="50178-120">Место, в котором идентификатор попадает в диапазон, указывает на его использование и право собственности.</span><span class="sxs-lookup"><span data-stu-id="50178-120">Where an identifier falls in the range indicates its use and ownership.</span></span> <span data-ttu-id="50178-121">Идентификаторы свойств помеченных свойств попадают в диапазоне 0x0001 0x7FFF.</span><span class="sxs-lookup"><span data-stu-id="50178-121">The property identifiers of tagged properties fall in the range of 0x0001 to 0x7FFF.</span></span> <span data-ttu-id="50178-122">В этом диапазоне находятся идентификаторы свойств, определяемого MAPI, которые находятся в диапазоне от 0x0001 до 0x3FFF.</span><span class="sxs-lookup"><span data-stu-id="50178-122">Within this range are the property identifiers of MAPI-defined properties, which fall in the range of 0x0001 to 0x3FFF.</span></span> <span data-ttu-id="50178-123">Идентификаторы именуемого свойства попадают в диапазон от 0x8000 до 0x8FFF.</span><span class="sxs-lookup"><span data-stu-id="50178-123">The property identifiers of named properties fall in the range from 0x8000 to 0x8FFF.</span></span> 
  
<span data-ttu-id="50178-124">Именуются также набором свойств, а также длинным именем (32-битным значением, неподписаным 32-битным) или строкой.</span><span class="sxs-lookup"><span data-stu-id="50178-124">Named properties are additionally attributed by a property set, and either a long ID (LID), which is an unsigned 32-bit value, or a string.</span></span> <span data-ttu-id="50178-125">Набор свойств — это GUID, который идентифицирует группу именуемых свойств с аналогичной целью.</span><span class="sxs-lookup"><span data-stu-id="50178-125">A property set is a GUID that identifies a group of named properties with a similar purpose.</span></span> <span data-ttu-id="50178-126">Набор свойств и имя строки ИЛИ ЛИД используются для получения или задайте именуемую свойство.</span><span class="sxs-lookup"><span data-stu-id="50178-126">The property set and LID or string name are used to get or set the named property.</span></span>
