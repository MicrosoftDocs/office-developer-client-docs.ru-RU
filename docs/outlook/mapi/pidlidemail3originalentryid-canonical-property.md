# <a name="pidlidemail3originalentryid-canonical-property"></a>Каноническое свойство PidLidEmail3OriginalEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает **entryId объекта,** соответствующего третьему адресу электронной почты. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail3OriginalEntryID  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080A5  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно быть как разовый **EntryId** для этого электронного адреса, так и допустимый объект Address Book **EntryId**.
