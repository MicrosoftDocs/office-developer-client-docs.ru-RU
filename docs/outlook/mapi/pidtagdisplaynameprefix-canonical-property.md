# <a name="pidtagdisplaynameprefix-canonical-property"></a>Каноническое свойство PidTagDisplayNamePrefix

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит префикс отображаемого имени (например, промахов, Mr., Mrs.) для пользователя обмена сообщениями. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DISPLAY_NAME_PREFIX, PR_DISPLAY_NAME_PREFIX_A PR_DISPLAY_NAME_PREFIX_W  <br/> |
|Идентификатор:  <br/> |0x3A45  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства обеспечивают идентификацию и доступ к информации о пользователе для обмена сообщениями. Содержимое определяется пользователем обмена сообщениями и организацией пользователя обмена сообщениями.
