# <a name="pidlidemail3emailaddress-canonical-property"></a>Каноническое свойство PidLidEmail3EmailAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Задает третий адрес электронной почты контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidEmail3EmailAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000080A3  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Значение этого свойства должно соответствовать типу адреса, указанному для этого адреса электронной почты.
