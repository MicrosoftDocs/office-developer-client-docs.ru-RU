# <a name="pidlidimapdeleted-canonical-property"></a>Каноническое свойство PidLidImapDeleted

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Обозначает элементы протокола доступа к интернет-почте (IMAP), помеченные для удаления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidImapDeleted  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008570  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |IMAP  <br/> |
   
## <a name="remarks"></a>Примечания

Если установлено значение nonzero, элемент помечен для удаления.
