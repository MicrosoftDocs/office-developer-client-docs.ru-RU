# <a name="pidlidimapdeleted-canonical-property"></a>Каноническое свойство PidLidImapDeleted

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает элементы протокола IMAP, помеченные для удаления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидимапделетед  <br/> |
|Набор свойств:  <br/> |Псетид_коммон  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008570  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |IMAP  <br/> |
   
## <a name="remarks"></a>Замечания

Если задано ненулевое значение, элемент помечен для удаления.
