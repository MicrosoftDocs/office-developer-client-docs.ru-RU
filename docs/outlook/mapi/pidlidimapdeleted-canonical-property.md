# <a name="pidlidimapdeleted-canonical-property"></a>Каноническое свойство PidLidImapDeleted

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Обозначает протокол доступа почты Интернета (IMAP) элементов, помеченных для удаления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidImapDeleted  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008570  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |IMAP  <br/> |
   
## <a name="remarks"></a>Замечания

Если параметр имеет значение ненулевое значение, элемент был помечен для удаления.
