# <a name="table-positioning"></a>Размещение таблиц

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Курсор всегда указывает текущую позицию в таблице. Один курсор для каждого представления таблицы. его значение с разработчик таблицы. Когда клиента или поставщика, с помощью таблицы вызывает изменение положения таблицы, сбрасывается значение курсора. Положение таблицы можно изменить с помощью:
  
- Закладка.
    
- Дробное значение.
    
- Фильтр.
