# <a name="table-positioning"></a>Расположение таблицы

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Текущее положение в таблице всегда указывается курсором. Существует один курсор для каждого представления таблицы; его значение занося в реализацию таблицы. Когда клиент или поставщик услуг, использующий таблицу, совершает вызов для изменения положения таблицы, сбрасывается значение курсора. Положение таблицы можно изменить с помощью:
  
- Закладка.
    
- Дробное значение.
    
- Фильтр.
