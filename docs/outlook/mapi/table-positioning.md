# <a name="table-positioning"></a>Позиционирование таблицы

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Текущая позиция в таблице всегда указывается курсором. Для каждого представления таблицы существует один курсор; его значение задается средством реализации таблицы. Когда клиент или поставщик услуг, использующий таблицу, выдает запрос на изменение позиции таблицы, значение курсора сбрасывается. Положение таблицы можно изменить с помощью следующих компонентов:
  
- Закладка.
    
- Дробное значение.
    
- Фильтр.
