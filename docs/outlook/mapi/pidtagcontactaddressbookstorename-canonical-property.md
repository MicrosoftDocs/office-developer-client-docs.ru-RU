# <a name="pidtagcontactaddressbookstorename-canonical-property"></a>Каноническое свойство PidTagContactAddressBookStoreName

  
  
**Относится к**: Outlook 
  
Содержит отображаемое имя хранилища, связанного с поставщиком адресной книги. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONTAB_STORE_NAME, PR_CONTAB_STORE_NAME_A, PR_CONTAB_STORE_NAME_W  <br/> |
|Идентификатор:  <br/> |0x6612  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Адресная книга контактов  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства доступны на контейнер контактов адресной книги и столбца в таблице контейнеров контактов адресной книги.
