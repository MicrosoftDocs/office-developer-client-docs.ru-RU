# <a name="pidtagcontactaddressbookstorename-canonical-property"></a>Каноническое свойство PidTagContactAddressBookStoreName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит отображаемое имя хранилища, связанного с поставщиком адресной книги. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КОНТАБ_СТОРЕ_НАМЕ, ПР_КОНТАБ_СТОРЕ_НАМЕ_А, ПР_КОНТАБ_СТОРЕ_НАМЕ_В  <br/> |
|Идентификатор:  <br/> |0x6612  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Адресная книга контактов  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства находятся в контейнере адресной книги контактов и в таблице контейнеров адресных книг контактов.
