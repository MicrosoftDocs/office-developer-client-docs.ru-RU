# <a name="pidlidallattendeesstring-canonical-property"></a>Каноническое свойство PidLidAllAttendeesString

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает список всех участников, за исключением организатора, включая ресурсы и неисключаемых участников.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidAllAttendeesString  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008238  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Значение для каждого участника — отображаемая фамилия участника. Отдельные записи должны быть делимитированы полуколоном, за которым следует пробел. Это свойство не требуется.
