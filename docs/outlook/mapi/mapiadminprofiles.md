# <a name="mapiadminprofiles"></a>MAPIAdminProfiles

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Создает объект администрирования профиля. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapix.h  <br/> |
|Реализовано в:  <br/> |MAPI  <br/> |
|Вызывающая сторона:  <br/> |Клиентские приложения  <br/> |
   
```cpp
HRESULT MAPIAdminProfiles(
  ULONG ulFlags,
  LPPROFADMIN FAR * lppProfAdmin
);
```

## <a name="parameters"></a>Параметры

 _ulFlags_
  
> [in] Битоваяmas флагов, указывающих параметры функции записи службы. 
    
 _lppProfAdmin_
  
> [out] Указатель на указатель на новый объект администрирования профиля.
    
## <a name="return-value"></a>Возвращаемое значение

S_OK
