# <a name="pidlidformstorage-canonical-property"></a>Каноническое свойство PidLidFormStorage

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Представляет свойства формы, связанный с сообщением.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFormStorage  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000850F  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Конфигурация во время выполнения  <br/> |
