# <a name="pidlidtaskdatecompleted-canonical-property"></a>Каноническое свойство PidLidTaskDateCompleted

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату завершения задачи пользователем.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидтаскдатекомплетед  <br/> |
|Набор свойств:  <br/> |Псетид_таск  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000810F  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Замечания

Если этот параметр задан, то у этого свойства должен быть компонент времени полуночи в местном часовом поясе, преобразованный в формат UTC.
