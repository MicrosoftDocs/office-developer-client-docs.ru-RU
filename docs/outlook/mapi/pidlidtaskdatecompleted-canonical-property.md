# <a name="pidlidtaskdatecompleted-canonical-property"></a>Каноническое свойство PidLidTaskDateCompleted

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату выполнения задачи пользователем.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskDateCompleted  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x0000810F  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Примечания

Если установлено, это свойство должно иметь компонент времени полуночи в локальном часовом поясе, преобразованный в скоординированное универсальное время (UTC).
