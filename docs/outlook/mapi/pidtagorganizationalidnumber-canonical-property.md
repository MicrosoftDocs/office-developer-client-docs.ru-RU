# <a name="pidtagorganizationalidnumber-canonical-property"></a>Каноническое свойство PidTagOrganizationalIdNumber

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит организационный ID-номер для контакта, например номер сотрудника.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORGANIZATIONAL_ID_NUMBER, PR_ORGANIZATIONAL_ID_NUMBER_A, PR_ORGANIZATIONAL_ID_NUMBER_W  <br/> |
|Идентификатор:  <br/> |0x3A10  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Адрес  <br/> |
   
## <a name="remarks"></a>Примечания

Это необязательные свойства. Их использование определяется пользователем или организацией обмена сообщениями.
