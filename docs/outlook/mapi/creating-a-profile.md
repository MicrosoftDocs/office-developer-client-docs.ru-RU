# <a name="creating-a-profile"></a>Создание профиля

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Клиент может создать профиль одним из двух способов: с помощью мастера профилей или путем написания пользовательского кода.
  
## <a name="in-this-section"></a>Содержание

[Создание профиля с помощью мастера профилей](creating-a-profile-by-using-the-profile-wizard.md)
  
> В этой статье описывается, как вызвать мастер профилей — приложение, которое создает профиль, позволяя пользователю выбирать службы сообщений и поставщики услуг в интерактивном режиме.
    
[Создание профиля с помощью пользовательского кода](creating-a-profile-by-using-custom-code.md)
  
> Описывается, как писать код C или C++, часто лучший выбор для неинтерактивных клиентских приложений, которым требуется определенный набор служб сообщений.
