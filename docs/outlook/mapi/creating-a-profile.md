# <a name="creating-a-profile"></a>Создание профиля

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Клиент может создать профиль одним из двух способов, используя мастер профиля или написав настраиваемый код.
  
## <a name="in-this-section"></a>В этом разделе

[Создание профиля с помощью мастера профилей](creating-a-profile-by-using-the-profile-wizard.md)
  
> Описывает, как вызвать мастер профилей , приложение, которое создает профиль, позволяя пользователю выбирать службы сообщений и поставщиков услуг интерактивно.
    
[Создание профиля с помощью пользовательского кода](creating-a-profile-by-using-custom-code.md)
  
> Описывает написание кода C или C++ , что часто является оптимальным выбором для неинтерактивных клиентских приложений, для которых требуется определенный набор служб сообщений.
