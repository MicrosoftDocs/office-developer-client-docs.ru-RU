# <a name="pidlidfax3displayname-canonical-property"></a>Каноническое свойство PidLidFax3DisplayName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает отображаемое имя другого Факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax3DisplayName  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000080D0  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
