# <a name="pidlidfax3emailtype-canonical-property"></a>Каноническое свойство PidLidFax3EmailType

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает тип электронной почты другого Факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax3EmailType  <br/> |
|Набор свойств:  <br/> |Псетид_аддресс  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000080D7  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
