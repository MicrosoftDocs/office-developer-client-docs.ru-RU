# <a name="pidlidworkaddresscountrycode-canonical-property"></a>Каноническое свойство PidLidWorkAddressCountryCode

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает часть кода страны в рабочий адрес контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidWorkAddressCountryCode  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x000080DB  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
