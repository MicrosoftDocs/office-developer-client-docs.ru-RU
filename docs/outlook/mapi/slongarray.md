# <a name="slongarray"></a>SLongArray

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит массив типов ДЛИННых значений, которые используются для описания свойства типа PT_MV_LONG. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
   
```cpp
typedef struct _SLongArray
{
  ULONG cValues;
  LONG FAR *lpl;
} SLongArray;

```

## <a name="members"></a>"Участники"

 **квалуес**
  
> Количество значений в массиве, на которое указывает элемент **ЛПЛ** . 
    
 **лпл**
  
> Указатель на массив ДЛИННых значений.

Дополнительные сведения о PT_MV_LONG приведены в разделе [список типов свойств](property-types.md).
