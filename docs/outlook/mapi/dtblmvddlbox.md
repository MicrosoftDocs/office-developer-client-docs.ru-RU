# <a name="dtblmvddlbox"></a>DTBLMVDDLBOX

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Описывает раскрывающийся список, который будет использоваться в диалоговом окне, созданном из таблицы отображения.
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
   
```cpp
typedef struct _DTBLMVDDLBX
{
  ULONG ulFlags;
  ULONG ulMVPropTag;
} DTBLMVDDLBX, FAR * LPDTBLMVDDLBX;

```

## <a name="members"></a>Элементы

 **ulFlags**
  
> Резервирования должно быть равно нулю.
    
 **Улмвпроптаг**
  
> Тег свойства с многозначным свойством типа ПТ_МВ_ТСТРИНГ. Различные значения этого свойства отображаются в раскрывающемся списке как отдельные записи.
    
## <a name="remarks"></a>Примечания

Структура **дтблмвддлбокс** описывает раскрывающийся список, поддерживающий несколько значений, список элементов, предназначенный только для чтения. При использовании раскрывающегося списка с несколькими значениями значения отображаются, когда пользователь нажимает полосу прокрутки. 
  
Отображаемые данные берутся из свойства, указанного в элементе **улмвпроптаг** . Нет необходимости считывать сведения из интерфейса свойства, связанного с таблицей отображения. Кроме того, так как пользователи не могут выбирать из этих типов списков, данные не записываются в интерфейс свойств. 
  
В раскрывающемся списке с несколькими значениями поддерживаются только многозначные строковые свойства; другие многозначные типы свойств не поддерживаются. 
  
Общие сведения о отображаемых таблицах приведены в разделе [Display Tables](display-tables.md). Сведения о том, как реализовать таблицу отображения, можно найти [в статье реализация таблицы отображения](display-table-implementation.md).
