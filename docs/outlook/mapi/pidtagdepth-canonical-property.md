# <a name="pidtagdepth-canonical-property"></a>Каноническое свойство PidTagDepth

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит integer, который представляет относительный уровень отступа или глубину объекта в таблице иерархии.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DEPTH  <br/> |
|Идентификатор:  <br/> |0x3005  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |MAPI общие  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство также может указать уровень классификации строки в таблице контента или глубину иерархии в таблице иерархии. Глубина нулевой, где ноль представляет собой левую категорию. Во всех случаях значение свойства представляет относительное значение, а не абсолютное значение. Например, в таблице иерархии значение глубины относительно контейнера, из которого была извлечена таблица иерархии. Глубина не представляет абсолютной глубины из корневого контейнера.
