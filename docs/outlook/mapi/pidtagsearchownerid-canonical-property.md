# <a name="pidtagsearchownerid-canonical-property"></a>Каноническое свойство PidTagSearchOwnerId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит ID процесса отправки уведомлений об объектах, готовых к индексации.
  
## <a name="quick-info"></a>Краткие сведения

|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SEARCH_OWNER_ID  <br/> |
|Подвергается:  <br/> |Объект магазина сообщений  <br/> |
|Идентификатор:  <br/> |0x3419  <br/> |
|Тип свойства:  <br/> |PT_LONG  <br/> |
|Тег свойства:  <br/> |0x34190003  <br/> |
|Доступ:  <br/> |Только для чтения  <br/> |
