# <a name="pidtagsearchownerid-canonical-property"></a>Каноническое свойство PidTagSearchOwnerId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор процесса, отправляющего в индексатор уведомления об объектах, готовых для индексирования.
  
## <a name="quick-info"></a>Краткие сведения

|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_СЕАРЧ_ОВНЕР_ИД  <br/> |
|Предоставлено:  <br/> |Объект хранилища сообщений  <br/> |
|Идентификатор:  <br/> |0x3419  <br/> |
|Тип свойства:  <br/> |PT_LONG  <br/> |
|Тег свойства:  <br/> |0x34190003  <br/> |
|Обращения  <br/> |Только для чтения  <br/> |
