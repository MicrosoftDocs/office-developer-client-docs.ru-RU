# <a name="pidtagproviderdisplayname-canonical-property"></a>Каноническое свойство PidTagProviderDisplayName

  
  
**Относится к**: Outlook 
  
Содержит отображаемое имя поставщика.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PROVIDER_DISPLAY_NAME, PR_PROVIDER_DISPLAY_NAME_A, PR_PROVIDER_DISPLAY_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3418  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Хранение сообщений MAPI  <br/> |
