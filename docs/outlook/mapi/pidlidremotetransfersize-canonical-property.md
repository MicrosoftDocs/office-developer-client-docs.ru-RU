# <a name="pidlidremotetransfersize-canonical-property"></a>Каноническое свойство PidLidRemoteTransferSize

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает размер передачи для удаленного элемента (в байтах).
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |диспидремотексферсизе  <br/> |
|Набор свойств:  <br/> |PSETID_Remote  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008F05  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Удаленное сообщение  <br/> |
