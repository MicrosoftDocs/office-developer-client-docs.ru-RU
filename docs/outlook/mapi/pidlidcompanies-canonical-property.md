# <a name="pidlidcompanies-canonical-property"></a>Каноническое свойство PidLidCompanies

  
  
**Относится к**: Outlook 
  
Содержит имена компаний, связанных с элементом.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidCompanies  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008539  <br/> |
|Тип данных:  <br/> |PT_MV_UNICODE  <br/> |
|Область:  <br/> |Общие системы обмена сообщениями  <br/> |
