# <a name="cbnewsrowset"></a>CbNewSRowSet

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Вычисляет количество ветвей, выделяемого для новой структуры [SRowSet,](srowset.md) которая содержит указанное число строк, представленных структурами [SRow.](srow.md) 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |Mapidefs.h  <br/> |
|Связанная структура:  <br/> |**SRowSet** <br/> |
   
```cpp
CbNewSRowSet (_crow)
```

## <a name="parameters"></a>Параметры

 _ ___
  
> Количество структур **SRow,** указывающих количество строк, которые необходимо включить в новую структуру **SRowSet.**
