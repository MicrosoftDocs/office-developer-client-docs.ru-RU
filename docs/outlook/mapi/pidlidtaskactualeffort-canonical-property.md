# <a name="pidlidtaskactualeffort-canonical-property"></a>Каноническое свойство PidLidTaskActualEffort

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает количество минут, в течение которые пользователь выполнил задачу.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskActualEffort  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008110  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Примечания

Значение должно быть больше или равно 0 и меньше 0x5AE980DF (1 525 252 319), где 480 минут равно одному дню, а 2400 минут — одной неделе (восемь часов в рабочий день и пять дней в неделю).
