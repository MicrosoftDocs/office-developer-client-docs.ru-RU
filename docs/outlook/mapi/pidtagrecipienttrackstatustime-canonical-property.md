# <a name="pidtagrecipienttrackstatustime-canonical-property"></a>Каноническое свойство PidTagRecipientTrackStatusTime

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит дату и время ответа участника.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RECIPIENT_TRACKSTATUS_TIME  <br/> |
|Идентификатор:  <br/> |0x5FFB  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Получатель транспорта  <br/> |
   
## <a name="remarks"></a>Примечания

Значение должно быть указано в UTC.
