# <a name="pidlidworkaddresspostofficebox-canonical-property"></a>Каноническое свойство PidLidWorkAddressPostOfficeBox

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает часть почтового ящика для работы контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidWorkAddressPostOfficeBox  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000804A  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
