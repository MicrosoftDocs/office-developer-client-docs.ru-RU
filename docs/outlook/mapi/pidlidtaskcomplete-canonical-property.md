# <a name="pidlidtaskcomplete-canonical-property"></a>Каноническое свойство PidLidTaskComplete

  
  
**Относится к**: Outlook 
  
Указывает, что задача завершена.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskComplete  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x0000811C  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Task  <br/> |
