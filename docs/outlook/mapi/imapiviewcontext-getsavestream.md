# <a name="imapiviewcontextgetsavestream"></a>IMAPIViewContext::GetSaveStream

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Получает поток, который будет использоваться для сохранения текущего сообщения.
  
```cpp
HRESULT GetSaveStream(
ULONG FAR * pulFlags,
ULONG FAR * pulFormat,
LPSTREAM FAR * ppstm
);
```

## <a name="parameters"></a>Параметры

 _пулфлагс_
  
> вышли Указатель на битовую маску флагов, определяющих способ сохранения текста сообщения. Можно задать следующий флаг:
    
MAPI_UNICODE 
  
> Текст сообщения сохраняется в формате Юникод. Если флаг MAPI_UNICODE не установлен, текст сохраняется в формате ANSI.
    
 _пулформат_
  
> вышли Указатель на битовую маску флагов, определяющих формат сохраняемого текста. Можно задать следующие флаги:
    
SAVE_FORMAT_RICHTEXT 
  
> Текст сообщения должен быть сохранен в виде форматированного текста в формате RTF. 
    
SAVE_FORMAT_TEXT 
  
> Текст сообщения должен быть сохранен в виде обычного текста. 
    
 _ппстм_
  
> вышли Указатель на указатель на поток, который будет содержать сохраненное сообщение.
    
## <a name="return-value"></a>Возвращаемое значение

S_OK 
  
> Поток успешно получен.
    
## <a name="remarks"></a>Примечания

Объекты формы вызывают метод **имапивиевконтекст:: жетсавестреам** для получения потока объект, который реализует интерфейс **IStream** для поддержки обработки команды "Сохранить как" в средстве просмотра форм. Метод [имапиформ::D оверб](imapiform-doverb.md) , который реализован на сервере формы и вызывается средством просмотра форм для вызова команды, не должен возвращать сообщение, пока сообщение не будет полностью преобразовано в соответствующий текстовый формат и помещается в соответствующий поток. 
  
## <a name="notes-to-callers"></a>Примечания для вызывающих методов

Не записываете в поток, на который указывает _ппстм_ , перед вызовом **жетсавестреам**. Когда возвращается **жетсавестреам** , не сбрасывать положение указателя поиска. Этот указатель должен находиться в конце сохраненного текста сообщения.
