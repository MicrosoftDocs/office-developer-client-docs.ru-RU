# <a name="pidtagoriginalentryid-canonical-property"></a>Каноническое свойство PidTagOriginalEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит исходный идентификатор записи, скопированной из адресной книги в личную адресную книгу или другую доступную для записи адресную книгу.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ОРИГИНАЛ_ЕНТРИД  <br/> |
|Идентификатор:  <br/> |0x3A12  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
   
## <a name="remarks"></a>Комментарии

Это свойство является одним из свойств, содержащих сведения о первоначальном источнике скопированной записи.
  
Для отчета о непрочтении это свойство содержит копию идентификатора записи исходного получателя сообщения, для которого создается отчет. Когда исходный получатель входит в список рассылки, идентификатор записи списка рассылки сохраняется для отчета.
