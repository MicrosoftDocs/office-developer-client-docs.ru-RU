# <a name="pidlidcontactuserfield3-canonical-property"></a>Каноническое свойство PidLidContactUserField3

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает третье поле контакта, предназначенное для разного использования для контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidContactUserField3  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008051  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство содержит текст, не связанный с любым другим свойством, связанным с контактом.
