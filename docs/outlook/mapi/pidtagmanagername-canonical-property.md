# <a name="pidtagmanagername-canonical-property"></a>Каноническое свойство PidTagManagerName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя руководителя получателя. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_MANAGER_NAME, PR_MANAGER_NAME_A, PR_MANAGER_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A4E  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения об идентификации и доступе для получателя. Они определяются получателем и его организацией.
