# <a name="file-format-of-form-configuration-files"></a>Формат файлов конфигурации форм

**Область применения**: Outlook 2013 | Outlook 2016 
  
Файл конфигурации формы — это форматированный файл, созданный разработчиками форм для определения формы.
  
Так как файлы конфигурации форм используются руководителями форм для загрузки форм, каждая форма должна быть определена с помощью файла конфигурации формы. Файлы конфигурации форм должны иметь расширение файла .cfg. Файл следует общему синтаксису файла Windows инициализации (.ini файл). 

Он разделен на названные разделы, и каждый раздел содержит ряд записей и значений. Значения имеют один из следующих типов: строка, отображаемая строка, строка платформы, путь, integer или глобальный уникальный идентификатор **GUID.** Файлы конфигурации форм можно создавать с помощью любого текстового редактора или текстовых процессоров, способных сохранять текстовые файлы.
