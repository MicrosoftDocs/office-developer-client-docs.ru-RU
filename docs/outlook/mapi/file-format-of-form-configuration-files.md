# <a name="file-format-of-form-configuration-files"></a>Формат файлов конфигурации форм

**Относится к**: Outlook 2013 | Outlook 2016 
  
Файл конфигурации формы — это форматированный файл, созданный разработчиками форм для определения формы.
  
Поскольку диспетчеры форм используют файлы конфигурации форм для загрузки форм, каждая форма должна быть определена с помощью файла конфигурации формы. Файлы конфигурации формы должны иметь расширение CFG. Файл следует общему синтаксис файла инициализации Windows (INI-файла). 

Он делится на именуемые разделы, и каждый раздел содержит ряд записей и значений. Значения имеют один из следующих типов: строка, отображаемая строка, строка платформы, путь, integer или глобальный уникальный идентификатор **GUID.** Файлы конфигурации формы можно создавать с помощью любого текстового редактора или текстового процессора, который может сохранять текстовые файлы.
