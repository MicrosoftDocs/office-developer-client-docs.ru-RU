# <a name="pidtagreporttime-canonical-property"></a>Каноническое свойство PidTagReportTime

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит дату и время создания отчета системой обмена сообщениями.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_REPORT_TIME  <br/> |
|Идентификатор:  <br/> |0x0032  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство представляет свойство для каждого получателя в отчетах о доставке и недоставке и свойство для каждого сообщения в отчетах о чтении и непрочтении.
