# <a name="pidtagreporttime-canonical-property"></a>Каноническое свойство PidTagReportTime

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит дату и время, когда система обмена сообщениями создает отчет.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_REPORT_TIME  <br/> |
|Идентификатор:  <br/> |0x0032  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство представляет свойство для каждого получателя в отчетах о доставке и недостистительности, а также свойство для каждого сообщения в отчетах о прочтениях и прочтениях.
