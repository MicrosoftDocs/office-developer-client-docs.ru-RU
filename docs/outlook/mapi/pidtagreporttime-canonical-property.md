# <a name="pidtagreporttime-canonical-property"></a>Каноническое свойство PidTagReportTime

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит дату и время создания отчета в системе обмена сообщениями.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_REPORT_TIME  <br/> |
|Идентификатор:  <br/> |0x0032  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство представляет свойство каждого получателя на отчеты о доставке и недоставке и свойства каждого сообщения на чтение и nonread отчетов.
