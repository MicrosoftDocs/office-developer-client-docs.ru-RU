# <a name="pidtagrulesequence-canonical-property"></a>Каноническое свойство PidTagRuleSequence

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Значение, используемое для определения порядка, в котором оцениваются и выполняются правила. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РУЛЕ_СЕКУЕНЦЕ  <br/> |
|Идентификатор:  <br/> |0x6676  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Правила на стороне сервера  <br/> |
   
## <a name="remarks"></a>Замечания

Правила оцениваются по последовательности в соответствии с порядком возрастания этого значения. Порядок оценки правил с одинаковым значением в этом свойстве не определен.
