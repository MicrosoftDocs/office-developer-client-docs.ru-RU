# <a name="imscapabilities--iunknown"></a>IMSCapabilities : IUnknown

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Предоставляет сведения о том, что может поддерживать магазин.
  
|||
|:-----|:-----|
|Предоставлено:  <br/> |Поставщик службы хранения сообщений  <br/> |
|Идентификатор интерфейса:  <br/> |IID_IMSCapabilities  <br/> |
   
## <a name="vtable-order"></a>Порядок ветвей

|||
|:-----|:-----|
|**[GetCapabilities](imscapabilities-getcapabilities.md)** <br/> |Возвращает возможности магазина на основе указанного выбора.  <br/> |
   
## <a name="remarks"></a>Примечания

 **IMSCapabilities** предоставляет сведения о том, что может поддерживать хранилище, если магазин поддерживает этот интерфейс.
