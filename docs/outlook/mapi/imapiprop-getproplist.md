# <a name="imapipropgetproplist"></a>IMAPIProp::GetPropList

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Возвращает теги свойств для всех свойств. 
  
```cpp
HRESULT GetPropList(
  ULONG ulFlags,
  LPSPropTagArray FAR * lppPropTagArray
);
```

## <a name="parameters"></a>Параметры

 _ulFlags_
  
> возврата Битовая маска флагов, которая управляет форматом строк в возвращенных тегах свойств. Можно задать следующий флаг:
    
MAPI_UNICODE 
  
> Возвращаемые строки представлены в формате Юникод. Если флаг МАПИ_УНИКОДЕ не установлен, строки представлены в формате ANSI.
    
 _Лпппроптагаррай_
  
> вышли Указатель на указатель на массив тегов свойств, который содержит теги для всех свойств объекта.
    
## <a name="return-value"></a>Возвращаемое значение

S_OK 
  
> Все теги свойств успешно возвращены.
    
МАПИ_Е_БАД_ЧАРВИДС 
  
> Установлен либо флаг МАПИ_УНИКОДЕ, либо реализация не поддерживает Юникод, или МАПИ_УНИКОДЕ не задано, а реализация поддерживает только Юникод.
    
## <a name="remarks"></a>Примечания

Метод **IMAPIProp:: жетпроплист** извлекает тег свойства для каждого свойства, которое в настоящее время поддерживается объектом. Если объект в настоящее время не поддерживает какие бы то ни было свойства, **жетпроплист** возвращает массив тегов свойств с набором элементов **квалуес** , равным 0. 
  
Область действия свойств, возвращаемых **жетпроплист** , отличается от поставщика к поставщику. Некоторые поставщики услуг исключают те свойства, для которых у вызывающего абонента нет доступа. Все поставщики возвращают свойства типа **пт_обжект**.
  
Если объект не поддерживает Юникод, **жетпроплист** возвращает мапи_е_бад_чарвидс, даже если для объекта не определено строковые свойства. 
  
## <a name="notes-to-implementers"></a>Примечания для исполнителей

Удаленные поставщики транспорта реализуют **жетпроплист** точно так, как указано здесь. Нет особых вопросов. Реализация должна, конечно же, возвращать тот же список свойств, который поддерживается методом [IMAPIProp::](imapiprop-getprops.md) /PROPS. 
  
## <a name="notes-to-callers"></a>Примечания для вызывающих методов

ВыЗовите функцию [мапифрибуффер](mapifreebuffer.md) , чтобы освободить массив тегов свойств, на который указывает _лпппроптагаррай_.
