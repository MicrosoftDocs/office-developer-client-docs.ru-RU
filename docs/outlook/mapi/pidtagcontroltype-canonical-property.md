# <a name="pidtagcontroltype-canonical-property"></a>Каноническое свойство PidTagControlType

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит значение, указывающее тип элемента управления для элемента управления, используемого в диалоговом окне. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_КОНТРОЛ_ТИПЕ  <br/> |
|Идентификатор:  <br/> |0x3F02  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Таблица отображения MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство может иметь только одно из следующих значений:
  
ДТКТ_БУТТОН 
  
> Элемент управления "Кнопка" диалогового окна.
    
ДТКТ_ЧЕККБОКС 
  
> Диалоговое окно.
    
ДТКТ_КОМБОБОКС 
  
> Диалоговое окно С раскрывающимся списком.
    
ДТКТ_ДДЛБКС 
  
> Раскрывающийся список диалоговых окон.
    
ДТКТ_ЕДИТ 
  
> Диалоговое окно редактирования текстового поля.
    
ДТКТ_ГРАУПБОКС 
  
> Поле группы диалоговых окон.
    
ДТКТ_ЛАБЕЛ 
  
> Метка диалогового окна.
    
ДТКТ_ЛБКС 
  
> Диалоговое окно списка.
    
ДТКТ_ЛИСТБОКС 
  
> Диалоговое окно списка.
    
ДТКТ_МВДДЛБКС 
  
> Многозначный список, заполненный многозначным свойством типа String.
    
ДТКТ_ПАЖЕ 
  
> Страница С вкладками диалогового окна.
    
ДТКТ_РАДИОБУТТОН 
  
> Переключатель диалогового окна.
