# <a name="pidlidremoteentryid-canonical-property"></a>Каноническое свойство PidLidRemoteEntryId

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает идентификатор элемента удаленного элемента.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидремотиид  <br/> |
|Набор свойств:  <br/> |Псетид_ремоте  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008F01  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Удаленное сообщение  <br/> |
