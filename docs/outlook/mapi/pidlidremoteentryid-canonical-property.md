# <a name="pidlidremoteentryid-canonical-property"></a>Каноническое свойство PidLidRemoteEntryId

  
  
**Относится к**: Outlook 
  
Указывает идентификатор записи удаленного элемента.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidRemoteEID  <br/> |
|Набор свойств:  <br/> |PSETID_Remote  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008F01  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Удаленные сообщения  <br/> |
