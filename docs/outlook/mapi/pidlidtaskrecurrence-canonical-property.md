# <a name="pidlidtaskrecurrence-canonical-property"></a>Каноническое свойство PidLidTaskRecurrence

  
  
**Относится к**: Outlook 
  
Содержит структуру RecurrencePattern, предоставляющий информацию о повторяющихся задач.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidTaskRecur  <br/> |
|Набор свойств:  <br/> |PSETID_Task  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008116  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Замечания

Сведения о том, как создать и указать структуру RecurrencePattern перейдите в раздел [Create простого повторяющегося элемента задачи](how-to-create-a-simple-recurrent-task-item.md).
