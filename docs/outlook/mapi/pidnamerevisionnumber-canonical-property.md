# <a name="pidnamerevisionnumber-canonical-property"></a>Каноническое свойство PidNameRevisionNumber

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает номер редакции файл, подключенный к документу.
  
|||
|:-----|:-----|
|Понятные имена:  <br/> |Отсутствует  <br/> |
|Набор свойств:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Имя свойства:  <br/> |RevNumber  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Common  <br/> |
