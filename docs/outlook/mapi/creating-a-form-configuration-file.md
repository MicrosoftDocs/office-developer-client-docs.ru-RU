# <a name="creating-a-form-configuration-file"></a>Создание файла конфигурации формы

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Файл конфигурации формы предоставляет сведения о форме как для используемого диспетчера форм, так и для клиентских приложений. Файл конфигурации формы содержит обширную спецификацию формы, включая свойства, опубликованные формой для использования клиентами обмена сообщениями, команды, реализованные в форме, и платформы, поддерживаемые формой.
  
Файл конфигурации формы — это файл с расширением CFG и имеет формат, аналогичный файлу инициализации Windows. Это обычный текстовый файл с числом разделов. Каждый раздел начинается с имени раздела, заключенного в квадратные скобки. Каждый раздел содержит одну или несколько строк, которые определяют значения и параметры, соответствующие этому разделу. Значения имеют один из следующих типов:
  
- String
    
- Отображаемая строка
    
- Строка платформы
    
- Имя пути
    
- Целое число
    
- GUID
    
Дополнительные сведения о разделах файла CFG см. в файле ["Формат файлов конфигурации формы".](file-format-of-form-configuration-files.md)
