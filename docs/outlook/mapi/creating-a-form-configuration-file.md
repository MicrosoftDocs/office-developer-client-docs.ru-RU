# <a name="creating-a-form-configuration-file"></a>Создание файла конфигурации формы

  
  
**Относится к**: Outlook 
  
Файл конфигурации формы сведения о формы в диспетчер форм используются и в клиентских приложениях. Файл конфигурации формы содержит широкое спецификацию для формы, включая свойства, опубликованное формы для использования системой обмена сообщениями клиентов, команды, реализованный формы и платформ, поддерживаемых формы.
  
Файл конфигурации формы представляет собой файл с расширением .cfg и имеет формат, аналогичный файла инициализации Windows. Это обычный текстовый файл с номером из разделов. Каждый раздел начинается с именем раздела, заключенные в квадратные скобки. Каждый раздел содержит одну или несколько строк, которые определяют значения и параметры, относящиеся к этого раздела. Значения использовать одну из следующих типов:
  
- Строка
    
- Отображается строка
    
- Строка платформы
    
- Имя пути
    
- Целое число
    
- GUID
    
Дополнительные сведения о разделах файла .cfg см [файл формата из формы](file-format-of-form-configuration-files.md).
