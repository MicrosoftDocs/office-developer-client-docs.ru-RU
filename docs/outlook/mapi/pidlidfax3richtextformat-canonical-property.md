# <a name="pidlidfax3richtextformat-canonical-property"></a>Каноническое свойство PidLidFax3RichTextFormat

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает формат RICH TEXT другого факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax3RTF  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x000080D6  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Контакт  <br/> |
