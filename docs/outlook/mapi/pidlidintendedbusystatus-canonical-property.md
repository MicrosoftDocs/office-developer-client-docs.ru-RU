# <a name="pidlidintendedbusystatus-canonical-property"></a>Каноническое свойство PidLidIntendedBusyStatus

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает значение свойства **диспидбусистатус** ([PidLidBusyStatus](pidlidbusystatus-canonical-property.md)) на собрании в календаре организатора при отправке приглашения на собрание или собрания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидинтендедбусистатус  <br/> |
|Набор свойств:  <br/> |Псетид_аппоинтмент  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008224  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Комментарии

Допустимые значения этого свойства такие же, как и для свойства **диспидбусистатус**.
