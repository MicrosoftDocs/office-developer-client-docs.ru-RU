# <a name="pidlidintendedbusystatus-canonical-property"></a>Каноническое свойство PidLidIntendedBusyStatus

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает значение свойства **dispidBusyStatus** [(PidLidBusyStatus)](pidlidbusystatus-canonical-property.md)собрания в календаре организатора при отправлении запроса на собрание или обновления собрания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidIntendedBusyStatus  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x00008224  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Допустимые значения этого свойства такие же, как и для **свойства dispidBusyStatus.**
