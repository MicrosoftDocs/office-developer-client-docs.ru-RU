# <a name="pidtagphysicalrenditionattributes-canonical-property"></a>Каноническое свойство PidTagPhysicalRenditionAttributes

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор объекта ASN.1, который используется для преобразования вложений сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_PHYSICAL_RENDITION_ATTRIBUTES  <br/> |
|Идентификатор:  <br/> |0x0C10  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Получатель MAPI  <br/> |
