# <a name="pidnamenotecount-canonical-property"></a>Каноническое свойство PidNameNoteCount

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает количество заметок в файле, вложенном в документ.
  
|||
|:-----|:-----|
|Понятные имена:  <br/> |Нет  <br/> |
|Набор свойств:  <br/> |ПС_ПУБЛИК_СТРИНГС  <br/> |
|Имя свойства:  <br/> |Нотекаунт  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Распространенная  <br/> |
