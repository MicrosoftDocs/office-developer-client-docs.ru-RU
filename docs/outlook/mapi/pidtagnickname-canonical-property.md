# <a name="pidtagnickname-canonical-property"></a>Каноническое свойство PidTagNickname

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит понятное имя контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_NICKNAME, PR_NICKNAME_A, PR_NICKNAME_W  <br/> |
|Идентификатор:  <br/> |0x3A4F  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства являются необязательными. Их содержимое определяются пользователем.
