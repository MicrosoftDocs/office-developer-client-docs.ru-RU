# <a name="imapiprogress--iunknown"></a>IMAPIProgress : IUnknown

  
  
**Относится к**: Outlook 
  
Реализует объект хода выполнения, который предоставляет клиентских приложений с индикатором хода выполнения. Индикатор выполнения отображается пользовательский интерфейс, который показывает процент завершения операции, такие как копирование папок между хранилищами сообщений. MAPI и клиентских приложениях внедрение объектов хода выполнения и их использовании поставщиков услуг.

|:-----|:-----|

|Предоставляемые:  <br/> |Объекты хода выполнения  <br/> |
|Реализованный:  <br/> |MAPI и клиентских приложениях  <br/> |
|Вызывается:  <br/> |Поставщики услуг  <br/> |
|Идентификатор интерфейса:  <br/> |IID_IMAPIProgress  <br/> |
|Тип указателя:  <br/> |LPMAPIPROGRESS  <br/> |

## <a name="remarks"></a>Замечания

MAPI включает параметр _lpProgress_ во многих методов, выполняющих потенциально длительных операций.  _lpProgress_ указывает реализации клиентского объекта хода выполнения. Клиенты, реализующие интерфейс **IMAPIProgress** установите для этого параметра для указания на их реализации; Клиенты, которые не реализует **IMAPIProgress** параметру значение NULL. Чтобы отобразить индикатор выполнения во время обработки операции, поставщиков услуг использовать объект ход выполнения, полученных от клиента, если он доступен или для реализации интерфейса MAPI (указывается при _lpProgress_ имеет значение NULL).
