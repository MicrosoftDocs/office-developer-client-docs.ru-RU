# <a name="pidtagexceptionstarttime-canonical-property"></a>Каноническое свойство PidTagExceptionStartTime

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату начала и время исключения в локальном часовом поясе машины, когда создается исключение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_EXCEPTION_STARTTIME  <br/> |
|Идентификатор:  <br/> |0x7FFB  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Сообщение, определенное классом, не передаваемое  <br/> |
   
## <a name="remarks"></a>Примечания

> [!NOTE]
> Это свойство является информационным и не должно полагаться на критически важные сведения.
