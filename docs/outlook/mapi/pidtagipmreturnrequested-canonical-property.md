# <a name="pidtagipmreturnrequested-canonical-property"></a>Каноническое свойство PidTagIpmReturnRequested

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение TRUE, если сообщение должно быть возвращено с отчетом.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ИПМ_РЕТУРН_РЕКУЕСТЕД  <br/> |
|Идентификатор:  <br/> |0x0C02  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Общий обмен сообщениями  <br/> |
