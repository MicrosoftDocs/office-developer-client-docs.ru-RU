# <a name="pidtaglanguage-canonical-property"></a>Каноническое свойство PidTagLanguage

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит значение, указывающее язык, на котором пользователь обмена сообщениями записывает сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ЛАНГУАЖЕ, ПР_ЛАНГУАЖЕ_А, ПР_ЛАНГУАЖЕ_В  <br/> |
|Идентификатор:  <br/> |0x3A0C  <br/> |
|Тип данных:  <br/> |ПТ_УНИКОДЕ, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Строка содержит один код страны или региона, состоящего из двух символов.
