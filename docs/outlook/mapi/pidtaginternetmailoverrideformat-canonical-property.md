# <a name="pidtaginternetmailoverrideformat-canonical-property"></a>Каноническое свойство PidTagInternetMailOverrideFormat

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает метод кодирования и включение HTML для вложений, которые должны быть установлены для исходящей почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_INETMAIL_OVERRIDE_FORMAT  <br/> |
|Идентификатор:  <br/> |0x5902  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Примечания

Сведения о вложенных компонентах, в которых это свойство разделено, приведено в разделе [[MS-оксомсг]](https://msdn.microsoft.com/library/daa9120f-f325-4afb-a738-28f91049ab3c%28Office.15%29.aspx).
