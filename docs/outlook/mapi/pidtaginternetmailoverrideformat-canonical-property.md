# <a name="pidtaginternetmailoverrideformat-canonical-property"></a>Каноническое свойство PidTagInternetMailOverrideFormat

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает метод кодизания и включение HTML для вложений, которые следует установить на исходячей почте.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_INETMAIL_OVERRIDE_FORMAT  <br/> |
|Идентификатор:  <br/> |0x5902  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Примечания
