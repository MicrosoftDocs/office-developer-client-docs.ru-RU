# <a name="pidlidfax2emailaddress-canonical-property"></a>Каноническое свойство PidLidFax2EmailAddress

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает адрес электронной почты домашнего факса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax2EmailAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080C3  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Примечания

В этом свойстве, если он присутствует, должно содержаться имя отображаемого пользователя, за которым следует символ "@", а затем номер факса.
