# <a name="swstringarray"></a>SWStringArray

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит массив строк символов, которые используются для описания свойства типа PT_MV_UNICODE. 
  
|||
|:-----|:-----|
|Файл заголовка:  <br/> |MAPIDEFS. h  <br/> |
   
```cpp
typedef struct _SWStringArray
{
  ULONG cValues;
  LPWSTR FAR *lppszW;
} SWStringArray;

```

## <a name="members"></a>"Участники"

 **квалуес**
  
> Количество строк в массиве, на которое указывает элемент **лппсзв** . 
    
 **лппсзв**
  
> Указатель на массив строк Юникода с символами NULL.
