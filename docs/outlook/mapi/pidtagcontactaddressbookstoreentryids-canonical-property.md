# <a name="pidtagcontactaddressbookstoreentryids-canonical-property"></a>Каноническое свойство PidTagContactAddressBookStoreEntryIds

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит входные ID-адреса магазинов, в которых находятся папки, которые выступают в качестве записей контактной книги.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONTAB_STORE_ENTRYIDS  <br/> |
|Идентификатор:  <br/> |0x6626  <br/> |
|Тип данных:  <br/> |PT_MV_BINARY  <br/> |
|Область:  <br/> |Адресная книга контактов  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство в разделе Профиль адресной книги контактов.
