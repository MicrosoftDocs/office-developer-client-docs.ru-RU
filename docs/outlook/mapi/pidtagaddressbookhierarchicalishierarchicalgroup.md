# <a name="pidtagaddressbookhierarchicalishierarchicalgroup"></a>Каноническое свойство PidTagAddressBookHierarchicalIsHierarchicalGroup

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит TRUE, если список рассылки также является отделной группой. Это свойство доступно только в списках рассылки.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_EMS_AB_HAB_IS_HIERARCHICAL_GROUP  <br/> |
|Набор свойств:  <br/> |Адресная книга  <br/> |
|Long ID (LID):  <br/> |0x8CDD  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Адресная книга Exchange  <br/> |
