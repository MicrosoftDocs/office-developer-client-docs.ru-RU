# <a name="pidtagcontactaddressbookuid-canonical-property"></a>Каноническое свойство PidTagContactAddressBookUid

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор UID, идентифицирует этот сеанс адресной книги контактов.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONTAB_UID  <br/> |
|Идентификатор:  <br/> |0x6601  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Адресная книга контакта  <br/> |
   
## <a name="remarks"></a>Примечания

Если настроено несколько адресных книг контактов, это свойство должно быть установлено для каждой из них.
