# <a name="pidlidlogtypedesc-canonical-property"></a>Каноническое свойство PidLidLogTypeDesc

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Описывает действия, которые записывают.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidLogTypeDesc  <br/> |
|Набор свойств:  <br/> |PSETID_Log  <br/> |
|Long ID (LID):  <br/> |0x00008712  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Журнал  <br/> |
