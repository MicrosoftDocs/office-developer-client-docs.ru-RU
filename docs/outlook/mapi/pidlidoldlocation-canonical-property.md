# <a name="pidlidoldlocation-canonical-property"></a>Каноническое свойство PidLidOldLocation

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает исходное значение свойства **диспидлокатион** ([PidLidLocation](pidlidlocation-canonical-property.md)) перед обновлением собрания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидолдлокатион  <br/> |
|Набор свойств:  <br/> |Псетид_митинг  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00000028  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Комментарии

Это свойство не является обязательным.
