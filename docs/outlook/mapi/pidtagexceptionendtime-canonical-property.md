# <a name="pidtagexceptionendtime-canonical-property"></a>Каноническое свойство PidTagExceptionEndTime

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает даты окончания и время исключения в локальном часовом поясе машины при его создания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_EXCEPTION_ENDTIME  <br/> |
|Идентификатор:  <br/> |0x7FFC  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Сообщение, определенное классом, не передаваемое  <br/> |
   
## <a name="remarks"></a>Примечания

> [!NOTE]
> Это свойство является информационным и не должно полагаться на критически важные сведения.
