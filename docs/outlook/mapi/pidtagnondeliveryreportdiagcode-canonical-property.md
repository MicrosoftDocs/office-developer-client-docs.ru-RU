# <a name="pidtagnondeliveryreportdiagcode-canonical-property"></a>Каноническое свойство PidTagNonDeliveryReportDiagCode

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит диагностический код, который является частью отчета о ненастройности.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_NDR_DIAG_CODE  <br/> |
|Идентификатор:  <br/> |0x0C05  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
