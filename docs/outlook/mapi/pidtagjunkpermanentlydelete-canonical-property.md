# <a name="pidtagjunkpermanentlydelete-canonical-property"></a>Каноническое свойство PidTagJunkPermanentlyDelete

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Означает, что сообщения, идентифицированные как нежелательные, могут быть удалены без возможности удаления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_JUNK_PERMANENTLY_DELETE  <br/> |
|Идентификатор:  <br/> |0x6102  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Спам  <br/> |
