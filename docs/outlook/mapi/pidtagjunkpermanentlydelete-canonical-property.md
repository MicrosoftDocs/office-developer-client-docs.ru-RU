# <a name="pidtagjunkpermanentlydelete-canonical-property"></a>Каноническое свойство PidTagJunkPermanentlyDelete

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает, если параметр имеет значение 1, которого сообщения распознан как нежелательной почты могут быть удалены без возможности восстановления.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_JUNK_PERMANENTLY_DELETE  <br/> |
|Идентификатор:  <br/> |0x6102  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Нежелательной почты  <br/> |
