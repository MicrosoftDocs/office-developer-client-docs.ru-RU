# <a name="pidlidfexceptionalattendees-canonical-property"></a>Каноническое свойство PidLidFExceptionalAttendees

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, является ли это свойство повторяющимся объектом Calendar с одним или несколькими исключениями, а по крайней мере одно из внедренных сообщений Exception содержит по крайней мере один РеЦипиентров.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидфексцептионалаттендис  <br/> |
|Набор свойств:  <br/> |Псетид_аппоинтмент  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x0000822B  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Замечания

Значение FALSE или отсутствие этого свойства указывает на то, что объект Calendar либо не содержит исключений, либо ни одно исключение внедренных сообщений РеЦипиентровс.
