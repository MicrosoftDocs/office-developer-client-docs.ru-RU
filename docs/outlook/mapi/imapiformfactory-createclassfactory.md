# <a name="imapiformfactorycreateclassfactory"></a>IMAPIFormFactory::CreateClassFactory

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Возвращает объект фабрики класса для формы.
  
```cpp
HRESULT CreateClassFactory(
  REFCLSID clsidForm,
  ULONG ulFlags,
  LPCLASSFACTORY FAR * lppClassFactory
);
```

## <a name="remarks"></a>Примечания

Средства просмотра форм вызывают метод **имапиформфактори:: креатеклассфактори** для получения фабрики класса для определенной формы. Фабрика классов используется для создания экземпляров формы, которые обрабатывают сообщения определенного класса и контролируют доступ к этим экземплярам. 
  
Метод **креатеклассфактори** вызывается средствами просмотра формы для получения объекта фабрики класса для серверов форм, которые реализуют несколько классов сообщений. Этот метод получает идентификатор класса (CLSID) в качестве параметра. В зависимости от этого параметра этот метод может определить конкретный вид объекта фабрики класса, который требуется возвратить. 
  
## <a name="notes-to-implementers"></a>Примечания для исполнителей

Вы можете вернуться от реализации **креатеклассфактори** к тому же объекту фабрики класса для нескольких вызовов для одного и того же идентификатора класса. Создание нового экземпляра фабрики классов не является обязательным. 
  
Вы можете реализовать фабрику одного класса, которая создает соответствующие экземпляры фабрики классов по запросу или несколько реализаций фабрики классов, по одному для каждого класса сообщений.
