# <a name="pidtagoriginatorrequestedalternaterecipient-canonical-property"></a>Каноническое свойство PidTagOriginatorRequestedAlternateRecipient

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор записи для альтернативного получателя, назначенного отправищиком.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINATOR_REQUESTED_ALTERNATE_RECIPIENT  <br/> |
|Идентификатор:  <br/> |0x0C09  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство используется в автоматических сообщениях. Если автопередача запрещена или альтернативный получатель не назначен, должен быть создан отчет о ненастройке.
