# <a name="iabcontainerdeleteentries"></a>IABContainer::DeleteEntries

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Удаляет одну или несколько записей, обычно для обмена сообщениями пользователей, списков рассылки или других контейнеров.
  
```cpp
HRESULT DeleteEntries(
  LPENTRYLIST lpEntries,
  ULONG ulFlags
);
```

## <a name="parameters"></a>Параметры

 _lpEntries_
  
> [in] Указатель на массив структур [ENTRYLIST,](entrylist.md) содержащих идентификаторы записей, которые представляют удаляемую запись.

## <a name="return-value"></a>Возвращаемое значение

S_OK 
  
> Указанные записи успешно удалены. 
    
MAPI_W_PARTIAL_COMPLETION 
  
> Вызов был успешным, но не удалось удалить одну или несколько записей. При возвращении этого значения вызов должен быть обработан как успешный. Чтобы проверить это значение,  используйте HR_FAILED макрос. Дополнительные сведения см. в теме ["Использование макроса для обработки ошибок".](using-macros-for-error-handling.md)
