# <a name="pidtagformhidden-canonical-property"></a>Каноническое свойство PidTagFormHidden

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит true, если форма должна быть не отображаться с помощью меню составить и диалоговое окно. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_FORM_HIDDEN  <br/> |
|Идентификатор:  <br/> |0x3307  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Общие MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Свойства, связанные с формой, можно только для чтения.
