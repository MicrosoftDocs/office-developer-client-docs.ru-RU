# <a name="pidtagrulelevel-canonical-property"></a>Каноническое свойство PidTagRuleLevel

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит уровень выхода правила.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_РУЛЕ_ЛЕВЕЛ  <br/> |
|Идентификатор:  <br/> |0x6683  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Правила на стороне сервера  <br/> |
   
## <a name="remarks"></a>Примечания

Если вы настраиваете это свойство, клиент должен пройти в 0x00000000.
