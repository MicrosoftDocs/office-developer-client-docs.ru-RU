# <a name="pidlidinternetaccountname-canonical-property"></a>Каноническое свойство PidLidInternetAccountName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает имя учетной записи электронной почты, видимой пользователем, через которую отправляется сообщение электронной почты.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidInetAcctName  <br/> |
|Набор свойств:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008580  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Формат этой строки зависит от реализации. Это свойство может быть использовано клиентом для определения сервера, на который направляется почта, но является необязательным и значение не имеет значения для сервера.
