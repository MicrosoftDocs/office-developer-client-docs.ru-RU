# <a name="pidtagmessagehandlingsystemcommonname-canonical-property"></a>Каноническое свойство PidTagMessageHandlingSystemCommonName

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит общее имя системы обработки сообщений.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_MHS_COMMON_NAME, PR_MHS_COMMON_NAME_A, PR_MHS_COMMON_NAME_W  <br/> |
|Идентификатор:  <br/> |0x3A0F  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Адрес  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства содержат общее имя системы обработки сообщений.
