<span data-ttu-id="0e14c-119">В следующем примере кода показано, как поставщик службы C можно определить простой состояние объекта.</span><span class="sxs-lookup"><span data-stu-id="0e14c-119">The following code example shows how a C service provider can define a simple status object.</span></span> <span data-ttu-id="0e14c-120">Первый элемент является указателя vtable; Остальная часть объекта состоит из элементов данных.</span><span class="sxs-lookup"><span data-stu-id="0e14c-120">The first member is the vtable pointer; the rest of the object is made up of data members.</span></span>

```C
    const STATUS_Vtbl FAR *lpVtbl;

```
