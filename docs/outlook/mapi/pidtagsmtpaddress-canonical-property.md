# <a name="pidtagsmtpaddress-canonical-property"></a>Каноническое свойство PidTagSmtpAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит SMTP-адрес для объекта адресной книги.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SMTP_ADDRESS, PR_SMTP_ADDRESS_A, PR_SMTP_ADDRESS_W  <br/> |
|Идентификатор:  <br/> |0x39FE  <br/> |
|Тип данных:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
