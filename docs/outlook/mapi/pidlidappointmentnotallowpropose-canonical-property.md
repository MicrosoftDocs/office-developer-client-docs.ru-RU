# <a name="pidlidappointmentnotallowpropose-canonical-property"></a>Каноническое свойство PidLidAppointmentNotAllowPropose

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, не разрешено ли участникам предлагать новую дату и время собрания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidApptNotAllowPropose  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x0000825A  <br/> |
|Тип данных:  <br/> |PT_BOOLEAN  <br/> |
|Область:  <br/> |Собрания  <br/> |
   
## <a name="remarks"></a>Примечания

Значение FALSE или отсутствие этого свойства указывает на то, что участникам разрешено предлагать новую дату и время.
