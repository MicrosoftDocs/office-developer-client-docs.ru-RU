# <a name="pidlidtaskversion-canonical-property"></a>Каноническое свойство PidLidTaskVersion

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает, какая копия является последним обновлением задачи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидтаскверсион  <br/> |
|Набор свойств:  <br/> |Псетид_таск  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008112  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Задача  <br/> |
   
## <a name="remarks"></a>Примечания

Обновления с более низкими версиями, чем в задаче, игнорируются. 
  
При внедрении задачи в общении задач клиент устанавливает текущую версию внедренной задачи в связи задач.
