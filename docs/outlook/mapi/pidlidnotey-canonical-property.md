# <a name="pidlidnotey-canonical-property"></a>Каноническое свойство PidLidNoteY

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Задает расстояние от верхнего края экрана, на котором отображается заметка, в пикселях.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспиднотэй  <br/> |
|Набор свойств:  <br/> |Псетид_ноте  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008B05  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |ЗаМетка  <br/> |
