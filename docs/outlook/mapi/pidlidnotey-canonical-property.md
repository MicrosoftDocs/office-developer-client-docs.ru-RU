# <a name="pidlidnotey-canonical-property"></a>Каноническое свойство PidLidNoteY

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает расстояние в пикселях от верхнего края экрана, где пользовательский интерфейс отображает примечание.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidNoteY  <br/> |
|Набор свойств:  <br/> |PSETID_Note  <br/> |
|Long ID (LID):  <br/> |0x00008B05  <br/> |
|Тип данных:  <br/> |PT_LONG  <br/> |
|Область:  <br/> |Липкий примечание  <br/> |
