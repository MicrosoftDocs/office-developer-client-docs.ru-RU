# <a name="mapi-service-provider-samples"></a>Примеры поставщиков услуг MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
В этом разделе содержатся примеры кода трех поставщиков услуг MAPI:
  
- [Загрузка примеров MAPI для Outlook](downloading-the-outlook-mapi-samples.md)
    
    В этом разделе описывается загрузка примеров.
    
- [Пример поставщика адресной книги](address-book-provider-sample.md)
    
    В этом примере поддерживается один контейнер только для чтения для отображаемого имени и адресов электронной почты, которые считыются из плоского двоичного файла.
    
- [Пример поставщика store сообщений](message-store-provider-sample.md)
    
    В этом примере каталоги используются в качестве папок, а файлы документов OLE 2.0 — в качестве сообщений. В примере поддерживаются все необходимые функции для хранения сообщений по умолчанию.
    
- [Пример поставщика транспорта](transport-provider-sample.md)
    
    В этом примере для передачи и получения сообщений используются файлы и каталоги.
    
Чтобы запустить любой из этих примеров в операционных системах Windows 7, Windows Vista, Windows Server 2008, Windows XP или Windows Server 2003, необходимо сначала настроить систему времени работы MAPI. Дополнительные сведения см. в статье [Общие сведения о программировании MAPI](mapi-programming-overview.md).
