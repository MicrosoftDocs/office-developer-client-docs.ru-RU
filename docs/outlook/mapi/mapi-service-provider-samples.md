# <a name="mapi-service-provider-samples"></a>Примеры поставщика службы MAPI

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
В этом разделе приведены примеры кода трех поставщиков службы MAPI:
  
- [Загрузка примеров Outlook MAPI](downloading-the-outlook-mapi-samples.md)
    
    В этом разделе описывается загрузка примеров.
    
- [Пример поставщика адресной книги](address-book-provider-sample.md)
    
    В этом примере поддерживается один контейнер "только для чтения" для отображаемых имен и адресов электронной почты, которые считываются из плоского двоичного файла.
    
- [Пример поставщика хранилища сообщений](message-store-provider-sample.md)
    
    В этом примере используются каталоги как папки и файлы документов OLE 2,0 в качестве сообщений. В этом примере поддерживаются все необходимые компоненты для хранилища сообщений по умолчанию.
    
- [Пример поставщика транспорта](transport-provider-sample.md)
    
    В этом примере используются файлы и каталоги для передачи и получения сообщений.
    
Для запуска любого из этих примеров в операционных системах Windows 7, Windows Vista, Windows Server 2008, Windows XP и Windows Server 2003 необходимо сначала настроить систему среды выполнения MAPI. Дополнительные сведения см. в статье [Общие сведения о программировании MAPI](mapi-programming-overview.md).
