# <a name="mapi-service-provider-samples"></a>Примеры поставщиков услуг MAPI

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
В этом разделе представлены примеры кода трех поставщиков услуг MAPI:
  
- [Загрузка Outlook MAPI](downloading-the-outlook-mapi-samples.md)
    
    В этом разделе описывается, как скачать образцы.
    
- [Пример поставщика адресных книг](address-book-provider-sample.md)
    
    В этом примере поддерживается один контейнер только для чтения для отображаемого имени и адресов электронной почты, которые читают из плоского двоичного файла.
    
- [Пример поставщика магазина сообщений](message-store-provider-sample.md)
    
    В этом примере каталоги используются в качестве папок, а файлы документов OLE 2.0 — в качестве сообщений. Пример поддерживает все необходимые функции для магазина сообщений по умолчанию.
    
- [Пример поставщика транспорта](transport-provider-sample.md)
    
    В этом примере используются файлы и каталоги для передачи и получения сообщений.
    
Чтобы запустить любой из этих образцов на операционных системах Windows, Windows Vista, Windows Server 2008, Windows XP или Windows Server 2003, необходимо сначала настроить систему запуска MAPI. Дополнительные сведения см. в статье [Общие сведения о программировании MAPI](mapi-programming-overview.md).
