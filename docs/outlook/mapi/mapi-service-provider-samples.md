# <a name="mapi-service-provider-samples"></a>Примеры поставщика службы MAPI

  
  
**Относится к**: Outlook 
  
В этом разделе содержатся примеры кода три поставщиков служб MAPI:
  
- [Скачивание примеров MAPI для Outlook](downloading-the-outlook-mapi-samples.md)
    
    В этом разделе описывается, как можно загрузить в примерах.
    
- [Пример поставщика адресной книги](address-book-provider-sample.md)
    
    В этом примере поддерживает один контейнер только для чтения отображаемые имена и адреса электронной почты, которые читают плоской двоичных файлов.
    
- [Пример поставщика хранилища сообщений](message-store-provider-sample.md)
    
    В этом примере использует каталоги как папки и файлы OLE 2.0 документа как сообщения. Пример поддерживает все необходимые компоненты для хранилища сообщений по умолчанию.
    
- [Пример поставщика транспорта](transport-provider-sample.md)
    
    В этом примере использует каталоги и файлы для передачи и получения сообщений.
    
Чтобы выполнить какие-либо из этих примеров на Windows 7, Windows Vista, Windows Server 2008, Windows XP или операционных систем Windows Server 2003, сначала необходимо настроить системы среды выполнения MAPI. Для получения дополнительных сведений см [MAPI программирования](mapi-programming-overview.md).
