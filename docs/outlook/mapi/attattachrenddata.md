# <a name="attattachrenddata"></a>attAttachRenddata

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Атрибут **аттаттачренддата** кодируется как структура **ренддата** , описывающая способ и место отображения вложения в тексте сообщения. Структура **ренддата** просто кодируется в потоке TNEF как **sizeof (ренддата)** байтов, начиная с первого элемента структуры **ренддата** . Если для элемента **dwFlags** структуры **ренддата** задано значение **MAC_BINARY**, то данные для следующего вложения хранятся в формате макбинари; в противном случае данные вложения кодируются как обычно.
