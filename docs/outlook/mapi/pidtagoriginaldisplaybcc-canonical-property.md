# <a name="pidtagoriginaldisplaybcc-canonical-property"></a>Каноническое свойство PidTagOriginalDisplayBcc

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит отображаемую информацию о любых получателях с незрячими копиями (BCC) исходного сообщения.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_ORIGINAL_DISPLAY_BCC, PR_ORIGINAL_DISPLAY_BCC_A, PR_ORIGINAL_DISPLAY_BCC_W  <br/> |
|Идентификатор:  <br/> |0x0072  <br/> |
|Тип данных:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Область:  <br/> |Общие сообщения  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства содержат список, разделенный с зазнайки. Они печатаются с помощью MAPI и копируется непосредственно из PR_DISPLAY_BCC **(** [PidTagDisplayBcc)](pidtagdisplaybcc-canonical-property.md)при составлении отчета о доставке или ненадежаности, а также чтения или непрочитанные отчеты. Эти свойства могут присутствовать в других сообщениях в зависимости от их классов сообщений.
