# <a name="pidtagabsearchpath-canonical-property"></a>Каноническое свойство PidTagAbSearchPath

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит список идентификаторов записей для контейнеров адресных книг, которые необходимо искать для разрешения имен. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_AB_SEARCH_PATH  <br/> |
|Идентификатор:  <br/> |0x3D05  <br/> |
|Тип данных:  <br/> |PT_MV_BINARY  <br/> |
|Область:  <br/> |Адресная книга  <br/> |
   
## <a name="remarks"></a>Примечания

Не используйте это свойство. Оно зарезервировано для использования MAPI.
