# <a name="pidtagabsearchpath-canonical-property"></a>Каноническое свойство PidTagAbSearchPath

  
  
**Относится к**: Outlook 
  
Содержит список идентификаторов запись для контейнеров адресной книги, которые требуется выполнить поиск для разрешения имен. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_AB_SEARCH_PATH  <br/> |
|Идентификатор:  <br/> |0x3D05  <br/> |
|Тип данных:  <br/> |PT_MV_BINARY  <br/> |
|Область:  <br/> |Адресная книга  <br/> |
   
## <a name="remarks"></a>Замечания

Это свойство не используется. Зарезервирован для использования с MAPI.
