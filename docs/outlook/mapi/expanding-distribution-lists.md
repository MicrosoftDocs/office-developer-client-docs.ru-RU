# <a name="expanding-distribution-lists"></a>Развертывание списков рассылки

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
 **Чтобы запрашивать MAPI разверните список рассылки**
  
- Присвойте свойству **PR_ADDRTYPE** ([PidTagAddressType](pidtagaddresstype-canonical-property.md)) MAPIPDL.
    
    MAPI расширяет адреса с этим типом перед отправкой сообщения поставщика транспорта.
