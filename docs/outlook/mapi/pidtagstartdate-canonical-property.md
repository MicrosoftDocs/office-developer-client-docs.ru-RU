# <a name="pidtagstartdate-canonical-property"></a>Каноническое свойство PidTagStartDate

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит дату и время начала встречи, управляемой приложением планирования.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_СТАРТ_ДАТЕ  <br/> |
|Идентификатор:  <br/> |0x0060  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Конверт MAPI  <br/> |
   
## <a name="remarks"></a>Замечания

При отправке приглашений на собрание приложения планирования должны задать свойства и свойства **пр_енд_дате** ([PidTagEndDate](pidtagenddate-canonical-property.md)).
