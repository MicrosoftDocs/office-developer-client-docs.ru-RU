# <a name="pidtagpostofficebox-canonical-property"></a>Каноническое свойство PidTagPostOfficeBox

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит номер или идентификатор почтового ящика получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_ПОСТ_ОФФИЦЕ_БОКС, ПР_ПОСТ_ОФФИЦЕ_БОКС_А, ПР_ПОСТ_ОФФИЦЕ_БОКС_В, ПР_БУСИНЕСС_АДДРЕСС_ПОСТ_ОФФИЦЕ_БОКС, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX_A, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Идентификатор:  <br/> |0x3A2B  <br/> |
|Тип данных:  <br/> |PT_STRING8, ПТ_УНИКОДЕ  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Комментарии

Эти свойства предоставляют сведения о идентификации и доступе для получателя. Они определяются получателем и их организацией.
