# <a name="pidlidlocation-canonical-property"></a>Каноническое свойство PidLidLocation

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Представляет место встречи.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidLocation  <br/> |
|Набор свойств:  <br/> |PSETID_Appointment  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008208  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Календарь  <br/> |
