# <a name="pidtagdefaultpostmessageclass-canonical-property"></a>Каноническое свойство PidTagDefaultPostMessageClass

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит имя настраиваемого класса message формы.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_DEF_POST_MSGCLASS  <br/> |
|Идентификатор:  <br/> |0x36E5  <br/> |
|Тип данных:  <br/> |PT_STRING8  <br/> |
|Область:  <br/> |Контейнер MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Если это свойство установлено для папки, значение должно содержать либо точно базовый класс сообщения (например, "IPM. Контакт" для папки контактов или "IPM. Встреча" для папки календаря) или начать с базового класса сообщения (например, "IPM." Contact.MyContact").
