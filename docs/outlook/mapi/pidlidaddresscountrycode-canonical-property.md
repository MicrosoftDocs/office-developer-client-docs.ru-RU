# <a name="pidlidaddresscountrycode-canonical-property"></a>Каноническое свойство PidLidAddressCountryCode

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает код страны для адреса электронной почты контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidAddressCountryCode  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный ИД (КРЫШКА):  <br/> |0x000080DD  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
