# <a name="pidtagcreationtime-canonical-property"></a>Каноническое свойство PidTagCreationTime

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит дату и время создания сообщения. 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CREATION_TIME  <br/> |
|Идентификатор:  <br/> |0x3007  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Время сообщения  <br/> |
   
## <a name="remarks"></a>Замечания

Хранилище сообщений этому свойству для каждого сообщения, он создает.
