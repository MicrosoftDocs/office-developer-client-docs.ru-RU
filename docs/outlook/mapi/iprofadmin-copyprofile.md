# <a name="iprofadmincopyprofile"></a>IProfAdmin::CopyProfile

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Копирует профиль.
  
```cpp
HRESULTCopyProfile(
  LPSTR lpszOldProfileName,
  LPSTR lpszOldPassword,
  LPSTR lpszNewProfileName,
  ULONG_PTR ulUIParam,
  ULONG ulFlags
);
```

## <a name="parameters"></a>Параметры

 _Лпсзолдпрофиленаме_
  
> возврата Указатель на имя профиля, который необходимо скопировать.
    
 _Лпсзолдпассворд_
  
> возврата Указатель на пароль, который необходимо скопировать.
    
 _Лпсзневпрофиленаме_
  
> возврата Указатель на новое имя скопированного профиля.
    
 _Улуипарам_
  
> возврата Дескриптор родительского окна всех диалоговых окон и окон, отображаемых данным методом.
    
 _ulFlags_
  
> возврата Битовая маска флагов, определяющих способ копирования профиля. Можно задать следующие флаги:
    
МАПИ_ДИАЛОГ 
  
> Отображает диалоговое окно, в котором пользователю предлагается ввести правильный пароль для копирования профиля. Если этот флаг не установлен, диалоговое окно не отображается.
    
## <a name="return-value"></a>Возвращаемое значение

S_OK 
  
> Профиль успешно скопирован.
    
МАПИ_Е_АКЦЕСС_ДЕНИЕД 
  
> Имя нового профиля совпадает с именем существующего профиля.
    
МАПИ_Е_ЛОГОН_ФАИЛЕД 
  
> Неправильный пароль для копирования профиля, и пользователю не удалось отобразить диалоговое окно, чтобы запросить правильный пароль, так как МАПИ_ДИАЛОГ не задан в параметре _ulFlags_ . 
    
МАПИ_Е_НОТ_ФАУНД 
  
> Указанный профиль не существует.
    
МАПИ_Е_УСЕР_КАНЦЕЛ 
  
> Пользователь отменил операцию, как правило, нажав кнопку **Отмена** в диалоговом окне. 
    
## <a name="remarks"></a>Замечания

Метод **ипрофадмин:: копипрофиле** создает копию профиля, на которую указывает _лпсзолдпрофиленаме_, предоставляя ему имя, на которое указывает _лпсзневпрофиленаме_. Копирование профиля оставляет копию с тем же паролем, что и исходный.
  
Имя исходного профиля, его пароль и копия могут иметь длину до 64 символов и могут включать следующие символы:
  
- Все буквенно-цифровые символы, включая диакритические знаки и символ подчеркивания.
    
- Внутренние пробелы, но не ведущие и замыкающие пробелы.
    
Пароли профилей не поддерживаются во всех операционных системах. В операционных системах, не поддерживающих пароли профилей, _лпсзолдпассворд_ может иметь значение null или указатель на строку нулевой длины. 
  
Если для параметра _лпсзолдпассворд_ ЗАДАНО значение null, то для профиля требуется пароль, а флаг мапи_диалог установлен; диалоговое окно, предлагающее пользователю ввести пароль. Если требуется пароль, но для параметра _лпсзолдпассворд_ ЗАДАНО значение null, а флаг мапи_диалог не установлен, **копипрофиле** возвращает мапи_е_логон_фаилед.
