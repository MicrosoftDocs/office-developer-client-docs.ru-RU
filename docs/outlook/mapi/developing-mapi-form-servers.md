# <a name="developing-mapi-form-servers"></a>Разработка серверов формы MAPI

  
  
**Применимо к**: Outlook 
  
В этом разделе описывается процесс создания исполняемый файл сервера форм и файлах конфигурации формы для создания пользовательских форм MAPI. Прежде чем читать этот раздел, следует ознакомиться со сведениями в [Формах MAPI](mapi-forms.md).
  
Разработка формы server состоит из следующих этапов:
  
1. Решить, какие сведения будут содержаться формы с выбором набор свойств для хранения эти сведения. Для получения дополнительных сведений см [Формы Property Set](choosing-a-form-s-property-set.md).
    
2. Создание пользовательского интерфейса с помощью которой пользователи могут работать со свойствами формы.
    
3. Выбор класса сообщения и создания уникальный идентификатор класса (CLSID). Обзор классов сообщений содержатся в разделе [Классы сообщений MAPI](mapi-message-classes.md). Дополнительные сведения о формы и классы сообщений [Класс сообщения](choosing-a-message-class.md)см.
    
4. Реализация необходимых интерфейсов формы MAPI, а также дополнительные интерфейсы, необходимые серверу определенной формы. Для получения дополнительных сведений см [Написание кода формы сервера](writing-form-server-code.md). 
    
5. Написание кода интерфейса пользователя для обработки взаимодействия с пользователем с помощью объекта формы и свойства используется форма.
    
6. Создание файла конфигурации формы для формы. Для получения дополнительных сведений см [файл формата из формы](file-format-of-form-configuration-files.md).
    
7. Установка формы на компьютерах пользователей. Для получения дополнительных сведений см [формы в библиотеку](installing-a-form-into-a-library.md).
    
Вероятнее всего, будет выполнять действия с 1 по 5 вместо завершения их в последовательности. Процесс разработки форм сервера, как количество проектов программирования, не в котором имеется особенно строго определенную последовательность. Например создание файла конфигурации формы отображается как второй к последний шаг выше, но, вероятно, создайте постепенного файла конфигурации формы и станет более полную как добавление компонентов к серверу формы.
