# <a name="property-identifiers-and-types"></a>Идентификаторы и типы свойств

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Все свойства MAPI представлены в виде тегов свойств. Тег свойства — это 32-разрядное целое число без знака, которое содержит идентификатор свойства в порядке 16 бит высокого порядка и тип свойства в порядке 16 младших разрядов. Теги свойств для всех свойств, определенных в MAPI, включены в файл заголовков мапитагс. h.
  
Идентификаторы свойств используются для указания того, что используется для свойства, а также от того, кто отвечает за него. Идентификаторы свойств делятся MAPI на диапазоны; где идентификатор попадает в диапазон, указывающий на его использование и владение. 
  
Типы свойств используются для указания формата данных свойства. MAPI определяет все допустимые типы. Клиенты и поставщики услуг, создающие новые свойства, должны использовать один из этих типов. Все типы свойств включены в файл заголовков MAPIDEFS. h.
