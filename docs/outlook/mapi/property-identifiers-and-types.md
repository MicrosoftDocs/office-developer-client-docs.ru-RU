# <a name="property-identifiers-and-types"></a>Типы и идентификаторы свойств

  
  
**Относится к**: Outlook 
  
Все свойства MAPI, представлены тегов свойств. Свойство tag — это значение 32-разрядное целое число, содержащее идентификатор свойства в высоким заказ 16-разрядный и тип свойства в небольшой упорядочивания 16 бит. Теги свойства для всех свойств, определенных в MAPI включены в файл заголовка mapitags.h.
  
Идентификаторы свойств используются для указания, для чего используется свойство и кто несет ответственность за его. Идентификаторы свойств, разделены на с MAPI диапазонов; где идентификатор попадают в диапазон указывает его использованием и владельца. 
  
Типы свойств используются для указания формата данных свойства. MAPI определяет все из допустимых типов. Клиенты и поставщиков услуг, создание новых свойств необходимо использовать один из этих типов. Включаются все типы свойств в файле заголовка mapidefs.h.
