# <a name="property-identifiers-and-types"></a>Идентификаторы и типы свойств

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Все свойства MAPI представлены тегами свойств. Тег свойства — это 32-битное неподписаное значение, которое содержит идентификатор свойства в 16 битах высокого порядка, а тип свойства в низком порядке — в 16 битах. Теги свойств для всех свойств, определенных MAPI, включаются в файл загона mapitags.h.
  
Идентификаторы свойств используются для того, чтобы указать, для чего используется свойство и кто несет за него ответственность. Идентификаторы свойств разделены по MAPI на диапазоны; где идентификатор попадает в диапазон, указывает на его использование и право собственности. 
  
Типы свойств используются для указать формат данных свойства. MAPI определяет все допустимые типы. Клиенты и поставщики услуг, создав новые свойства, должны использовать один из этих типов. Все типы свойств включены в файл загона mapidefs.h.
