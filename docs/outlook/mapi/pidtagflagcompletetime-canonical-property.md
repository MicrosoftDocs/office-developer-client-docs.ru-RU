# <a name="pidtagflagcompletetime-canonical-property"></a>Каноническое свойство PidTagFlagCompleteTime

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату и время в Координируется универсальное время (UTC), что объект сообщения был помечен как завершен.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_FLAG_COMPLETE_TIME  <br/> |
|Идентификатор:  <br/> |0x1091  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Разное  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство удаляется, если объект сообщения не помечен полным. Наименьшее разрешение времени должно быть минут (значение должно быть несколько 600 000 000). Это свойство не должно существовать, если объект является объектом, связанным с собранием, и не должен существовать на объекте задачи.
