# <a name="pidtagipmoutboxentryid-canonical-property"></a>Каноническое свойство PidTagIpmOutboxEntryId

  
  
**Применимо к**: Outlook 2013 | Outlook 2016 
  
Содержит идентификатор записи папке Исходящие стандартные сообщения электронной почты — это (IPM). 
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_IPM_OUTBOX_ENTRYID  <br/> |
|Идентификатор:  <br/> |0x35E2  <br/> |
|Тип данных:  <br/> |PT_BINARY  <br/> |
|Область:  <br/> |Folder  <br/> |
   
## <a name="remarks"></a>Замечания

Исходящие сообщения обычно создаются в папке «Исходящие». Сообщения электронной почты — это должны находиться в этой папке для отправки.
