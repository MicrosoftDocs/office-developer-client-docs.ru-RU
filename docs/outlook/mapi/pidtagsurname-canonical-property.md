# <a name="pidtagsurname-canonical-property"></a>Каноническое свойство PidTagSurname

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит фамилию или фамилию получателя.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_SURNAME, PR_SURNAME_A PR_SURNAME_W  <br/> |
|Идентификатор:  <br/> |0x3A11  <br/> |
|Тип данных:  <br/> |PT_STRING8 PT_UNICODE  <br/> |
|Область:  <br/> |Пользователь почты MAPI  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства предоставляют сведения о идентификации и доступе для получателя. Они определяются получателем и Организацией получателя.
