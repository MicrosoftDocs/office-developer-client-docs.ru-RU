# <a name="mapisvcinf-services-section"></a>Раздел MapiSvc. INF [службы]

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
В разделе **[Services]** перечислены службы сообщений, установленные на компьютере. В записях этого раздела используется следующий формат: 
  
 **Служб**
  
 _имя раздела "_ =  служба сообщений" имя_службы сообщений_
  
Имя раздела Message-Service — это строка, определяемая службой сообщений, которая связывает эту запись с соответствующим разделом для службы в другом месте в Mapisvc. INF. Имя службы messages — это имя установленной службы. В следующем разделе показаны три службы сообщений: адресная книга по умолчанию, моя собственная служба и служба хранилища сообщений. Эти службы выявляются вымышленными и предназначены только для примера. Каждый разработчик службы сообщений должен заменить соответствующую запись в службе сообщений в этом разделе.
  
```cpp
[Services]
AB=Default Address Book
MsgService=My Own Service
MS=Message Store Service

```

Каждая запись в этом разделе содержит соответствующий раздел, где хранятся сведения для службы сообщений. Например, соответствующий раздел для адресной книги по умолчанию называется [AB].
