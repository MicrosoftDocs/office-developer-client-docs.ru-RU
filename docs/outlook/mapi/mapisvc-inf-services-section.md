# <a name="mapisvcinf-services-section"></a>Раздел MapiSvc.inf [Services]

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
В **разделе [Services]** перечислены службы сообщений, установленные на компьютере. Записи в этом разделе используют следующий формат: 
  
 **[Services]**
  
 _Имя раздела "служба сообщений"_  =   _Имя службы сообщений_
  
Имя раздела служба сообщений — это строка, определяемая службой сообщений, которая связывает эту запись с соответствующим разделом для службы в другом месте в mapisvc.inf. Имя службы сообщений — это имя установленной службы. В следующем разделе показаны три службы сообщений: адресная книга по умолчанию, моя собственная служба и служба магазина сообщений. Эти службы являются вымышленными только для иллюстраций. Каждый реализующий службу сообщений заменит соответствующую запись для службы сообщений в этом разделе.
  
```cpp
[Services]
AB=Default Address Book
MsgService=My Own Service
MS=Message Store Service

```

Каждая запись в этом разделе имеет соответствующий раздел, в котором хранятся сведения для службы сообщений. Например, соответствующий раздел адресной книги по умолчанию называется [AB].
