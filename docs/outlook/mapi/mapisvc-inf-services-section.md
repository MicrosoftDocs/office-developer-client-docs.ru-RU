# <a name="mapisvcinf-services-section"></a>MapiSvc.inf: раздел [Службы]

  
  
**Относится к**: Outlook 
  
В разделе **[Services]** перечислены службы сообщений, установленные на компьютере. Записи в этом разделе используйте следующий формат: 
  
 **[Services]**
  
 _Имя раздела службы сообщений_ =  _имя службы сообщений_
  
Имя раздела службы сообщений — это строка, определенного службой сообщения для ссылки в этой записи в соответствующий раздел для службы в любом месте mapisvc.inf. Имя службы сообщений — это имя службы, установленные. В следующем разделе описываются три службы сообщений: адресной книги по умолчанию, мои собственные службы и службы хранилища сообщений. Эти службы, вымышленной только для иллюстрации. Разработчик службы каждого сообщения подставить средством для свой службы сообщений в этом разделе.
  
```cpp
[Services]
AB=Default Address Book
MsgService=My Own Service
MS=Message Store Service

```

Каждая запись в данном разделе имеет соответствующий раздел собственных хранения сведений о службе сообщений. Например соответствующий раздел для адресной книги по умолчанию называется [AB].
