# <a name="pidlidpostrsschannel-canonical-property"></a>Каноническое свойство PidLidPostRssChannel

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит содержимое \<заголовок\> из XML Atom \<веб-канала\> или RSS-канал \<канала\>.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidPostRssChannel  <br/> |
|Набор свойств:  <br/> |PSETID_PostRss  <br/> |
|Длинный идентификатор (КРЫШКА):  <br/> |0x00008904  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |RSS-КАНАЛ  <br/> |
