**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит содержимое поля заголовка из XML канала Atom или \< \> \< \> RSS-канала. \< \>
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidPostRssChannel  <br/> |
|Набор свойств:  <br/> |PSETID_PostRss  <br/> |
