# <a name="pidtagreceivedbyname-canonical-property"></a>Каноническое свойство PidTagReceivedByName

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Содержит отображаемое имя пользователя обмена сообщениями, который получает сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RECEIVED_BY_NAME, PR_RECEIVED_BY_NAME_A PR_RECEIVED_BY_NAME_W  <br/> |
|Идентификатор:  <br/> |0x0040  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства представляют собой пример свойств адреса для пользователя обмена сообщениями, который получает сообщение. Они должны быть заданы входящим поставщиком транспорта.
