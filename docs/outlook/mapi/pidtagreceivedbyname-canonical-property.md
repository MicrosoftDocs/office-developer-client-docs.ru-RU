# <a name="pidtagreceivedbyname-canonical-property"></a>Каноническое свойство PidTagReceivedByName

  
  
**Относится к**: Outlook 
  
Содержит отображаемое имя пользователя системы обмена сообщениями, который получает сообщение.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_RECEIVED_BY_NAME, PR_RECEIVED_BY_NAME_A, PR_RECEIVED_BY_NAME_W  <br/> |
|Идентификатор:  <br/> |0x0040  <br/> |
|Тип данных:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Область:  <br/> |Address  <br/> |
   
## <a name="remarks"></a>Замечания

Эти свойства — это пример свойств адрес для обмена мгновенными сообщениями пользователя, который получает сообщение. Им должен иметь значение входящих поставщика транспорта.
