# <a name="pidlidfax3emailaddress-canonical-property"></a>Каноническое свойство PidLidFax3EmailAddress

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Указывает адрес электронной почты другого Факс-адреса контакта.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |dispidFax3EmailAddress  <br/> |
|Набор свойств:  <br/> |PSETID_Address  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x000080D3  <br/> |
|Тип данных:  <br/> |PT_UNICODE  <br/> |
|Область:  <br/> |Контакт  <br/> |
   
## <a name="remarks"></a>Примечания

Это свойство, если оно задано, должно содержать отображаемое имя пользователя, за которым следует символ "@", а затем номер факса.
