# <a name="pidlidappointmentreplytime-canonical-property"></a>Каноническое свойство PidLidAppointmentReplyTime

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Указывает дату и время, когда участник ответил на полученное приглашение на собрание или обновление собрания.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |Диспидапптреплитиме  <br/> |
|Набор свойств:  <br/> |Псетид_аппоинтмент  <br/> |
|Длинный идентификатор (крышка):  <br/> |0x00008220  <br/> |
|Тип данных:  <br/> |PT_SYSTIME  <br/> |
|Область:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Комментарии

Значение должно быть указано в формате UTC.
