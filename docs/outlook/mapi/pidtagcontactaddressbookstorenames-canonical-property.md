# <a name="pidtagcontactaddressbookstorenames-canonical-property"></a>Каноническое свойство PidTagContactAddressBookStoreNames

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит отображаемую информацию о магазинах, связанных с адресной книгой.
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |PR_CONTAB_STORE_NAMES, PR_CONTAB_STORE_NAMES_A, PR_CONTAB_STORE_NAMES_W  <br/> |
|Идентификатор:  <br/> |0x6622  <br/> |
|Тип данных:  <br/> |PT_MV_UNICODE, PT_MV_STRING8  <br/> |
|Область:  <br/> |Адресная книга контактов  <br/> |
   
## <a name="remarks"></a>Примечания

Эти свойства находятся в разделе Профиль адресной книги контактов.
