## <a name="description"></a>Описание

Это состояние означает инициируется один из двух процессов репликации: отправка содержимого заданной папки локального хранилища или полной синхронизации. В полной синхронизации для каждой указанной папки содержимое сначала загружаются и загружаются. В зависимости от *ulFlags* в структуре соответствующего **[СИНХРОНИЗАЦИИ](sync.md)** в предыдущем установите синхронизировать состояние, Outlook инициализирует [элементы в структуре **SYNCCONT** для предоставления сведений о содержимом out]. 
  
По той же структурой **SYNCCONT** клиент получает число папок, для которых материалы для загрузки или отправки. Клиент будет выполняют цикл по каждой из этих папок перемещение локального хранилища в состояние таблицы отправить отправка в папку и перемещение локального хранилища в таблице состояние загрузки можно загрузить в папку. 
  
Кроме того клиент получает запись идентификаторы для папки, требующие репликации.
  
По окончании этого состояния Outlook очищает внутренние сведения. Локальное хранилище возвращается в состояние синхронизации.
