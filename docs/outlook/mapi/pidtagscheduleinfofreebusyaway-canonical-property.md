# <a name="pidtagscheduleinfofreebusyaway-canonical-property"></a>Каноническое свойство PidTagScheduleInfoFreeBusyAway

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Содержит время, в течение которого в качестве состояния занятости устанавливается значение "отсутствие на работе".
  
|||
|:-----|:-----|
|Связанные свойства:  <br/> |ПР_СЧДИНФО_ФРИБУСИ_УФ  <br/> |
|Идентификатор:  <br/> |0x6856  <br/> |
|Тип данных:  <br/> |PT_MV_BINARY  <br/> |
|Область:  <br/> |Сведения о доступности  <br/> |
   
## <a name="remarks"></a>Комментарии

Формат, вычисления и ограничения этого свойства такие же, как и в **пр_счдинфо_фрибуси_тентативе** ([PidTagScheduleInfoFreeBusyTentative](pidtagscheduleinfofreebusytentative-canonical-property.md)), но ссылаются на встречи, помеченные как неиспользуемые для соответствующего календаря.
