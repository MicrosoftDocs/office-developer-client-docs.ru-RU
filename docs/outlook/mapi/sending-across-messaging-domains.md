# <a name="sending-across-messaging-domains"></a>Отправка между доменами обмена сообщениями

  
  
**Относится к**: Outlook 2013 | Outlook 2016 
  
Домен обмена сообщениями представляет одну или несколько систем обмена сообщениями, которые имеют общий формат адресов. Обмен сообщениями между несколькими доменами обмена сообщениями включает перевод сообщения, отправленного в формате исходного домена обмена сообщениями, в формат конечного домена обмена сообщениями. Так как не все форматы адресов совместимы, шлюз необходим для перевода адресных данных из формата источника в формат назначения. Чтобы обеспечить действительность для доменов обмена сообщениями, клиентские приложения хранят важные сведения об адресовке в свойствах MAPI. Кроме того, шлюзы выполняют перевод, проверяя свойства, которые, как известно, требуют перевода, и меняя их на формат, который может использовать домен обмена сообщениями назначения.
  
Ранее MAPI позволял связывать эту адресную информацию только с пользователями, составляющими текущий список получателей сообщения. Свойства, описывающие каждого участника списка получателей, были переведены шлюзом, чтобы обеспечить правильность их работы в доменах обмена сообщениями. Однако некоторые приложения требуют, чтобы их сообщения включали адресную информацию о пользователях, которые, возможно, были получателями в прошлом, будут получателями в будущем или никогда не будут получателями. Например, приложения маршрутинга, которые отправляют сообщения группе пользователей в указанном порядке, встраив сведения об этих пользователях в сообщения. Внедренные сведения обычно включают адрес и тип адреса будущих получателей, а также, возможно, их роли и позиции в порядке маршрутки, их имена и один или несколько двоичных идентификаторов для каждого получателя.
  
Чтобы сообщения включали сведения об этих нереципличных пользователях, MAPI теперь включает стратегию для обеспечения правильного перевода этих нерефициентных сведений между доменами обмена сообщениями. Эта стратегия основана на концепции свойств, которые можно смоблить шлюзом.
