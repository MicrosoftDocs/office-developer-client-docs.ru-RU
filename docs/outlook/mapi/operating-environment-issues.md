# <a name="operating-environment-issues"></a>Проблемы с операционной средой

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Проблемы операционной системы и операционной среды важны, когда разработчики MAPI выбирают среду разработки и среду для установки Windows SDK.
  
Разработчики MAPI должны знать, что эти варианты влияют на операционную систему, в которой работает компонент, и на диапазон компонентов, которые они могут нацелить.
