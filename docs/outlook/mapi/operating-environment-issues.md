# <a name="operating-environment-issues"></a>Проблемы с операционной средой

  
  
**Область применения**: Outlook 2013 | Outlook 2016 
  
Проблемы операционной системы и операционной среды важны, когда разработчики MAPI выбирают среду разработки и среду для Windows установки SDK.
  
Разработчики MAPI должны знать, что эти решения влияют на операционную систему, на которую будет работать их компонент, и на диапазон компонентов, на которые они смогут нацелить.
