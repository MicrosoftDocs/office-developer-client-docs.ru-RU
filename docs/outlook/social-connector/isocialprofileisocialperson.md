# <a name="isocialprofile--isocialperson"></a>ISocialProfile : ISocialPerson

Представляет зарегистрированного пользователя. 
  
## <a name="members"></a>"Участники"

В следующей таблице показаны участники, доступные в **интерфейсе ISocialProfile.** 
  
|**Имя**|**Тип участника**|**Описание**|
|:-----|:-----|:-----|
|[AreFriendsOrColleagues](isocialprofile-arefriendsorcolleagues.md) <br/> |Метод  <br/> |Определяет, являются ли указанные пользователи друзьями.  <br/> |
|[GetActivitiesOfFriendsAndColleagues](isocialprofile-getactivitiesoffriendsandcolleagues.md) <br/> |Метод  <br/> |Этот метод был обесценив с Outlook Social Connector 2013.  <br/> |
|[SetStatus](isocialprofile-setstatus.md) <br/> |Метод  <br/> |Этот метод в настоящее время не поддерживается.  <br/> |
   
## <a name="remarks"></a>Примечания

Поставщик Outlook social Connector (OSC) должен реализовать этот интерфейс для связи с OSC.
