# <a name="isocialsessionloggedonuserid"></a>ISocialSession::LoggedOnUserID

Возвращает строку, представляющую идентификатор пользователя социальной сети пользователя, вошедшего в систему. 
  
```cpp
[propget] HRESULT _stdcall LoggedOnUserID([out, retval] BSTR* result);
```

## <a name="property-value"></a>Значение свойства

Строка, содержащая идентификатор пользователя социальной сети пользователя, вошедшего в систему.
