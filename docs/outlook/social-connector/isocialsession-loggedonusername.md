# <a name="isocialsessionloggedonusername"></a>ISocialSession::LoggedOnUserName

Возвращает строку, представляющую имя пользователя, который используется при входе в систему.
  
```cpp
[propget] HRESULT _stdcall LoggedOnUserName([out, retval] BSTR* result);
```

## <a name="property-value"></a>Значение свойства

Строка, представляющая имя пользователя, вошедшего в систему пользователя.
