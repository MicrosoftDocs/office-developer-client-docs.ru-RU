# <a name="applying-a-sample-provider-template"></a>Применение образец шаблона поставщика

Образцы шаблонов поставщика Outlook Social Connector (OSC) предоставить структуру для реализации поставщика OSC. Шаблоны поставщика доступны на C++, C# и Visual Basic. Эти шаблоны, просто отправной точки для разработки поставщика; шаблоны, не относятся к написанию кода реализации поставщика и создание пакета установки для поставщика. Следующей процедуре показано, как применить шаблон поставщика OSC при начале разработки поставщика.
  
### <a name="to-apply-an-osc-provider-template"></a>Чтобы применить шаблон поставщика OSC

1. В меню **Пуск** щелкните правой кнопкой мыши **Microsoft Visual Studio 2010** и выберите команду **Запуск от имени администратора** . При появлении запроса нажмите кнопку **Да** для запуска Visual Studio с правами администратора. 
    
2. Измените имя проекта и пространства имен в шаблоне на идентификаторов и пространства имен проекта.
    
3. Измените класс **AssemblyInfo** для определения информации, соответствующей сборки. 
    
4. Реализация членов интерфейса, помеченные как **задачи** и добавьте дополнительные справочные материалы и зависимостей при необходимости. 
    
5. Выполните построение проекта.
    
6. Убедитесь, что программный идентификатор сборки поставщика из списка в разделе реестра `HKEY_CURRENT_USER\Software\Microsoft\Office\Outlook\SocialConnector\SocialProviders`.
    
7. Чтобы распространить проект программы установки, создайте проект программы установки в Visual Studio или средство установки собственное.
    
8. Проект программы установки следует завершения регистрации COM для сборки и также создать ProgID ключ, указанный в шаге 5.
