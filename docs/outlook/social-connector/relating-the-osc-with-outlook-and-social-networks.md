<span data-ttu-id="54855-108">Для поддержки общего доступа к функциям в клиентских приложениях Office ядро компонента OSC реализовано как часть общего компонента Office, а область пользователей реализована как надстройка Outlook.</span><span class="sxs-lookup"><span data-stu-id="54855-108">To support sharing of functionality across Office client applications, the OSC core engine is implemented as part of an Office shared component, and the People Pane is implemented as an Outlook add-in.</span></span> <span data-ttu-id="54855-109">Чтобы использовать OSC, пользователь Office должен установить Outlook на этом клиентском компьютере и настроить Outlook с использованием профиля, чтобы OSC мог кэшировать контакты в папке "Контакты".</span><span class="sxs-lookup"><span data-stu-id="54855-109">To use the OSC, an Office user must have installed Outlook on that client computer and configured Outlook with a profile, so that the OSC can cache contacts in a Contacts folder.</span></span> 
  
<span data-ttu-id="54855-110">Поставщик OSC — это DLL-библиотека COM, которая позволяет OSC получать доступ к данным социальных сетей так, как это не зависит от API каждой социальной сети.</span><span class="sxs-lookup"><span data-stu-id="54855-110">An OSC provider is a Component Object Model (COM) DLL that allows the OSC to access social network data in a way that is independent of the APIs of each social network.</span></span> <span data-ttu-id="54855-111">Библиотека DLL поставщика OSC должна быть установлена локально на клиентском компьютере.</span><span class="sxs-lookup"><span data-stu-id="54855-111">An OSC provider DLL must be installed locally on a client computer.</span></span> <span data-ttu-id="54855-112">Поставщик OSC в социальной сети подключается к OSC, который входит в состав Outlook, с социальными сетями в Интернете.</span><span class="sxs-lookup"><span data-stu-id="54855-112">A social network's OSC provider connects the OSC, which is part of Outlook, with the social network on the Internet.</span></span>
  
<span data-ttu-id="54855-113">Поставщик OSC должен реализовать набор интерфейсов, определенных как часть расширяемости поставщика OSC, для подключения к OSC.</span><span class="sxs-lookup"><span data-stu-id="54855-113">An OSC provider must implement a set of interfaces, defined as part of the OSC provider extensibility, to communicate with the OSC.</span></span> <span data-ttu-id="54855-114">Расширяемость поставщика OSC доступна в виде открытой платформы.</span><span class="sxs-lookup"><span data-stu-id="54855-114">OSC provider extensibility is available as an open platform.</span></span>
  
<span data-ttu-id="54855-115">Архитектура поставщика OSC позволяет нескольким поставщикам работать с ядром компонента OSC и собирать социальные сведения, такие как друзья и действия.</span><span class="sxs-lookup"><span data-stu-id="54855-115">The provider architecture of the OSC enables multiple providers to work with the OSC core engine and aggregate social information such as friends and activities.</span></span> <span data-ttu-id="54855-116">На рисунке 1 показана архитектура поставщика OSC.</span><span class="sxs-lookup"><span data-stu-id="54855-116">Figure 1 illustrates the OSC provider architecture.</span></span>
