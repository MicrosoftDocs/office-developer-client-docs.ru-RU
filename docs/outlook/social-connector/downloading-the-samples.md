# <a name="downloading-the-samples"></a>Скачивание примеров

В этом разделе описывается, как можно загрузить примеры кода, которые прилагаются к этой документации.
  
### <a name="to-download-the-sample-code-installation-package"></a>Чтобы загрузить образец кода установки пакета

1. Загрузите файл OSCProviderSamples.zip из [Outlook Social Connector 2013: шаблоны поставщика](http://code.msdn.microsoft.com/Outlook-Social-Connector-73fd8d2c).
    
2. Извлеките ZIP-файл в папку. В Windows 7 путь по умолчанию для Visual Studio 2010 — \Documents\Visual _пользователя_C:\Users\ Studio 2010\Projects.
    
3. После извлечения ZIP-файл, вы найдете следующие проекты в папку проектов:
    
   - OSCProvider_CPP — содержит шаблон поставщика C++.
    
   - OSCProvider_CS — содержит шаблон поставщика C#.
    
   - OSCProvider_VB — содержит шаблон поставщика Visual Basic.
