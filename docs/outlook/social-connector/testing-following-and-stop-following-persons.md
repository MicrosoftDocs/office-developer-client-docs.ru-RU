# <a name="testing-following-and-stop-following-persons"></a>Тестирование подписки на пользователей и ее отмены

В этом разделе описываются сценарии проверки способности поставщика Outlook social Connector (OSC) следовать за другом и прекратить следовать за другом в социальной сети.
  
## <a name="following-a-person"></a>Следуя за человеком

Чтобы следовать за человеком, следует добавить человека в качестве друга в социальной сети с помощью SMTP-адреса, предоставленного выбранным элементом Outlook. После кого-то в социальной сети, как правило, требует разрешения от этого человека по электронной почте, что SMTP-адрес. Чтобы предоставить разрешение, этот человек должен либо иметь этот SMTP-адрес, уже включенный в его учетную запись в социальной сети, либо быть готовым добавить этот SMTP в учетную запись социальной сети. Проверьте каждый из следующих сценариев, чтобы убедиться, что поставщик OSC ведет себя правильно.
  
|**Сценарий**|**Ожидаемое поведение**|
|:-----|:-----|
|Попытка следовать за человеком в социальной сети, который существует в социальной сети.  <br/> |Для социальной сети, которая не требует разрешения от человека, социальная сеть немедленно добавляет человека в качестве друга.  <br/> Для сети, которая требует разрешения от этого человека, социальная сеть отправляет уведомление. В Outlook для этого человека отображается ожидаемая значок.  <br/> |
|Попытка следовать за человеком в социальной сети, который не существует в социальной сети.  <br/> |Поставщик OSC отображает соответствующую ошибку [в ISocialSession::FollowPerson](isocialsession-followperson.md) или [ISocialSession2::FollowPersonEx](isocialsession2-followpersonex.md).  <br/> |
|Следуя за другом в социальной сети.  <br/> |Для друга, выбранного в области people, отображаются значок социальной сети и изображение профиля друга для этой социальной сети.  <br/> |
|Выбор ссылки на страницу профиля друга.  <br/> |Страница друга в социальной сети открывается в браузере по умолчанию пользователя.  <br/> |
   
## <a name="stop-following-a-person"></a>Остановите следовать за человеком

Чтобы перестать следовать за человеком, следует удалить его в качестве друга в социальной сети. Проверьте следующий сценарий, чтобы убедиться, что поставщик OSC перестает правильно следовать за человеком.
  
|**Сценарий**|**Ожидаемое поведение**|
|:-----|:-----|
|Попытка удалить человека в качестве друга в социальной сети.  <br/> |Социальная сеть больше не перечисляет этого человека в качестве друга в журнале учетной записи пользователя.  <br/> |
