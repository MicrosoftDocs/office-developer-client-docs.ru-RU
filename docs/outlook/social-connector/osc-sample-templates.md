# <a name="osc-sample-templates"></a>Шаблоны образцов OSC

Сопровождающие ссылку Outlook social Connector 2013— несколько шаблонов поставщиков в C++, C# и Visual Basic. Шаблоны можно скачать в [Outlook Social Connector 2013: Provider templates](https://code.msdn.microsoft.com/Outlook-Social-Connector-73fd8d2c).
  
Шаблоны поставщика доступны для использования при создании поставщика на языке по вашему выбору. Чтобы свести к минимуму размер загрузки поставщика и свести к минимуму время загрузки поставщика, необходимо создать поставщика с помощью C++. Если вы разрабатываете поставщика для пользователей с платформа .NET Framework 3,5 SP1 или более поздней версией платформа .NET Framework, установленной на их компьютерах, вы также можете написать управляемого поставщика с помощью C# или Visual Basic.
