# <a name="osc-provider-xml-examples"></a>Примеры XML поставщика OSC

В этом разделе содержатся примеры XML, использующих **возможности**, **друзей**и **activityFeed** элементы XML, соответствие схеме XML поставщика Outlook Social Connector (OSC). 
  
- [Пример XML возможности](capabilities-xml-example.md) представлены **возможности** XML поставщика OSC, указав его возможности и требования для OSC. 
    
- [Друзья XML](friends-xml-example.md) пример **друзья** XML для двух пользователей в социальных сетях. 
    
- [Пример XML-файла веб-канала активности](activity-feed-xml-example.md) показывает **activityFeed** XML четыре действия: два профиля обновления изображения, обновление состояния и запись блога. XML-код также приведены три активности шаблоны отображения для отображения соответствующими действиями.
