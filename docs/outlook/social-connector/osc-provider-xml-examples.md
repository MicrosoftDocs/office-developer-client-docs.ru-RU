# <a name="osc-provider-xml-examples"></a>Примеры XML поставщика OSC

В этом разделе представлены примеры XML, которые используют элементы XML-элементов, друзья и **activityFeed,** соответствующие схеме XML Outlook поставщика социальных соединителем (OSC). 
  
- [В примере возможностей XML](capabilities-xml-example.md) показаны возможности **XML** поставщика OSC, указывая его возможности и требования к OSC. 
    
- [В примере Friends XML](friends-xml-example.md) показано, как друзья XML для двух пользователей в социальной сети.  
    
- [В примере Канала действий XML](activity-feed-xml-example.md) показан XML **activityFeed** из четырех действий: двух обновлений изображений профиля, обновления состояния и сообщения в блоге. XML также указывает три шаблона отображения действий для отображения соответствующих действий.
