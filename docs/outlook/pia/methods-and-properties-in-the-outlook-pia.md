<span data-ttu-id="fbcdc-p101">При создании PIA в Outlook применяется программа импорта библиотек типов (TLBIMP) в .NET Framework, преобразующая определения типов в библиотеке типа COM в эквивалентные определения из сборки среды CLR. - <span data-ttu-id="fbcdc-107">основной интерфейс (например, интерфейс [\_FormRegion](https://msdn.microsoft.com/library/bb645761\(v=office.15\)));</span><span class="sxs-lookup"><span data-stu-id="fbcdc-107">The primary interface (for example, the _FormRegion interface).</span></span>

<span data-ttu-id="fbcdc-109">TLBIMP импортирует основной интерфейс и интерфейс события для каждого объекта и создает ряд интерфейсов, делегатов и классов, среди которых встречаются следующие:</span><span class="sxs-lookup"><span data-stu-id="fbcdc-109">TLBIMP imports the primary interface and the event interface for each object and creates a number of interfaces, delegates, and classes, among which are the following:</span></span>

- <span data-ttu-id="fbcdc-110">интерфейс события .NET (например, интерфейс [FormRegionEvents\_Event](https://msdn.microsoft.com/library/bb647619\(v=office.15\)));</span><span class="sxs-lookup"><span data-stu-id="fbcdc-110">The .NET event interface (for example, the FormRegionEvents_Event interface).</span></span>

- <span data-ttu-id="fbcdc-111">класс .NET (например, класс [FormRegionClass](https://msdn.microsoft.com/library/bb624204\(v=office.15\)));</span><span class="sxs-lookup"><span data-stu-id="fbcdc-111">The .NET class (for example, the [FormRegionClass](https://msdn.microsoft.com/library/bb624204\(v=office.15\)) class).</span></span>
