# <a name="electronic-business-cards"></a>Электронные визитные карточки

В этом разделе представлены примеры задач, в которых используются электронные визитные карточки. Электронные визитные карточки — это настраиваемые подписи, которые вы можете включать в свои электронные письма. С помощью электронных визитных карточек можно легко передавать свои контактные данные другим пользователям.
