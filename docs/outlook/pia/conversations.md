# <a name="conversations"></a>Conversations

В этом разделе приведены примеры задач, в которых используются беседы. Беседы в Outlook представлены объектом [Conversation](https://msdn.microsoft.com/library/ff184711\(v=office.15\)) и включают один или несколько элементов в папке, которые совместно используют один и тот же заголовок темы или цепочку.
