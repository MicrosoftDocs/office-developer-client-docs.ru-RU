# <a name="data-types-outlook-exported-apis"></a>Типы данных (экспортированные API Outlook)

В этом разделе описываются типы данных, экспортируемые в Outlook.
  
|**Тип данных**|**Описание**|
|:-----|:-----|
|[TZDEFINITION](tzdefinition.md) <br/> |Представляет весь часовой пояс, включая все правила сдвига в прошлом, текущих и будущих часовых поясов для летнего времени.  <br/> |
|[TZREG](tzreg.md) <br/> |Определяет, когда происходит переход на летнее время, когда происходит возврат к стандартному времени и сколько часов сдвига на летнее время.  <br/> |
|[TZRULE](tzrule.md) <br/> |Задает сведения о том, когда начинается летнее время, а также год вступления в силу правила с часовым поясом.  <br/> |
