# <a name="data-types-outlook-exported-apis"></a>Типы данных (Outlook экспортировать API-интерфейсы)

В этом разделе описываются типы данных, которые экспортируются Outlook.
  
|**Тип данных**|**Описание**|
|:-----|:-----|
|[TZDEFINITION](tzdefinition.md) <br/> |Представляет всей часового пояса, включая все правила shift журналу, текущие и будущие часового пояса на летнее время.  <br/> |
|[TZREG](tzreg.md) <br/> |Определяет при запуске летнего времени, когда происходит возврат стандартного времени и сколько часов — это летнего shift.  <br/> |
|[TZRULE](tzrule.md) <br/> |Задает сведения о при запуске летнее время и года, в котором это правило часового пояса сначала вступает в силу для часового пояса.  <br/> |
