# <a name="propsmtpssl"></a>PROP_SMTP_SSL

Указывает, следует ли использовать протокол Secure сокетов протокол SSL для учетной записи SMTP.
  
## <a name="quick-info"></a>Краткие сведения

|||
|:-----|:-----|
|Идентификатор:  <br/> |0x0202  <br/> |
|Тип свойства:  <br/> |PT_DWORD  <br/> |
|Свойство tag:  <br/> |0x02020003  <br/> |
|Access:  <br/> |Только для чтения  <br/> |
   
## <a name="remarks"></a>Замечания

Нуль означает значение не использовать SSL-шифрование, в противном случае использовать SSL-шифрование.
