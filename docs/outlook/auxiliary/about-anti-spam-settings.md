# <a name="about-anti-spam-settings"></a>Сведения о параметрах защиты от нежелательной почты

Outlook позволяет пользователям задавать параметры для каждой учетной записи, чтобы защитить учетную запись от спама. Такие параметры защиты от нежелательной почты хранятся в разделе, предназначенном для этой учетной записи в профиле пользователя. Используйте свойство [PROP_ACCT_PREFERENCES_UID](prop_acct_preferences_uid.md) , чтобы получить уникальный идентификатор (UID) для раздела в профиле, в котором хранятся параметры пользователя для этой учетной записи, в том числе параметры защиты от нежелательной почты. 
  
Используйте следующие свойства для получения параметров защиты от нежелательной почты для учетной записи:
  
- [PidTagSpamJunkSenders](https://msdn.microsoft.com/library/3c5182a7-7d7a-48e8-b9cb-5abd7739f0fd%28Office.15%29.aspx)— указывает список адресов электронной почты и доменов, разделенных точкой с запятой, которые пользователь указал как заблокированные отправители для учетной записи.
    
- [PidTagSpamThreshold](https://msdn.microsoft.com/library/2b2d6b8e-e3dd-4a9b-8bb5-53add675605d%28Office.15%29.aspx)— задает уровень фильтрации нежелательной почты, который пользователь указал для учетной записи. В следующей таблице приведены поддерживаемые значения.
    
|Поддерживаемое значение |Определение |
|:-----|:-----|
|Отсутствует  <br/> |Равен  <br/> |
|Низкий  <br/> |0x00000006  <br/> |
|Средние  <br/> |0x00000005  <br/> |
|Высокий  <br/> |0x00000003  <br/> |
   
- [PidTagSpamTrustedRecipients](https://msdn.microsoft.com/library/59f43316-3ff6-4ed0-bc29-b31039192b08%28Office.15%29.aspx)— указывает список адресов электронной почты и доменов, разделенных точкой с запятой, которые пользователь указал как доверенных получателей для учетной записи.
    
- [PidTagSpamTrustedSenders](https://msdn.microsoft.com/library/8e3f0094-e64b-4828-ba8f-5eed35f85366%28Office.15%29.aspx)— указывает список адресов электронной почты и доменов, разделенных точкой с запятой, которые пользователь указал как доверенные отправители для учетной записи.
