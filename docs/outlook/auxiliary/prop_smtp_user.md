# <a name="propsmtpuser"></a>PROP_SMTP_USER

Представляет имя пользователя для учетной записи SMTP.
  
## <a name="quick-info"></a>Краткие сведения

|||
|:-----|:-----|
|Идентификатор:  <br/> |0x0204  <br/> |
|Тип свойства:  <br/> |PT_UNICODE  <br/> |
|Свойство tag:  <br/> |0x0204001F  <br/> |
|Access:  <br/> |Только для чтения  <br/> |
