# <a name="interfaces-freebusy-api"></a>Интерфейсы (API free/busy)

В этом разделе описываются интерфейсы aPI Free/Busy.
  
|**Интерфейс**|**Описание**|
|:-----|:-----|
|[IEnumFBBlock](ienumfbblock.md) <br/> |Поддерживает доступ и переумеживающие свободные и загруженные блоки данных для пользователя в пределах диапазона времени.  <br/> |
|[IFreeBusyData](ifreebusydata.md) <br/> |Для данного пользователя получает и задает диапазон времени и возвращает интерфейс для перенамерения свободных и загруженных блоков данных в этом диапазоне времени.  <br/> |
|[IFreeBusySupport](ifreebusysupport.md) <br/> |Поддерживает спецификацию интерфейсов, которые имеют доступ к свободным и загруженным данным для указанных пользователей.  <br/> |
