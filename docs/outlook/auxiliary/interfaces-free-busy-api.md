# <a name="interfaces-freebusy-api"></a>Интерфейсы (API занятости)

В этом разделе описываются интерфейсы API свободен/занят.
  
|**Интерфейс**|**Описание**|
|:-----|:-----|
|[IEnumFBBlock](ienumfbblock.md) <br/> |Поддерживает доступ к и перечисление занятости блоков данных для пользователя в интервал времени.  <br/> |
|[IFreeBusyData](ifreebusydata.md) <br/> |Для определенного пользователя, получает и задает диапазон времени и возвращает интерфейс для перечисления сведений о доступности блоков данных в пределах этого диапазона времени.  <br/> |
|[IFreeBusySupport](ifreebusysupport.md) <br/> |Поддерживает спецификацию интерфейсов, доступ к сведениям о доступности данных для указанных пользователей.  <br/> |
