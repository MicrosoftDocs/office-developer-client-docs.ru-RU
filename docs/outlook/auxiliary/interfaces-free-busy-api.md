# <a name="interfaces-freebusy-api"></a>Интерфейсы (API занятости)

В этом разделе описаны интерфейсы в API занятости.
  
|**Интерфейс**|**Описание**|
|:-----|:-----|
|[IEnumFBBlock](ienumfbblock.md) <br/> |Поддерживает доступ к блокам данных о доступности и их нумерации для пользователя в течение задающегося диапазона времени.  <br/> |
|[IFreeBusyData](ifreebusydata.md) <br/> |Для данного пользователя возвращает и задает диапазон времени и возвращает интерфейс для нумерации блоков данных о занятости в пределах этого диапазона времени.  <br/> |
|[IFreeBusySupport](ifreebusysupport.md) <br/> |Поддерживает спецификацию интерфейсов, которые имеют доступ к сведениям о доступности для указанных пользователей.  <br/> |
