# <a name="interfaces-freebusy-api"></a>Интерфейсы (API сведений о доступности)

В этом разделе описываются интерфейсы API сведений о доступности.
  
|**Интерфейс**|**Описание**|
|:-----|:-----|
|[IEnumFBBlock](ienumfbblock.md) <br/> |Поддерживает доступ и перечисление блоков сведений о занятости для пользователя в диапазоне времени.  <br/> |
|[IFreeBusyData](ifreebusydata.md) <br/> |Для определенного пользователя возвращает и задает диапазон времени и возвращает интерфейс для перечисления блоков сведений о занятости данных в пределах этого диапазона времени.  <br/> |
|[IFreeBusySupport](ifreebusysupport.md) <br/> |Поддерживает спецификацию интерфейсов, которые обращаются к данным о занятости для указанных пользователей.  <br/> |
