# <a name="prop_smtp_secure_connection"></a>PROP_SMTP_SECURE_CONNECTION

Указывает тип зашифрованного подключения, используемого для учетной записи SMTP.
  
## <a name="quick-info"></a>Краткие сведения

|||
|:-----|:-----|
|Идентификатор:  <br/> |0x020A  <br/> |
|Тип свойства:  <br/> |PT_DWORD  <br/> |
|Тег свойства:  <br/> |0x020A0003  <br/> |
|Обращения  <br/> |Только для чтения  <br/> |
   
## <a name="remarks"></a>Примечания

Значение может быть одной из следующих констант. В этом разделе приведены [константы (API управления учетными записями)](constants-account-management-api.md) для их значений. 
  
|**Константы**|**Описание**|
|:-----|:-----|
|**ENCRYPT_CONN_NO_SECURITY** <br/> |Не используйте шифрование.  <br/> |
|**ENCRYPT_CONN_SSL** <br/> |Используйте шифрование SSL.  <br/> |
|**ENCRYPT_CONN_TLS** <br/> |Используйте шифрование TLS и протокол проверки подлинности.  <br/> |
|**ENCRYPT_CONN_AUTO** <br/> |Автоматически определять и использовать метод шифрования, поддерживаемый почтовым сервером.  <br/> |
