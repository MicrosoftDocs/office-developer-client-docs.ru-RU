|**Function**|**Описание**|
|:-----|:-----|

|[HrProcessConvActionForSentItem](hrprocessconvactionforsentitem.md) <br/> |Выполняет категоризацию почтового элемента после отправки на основе [его PidTagConversationId.](https://msdn.microsoft.com/library/f8e4a5fa-cb73-4eca-b174-72e1fda821a6%28Office.15%29.aspx)  <br/> |
|[RebaseTaskComplete](rebasetaskcomplete.md) <br/> |Сообщает о завершении повторного сбоя встреч.  <br/> |
|[RebaseTaskProgress](rebasetaskprogress.md) <br/> |Сообщает о ходе выполнения для нумерации и повторного выполнения встреч.  <br/> |
