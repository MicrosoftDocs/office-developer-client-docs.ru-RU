# <a name="ifreebusydataplaceholder4"></a>IFreeBusyData::Placeholder4

Этот элемент является заполнителем и не поддерживается.
  
## <a name="quick-info"></a>Краткие сведения

Обратитесь к разделу [ифрибусидата](ifreebusydata.md).
  
```cpp
HRESULT Placeholder4( 
     LONG, 
     LONG,  
     LONG,  
     BOOL, 
     LONG, 
     LONG,  
     LONG, 
     void * 
);

```

## <a name="return-values"></a>Возвращаемые значения

Всегда S_OK, так как это заполнитель и не поддерживается.
  
## <a name="remarks"></a>Примечания

Этот элемент должен возвращать S_OK.
