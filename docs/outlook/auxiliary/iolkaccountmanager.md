# <a name="iolkaccountmanager"></a>IOlkAccountManager

Управляет доступом к учетным записям и выполняет настройку уведомлений о изменения учетной записи.

|||
|:-----|:-----|
|Наследует от:  <br/> |[IOlkErrorUnknown](iolkerrorunknown.md) <br/> |
|Реализованный:  <br/> |Outlook  <br/> |
|Предоставлено:  <br/> |CLSID_OlkAccountManager  <br/> |
|Вызывается:  <br/> |Клиент  <br/> |
|Идентификатор интерфейса:  <br/> |IID_IOlkAccountManager  <br/> |
