# <a name="prop_smtp_use_spa"></a>PROP_SMTP_USE_SPA

Указывает, следует ли использовать проверку подлинности безопасного пароля (SPA) для учетной записи SMTP.
  
## <a name="quick-info"></a>Краткие сведения

|||
|:-----|:-----|
|Идентификатор:  <br/> |0x0207  <br/> |
|Тип свойства:  <br/> |PT_DWORD  <br/> |
|Тег свойства:  <br/> |0x02070003  <br/> |
|Доступ:  <br/> |Только для чтения  <br/> |
   
## <a name="remarks"></a>Примечания

Нулевое значение означает, что не используйте SPA, в противном случае используйте SPA.
