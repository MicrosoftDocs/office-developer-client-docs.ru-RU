# <a name="named-properties-account-management-api"></a>Именованные свойства (API управления учетными записями)

В этом разделе описываются именованные свойства в API управления учетными записями.
  
|**Именованное свойство**|**Описание**|
|:-----|:-----|
|[PidLidInternetAccountName](pidlidinternetaccountname.md) <br/> |Возвращает отображаемое имя учетной записи, которая доставила сообщение.  <br/> |
|[PidLidInternetAccountStamp](pidlidinternetaccountstamp.md) <br/> |Возвращает метку учетной записи, которая доставила сообщение.  <br/> |
