# <a name="named-properties-account-management-api"></a>Именуемые свойства (API управления учетной записью)

В этом разделе описываются именуемые свойства в API управления учетной записью.
  
|**Именованое свойство**|**Описание**|
|:-----|:-----|
|[PidLidInternetAccountName](pidlidinternetaccountname.md) <br/> |Возвращает отображаемую имя учетной записи, которая доставляла сообщение.  <br/> |
|[PidLidInternetAccountStamp](pidlidinternetaccountstamp.md) <br/> |Возвращает отметку учетной записи, которая доставляла сообщение.  <br/> |
