# <a name="prop_inet_use_spa"></a>PROP_INET_USE_SPA

Указывает, следует ли использовать проверку подлинности безопасного пароля (SPA) для общего почтового ящика в Интернете.
  
## <a name="quick-info"></a>Краткие сведения

|||
|:-----|:-----|
|Идентификатор:  <br/> |0x0108  <br/> |
|Тип свойства:  <br/> |PT_DWORD  <br/> |
|Тег свойства:  <br/> |0x01080003  <br/> |
|Доступ:  <br/> |Только для чтения  <br/> |
   
## <a name="remarks"></a>Примечания

Нулевое значение означает, что не используйте SPA, в противном случае используйте SPA.
