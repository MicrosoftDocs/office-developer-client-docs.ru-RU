# <a name="about-the-data-degradation-layer-api"></a>Сведения об API уровня ухудшения данных

API уровня ухудшения данных позволяет скрыть в основном основном формате символов объекта Outlook, позволяя клиенту получить доступ к объекту в предпочитаемом формате символов, а не в формате символов в основном объекте. Он предоставляет определения констант, показанные в [Constants (API](constants-data-degradation-layer-api.md)уровня ухудшения данных) и функцию [HrCreateNewWrappedObject,](hrcreatenewwrappedobject.md) которая возвращает объект Outlook, упакованный в указанный формат.
