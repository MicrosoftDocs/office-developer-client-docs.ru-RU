# <a name="about-the-data-degradation-layer-api"></a>Сведения об API уровня ухудшения данных

API уровня падения данных позволяет скрывать основной формат символов объекта Outlook, позволяя клиенту получать доступ к объекту в предпочтительном символьном формате, а не в собственном формате символов объекта. Он предоставляет определения констант, показанные в [константах (API уровня замедленИя данных)](constants-data-degradation-layer-api.md), и функцию [хркреатеневвраппедобжект](hrcreatenewwrappedobject.md) , возвращающую объект Outlook, упакованный в заданном формате.
