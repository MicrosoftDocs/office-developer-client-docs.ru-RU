# <a name="about-the-data-degradation-layer-api"></a>Сведения об API уровня ухудшения данных

API уровня деградации данных предоставляет возможность скрыть родной формат объекта Outlook, позволяя клиенту получить доступ к объекту в предпочтительном формате символов, а не в формате родного символа объекта. Он предоставляет постоянные определения, показанные в [Constants (API](constants-data-degradation-layer-api.md)уровня деградации данных) и функцию [HrCreateNewWrappedObject,](hrcreatenewwrappedobject.md) которая возвращает объект Outlook, завернутый в указанном формате.
