# <a name="interfaces-account-management-api"></a>Интерфейсы (API управления учетной записью)

В этом разделе описываются интерфейсы в API управления учетной записью.
  
|**Интерфейс**|**Описание**|
|:-----|:-----|
|[IOlkAccount](iolkaccount.md) <br/> |Поддерживает получение и настройку свойств и других сведений об учетной записи.  <br/> |
|[IOlkAccountHelper](iolkaccounthelper.md) <br/> |Предоставляет дополнительные функции в текущем сеансе MAPI для управления учетной записью.  <br/> |
|[IOlkAccountManager](iolkaccountmanager.md) <br/> |Управляет доступом к учетным записям и настраивает уведомления об изменениях учетных записей.  <br/> |
|[IOlkAccountNotify](iolkaccountnotify.md) <br/> |Предоставляет клиенту вызов для изменения учетной записи.  <br/> |
|[IOlkEnum](iolkenum.md) <br/> |Поддерживается enumerating accounts as [IUnknown](https://docs.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown) objects.  <br/> |
|[IOlkErrorUnknown](iolkerrorunknown.md) <br/> |Предоставляет дополнительные сведения о последней ошибке.  <br/> |
