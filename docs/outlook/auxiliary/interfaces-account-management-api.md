# <a name="interfaces-account-management-api"></a>Интерфейсы (Управление учетными записями API)

В этом разделе описываются интерфейсы API для управления учетными записями.
  
|**Интерфейс**|**Описание**|
|:-----|:-----|
|[IOlkAccount](iolkaccount.md) <br/> |Поддерживает получение и задание свойств и другие сведения об учетной записи.  <br/> |
|[IOlkAccountHelper](iolkaccounthelper.md) <br/> |Предоставляет функциональные возможности модуля поддержки в текущем сеансе MAPI к управлению учетными записями.  <br/> |
|[IOlkAccountManager](iolkaccountmanager.md) <br/> |Управляет доступом к учетным записям и выполняет настройку уведомлений о изменения учетной записи.  <br/> |
|[IOlkAccountNotify](iolkaccountnotify.md) <br/> |Предоставляет обратный вызов клиента для изменения учетной записи.  <br/> |
|[IOlkEnum](iolkenum.md) <br/> |Поддерживает перечисления учетных записей как объекты [IUnknown](https://docs.microsoft.com/en-us/windows/desktop/api/unknwn/nn-unknwn-iunknown) .  <br/> |
|[IOlkErrorUnknown](iolkerrorunknown.md) <br/> |Дополнительные сведения о последней ошибки.  <br/> |
