Представляет номер порта SMTP учетной записи.

|||
|:-----|:-----|
|Идентификатор:  <br/> |0x0201  <br/> |
|Тип свойства:  <br/> |PT_DWORD  <br/> |
|Свойство tag:  <br/> |0x02010003  <br/> |
|Access:  <br/> |Только для чтения  <br/> |
