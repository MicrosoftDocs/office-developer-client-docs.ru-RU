В этом разделе описываются типы данных в API Free/Busy.
  
|**Тип данных**|**Описание**|
|:-----|:-----|
|[FBBlock_1](fbblock_1.md) <br/> |Определяет свободный или загруженный блок данных.  <br/> |
|[FBStatus](fbstatus.md) <br/> |Переумеление для свободного/занятого состояния бесплатных/занятых блоков.  <br/> |
|[FBUser](fbuser.md) <br/> |Определяет пользователя, который может иметь или не иметь доступных бесплатных и загруженных данных.  <br/> |
