# <a name="prop_acct_mini_uid"></a>PROP_ACCT_MINI_UID

Возвращает идентификатор учетной записи, уникальный для Outlook профилей.
  
## <a name="quick-info"></a>Краткие сведения

См. [iOlkAccount](iolkaccount.md).
  
|||
|:-----|:-----|
|Идентификатор:  <br/> |0x0003  <br/> |
|Тип свойства:  <br/> |PT_LONG  <br/> |
|Тег свойства:  <br/> |0x00030003  <br/> |
|Доступ:  <br/> |Только для чтения  <br/> |
   
## <a name="remarks"></a>Примечания

Получите это свойство с помощью [IOlkAccount::GetProp](iolkaccount-getprop.md). Если клиент пытается установить это свойство, это свойство возвращает **E_OLK_PROP_READ_ONLY.** 
  
Это свойство отличается от [PROP_ACCT_ID](prop_acct_id.md) тем, что его значение уникально определяет учетную запись в профиле, в  котором была создана учетная запись, и за ее пределами, в то время как PROP_ACCT_ID уникальна только среди всех учетных записей в том профиле, в котором была создана учетная запись. Когда сообщение с **этими** свойствами перемещается на второй компьютер с другим профилем Outlook и другим набором учетных записей, PROP_ACCT_MINI_UID может однозначно определить исходную учетную запись в исходном профиле. Однако **PROP_ACCT_ID** может привести к конфликту с учетной записью в профиле второго компьютера.
