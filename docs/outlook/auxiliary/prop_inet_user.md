Представляет имя пользователя для общего почтового ящика в Интернете.

|||
|:-----|:-----|
|Идентификатор:  <br/> |0x0101  <br/> |
|Тип свойства:  <br/> |PT_UNICODE  <br/> |
|Тег свойства:  <br/> |0x0101001F  <br/> |
|Обращения  <br/> |Только для чтения  <br/> |
