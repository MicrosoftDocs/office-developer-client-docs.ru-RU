# <a name="project-conceptual-and-how-to-articles"></a>Общие и практические статьи по проектам

Найдите ссылки на сведения в этом разделе о основных частях платформы Project 2013 г., приобщение к разработке CSOM с помощью javaScript и надстройки платформа .NET Framework, надстройки области задач в Project профессиональный 2013 г., интерфейса Project Server и службы отчетов OData.
  
Project Сервер 2013 основан на Project интерфейсе сервера (PSI), внедренной в Office Project Server 2007. Платформа Project Server 2013 построена с платформа .NET Framework 4. Project Web App требует SharePoint Server 2013 в качестве базы для функциональных возможностей новых функций, таких как клиентская объектная модель (CSOM) для доступа к Project Server в облаке, версии 4 Windows Workflow Foundation (WF4) и службы OData для таблиц баз данных отчетов. Project профессиональный 2013 г. можно использовать новую Office надстройки.
