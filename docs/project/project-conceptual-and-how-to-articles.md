# <a name="project-conceptual-and-how-to-articles"></a>Общие и практические статьи по проектам

Ссылки на сведения, приведенные в этом разделе, посвящены основным частям платформы Project 2013, началу работу с разработкой CSOM с помощью JavaScript и .NET Framework, надстроек области задач в Project профессиональный 2013, интерфейса Project Server и Служба отчетов OData.
  
Project Server 2013 основан на платформе веб-службы Project Server Interface (PSI), которая появилась в Office Project Server 2007. Платформа Project Server 2013 создана с помощью .NET Framework 4. В Project Web App необходимо использовать SharePoint Server 2013 в качестве основы для функций новых функций, таких как клиентская объектная модель (CSOM) для доступа к Project Server в облаке, версия 4 для Windows Workflow Foundation (WF4) и служба OData для таблицы базы данных отчетов. Project профессиональный 2013 может использовать надстройки Office новой области задач.
