# <a name="project-conceptual-and-how-to-articles"></a>Общие и практические статьи по проектам

Ссылки на сведения в этом разделе об основных частях платформы Project 2013, начало разработки CSOM с помощью JavaScript и .NET Framework, надстройки области задач в Project профессиональный 2013, интерфейс Project Server и служба отчетов OData.
  
Project Server 2013 основан на структуре веб-служб интерфейса Project Server (PSI), которая была представлена в Office Project Server 2007. Платформа Project Server 2013 построена с помощью .NET Framework 4. Project Web App sharePoint Server 2013 требуется в качестве основы для функциональности новых функций, таких как клиентская объектная модель (CSOM) для доступа к Project Server в облаке версии 4 Windows Workflow Foundation (WF4) и служба OData для таблиц базы данных отчетов. Project профессиональный 2013 может использовать новые надстройки области задач Office.

- [Архитектура и программируемость Project Server 2013](project-server-2013-architecture-and-programmability.md) описывают основные части платформы Project 2013, включая клиенты Project и Project Server. 
  
- [Клиентская объектная модель (CSOM) для Project 2013](client-side-object-model-csom-for-project-2013.md) содержит разделы о том, как начать разработку CSOM с помощью JavaScript и .NET Framework. 
  
- [Программирование клиента Project](project-client-programming.md) включает введение в надстройки области задач в Project профессиональный 2013.
