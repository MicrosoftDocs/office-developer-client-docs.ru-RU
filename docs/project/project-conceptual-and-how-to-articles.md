# <a name="project-conceptual-and-how-to-articles"></a>Общие и практические статьи проекта

Найдите ссылки на сведения в этом разделе о основные части платформы Project 2013, Приступая к разработке CSOM с помощью JavaScript и .NET Framework, задача области надстроек в Project Professional 2013 интерфейс Project Server и Службы отчетов OData.
  
Project Server 2013 основано на интерфейс Project Server (PSI) Инфраструктура веб-служб, внедренные в Office Project Server 2007. Платформа Project Server 2013 созданных с помощью .NET Framework 4. Требуется SharePoint Server 2013 основой для функции новых функций, таких как клиентской объектной модели (CSOM) для доступа к Project Server в облаке, версии 4 (WF4) Windows Workflow Foundation и службы OData для Project Web App Отчетные таблицы базы данных. Project профессиональный 2013 можно использовать новые области задач Office Add-ins.
