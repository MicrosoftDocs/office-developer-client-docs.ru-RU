# <a name="color-cell-layers-section"></a>Ячейка Color (раздел "Слои")

Определяет цвет, используемый для отображения на уровне.
  
## <a name="remarks"></a>Замечания

Чтобы задать цвет, введите число от 0 до 23.
  
Это значение ячейки соответствует параметру **цвет слоя** в диалоговом окне **Свойства слоя** (в группе **Правка** на вкладке **Главная** нажмите кнопку **уровни** и выберите пункт **Свойства Layer**).
  
Чтобы ввести другой цвет, используйте функцию RGB или HSL. Настраиваемые цвета значение цвета RGB, а RGB ( *r, g, b*), а не числа, будут отображаться в окне таблицы свойств фигуры. При использовании в операции, настраиваемые цвета имеют значения из 24 и выше. Значение 255 указывает уровень без цвета. 
  
Можно задать прозрачность цвет слоя в ячейке прозрачность.
  
Для получения ссылки на ячейки цвет по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Layers.Color [ *i* ] где *i* = < 1 > 2, 3,...  <br/> |
   
Для получения ссылки на ячейки цвет по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionLayer** <br/> |
|Индекс строки:  <br/> |**visRowLayer** +  *i* где *i* = 0, 1, 2,...  <br/> |
|Индекс ячейки:  <br/> |**visLayerColor** <br/> |
