# <a name="sum-function"></a>Функция SUM

Возвращает сумму списка чисел.
  
## <a name="syntax"></a>Синтаксис

SUM (* * *число1* * *, * * *число2* * *,..., * * *[нумберн]* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первый номер.  <br/> |
| _число2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второе число.  <br/> |
| _Нумберн_ <br/> |Необязательный  <br/> |**Числовой** <br/> |Значение n.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="example"></a>Пример

SUM (5, 7, 12)
  
Возвращает 24.
