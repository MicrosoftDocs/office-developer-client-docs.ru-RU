# <a name="sum-function"></a>Функция сумм

Возвращает сумму список номеров.
  
## <a name="syntax"></a>Синтаксис

Сумма (** *Число1* **, ** *число2* **,..., ** *[numberN]* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Число1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первый номер.  <br/> |
| _число2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второй номер.  <br/> |
| _numberN_ <br/> |Optional  <br/> |**Числовой** <br/> |N-й номер.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Числовой
  
## <a name="example"></a>Пример

SUM(5,7,12)
  
Возвращает 24.
