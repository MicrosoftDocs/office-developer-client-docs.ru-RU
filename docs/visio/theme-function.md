# <a name="theme-function"></a>Функция ТЕМЫ

Получает параметры формата текущей темы.
  
## <a name="syntax"></a>Синтаксис

THEME()
  
## <a name="example"></a>Пример

```vb
Shape.FillForegnd = THEME("FillColor")
```

Назначает текущей темы цвета заливки фигуры.
