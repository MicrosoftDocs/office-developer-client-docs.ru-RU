# <a name="shapetext-function"></a>Функция SHAPETEXT

Получает текст из фигуры. 
  
## <a name="syntax"></a>Синтаксис

SHAPETEXT (** указанной фигуре *! TheText* ** ** *[, флаг]* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _указанной фигуре! TheText_ <br/> |Обязательный  <br/> ||Ссылка на ячейку с именем TheText в конечную фигуру.  _Указанной фигуре!_ — Это имя фигуры, из которого требуется получить текст.  <br/> |
| _флаг_ <br/> |Optional  <br/> |**Числовой** <br/> |Бит, который определяет формат текста. Флаг по умолчанию (0) отображается текст точно так, как он отображается в форме.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Строка
  
## <a name="remarks"></a>Замечания

Можно использовать любое сочетание следующих флагов с помощью функции SHAPETEXT.
  
|**Flag**|**Описание**|
|:-----|:-----|
|0  <br/> |Отображение текста так же, как показано в форму.  <br/> |
|1  <br/> |Включите дефисы на уровне пользователей.  <br/> |
|2  <br/> |Не включать расширенное текст в полях.  <br/> |
|4  <br/> |Преобразование в один пробел.  <br/> |
|8  <br/> |Преобразование в набор пробелов.  <br/> |
|16  <br/> |Преобразование возврат каретки и перевода строки в пробелы.  <br/> |
|32  <br/> |Преобразования typographer квот для регулярного кавычки.  <br/> |
|64  <br/> |Преобразование рядом с пробелов в пробел.  <br/> |
   
## <a name="example-1"></a>Пример 1

SHAPETEXT(sheetN!theText)
  
Возвращает текст фигуры, с именем sheetN, точно так, как он отображается в форме.
  
## <a name="example-2"></a>Пример 2

SHAPETEXT(theText)
  
Возвращает текст текущего фигуры точно так, как он отображается в форме.
  
## <a name="example-3"></a>Пример 3

SHAPETEXT (theText, 84)
  
Возвращает текст текущей фигуры. Также преобразует рядом с пробелы пробелом (64), возврат каретки преобразует и в строке каналов пробелов (16) и преобразует знаки табуляции в пробелом (4). Сумма эти флаги — 84.
