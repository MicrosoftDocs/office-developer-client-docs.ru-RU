# <a name="shapetext-function"></a>Функция SHAPETEXT

Получает текст из фигуры. 
  
## <a name="syntax"></a>Синтаксис

SHAPETEXT (** *shapename! TheText* ** ** *[,flag]* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _shapename! TheText_ <br/> |Обязательный  <br/> ||Ссылка на ячейку с именем TheText в целевой форме.  _Shapename!_ это имя фигуры, из которой нужно получить текст.  <br/> |
| _flag_ <br/> |Необязательный  <br/> |**Числовой** <br/> |Немного, который указывает формат текста. Флаг по умолчанию (0) показывает текст точно так, как он показан в форме.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Вы можете использовать любое сочетание следующих флагов с функцией SHAPETEXT.
  
|**Флаг**|**Описание**|
|:-----|:-----|
|0  <br/> |Показать текст точно так, как показано в форме.  <br/> |
|1  <br/> |Включай дискреционные дефисы.  <br/> |
|2  <br/> |Не включайте расширенный текст в поля.  <br/> |
|4   <br/> |Преобразование вкладок в одно пространство.  <br/> |
|8   <br/> |Преобразование вкладок в набор пробелов.  <br/> |
|16   <br/> |Преобразование возвращаемой кареты и каналов строки в пробелы.  <br/> |
|32  <br/> |Преобразование кавычка опечатщика в обычные кавычка.  <br/> |
|64  <br/> |Преобразование прилегающего белого пространства в одно пространство.  <br/> |
   
## <a name="example-1"></a>Пример 1

SHAPETEXT(sheetN!theText)
  
Возвращает текст фигуры с именем sheetN, точно так же, как показано в фигуре.
  
## <a name="example-2"></a>Пример 2

SHAPETEXT (theText)
  
Возвращает текст текущей фигуры точно так, как показано в фигуре.
  
## <a name="example-3"></a>Пример 3

SHAPETEXT (theText, 84)
  
Возвращает текст текущей фигуры. Кроме того, он преобразует прилегающее белое пространство в одно пространство (64), преобразует возвраты вагона и каналы строки в пробелы (16) и преобразует вкладки в одно пространство (4). Сумма этих флагов — 84.
