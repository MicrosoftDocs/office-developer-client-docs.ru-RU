# <a name="shapetext-function"></a>Функция SHAPETEXT

Получает текст из фигуры. 
  
## <a name="syntax"></a>Синтаксис

SHAPETEXT (** *shapename! TheText* ** ** *[,flag]* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _shapename! TheText_ <br/> |Обязательна  <br/> ||Ссылка на ячейку с именем TheText в целевой фигуре.  _Shapename!_ — имя фигуры, из которой вы хотите получить текст.  <br/> |
| _flag_ <br/> |Необязательный  <br/> |**Числовой** <br/> |Бит, который определяет формат текста. Флаг по умолчанию (0) показывает текст точно так, как он отображается в фигуре.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

С функцией SHAPETEXT можно использовать любое сочетание следующих флагов.
  
|**Flag**|**Описание**|
|:-----|:-----|
|0  <br/> |Отобразить текст точно так, как показано в фигуре.  <br/> |
|1   <br/> |Включаем дискреционные дефисы.  <br/> |
|2   <br/> |Не включайте расширенный текст в поля.  <br/> |
|4   <br/> |Преобразование вкладок в одно пространство.  <br/> |
|8   <br/> |Преобразование вкладок в набор пробелов.  <br/> |
|16   <br/> |Преобразуем возвраты каретки и переводимые строки в пробелы.  <br/> |
|32  <br/> |Преобразуйте кавычка опечатки в обычные кавычка.  <br/> |
|64  <br/> |Преобразуем смежный пробел в одно пространство.  <br/> |
   
## <a name="example-1"></a>Пример 1

SHAPETEXT(sheetN!theText)
  
Возвращает текст фигуры с именем sheetN точно так же, как он показан в фигуре.
  
## <a name="example-2"></a>Пример 2

SHAPETEXT(theText)
  
Возвращает текст текущей фигуры точно так же, как он показан в фигуре.
  
## <a name="example-3"></a>Пример 3

SHAPETEXT(theText, 84)
  
Возвращает текст текущей фигуры. Он также преобразует смежное пробел в одно пространство (64), преобразует возврат каретки и каналы строк в пробелы (16) и преобразует вкладки в одно пространство (4). Сумма этих флагов составляет 84.
