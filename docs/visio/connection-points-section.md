# <a name="connection-points-section"></a>Раздел "Точки соединения"

Содержит строку и ячейки для каждой точки подключения, указанных для фигуры.
  
## <a name="remarks"></a>Замечания

Можно определить точек подключения в разделе **Точек подключения** в окне таблицы свойств фигуры или с помощью средства **Точки подключения** на вкладке **Главная** . 
  
Если фигура содержит раздел без **Точек подключения** можно вставить раздел **Точек подключения** в окне таблицы свойств фигуры. Щелкните правой кнопкой мыши в окне таблицы свойств фигуры, нажмите кнопку **Вставить раздел**и нажмите кнопку **точек подключения**. Чтобы добавить точек подключения в существующий раздел **Точек подключения** , щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить строку** в контекстном меню. 
  
Можно изменить тип строки точки подключения в этом разделе, чтобы отобразить дополнительные ячеек, которые можно использовать для вычисления вспомогательной. Чтобы сделать это, щелкните правой кнопкой мыши строку и нажмите кнопку **Изменить тип строки** в контекстном меню.
