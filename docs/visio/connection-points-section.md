# <a name="connection-points-section"></a>Connection Points Section

Содержит строку и ячейки для каждой точки подключения, определенной для фигуры.
  
## <a name="remarks"></a>Примечания

Точки подключения можно определить в разделе **Точки** подключения в окне ShapeSheet или щелкнув средство **Точка** подключения на вкладке **Главная.** 
  
Если в вашей фигуре нет раздела **Точки** подключения, можно вставить раздел **Точки** подключения в окне ShapeSheet. Щелкните правой кнопкой мыши в окне ShapeSheet, щелкните **Раздел Вставки** и нажмите **кнопку Точки подключения.** Чтобы добавить точки подключения к существующему разделу **Точки** подключения, щелкните правой кнопкой мыши строку и нажмите **строку Вставить** строку в меню ярлыка. 
  
Вы можете изменить тип строки точки подключения в этом разделе, чтобы отобразить дополнительные ячейки, которые можно использовать для расчетов царапин. Для этого щелкните правой кнопкой мыши строку и нажмите **кнопку Изменить** строку в меню ярлыка.
