# <a name="text-block-format-section"></a>Text Block Format Section

Содержит ячейки, которые контролируют выравнивание, поля и стоп-положение стоп-вкладки по умолчанию в текстовом блоке формы.
  
## <a name="remarks"></a>Примечания

Эти текстовые атрибуты можно также установить с помощью диалогового окна **Text** (на вкладке **Home** нажмите кнопку запуска диалогового окна в **группе Font).**
