# <a name="text-block-format-section"></a>Text Block Format Section

Содержит ячейки, которые управляют выравниванием, полями и позицией табуляции по умолчанию в блоке текста фигуры.
  
## <a name="remarks"></a>Примечания

Вы также можете задать эти атрибуты текста с помощью диалогового окна " **текст** " (на вкладке " **Главная** " щелкните средство запуска диалогового окна в группе **Шрифт** ).
