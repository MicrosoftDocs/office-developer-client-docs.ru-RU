# <a name="doclastedit-function"></a>Функция DOCLASTEDIT

Возвращает в качестве серийного значения дату и время последнего редактирования документа.
  
## <a name="syntax"></a>Синтаксис

DOCLASTEDIT()
  
## <a name="remarks"></a>Примечания

Используйте [функцию FORMAT](format-function.md) для формата значения.
