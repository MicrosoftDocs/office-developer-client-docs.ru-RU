# <a name="doclastedit-function"></a>Функция DOCLASTEDIT

Возвращает в виде порядкового значения Дата и время последнего изменения документа.
  
## <a name="syntax"></a>Синтаксис

DOCLASTEDIT ()
  
## <a name="remarks"></a>Комментарии

Чтобы отформатировать значение, используйте функцию [Format](format-function.md) .
