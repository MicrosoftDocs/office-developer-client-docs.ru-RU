# <a name="doclastedit-function"></a>Функция DOCLASTEDIT

Возвращает в виде числа, Дата и время последнего изменения документа.
  
## <a name="syntax"></a>Синтаксис

DOCLASTEDIT()
  
## <a name="remarks"></a>Замечания

Функция [FORMAT](format-function.md) используется для форматирования значения.
