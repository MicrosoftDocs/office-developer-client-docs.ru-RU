# <a name="group-properties-section"></a>Раздел "Свойства группы"

Содержит ячейки для группы, которые управляют, как добавить фигуры в группу, перемещение участников группы и выберите группы. 
  
## <a name="remarks"></a>Замечания

Эти значения также можно настроить на вкладке **поведение** в диалоговом окне **поведение** (выберите **поведение** группы **Разработки фигуры** на вкладки " [Разработчик](run-in-developer-mode-display-the-developer-tab.md) ").
