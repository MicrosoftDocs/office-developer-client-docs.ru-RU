# <a name="group-properties-section"></a>Group Properties Section

Содержит ячейки для группы, которые управляют способом добавления фигур в группу, перемещения участников группы и выбора групп. 
  
## <a name="remarks"></a>Примечания

Вы также можете задать эти значения на вкладке **поведение** в диалоговом окне **поведение** (нажмите **поведение** в группе **Макет фигуры** на вкладке [разработчик](run-in-developer-mode-display-the-developer-tab.md) ).
