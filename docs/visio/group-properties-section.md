# <a name="group-properties-section"></a>Group Properties Section

Содержит ячейки для группы, которые контролируют добавление фигур в группу, перемещение членов группы и выбор групп. 
  
## <a name="remarks"></a>Примечания

Эти значения также можно установить на вкладке **"Поведение"**  в диалоговом окне "Поведение" (щелкните "Поведение" в группе "Конструктор фигур" на вкладке  ["Разработчик").](run-in-developer-mode-display-the-developer-tab.md)
