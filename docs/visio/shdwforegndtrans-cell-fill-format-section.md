# <a name="shdwforegndtrans-cell-fill-format-section"></a>ShdwForegndTrans Cell (Fill Format Section)

Определяет уровень прозрачности для цвета, используемого для переднего плана (росчерка) шаблона заливки тени для отпадения фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 - 100  <br/> |Представляет процент прозрачности. Значение по умолчанию — 0 % (полностью непрозрачной).  <br/> |
   
## <a name="remarks"></a>Примечания

Значения округлются до ближайшей половины процента. Значение 100% полностью прозрачно. Хотя тень с полностью прозрачной заливки отображается на странице рисования так же, как и тени без заливки, она взаимодействует с другими объектами на странице так же, как если бы ее прозрачность была 0%.
  
Это значение также можно установить с помощью  ползунок в диалоговом окне "Тени" (на вкладке "Главная" в группе фигур щелкните "Тени" и выберите **"Параметры теней").**    Это значение управляет значением как фоновой, так и передней теней. Чтобы установить эти значения независимо друг от друга, необходимо ввести их в окне ShapeSheet.
  
Чтобы получить ссылку на ячейку ShdwForegndTrans по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |ShdwForegndTrans  <br/> |
   
Чтобы получить ссылку на ячейку ShdwForegndTrans по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowFill** <br/> |
|Индекс ячейки:  <br/> |**visFillShdwForegndTrans** <br/> |
