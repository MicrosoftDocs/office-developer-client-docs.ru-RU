# <a name="shdwforegndtrans-cell-fill-format-section"></a>ShdwForegndTrans Cell (Fill Format Section)

Определяет уровень прозрачности для цвета, используемого для переднего плана (штриха) узора заливки тени фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0-100  <br/> |Представляет процент прозрачности. Значение по умолчанию: 0% (полностью непрозрачно).  <br/> |
   
## <a name="remarks"></a>Примечания

Значения округляются до ближайшей половины процентов. Значение 100% полностью прозрачно. Несмотря на то, что тень с полностью прозрачной заливкой отображается на странице рисунка в виде тени, не имеющей заливки, она взаимодействует с другими объектами на странице тем же способом, что и его прозрачность: 0%.
  
Это значение можно также задать с помощью ползунка элемента управления в диалоговом окне **теневОй копии** (на вкладке **Главная** , в группе **фигур** щелкните элемент **тень**, а затем выберите **параметры тени**). Это значение определяет значение для прозрачности фона и фоновой тени. Чтобы задать эти значения независимо друг от друга, необходимо ввести их в окне таблицы свойств фигуры.
  
Чтобы получить ссылку на ячейку ShdwForegndTrans по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |ShdwForegndTrans  <br/> |
   
Чтобы получить ссылку на ячейку ShdwForegndTrans по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровфилл** <br/> |
|Индекс ячейки:  <br/> |**Висфиллшдвфорегндтранс** <br/> |
