# <a name="shdwforegndtrans-cell-fill-format-section"></a>Ячейка ShdwForegndTrans (раздел "Формат заливки")

Определяет уровень прозрачности цвета, используемого для переднего плана (числу штрихов) узора заливки тени фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 — 100  <br/> |Представляет собой процентную долю прозрачность. Значение по умолчанию — 0% (полностью непрозрачный).  <br/> |
   
## <a name="remarks"></a>Замечания

Значения округляются до ближайшего большего половины процентов. Значение 100% является полностью прозрачным. Несмотря на то, что тени, полностью прозрачной заливкой отображается на странице документа как тени, без заливки, взаимодействия с других объектов на странице же образом как если бы его прозрачность 0%.
  
Также можно задать это значение с помощью элемента управления ползунок в диалоговом окне **теневой** (на вкладке **Главная** в группе **фигуру** нажмите кнопку **тень**и нажмите кнопку **Параметры тени**). Это значение определяет значение теневой прозрачность переднего плана и фона. Чтобы задать эти значения независимо друг от друга, необходимо ввести их в окне таблицы свойств фигуры.
  
Чтобы получить ссылку на ячейку ShdwForegndTrans по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |ShdwForegndTrans  <br/> |
   
Для получения ссылки на ячейки ShdwForegndTrans по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowFill** <br/> |
|Индекс ячейки:  <br/> |**visFillShdwForegndTrans** <br/> |
