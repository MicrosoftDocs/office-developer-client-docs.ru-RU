# <a name="endarrow-cell-line-format-section"></a>Ячейка EndArrow (раздел "Формат линий")

Указывает, будет ли линия стрелки или другого плана формата в последней вершине.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0  <br/> |Не стрелки.  <br/> |
|1 - 45  <br/> |Различные стрелки стили, которые соответствуют индексирование записей в диалоговом окне " **строка** ".  <br/> |
   
## <a name="remarks"></a>Замечания

Также можно задать это значение в поле **строка** (на вкладке **Главная** в группе **фигуру** щелкните **строку**, выберите пункт **стрелки**и нажмите кнопку **Дополнительно стрелки**). Размер стрелки имеет значение в ячейке EndArrowSize.
  
Можно указать конец настраиваемой строки, с помощью функции используется в этой ячейке. 
  
Чтобы получить ссылку на ячейку EndArrow по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |EndArrow  <br/> |
   
Для получения ссылки на ячейки EndArrow по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowLine** <br/> |
|Индекс ячейки:  <br/> |**visLineEndArrow** <br/> |
