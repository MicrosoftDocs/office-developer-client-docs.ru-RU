# <a name="sat-function"></a>Функция SAT

Возвращает значение компонента насыщенность цвета. 
  
## <a name="syntax"></a>Синтаксис

СБ (** *выражение* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**Разные** <br/> |Индекс цвета в таблице цветов документа, выражение, которое разрешается в настраиваемый цвет (например, RGB или HSL) или ссылку на ячейку, содержащую цвет индекса или цветовой результатов.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Числовой
  
## <a name="remarks"></a>Замечания

Возвращаемое значение — число в диапазоне от 0 до 240 включительно. Функция возвращает 0 для недопустимые данные.
  
## <a name="example-1"></a>Пример 1

Суббота (Sheet.4! FillForegnd)
  
Возвращает насыщенность цвет заливки Sheet. 4.
  
## <a name="example-2"></a>Пример 2

SAT(8)
  
Возвращает 240, если в документе используется по умолчанию Visio цветовой палитры, в котором Темно-красный цвет по индексу 8.
  
## <a name="example-3"></a>Пример 3

СУББОТА (HSL (10, 20, 30))
  
Возвращает 20.
