# <a name="datetime-function"></a>Функция DATETIME

Возвращает значение даты и времени, _даты и времени_ или _выражения_.
  
## <a name="syntax"></a>Синтаксис

Даты и времени ("** *даты и времени* **" | ** *выражение* ** [, ** *lcid* **]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _даты и времени_ <br/> |Обязательный  <br/> |**Строка** <br/> |Любая строка, часто распознается как даты и времени или ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**Строка** <br/> |Любое выражение, возвращающее даты и времени.  <br/> |
| _код языка_ <br/> |Optional  <br/> |**Число** <br/> |Задает идентификатор языкового стандарта для использования при оценке нелокальных datetime. Идентификатор языкового стандарта — это число, описанных в файлы заголовков системы.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Даты и времени
  
## <a name="remarks"></a>Замечания

Если *даты и времени* отсутствует или не может использоваться как даты или времени, даты и времени возвращает #VALUE! Ошибка. 
  
Возвращаемое значение форматируется в соответствии с краткий формат даты и времени стиль в текущих региональных параметров компьютера. 
  
Функции даты и времени также принимает одно значение номеров для *выражения* , где целую часть результата представляет число дней, прошедших с 30 декабря 1899, а дробная часть представляет долю день после полночи. 
  
## <a name="example-1"></a>Пример 1

Даты и времени ("30 мая 1997 г.") + 5 ed.
  
Возвращает значение, представляющее 6/4/1997 г.
  
## <a name="example-2"></a>Пример 2

ФОРМАТ (ДАТЫ И ВРЕМЕНИ ("5/20/1997 14:30:45"), «C»)
  
Возвращает строку «Вторник 20 мая, 1997 г. 2:30:45 PM.»
  
## <a name="example-3"></a>Пример 3

Даты и времени ("1:30 PM 19 июля")
  
Возвращает значение, представляющее 7/19/2001 1:30:00 PM (при условии текущего года 2001 г.).
  
## <a name="example-4"></a>Пример 4

DATETIME(35580.6337)
  
Возвращает значение, представляющее 5/30/1997 г., 15:12:32 по.
