# <a name="spline-cell-paragraph-section"></a>SpLine Cell (Paragraph Section)

Определяет расстояние между одной строкой текста и следующим, выраженное в процентном выражении, где 100% — высота строки текста.
  
|**Значение**|**Описание**|
|:-----|:-----|
| \>нуль  <br/> | Абсолютные интервалы, независимо от размера шрифта  <br/> |
| = 0  <br/> | Задать сплошной (интервал = 100% размера шрифта)  <br/> |
| \<нуль  <br/> | Процент размера шрифта (например,-120% дает 120% интервал)  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку сплайна по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Para. Сплайн [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку сплайна по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**Виссектионпараграф** <br/> |
| Индекс строки:  <br/> |**висровпараграф** +  *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**Висспацелине** <br/> |
