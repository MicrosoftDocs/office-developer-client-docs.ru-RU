# <a name="beginx-cell-1-d-endpoints-section"></a>Ячейка BeginX (раздел "Одномерные конечные точки")

Представляет *x* -координаты начальной точки одномерной фигуры относительно начала родительского элемента. 
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку BeginX по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | BeginX  <br/> |
   
Для получения ссылки на ячейки BeginX по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowXForm1D** <br/> |
| Индекс ячейки:  <br/> |**vis1DBeginX** <br/> |
