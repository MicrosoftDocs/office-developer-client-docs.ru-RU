# <a name="shapeshdwtype-cell-fill-format-section"></a>ShapeShdwType Cell (Fill Format Section)

Указывает тип тени для фигуры. 
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Использование страницы по умолчанию (по умолчанию)  <br/> |**visFSTPageDefault** <br/> |
|1   <br/> |Простой  <br/> |**visFSTSimple** <br/> |
|2   <br/> |Oblique  <br/> |**visFSTOblique** <br/> |
   
## <a name="remarks"></a>Примечания

Используйте эту ячейку для применения тени фигуры, которая отличается от страницы по умолчанию (тип тени страницы по умолчанию определяется в ячейке ShdwType в разделе "Свойства страницы").
  
Простые типы теней описываются как тени смещения в пользовательском интерфейсе. Простая тень дает эффект тени фигуры на параллельную плоскость, расположенную за фигурой. Косые тени описываются как косые тени в пользовательском интерфейсе и дают эффект отвершения тени на плоскость, перпендикулярную фигуре. 
  
Список предопределений простых и косых типов теней см.  в поле "Стиль" в диалоговом окне "Тени" (на вкладке "Главная" в группе "Фигура" щелкните "Тени" и выберите "Параметры тени").    
  
Чтобы получить ссылку на ячейку ShapeShdwType по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |ShapeShdwType  <br/> |
   
Чтобы получить ссылку на ячейку ShapeShdwType по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowFill** <br/> |
|Индекс ячейки:  <br/> |**visFillShdwType** <br/> |
