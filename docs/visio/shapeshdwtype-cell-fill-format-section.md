# <a name="shapeshdwtype-cell-fill-format-section"></a>Ячейка ShapeShdwType (раздел "Формат заливки")

Указывает тип тени фигуры. 
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Использование страницы по умолчанию (по умолчанию)  <br/> |**visFSTPageDefault** <br/> |
|1  <br/> |Простое  <br/> |**visFSTSimple** <br/> |
|2  <br/> |Наклон  <br/> |**visFSTOblique** <br/> |
   
## <a name="remarks"></a>Замечания

Используйте эту ячейку для применения тени фигуры, отличный от страницы по умолчанию (типа тени по умолчанию определяется в ячейке ShdwType в разделе Страница свойств).
  
Простая тень типов описаны как смещения тени пользовательского интерфейса (UI). Простая тень дает эффект фигуры, теневую на параллельный плоскости, расположенной за фигуры. Наклон тени описанными наклон теней в пользовательском Интерфейсе и создания эффекта тени приведение на плоскости перпендикулярно фигуры. 
  
Список типов предварительно заданных простой и наклон тени, просмотрите окно **стиль** в диалоговом окне **теневой** (на вкладке **Главная** в группе **фигуру** нажмите кнопку **тень**и нажмите кнопку **Параметры тени**).
  
Чтобы получить ссылку на ячейку ShapeShdwType по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |ShapeShdwType  <br/> |
   
Для получения ссылки на ячейки ShapeShdwType по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowFill** <br/> |
|Индекс ячейки:  <br/> |**visFillShdwType** <br/> |
