# <a name="shapeshdwtype-cell-fill-format-section"></a>ShapeShdwType Cell (Fill Format Section)

Указывает тип тени для фигуры. 
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Использование по умолчанию страницы (по умолчанию)  <br/> |**visFSTPageDefault** <br/> |
|1  <br/> |Простой  <br/> |**visFSTSimple** <br/> |
|2  <br/> |Косая  <br/> |**visFSTOblique** <br/> |
   
## <a name="remarks"></a>Примечания

Используйте эту ячейку для применения тени фигуры, которая отличается от страницы по умолчанию (тип тени страницы определяется в ячейке ShdwType в разделе Свойства страниц).
  
Простые типы теней описываются как офсетные тени в пользовательском интерфейсе (пользовательском интерфейсе). Простая тень дает эффект тени фигуры на параллельной плоскости, расположенной за фигурой. Косые тени описываются как косые тени в пользовательском интерфейсе и дают эффект отбрасываемой тени на плоскость перпендикулярно фигуре. 
  
Список предопределяемых простых и косых типов  теней см. в поле Стиль в диалоговом окне **Shadow** (на вкладке **Главная,** в группе **Shape** нажмите кнопку **Тень,** а затем нажмите параметры **Shadow).**
  
Чтобы получить ссылку на ячейку ShapeShdwType по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |ShapeShdwType  <br/> |
   
Чтобы получить ссылку на ячейку ShapeShdwType по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowFill** <br/> |
|Индекс ячейки:  <br/> |**visFillShdwType** <br/> |
