# <a name="relationships-cell-shape-layout-section"></a>Ячейка Relationships (раздел "Макет фигуры")

Сохранение отношения между контейнеров, списков, выноски и фигур. 
  
## <a name="remarks"></a>Замечания

 Microsoft Visio использует ячейки связей для хранения отношений с участием этой фигуры. Ряд функций DEPENDSON, с параметрами, используются для представления отношения с этой фигуры, как показано в следующей таблице. 
  
|**Первым параметром**|**Дополнительные параметры**|
|:-----|:-----|
|1  <br/> |Фигуры, которые являются членами этого контейнера.  <br/> |
|2  <br/> |Фигуры, которые являются членами этого списка.  <br/> |
|3  <br/> |Выноски, связанных с этой фигуры.  <br/> |
|4  <br/> |Контейнеры, которые принадлежит этой фигуры.  <br/> |
|5  <br/> |Список, принадлежит этот элемент списка.  <br/> |
|6  <br/> |Фигура, связанный с этой выноски.  <br/> |
|7  <br/> |Контейнер на левой границе, из которых стоит этой фигуры.  <br/> |
|8  <br/> |Контейнер на правой границе, из которых стоит этой фигуры.  <br/> |
|9  <br/> |Контейнер на верхней границе, из которых стоит этой фигуры.  <br/> |
|10  <br/> |Контейнер на нижней границе которого располагается этой фигуры.  <br/> |
|11  <br/> |Список, перекрывается этого списка.  <br/> |
   
Для получения ссылки на ячейку связи по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Связи  <br/> |
   
Для получения ссылки на ячейки связи по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowShapeLayout** <br/> |
|Индекс ячейки:  <br/> |**visSLORelationships** <br/> |
