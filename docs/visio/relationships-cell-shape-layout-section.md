# <a name="relationships-cell-shape-layout-section"></a>Relationships Cell (Shape Layout Section)

Хранит отношения между контейнерами, списками, выносками и фигурами. 
  
## <a name="remarks"></a>Примечания

 В Microsoft Visio используется ячейка связи для хранения отношений, включающих эту фигуру. Ряд функций DEPENDSON с отображаемыми параметрами используются для представления связей с этой фигурой, как показано в следующей таблице. 
  
|**Первый параметр**|**Дополнительные параметры**|
|:-----|:-----|
|1,1  <br/> |Фигуры, которые являются членами этого контейнера.  <br/> |
|2  <br/> |Фигуры, которые являются участниками этого списка.  <br/> |
|4  <br/> |ВыНоски, связанные с этой фигурой.  <br/> |
|SP4  <br/> |Контейнеры, участником которых является эта фигура.  <br/> |
|17:00  <br/> |Список, участником которого является данный элемент списка.  <br/> |
|ICMPv6  <br/> |Фигура, связанная с этим вызываемым выноски.  <br/> |
|см  <br/> |Контейнер на границе слева, на которой располагается эта фигура.  <br/> |
|8,5  <br/> |Контейнер на правой границе, на которой располагается эта фигура.  <br/> |
|10  <br/> |Контейнер для верхнего края, на котором располагается эта фигура.  <br/> |
|10   <br/> |Контейнер для нижнего края, на котором располагается эта фигура.  <br/> |
|11   <br/> |Список перекрывает этот список.  <br/> |
   
Чтобы получить ссылку на ячейку Relationships по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Связи  <br/> |
   
Чтобы получить ссылку на ячейку Relationships по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровшапелайаут** <br/> |
|Индекс ячейки:  <br/> |**Висслорелатионшипс** <br/> |
