# <a name="relationships-cell-shape-layout-section"></a>Relationships Cell (Shape Layout Section)

Сохраняет связи между контейнерами, списками, вызовами и фигурами. 
  
## <a name="remarks"></a>Примечания

 Microsoft Visio использует ячейку Relationships для хранения связей, которые связаны с этой фигурой. Для представления отношений с этой фигурой используется серия функций DEPENDSON с показанными параметрами, как показано в следующей таблице. 
  
|**Первый параметр**|**Дополнительные параметры**|
|:-----|:-----|
|1  <br/> |Фигуры, которые являются членами этого контейнера.  <br/> |
|2  <br/> |Фигуры, которые являются участниками этого списка.  <br/> |
|3  <br/> |Вызовы, связанные с этой фигурой.  <br/> |
|4   <br/> |Контейнеры, в которые входит эта фигура.  <br/> |
|5   <br/> |Список, в который входит этот элемент списка.  <br/> |
|6   <br/> |Форма, связанная с этим вызовом.  <br/> |
|7   <br/> |Контейнер на левом краю границы, на котором расположена эта фигура.  <br/> |
|8   <br/> |Контейнер на правом пограничном краю, на котором расположена эта фигура.  <br/> |
|9   <br/> |Контейнер на верхнем краю границы, на котором расположена эта фигура.  <br/> |
|10   <br/> |Контейнер на нижнем краю границы, на котором расположена эта фигура.  <br/> |
|11  <br/> |Перечислить, что этот список перекрывается.  <br/> |
   
Чтобы получить ссылку на ячейку Relationships по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Связи  <br/> |
   
Чтобы получить ссылку на ячейку Relationships по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowShapeLayout** <br/> |
|Индекс ячейки:  <br/> |**visSLORelationships** <br/> |
