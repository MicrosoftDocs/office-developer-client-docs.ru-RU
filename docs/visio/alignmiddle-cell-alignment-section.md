# <a name="alignmiddle-cell-alignment-section"></a>Ячейка AlignMiddle (раздел "Выравнивание")

Определяет позицию по вертикали, по отношению к источнику родительского объекта по горизонтали, направляющей или направляющей точки, относительно которого выровнен вертикальный центр фигуры.
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку AlignMiddle по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AlignMiddle  <br/> |
   
Для получения ссылки на ячейки AlignMiddle по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignMiddle** <br/> |
