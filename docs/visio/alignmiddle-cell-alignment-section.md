# <a name="alignmiddle-cell-alignment-section"></a>AlignMiddle Cell (Alignment Section)

Определяет вертикальное положение (относительно начала родительского) горизонтального руководства или точки направляния, к которой выравнивается центр вертикальной фигуры.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку AlignMiddle по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | AlignMiddle  <br/> |
   
Чтобы получить ссылку на ячейку AlignMiddle по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignMiddle** <br/> |
