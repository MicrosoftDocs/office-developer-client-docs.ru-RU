# <a name="doclastsave-function"></a>Функция DOCLASTSAVE

Возвращает дату и время последнего с сохранения документа в качестве последовательного значения.
  
## <a name="syntax"></a>Синтаксис

DOCLASTSAVE ()
  
## <a name="remarks"></a>Примечания

Используйте функцию FORMAT для формата значения.
