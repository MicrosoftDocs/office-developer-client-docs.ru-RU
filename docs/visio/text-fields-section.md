# <a name="text-fields-section"></a>Text Fields Section

Отображает функции и формулы, вставленные в текст фигуры, с помощью **диалогового** окна "Поле". 
  
## <a name="remarks"></a>Примечания

Единственный способ добавить этот раздел — вставить поле в фигуру с помощью диалоговых окна **"Поле"** (вкладка **"Вставка").** 
  
Каждое текстовое поле в текстовом блоке имеет соответствующую строку в окне ShapeSheet. Имя строки представляет количество символов, которые содержит поле.
