# <a name="text-fields-section"></a>Раздел текстовых полей

Отображает функций и формул вставлен в тексте фигуры с помощью диалогового окна **поля** . 
  
## <a name="remarks"></a>Замечания

Чтобы вставить поле в форму с помощью диалогового окна **поля** (вкладка "**Вставка** ") является единственным способом для добавления в этом разделе. 
  
Каждое поле текста в блоке текста имеет соответствующей строки в окне таблицы свойств фигуры. Имя строки представляет число знаков, которое содержит поля.
