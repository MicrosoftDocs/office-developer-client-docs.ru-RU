# <a name="avenuesizey-cell-page-layout-section"></a>Ячейка AvenueSizeY (раздел "Макет страницы")

Определяет интервал по вертикали между фигурами на странице документа при расположении фигур с помощью диалогового окна " **Настройка макета** " (на вкладке **Конструктор** в группе **Макет** выберите пункт **Изменить** расположение и нажмите кнопку **более Параметры расположения**).
  
## <a name="remarks"></a>Замечания

Также можно задать это значение в диалоговом окне **Макет и маршрутизация интервалы** (на вкладку **Конструктор** , щелкните стрелку в группе **Параметры страницы** , перейдите на вкладку **Макет и маршрутизации** и нажмите кнопку **интервала**).
  
Динамические сетки используется в ячейке AvenueSizeY Если доступен только один фигуры для расчета интервал по вертикали. Использование динамических сетки на вкладке **Вид** в группе **Вспомогательные элементы** выберите **Динамических сетки**.
  
Чтобы получить ссылку на ячейку AvenueSizeY по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AvenueSizeY  <br/> |
   
Для получения ссылки на ячейки AvenueSizeY по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowPageLayout** <br/> |
| Индекс ячейки:  <br/> |**visPLOAvenueSizeY** <br/> |
