# <a name="iferror-function"></a>Функция IFERROR

Возвращает результат оценки основного выражения, если он не оценивает ошибку. В противном случае возвращает результат оценки альтернативного выражения.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

IFERROR(** *primary expression* **, ** alternate *expression* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _основное выражение_ <br/> |Обязательно  <br/> |**Строка** <br/> |Первое выражение для оценки.  <br/> |
| _альтернативное выражение_ <br/> |Обязательно  <br/> |**Строка** <br/> |Альтернативное выражение для оценки ошибки основным выражением.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Разные
