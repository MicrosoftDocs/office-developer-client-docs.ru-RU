# <a name="iferror-function"></a>Функция IFERROR

Возвращает оцененный результат первичного выражения, если оно не оценивает ошибку. В противном случае возвращается оцененный результат альтернативного выражения.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

IFERROR(** *первичное* выражение **, ** *альтернативное* выражение ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _первичное выражение_ <br/> |Обязательный  <br/> |**String** <br/> |Первое выражение для оценки.  <br/> |
| _альтернативное выражение_ <br/> |Обязательный  <br/> |**String** <br/> |Альтернативное выражение, чтобы оценить, оценивает ли основное выражение ошибку.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Разные
