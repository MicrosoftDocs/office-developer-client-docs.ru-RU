# <a name="iferror-function"></a>Функция IFERROR

Возвращает вычисленный результат первичного выражения, если он не вычисляется как ошибка. В противном случае возвращает результат вычисления альтернативного выражения.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

IFERROR (* * *первичное выражение* * *, * * *альтернативное выражение* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _основное выражение_ <br/> |Обязательный  <br/> |**String** <br/> |Первое вычисляемое выражение.  <br/> |
| _альтернативное выражение_ <br/> |Обязательный  <br/> |**String** <br/> |Альтернативное выражение для оценки того, является ли основное выражение ошибкой.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Разные
