# <a name="blend-function"></a>Функция BLEND

Сочетает двух цветов в пропорции, указанного с помощью параметра _с плавающей запятой_ . 
  
## <a name="syntax"></a>Синтаксис

Переход (** *color1* **, ** *цвет2* **, ** *с плавающей запятой [0,1]* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Индекс цвет Visio или значение первого цвета RGB.  <br/> |
| _цвет2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Индекс цвет Visio или значение второй цвета RGB.  <br/> |
| _число с плавающей запятой [0,1]_ <br/> |Обязательный  <br/> |**Double** <br/> |Размеры наложения _цвет2_ и _color1_соответственно. Действительное число от 0 до 1 включительно.  <br/> |
   
### <a name="return-value"></a>������������ ��������

 **RGB**
  
## <a name="remarks"></a>Замечания

Цвет возвращаемых определяется относительный пропорции, в котором следует наложения _цвет2_ и _color1_, соответственно, как указано в параметре _с плавающей запятой_ . Например если _число с плавающей запятой_ 0,25, возвращаемых цвет — составных 75% _color1_ и 25% _цвет2_. 
  
Другим способом подумайте, он является, что значение _с плавающей запятой_ соответствует точка в спектре из _color1_ для _цвет2_. Таким образом меньшего числа (ближе к нулю) для переходов продукты _с плавающей запятой_ ближе к _color1_во время большего числа (ближе к 1) необходимо создать переходы ближе к _цвет2_.
