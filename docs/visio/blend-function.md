# <a name="blend-function"></a>Функция BLEND

Смешивает два цвета в пропорции, указанной _параметром float._ 
  
## <a name="syntax"></a>Синтаксис

BLEND(** *color1* **, ** *color2* **, ** *float[0,1]* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color1_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Индекс цвета Visio или значение RGB первого цвета.  <br/> |
| _color2_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Индекс цвета Visio или значение RGB второго цвета.  <br/> |
| _float[0,1]_ <br/> |Обязательна  <br/> |**Double** <br/> |Пропорции для смешивания  _цветов2_  _и color1_ соответственно. Реальное число от 0 до 1 включительно.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 **RGB**
  
## <a name="remarks"></a>Примечания

Возвращаемый цвет определяется относительным соотношением, в котором необходимо смешивать _color2_ и _color1_ соответственно, как указано в _параметре float._ Например, если  _float_ составляет 0,25, возвращаемого цвета составляет 75 %  _от color1_ и 25 %  _цвета2_. 
  
Другой способ продумать, что  значение с плавающей точкой соответствует точке спектра цветов _от color1_ до _color2._ Таким образом, меньшие числа (ближе к нулю) для с плавающей заготовки создают смешение ближе _к color1,_ а большие числа (ближе к 1) создают смешение ближе _к color2._
