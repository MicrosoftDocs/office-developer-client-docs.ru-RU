# <a name="blend-function"></a>Функция BLEND

Смешивает два цвета в соответствии с пропорциями, указанными в параметре _float_ . 
  
## <a name="syntax"></a>Синтаксис

BLEND (* * *color1* * *, * * *color2* * *, * * *float [0, 1]* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color1_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Цветовой индекс Visio или значение RGB для первого цвета.  <br/> |
| _color2_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Цветовой индекс Visio или значение RGB второго цвета.  <br/> |
| _float [0, 1]_ <br/> |Обязательна  <br/> |**Double** <br/> |Пропорция, в которой накладываются _color2_ и _color1_соответственно. Вещественное число от 0 до 1 включительно.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 **RGB**
  
## <a name="remarks"></a>Примечания

Возвращаемый цвет определяется относительными пропорциями, в которых выполняется смешение _color2_ и _color1_, соответственно, как указано параметром _float_ . Например, если значение _float_ равно 0,25, возвращаемый цвет составляет 75% от _color1_ и 25% от _color2_. 
  
Еще один способ обдумать то, что значение _float_ соответствует точке цветового спектра из _color1_ в _color2_. Таким образом, меньшее число (ближе к нулю _) для_ наложения накладывается ближе к _color1_, а большее число (ближе к 1) приводит к переходу ближе к _color2_.
