# <a name="lower-function"></a>Функция LOWER

Возвращает строку, преобразованную в нижний регистр.
  
## <a name="syntax"></a>Синтаксис

LOWER(** *expression* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**Разные** <br/> | Строка, ссылка на ячейку или выражение; результат преобразуется в строку, которая затем преобразуется в нижний регистр.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Преобразование дела зависит от региональных параметров на основе текущих пользовательских параметров. 
  
## <a name="example"></a>Пример

LOWER("mIxEd CAse") 
  
Возвращает "смешанный случай".
