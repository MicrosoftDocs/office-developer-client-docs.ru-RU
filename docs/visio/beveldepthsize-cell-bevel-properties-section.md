# <a name="beveldepthsize-cell-bevel-properties-section"></a>Ячейка BevelDepthSize (раздел "Свойства багетной рамки")

Определяет размер рельефов глубины в точках. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **BevelDepthSize** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | BevelDepthSize  <br/> |
   
Для получения ссылки на ячейки **BevelDepthSize** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowBevelProperties** <br/> |
| Индекс ячейки:  <br/> |**visBevelDepthSize** <br/> |
