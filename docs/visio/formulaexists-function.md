# <a name="formulaexists-function"></a>Функция FORMULAEXISTS

Указывает, содержит ли указанная в ссылке ячейка формулу. 
  
## <a name="syntax"></a>Синтаксис

FORMULAEXISTS (** *cellref* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _cellref_ <br/> |Обязательно  <br/> |**Строка** <br/> |Ячейка, которую необходимо проверить на наличие формулы.  <br/> |
   
## <a name="remarks"></a>Примечания

Функция FORMULAEXISTS возвращает 1, если ячейка содержит формулу; Если он не содержит формулу, возвращается ноль (0).
