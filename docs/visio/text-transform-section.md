# <a name="text-transform-section"></a>Text Transform Section

Содержит сведения о размещении текстового блока фигуры.
  
## <a name="remarks"></a>Примечания

Чтобы добавить этот раздел, используйте диалоговое окно **Insert Section** (щелкните правой кнопкой мыши в окне ShapeSheet и нажмите **кнопку Вставить раздел**). Настройка значений в разделе **Преобразование** текста сопоставима с работой с текстовым блоком с помощью средства **Text Block** (группа **Инструментов** на вкладке **Главная).**
