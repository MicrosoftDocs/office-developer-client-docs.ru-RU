# <a name="text-transform-section"></a>Раздел Преобразование текста

Содержит данные о блок текста фигуры размещения.
  
## <a name="remarks"></a>Замечания

Чтобы добавить в этом разделе, используйте диалоговое окно " **Вставка раздела** " (щелкните правой кнопкой мыши в окне таблицы свойств фигуры и нажмите кнопку **Вставить раздел**). Задание значений в разделе **Преобразование текста** можно представить как работа с блок текста с помощью средства **Блок текста** (группа **средства** на вкладке **Главная** ).
