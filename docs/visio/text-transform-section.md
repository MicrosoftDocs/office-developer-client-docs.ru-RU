# <a name="text-transform-section"></a>Text Transform Section

Содержит сведения о размещении текстового блока фигуры.
  
## <a name="remarks"></a>Примечания

Чтобы добавить этот раздел, используйте диалоговое окно **"Вставка** раздела" (щелкните правой кнопкой мыши в окне таблицы фигур и выберите **"Вставка раздела").** Параметры в разделе **"Преобразование** текста" сравнимы с  работой с  блоком текста с помощью средства "Блок текста" (группа "Инструменты" на вкладке **"Главная").**
