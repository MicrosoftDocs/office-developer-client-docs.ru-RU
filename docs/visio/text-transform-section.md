# <a name="text-transform-section"></a>Text Transform Section

Содержит сведения о размещении блока текста фигуры.
  
## <a name="remarks"></a>Примечания

Чтобы добавить этот раздел, используйте диалоговое окно " **Вставка раздела** " (щелкните правой кнопкой мыши в окне таблицы свойств фигур и выберите команду " **Вставить раздел**"). Значения параметра в разделе **Преобразование текста** сравнимы для работы с блоком текста с помощью инструмента " **текстовый блок** " (Группа " **инструменты** " на вкладке " **Главная** ").
