# <a name="loctoloc-function"></a>Функция LOCTOLOC

Возвращает преобразованную точку в локальных координатах в системе координат назначения.
  
## <a name="syntax"></a>Синтаксис

LOCTOLOC(** *srcPoint* **, ** *srcRef* **, ** *dstRef* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _srcPoint_ <br/> |Обязательно  <br/> |**Строка** <br/> | Точка в локальных координатах в системе координат источника.  <br/> |
| _srcRef_ <br/> |Обязательно  <br/> |**Строка** <br/> | Ссылка на ячейку в объекте-источнике.  <br/> |
| _dstRef_ <br/> |Обязательно  <br/> |**Строка** <br/> | Ссылка на ячейку в объекте назначения.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Функция LOCTOLOC преобразует точку из локальных координат в форме источника в локальные координаты в фигуре назначения. Эту функцию можно использовать для создания фигуры, например в терминах точки из другого пространства координат. Эту функцию также можно использовать для преобразования локальной точки в координаты страницы или наоборот.
  
Эта функция работает, даже если исходные и пунктов назначения фигуры находятся в группах. Он также настраивается для поворота и пролистывок в промежуточном преобразовании.
  
Координаты источника и назначения должны быть на одной странице.
  
## <a name="example"></a>Пример

Следующая формула преобразует локальный контакт фигуры, связанной с формулой, в точку на странице.
  
```vb
LOCTOLOC(PNT(LocPinX, LocPinY), Width, ThePage!PageWidth)
```
