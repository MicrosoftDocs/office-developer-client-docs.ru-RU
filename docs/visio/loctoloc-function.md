# <a name="loctoloc-function"></a>Функция LOCTOLOC

Возвращает преобразованную точку в локальных координатах в конечной системе координат.
  
## <a name="syntax"></a>Синтаксис

LOCTOLOC (* * *сркпоинт* * *, * * *сркреф* * *, * * *дстреф* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Сркпоинт_ <br/> |Обязательный  <br/> |**String** <br/> | Точка в локальных координатах в исходной системе координат.  <br/> |
| _Сркреф_ <br/> |Обязательный  <br/> |**String** <br/> | Ссылка на ячейку в исходном объекте.  <br/> |
| _Дстреф_ <br/> |Обязательный  <br/> |**String** <br/> | Ссылка на ячейку в целевом объекте.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Функция LOCTOLOC преобразует точку из локальных координат исходной фигуры в локальные координаты конечной фигуры. Эту функцию можно использовать для создания фигуры, например, с точки зрения точки из другого координатного пространства. Эту функцию также можно использовать для преобразования локальной точки в координаты страницы или наоборот.
  
Эта функция работает, даже если исходная и конечная фигуры находятся в группах. Он также настраивается для вращения и отражения в промежуточном преобразовании.
  
Координаты источника и назначения должны находиться на одной странице.
  
## <a name="example"></a>Пример

Следующая формула преобразует локальный ПИН-код фигуры, связанной с формулой, в точку на странице.
  
```vb
LOCTOLOC(PNT(LocPinX, LocPinY), Width, ThePage!PageWidth)
```
