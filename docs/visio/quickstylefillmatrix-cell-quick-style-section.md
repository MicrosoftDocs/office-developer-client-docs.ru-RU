# <a name="quickstylefillmatrix-cell-quick-style-section"></a>QuickStyleFillMatrix Cell (Quick Style Section)

Определяет стиль Экспресс заливки, который фигура наследует от активной темы, в виде целого числа от 0-6. 
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку **QuickStyleFillMatrix** по имени из другой формулы, по значению атрибута **N** элемента **ячейки** или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | QuickStyleFillMatrix  <br/> |
   
Чтобы получить ссылку на ячейку **QuickStyleFillMatrix** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**Висровкуиккстилепропертиес** <br/> |
| Индекс ячейки:  <br/> |**Вискуиккстилефиллматрикс** <br/> |
