# <a name="quickstylefillmatrix-cell-quick-style-section"></a>Ячейка QuickStyleFillMatrix (раздел "Экспресс-стиль")

Определяет стиль заливки экспресс-стилей, который фигуры наследует от активной темы, как целое число от 0 до 6. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **QuickStyleFillMatrix** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | QuickStyleFillMatrix  <br/> |
   
Для получения ссылки на ячейки **QuickStyleFillMatrix** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowQuickStyleProperties** <br/> |
| Индекс ячейки:  <br/> |**visQuickStyleFillMatrix** <br/> |
