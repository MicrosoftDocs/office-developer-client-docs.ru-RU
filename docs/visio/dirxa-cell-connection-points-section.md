# <a name="dirx--a-cell-connection-points-section"></a>Ячейка DirX / A (раздел "Точки соединения")

Определяет значение *x* -компонент для вектор Обязательное выравнивание совпадающих точки подключения. DirX аудио- и ячейки также используется для сориентироваться вложенные ветвь динамических соединителя. В этой ячейке принимает число с плавающей запятой. 
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на DirX / ячейки по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Connections.DirX [ *i* ] где *i* = < 1 > 2, 3...  <br/> |
   
Чтобы получить ссылку на DirX / ячейки по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionConnectionPts** <br/> |
| Индекс строки:  <br/> |**visRowConnectionPts** +  *i* где *i* = 0, 1, 2  <br/> |
| Индекс ячейки:  <br/> |**visCnnctDirX** (не только строк)           **visCnnctA** (расширенные строк)  <br/> |
   
Сведения о не расширены и расширенного строк, в разделе точек подключения строки.
