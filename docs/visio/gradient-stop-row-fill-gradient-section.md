# <a name="gradient-stop-row-fill-gradient-section"></a>Gradient Stop Row (Fill Gradient Section)

Содержит цвет, прозрачность и расположение остановки градиента для градиента заполнения.
  
Строка **"Стоп Градиент"** содержит следующие ячейки. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|**Color** <br/> |Цветовое значение остановки градиента. Это значение может быть выражено в качестве индексного номера цвета в палитре документов или с помощью функций [RGB,](rgb-function-visioshapesheet.md) [THEMEVAL](themeval-function.md)или [HSL](hsl-function.md) (например).  <br/> |
|**ColorTrans** <br/> |Количество прозрачности цвета градиента в процентах.  <br/> |
|**Position** <br/> |Положение градиента остановится в направлении градиента заполнения в процентах от точки происхождения градиента до края области заполнения.  <br/> |
