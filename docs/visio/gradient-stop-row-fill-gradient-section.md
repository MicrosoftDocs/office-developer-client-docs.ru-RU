# <a name="gradient-stop-row-fill-gradient-section"></a>Строка Gradient Stop (раздел "Градиентная заливка")

Содержит цвет, прозрачность и положение градиента для градиентной заливки.
  
Строка **Градиента остановить** содержит следующие ячейки. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|**Цвет** <br/> |Значение цвета градиента. Это значение может быть выражено как номер индекса цвета в палитре документа или с помощью функции [RGB](rgb-function-visioshapesheet.md), [THEMEVAL](themeval-function.md)или [HSL](hsl-function.md) (например).  <br/> |
|**ColorTrans** <br/> |Степень прозрачности цвета градиента в процентном выражении.  <br/> |
|**Position** <br/> |Позиция градиента по Направление градиентной заливки, в процентном соотношении от точки origin градиента краю области заливки.  <br/> |
