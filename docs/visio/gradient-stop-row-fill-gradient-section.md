# <a name="gradient-stop-row-fill-gradient-section"></a>Gradient Stop Row (Fill Gradient Section)

Содержит цвет, прозрачность и положение остановки градиента для градиента заливки.
  
Строка **остановки градиента** содержит следующие ячейки. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|**Цвет** <br/> |Значение цвета остановки градиента. Это значение может быть выражено как номер индекса цвета в палитре документа или с помощью функций [RGB](rgb-function-visioshapesheet.md), [THEMEVAL](themeval-function.md)или [HSL](hsl-function.md) (например,).  <br/> |
|**Колортранс** <br/> |Степень прозрачности цвета градиента в процентах.  <br/> |
|**Position** <br/> |Положение остановки градиента вдоль направления градиента заполнения в процентах от точки начала градиента до края области заливки.  <br/> |
