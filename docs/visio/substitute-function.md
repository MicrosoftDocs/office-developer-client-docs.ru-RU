# <a name="substitute-function"></a>Функция SUBSTITUTE

Заменяет часть текстовой строки другой текстовой строкой. 
  
## <a name="syntax"></a>Синтаксис

 *Start_num* [* old_text ** ** * * * * *old_text* * * *new_text* * * * * * * * * * * * * * * * * * * * * * * *] [, * * * * *] [, * * *ignore_case_opt* * * 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _text_ <br/> |Обязательный  <br/> |**String** <br/> | Текст или ссылка на ячейку, содержащую текст, для которого нужно заменять символы.  <br/> |
| _old_text_ <br/> |Обязательный  <br/> |**String** <br/> | Текст, который необходимо заменить.  <br/> |
| _new_text_ <br/> |Обязательный  <br/> |**String** <br/> | Текст, который вы хотите использовать для замены _old_text_.  <br/> |
| _start_num_opt_ <br/> |Необязательный  <br/> |**Числовой** <br/> |Указывает, какие экземпляры old_text заменять.  <br/> |
| _ignore_case_opt_ <br/> |Необязательный  <br/> |**Логический** <br/> |Значение FALSE в случае учета регистра; в противном случае — значение TRUE. Значение по умолчанию — FALSE.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

 Если указать _start_num_opt_, заменяется только это вхождение _old_text_ . В противном случае каждое вхождение _old_text_ в _тексте_ изменяется на _new_text._
  
Используйте функцию подстановки, если хотите заменить определенный текст в текстовой строке. Если вы хотите заменить текст, который находится в определенном месте в текстовой строке, используйте функцию REPLACE.
  
## <a name="example"></a>Пример

Замена ("1 января 2003 января", "Январь", "Янв") 
  
Возвращает значение "1 января 2003". 
  
Замена ("1 января 2003 г.", "Январь", "Янв") 
  
Возвращает значение "1 января 2003". Изменения не выполняются, так как при поиске текста учитывается регистр.
