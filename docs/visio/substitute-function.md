# <a name="substitute-function"></a>Функция SUBSTITUTE

Заменяет часть текстовой строки другой текстовой строкой. 
  
## <a name="syntax"></a>Синтаксис

 Замена (* * *текстовый* * *, * * *стар_текст* * *, * * *новый_текст* * * [, * * *Нач_позиция* * *] [, * * *игноре_касе_опт* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _text_ <br/> |Обязательный  <br/> |**String** <br/> | Текст или ссылка на ячейку, содержащую текст, для которого нужно заменять символы.  <br/> |
| _стар_текст_ <br/> |Обязательный  <br/> |**String** <br/> | Текст, который необходимо заменить.  <br/> |
| _новый_текст_ <br/> |Обязательный  <br/> |**String** <br/> | Текст, который будет использоваться для замены _стар_текст_.  <br/> |
| _старт_нум_опт_ <br/> |Необязательный  <br/> |**Числовой** <br/> |Указывает, какие экземпляры стар_текст заменяют.  <br/> |
| _игноре_касе_опт_ <br/> |Необязательно  <br/> |**Boolean** <br/> |ЗНАЧЕНИЕ FALSE в случае учета регистра; в противном случае — значение TRUE. Значение по умолчанию — FALSE.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Строка
  
## <a name="remarks"></a>Примечания

 Если вы укажете _старт_нум_опт_, заменяется только это вхождение _стар_текст_ . В противном случае каждое вхождение _стар_текст_ в _тексте_ изменится на _новый_текст._
  
Используйте функцию подСТАНОВКи, если хотите заменить определенный текст в текстовой строке. Если вы хотите заменить текст, который находится в определенном месте в текстовой строке, используйте функцию rePLACE.
  
## <a name="example"></a>Пример

Замена ("1 января января", "Январь", "Янв") 
  
Возвращает значение "1 января 2003". 
  
Замена ("1 января 2003 г.", "Январь", "Янв") 
  
Возвращает значение "1 января 2003". Изменения не выполняются, так как при поиске текста учитывается регистр.
