# <a name="substitute-function"></a>Функция SUBSTITUTE

Заменяет часть текстовой строки другой строкой текста. 
  
## <a name="syntax"></a>Синтаксис

 ЗАМЕНА (** *текст* **, ** *стар_текст* **, ** *нов_текст* ** [, ** *Начальная позиция* **] [, ** *ignore_case_opt* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _text_ <br/> |Обязательный  <br/> |**Строка** <br/> | Текст или ссылка на ячейку, содержащую текст, для которого требуется замена знаков.  <br/> |
| _стар_текст_ <br/> |Обязательный  <br/> |**Строка** <br/> | Текст, который вы хотите заменить.  <br/> |
| _нов_текст_ <br/> |Обязательный  <br/> |**Строка** <br/> | Текст, который будет использоваться для замены _стар_текст_.  <br/> |
| _start_num_opt_ <br/> |Необязательный  <br/> |**Числовой** <br/> |Указывает, какие вхождения стар_текст для замены.  <br/> |
| _ignore_case_opt_ <br/> |Необязательный  <br/> |**Логическое значение** <br/> |FALSE, если с учетом регистра; в противном случае — значение TRUE. Значение по умолчанию — FALSE.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

 При указании _start_num_opt_заменяется только это вхождение _стар_текст_ . В противном случае каждого вхождения _стар_текст_ в _тексте_ изменяется на _нов_текст._
  
Используйте функцию ЗАМЕНЫ, когда необходимо заменить определенный текст в текстовой строки. Если вы хотите заменить текст, который встречается в определенное расположение в строку текста, используйте функцию REPLACE.
  
## <a name="example"></a>Пример

Заменить («1 января 2003», «Январь», «Янв») 
  
Возвращает «1 ЯНВАРЯ 2003». 
  
Заменить («1 января 2003», «январь», «Янв») 
  
Возвращает «1 января 2003». Не является изменений, так как поиск текста задается с учетом регистра.
