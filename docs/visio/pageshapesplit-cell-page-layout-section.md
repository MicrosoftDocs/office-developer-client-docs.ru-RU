# <a name="pageshapesplit-cell-page-layout-section"></a>PageShapeSplit Cell (Page Layout Section)

Указывает, можно ли автоматически разделить фигуры на странице.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|нуль  <br/> |Не разрешать автоматическое разбиение фигур.  <br/> |**висплосплитноне** <br/> |
|1,1  <br/> |Разрешить автоматическое разбиение фигур (по умолчанию).  <br/> |**висплосплиталлов** <br/> |
   
## <a name="remarks"></a>Примечания

Автоматическое разбиение на фигуры включено и отключено на трех разных уровнях: приложение, страница и фигура. По умолчанию разделение разрешено на уровне приложений и страниц. Параметры по умолчанию для фигур зависят от типа документа. 
  
Чтобы включить или отключить разбиение на уровне приложения, используйте параметр **включить разделение соединителей** на вкладке **Дополнительно** диалогового окна **Параметры Visio** (нажмите кнопку **Office** , выберите пункт **Параметры** на вкладке **Visio** , а затем нажмите кнопку **Дополнительно** ). 
  
Чтобы включить или отключить разбивку на уровне фигур, просмотрите ячейки ShapeSplit и ShapeSplittable. 
  
Чтобы получить ссылку на ячейку PageShapeSplit по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |PageShapeSplit  <br/> |
   
Чтобы получить ссылку на ячейку PageShapeSplit по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**висровпажелайаут** <br/> |
|Индекс ячейки:  <br/> |**висплосплит** <br/> |
