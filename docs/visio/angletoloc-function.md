# <a name="angletoloc-function"></a>Функция ANGLETOLOC

Возвращает преобразованный угол в локальной системе координат конечной фигуры. Преобразует угол из локальных координат исходной фигуры в локальные координаты конечной фигуры. 
  
## <a name="syntax"></a>Синтаксис

ANGLETOLOC (* * *сркангле* * *, * * *сркреф* * *, * * *дстреф* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Сркангле_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Угол в исходной системе координат.  <br/> |
| _Сркреф_ <br/> |Обязательный  <br/> |**String** <br/> | Ссылка на ячейку в исходном объекте, например фигуру, группу, страницу и т. д.  <br/> |
| _Дстреф_ <br/> |Обязательный  <br/> |**String** <br/> |Ссылка на ячейку в целевом объекте, например фигуру, группу, страницу и т. д.  <br/> |
   
## <a name="remarks"></a>Примечания

Функцию ANGLETOLOC можно использовать для установки локальных ячеек угла в фигуре с точки зрения угла из другого пространства координат.
  
Эта функция работает, даже если исходная и конечная фигуры находятся в группах. Он также настраивается для вращения и отражения в промежуточном преобразовании.
  
Координаты источника и назначения должны находиться на одной странице.
