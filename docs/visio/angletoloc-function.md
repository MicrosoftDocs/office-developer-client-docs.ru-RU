# <a name="angletoloc-function"></a>Функция ANGLETOLOC

Возвращает преобразованный угол в локальной системе координат фигуры назначения. Преобразует угол с локальных координат в форме источника в локальные координаты в форме назначения. 
  
## <a name="syntax"></a>Синтаксис

ANGLETOLOC ***(srcAngle***, ***srcRef***, ***dstRef*** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _srcAngle_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Угол в системе координат источника.  <br/> |
| _srcRef_ <br/> |Обязательный  <br/> |**String** <br/> | Ссылка на ячейку в объекте источника, например фигуру, группу, страницу и так далее.  <br/> |
| _dstRef_ <br/> |Обязательный  <br/> |**String** <br/> |Ссылка на ячейку в объекте назначения, например фигуру, группу, страницу и так далее.  <br/> |
   
## <a name="remarks"></a>Примечания

Вы можете использовать функцию ANGLETOLOC для набора локальных угловых ячеек в форме с точки зрения угла из другого пространства координат.
  
Эта функция работает даже в том случае, если исходные и назначения фигуры находятся в группах. Он также настраивается для вращения и сальто в промежуточной трансформации.
  
Координаты источника и назначения должны быть на одной странице.
