# <a name="controls-section"></a>Раздел "Элементы управления"

Содержит строку и ячейки для каждого управляющего маркера, указанного для фигуры.
  
## <a name="remarks"></a>Замечания

Можно определить управляющие маркеры в разделе **элементы управления** в окне таблицы свойств фигуры. Если фигура имеет раздел без **элементов управления** , его можно вставить в окне таблицы свойств фигуры. Щелкните правой кнопкой мыши в окне таблицы свойств фигуры, нажмите кнопку **Вставить раздел**и установите флажок **элементы управления** . Чтобы добавить маркеры элемента управления в существующий раздел **элементы управления** , щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить строку** в контекстном меню.
