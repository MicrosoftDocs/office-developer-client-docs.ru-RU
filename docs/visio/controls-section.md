# <a name="controls-section"></a>Controls Section

Содержит строку и ячейки для каждой ручки управления, определенной для фигуры.
  
## <a name="remarks"></a>Примечания

Вы можете определить ручки управления в разделе **Элементы** управления в окне ShapeSheet. Если в вашей фигуре нет **раздела Элементы** управления, вы можете вставить его в окно ShapeSheet. Щелкните правой кнопкой мыши в окне ShapeSheet, щелкните **Раздел Вставить,** а затем выберите поле **управления.** Чтобы добавить ручки управления в существующий раздел **Элементы** управления, щелкните правой кнопкой мыши строку и нажмите **кнопку Вставить строку** в меню ярлыка.
