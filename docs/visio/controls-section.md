# <a name="controls-section"></a>Controls Section

Содержит строку и ячейки для каждого управляющего маркера, определенного для фигуры.
  
## <a name="remarks"></a>Примечания

Вы можете определить управляющие маркеры в разделе **элементы управления** в окне таблицы свойств фигуры. Если у фигуры нет раздела **элементы управления** , вы можете вставить его в окно таблицы свойств фигуры. Щелкните правой кнопкой мыши в окне таблицы свойств фигуры, выберите команду **Вставить раздел**, а затем установите флажок **элементы управления** . Чтобы добавить управляющие маркеры в существующий раздел " **элементы управления** ", щелкните строку правой кнопкой мыши и выберите команду **Вставить строку** в контекстном меню.
