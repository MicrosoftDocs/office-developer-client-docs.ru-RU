# <a name="controls-section"></a>Controls Section

Содержит строку и ячейки для каждого окна управления, определенного для фигуры.
  
## <a name="remarks"></a>Примечания

Вы можете определить органы управления в разделе **"Элементы** управления" в окне таблицы фигур. Если в фигуре нет раздела **"Элементы** управления", его можно вставить в окно таблицы фигур. Щелкните правой кнопкой мыши в окне таблицы фигур, выберите **"Вставить раздел"** и выберите **"Элементы** управления". Чтобы добавить органы управления в существующий раздел **"Элементы** управления", щелкните правой кнопкой мыши строку и выберите пункт **"Вставка** строки" в ярлыковом меню.
