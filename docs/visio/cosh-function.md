# <a name="cosh-function"></a>Функция COSH

Возвращает гиперболический косинус угла.
  
## <a name="syntax"></a>Синтаксис

COSH ( *угол*) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _угол_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Угол которого необходимо получить гиперболический косинус.  <br/> |
   
## <a name="remarks"></a>Замечания

Функция COSH использует радианах, если не указать разные единицы измерения в *угол* . 
  
## <a name="example"></a>Пример

Cosh(45 deg) 
  
Возвращает 1.3246.
