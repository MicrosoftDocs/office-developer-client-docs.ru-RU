# <a name="cosh-function"></a>Функция COSH

Возвращает гиперболический косинус угла.
  
## <a name="syntax"></a>Синтаксис

COSH ( *угол*) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _градусов_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Угол, для которого требуется получить гиперболический косинус.  <br/> |
   
## <a name="remarks"></a>Примечания

Функция COSH использует радианы, если вы не укажете другую единицу измерения в качестве *угла* . 
  
## <a name="example"></a>Пример

COSH (45 градусов) 
  
Возвращает 1,3246.
