# <a name="fill-format-section"></a>Fill Format Section

Отображает текущие атрибуты форматирования заливки для фигуры и тени перепада фигуры, включая шаблон, цвет переднего плана и цвет фона. 
  
## <a name="remarks"></a>Примечания

Форматы заливки можно  настроить в области "Формат фигуры"  (на вкладке "Главная", в группе "Стили фигуры" щелкните "Заполнить"  и "Параметры заливки"), применив стиль заливки или сделав запись в ячейке в разделе "Формат заливки".
