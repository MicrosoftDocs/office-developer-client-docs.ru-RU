# <a name="fill-format-section"></a>Раздел "Формат заливки"

Отображает текущую заливку форматирование атрибутов фигуры и тени фигуры, включая шаблон, цвет переднего плана и цвета фона. 
  
## <a name="remarks"></a>Замечания

Можно установить форматы заливки в области **Формат фигуры** (на вкладке **Главная** в группе **Стили фигур** щелкните **заливки**и выберите пункт **Параметры заливки**), применив стиль заливки или путем создания записи в ячейку в **Формате заливки** раздел.
