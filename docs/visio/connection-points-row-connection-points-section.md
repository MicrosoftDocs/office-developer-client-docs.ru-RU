# <a name="connection-points-row-connection-points-section"></a>Connection Points Row (Connection Points Section)

Содержит  *x*  и  *y-координаты,*  горизонтальное и вертикальное направление и введите для одной точки подключения на фигуре. Координаты точек подключения измеряются по происхождению фигуры. Фигура содержит по одной строке Точки подключения для каждой точки подключения. 
  
Если строки точек подключения называются, эти имена отображаются как Подключения. *имя*  в окне ShapeSheet. Строки Точки подключения содержат следующие ячейки. Дополнительные сведения см. в конкретных разделах ячейки. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-connection-points-section.md) <br/> |X-координата для точки подключения в локальных координатах.   <br/> |
|[Да](y-cell-connection-points-section.md) <br/> |*y-coordinate* для точки подключения в локальных координатах.  <br/> |
|[DirX/A](dirxa-cell-connection-points-section.md) <br/> |*X-компонент* для необходимого вектора выравнивания точки подключения. Он также используется для сориентировать присоединенную ногу динамического соединителя. Эта ячейка принимает значение плавающей точки.  <br/> |
|[DirY/B](diryb-cell-connection-points-section.md) <br/> |*Компонент y* для необходимого вектора выравнивания точки подключения. Он также используется для сориентировать присоединенную ногу динамического соединителя. Эта ячейка принимает значение плавающей точки.  <br/> |
|[Тип/C](typec-cell-connection-points-section.md) <br/> |Тип точки подключения (0 = внутрь; 1 = наружу; 2 = внутрь + наружу).  <br/> |
|[D](d-cell-connection-points-section.md) <br/> |Ячейка нуля, которую можно использовать для ввода или тестирования формул. Чтобы получить доступ к этой ячейке, щелкните правой кнопкой мыши строку и нажмите **кнопку Изменить** строку в меню ярлыка.  <br/> |
   
## <a name="remarks"></a>Примечания

Ячейки в подключениях. *Строки*  имен помечены DirX/A, DirY/B и Type/C, так как эти строки можно расширить или не расширить. 
  
Большинство точек подключения (все точки подключения, созданные с помощью пользовательского интерфейса) не являются расширенными и имеют ячейки DirX, DirY и Type. Тип **строки — visTagCnnctPt** или **visTagCnnctNamed.**
  
В неудлиненых строках ячейки DirX и DirY вместе определяют вектор направления, который влияет на поворот фигур, участвующих в подключениях с помощью точки подключения. Если обе нулевые, точка ненаправленна. Точки подключения имеют следующие типы:
  
- Внутрь (0), что означает, что фигуры приклеиваются к ним. Этот параметр используется по умолчанию.
    
- Наружу (1), что означает, что эти точки подключения будут приклеены к внешним точкам подключения.
    
- И внутрь, и наружу (2), в этом случае направление — это внутреннее направление, которое будет обратным, если используется в качестве внешнего подключения.
    
Расширенные строки имеют ячейки A, B, C и D и ведут себя как беснаправленные невытянутые строки типа Inward. Расширенные строки обычно не используются, но их можно использовать для связывать данные с точкой подключения в ячейках A, B, C и D. Их тип **строки visTagCnnctPtABCD** или **visTagCnnctNamedABCD**. Расширенные строки можно определить по присутствию формулы в ячейке D. 
  
 Можно добавить как можно больше подключений.  *Строки*  имен по мере необходимости назначьте значимые имена строкам и установите значения ячейки. Чтобы добавить точку подключения к существующему разделу Точки подключения, щелкните правой кнопкой мыши строку и нажмите **строку Вставить** строку в меню ярлыка. 
  
Вы можете ссылаться на ячейки строки Точки подключения по имени строки, которое отображается в окне ShapeSheet в красном тексте. Чтобы изменить имя строки, щелкните его, а затем введите имя, например  *Custom,*  чтобы создать имя строки Connections.Custom. Затем вы можете ссылаться на X-ячейку с помощью, например, Connections.Custom.X или Connections.X1, если вы хотите использовать номер строки. 
  
Имя строки, которое вы вводите, должно быть уникальным в разделе. При создании имени одной строки в разделе Точки подключения Microsoft Office Visio имена всех строк в разделе с именем по умолчанию, Connections.Row_ *n* . 
  
Строки с именами точек подключения несовместимы с версиями Visio более ранних 5.0. При сохранении файла Visio с именами строк Точки подключения в более ранний формат ссылки на названные строки точки подключения преобразуются в индексируемые ссылки, а имена строк теряются.
