# <a name="connection-points-row-connection-points-section"></a>Connection Points Row (Connection Points Section)

Содержит координаты *x* и *y* , горизонтальное и вертикальное направление и тип для одной точки подключения на фигуре. Координаты точек соединения измеряются относительно начала фигуры. Фигура содержит одну строку точек подключения для каждой точки подключения. 
  
Если строки точек подключения именуются, эти имена отображаются как подключения. *имя* в окне таблицы свойств фигуры. Строки точек подключения содержат следующие ячейки. Более подробную информацию можно найти в разделах, посвященных определенным ячейкам. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-connection-points-section.md) <br/> |Координата *x* точки подключения в локальных координатах.  <br/> |
|[Y (да)](y-cell-connection-points-section.md) <br/> |Координата *y* точки подключения в локальных координатах.  <br/> |
|[Ячейка dirx/A](dirxa-cell-connection-points-section.md) <br/> |Компонент *x* для требуемого вектора выравнивания соответствующей точки подключения. Он также используется для ориентации присоединенного участка динамической соединительной линии. Эта ячейка принимает значение с плавающей запятой.  <br/> |
|[Ячейка diry/B](diryb-cell-connection-points-section.md) <br/> |Компонент *y* для требуемого вектора выравнивания соответствующей точки подключения. Он также используется для ориентации присоединенного участка динамической соединительной линии. Эта ячейка принимает значение с плавающей запятой.  <br/> |
|[Type/C](typec-cell-connection-points-section.md) <br/> |Тип точки подключения (0 = в сторону; 1 = наружу; 2 = внутрь + наружу).  <br/> |
|[D](d-cell-connection-points-section.md) <br/> |Вспомогательная ячейка, которую можно использовать для ввода или тестирования формул. Чтобы получить доступ к этой ячейке, щелкните строку правой кнопкой мыши и выберите в контекстном меню команду **изменить тип строки** .  <br/> |
   
## <a name="remarks"></a>Примечания

Ячейки в подключениях. Строка *имени* имеет метки ячейка dirx/A, ячейка diry/B и Type/C, так как эти строки могут быть расширенными или нерасширенными. 
  
Большинство точек подключения (все точки подключения, созданные с помощью пользовательского интерфейса) не являются расширенными и содержат ячейки ячейка dirx, ячейка diry и Type. Тип строки — **вистагкннктпт** или **вистагкннктнамед.**
  
В нерасширенных строках ячейки ячейка dirx и ячейка diry вместе определяют вектор направления, который влияет на поворот фигур, участвующих в подключениях, с помощью точки подключения. Если оба значения равны нулю, точка — без направления. Точки подключения имеют следующие типы:
  
- В обратном направлении (0) это означает, что фигуры привязываются к ним. Это значение используется по умолчанию.
    
- Наружу (1), что означает, что эти точки подключения будут приклеить к точкам подключения.
    
- Как внутрь, так и наружу (2), в этом случае направлением является направление, которое изменяется при использовании в качестве соединения по внешнему интерфейсу.
    
Расширенные строки имеют ячейки, B, C и D и ведут себя как нерасширенные строки типа "без направления". Расширенные строки обычно не используются, но их можно использовать для связывания данных с точкой подключения в ячейках A, B, C и D. Тип строки — **вистагкннктптабкд** или **вистагкннктнамедабкд**. Расширенные строки можно идентифицировать по наличию формулы в ячейке D. 
  
 Можно добавить любое количество подключений.  ** укажите имена строк, назначьте им значимые имена и задайте значения ячеек. Чтобы добавить точку подключения к существующему разделу точек подключения, щелкните строку правой кнопкой мыши и выберите команду **Вставить строку** в контекстном меню. 
  
Вы можете ссылаться на ячейки строк точек подключения по имени строки, которое отображается в окне таблицы свойств фигуры в виде красного текста. Чтобы изменить имя строки, щелкните его, а затем введите имя, например *Custom* , например, чтобы создать имя строки Connections. Custom. После этого вы можете ссылаться на ячейку X с помощью Connections. Custom. X, например, или Connections. x1, если вы хотите использовать номер строки. 
  
Введенное имя строки должно быть уникальным в пределах раздела. При создании имени для одной строки в разделе точки подключения Microsoft Office Visio называет все строки в разделе на имя по умолчанию (Connections. Ров_ *n* ). 
  
Именованные точки подключения строки несовместимы с версиями Visio, предшествующими 5,0. При сохранении файла документа Visio с именованными точками подключения к более раннему формату ссылки на именованные точки подключения преобразуются в индексируемые ссылки, а имена строк теряются.
