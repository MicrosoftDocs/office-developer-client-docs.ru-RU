# <a name="topmargin-cell-text-block-format-section"></a>Ячейка TopMargin (раздел "Формат текстового блока")

Определяет расстояние между верхней границей блока текста и первой строки текста, которые он содержит. Значение по умолчанию — 4.0000 точка. Это значение не зависит от масштаба документа. Если документа изменяется, верхнее поле остается неизменным.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки TopMargin по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | TopMargin  <br/> |
   
Для получения ссылки на ячейки TopMargin по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowText** <br/> |
| Индекс ячейки:  <br/> |**visTxtBlkTopMargin** <br/> |
