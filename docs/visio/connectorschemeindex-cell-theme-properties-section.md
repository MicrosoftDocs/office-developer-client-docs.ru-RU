# <a name="connectorschemeindex-cell-theme-properties-section"></a>ConnectorSchemeIndex Cell (Theme Properties Section)

Определяет схему соединители темы, применяемой к фигуре, в виде integer. 
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку **ConnectorSchemeIndex** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ConnectorSchemeIndex  <br/> |
   
Чтобы получить ссылку на ячейку **ConnectorSchemeIndex** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowThemeProperties** <br/> |
| Индекс ячейки:  <br/> |**visConnectorSchemeIndex ** <br/> |
