# <a name="listsheetref-function"></a>Функция LISTSHEETREF

Возвращает ссылку на лист фигуры контейнера списка, который содержит фигуру.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010 
  
## <a name="syntax"></a>Синтаксис

LISTMEMBERCOUNT()
  
### <a name="return-value"></a>������������ ��������

Справочник по таблице свойств фигуры
  
## <a name="remarks"></a>Замечания

Если фигура не является список компонентов, функция LISTSHEETREF возвращает #REF!.
  
## <a name="example"></a>Пример

LISTSHEETREF(1)! Высота 
  
Возвращает значение в ячейке высоту фигуры контейнера списка, который содержит фигуру.
