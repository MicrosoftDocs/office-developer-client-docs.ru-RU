# <a name="reflectionsize-cell-additional-effect-properties-section"></a>Ячейка ReflectionSize (раздел "Дополнительные свойства эффекта")

Определяет размер отражения относительно фигуры в процентном соотношении от 0,0 100,0%. Фигура со значением 0% в ячейке **ReflectionSize** не имеет отражения; значение 100% отображает полный зеркальным отображением фигуры. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **ReflectionSize** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | ReflectionSize  <br/> |
   
Для получения ссылки на ячейки **ReflectionSize** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowOtherEffectProperties** <br/> |
| Индекс ячейки:  <br/> |**visReflectionSize** <br/> |
