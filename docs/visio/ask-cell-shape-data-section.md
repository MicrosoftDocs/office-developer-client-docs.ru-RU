# <a name="ask-cell-shape-data-section"></a>Ask Cell (Shape Data Section)

Определяет, запрашивается ли у пользователя ввод данных фигуры при ее создания или копировании.
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Попросите пользователя ввести данные фигуры в диалоговом окне **"Определение данных** фигуры".  <br/> |
|FALSE  <br/> |Не просите пользователя вводить данные.  <br/> |
   
## <a name="remarks"></a>Примечания

Значение в этой ячейке  соответствует установленного в  диалоговом окне "Определение данных фигуры" в поле "Задать данные фигуры" (щелкните правой кнопкой мыши фигуру, найдите пункт **"Данные"** и выберите пункт **"Определить данные фигуры").**
  
Чтобы получить ссылку на ячейку Ask по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Prop. *name*  . Проверьте, где находится реквизит.  *это*  имя строки настраиваемого свойства.  <br/> |
   
Чтобы получить ссылку на ячейку Ask по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionProp** <br/> |
|Индекс строки:  <br/> |**visRowProp**  +   *i,* *где i* = 0, 1, 2,...  <br/> |
|Индекс ячейки:  <br/> |**visCustPropsAsk** <br/> |
