# <a name="ask-cell-shape-data-section"></a>Ячейка Ask (раздел "Данные фигуры")

Определяет, является ли пользователь будет опрошен для ввода данных фигур для фигуры при создании экземпляра или фигуры и копировании.
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Попросите пользователя ввести данные фигуры в диалоговом окне **Определение данных фигуры** .  <br/> |
|FALSE  <br/> |Не запрашивать пользователя для ввода данных.  <br/> |
   
## <a name="remarks"></a>Замечания

Значение в этой ячейке соответствующий флажок **Запрашивать при перетаскивании** в диалоговом окне **Определение данных фигуры** (щелкните правой кнопкой мыши фигуру, выберите пункт **данные**и выберите **Определение данных фигуры**).
  
Для получения ссылки на ячейку Ask по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Свойства. *имя* . Убедитесь, где свойств.  *имя* — это имя настраиваемого свойства строки.  <br/> |
   
Для получения ссылки на ячейки Ask по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionProp** <br/> |
|Индекс строки:  <br/> |**visRowProp** +  *i* где *i* = 0, 1, 2,...  <br/> |
|Индекс ячейки:  <br/> |**visCustPropsAsk** <br/> |
