# <a name="imgoffsety-cell-foreign-image-info-section"></a>Ячейка ImgOffsetY (раздел "Сведения о внешнем изображении")

Определяет, что объект расстояние от начала границы объекта смещения по вертикали. Значение по умолчанию равно 0. Режим панорамирования объект с изменениями средство **обрезки** это значение. 
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку ImgOffsetY по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | ImgOffsetY  <br/> |
   
Для получения ссылки на ячейки ImgOffsetY по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowForeign** <br/> |
| Индекс ячейки:  <br/> |**visFrgnImgOffsetY** <br/> |
