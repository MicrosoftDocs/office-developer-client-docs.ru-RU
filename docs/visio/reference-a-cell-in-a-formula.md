# <a name="reference-a-cell-in-a-formula"></a>Ссылка на ячейку в формуле

1. Дважды щелкните ячейку, в которую требуется поместить ссылку, а затем поместите точку вставки в нужное место в формуле.
    
2. Щелкните ячейку, на которую вы хотите добавить ссылку.
    
3. Нажмите клавишу ВВОД, чтобы принять формулу. Или нажмите клавишу ESC, чтобы отменить изменения.
