# <a name="reference-a-cell-in-a-formula"></a>Ссылка на ячейку в формуле

1. Дважды щелкните ячейку, в которую требуется поместить ссылку и затем поместите курсор в расположении в формулу.
    
2. Щелкните ячейку, которую требуется создать ссылку.
    
3. Нажмите клавишу ВВОД, чтобы принять формулу. Или нажмите клавишу ESC, чтобы отменить изменения.
