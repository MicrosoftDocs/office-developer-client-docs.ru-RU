# <a name="reference-a-cell-in-a-formula"></a>Ссылка на ячейку в формуле

1. Дважды щелкните ячейку, в которую нужно разместить ссылку, а затем поместите точку вставки в нужное расположение в формуле.
    
2. Щелкните ячейку, на которая вы хотите ссылаться.
    
3. Нажмите ввод, чтобы принять формулу. Или нажмите клавишу ESC, чтобы отменить изменения.
