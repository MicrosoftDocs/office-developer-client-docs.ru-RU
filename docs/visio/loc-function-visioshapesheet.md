# <a name="loc-function-visioshapesheet"></a>Функция LOC (VisioShapeSheet)

Принимает точку, определенную в локальной системе координат одной фигуры и возвращает эквивалентную точку, выраженную в локальной системе координат фигуры, связанной с формулой. 
  
## <a name="syntax"></a>Синтаксис

LOC (** *пункт* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _точка_ <br/> |Обязательный  <br/> |**Строка** <br/> | Точка, определенные в локальной системе координат одной фигуры.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Строка
  
## <a name="remarks"></a>Замечания

Локальные координаты отсчитываются с левого нижнего угла прямоугольник выделения фигуры. Обе фигуры должны располагаться на той же странице.
  
## <a name="example"></a>Пример

LOC (раздела PNT (Sheet.5! LocPinX Sheet.5! LocPinY)) 
  
В этом выражении раздела PNT преобразует набор локальных координат Sheet.5 точки. (Sheet.5 является другую фигуру на одной странице документа). Затем LOC преобразует текущего момента эквивалентный точки в текущей форме локальной системе координат, относящиеся к левом нижнем углу прямоугольник выделения текущей фигуры. 
  
5 в Sheet.5 — это идентификатор для фигуры, которая отображается в диалоговом окне **Имени фигуры** (вкладки "**Разработчик** ").
