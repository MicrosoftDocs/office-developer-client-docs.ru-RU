# <a name="loc-function-visioshapesheet"></a>LOC Function (VisioShapeSheet)

Принимает точку, определяемую в локальных координатах одной фигуры, и возвращает эквивалентную точку, выраженную в локальных координатах фигуры, связанной с формулой. 
  
## <a name="syntax"></a>Синтаксис

LOC(** *point* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _point_ <br/> |Обязательно  <br/> |**Строка** <br/> | Точка, определяемая в локальных координатах одной фигуры.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Локальные координаты измеряются из левого нижнего угла прямоугольника выделения фигуры. Обе фигуры должны быть на одной странице.
  
## <a name="example"></a>Пример

LOC(PNT(Sheet.5! LocPinX, Sheet.5! LocPinY)) 
  
В этом выражении PNT преобразует набор локальных координат в листе Sheet.5 в точку. (Лист.5 — это другая фигура на той же странице рисования.) Затем LOC преобразует эту точку в эквивалентную точку в локальной системе координат текущей фигуры относительно левого нижнего угла прямоугольника выделения текущей фигуры. 
  
5 в листе Sheet.5 — это номер ИД фигуры, который отображается в диалоговом окне **"Имя** фигуры" (вкладка **"Разработчик").**
