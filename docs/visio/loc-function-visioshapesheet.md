# <a name="loc-function-visioshapesheet"></a>LOC Function (VisioShapeSheet)

Принимает точку, определенную в локальных координатах одной фигуры, и возвращает эквивалентную точку, выраженную в локальных координатах фигуры, связанной с формулой. 
  
## <a name="syntax"></a>Синтаксис

LOC (* * *Point* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Указывает_ <br/> |Обязательный  <br/> |**String** <br/> | Точка, определенная в локальных координатах одной фигуры.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Локальные координаты измеряются в левом нижнем углу прямоугольника выделения фигуры. Обе фигуры должны размещаться на одной странице.
  
## <a name="example"></a>Пример

LOC (PNT (sheet. 5! LocPinX, лист. 5! LocPinY)) 
  
В этом выражении PNT преобразует набор локальных координат в Sheet. 5 в точку. (Sheet. 5 — это другая фигура на той же странице документа.) Затем LOC преобразует эту точку в эквивалентную точку в локальной системе координат текущей фигуры относительно левого нижнего угла прямоугольника выделения текущей фигуры. 
  
5 in Sheet. 5 — это ИДЕНТИФИКАЦИОНный номер фигуры, который отображается в диалоговом окне **имя фигуры** (вкладка "**разработчик** ").
