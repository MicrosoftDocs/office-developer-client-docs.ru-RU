# <a name="fill-gradient-section"></a>Fill Gradient Section

Отображает значения цвета для градиента заливки, примененного к фигуре. 
  
## <a name="remarks"></a>Примечания

Каждая [строка Градиентной остановки (раздел "Заполнение градиента")](gradient-stop-row-fill-gradient-section.md) в разделе представляет собой остановку градиента в заливке. Этот раздел может быть пустым. 
  
Параметры градиента используют только первые 10 строк, содержащихся в разделе. Все **строки остановки** градиента после 10 игнорируются. 
  
Вы можете настроить свойства градиента заливки в  области "Форма  формата" (на вкладке "Главная" в группе стилей фигур щелкните "Заполнить" и выберите "Параметры  заливки").  
  
Вы можете включить или отключить заливку градиента для фигуры, изменив значение ячейки [FillGradientEnabled (Gradient Properties Section).](fillgradientenabled-cell-gradient-properties-section.md)
