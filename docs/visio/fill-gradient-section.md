# <a name="fill-gradient-section"></a>Fill Gradient Section

Отображает значения цвета для градиента заполнения, примененного к фигуре. 
  
## <a name="remarks"></a>Примечания

Каждая [строка стоп-строки Gradient (Заполните раздел Градиент)](gradient-stop-row-fill-gradient-section.md) в разделе представляет остановку градиента в заполнении. Этот раздел может быть пустым. 
  
Только первые 10 строк, содержащиеся в разделе, используются для параметров градиента. Все **строки остановки Gradient** за 10 игнорируются. 
  
Свойства градиента заполнения можно установить в области **Формы** формата (на вкладке Главная, в группе **Стилей** формы нажмите кнопку **Заполните** и нажмите кнопку  **Параметры заполнения).** 
  
Вы можете включить или отключить заполнение градиента для фигуры, изменив значение ячейки [FillGradientEnabled (Gradient Properties Section).](fillgradientenabled-cell-gradient-properties-section.md)
