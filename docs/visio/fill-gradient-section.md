# <a name="fill-gradient-section"></a>Раздел "Градиентная заливка"

Отображаются значения цвета для градиентной заливки, применяемой к фигуре. 
  
## <a name="remarks"></a>Замечания

Каждая строка [градиента остановить строки (раздел градиентной заливки)](gradient-stop-row-fill-gradient-section.md) в разделе представляет градиента в заливки. В этом разделе может быть пустым. 
  
Только первые 10 строк, содержащихся в разделе используются параметры градиента. Все строки **Градиента остановить** за пределы 10 игнорируются. 
  
Заливки градиента свойства задаются в области **Формат фигуры** (на вкладке **Главная** в группе **Стили фигур** щелкните **заливки**и выберите пункт **Параметры заливки**). 
  
Можно включить или отключить градиентной заливки фигуры, при изменении значения ячейки [FillGradientEnabled ячейки (градиента Properties Section)](fillgradientenabled-cell-gradient-properties-section.md) .
