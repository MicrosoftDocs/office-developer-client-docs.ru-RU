# <a name="rewiden-function"></a>Функция REWIDEN

Преобразует формулу, которая создала коды 16-разрядный символов, которые являются расширенными однобайтовые или многобайтовые кодов набор символов в строку коды знаков Юникод 16-разрядный, с помощью указанных наборов символов. 
  
## <a name="syntax"></a>Синтаксис

REWIDEN (** *srcCharSet* **, ** *dstCharSet* **, ** *текст* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _srcCharSet_ <br/> |Обязательный  <br/> |**Строка** <br/> |Набор символов в исходном документе.  <br/> |
| _dstCharSet_ <br/> |Обязательный  <br/> |**Строка** <br/> | Набор символов в целевой документ.  <br/> |
| _text_ <br/> |Обязательный  <br/> |**Строка** <br/> |Текст для преобразования.  <br/> |
   
## <a name="remarks"></a>Замечания

Функция REWIDEN используется в автоматическое преобразование документов Visio 2002 в документы Visio 2003. Не рекомендуется использовать другие.
