# <a name="rewiden-function"></a>Функция REWIDEN

Преобразует формулу, которая создает 16-разрядные коды символов, которые расширяются в однобайтовом или многобайтовом коде набора символов, в строку 16-разрядных кодов символов Юникода с использованием указанных кодов символов. 
  
## <a name="syntax"></a>Синтаксис

REWIDEN(** *srcCharSet* **, ** *dstCharSet* **, ** *text* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _srcCharSet_ <br/> |Обязательно  <br/> |**Строка** <br/> |Набор символов в документе источника.  <br/> |
| _dstCharSet_ <br/> |Обязательно  <br/> |**Строка** <br/> | Набор символов в документе назначения.  <br/> |
| _text_ <br/> |Обязательно  <br/> |**Строка** <br/> |Текст для преобразования.  <br/> |
   
## <a name="remarks"></a>Примечания

Функция REWIDEN используется при автоматическом преобразовании документов Visio 2002 в документы Visio 2003. Другое использование не рекомендуется.
