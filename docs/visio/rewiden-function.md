# <a name="rewiden-function"></a>Функция REWIDEN

Преобразует формулу, которая создает 16 – битовые коды символов, которые являются расширенными кодами символов с одним или более многобайтовым кодом, в строку 16 – разрядных символов Юникода с использованием указанных наборов символов. 
  
## <a name="syntax"></a>Синтаксис

ReWIDEN (* * *сркчарсет* * *, * * *дстчарсет* * *, * * *Text* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Сркчарсет_ <br/> |Обязательный  <br/> |**String** <br/> |Набор символов в исходном документе.  <br/> |
| _Дстчарсет_ <br/> |Обязательный  <br/> |**String** <br/> | Набор символов в целевом документе.  <br/> |
| _text_ <br/> |Обязательный  <br/> |**String** <br/> |Преобразуемый текст.  <br/> |
   
## <a name="remarks"></a>Замечания

Функция reWIDEN используется для автоматического преобразования документов Visio 2002 в документы Visio 2003. Использовать другие рекомендации не рекомендуется.
