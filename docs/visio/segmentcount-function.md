# <a name="segmentcount-function"></a>Функция SEGMENTCOUNT

Возвращает количество сегментов строк, которые составляют путь.
  
## <a name="syntax"></a>Синтаксис

SEGMENTCOUNT(** *pathRef* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _pathRef_ <br/> |Обязательный  <br/> |**Integer** <br/> |Раздел Геометрия, который представляет путь, заданный ссылкой на ячейку Path (например, Geometry1.Path).  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Целое число
  
## <a name="remarks"></a>Замечания

Прыжки строки не включаются в общее число возвращаемого сегмента.
