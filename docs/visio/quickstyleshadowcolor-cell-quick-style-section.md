# <a name="quickstyleshadowcolor-cell-quick-style-section"></a>Ячейка QuickStyleShadowColor (экспресс-стилей раздел)

Определяет, какой цвет темы, которая использует тени фигуры, как целое число от 0 до 7.
  
|||
|:-----|:-----|
|Значение  <br/> |Описание  <br/> |
|0  <br/> |Цвет затенения фигуры наследуется от темы темный цвет.  <br/> |
|1  <br/> |Цвет затенения фигуры наследуется от цвет быстрое темы.  <br/> |
|2  <br/> |Цвет затенения фигуры наследуется от 1 контрастный цвет темы.  <br/> |
|3  <br/> |Цвет затенения фигуры наследуется от 2 контрастный цвет темы.  <br/> |
|4  <br/> |Цвет затенения фигуры наследуется от 3 контрастный цвет темы.  <br/> |
|5  <br/> |Цвет затенения фигуры наследуется от 4 контрастный цвет темы.  <br/> |
|6  <br/> |Цвет затенения фигуры наследуется от 5 контрастный цвет темы.  <br/> |
|7  <br/> |Цвет затенения фигуры наследуется от 6 контрастный цвет темы.  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **QuickStyleShadowColor** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | QuickStyleShadowColor  <br/> |
   
Для получения ссылки на ячейки **QuickStyleShadowColor** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowQuickStyleProperties** <br/> |
| Индекс ячейки:  <br/> |**visQuickStyleShadowColor** <br/> |
