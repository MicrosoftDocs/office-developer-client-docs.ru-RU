# <a name="bevellightingangle-cell-bevel-properties-section"></a>Ячейка BevelLightingAngle (раздел "Свойства багетной рамки")

Определяет угол изображением при использовании рельефов в градусов. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **BevelLightingAngle** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | BevelLightingAngle  <br/> |
   
Для получения ссылки на ячейки **BevelLightingAngle** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowBevelProperties** <br/> |
| Индекс ячейки:  <br/> |**visBevelLightingAngle** <br/> |
