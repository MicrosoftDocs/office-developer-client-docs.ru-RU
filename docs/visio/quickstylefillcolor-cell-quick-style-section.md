# <a name="quickstylefillcolor-cell-quick-style-section"></a>QuickStyleFillColor Cell (Quick Style Section)

Определяет цвет темы, который использует заполняемая фигура, в виде integer от 0 до 7
  
|||
|:-----|:-----|
|Значение  <br/> |Описание  <br/> |
|0  <br/> |Цвет заполнения формы наследуется от цвета темной темы.  <br/> |
|1  <br/> |Цвет заполнения формы наследуется от цвета темы Light.  <br/> |
|2  <br/> |Цвет заполнения формы наследуется от цвета темы Accent 1  <br/> |
|3  <br/> |Цвет заполнения формы наследуется от цвета темы Accent 2  <br/> |
|4   <br/> |Цвет заполнения формы наследуется от цвета темы Accent 3  <br/> |
|5   <br/> |Цвет заполнения формы наследуется от цвета темы Accent 4  <br/> |
|6   <br/> |Цвет заполнения формы наследуется от цвета темы Accent 5  <br/> |
|7   <br/> |Цвет заполнения формы наследуется от цвета темы Accent 6  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на **ячейку QuickStyleFillColor** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | QuickStyleFillColor  <br/> |
   
Чтобы получить ссылку на **ячейку QuickStyleFillColor** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowQuickStyleProperties** <br/> |
| Индекс ячейки:  <br/> |**visQuickStyleFillColor** <br/> |
