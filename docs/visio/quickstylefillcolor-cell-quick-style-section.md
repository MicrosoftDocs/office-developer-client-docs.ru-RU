# <a name="quickstylefillcolor-cell-quick-style-section"></a>Ячейка QuickStyleFillColor (раздел "Экспресс-стиль")

Определяет, какой цвет темы, использующий заливки фигуры в качестве целое число от 0 до 7
  
|||
|:-----|:-----|
|Значение  <br/> |Описание  <br/> |
|0  <br/> |Цвета заливки фигуры наследуется от темы темный цвет.  <br/> |
|1  <br/> |Цвета заливки фигуры наследуется от цвет быстрое темы.  <br/> |
|2  <br/> |Цвета заливки фигуры наследует от 1 контрастный цвет темы  <br/> |
|3  <br/> |Цвета заливки фигуры наследует от 2 контрастный цвет темы  <br/> |
|4  <br/> |Цвета заливки фигуры наследует от 3 контрастный цвет темы  <br/> |
|5  <br/> |Цвета заливки фигуры наследует от 4 контрастный цвет темы  <br/> |
|6  <br/> |Цвета заливки фигуры наследует от 5 контрастный цвет темы  <br/> |
|7  <br/> |Цвета заливки фигуры наследует от 6 контрастный цвет темы  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **QuickStyleFillColor** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | QuickStyleFillColor  <br/> |
   
Для получения ссылки на ячейки **QuickStyleFillColor** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowQuickStyleProperties** <br/> |
| Индекс ячейки:  <br/> |**visQuickStyleFillColor** <br/> |
