# <a name="linetoliney-cell-page-layout-section"></a>Ячейка LineToLineY (раздел "Макет страницы")

Определяет вертикали свободное пространство между все соединители на странице документа.
  
## <a name="remarks"></a>Замечания

Также можно задать значение данной ячейки в диалоговом окне **Макет и интервал по маршрутизации** . (На вкладке **Конструктор** щелкните стрелку **Параметры страницы** , нажмите кнопку **Макет и маршрутизации**и нажмите кнопку **интервалы**.)
  
Для получения ссылки на ячейки LineToLineY по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |LineToLineY  <br/> |
   
Для получения ссылки на ячейки LineToLineY по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPageLayout** <br/> |
|Индекс ячейки:  <br/> |**visPLOLineToLineY** <br/> |
