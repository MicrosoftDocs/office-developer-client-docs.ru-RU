# <a name="blocksizey-cell-page-layout-section"></a>Ячейка BlockSizeY (раздел "Макет страницы")

Определяет размер вертикального блока должна помещаться область, в которой каждый фигур на странице документа при расположении фигур с помощью диалогового окна " **Настройка макета** " (на вкладке **Конструктор** в группе **Макет** выберите **Изменить расположение** и нажмите кнопку **Дополнительные параметры расположения**).
  
## <a name="remarks"></a>Замечания

Также можно задать это значение в диалоговом окне **Макет и маршрутизация интервалы** (на вкладку **Конструктор** , щелкните стрелку в группе **Параметры страницы** , перейдите на вкладку **Макет и маршрутизации** и нажмите кнопку **интервала**).
  
Чтобы получить ссылку на ячейку BlockSizeY по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | BlockSizeY  <br/> |
   
Для получения ссылки на ячейки BlockSizeY по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowPageLayout** <br/> |
| Индекс ячейки:  <br/> |**visPLOBlockSizeY** <br/> |
