# <a name="upper-function"></a>Функция UPPER

Возвращает строку, преобразованную в верхний шкаф.
  
## <a name="syntax"></a>Синтаксис

UPPER(** *expression* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**Разные** <br/> | Строка, ссылка ячейки или выражение; результат преобразуется в строку, которая затем преобразуется в верхний шкаф.  <br/> |
   
## <a name="remarks"></a>Примечания

Преобразование дела зависит от конкретного локального параметра, основанного на текущих параметрах пользователя. 
  
## <a name="example"></a>Пример

UPPER ("mIxEd CAse") 
  
Возвращает "СМЕШАННЫЙ СЛУЧАЙ".
