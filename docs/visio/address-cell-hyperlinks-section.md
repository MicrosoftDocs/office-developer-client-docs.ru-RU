# <a name="address-cell-hyperlinks-section"></a>Address Cell (Hyperlinks Section)

Задает URL-адрес, имя файла или UNC-путь для перехода.
  
Вы можете указать адрес как относительный путь в зависимости от базового пути, определенного для документа в поле **база** данных на вкладке **Сводка** диалогового окна **Свойства** (перейдите на вкладку **файл** , щелкните **сведения**, выберите * * свойства * *, а затем щелкните **Дополнительные свойства**). Если в документе нет базового пути, приложение переходит на основе пути документа. Если документ не был сохранен, гиперссылка не определена.
  
## <a name="remarks"></a>Примечания

Кроме того, можно задать значение ячейки адреса в диалоговом окне **гиперссылки** (щелкните **Гиперссылка** на вкладке **Вставка** ). 
  
Чтобы получить ссылку на ячейку Address по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Гиперссылки. *Name (имя* ). Адрес гиперссылки. *Name* — имя строки гиперссылки  <br/> |
   
Чтобы получить ссылку на ячейку Address по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**виссектионхиперлинк** <br/> |
| Индекс строки:  <br/> |**visRow1stHyperlink** +  *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**вишлинкаддресс** <br/> |
