# <a name="address-cell-hyperlinks-section"></a>Address Cell (Hyperlinks Section)

Указывает URL-адрес, имя файла или путь UNC, к которому необходимо перейти.
  
Адрес можно указать как относительный путь на основе базового  пути, определенного  для документа в базовом окне Гиперссылка, на вкладке Сводка диалогового окна **Свойства** (щелкните вкладку **Файл,** щелкните **Info,** щелкните ** Свойства **, а затем нажмите **кнопку Расширенные свойства**). Если в документе нет базового пути, приложение перемещается по пути документа. Если документ не сохранен, гиперссылка не установлена.
  
## <a name="remarks"></a>Примечания

Вы также можете установить значение ячейки Address в диалоговом окне **Гиперссылки** (щелкните **Гиперссылку** на вкладке **Вставка).** 
  
Чтобы получить ссылку на ячейку Address по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Гиперссылка. *имя*  . Адрес, где гиперссылка. *имя*  строки гиперссылки  <br/> |
   
Чтобы получить ссылку на ячейку Address по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionHyperlink** <br/> |
| Индекс строки:  <br/> |**visRow1stHyperlink**  +   *i,* *где i* = 0, 1, 2 ...  <br/> |
| Индекс ячейки:  <br/> |**visHLinkAddress** <br/> |
