# <a name="address-cell-hyperlinks-section"></a>Address Cell (Hyperlinks Section)

Указывает URL-адрес, имя файла или UNC-путь, к которому необходимо перейти.
  
Вы можете указать адрес в качестве относительного пути на  основе базового пути, определенного  для документа, в базовом поле гиперссылки на вкладке "Сводка" диалогового окна "Свойства" (щелкните вкладку "Файл", щелкните "Сведения", "** "Свойства" **, а затем "Дополнительные  свойства").    Если в документе нет базового пути, приложение переходит по этому пути. Если документ не был сохранен, гиперссылка не будет запределена.
  
## <a name="remarks"></a>Примечания

Вы также можете установить значение ячейки Address в диалоговом  окне **"Гиперссылки"** (щелкните "Гиперссылка" на вкладке **"Вставка").** 
  
Чтобы получить ссылку на ячейку Address по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Гиперссылка. *name*  . Адрес, где находится гиперссылка. *имя*  — имя строки гиперссылки  <br/> |
   
Чтобы получить ссылку на ячейку Address по имени из другой формулы или из программы, используя свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionHyperlink** <br/> |
| Индекс строки:  <br/> |**visRow1stHyperlink**  +   *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visHLinkAddress** <br/> |
