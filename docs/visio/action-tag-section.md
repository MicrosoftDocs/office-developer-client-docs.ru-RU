# <a name="action-tag-section"></a>Action Tag Section

Содержит строку для каждого тега действия, определенного для фигуры или страницы.
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
## <a name="remarks"></a>Замечания

Раздел **"Тег действия"** управляет внешним видом и положением тегов действий в рисунке. С помощью **ячейки TagName** можно связать каждый тег действия со строкой или строками в разделе "Действия", где определяются элементы меню, которые отображаются в меню тегов действий. 
  
 Одна фигура или страница может содержать несколько тегов действий. 
  
Если тег действия не имеет совпадающих действий, появляется и исчезает кнопка тега действия, как определено, но его меню пустое.
  
В окне  "Таблица фигур" можно вставить только раздел "Тег действия" (щелкните правой кнопкой мыши в окне "Таблица фигур", выберите "Вставить раздел" и выберите "Теги действий").  Чтобы добавить тег действия  в существующий раздел тега действий, щелкните правой кнопкой мыши строку и выберите пункт **"Вставка** строки" в ярлыковом меню.
