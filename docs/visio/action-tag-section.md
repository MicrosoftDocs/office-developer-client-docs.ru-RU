# <a name="action-tag-section"></a>Action Tag Section

Содержит строку для каждого тега действия, определенного для формы или страницы.
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
## <a name="remarks"></a>Замечания

Раздел **Тег действия** контролирует внешний вид и положение тегов действий в рисунке. С помощью **ячейки TagName** можно связать каждый тег действия со строкой или строками в разделе Действия, где определяются элементы меню, которые отображаются в меню тегов действий. 
  
 Одна фигура или страница может содержать несколько тегов действий. 
  
Если тег действия не имеет совпадающих действий, кнопка тега действий отображается и исчезает по определению, но ее меню пусто.
  
В окно ShapeSheet можно вставить только раздел Тег действий (щелкните правой кнопкой мыши в окне ShapeSheet, нажмите кнопку **Вставить** раздел , а затем выберите поле Теги **действия).**  Чтобы добавить тег действий в существующий раздел **Тег** действий, щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить** строку в меню ярлыка.
