# <a name="action-tag-section"></a>Раздел "Теги действий"

Содержит строку для каждого тега действия, определенного для фигуры или страницы.
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий, называются смарт-тегов. 
  
## <a name="remarks"></a>Замечания

В разделе **Тег действия** определяет внешний вид и положение теги действий в документе. Ячейка **TagName** можно связать тег каждого действия с строку или строки в разделе действия, где определены пунктов меню, которые отображаются в меню Действие тега. 
  
 Один фигуры или страницы может содержать несколько тегов действие. 
  
Если тег действие не имеет соответствующего действий, кнопки тега действия появляется и исчезает, как определено, но его меню не задан.
  
Раздел **Тег действия** можно добавить только в окне таблицы свойств фигуры (щелкните правой кнопкой мыши в окне таблицы свойств фигуры, нажмите кнопку **Вставить раздел**и установите флажок **теги действий** ). Чтобы добавить тег действие в существующий раздел **Тег действия** , щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить строку** в контекстном меню.
