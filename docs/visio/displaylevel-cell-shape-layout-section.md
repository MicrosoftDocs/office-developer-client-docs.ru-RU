# <a name="displaylevel-cell-shape-layout-section"></a>Ячейка DisplayLevel (раздел "Макет фигуры")

Определяет диапазон уровня отображения (относительное диапазон из Z-порядка группировки) для фигуры.
  
## <a name="remarks"></a>Замечания

Z-порядка является порядок отображения для фигур на странице документа. Фигура, находящегося выше в Z-порядке отображается перед фигуры нижнего уровня в Z-порядке, когда один из другой перекрытий фигур. 
  
Разделяет уровня отображения с фигурами в группы или полосами. Все фигуры в полосе заданного иметь выше Z-порядка чем фигур в нижней панели. По умолчанию большинство фигур имеют уровень отображения нуль (0).
  
Диапазон отображаемых уровней — от-32,767 +32,767. Фигуры, которые имеют тот же уровень отображения объединяются в одном регулирования, в котором они также ранжируются относительно друг с другом по оси z.
  
Z-порядка фигур в полосе можно изменить с помощью команды **Переместить вперед**, **Переместить назад**, **на передний план**и **на задний план**. Если эти команды Переместить фигуру из заданного расположения, Microsoft Visio значение зарезервировано -32768 в ячейке DisplayLevel фигуры, если не защищены ячейки. В этом случае фигуры не могут перемещаться в другом и Visio отображается предупреждение «защиты и/или уровень со свойствами не удается завершить выполнение этой команды». 
  
Для получения ссылки на ячейку DisplayLevel по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте следующее. 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |DisplayLevel  <br/> |
   
Для получения ссылки на ячейки DisplayLevel по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowShapeLayout** <br/> |
|Индекс ячейки:  <br/> |**visSLODisplayLevel** <br/> |
