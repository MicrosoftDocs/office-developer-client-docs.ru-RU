# <a name="shape-data-section"></a>Shape Data Section

Содержит по одной строке для каждого элемента данных фигуры и ячеек для связывания данных с фигурой.
  
## <a name="remarks"></a>Примечания

Данные фигуры можно определить с помощью окна " **данные фигуры** " или ячеек в разделе "данные фигуры". 
  
> [!NOTE]
> В предыдущих версиях Visio раздел " **данные фигуры** " назывался **пользовательскими свойствами**. 
  
- В окне таблицы свойств фигуры можно определить данные фигуры в разделе **данные фигуры** . Щелкните правой кнопкой мыши в окне таблицы свойств фигуры и выберите команду **Вставить раздел**, а затем установите флажок **данные фигуры** . Можно добавить любое количество свойств prop.  Укажите *имена строк,* назначьте им значимые имена и задайте значения ячеек. 
    
- В окне рисунок можно определить данные фигуры в окне **данные фигуры** . На вкладке **вид** в группе **Показать/скрыть** щелкните **области задач**, а затем щелкните **данные фигуры**. Чтобы открыть диалоговое окно **Определение данных фигуры** , щелкните правой кнопкой мыши окно **данные фигуры** , а затем выберите команду **Определение данных фигуры**.
