# <a name="shape-data-section"></a>Shape Data Section

Содержит одну строку для каждого элемента данных фигуры и ячеек для связи данных с фигурой.
  
## <a name="remarks"></a>Примечания

Данные фигуры можно определить с помощью окна **"Данные** фигуры" или ячеек в разделе "Данные фигуры". 
  
> [!NOTE]
> В предыдущих версиях Visio раздел **"Данные** фигуры" назывался **"Настраиваемые свойства".** 
  
- В окне ShapeSheet можно определить данные фигуры в разделе **"Данные** фигуры". Щелкните правой кнопкой мыши в окне таблицы фигур и выберите **"Вставить** раздел", а затем нажмите кнопку **"Данные** фигуры". Можно добавить максимальное количество реквизитов.  *назначать*  строкам имена по мере необходимости, назначать осмысленные имена строкам и задавать значения ячеей. 
    
- В окне рисования можно определить данные фигуры в **окне "Данные** фигуры". На **вкладке "Вид"** в группе **"Показать/скрыть"** щелкните **"Области** задач" и выберите **"Данные фигуры".** Чтобы получить доступ **к диалоговом окну** "Определение данных фигуры", щелкните правой кнопкой мыши окно **"Данные** фигуры" и выберите **"Определить данные фигуры".**
