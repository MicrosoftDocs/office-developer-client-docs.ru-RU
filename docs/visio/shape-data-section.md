# <a name="shape-data-section"></a>Раздел "Данные фигуры"

Содержит одну строку для каждого элемента данных фигуры и ячейки для связывания данных с фигурой.
  
## <a name="remarks"></a>Замечания

Данные фигуры может быть определен с использованием окна **Данных фигуры** и ячейки в разделе данные фигуры. 
  
> [!NOTE]
> В предыдущих версиях Visio в разделе **Данных фигуры** вызван **Настраиваемые свойства**. 
  
- В окне таблицы свойств фигуры можно определить данные фигуры в разделе **Данных фигуры** . Щелкните правой кнопкой мыши в окне таблицы свойств фигуры и нажмите кнопку **Вставить раздел**и установите флажок **Данные фигуры** . Можно добавить столько свойства.  *имя* строки при необходимости назначить понятные имена строк и задайте значения ячейки. 
    
- В окне документа можно определить данные фигуры в окне **Данных фигуры** . На вкладке **Вид** в группе **Показать/скрыть** нажмите кнопку **Панели задач**и нажмите кнопку **Данные фигуры**. Чтобы открыть диалоговое окно " **Определение данных фигуры** ", щелкните правой кнопкой мыши окно **Данных фигуры** и нажмите кнопку **Определение данных фигуры**.
