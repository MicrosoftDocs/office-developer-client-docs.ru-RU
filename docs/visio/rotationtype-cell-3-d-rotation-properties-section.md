# <a name="rotationtype-cell-3-d-rotation-properties-section"></a>Ячейка RotationType (раздел "Свойства поворота объемной фигуры")

Определяет, следует ли фигуры параллельный цикл, перспективы поворот или наклон вращение, как целое число от 0 до 6. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|0  <br/> |Фигура не имеет любое вращение.  <br/> |
|1  <br/> |Фигура имеет параллельный цикл.  <br/> |
|2  <br/> |Фигура имеет вращение Перспектива.  <br/> |
|3  <br/> |Фигура имеет верхней левой наклон вращение.  <br/> |
|4  <br/> |Фигура имеет верхней прямо наклон вращение.  <br/> |
|5  <br/> |Фигура имеет наклонный вращение нижней левой.  <br/> |
|6  <br/> |Фигура имеет нижней правой кнопкой наклон вращение.  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **RotationType** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |RotationType  <br/> |
   
Для получения ссылки на ячейки **RotationType** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRow3DRotationProperties** <br/> |
|Индекс ячейки:  <br/> |**visRotationType** <br/> |
