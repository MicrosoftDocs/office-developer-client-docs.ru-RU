# <a name="find-function"></a>Функция FIND

Находит одну текстовую строку, содержаную в другой строке текста, и возвращает исходное положение текстовой строки, которую вы ищете относительно ее положения в текстовой строке, содержащем ее.
  
## <a name="syntax"></a>Синтаксис

FIND (** *find_text* **, **** within_text **,[ ** *start_num* ** ], [* *ignore_case* ** ]] 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _find_text_ <br/> |Обязательный  <br/> |**String** <br/> |Текстовая строка, которая вам нужна.  <br/> |
| _format_ <br/> |Обязательный  <br/> |**String** <br/> |Текстовая строка, содержаная текст, который необходимо найти.  <br/> |
| _start_num_ <br/> |Необязательный  <br/> |**Number** <br/> |Символ, с которого начинается поиск. Первый символ в  _within_text_ — 1. Если  _start_num_ отсутствует, предполагается, что это будет 1.  <br/> |
| _ignore_case_ <br/> |Необязательный  <br/> |**Логический** <br/> |По умолчанию функция FIND является чувствительной к делу. Если вы хотите, чтобы функция FIND проигнорировала случай, установите этот аргумент true.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Номер
  
## <a name="remarks"></a>Примечания

Если найдено несколько совпадений, функция FIND возвращает исходное положение первого совпадения в строке. Аргумент  _find_text_ не считает какие-либо символы подкардами. 
  
Если _find_text:_
  
-  Является пустым (""), FIND соответствует первому символу строки поиска  _(т._ е. символу с номером start_num или 1). 
    
- Не появляется в  _within_text_, FIND возвращает #VALUE! значение ошибки. 
    
Если _start_num:_
  
- Не больше нуля (0), FIND возвращает #VALUE! значение ошибки. 
    
- Больше, чем длина  _within_text,_ FINDreturns #VALUE! значение ошибки. 
    
## <a name="example"></a>Пример

FIND ("2003", "1 января 2003 г.") 
  
Возвращает 12.
