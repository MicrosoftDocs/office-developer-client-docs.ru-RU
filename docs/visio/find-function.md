# <a name="find-function"></a>Функция FIND

Находит одну текстовую строку, содержащуюся в другой текстовой строке, и возвращает начальную позицию искомой текстовой строки относительно ее позиции в текстовой строке, содержащей ее.
  
## <a name="syntax"></a>Синтаксис

Find (* * *искомый_текст* * *, * * *просматриваемый_текст* * *, [* * *Нач_позиция* * *], [* * *игноре_касе* * *]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _искомый_текст_ <br/> |Обязательный  <br/> |**String** <br/> |Текстовая строка, которую нужно найти.  <br/> |
| _format_ <br/> |Обязательный  <br/> |**String** <br/> |Текстовая строка, содержащая текст, который требуется найти.  <br/> |
| _равн_ <br/> |Необязательный  <br/> |**Number** <br/> |Символ, с которого начинается поиск. Первый символ в параметре _текст_для_поиска_ равен 1. Если _аргумент нач_позиция_ отсутствует, предполагается, что он равен 1.  <br/> |
| _игноре_касе_ <br/> |Необязательный  <br/> |**Логический** <br/> |По умолчанию функция поиска учитывает регистр. Если вы хотите, чтобы функция поиска игнорировала регистр, установите для этого аргумента значение TRUE.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Число
  
## <a name="remarks"></a>Примечания

Если найдено несколько совпадений, функция найти Возвращает начальную позицию первого совпадения в строке. Аргумент _искомый_текст_ не учитывает символы с подстановочными знаками. 
  
Если _искомый_текст_:
  
-  Является пустым (""), то функция найти соответствует первому символу в строке поиска (то есть символу номер _Нач_позиция_ или 1). 
    
- Не отображается в _просматриваемый_текст_, то FIND возвращает #VALUE! значение ошибки. 
    
Если _Нач_позиция_:
  
- Не больше нуля (0), то FIND возвращает #VALUE! значение ошибки. 
    
- Больше, чем длина _просматриваемый_текст_, финдретурнс #VALUE! значение ошибки. 
    
## <a name="example"></a>Пример

FIND ("2003", "1 января, 2003") 
  
Возвращает 12.
