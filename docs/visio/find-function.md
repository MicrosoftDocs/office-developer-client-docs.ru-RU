# <a name="find-function"></a>Функция FIND

Находит одну текстовую строку, содержащемся в другой текстовой строке, и возвращает начальное положение текстовой строки, которую вы ищете относительно ее позиции в текстовой строке, содержащем ее.
  
## <a name="syntax"></a>Синтаксис

FIND (** *find_text* **, ** *within_text* **,[ ** *start_num* ** ], [ ** *ignore_case* ** ]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _find_text_ <br/> |Обязательно  <br/> |**Строка** <br/> |Текстовая строка, которая вам нужна.  <br/> |
| _format_ <br/> |Обязательно  <br/> |**Строка** <br/> |Текстовая строка, содержаная текст, который вы хотите найти.  <br/> |
| _start_num_ <br/> |Необязательна  <br/> |**Number** <br/> |Символ, с которого начинается поиск. Первый символ  _в_ within_text 1. Если  _start_num_ отсутствует, предполагается, что имеется 1.  <br/> |
| _ignore_case_ <br/> |Необязательный  <br/> |**Логический** <br/> |По умолчанию функция FIND чувствительна к делу. Если вы хотите, чтобы функция FIND проигнорировала дело, установите для этого аргумента true.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="remarks"></a>Примечания

Если найдено несколько совпадений, функция FIND возвращает точки начала первого совпадения в строке. Аргумент  _find_text_ не считает какие-либо символы поддиаными знаками. 
  
Если _find_text:_
  
-  Пуст (""), FIND соответствует первому символу в строке поиска (то есть символу с номером  _start_num_ или 1). 
    
- Не появляется  _в_ within_text , FIND возвращает #VALUE! значение ошибки. 
    
Если _start_num:_
  
- Не больше нуля (0), FIND возвращает #VALUE! значение ошибки. 
    
- Больше, чем длина within_text  _,_ FINDreturns #VALUE! значение ошибки. 
    
## <a name="example"></a>Пример

FIND ("2003","January 1, 2003") 
  
Возвращает 12.
