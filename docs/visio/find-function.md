# <a name="find-function"></a>Функция FIND

Находит одной строки текста, содержащихся в другую строку текста и возвращает положение начала текстовой строки, вы ищете относительно его положения текстовой строки, содержащей его.
  
## <a name="syntax"></a>Синтаксис

Поиск (** *искомого* **, ** *левого края текста* **, [** *Начальная позиция* **], [** *ignore_case* **]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _искомого_ <br/> |Обязательный  <br/> |**Строка** <br/> |Строка текста, который требуется найти.  <br/> |
| _format_ <br/> |Обязательный  <br/> |**Строка** <br/> |Текстовая строка, содержащая текст, который требуется найти.  <br/> |
| _Начальная позиция_ <br/> |Optional  <br/> |**Число** <br/> |Символ, с которого следует начать поиск. Первого знака _Укажите_ -1. Если _Начальная позиция_ отсутствует, то предполагается, что оно является 1.  <br/> |
| _ignore_case_ <br/> |Optional  <br/> |**Boolean** <br/> |По умолчанию функция поиска с учетом регистра. Функции поиска, следует ли игнорировать регистр, установите этот аргумент значение TRUE.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Number
  
## <a name="remarks"></a>Замечания

Если несколько совпадений, функция найти возвращает позицию первого соответствия в строке. Аргумент _искомого_ необходимо учитывать символы быть подстановочные знаки. 
  
Если _искомого_:
  
-  Пустое ("»), найти совпадает с первым символом в строке поиска (то есть, знак с номером _Начальная позиция_ или 1). 
    
- Не отображается в _левого края текста_, FIND возвращает #VALUE! значение ошибки. 
    
Если _Начальная позиция_:
  
- Не больше, чем нуль (0), FIND возвращает #VALUE! значение ошибки. 
    
- Больше, чем длина _левого края текста_, FINDreturns #VALUE! значение ошибки. 
    
## <a name="example"></a>Пример

Поиск («2003», «1 января 2003») 
  
Возвращает 12.
