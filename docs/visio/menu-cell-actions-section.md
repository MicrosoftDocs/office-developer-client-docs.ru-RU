# <a name="menu-cell-actions-section"></a>Menu Cell (Actions Section)

Определяет имя элемента меню, который отображается в меню ярлыка или тега действий для фигуры или страницы. 
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
## <a name="remarks"></a>Замечания

Чтобы вставить сепаратор в меню над этим элементом, используйте ячейку BeginGroup. Чтобы отобразить команду в нижней части меню, применим имя с процентным характером (%).
  
Чтобы получить ссылку на ячейку Меню по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Действия. *имя*  . Действия menuwhere.  *имя*  — это имя строки Действия  <br/> |
   
Чтобы получить ссылку на ячейку Меню по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionAction** <br/> |
|Индекс строки:  <br/> |**visRowAction**  +   *i,* где i = 0, 1, 2, ...  <br/> |
|Индекс ячейки:  <br/> |**visActionMenu** <br/> |
