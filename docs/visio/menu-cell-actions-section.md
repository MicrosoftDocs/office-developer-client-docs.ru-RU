# <a name="menu-cell-actions-section"></a>Menu Cell (Actions Section)

Определяет имя элемента меню, которое отображается в контекстном меню или меню тегов действий для фигуры или страницы. 
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
## <a name="remarks"></a>Замечания

Чтобы вставить разделитель в меню над элементом, используйте ячейку BeginGroup. Чтобы отобразить команду в нижней части меню, введите перед именем символ процента (%).
  
Чтобы получить ссылку на ячейку меню по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Действия. *Name (имя* ). Действия Менувхере.  *Name* — имя строки действий  <br/> |
   
Чтобы получить ссылку на ячейку меню по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**Виссектионактион** <br/> |
|Индекс строки:  <br/> |**висровактион** +  *i* , где i = 0, 1, 2,...  <br/> |
|Индекс ячейки:  <br/> |**Висактионмену** <br/> |
