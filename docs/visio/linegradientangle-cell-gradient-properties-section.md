# <a name="linegradientangle-cell-gradient-properties-section"></a>Ячейка LineGradientAngle (раздел "Свойства градиента")

Определяет угол градиентной строки для линейного градиента от 0 до 359.9 градусов.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **LineGradientAngle** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | LineGradientAngle  <br/> |
   
Для получения ссылки на ячейки **LineGradientAngle** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowGradientProperties** <br/> |
| Индекс ячейки:  <br/> |**visLineGradientAngle** <br/> |
