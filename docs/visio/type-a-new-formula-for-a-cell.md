# <a name="type-a-new-formula-for-a-cell"></a>Введите новую формулу для ячейки

1. Выберите ячейку.
    
    Формула ячейки отображается в ячейке и в панели формул.
    
2. Введите новую формулу.
    
    Введите формулу, которая заменяет предыдущую формулу ячейки.
    
    Если вы введите номер, но не поставляете единицы измерения, microsoft Visio использует единицы из предыдущего значения ячейки, если она была. В противном случае, если ячейке требуются единицы, Visio для ячейки используются единицы по умолчанию.
    
3. Чтобы принять формулу, нажмите кнопку ENTER.
    
    Если номер или формула содержит ошибку, Visio отображает сообщение, а затем выделяет ошибку в формуле или ячейке. Исправление ошибки и нажмите КНОПКУ ENTER, чтобы принять исправление.
