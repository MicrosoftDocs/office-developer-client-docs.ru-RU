# <a name="type-a-new-formula-for-a-cell"></a>Введите новую формулу для ячейки

1. Выберите ячейку.
    
    Формула ячейки отображается в ячейке и на панели формул.
    
2. Введите новую формулу.
    
    Введите формулу, которая заменяет предыдущую формулу ячейки.
    
    Если вы введите число, но не укалит единицы измерения, Microsoft Visio использует единицы из предыдущего значения ячейки, если таковой был. В противном случае, если ячейке требуются единицы, Visio использует для нее единицы по умолчанию.
    
3. Чтобы принять формулу, нажмите ввод.
    
    Если число или формула содержат ошибку, Visio отображает сообщение, а затем выделяет ошибку в панели формул или ячейке. Исправим ошибку и нажмите ввод, чтобы принять исправление.
