# <a name="type-a-new-formula-for-a-cell"></a>Введите новую формулу для ячейки

1. Выберите ячейку.
    
    Формула ячейки отображается в ячейке и в строке формул.
    
2. Введите новую формулу.
    
    Формула, которую вы введите заменяет предыдущие ячейка.
    
    Если ввести номер, но не указать единицы измерения, Microsoft Visio использует единиц измерения из предыдущего значения ячейки в случае отсутствия один. В противном случае если ячейки требуется единиц измерения, Visio использует единиц измерения по умолчанию для ячейки.
    
3. Чтобы принять формулу, нажмите клавишу ВВОД.
    
    Если номер или формула содержит ошибку, Visio выводится сообщение и затем основные сведения о ошибку в строке формул или ячейки. Исправьте ошибки и нажмите клавишу ВВОД, чтобы принять исправления.
