# <a name="type-a-new-formula-for-a-cell"></a>Ввод новой формулы для ячейки

1. Выберите ячейку.
    
    Формула ячейки отображается в ячейке и в строке формул.
    
2. Введите новую формулу.
    
    Введенная формула заменяет предыдущую формулу ячейки.
    
    Если вы вводите число, но не вводите единицы измерения, в Microsoft Visio используются единицы измерения из предыдущего значения ячейки, если таковая имеется. В противном случае, если ячейка требует единиц, Visio использует единицы измерения по умолчанию для ячейки.
    
3. Чтобы принять формулу, нажмите клавишу ВВОД.
    
    Если число или формула содержит ошибку, Visio выводит сообщение, а затем выделяет ошибку в строке формул или ячейке. ИсПравьте ошибку и нажмите клавишу ВВОД, чтобы принять исправление.
