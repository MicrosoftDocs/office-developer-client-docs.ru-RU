# <a name="shdwforegnd-cell-fill-format-section"></a>Ячейка ShdwForegnd (раздел "Формат заливки")

Определяет цвет, используемый для переднего плана (числу штрихов) узора заливки тени фигуры.
  
## <a name="remarks"></a>Замечания

Для настройки цвета, введите число от 0 до 23, которая является индексом в коллекции цветов.
  
Чтобы ввести другой цвет, используйте функцию RGB или HSL. Настраиваемые цвета значение цвета RGB, а RGB ( *r, g, b*), а не числа, будут отображаться в окне таблицы свойств фигуры. При использовании в операции, настраиваемые цвета имеют значения из 24 и выше. 
  
В ячейке ShdwForegndTrans можно задать прозрачность цвет переднего плана узора заливки тени фигуры.
  
Чтобы получить ссылку на ячейку ShdwForegnd по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | ShdwForegnd  <br/> |
   
Для получения ссылки на ячейки ShdwForegnd по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowFill** <br/> |
| Индекс ячейки:  <br/> |**visFillShdwForegnd** <br/> |
