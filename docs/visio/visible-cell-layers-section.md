# <a name="visible-cell-layers-section"></a>Visible Cell (Layers Section)

Указывает, видимы ли фигуры, принадлежащие слою, на странице документа.
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Фигуры видимы.  <br/> |
|FALSE  <br/> |Фигуры скрыты.  <br/> |
   
## <a name="remarks"></a>Примечания

Эта ячейка соответствует параметру **Visible** в диалоговом окне **Свойства слоя** (на вкладке **Главная** , в группе **Правка** щелкните **слои**, а затем выберите **Свойства слоя** ). 
  
Чтобы получить ссылку на видимую ячейку по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Слои. Visible [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на видимую ячейку по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**виссектионлайер** <br/> |
|Индекс строки:  <br/> |**висровлайер** +  *i* , где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**вислайервисибле** <br/> |
