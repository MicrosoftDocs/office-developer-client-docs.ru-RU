# <a name="run-in-developer-mode-display-the-developer-tab"></a>Запуск в режиме разработчика (отображение вкладки "Разработчик")

- Перейдите на вкладку **файл** и нажмите кнопку **Параметры**.
    
- В диалоговом окне **Параметры Visio** нажмите кнопку **Дополнительно**, а затем в разделе **Общие**выберите команду **запустить в режиме разработчика**.
    
Затем можно выбрать объект и нажать кнопку **Показать таблицу свойств** фигуры для быстрого доступа к окну таблица свойств фигуры.
