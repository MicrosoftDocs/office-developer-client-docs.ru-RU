# <a name="run-in-developer-mode-display-the-developer-tab"></a>Запуск в режиме разработчика (отображение вкладки "Разработчик")

- Перейдите на вкладку **файл** и нажмите кнопку **Параметры**.
    
- В диалоговом окне **Параметры Visio** нажмите кнопку **Дополнительно**и затем на вкладке **Общие**выберите **Запуск в режиме разработчика**.
    
Затем можно выберите объект и нажмите кнопку **Показать таблицу свойств фигуры** для быстрого доступа в окно таблицы свойств фигуры.
