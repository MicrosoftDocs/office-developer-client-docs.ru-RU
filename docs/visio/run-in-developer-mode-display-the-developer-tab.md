# <a name="run-in-developer-mode-display-the-developer-tab"></a>Запуск в режиме разработчика (Отображение вкладки разработчика)

- Щелкните **вкладку File** и нажмите кнопку **Параметры**.
    
- В **диалоговом окне Visio Параметры** нажмите **кнопку Advanced,** а затем в **general** выберите **Выполнить в режиме разработчика.**
    
Затем можно выбрать объект и щелкнуть **Show ShapeSheet** для более быстрого доступа к окну ShapeSheet.
