# <a name="run-in-developer-mode-display-the-developer-tab"></a>Запуск в режиме разработчика (вкладка "Отображение разработчика")

- Щелкните **вкладку "Файл"** и выберите **"Параметры".**
    
- В **диалоговом окне "Параметры Visio"** нажмите кнопку **"Дополнительные",** а затем в области "Общие" выберите "Выполнить **в режиме разработчика".** 
    
Затем можно выбрать объект и щелкнуть **"Показать таблицу фигур",** чтобы быстрее получить доступ к окну ShapeSheet.
