# <a name="cell-element-infiniteline-row-visio-xml"></a>Элемент Cell (строка InfiniteLine) (Visio XML)

Содержит координаты X или Y двух точек на бесконечной линии.
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|**Тип элемента** <br/> |[Cell_Type](cell_type-complextypevisio-xml.md) <br/> |
|**Пространство имен** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Файл схемы** <br/> |VisioSchema15.xsd  <br/> |
|**Части документа** <br/> |master#.xml, page#.xml  <br/> |
   
## <a name="definition"></a>Определение

```XML
< xs:element name="Cell" type="Cell_Type" minOccurs="0" maxOccurs="unbounded" >
</xs:element >
```

## <a name="remarks"></a>Примечания

Атрибут **N** этого элемента **Cell** должен быть одним из ограниченного набора значений, соответствующих ячейкам ShapeSheet. Чтобы определить значения атрибута **N,** допустимого для этого элемента Cell, обратитесь к таблице **ниже.** 
  
|**Значение**|**Описание**|**Дополнительные сведения**|
|:-----|:-----|:-----|
|X  <br/> |X-координата точки на бесконечной линии; сопряжено с координатой Y, представленной ячейкой Y.  <br/> |[InfiniteLine Row (Geometry Section)](infiniteline-row-geometry-section.md) <br/> |
|Да  <br/> |Y-координата точки на бесконечной линии; сопряжено с X-координатой, представленной ячейкой X.  <br/> |[InfiniteLine Row (Geometry Section)](infiniteline-row-geometry-section.md) <br/> |
|A  <br/> |X-координата точки на бесконечной линии; сопряжено с координатой Y, представленной ячейкой B.  <br/> |[InfiniteLine Row (Geometry Section)](infiniteline-row-geometry-section.md) <br/> |
|B  <br/> |Y-координата точки на бесконечной линии; в паре с X-координатой, представленной ячейкой A.  <br/> |[InfiniteLine Row (Geometry Section)](infiniteline-row-geometry-section.md) <br/> |
