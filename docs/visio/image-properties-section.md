# <a name="image-properties-section"></a>Раздел "Свойства изображения"

Содержит гамма, яркость, контраст, чтобы минимизировать, Резкость и Устранение шума значения для растровых изображений (изображения, состоящую из пикселов) и будет отображаться в окне таблицы свойств фигуры только для таких объектов.
