# <a name="annotation-section"></a>Раздел "Примечание"

Содержит сведения о комментарии, вставляется на страницу документа. 
  
> [!NOTE]
> В этом разделе используется для отслеживания комментариев только при открытии VSD-файл в Microsoft Visio 2013 или при сохранении файла vsdx (en) в формате .vsd. Он не используется для отслеживания комментариев в новые документы vsdx (en) в Visio 2013. 
  
## <a name="remarks"></a>Замечания

В разделе **комментариев** из таблицы свойств фигуры на странице содержит одну строку для каждого комментария, добавлены на страницу в документе, независимо от того, исходного документа страницы или страницы разметки перекрытия (страница, содержащая разметку, который был добавлен в документ рецензент). Сведения о разметки, которое было добавлено с помощью рукописного ввода или другие фигуры, а не комментариев, не включенные в разделе **заметки** . 
  
Чтобы просмотреть таблицы свойств фигуры для страницы, убедитесь, что ничего не выбрано на странице, щелкните правой кнопкой мыши страницу и нажмите кнопку **Показать таблицу свойств фигуры**.
  
Сведения о конкретной рецензентов обратитесь к разделу **Редактор** в таблице свойств фигуры документа.
