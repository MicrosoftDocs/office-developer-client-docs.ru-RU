# <a name="annotation-section"></a>Annotation Section

Содержит сведения о комментариях, вставленных на страницу документа. 
  
> [!NOTE]
> Этот раздел используется для отслеживания комментариев только при открытии файла .vsd в Microsoft Visio 2013 г. или при сохранении файла .vsdx в формате .vsd file. Он не используется для отслеживания комментариев в новых документах .vsdx в Visio 2013 г. 
  
## <a name="remarks"></a>Примечания

Раздел **Аннотации** таблицы ShapeSheet страницы содержит одну строку для каждого комментария, добавленного на страницу рисунка, будь то оригинальная страница рисунка или страница наложения разметки (страница, содержащая разметку, добавленную в рисунок рецензентом). Сведения о разметке, которая была добавлена с помощью ink или других форм, а не комментарии, не включены в раздел **Аннотация.** 
  
Чтобы просмотреть таблицу ShapeSheet для страницы, убедитесь, что на странице ничего не выбрано, щелкните правой кнопкой мыши страницу и щелкните **Show ShapeSheet**.
  
Сведения о конкретных рецензентах см. в разделе **Reviewer** в разделе ShapeSheet документа.
