# <a name="annotation-section"></a>Annotation Section

Содержит сведения о комментариях, вставленных на страницу документа. 
  
> [!NOTE]
> Этот раздел используется для отслеживания комментариев только при открытии VSD-файла в Microsoft Visio 2013 или при сохранении VSDX-файла в формате VSD. Он не используется для отслеживания комментариев в новых VSDX-документах в Visio 2013. 
  
## <a name="remarks"></a>Примечания

Раздел  "Аннотация" таблицы shapeSheet страницы содержит по одной строке для каждого комментария, добавленного на страницу в документе, будь то исходная страница рисования или страница наложения разметки (страница, содержащая разметку, добавленную в документ рецензентом). Сведения о разметке, добавленной с помощью ink или других фигур, а не комментариев, не включаются в раздел **"Аннотация".** 
  
Чтобы просмотреть таблицу фигур для страницы, убедитесь, что на странице ничего не выбрано, щелкните ее правой кнопкой мыши и выберите **"Показать таблицу фигур".**
  
Сведения об определенных рецензентах см. в разделе **"Рецензент"** таблицы shapeSheet документа.
