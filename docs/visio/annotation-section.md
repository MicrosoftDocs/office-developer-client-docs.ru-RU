# <a name="annotation-section"></a>Annotation Section

Содержит сведения о комментариях, вставленных в страницу документа. 
  
> [!NOTE]
> Этот раздел используется для отслеживания комментариев только при открытии файла VSD в Microsoft Visio 2013 или при сохранении vsdx-файла в формате VSD. Он не используется для отслеживания комментариев в новых vsdx-документах в Visio 2013. 
  
## <a name="remarks"></a>Примечания

Раздел **аннотации** таблицы свойств фигуры содержит по одной строке для каждого комментария, добавленного на страницу в документе, независимо от того, является ли она исходной страницей документа или разметкой разметки (страница с разметкой, которая была добавлена в документ рецензентом). Сведения о разметке, добавленной с помощью рукописных фрагментов или других фигур, а не примечаний **** , не включаются в раздел примечаний. 
  
Чтобы просмотреть таблицу свойств фигуры на странице, убедитесь, что на странице ничего не выбрано, щелкните страницу правой кнопкой мыши и выберите пункт **Показать таблицу свойств фигуры**.
  
Сведения о конкретных проверяющих можно найти в разделе **рецензент** в таблице свойств фигуры.
