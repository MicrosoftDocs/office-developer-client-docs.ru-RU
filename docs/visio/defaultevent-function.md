# <a name="defaultevent-function"></a>Функция DEFAULTEVENT

Выполняет событие по умолчанию, связанное с объектом.
  
## <a name="syntax"></a>Синтаксис

DEFAULTEVENT()
  
## <a name="remarks"></a>Примечания

В следующей таблице перечислены события по умолчанию, связанные с несколькими объектами.
  
|**Object**|**Событие по умолчанию**|
|:-----|:-----|
|Shape  <br/> |Изменение текста.  <br/> |
|Group  <br/> |Ничего не предпринимать.  <br/> |
|OLE  <br/> |Сделайте основной глагол.  <br/> |
|Другое  <br/> |Ничего не предпринимать.  <br/> |
   
В более ранних версиях Visio эта функция отображается как _DEFAULTEVENT и событие по умолчанию для группы состоит в том, чтобы открыть групповое окно.
