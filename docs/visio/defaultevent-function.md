# <a name="defaultevent-function"></a>Функция DEFAULTEVENT

Выполняет событие по умолчанию, связанного с объектом.
  
## <a name="syntax"></a>Синтаксис

(DEFAULTEVENT)
  
## <a name="remarks"></a>Замечания

В следующей таблице приведены события по умолчанию, связанные с несколько объектов.
  
|**Объект**|**Событием по умолчанию**|
|:-----|:-----|
|Shape  <br/> |Изменение текста.  <br/> |
|Группа  <br/> |Ничего не предпринимать.  <br/> |
|OLE  <br/> |Выполните основные команды.  <br/> |
|Other (другие)  <br/> |Ничего не предпринимать.  <br/> |
   
В более ранних версиях Visio эта функция отображается как _DEFAULTEVENT и события по умолчанию для группы необходимо открыть окно группы.
