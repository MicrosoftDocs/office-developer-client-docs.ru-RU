# <a name="defaultevent-function"></a>Функция DEFAULTEVENT

Выполняет событие по умолчанию, связанное с объектом.
  
## <a name="syntax"></a>Синтаксис

DEFAULTEVENT ()
  
## <a name="remarks"></a>Примечания

В следующей таблице перечислены события по умолчанию, связанные с несколькими объектами.
  
|**Object**|**Событие по умолчанию**|
|:-----|:-----|
|Shape  <br/> |Редактирование текста.  <br/> |
|Группа  <br/> |Ничего не предпринимать.  <br/> |
|OLE  <br/> |Выполнение основной команды.  <br/> |
|Прочее  <br/> |Ничего не предпринимать.  <br/> |
   
В более ранних версиях Visio Эта функция отображается в виде _ДЕФАУЛТЕВЕНТ, а для группы событие по умолчанию — открытие окна группы.
