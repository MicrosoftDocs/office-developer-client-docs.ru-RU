# <a name="placeflip-cell-page-layout-section"></a>Ячейка PlaceFlip (раздел "Макет страницы")

Определяет, как размещаемые фигуры отразить и/или поворот на странице при использовании диалоговое окно " **Настройка макета** " (на вкладке **Конструктор** в группе **Макет** нажмите кнопку **Изменить расположение**и нажмите кнопку **Дополнительные параметры расположения**).
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|&amp;H0  <br/> |Значение, используемое по умолчанию. Не отражение.  <br/> |**visLOFlipDefault** <br/> |
|&amp;H1  <br/> |Слева направо.  <br/> |**visLOFlipX** <br/> |
|&amp;H2  <br/> |Отразить сверху вниз.  <br/> |**visLOFlipY** <br/> |
|&amp;H4  <br/> |Отразить на 90 градусов.  <br/> |**visLOFlipRotate** <br/> |
|&amp;H8  <br/> |Не отражение.  <br/> |**visLOFlipNone** <br/> |
   
## <a name="remarks"></a>Замечания

Значение в ячейке PlaceFlip поможет сориентироваться размещаемой фигуры к следующей размещаемой фигуры, которым подключаются к. Обычно используется при создании макетов документы, которые используют статические связывающих.
  
Чтобы задать это поведение для определенного фигуры, используйте ShapePlaceFlip ячейки в разделе Макет фигуры.
  
Чтобы получить ссылку на ячейку PlaceFlip по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |PlaceFlip  <br/> |
   
Для получения ссылки на ячейки PlaceFlip по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPageLayout** <br/> |
|Индекс ячейки:  <br/> |**visPLOPlaceFlip** <br/> |
