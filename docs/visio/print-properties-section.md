# <a name="print-properties-section"></a>Print Properties Section

Содержит ячейки, которые контролируют форматирование страницы рисования (отображается) на странице принтера. 
  
## <a name="remarks"></a>Примечания

Используйте раздел **"Свойства печати"** для управления печатью на уровне страницы. Эти параметры используются для каждой страницы и не зависят от параметров страницы рисования, например, страница рисования, использующая альбомную ориентацию, может печататься с использованием крисовской ориентации. Кроме того, документы можно печатать с помощью сочетания ориентаций. 
  
Параметры в  разделе "Свойства печати" соответствуют значениям, установленным  на вкладке "Настройка  печати" в диалоговом окне "Настройка страницы" (на вкладке "Конструктор" щелкните диалоговое окно запуска в группе "Настройка **страницы").**  **Параметры,** которые применяются к странице рисования, см. в разделе "Свойства страницы".
