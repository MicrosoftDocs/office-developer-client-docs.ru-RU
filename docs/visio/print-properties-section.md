# <a name="print-properties-section"></a>Print Properties Section

Содержит ячейки, которые контролируют форматирование страницы рисования (отображается) на странице принтера. 
  
## <a name="remarks"></a>Примечания

Используйте раздел **Print Properties** для управления печатью на уровне страницы. Эти параметры являются на одну страницу и не зависят от параметров страницы рисования, например, страница рисования, использующая ландшафтную ориентацию, может быть напечатана с помощью портретной ориентации. Кроме того, документы можно распечатать с помощью сочетания ориентаций. 
  
Параметры в разделе **Print Properties** соответствуют значениям,  установленным на  вкладке Настройка печати  в диалоговом окне Настройка страницы (на вкладке Дизайн нажмите кнопку запуска диалогового окна в группе **Установки** страницы). **Параметры,** применимые к странице рисования, см. в разделе Свойства страниц.
