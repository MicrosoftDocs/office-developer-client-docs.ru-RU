# <a name="print-properties-section"></a>Раздел "Свойства печати"

Содержит ячейки, которые управляют форматирование страницы рисунка (отображается) на странице принтера. 
  
## <a name="remarks"></a>Замечания

Используйте раздел **Свойства печати** для управления печатью на уровне страницы. Эти параметры на странице и не зависят от рисунка параметры страницы, например, страницу документа, что используется альбомная ориентация можно распечатать с помощью книжную. Кроме того документы можно распечатать с помощью сочетания ориентации. 
  
Параметры в разделе **Свойства печати** соответствующие значения, установленные на вкладке **Параметры печати** в диалоговом окне " **Параметры страницы** " (на вкладке " **Конструктор** ", нажмите кнопку вызова диалогового окна в группе **Параметры страницы** ). Параметры, применимые к странице документа в разделе **Свойства страницы** .
