# <a name="print-properties-section"></a>Print Properties Section

Содержит ячейки, которые контролируют способ форматирования страницы документа (отображается) на странице принтера. 
  
## <a name="remarks"></a>Замечания

Раздел **Свойства печати** используется для управления печатью на уровне страниц. Эти параметры размещаются на странице и не зависят от параметров страницы документа, например страница документа, на которой используется альбомная ориентация, может печататься с использованием книжной ориентации. Кроме того, документы можно распечатать с помощью комбинации ориентаций. 
  
Параметры в разделе " **Свойства печати** " соответствуют значениям, заданным на вкладке " **Настройка печати** " диалогового окна **Параметры страницы** (на вкладке **конструктор** щелкните средство запуска диалогового окна в группе **Параметры страницы** ). В разделе **Свойства страницы** содержатся параметры, которые применяются к странице документа.
