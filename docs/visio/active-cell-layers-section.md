# <a name="active-cell-layers-section"></a>Ячейка Active (раздел "Слои")

Указывает, активна ли уровень. Фигуры без назначенный слои назначаются active слои при перетаскивании на страницу документа.
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Слой активен.  <br/> |
|FALSE  <br/> |Уровень не является активным.  <br/> |
   
## <a name="remarks"></a>Замечания

Значение в этой ячейке соответствует параметру **Active** в диалоговом окне **Свойства слоя** (в группе **Правка** на вкладке **Главная** нажмите кнопку **уровни**и выберите пункт **Свойства Layer**).
  
Чтобы получить ссылку на активную ячейку по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Layers.Active [ *i* ] где *i* = < 1 > 2, 3...  <br/> |
   
Для получения ссылки на активную ячейку по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionLayer** <br/> |
|Индекс строки:  <br/> |**visRowLayer** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visLayerActive** <br/> |
