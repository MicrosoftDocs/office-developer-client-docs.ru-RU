# <a name="ln-function"></a>Функция LN

Возвращает натуральный логарифм числа. Число должно быть положительным, а LN Возвращает значение ошибки #NUM.
  
## <a name="syntax"></a>Синтаксис

LN (* * *Number* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательный  <br/> |**Числовой** <br/> | Число, натуральный логарифм которого требуется найти.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="example"></a>Пример

LN (42) 
  
Возвращает 3,7377.
