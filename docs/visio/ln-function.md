# <a name="ln-function"></a>Функция LN

Возвращает натуральный логарифм числа. Число должно быть положительное или LN возвращает значение ошибки # число!.
  
## <a name="syntax"></a>Синтаксис

LN (** *номер* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Обязательный  <br/> |**Числовой** <br/> | Номер, который требуется найти натуральный логарифм.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Числовой
  
## <a name="example"></a>Пример

LN(42) 
  
Возвращает 3.7377.
