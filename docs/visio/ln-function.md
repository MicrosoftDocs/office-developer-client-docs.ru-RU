# <a name="ln-function"></a>Функция LN

Возвращает естественный логарифм номера. Номер должен быть положительным, или LN возвращает значение ошибки #NUM.
  
## <a name="syntax"></a>Синтаксис

Номер LN(**  ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательный  <br/> |**Числовой** <br/> | Номер, естественный логарифм которого вы хотите найти.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="example"></a>Пример

LN (42) 
  
Возвращает 3.7377.
