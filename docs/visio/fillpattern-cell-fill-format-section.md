# <a name="fillpattern-cell-fill-format-section"></a>Ячейка FillPattern (раздел "Формат заливки")

Определяет узор заливки для фигуры. Чтобы указать пользовательский узор заливки, используйте функцию использования в этой ячейке.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0  <br/> |None (прозрачной заливки).  <br/> |
|1  <br/> |Сплошной цвет.  <br/> |
|2 - 40  <br/> |Шаблоны различные заполнения, соответствующие индексирование записей в диалоговом окне **заливки** .  <br/> |
   
## <a name="remarks"></a>Замечания

Также можно задать это значение, с помощью диалогового окна **заполните поля** (на вкладке **Главная** в группе **фигуры** выберите **заполните поля** и нажмите кнопку **Параметры заливки**).
  
Чтобы получить ссылку на ячейку узор заливки по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Узор заливки  <br/> |
   
Для получения ссылки на ячейки узор заливки по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowFill** <br/> |
|Индекс ячейки:  <br/> |**visFillPattern** <br/> |
