# <a name="user-defined-cells-section"></a>User-defined Cells Section

Область работы для ввода формул в ячейках, на которые могут ссылаться другие ячейки и средства надстройки.
  
## <a name="remarks"></a>Примечания

Как и [в разделе "С](scratch-section.md) нуля", раздел **"Пользовательские** ячейки" может использоваться решением различными способами. Преимущество раздела **"Пользовательские ячейки"** заключается в том, что ячейки можно именовать; ячейки в разделе **"Scratch"** нельзя назвать. 
  
Если ваше решение имеет четко определенное назначение, для ясности  рекомендуем использовать ячейки в разделе "Пользовательские ячейки", так как ячейки пользователей можно назвать. Раздел **"С нуля"** обычно используется для изоляции повторяются сложные вычисления. 
  
Чтобы добавить этот раздел, используйте диалоговое окно **"Вставка** раздела" (щелкните правой кнопкой мыши в окне "Таблица фигур" и выберите **"Вставка раздела").**
