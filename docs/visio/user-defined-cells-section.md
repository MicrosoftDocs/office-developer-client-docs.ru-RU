# <a name="user-defined-cells-section"></a>User-defined Cells Section

Область работы для ввода формул в пользовательских ячейках, на которые можно ссылаться другими ячейками и средствами надстройки.
  
## <a name="remarks"></a>Примечания

Как и [раздел Scratch,](scratch-section.md) раздел **Ячейки,** определенные пользователем, может использоваться вашим решением различными способами. Преимущество раздела **Ячейки, определяемого пользователем,** заключается в том, что ячейки можно назвать; ячейки в **разделе Scratch** нельзя назвать. 
  
Если ваше решение имеет четко определенные цели, предлагается использовать  ячейки в разделе Ячейки, определенные пользователем, для ясности, так как ячейки пользователей можно назвать. Раздел **Scratch** обычно используется для изоляции повторных сложных вычислений. 
  
Чтобы добавить этот раздел, используйте диалоговое окно **Insert Section** (щелкните правой кнопкой мыши в окне ShapeSheet и нажмите **кнопку Вставить раздел**).
