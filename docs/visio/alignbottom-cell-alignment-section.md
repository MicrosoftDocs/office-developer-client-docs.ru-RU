# <a name="alignbottom-cell-alignment-section"></a>Ячейка AlignBottom (раздел "Выравнивание")

Определяет позицию по вертикали, по отношению к источнику родительского объекта по горизонтали, направляющей или направляющей точки, относительно которого выровнена нижняя граница фигуры.
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку AlignBottom по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AlignBottom  <br/> |
   
Для получения ссылки на ячейки AlignBottom по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignBottom** <br/> |
