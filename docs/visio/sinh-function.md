# <a name="sinh-function"></a>Функция SINH

Возвращает гиперболический синус угла. 
  
## <a name="syntax"></a>Синтаксис

SINH (* * *угол* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _градусов_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Угол, для которого требуется получить гиперболический синус.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Номер
  
## <a name="remarks"></a>Примечания

Функция SINH использует радианы, если вы не укажете другую единицу измерения в качестве _угла_.
  
## <a name="example"></a>Пример

SINH (45 градусов) 
  
Возвращает 0,8687.
