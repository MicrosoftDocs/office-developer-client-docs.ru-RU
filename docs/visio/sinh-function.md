# <a name="sinh-function"></a>Функция SINH

Возвращает гиперболический синус угла. 
  
## <a name="syntax"></a>Синтаксис

SINH (** *угол* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _угол_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Угол которого необходимо получить гиперболический синус.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Number
  
## <a name="remarks"></a>Замечания

Функция SINH использует радианах, если не указать разные единицы измерения в _угол_.
  
## <a name="example"></a>Пример

SINH(45 deg) 
  
Возвращает 0.8687.
