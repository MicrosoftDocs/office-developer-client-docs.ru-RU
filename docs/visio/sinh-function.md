# <a name="sinh-function"></a>Функция SINH

Возвращает гиперболический сина угла. 
  
## <a name="syntax"></a>Синтаксис

SINH(** *угол* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _угол_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Угол, в котором можно получить гиперболический сина.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Номер
  
## <a name="remarks"></a>Примечания

Функция SINH использует радианы, если не указать другую единицу измерения в _угол._
  
## <a name="example"></a>Пример

SINH (45 дег) 
  
Возвращает 0.8687.
