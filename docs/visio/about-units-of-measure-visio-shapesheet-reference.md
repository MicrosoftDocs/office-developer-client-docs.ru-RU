# <a name="about-units-of-measure-visio-shapesheet-reference"></a>О единиц измерения (Справочник по таблице свойств фигуры Visio)

При вставке полей в текст или построение формулы, часто указать единицы измерения для значений, вводимых.
  
Microsoft Visio оценивает результат формулы по-разному в зависимости от ячейки, в котором введите формулу. В общем случае ячеек, которые представляют положения фигуры, измерения или угол требуется пара номер подразделения, состоящего из числа и классификации единиц измерения, необходимый для интерпретации номер. Многие другие ячейки не требуют единиц измерения и оценка в строку, значение TRUE или FALSE или индекса. Например, в ячейке **FillForegnd** означает, что цвет 5 из палитры цветов документа и той же формуле означает, что значение TRUE (и блокирует ширину фигуры) в ячейке LockWidth. 
  
При вводе формулы в ячейке, требующей измерения значение всегда укажите единицы измерения. Если не указать единицы измерения, Visio использует подразделения по умолчанию для этой ячейки, который может быть страница единицы, размеры единиц измерения, единицы измерения, единицы длительности или угловые.

## <a name="implicit-units-of-measure"></a>Неявные единиц измерения

Когда Visio анализирует и хранит пары номер подразделения, можно использовать единицы явных или неявных единиц измерения. Число единиц выраженное в явных всегда отображается в единицы измерения, которые изначально были введены. Число единиц выраженное в неявных всегда преобразуется в эквивалентное значение документа, страницы или угловые единицы при выборе ячейки.
  
Предположим, например, в ячейке A введите эквивалент 1 дюйм с помощью явных единиц измерения и в ячейке B Использование неявных единиц измерения и ячейки A и B ячейки использовать размеры единиц измерения. Далее следует изменить единиц измерения по умолчанию для страницы на см. Ячейка A по-прежнему отображается дюймов 1, так как они используют явные единиц измерения, которые не изменяйте значения по умолчанию. Ячейка B теперь отображается 2,54 см, эквивалентное значение в единиц измерения по умолчанию.
  
Чтобы ввести единицы неявно, используйте следующий синтаксис.
  
```vb
number  [unit , flag ]
```

|||
|:-----|:-----|
| _number_ <br/> |Исходное значение, например 3,7, 1, 7E-4 или 5 1/2.  <br/> |
| _подразделения_ <br/> |Единицы, в которых _номер_ изначально выражается.  <br/> |
| _флаг_ <br/> |Система измерения для использования при отображении неявных значение единицы. Ниже приведены значения.  <br/> |
   
Параметр _флаг_ — это один из следующих буквы (прописные или строчные), указывающее, системы измерений, который должен использоваться при отображении неявных значение единицы. 
  
|_**Флаг**_|**Система мер**|**Пример**|
|:-----|:-----|:-----|
| , А  <br/> | Angular  <br/> | = 5 [градусов A]  <br/> |
| d, D  <br/> | Drawing  <br/> | [в, D] = 5  <br/> |
| e, E  <br/> | Продолжительность  <br/> | = 5 [а, E]  <br/> |
| p, P  <br/> | Page  <br/> | [в, P] = 5  <br/> |
| t, T  <br/> | Тип  <br/> | = 5 [pt или T]  <br/> |
   
Кроме того можно использовать неявных единицы списка Рассылки, DP, DT, DA, DE для неявных документа, страницы-, текст-, угловые- и единицы времени, соответственно. Эти единицы Предположим, что связанное значение — внутренний единиц измерения. Например если текущей системы измерения см, *= 2 списка Рассылки* как 2 внутренних единиц (см) и отображается как 5,08 cm. 
  
Использование неявных синтаксис, описанных выше, это выражение (= 2 DL) является эквивалентом 2 d] [в. Синтаксис неявных дает возможность выбора способа интерпретации значения, поэтому можно также задать 2 [ft, d], который будет как 2 футов и отображается как 60,96 см. Неявные единицы списка Рассылки, DP, DT, DA и DE универсальные и не имеющих локализованные версии.
