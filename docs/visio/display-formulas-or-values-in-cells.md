# <a name="display-formulas-or-values-in-cells"></a>Отображение формул или значений в ячейках

1. Щелкните окно ShapeSheet, чтобы сделать его активным.
    
2. На **вкладке "Инструменты таблицы фигур"**  в группе "Представление" щелкните "Формулы", чтобы просмотреть формулы, или щелкните "Значения", чтобы просмотреть значения.   
    
    > [!NOTE]
    > Можно также щелкнуть правой кнопкой мыши в  окне таблицы фигур, а затем щелкнуть "Формулы" или "Значения" в **ярлыковом** меню.
