# <a name="display-formulas-or-values-in-cells"></a>Отображение формул или значений в ячейках

1. Щелкните в окне ShapeSheet, чтобы сделать его активным.
    
2. На **вкладке ShapeSheet Tools** в группе  **Просмотр** щелкните Формулы  для просмотра формул или щелкните Значения для просмотра значений. 
    
    > [!NOTE]
    > Вы также можете щелкнуть правой кнопкой мыши в окне ShapeSheet, а затем **нажать** формулы или значения **в** меню ярлыка.
