# <a name="gamma-cell-image-properties-section"></a>Ячейка Gamma (раздел "Свойства изображения")

Настраивает или корректирует насыщенность изображения для определенного выходное устройство, например монитор или средства поиска вирусов. Значение по умолчанию — 1 (корректировка отсутствует).
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку гамма по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Гамма  <br/> |
   
Для получения ссылки на ячейки гамма по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowImage** <br/> |
| Индекс ячейки:  <br/> |**visImageGamma** <br/> |
