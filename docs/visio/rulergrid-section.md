# <a name="ruler-amp-grid-section"></a>Раздел &amp; Сетка линейки

Отображает текущие параметры линейки и сетки страницы.
  
## <a name="remarks"></a>Примечания

Этот раздел — свойство страницы, которое можно просмотреть только в окне ShapeSheet для страницы. Ячейки, которые в нем содержатся, соответствуют настройкам  в диалоговом окне **Линейка &amp;** Grid (на вкладке Просмотр в группе **Show/Hide** щелкните пусковое окно диалогового окна).
