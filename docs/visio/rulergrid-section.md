# <a name="ruler-amp-grid-section"></a>Раздел &amp; "Линейка сетки"

Показывает текущие параметры линейки и сетки страницы.
  
## <a name="remarks"></a>Примечания

Этот раздел является свойством страницы, которое можно просмотреть только в окне таблицы фигур для страницы. Ячейки, которые он содержит, соответствуют настройкам в диалоговом окне **"Линейка &amp;** сетки" (на вкладке "Вид" в группе "Показать/скрыть" щелкните окно запуска диалогового окна).
