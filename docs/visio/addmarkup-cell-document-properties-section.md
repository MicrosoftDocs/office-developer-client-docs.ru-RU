# <a name="addmarkup-cell-document-properties-section"></a>Ячейка AddMarkup (раздел "Свойства документа")

Указывает ли рецензирование документа для разметки.
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Время просмотра документа.  <br/> |
|FALSE  <br/> |Документ не находится на рассмотрении (по умолчанию).  <br/> |
   
## <a name="remarks"></a>Замечания

При AddMarkup, ячейки имеет значение TRUE, проверяющий является добавление разметки и изменения применяются к перекрытия страницы разметки, не следует страниц исходного документа. Когда ячейки AddMarkup имеет значение FALSE, отслеживание исправлений отключено и изменения применяются к страниц исходного документа.
  
> [!NOTE]
> Можно запретить разметки с документами с помощью функции защиты. Если ячейка AddMarkup содержит формулу = GUARD(FALSE), команда **Отслеживать разметку** отключена. 
  
Этот параметр соответствует параметру команду **Отслеживать разметку** **разметки** группы на вкладке **Review** . 
  
Чтобы получить ссылку на ячейку AddMarkup по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |AddMarkup  <br/> |
   
Для получения ссылки на ячейки AddMarkup по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowDoc** <br/> |
|Индекс ячейки:  <br/> |**visDocAddMarkup** <br/> |
