# <a name="alignleft-cell-alignment-section"></a>AlignLeft Cell (Alignment Section)

Определяет горизонтальное положение относительно начала координат родительского элемента, вертикальной направляющей или направляющей точки, относительно которого выровнена левая граница фигуры.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку AlignLeft по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | AlignLeft  <br/> |
   
Чтобы получить ссылку на ячейку AlignLeft по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**Висровалигн** <br/> |
| Индекс ячейки:  <br/> |**Висалигнлефт** <br/> |
