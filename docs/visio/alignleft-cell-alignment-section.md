# <a name="alignleft-cell-alignment-section"></a>Ячейка AlignLeft (раздел "Выравнивание")

Определяет позицию по горизонтали, по отношению к источнику родительского объекта вертикальной направляющей или направляющей точки, относительно которого выровнена левая граница фигуры.
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку AlignLeft по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AlignLeft  <br/> |
   
Для получения ссылки на ячейки AlignLeft по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignLeft** <br/> |
