# <a name="leftmargin-cell-text-block-format-section"></a>Ячейка LeftMargin (раздел "Формат текстового блока")

Определяет расстояние между левой границей блока текста и текст, которые он содержит. Значение по умолчанию — 0,1 дюйма. Это значение не зависит от масштаба документа. Если документа изменяется, левое поле остается неизменным.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки LeftMargin по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | LeftMargin  <br/> |
   
Для получения ссылки на ячейки LeftMargin по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowText** <br/> |
| Индекс ячейки:  <br/> |**visTxtBlkLeftMargin** <br/> |
