# <a name="fillgradientenabled-cell-gradient-properties-section"></a>Ячейка FillGradientEnabled (раздел "Свойства градиента")

Определяет, включена ли градиентной заливки для этой фигуры. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Градиентные заливки отображается на форме.  <br/> |
|FALSE  <br/> |Градиентные заливки не отображаются на форме.  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **FillGradientEnabled** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | FillGradientEnabled  <br/> |
   
Для получения ссылки на ячейки **FillGradientEnabled** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowGradientProperties** <br/> |
| Индекс ячейки:  <br/> |** visFillGradientEnabled ** <br/> |
