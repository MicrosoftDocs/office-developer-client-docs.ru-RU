# <a name="tabs-section"></a>Раздел "Вкладки"

Содержит ячейки для фигур или стилей, которые управляют позиции табуляции и выравнивание.
  
## <a name="remarks"></a>Замечания

Можно также задать эти значения с помощью вкладки **вкладки** в диалоговом окне **текст** (нажмите кнопку вызова диалогового окна в группе **Шрифт** на вкладке **Главная** ) или с помощью **Линейки**. Чтобы отобразить **Линейки**, щелкните правой кнопкой мыши текст в режиме редактирования текста и нажмите кнопку **Линейки** в контекстном меню. 
  
Можно задать позиции табуляции, определенных символов, выделив этот текст, а затем установить позиции табуляции. Если значение позиции табуляции, определенных символов, новая строка добавляется в окне таблицы свойств фигуры. Имя строки представляет число знаков, которое содержит позиции табуляции.
