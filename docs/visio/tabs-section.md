# <a name="tabs-section"></a>Tabs Section

Содержит ячейки для фигур или стилей, которые контролируют положение и выравнивание стоп-вкладок.
  
## <a name="remarks"></a>Примечания

Эти значения также можно установить с помощью вкладки "Вкладки" в текстовом  диалоговом  окне (в диалоговом окне "Щелчок" в группе шрифтов на вкладке "Главная") или с помощью линейки **текста.**   Чтобы  **отобразить** линейку текста, щелкните правой кнопкой мыши текст в режиме редактирования текста и выберите "Линейка текста" в ярлыковом меню. 
  
Вы можете установить остановки вкладок для определенных символов, выбрав текст, а затем настроив остановку вкладок. При наборе остановок вкладок для определенных символов в окне ShapeSheet добавляется новая строка. Имя строки представляет количество символов, которые содержит остановка вкладок.
