# <a name="tabs-section"></a>Tabs Section

Содержит ячейки для фигур или стилей, которые контролируют положение стоп-вкладки и выравнивание.
  
## <a name="remarks"></a>Примечания

Эти значения можно также установить с помощью вкладок **Tabs** в диалоговом окне **Текст** (щелкните диалоговое окно в группе **Шрифт** на вкладке **Home)** или с помощью текстового **линейки**. Чтобы отобразить **текстовую** линейку, щелкните правой кнопкой мыши текст в режиме редактирования текста, а затем нажмите **текстовую** линейку в меню ярлыка. 
  
Вы можете установить остановки вкладки для определенных символов, выбрав текст, а затем установив остановку вкладки. При наборе остановок вкладки для определенных символов в окне ShapeSheet добавляется новая строка. Имя строки представляет количество символов, которые содержит остановка вкладки.
