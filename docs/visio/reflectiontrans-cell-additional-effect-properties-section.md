# <a name="reflectiontrans-cell-additional-effect-properties-section"></a>Ячейка ReflectionTrans (раздел "Дополнительные свойства эффекта")

Определяет прозрачность отражения в процентах от 0 до 100%. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **ReflectionTrans** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | ReflectionTrans  <br/> |
   
Для получения ссылки на ячейки **ReflectionTrans** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowOtherEffectProperties** <br/> |
| Индекс ячейки:  <br/> |**visReflectionTrans** <br/> |
