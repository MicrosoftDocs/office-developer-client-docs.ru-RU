# <a name="setf-function"></a>Функция SETF

Задает формулу ячейки. 
  
## <a name="syntax"></a>Синтаксис

SETF( GETREF(** *cell* ** ), ** *formula* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _cell_ <br/> |Обязательно  <br/> |**Строка** <br/> |Ячейка, формулу которой необходимо установить.  <br/> |
| _formula_ <br/> |Обязательно  <br/> |**Строка** <br/> |Используемая формула.  <br/> |
   
## <a name="remarks"></a>Примечания

При оценке результат выражения в _формуле_ становится новой формулой в _ячейке._ Если _формула_ заключена в кавычках, выражение в кавычках будет записано в _ячейку._ Чтобы установить  _для ячейки_ строку, необходимо заключить  _формулу_ в три набора кавычка. 
  
Целевая ячейка должна быть указана с помощью ссылки GETREF() или в качестве строки, чтобы избежать цикличности. Предпочтительным является использование GETREF, так как Microsoft Visio может настраивать ссылки при перемещении фигуры в другой документ.
  
Если  _ячейка_ не указана с помощью GETREF или в качестве строки, функция возвращает ошибку, и формула ячейки не меняется. Если  _формула_ содержит ошибку синтаксиса, функция возвращает ошибку, а формула в  _ячейке_ не меняется. 
  
## <a name="example-1"></a>Пример 1

SETF( GETREF(Scratch.A1), 1,5 в \* 6 + 1 ft)
  
Задает для формулы Scratch.A1 21 дюйма.
  
## <a name="example-2"></a>Пример 2

SETF( GETREF(Scratch.A1), "1,5 в \* 6 + 1 ft")
  
Задает формулу "С нуля". A1 для выражения 1,5 в \* 6+1 ft.
  
## <a name="example-3"></a>Пример 3

SETF( GETREF(Scratch.A1), """Say """"ahh""""""
  
Задает формулу Scratch.A1 для строки "Say "ahh"" (Скажите "ahh"), которая оценивается как "ahh".
