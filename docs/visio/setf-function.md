# <a name="setf-function"></a>Функция SETF

Задает формулу ячейки. 
  
## <a name="syntax"></a>Синтаксис

SETF (GETREF (* * *ячейка* * *), * * *Формула* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _элемент_ <br/> |Обязательный  <br/> |**String** <br/> |Ячейка, для которой задается формула.  <br/> |
| _formula_ <br/> |Обязательный  <br/> |**String** <br/> |Используемая формула.  <br/> |
   
## <a name="remarks"></a>Примечания

При оценке результат выражения в _формуле_ становится новой формулой в _ячейке_. Если _Формула_ заключена в кавычки, выражение с кавычками записывается в _ячейку_. Чтобы присвоить _ячейке_ строку, заключите _формулу_ в три набора кавычек. 
  
Целевую ячейку необходимо указать с помощью ссылки на GETREF () или в виде строки, чтобы избежать цикличности. Рекомендуется использовать GETREF, так как Microsoft Visio может скорректировать ссылки при перемещении фигуры в другой документ.
  
Если _ячейка_ не указана с помощью GETREF или строки, функция возвращает ошибку, а формула ячейки не изменяется. Если _Формула_ содержит синтаксическую ошибку, функция возвращает ошибку, а формула в _ячейке_ не изменяется. 
  
## <a name="example-1"></a>Пример 1

SETF (GETREF ("с нуля. a1), \* 1,5 в 6 + 1 ФТ)
  
Задает формулу "с нуля" до 21 дюймов.
  
## <a name="example-2"></a>Пример 2

SETF (GETREF ("с нуля. a1"), \* "1,5" в 6 + 1 ФТ)
  
Задает формулу "с нуля". A1 на выражение 1,5 в\*6 + 1 ФТ.
  
## <a name="example-3"></a>Пример 3

SETF (GETREF ("с нуля. a1"), "" "" "" "" "АХХ" "" "" ")
  
Задает формулу "" с начала. a1 "на строку" Скажите "" АХХ "" "(" ""), которая оценивается как "АХХ".
