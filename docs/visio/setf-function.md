# <a name="setf-function"></a>Функция SETF

Задает формулу ячейки. 
  
## <a name="syntax"></a>Синтаксис

SETF (GETREF (** *ячейки* **), ** *Формула* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _ячейки_ <br/> |Обязательный  <br/> |**Строка** <br/> |Ячейки, формулы, чтобы задать.  <br/> |
| _formula_ <br/> |Обязательный  <br/> |**Строка** <br/> |Формула для.  <br/> |
   
## <a name="remarks"></a>Замечания

При выполнении функции, в результате выражение в _формуле_ становится новые формулы в _ячейке_. Если _формуле_ заключен в кавычки, кавычках выражение записывается в _ячейку_. Чтобы задать _ячейки_ в строку, заключите _формулы_ в трех наборов кавычки. 
  
Целевой ячейки должен быть указан с помощью ссылки на GETREF() или в виде строки во избежание циклические ссылки. С помощью GETREF является более предпочтительным, так как Microsoft Visio можно настроить ссылки, при перемещении фигуры в другой документ.
  
Если _ячейки_ не указан с помощью GETREF или в виде строки, функция возвращает ошибку и не ячейка изменяется. Если _Формула_ содержит синтаксическая ошибка, функция возвращает ошибку, и формулы в _ячейке_ не изменяется. 
  
## <a name="example-1"></a>Пример 1

SETF (GETREF(Scratch.A1), 1,5 в \* 6 + 1 футов)
  
Задает формулу Scratch.A1 21 см.
  
## <a name="example-2"></a>Пример 2

SETF (GETREF(Scratch.A1), «1,5 в \* 6 + 1 ft»)
  
Задает формулу нуля. A1 на него 1,5 в\*ft 6 + 1.
  
## <a name="example-3"></a>Пример 3

SETF (GETREF(Scratch.A1), «"«Сказать «««ahh»»»)
  
Задает формулу Scratch.A1 в строку «Сказать ««ahh»»» имеющее сказать «ahh».
