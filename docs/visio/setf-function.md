# <a name="setf-function"></a>Функция SETF

Задает формулу ячейки. 
  
## <a name="syntax"></a>Синтаксис

SETF (ячейка GETREF(**  ** ), ** *формула* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _ячейка_ <br/> |Обязательный  <br/> |**String** <br/> |Ячейка, формула которой устанавливается.  <br/> |
| _formula_ <br/> |Обязательный  <br/> |**String** <br/> |Формула для использования.  <br/> |
   
## <a name="remarks"></a>Примечания

При оценке результат выражения в  формуле становится новой формулой в _ячейке._ Если  _формула_ заключена в кавычках, цитируемое выражение пишется в  _ячейку_. Чтобы настроить  _ячейку_ на строку, заключив  _формулу_ в три набора кавычков. 
  
Целевая ячейка должна быть указана с помощью ссылки GETREF() или в качестве строки, чтобы избежать округлости. Использование GETREF является предпочтительным, так как microsoft Visio может корректировать ссылки при перемещении фигуры в другой документ.
  
Если  _ячейка_ не указана с помощью GETREF или в качестве строки, функция возвращает ошибку, и формула ячейки не меняется. Если  _формула_ содержит ошибку синтаксиса, функция возвращает ошибку, и формула в  _ячейке_ не меняется. 
  
## <a name="example-1"></a>Пример 1

SETF (GETREF(Scratch.A1), 1,5 в \* 6 + 1 фут)
  
Задает формулу Scratch.A1 до 21 дюйма.
  
## <a name="example-2"></a>Пример 2

SETF (GETREF(Scratch.A1), "1.5 в \* 6 + 1 фут")
  
Задает формулу Scratch. A1 для выражения 1.5 в \* 6+1 ft.
  
## <a name="example-3"></a>Пример 3

SETF (GETREF(Scratch.A1), """Say """ahh"""""")"
  
Задает формулу Scratch.A1 строке "Say "ahh"", которая оценивается как "ahh".
