# <a name="rounding-cell-line-format-section"></a>Rounding Cell (Line Format Section)

Указывает радиус закругления округления, который применяется, когда два непрерывных сегмента сопоставлены с контуром. Например, округление можно использовать для предоставления закругленных углов прямоугольника. Чтобы задать округление, введите значение с единицами измерения (числом, состоящий из единицы измерения).
  
## <a name="remarks"></a>Примечания

Вы также можете задать это значение в диалоговом окне **строка** (на вкладке **Главная** , в группе **фигура** , выберите пункт **линия**, затем выберите **вес**, а затем щелкните **дополнительные линии**).
  
Чтобы получить ссылку на ячейку округления по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Rounding  <br/> |
   
Чтобы получить ссылку на ячейку округления по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровлине** <br/> |
|Индекс ячейки:  <br/> |**Вислинераундинг** <br/> |
