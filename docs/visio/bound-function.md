# <a name="bound-function"></a>Функция BOUND

Ограничивает значение ячейки диапазоном или набором диапазонов.
  
## <a name="syntax"></a>Синтаксис

ГРАНИЦА (* * *значение* * * * * * * * * * * * ** * * * * * * * * * * * * ** * * * * * * * * | * * * * * * * * * после * * * * * * * * * | *, * * *Значение1* * *, * * * * * *] ** 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _value_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Текущее значение ограничено.  <br/> |
| _type_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Указывает, является ли ограничение инклюзивным (0), исключительным (1) или отключенным (2).  <br/> |
| _пропуск_ <br/> |Обязательный  <br/> |**Логический** <br/> | ЗНАЧЕНИЕ TRUE, чтобы игнорировать диапазон; Значение FALSE, чтобы ограничить значение ячейки диапазоном.  <br/> |
| _взнос_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первое значение в диапазоне.  <br/> |
| _value2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второе значение в диапазоне.  <br/> |
   
## <a name="remarks"></a>Примечания

Используйте функцию с приВЯЗКОЙ, чтобы ограничить значение ячейки верхней и нижней границей, например, для управления объектами, которые не должны растягиваться выше или ниже минимальной или максимальной высоты. Ограничение может быть инклюзивным или исключительным относительно диапазона или диапазонов. Если текущее значение не должно быть ограничено, задайте для параметра _Type_ значение 2 (отключено). 
  
Можно определить несколько диапазонов, указав несколько экземпляров параметров _Ignore_, _value1_и _value2_ . Используйте параметр _Ignore_ , чтобы отключить ограничения для определенного диапазона. 
  
Формула, содержащая СВЯЗАНную функцию, не перезаписывается, если ее значение изменяется; Вместо этого Формула сохраняется, а новое значение помещается в параметр _value_ .

В этом примере функция привязки используется для принудительного включения управляющего маркера в ограничивающий прямоугольник фигуры. 
  
Controls. x1 = граница (\*ширина 0,5, 0, ложь,\*ширина 0,\*ширина 1)
  
Controls. Y1 = граница (\*высота 0,5, 0, ложь,\*высота 0,\*высота 1)
