# <a name="bound-function"></a>Функция BOUND

Ограничивает значение ячейки диапазоном или набором диапазонов.
  
## <a name="syntax"></a>Синтаксис

ПРИВЯЗКИ (** *значение* **, ** *Тип* **, ** *Игнорировать* **, ** *значение1* **, ** *значение2* ** ** * [, ignore(n), value1(n), value2(n),...] * **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _value_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Текущее значение ограничены.  <br/> |
| _type_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Является ли ограничение включительно (0), монопольная (1) или отключено (2).  <br/> |
| _Пропуск_ <br/> |Обязательный  <br/> |**Boolean** <br/> | Значение TRUE, следует ли игнорировать диапазон; Значение FALSE, чтобы ограничить значения ячейки в диапазон.  <br/> |
| _Значение1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первое значение в диапазоне.  <br/> |
| _значение2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второе значение в диапазоне.  <br/> |
   
## <a name="remarks"></a>Замечания

Используйте функцию ПРИВЯЗКИ для ограничения ячейки значение на верхнюю и нижнюю границу, например, для управления объектами, не должен быть растянут выше или ниже минимального и максимального высоту. Ограничение может быть включительно или монопольной по отношению к диапазон или диапазоны. Если текущее значение не должен быть ограничен, установите для параметра _Тип_ 2 (отключено). 
  
Вы можете определить несколько диапазонов, передав несколько вхождений _Игнорировать_ _значение1_и _значение2_ параметров. Используйте параметр _Игнорировать_ отключить ограничения, определенного диапазона. 
  
Формула, содержащий ПРИВЯЗАННОЙ функции не будет перезаписан при изменении значения; Вместо этого сохраняются формулы и новое значение помещается в качестве _значения_ параметра.
