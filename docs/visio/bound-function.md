# <a name="bound-function"></a>Функция BOUND

Ограничивает значение ячейки диапазоном или набором диапазонов.
  
## <a name="syntax"></a>Синтаксис

BOUND (** *value* **, ** *type* **, ** *ignore* **, ** *value1* **, ** *value2* ** ** * [,ignore(n), value1(n), value2(n),...] * ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _value_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Текущее ограниченное значение.  <br/> |
| _type_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Является ли ограничение инклюзивным (0), монопольным (1) или отключенным (2).  <br/> |
| _ignore_ <br/> |Обязательный  <br/> |**Логический** <br/> | TRUE для игнорирования диапазона; ЗНАЧЕНИЕ FALSE, чтобы ограничить значение ячейки диапазоном.  <br/> |
| _value1_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Первое значение в диапазоне.  <br/> |
| _value2_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Второе значение в диапазоне.  <br/> |
   
## <a name="remarks"></a>Примечания

Используйте функцию BOUND, чтобы ограничить значение ячейки верхней и нижней границами, например для управления объектами, которые не должны быть растянуты выше или ниже минимальной или максимальной высоты. Ограничение может быть инклюзивным или монопольным в отношении диапазона или диапазонов. Если текущее значение не должно быть ограничено, установите для параметра  _типа_ значение 2 (отключено). 
  
Можно определить несколько диапазонов, заданы несколько вхождений параметров _ignore,_ _value1_ и _value2._ Параметр  _ignore_ используется для отключения ограничений для определенного диапазона. 
  
Формула, содержащая функцию BOUND, не перезаписывается при смене ее значения; вместо этого формула сохраняется, а новое значение помещается в _параметр value._ 
  
## <a name="example-1"></a>Пример 1

В этом примере функция BOUND используется для принудительного хранения окне управления в пределах границ фигуры. 
  
Controls.X1 = BOUND(Width \* 0.5, 0, FALSE, Width \* 0, Width \* 1)
  
Controls.Y1 = BOUND(Height \* 0.5, 0, FALSE, Height \* 0, Height \* 1)
  
## <a name="example-2"></a>Пример 2

В этом примере функция BOUND используется для ограничения ширины фигуры до 2, 4 и 6 дюймов. 
  
Width = BOUND(, 0, FALSE, 2 in, 2 in, FALSE, 4 in, 4 in, FALSE, 6 in, 6 in)
