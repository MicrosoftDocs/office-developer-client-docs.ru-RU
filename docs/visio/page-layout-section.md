# <a name="page-layout-section"></a>Раздел "Макет страницы"

Содержит ячейки, которые задают параметры страницы для фигуры и соединители, таких как интервалы между всех фигур на странице, интервал между все соединители на странице, а также маршрутизацию стилей для всех соединителей на странице.
  
## <a name="remarks"></a>Замечания

Чтобы переопределить страницы значения по умолчанию для определенного фигуры, используйте ячейки в разделе **Макет фигуры** . 
  
Также можно задать значения ячеек в этом разделе на вкладке **Макет и маршрутизации** в диалоговом окне " **Параметры страницы** " (на вкладке " **Конструктор** ", нажмите кнопку вызова диалогового окна в группе **Параметры страницы** ).
