# <a name="page-layout-section"></a>Page Layout Section

Содержит ячейки, которые контролируют параметры страниц для фигур и соединители, такие как интервал между всеми фигурами на странице, интервал между всеми соединителями на странице и стиль маршрутивки для всех соединители на странице.
  
## <a name="remarks"></a>Примечания

Для переопределения по умолчанию страницы для определенной фигуры используйте ячейки в разделе **Макет формы.** 
  
Вы также можете установить значения ячеек в  этом разделе на вкладке Макет и маршрутинг  в диалоговом окне Настройка страницы (на вкладке Дизайн нажмите кнопку запуска диалогового окна в группе **Установки** страницы).
