# <a name="page-layout-section"></a>Page Layout Section

Содержит ячейки, которые контролируют параметры страницы для фигур и соединителя, например интервалы между всеми фигурами на странице, интервалы между всеми соединителями на странице и стиль маршрутки для всех соединителях на странице.
  
## <a name="remarks"></a>Примечания

Чтобы переопредить значения по умолчанию страницы для определенной фигуры, используйте ячейки в разделе **"Макет** фигуры". 
  
Вы также можете установить значения ячеек в  этом разделе на вкладке "Макет и маршрут" в диалоговом окне  "Настройка страницы" (на вкладке "Конструктор" щелкните диалоговое окно запуска в группе "Настройка страницы").
