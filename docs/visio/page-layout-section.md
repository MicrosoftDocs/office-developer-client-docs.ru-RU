# <a name="page-layout-section"></a>Page Layout Section

Содержит ячейки, которые управляют параметрами страницы для фигур и соединителей, например расстояния между фигурами на странице, расстояния между всеми соединительными линиями на странице и стиль маршрутизации для всех соединителей на странице.
  
## <a name="remarks"></a>Примечания

Чтобы переопределить параметры страницы по умолчанию для определенной фигуры, используйте ячейки в разделе **Макет фигуры** . 
  
Вы также можете задать значения ячеек в этом разделе на вкладке **Макет и маршрутизация** в диалоговом окне **Параметры страницы** (на вкладке **конструктор** щелкните средство запуска диалогового окна в группе **Параметры страницы** ).
