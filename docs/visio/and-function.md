# <a name="and-function"></a>Функция AND

Возвращает значение TRUE (1), если все предоставленные логические выражения имеют значение TRUE. Если любое из логических выражений имеет значение FALSE или 0, функция и возвращает значение FALSE (0).
  
## <a name="syntax"></a>Синтаксис

AND (* * *logical выражение1* * *, * * *logical выражение2* * *,..., * * *логический експрессионн* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Логическое выражение_ <br/> |Обязательный  <br/> |**String** <br/> | Сочетание констант, операторов, функций и ссылок на ячейки таблицы свойств фигуры, в результате чего получается значение. Любое выражение, результатом которого является ненулевое значение, считается ИСТИНным.  <br/> |
   
## <a name="example"></a>Пример

AND (Height \> 1, PinX \> 1)
  
Возвращает значение TRUE, если оба выражения имеют значение TRUE. Возвращает значение FALSE, если любое выражение имеет значение FALSE.
