# <a name="hour-function-visioshapesheet"></a>HOUR Function (VisioShapeSheet)

Возвращает количество от 0 до 23, представляющее час дня даты _и времени_ или _выражения._
  
## <a name="syntax"></a>Синтаксис

HOUR(" ** *datetime* ** "| ** *expression* ** [, ** *lcid* ** ]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _datetime_ <br/> |Обязательно  <br/> |**Строка** <br/> | Строка, которая обычно распознается как дата и время или ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**Разные** <br/> |Выражение, которое дает дату и время.  <br/> |
| _lcid_ <br/> |Необязательный  <br/> |**Number** <br/> | Идентификатор локали, используемый при оценке нелокального даты-времени. Идентификатор языкового стандарта — это число, представленной в файлах системных заголовков.  <br/> |
   
## <a name="remarks"></a>Примечания

Компонент даты в  *дате и*  *выражении*  удаляется. 
  
Округление не выполняется. Если  *дата-время*  отсутствует или не может быть преобразована в допустимый результат, функция возвращает ошибку. 
  
Возвращаемая величина форматирована в соответствии со стилем времени, установленным текущими региональными настройками системы. 
  
Функция HOUR также принимает одно число  для выражения, где десятичная часть результата представляет часть дня с полуночи. 
  
## <a name="example-1"></a>Пример 1

HOUR("15:45")
  
Возвращает 15.
  
## <a name="example-2"></a>Пример 2

HOUR ("30 мая 1997 г., 15:45:24")
  
Возвращает 15.
  
## <a name="example-3"></a>Пример 3

HOUR(0,5)
  
Возвращает 12.
  
## <a name="example-4"></a>Пример 4

HOUR("30.05.1997")
  
Возвращает 0.
