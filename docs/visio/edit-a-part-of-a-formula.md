# <a name="edit-a-part-of-a-formula"></a>Изменение части формулы

1. Дважды щелкните ячейку. Или выберите ячейку и нажмите клавишу F2.
    
2. В ячейке или строке формул измените формулу, используя методы, описанные в следующей таблице.
    
|**Задача**|**Рекомендации**|
|:-----|:-----|
| Поместите точку вставки  <br/> | Щелкните место, в котором должна появиться точка вставки.  <br/> |
| Перемещение точки вставки  <br/> | Нажмите клавишу со стрелкой влево или вправо.  <br/> |
| Удаление знака слева  <br/> | Нажмите клавишу BACKSPACE.  <br/> |
| Удаление знака справа  <br/> | Нажмите клавишу DELETE.  <br/> |
| Выбор текста  <br/> | НаВедите указатель мыши на текст.  <br/> |
| Расширение выделенного фрагмента влево  <br/> | Нажмите клавиши SHIFT + стрелка влево.  <br/> |
| Расширение выделенного фрагмента вправо  <br/> | Нажмите клавиши SHIFT + стрелка вправо.  <br/> |
| Выбор слова  <br/> | Нажмите клавиши CTRL + SHIFT + стрелка вправо или двойным щелчком.  <br/> |
| Выделить до конца  <br/> | Нажмите клавиши SHIFT + END.  <br/> |
| Выделить до начала  <br/> | Нажмите клавиши SHIFT + HOME.  <br/> |
| Замена выбранного текста  <br/> | Введите новый текст.  <br/> |
| Удаление выделенного текста  <br/> | Нажмите клавишу DELETE. На вкладке **Главная** нажмите кнопку **Вырезать**.  <br/> |
