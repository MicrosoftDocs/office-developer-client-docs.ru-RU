# <a name="edit-a-part-of-a-formula"></a>Редактирование фрагмента формулы

1. Дважды щелкните ячейку. Выберите ячейку и нажмите клавишу F2.
    
2. В ячейке или строке формул Изменение формулы с помощью методов в следующей таблице.
    
|**Задача**|**Рекомендации**|
|:-----|:-----|
| Поместите курсор  <br/> | Выберите место точку вставки отображаться.  <br/> |
| Перемещение точки вставки  <br/> | Нажмите клавишу Стрелка влево или вправо.  <br/> |
| Удалите знак слева  <br/> | Нажмите клавишу BACKSPACE.  <br/> |
| Удалите знак справа  <br/> | Нажмите клавишу DELETE.  <br/> |
| Выберите текст  <br/> | Перетащите указатель мыши над текстом.  <br/> |
| Расширить выделение до слева  <br/> | Нажать клавиши SHIFT + Стрелка влево.  <br/> |
| Расширить выделение до вправо  <br/> | Нажать клавиши SHIFT + Стрелка вправо.  <br/> |
| Выберите слово  <br/> | Нажмите сочетание клавиш CTRL + SHIFT + Стрелка вправо или дважды щелкните значок.  <br/> |
| Выберите в конец  <br/> | Нажмите клавишу SHIFT + END.  <br/> |
| Выберите в начало  <br/> | Нажмите клавишу SHIFT + HOME.  <br/> |
| Замена выделенного текста  <br/> | Введите новый текст.  <br/> |
| Удаление выделенного текста  <br/> | Нажмите клавишу DELETE. Или на вкладке **Главная** щелкните **Вырезать**.  <br/> |
