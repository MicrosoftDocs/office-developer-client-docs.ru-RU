# <a name="edit-a-part-of-a-formula"></a>Изменение части формулы

1. Дважды щелкните ячейку. Или выберите ячейку и нажмите кнопку F2.
    
2. В ячейке или панели формул изменить формулу с помощью методов в следующей таблице.
    
|**Задача**|**Рекомендации**|
|:-----|:-----|
| Поместите точку вставки  <br/> | Щелкните, где вы хотите, чтобы точка вставки появлялись.  <br/> |
| Перемещение точки вставки  <br/> | Нажмите левый или правый клавиши стрелки.  <br/> |
| Удаление символа слева  <br/> | Нажмите BACKSPACE.  <br/> |
| Удаление символа справа  <br/> | Нажмите клавишу DELETE.  <br/> |
| Выбор текста  <br/> | Перетащите указатель над текстом.  <br/> |
| Расширение выбора влево  <br/> | Нажмите стрелку SHIFT+left.  <br/> |
| Расширение выбора справа  <br/> | Нажмите стрелку SHIFT+right.  <br/> |
| Выберите слово  <br/> | Нажмите стрелку CTRL+SHIFT+right или дважды щелкните.  <br/> |
| Выберите до конца  <br/> | Нажмите КНОПКУ SHIFT+END.  <br/> |
| Выберите начало  <br/> | Нажмите КНОПКУ SHIFT+HOME.  <br/> |
| Замена выбранного текста  <br/> | Введите новый текст.  <br/> |
| Удаление выбранного текста  <br/> | Нажмите клавишу DELETE. Или на **вкладке Главная** нажмите кнопку **Вырезать**.  <br/> |
