# <a name="edit-a-part-of-a-formula"></a>Изменение части формулы

1. Дважды щелкните ячейку. Или выберите ячейку и нажмите F2.
    
2. В ячейке или на панели формул отредактируете формулу, используя методы, которые следующую таблицу.
    
|**Задача**|**Рекомендации**|
|:-----|:-----|
| Поместите точку вставки  <br/> | Щелкните, где будет отображаться точка вставки.  <br/> |
| Перемещение точки вставки  <br/> | Нажмите клавишу со стрелкой влево или вправо.  <br/> |
| Удаление символа слева  <br/> | Нажмите кнопку BACKSPACE.  <br/> |
| Удаление символа справа  <br/> | Нажмите клавишу DELETE.  <br/> |
| Выбор текста  <br/> | Перетащите указатель на текст.  <br/> |
| Расширение выделения слева  <br/> | Нажмите клавиши SHIFT+СТРЕЛКА ВЛЕВО.  <br/> |
| Расширение выделения справа  <br/> | Нажмите клавиши SHIFT+СТРЕЛКА ВПРАВО.  <br/> |
| Выбор слова  <br/> | Нажмите клавиши CTRL+SHIFT+СТРЕЛКА ВПРАВО или дважды щелкните.  <br/> |
| Выберите до конца  <br/> | Нажмите shift+END.  <br/> |
| Выберите в начале  <br/> | Нажмите shift+HOME.  <br/> |
| Замена выбранного текста  <br/> | Введите новый текст.  <br/> |
| Удаление выбранного текста  <br/> | Нажмите клавишу DELETE. Или на вкладке **"Главная"** нажмите кнопку **"Вырезать".**  <br/> |
