# <a name="bevelmaterialtype-cell-bevel-properties-section"></a>Ячейка BevelMaterialType (раздел "Свойства багетной рамки")

Определяет тип материалов, которые рельефов состоит из. 
  
|**Описание**|**Значение**|
|:-----|:-----|
|0  <br/> |Нет специальных материалов  <br/> |
|1  <br/> |Подложки  <br/> |
|2  <br/> |Горячего подложки  <br/> |
|3  <br/> |Пластик  <br/> |
|4  <br/> |Системы  <br/> |
|5  <br/> |Темный пограничного сервера  <br/> |
|6  <br/> |Мягкий край  <br/> |
|7  <br/> |Плоский  <br/> |
|8  <br/> |Проволочная  <br/> |
|9  <br/> |Порошок  <br/> |
|10  <br/> |Прозрачные порошок  <br/> |
|11  <br/> |Очистить  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **BevelMaterialType** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | BevelMaterialType  <br/> |
   
Для получения ссылки на ячейки **BevelMaterialType** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowBevelProperties** <br/> |
| Индекс ячейки:  <br/> |**visBevelMaterialType** <br/> |
