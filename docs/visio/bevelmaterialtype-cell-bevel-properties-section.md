# <a name="bevelmaterialtype-cell-bevel-properties-section"></a>BevelMaterialType Cell (Bevel Properties Section)

Определяет тип материала, из которого состоит Багетная рамка. 
  
|**Описание**|**Значение**|
|:-----|:-----|
|нуль  <br/> |Нет специальных материалов  <br/> |
|1,1  <br/> |Матовая  <br/> |
|2  <br/> |Теплый матовый  <br/> |
|4  <br/> |PPGA  <br/> |
|4   <br/> |Системы  <br/> |
|5   <br/> |Темный край  <br/> |
|6   <br/> |Мягкая пограничная  <br/> |
|7   <br/> |Обычно  <br/> |
|8   <br/> |Каркасная  <br/> |
|9   <br/> |Туманно  <br/> |
|10   <br/> |Полупрозрачный порошок  <br/> |
|11   <br/> |Очистка  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку **BevelMaterialType** по имени из другой формулы, по значению атрибута **N** элемента **ячейки** или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | BevelMaterialType  <br/> |
   
Чтобы получить ссылку на ячейку **BevelMaterialType** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**висровбевелпропертиес** <br/> |
| Индекс ячейки:  <br/> |**висбевелматериалтипе** <br/> |
