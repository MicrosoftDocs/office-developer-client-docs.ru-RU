# <a name="replacelockformat-cell-change-shape-behavior-section"></a>ReplaceLockFormat Cell (Change Shape Behavior Section)

Указывает, переоформили ли значения указанных клеток в мастер-фигуре значения (в том числе локальные) фигуры, заменяемой во время операции замены фигуры. Если **ячейка ReplaceLockFormat** мастер-фигуры установлена для TRUE (1), значения форматирования мастера переоформят все соответствующие значения фигуры, заменяемой мастером. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Если **ячейка ReplaceLockFormat** в магистратуре настроена на ЗНАЧЕНИЕ TRUE, значения форматирования мастера переоформят все соответствующие значения фигуры, заменяемой мастером.  <br/> |
|FALSE  <br/> |Если **ячейка ReplaceLockFormat** мастерской формы настроена на FALSE, форма замены содержит локальные значения форматирования из старой формы после операции замены.  <br/> |
   
## <a name="remarks"></a>Примечания

Ячейка **ReplaceLockFormat** определяет, переопределяет ли в ходе операции замены фигуры значения локального форматирования ячеек в следующих разделах: 
  
- **Раздел Fill Format** 
    
- **Раздел Line Format** 
    
- **Раздел Быстрый стиль** 
    
- **Раздел Свойства темы** 
    
- **Раздел Gradient Properties** 
    
- **Раздел Bevel Properties** 
    
- **Раздел Дополнительные свойства эффекта** 
    
- **Раздел Остановки строки Градиент** 
    
- **Заполните раздел Остановки Градиента** 
    
Чтобы получить ссылку на ячейку **ReplaceLockFormat** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ReplaceLockFormat  <br/> |
   
Чтобы получить ссылку на ячейку **ReplaceLockFormat** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowReplaceBehaviors** <br/> |
| Индекс ячейки:  <br/> |**visReplaceLockFormat** <br/> |
