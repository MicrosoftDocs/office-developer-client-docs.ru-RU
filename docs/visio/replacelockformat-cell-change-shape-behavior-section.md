# <a name="replacelockformat-cell-change-shape-behavior-section"></a>Ячейка ReplaceLockFormat (раздел "Изменение поведения фигуры")

Указывает ли значений указанного ячеек в основной фигуры перезаписи значений (в том числе локального значения) во время операции замены фигуры в процессе замены фигуры. Если ячейка **ReplaceLockFormat** основной фигуры задано значение TRUE (1), форматирования значения хозяином перезаписать все соответствующие значения заменяется основной фигуры. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Если ячейка **ReplaceLockFormat** основной фигуры задано значение TRUE, форматирования значения хозяином перезаписать все соответствующие значения заменяется основной фигуры.  <br/> |
|FALSE  <br/> |Если ячейка **ReplaceLockFormat** основной фигуры задано значение FALSE, фигуры замещения содержит локального форматирования значения из старого фигуры после выполнения операции замены.  <br/> |
   
## <a name="remarks"></a>Замечания

Ячейки **ReplaceLockFormat** определяет ли образца фигуры перезаписывает во время операции замены фигуры локального форматирования значений ячеек в следующих разделах: 
  
- **Заполните поля формат** раздела 
    
- Раздел **Формат строки** 
    
- Раздел **Экспресс-стилей** 
    
- Раздел **Свойства темы** 
    
- В разделе **Градиента свойств** 
    
- В разделе **Свойств рельефов** 
    
- В разделе **Дополнительных свойств влияние** 
    
- Раздел **Строки градиента** 
    
- **Заполните поля градиента** раздел 
    
Для получения ссылки на ячейки **ReplaceLockFormat** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | ReplaceLockFormat  <br/> |
   
Для получения ссылки на ячейки **ReplaceLockFormat** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowReplaceBehaviors** <br/> |
| Индекс ячейки:  <br/> |**visReplaceLockFormat** <br/> |
