# <a name="replacelockformat-cell-change-shape-behavior-section"></a>ReplaceLockFormat Cell (Change Shape Behavior Section)

Указывает, перезаписывает ли значения заданных ячеек в главной фигуре значения (включая локальные значения) фигуры, которые заменяются во время операции замены фигуры. Если ячейка **ReplaceLockFormat** главной фигуры имеет значение true (1), то значения форматирования образца перезапишу все соответствующие значения фигуры, замещенной образцом. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Если ячейка **ReplaceLockFormat** главной фигуры имеет значение true, то значения форматирования образца перезапишу все соответствующие значения фигуры, замещенной образцом.  <br/> |
|FALSE  <br/> |Если ячейка **ReplaceLockFormat** главной фигуры имеет значение false, фигура замещения содержит локальные значения форматирования из старой фигуры после операции замены.  <br/> |
   
## <a name="remarks"></a>Примечания

Ячейка **ReplaceLockFormat** определяет, перезаписывает ли Главная фигура локальные значения форматирования ячеек в следующих разделах во время операции замены фигуры: 
  
- Раздел " **Формат заливки** " 
    
- Раздел " **Формат линий** " 
    
- Раздел " **экспресс — стиль** " 
    
- Раздел " **свойства темы** " 
    
- Раздел " **Свойства градиента** " 
    
- Раздел " **Свойства багетНой рамки** " 
    
- Раздел " **Дополнительные свойства эффекты** " 
    
- Раздел " **позиции градиента линий** " 
    
- Раздел " **позиции градиента заполнения** " 
    
Чтобы получить ссылку на ячейку **ReplaceLockFormat** по имени из другой формулы, по значению атрибута **N** элемента **ячейки** или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ReplaceLockFormat  <br/> |
   
Чтобы получить ссылку на ячейку **ReplaceLockFormat** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**Висровреплацебехавиорс** <br/> |
| Индекс ячейки:  <br/> |**Висреплацелоккформат** <br/> |
