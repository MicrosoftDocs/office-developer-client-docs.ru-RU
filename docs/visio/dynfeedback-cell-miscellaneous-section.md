# <a name="dynfeedback-cell-miscellaneous-section"></a>Ячейка DynFeedback (раздел "Прочее")

Изменяет тип визуальной обратной связи, предоставляемых пользователям при перетаскивании соединительной линии. При отпускании кнопки мыши, этот параметр не повлияет получившуюся форму соединительной линии. Этот параметр не применяется к маршрутизируемые соединители.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| 0  <br/> | Остается с текущими настройками прямых (не конечностей).  <br/> |**visDynFBDefault** <br/> |
| 1  <br/> | Отображает три конечностей при перетаскивании.  <br/> |**visDynFBUCon3Leg** <br/> |
| 2  <br/> | Показывает пять конечностей при перетаскивании.  <br/> |**visDynFBUCon5Leg** <br/> |
   
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку DynFeedback по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | DynFeedback  <br/> |
   
Для получения ссылки на ячейки DynFeedback по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowMisc** <br/> |
| Индекс ячейки:  <br/> |**visDynFeedback** <br/> |
