# <a name="dynfeedback-cell-miscellaneous-section"></a>DynFeedback Cell (Miscellaneous Section)

Изменяет тип визуальной обратной связи, предоставляемой пользователям при перетаскивания соединители. Когда кнопка мыши отпущена, на итоговую фигуру соединитела этот параметр не влияет. Этот параметр не применяется к соединитетелям, которые можно маршрутить.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| 0  <br/> | Остается прямым (без ногой).  <br/> |**visDynFBDefault** <br/> |
| 1   <br/> | Показывает три ногой при перетаскивание.  <br/> |**visDynFBUCon3Leg** <br/> |
| 2   <br/> | Показывает пять мужек при перетаскивание.  <br/> |**visDynFBUCon5Leg** <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку DynFeedback по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | DynFeedback  <br/> |
   
Чтобы получить ссылку на ячейку DynFeedback по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowMisc** <br/> |
| Индекс ячейки:  <br/> |**visDynFeedback** <br/> |
