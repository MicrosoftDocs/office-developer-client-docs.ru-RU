# <a name="cell-element-relellipticalarcto-row-visio-xml"></a>Элемент ячейки (строка RelEllipticalArcTo) ('Visio XML»)

Содержит или y координаты x конечной точки руководство пользователя относительно ширину и высоту фигуры, x и y координат элемента управления указывает на дугу относительно ширины и высоты, угол оси основные эллипса или отношение между фигуры основной и дополнительной осей эллипса.

|Да  <br/> |Координата конца вершины на дугу высоте фигуры.  <br/> |[Строка RelEllipticalArcTo (раздел "Геометрия")](relellipticalarcto-row-geometry-section.md) <br/> |
|A  <br/> |Точка управления дуги по оси x относительно ширины фигуры; точка дуги.  <br/> |[Строка RelEllipticalArcTo (раздел "Геометрия")](relellipticalarcto-row-geometry-section.md) <br/> |
|B  <br/> |Координата элемента управления arc точка относительно ширины фигуры.  <br/> |[Строка RelEllipticalArcTo (раздел "Геометрия")](relellipticalarcto-row-geometry-section.md) <br/> |
|C  <br/> |Угол arc главных осей относительно оси x родительского элемента.  <br/> |[Строка RelEllipticalArcTo (раздел "Геометрия")](relellipticalarcto-row-geometry-section.md) <br/> |
