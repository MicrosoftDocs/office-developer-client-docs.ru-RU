# <a name="index-function"></a>Функция INDEX

Возвращает подстроки с отсчетом от нуля расположение _индекса_ в _списке_ _разделителем_. Или, если индекс находится вне диапазона, возвращает пустую строку или маркер необязательно, представленные в качестве аргумента *errorvalue* . 
  
## <a name="syntax"></a>Синтаксис

Индекс (** *индекса* **, "** *списка* **" [, [** *разделитель* **] [, [** *errorvalue* **]]]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _index_ <br/> |Обязательный  <br/> |**Число** <br/> |Расположение, которое требуется найти.  <br/> |
| _списки_ <br/> |Обязательный  <br/> |**Строка** <br/> |Список, в котором требуется выполнить поиск.  <br/> |
| _delimiter_ <br/> |Optional  <br/> |**Строка** <br/> | Строка для использования в качестве разделителя в _списке_. Строка _разделителя_ может быть несколько символов в длину и включать многобайтовых символов. Значение по умолчанию — точка с запятой.  <br/> |
| _ERRORVALUE_ <br/> |Optional  <br/> |**Число** <br/> | Определенные пользователем значения для возврата, если индекс находится вне диапазона. По умолчанию используется пустая строка.  <br/> |
   
## <a name="remarks"></a>Замечания

Если список начинается или заканчивается на разделителя, предполагается, что строка значение null, существующих до или после списка. Последовательные разделители между подразумевает пустая строка. 
  
Если индекс находится вне диапазона, Visio возвращает пустую строку или необязательный маркер, предоставленный в качестве аргумента *errorvalue* . 
  
## <a name="example-1"></a>Пример 1

Индекс (3, "кошка; rat; goat»)
  
Возвращает «goat».
  
## <a name="example-2"></a>Пример 2

INDEX(54,";1;2;3;",,"ERROR")
  
Возврат значения «Ошибка».
