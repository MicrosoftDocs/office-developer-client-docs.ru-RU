# <a name="index-function"></a>Функция INDEX

Возвращает подстроку в индексе расположения  на  основе нуля в списке, относя к списку с помощью _делегировок._ Или, если индекс находится вне диапазона, возвращает пустую строку или необязательный маркер, предоставленный в качестве *аргумента errorvalue.* 
  
## <a name="syntax"></a>Синтаксис

INDEX(** *index* **," ** *list* ** "[,[ ** *delimiter* ** ][,[ ** *errorvalue* ** ]]]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _индекс_ <br/> |Обязательна  <br/> |**Number** <br/> |Расположение, которое вы хотите найти.  <br/> |
| _списке_ <br/> |Обязательно  <br/> |**Строка** <br/> |Список, в котором нужно найти.  <br/> |
| _delimiter_ <br/> |Необязательна  <br/> |**Строка** <br/> | Строка, используемая в качестве делегировки в _списке._ Строка  _с делегировкой_ может иметь длину более одного символа и включать многобайтные символы. Значение по умолчанию — 1 с зачетом.  <br/> |
| _errorvalue_ <br/> |Необязательна  <br/> |**Number** <br/> | Указанное пользователем значение, возвращаемого, если индекс находится вне диапазона. Значение по умолчанию — пустая строка.  <br/> |
   
## <a name="remarks"></a>Примечания

Если список начинается или заканчивается с помощью разбиений, предполагается, что строка null существует до или после списка. Последовательные заметивы подразумевают нулевую строку между ними. 
  
Если индекс находится вне диапазона, Visio возвращает пустую строку или необязательный маркер, предоставленный в качестве *аргумента errorvalue.* 
  
## <a name="example-1"></a>Пример 1

INDEX(3,"cat;cat;; ")
  
Возвращает "ветвь".
  
## <a name="example-2"></a>Пример 2

INDEX(54,";1;2;3;","ERROR")
  
Возвращает "ERROR".
