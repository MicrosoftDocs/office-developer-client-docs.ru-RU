# <a name="brightness-cell-image-properties-section"></a>Ячейка Brightness (раздел "Свойства изображения")

Регулировка яркости точечного рисунка. Увеличить яркость изображения, указав значение от 0 до 49%, или яркость, укажите значение от 51 до 100%. Значение по умолчанию: 50%.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки яркость по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Яркость  <br/> |
   
Для получения ссылки на ячейки яркость по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowImage** <br/> |
| Индекс ячейки:  <br/> |**visImageBrightness** <br/> |
