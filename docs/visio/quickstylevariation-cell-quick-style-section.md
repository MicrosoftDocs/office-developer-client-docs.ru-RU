# <a name="quickstylevariation-cell-quick-style-section"></a>Ячейка QuickStyleVariation (раздел "Экспресс-стиль")

Определяет, следует ли переопределить формулы и значения текста, строки и заливки цвет (или комбинацию этих свойств) с помощью цвета, контрастность с фон диаграммы. Значение — побитовое или 0, 1, 2, 4 и 8.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0  <br/> |Не изменяйте текстом фигуры, строки, или заливки цвет (или любое сочетание этих свойств) видимыми от цвета заданного фона темы.  <br/> |
|1  <br/> |Не изменяйте текстом фигуры, строки, или заливки цвет (или любое сочетание этих свойств) видимыми от цвета заданного фона темы.  <br/> |
|2  <br/> |Измените цвет текста фигуры, при необходимости видимыми от темы присваивает цвет фона.  <br/> |
|4  <br/> |Измените цвет линии фигуры, при необходимости видимыми от темы присваивает цвет фона.  <br/> |
|8  <br/> |Изменение цвета заливки фигуры, при необходимости видимыми от темы присваивает цвет фона.  <br/> |
   
## <a name="remarks"></a>Замечания

Используйте ячейку QuickStyleVariation, чтобы гарантировать видимости в текст или строки, когда они находятся вне любого геометрии видимым фигуры (например, в фигуру, чьи textbox является ниже в нижней части фигуры, некоторые из которых схем сети). Значение ячейки по умолчанию — 0, что означает, что его поведение неактивна. Любое другое значение запускает поведение ячейки.
  
Значение QuickStyleVariation переопределяет значение, созданное функцию THEMEVAL, когда он располагается в цвет (раздел символ), FillForegnd или цвет линии ячеек (или созданные средством прямые ссылки на эти три свойства с помощью THEMEVAL (» [[[CharacterColor»), THEMEVAL("FillColor") и THEMEVAL("LineColor")).
  
Для получения ссылки на ячейки **QuickStyleVariation** по имени из другой формулы, получив значение атрибута **N** элемента **ячейки** или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |QuickStyleVariation  <br/> |
   
Для получения ссылки на ячейки **QuickStyleVariation** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowQuickStyleProperties** <br/> |
|Индекс ячейки:  <br/> |**visQuickStyleVariation** <br/> |
