# <a name="paragraph-section"></a>Paragraph Section

Показывает атрибуты форматирования абзаца для текста фигуры, такие как отступы, интервалы строк, маркеры и горизонтальное выравнивание абзацев.
  
## <a name="remarks"></a>Примечания

Форматы абзацев можно установить, сделав запись в ячейке раздела **Paragraph,** применив текстовый стиль или  введите значения в диалоговом  окне "Текст" (на вкладке "Главная" щелкните диалоговое окно запуска в группе шрифтов).  
  
Раздел **"Абзац"** не включает ячейки для параметров вкладок. Чтобы установить вкладки для текста фигуры, используйте диалоговое окно **"Текст".**
