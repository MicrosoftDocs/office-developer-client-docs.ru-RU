# <a name="paragraph-section"></a>Paragraph Section

Показаны атрибуты форматирования абзаца для текста фигуры, такие как вмятины, интервалы строк, пули и горизонтальное выравнивание абзацев.
  
## <a name="remarks"></a>Примечания

Форматы абзаца можно установить, сделав запись в ячейке раздела **Абзац,** применяя текстовый стиль или введите значения в диалоговом окне **Text** (на вкладке **Home** нажмите кнопку запуска диалогового окна в группе **Font).** 
  
В **разделе Paragraph** не содержатся ячейки для параметров вкладок. Чтобы установить вкладки для текста фигуры, используйте диалоговое окно **Text.**
