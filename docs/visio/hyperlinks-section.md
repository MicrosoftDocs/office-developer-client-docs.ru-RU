# <a name="hyperlinks-section"></a>Раздел "Гиперссылки"

Содержит ячейки для создания различных переходов между фигурой или нарисовать страницы и другую страницу рисунка, другой файл или веб-сайта.
  
## <a name="remarks"></a>Замечания

Определение гиперссылки, введите значения в ячейках в этом разделе, или используйте диалоговое окно **гиперссылки** (щелкните **гиперссылку** на вкладке **Вставка** ). 
  
Можно добавить раздел гиперссылки на фигуры, группы, страницы или внешнего объекта в окне таблицы свойств фигуры.
