# <a name="hyperlinks-section"></a>Hyperlinks Section

Содержит ячейки для создания нескольких переходов между фигурой или страницей документа и другой странице документа, другим файлом или веб-сайтом.
  
## <a name="remarks"></a>Примечания

Чтобы определить гиперссылку, введите значения в ячейки этого раздела или **Используйте диалоговое окно гиперссылки** (нажмите кнопку **Гиперссылка** на вкладке **Вставка** ). 
  
Раздел "гиперссылки" можно добавить к фигуре, группе, странице или внешнему объекту в окне таблицы свойств фигуры.
