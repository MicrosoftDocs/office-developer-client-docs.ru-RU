# <a name="line-gradient-section"></a>Line Gradient Section

Показывает значения цвета для градиента строки, примененного к фигуре. 
  
## <a name="remarks"></a>Примечания

Каждая [строка стоп-строки Gradient (раздел Line Gradient)](gradient-stop-row-line-gradient-section.md) в разделе представляет остановку градиента в строке. Этот раздел может быть пустым. 
  
Только первые 10 строк, содержащиеся в разделе, используются для параметров градиента. Все **строки остановки Gradient** за 10 игнорируются. 
  
Свойства градиента строки можно установить в  разделе **Строка** области  Формы формата (на вкладке Главная, в группе **Стилей** фигур нажмите кнопку **Line** и нажмите кнопку **Параметры строки).** 
  
Вы можете включить или отключить цвет строки градиента для формы, изменив значение ячейки [LineGradientEnabled (Gradient Properties Section).](linegradientenabled-cell-gradient-properties-section.md)
