# <a name="line-gradient-section"></a>Раздел "Градиентная линия"

Отображаются значения цвета для градиента строки, применяемой к фигуре. 
  
## <a name="remarks"></a>Замечания

Каждая строка [градиента остановить строки (строка градиента раздел)](gradient-stop-row-line-gradient-section.md) в разделе представляет градиента в строке. В этом разделе может быть пустым. 
  
Только первые 10 строк, содержащихся в разделе используются параметры градиента. Все строки **Градиента остановить** за пределы 10 игнорируются. 
  
Строка градиента свойства задаются в **строке** области **Формат фигуры** (на вкладке **Главная** в группе **Стили фигур** щелкните **строку**и нажмите кнопку **Параметры строки**). 
  
Можно включить или отключить цвет градиента линии для фигуры, при изменении значения ячейки [LineGradientEnabled ячейки (градиента Properties Section)](linegradientenabled-cell-gradient-properties-section.md) .
