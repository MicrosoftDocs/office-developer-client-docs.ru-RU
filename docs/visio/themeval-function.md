# <a name="themeval-function"></a>Функция THEMEVAL

Извлекает значения из активной темы. 
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2013
 
  
## <a name="syntax"></a>Синтаксис

 **THEMEVAL**([ _"theme_value"_][, _default_]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _"theme_value"_ <br/> |Необязательна  <br/> |**Строка** <br/> |Имя ячейки в определении темы для получения значения.  <br/> |
| _default_ <br/> |Необязательна  <br/> |Разное  <br/> |Значение по умолчанию, если документ не является темой (определение темы не существует).  <br/> |
   
## <a name="remarks"></a>Примечания

Если функция **THEMEVAL** не получает аргументы, она возвращает значение темы ячейки хоста. Это значение хранится в определении текущей темы. Ячейка хоста должна иметь возможность использовать темы для возврата значения; если ячейка не может быть темой, **THEMEVAL** возвращает ошибку. 
  
Если функция **THEMEVAL** получает один аргумент, она получает значение из определения темы, переданного в качестве аргумента. Аргумент, переданный для первого параметра, должен быть числом или одной из точных строк, перечисленных в таблице ниже. 
  
Функция **THEMEVAL** также может принимать для первого параметра в качестве значения от 1 до 8. При использовании значений в качестве значений можно получить цвет по индексу из цветовой схемы темы. Таким образом, значение "1" возвращает цвет Dark из темы, "2" возвращает цвет Light, "3" возвращает цвет "Accent 1" и т. д. 
  
Если функция **THEMEVAL** получает два аргумента, она получает значение из определения темы, переданного в качестве первого аргумента. Однако если к документу не применена тема, функция **THEMEVAL** использует значение, указанное в качестве второго аргумента.

## <a name="example"></a>Пример

 `THEMEVAL("5")`
  
Возвращает цвет "Accent 3" из определения темы.
  
 `THEMEVAL("LineWeight", "0.7 pt.")`
  
Возвращает значение ячейки LineWeight из определения темы. Если к фигуре, содержащей эту функцию, не применена тема, функция возвращает "0,7 pt.".
