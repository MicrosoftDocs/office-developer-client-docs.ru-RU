# <a name="cell-element-relquadbezto-row-visio-xml"></a>Элемент ячейки (строка RelQuadBezTo) ('Visio XML»)

Содержит x или y координаты конечной точки кривая Безье относительно фигуры ширину и высоту или x и y координаты контрольной точки фигуры график относительно ширины и высоты.

</xs:element >

Атрибут **N** этого элемента **ячейки** должен быть один из ограниченный набор значений, которые соответствуют ячейки таблицы свойств фигуры. Обратитесь к таблице ниже для определения значений атрибутов **N** , разрешенных для этого элемента **ячейки** . 
  
|**Значение**|**Описание**|**Дополнительные сведения**|
|:-----|:-----|:-----|
|X   <br/> |X координата конца вершины кривая Безье ширине фигуры.  <br/> |[Строка RelQuadBezTo (раздел "Геометрия")](relquadbezto-row-geometry-section.md) <br/> |
|Да  <br/> |Координата конца вершины кривая Безье высоте фигуры.  <br/> |[Строка RelQuadBezTo (раздел "Геометрия")](relquadbezto-row-geometry-section.md) <br/> |
|A  <br/> |Точка управления график по оси x относительно ширины фигуры; точка дуги. Контрольная точка расположена лучше всего о пользователю между начала и окончания грани дуги.  <br/> |[Строка RelQuadBezTo (раздел "Геометрия")](relquadbezto-row-geometry-section.md) <br/> |
|B  <br/> |График элемента управления по оси y точки высоте фигуры.  <br/> |[Строка RelQuadBezTo (раздел "Геометрия")](relquadbezto-row-geometry-section.md) <br/> |
