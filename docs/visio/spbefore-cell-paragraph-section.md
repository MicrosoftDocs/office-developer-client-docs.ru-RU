# <a name="spbefore-cell-paragraph-section"></a>SpBefore Cell (Paragraph Section)

Определяет количество пробелов, вставленных перед каждым абзацем в блоке текста фигуры, в дополнение к любому пробелу из ячейки сплайна, если это первый абзац в текстовом блоке, ячейка TopMargin.
  
## <a name="remarks"></a>Примечания

Это значение не зависит от масштаба рисунка. Если масштаб документа изменяется, то параметр пробел до параметра остается прежним.
  
Чтобы получить ссылку на ячейку "до" по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Para. @ перед [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку CellsSRC по индексу из программы, используйте свойство **** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**Виссектионпараграф** <br/> |
| Индекс строки:  <br/> |**висровпараграф** +  *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**Висспацебефоре** <br/> |
