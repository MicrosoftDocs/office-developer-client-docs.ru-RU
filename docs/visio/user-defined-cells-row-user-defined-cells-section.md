# <a name="user-defined-cells-row-user-defined-cells-section"></a>User-defined Cells Row (User-defined Cells Section)

Содержит значение и описательный запрос для любых пользовательских ячеек в вашем решении. Фигура содержит одну строку пользовательских ячеек для каждой пары ячеек Value/Prompt, определяемой пользователем.
  
Строки пользовательских ячеек называются User. *имя*  и содержать следующие ячейки. Дополнительные сведения см. в определенных темах о ячейках. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[Значение](value-cell-user-defined-cells-section.md) <br/> |Указывает значение для соответствующей пользовательской ячейки.  <br/> |
|[Prompt](prompt-cell-user-defined-cells-section.md) <br/> |Указывает описательный запрос или комментарий для пользовательской ячейки.  <br/> |
   
## <a name="remarks"></a>Примечания

Пользовательские ячейки можно использовать для ввода формул или констант, на которые ссылается другая ячейка или надстройка. Значения в пользовательских ячейках переносимы, то есть если фигура, ссылаемая на пользовательную ячейку в одной фигуре, копируется в другую фигуру, которая не имеет той же пользовательской ячейки, ячейка добавляется в фигуру.
  
 Можно добавить максимальное количество пользователей.  *назначать*  строкам имена по мере необходимости, назначать осмысленные имена строкам и задавать значения ячеей. Чтобы добавить строку в существующий раздел "Пользовательские ячейки", щелкните правой кнопкой мыши строку и выберите пункт **"Вставка** строки" в ярлыковом меню. 
  
Вы можете ссылаться на эти ячейки по имени строки, которое отображается в окне таблицы фигур в красном тексте. Назначение осмысленных имен пользователю. *щелкните*  строку имени, а затем введите имя, например  *Offset*  , чтобы создать имя строки User.Offset. Затем можно ссылаться на ячейку Prompt с помощью User.Offset.Prompt. 
  
Введите уникальное имя строки в разделе.
