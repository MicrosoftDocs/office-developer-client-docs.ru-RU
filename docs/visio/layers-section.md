# <a name="layers-section"></a>Layers Section

Отображает все уровни, определенные для страницы, и свойства, задав для каждого слоя. 
  
## <a name="remarks"></a>Примечания

Этот раздел — свойство страницы, которое можно просмотреть только в окне ShapeSheet для страницы. Его ячейки соответствуют параметрам в диалоговом окне Layer  **Properties** (на вкладке **Главная,** в группе редактирования щелкните **Слои** и нажмите **кнопку Свойства слоя).**
