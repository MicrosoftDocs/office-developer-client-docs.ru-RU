# <a name="layers-section"></a>Раздел "Слои"

Отображает все слои, указанные для страницы и свойства для каждого уровня. 
  
## <a name="remarks"></a>Замечания

В этом разделе является свойством страницы, которое можно просмотреть только в окне таблицы свойств фигуры для страницы. Ячейки соответствуют параметрам в диалоговом окне **Свойства слоя** (на вкладке **Главная** в группе **Редактирование** щелкните **слои**и выберите пункт **Свойства Layer**).
