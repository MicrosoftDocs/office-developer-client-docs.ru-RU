# <a name="layers-section"></a>Layers Section

Показывает все уровни, определенные для страницы, и свойства, задав для каждого уровня. 
  
## <a name="remarks"></a>Примечания

Этот раздел является свойством страницы, которое можно просмотреть только в окне таблицы фигур для страницы. Его ячейки соответствуют параметрам в диалоговом  **окне "Свойства** слоя" (на вкладке "Главная" в группе редактирования щелкните "Слои" и выберите "Свойства **уровня").**
