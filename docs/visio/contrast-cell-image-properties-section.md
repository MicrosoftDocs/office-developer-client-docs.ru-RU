# <a name="contrast-cell-image-properties-section"></a>Ячейка Contrast (раздел "Свойства изображения")

Регулировка контрастность точечного рисунка. Уменьшить контрастность изображения, указав значение от 0 до 49% или увеличить контрастность, укажите значение от 51 до 100%. Значение по умолчанию: 50%.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки контрастности по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Контрастности  <br/> |
   
Для получения ссылки на ячейки контрастности по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowImage** <br/> |
| Индекс ячейки:  <br/> |**visImageContrast** <br/> |
