# <a name="iserr-function"></a>Функция ISERR

Возвращает значение TRUE, если значение _ссылка_на_ячейку_ имеет любой тип ошибки, кроме # н/д; в противном случае возвращает значение FALSE. Функция ЕОШ используется в формулах, которые ссылаются на другую ячейку. 
  
## <a name="syntax"></a>Синтаксис

ЕОШ (** *ссылка_на_ячейку* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _ссылка_на_ячейку_ <br/> |Обязательный  <br/> |**Строка** <br/> |Ссылка на ячейку.  <br/> |
   
## <a name="example-1"></a>Пример 1

|**Cell**|**Формулы**|**Возвращаемое значение**|
|:-----|:-----|:-----|
|Scratch.A1  <br/> |= (НД)  <br/> |# Н/Д!  <br/> |
|Scratch.B1  <br/> |=ISERR(Scratch.a1)  <br/> |FALSE  <br/> |
   
Возвращает значение FALSE, так как # н/д! Ошибка не распознается Функция ЕОШ. Чтобы найти все типы ошибок используйте ЕОШИБКА.
  
## <a name="example-2"></a>Пример 2

|**Cell**|**Формулы**|**Возвращаемое значение**|
|:-----|:-----|:-----|
|Scratch.X1  <br/> |= «Дом»  <br/> |#VALUE!  <br/> |
|Scratch.A1  <br/> |=ISERR(Scratch.X1)  <br/> |TRUE  <br/> |
   
Возвращает значение TRUE, так как #VALUE! Ошибка распознается Функция ЕОШ.
