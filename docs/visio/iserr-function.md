# <a name="iserr-function"></a>Функция ISERR

Возвращает значение TRUE, если значение _целлреференце_ имеет любой тип ошибки, кроме #N/a; в противном случае возвращает значение FALSE. Функция ЕОШ используется в формулах, которые ссылаются на другую ячейку. 
  
## <a name="syntax"></a>Синтаксис

ЕОШ (* * *целлреференце* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _целлреференце_ <br/> |Обязательный  <br/> |**String** <br/> |Ссылка на ячейку.  <br/> |
   
## <a name="example-1"></a>Пример 1

|**Cell**|**Формула**|**Возвращаемое значение**|
|:-----|:-----|:-----|
|". A1"  <br/> |= НД ()  <br/> |#N/A!  <br/> |
|"С нуля".  <br/> |= ЕОШ ("a. a1")  <br/> |FALSE  <br/> |
   
Возвращает значение FALSE, так как #N/A! ошибка не распознается функцией ЕОШ. Используйте ЕОШИБКА, чтобы найти все типы ошибок.
  
## <a name="example-2"></a>Пример 2

|**Cell**|**Формула**|**Возвращаемое значение**|
|:-----|:-----|:-----|
|"С начала. x1"  <br/> |= "Дом"  <br/> |#ЗНАЧ!  <br/> |
|". A1"  <br/> |= ЕОШ ("a. x1")  <br/> |TRUE  <br/> |
   
Возвращает значение TRUE, так как #VALUE! Ошибка распознаваться функцией ЕОШ.
