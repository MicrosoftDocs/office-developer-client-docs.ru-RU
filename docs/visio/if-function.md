# <a name="if-function"></a>Функция IF

Возвращает _валуеифтруе_ , если _ЛОГИКАЛЕКСПРЕССИОН_ имеет значение true. В противном случае возвращается _валуеиффалсе_.
  
## <a name="syntax"></a>Синтаксис

IF (* * *логикалекспрессион* * *, * * *валуеифтруе* * *, * * *валуеиффалсе* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _логикалекспрессион_ <br/> |Обязательный  <br/> |**String** <br/> |Вычисляемое выражение.  <br/> |
| _валуеифтруе_ <br/> |Обязательный  <br/> |**Разные** <br/> |Значение, возвращаемое, если _логикалекспрессион_ имеет значение true.  <br/> |
| _валуеиффалсе_ <br/> |Обязательный  <br/> |**Разные** <br/> | Значение, возвращаемое, если _логикалекспрессион_ имеет значение false.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Разные
  
## <a name="example"></a>Пример

Если (высота \> 1,25 в, 5, 7)
  
Возвращает 5, если высота фигуры больше 1,25 дюйма. Возвращает значение 7, если высота фигуры меньше или равна 1,25 сантиметра.
