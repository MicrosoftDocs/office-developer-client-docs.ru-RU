# <a name="modulus-function"></a>Функции МОДУЛЯ

Возвращает остаток от деления, полученные при делении числа на делителя.
  
## <a name="syntax"></a>Синтаксис

Операции с МОДУЛЯМИ (** *номер* **, ** *делителя* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Обязательный  <br/> |**Число** <br/> |Дивидендов.  <br/> |
| _делителя_ <br/> |Обязательный  <br/> |**Число** <br/> |Делителя.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Число
  
## <a name="remarks"></a>Замечания

Результат имеет тот же знак в качестве делителя. #DIV/0! Если делителя равно 0, возвращается ошибка. 
  
В большинстве случаев функции МОДУЛЯ можно использовать вместо функция МОД. 
  
## <a name="example-1"></a>Пример 1

ОПЕРАЦИИ С МОДУЛЯМИ (5, 1.4)
  
Возвращает 0,8.
  
## <a name="example-2"></a>Пример 2

ОПЕРАЦИИ С МОДУЛЯМИ (5,-1.4)
  
Возвращает-0.6.
  
## <a name="example-3"></a>Пример 3

ОПЕРАЦИИ С МОДУЛЯМИ (-5, 1.4)
  
Возвращает 0,6.
  
## <a name="example-4"></a>Пример 4

ОПЕРАЦИИ С МОДУЛЯМИ (-5,-1.4)
  
Возвращает-0.8.
