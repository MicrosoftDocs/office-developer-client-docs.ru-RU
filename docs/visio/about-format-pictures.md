# <a name="about-format-pictures"></a>Сведения о форматировании изображений

Формат рисунка используется для определения способа отображения значение. Например можно контролировать количество цифр справа или слева от десятичной запятой или ли текстовая строка отображается как прописные или строчные буквы.
  
> [!NOTE]
> Определение формата даты и времени с помощью форматирования системы Microsoft Office, заключите формат рисунка в двойные фигурные скобки, например, «{{m/d/гг}}». При использовании предварительно определенный формат, например 201, заключите его в фигурные скобки и угловые скобки, следующим образом: «{\<201\>}» 
  
В следующих разделах приведены символы, которые можно использовать для форматирования значения для отображения различных типов.
  
## <a name="string-and-numeric-values"></a>Строковые и числовые значения

|:-----|:-----|

Если больше цифр, чем заполнители справа от десятичной запятой, дроби округляется до номера заполнители. Для измерения Если заполнителем является самые левые цифры компоненты, которые являются 0 не отображаются.  <br/> Например ФОРМАТ (0 in 11,25 ft, "#. ##u") отображает 11.25 in.  <br/> |
|0  <br/> |Шаблон цифры (0). Отображает цифру или ничего. Начальные и конечные нули отображаются. Если больше цифр, чем заполнители слева от десятичной запятой, отображаются все цифры. Если больше цифр, чем заполнители справа от десятичной запятой, дроби округляется до номера заполнители. Для измерения отображаются компоненты, которые являются 0.  <br/> Например, ФОРМАТ (2ft 11.33 in "0. ## u") отображает 2 Форт 11.33 in.  <br/> |
|.  <br/> |Десятичный разделитель. Определяет количество цифр, отображаемых слева и справа от десятичного положение. Отображает десятичных цифр, определенное для системы **язык и региональные** параметры (панель управления).  <br/> Например, ФОРМАТ (250 cm "0.000 u") отображает 250.000 см.  <br/> |
|,  <br/> |Тысячи разделителя групп разрядов. Если окружены цифр (# или 0), разделитель отделяет тысячи от сотен в числе, имеющем четыре или более цифр слева от десятичной запятой. Вставляет такое же число пробелов перед меткой, прежде чем заполнитель.  <br/> Например ФОРМАТ (12 c 13 d, "#u") отображает 13 c 1.  <br/> |

Для японского языка отображает Gengo года в виде двух цифр арабские числа с нулем в начале при необходимости. Для корейского языка корейского год отображается как цифры арабский из четырех цифр.  <br/> |
