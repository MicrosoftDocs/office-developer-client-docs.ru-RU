# <a name="about-format-pictures"></a>О форматных картинках

Для определения отображения значения используется изображение формата. Например, можно контролировать количество цифр, отображаемых справа или слева от десятичной точки, или текстовая строка отображается в виде верхнего или нижнего шкафа.
  
> [!NOTE]
> Чтобы определить картину формата даты или времени с помощью Microsoft Office форматирования, заключите изображение формата в двойные фигурные скобки, например "{m/d/yy}}". Если используется заранее заранее заданной формат, например, 201, заключите его в фигурные скобки и угловые скобки, например: "{ \< 201 \> }" 
  
В следующих разделах показаны символы, которые можно использовать для формата различных типов значений для отображения.
  
## <a name="string-and-numeric-values"></a>Строковая и числовая значения

|**Символ**|**Описание**|
|:-----|:-----|
|#  <br/> |Заполнитель цифры. Отображает цифру или ничего. Ведущие и нулевые не отображаются. Если слева от десятичной области больше цифр, чем держателей, отображаются все цифры. Если справа от десятичной цифры больше цифр, чем у держателей, то фракция округляется до количества держателей. Для измерения, если местообладатель является самой левой цифрой, не отображаются подразделения, которые являются 0.  <br/> Например, FORMAT(0ft 11.25in,#.##u) отображает 11.25in.  <br/> |
|0  <br/> |Местообладатель цифры (ноль). Отображает цифру или ничего. Отображаются нулевые и ведущие нули. Если слева от десятичной области больше цифр, чем держателей, отображаются все цифры. Если справа от десятичной цифры больше цифр, чем у держателей, то фракция округляется до количества держателей. Для измерения отображаются подразделения с 0.  <br/> Например, FORMAT (2ft 11.33in,"0.##u") отображает 2 ft. 11.33 в.  <br/> |
|.  <br/> |Заполнитель десятичного разделителя. Определяет, сколько цифр отображается слева и справа от десятичной позиции. В многопартийной единице десятичной знак используется в самом маленьком (правом) подразделении. Отображает десятичной символ, определенный для  параметров региона и языка системы (панель управления).  <br/> Например, FORMAT (250 см,"0.000 u") отображает 250.000 см.  <br/> |
|,  <br/> |Тысячи сепараторов. Если в окружении значащего места (# или 0), разделитель отделяет тысячи от сотен в количестве, которое имеет четыре или более цифр слева от десятичной. Отображает тысячный сепаратор, определенный  для параметров региона и языка системы (Панель управления).  <br/> |
|E-E+ e-e+  <br/> |Экспоненциальный формат. Если формат содержит хотя бы один значок справа от этих символов, номер отображается в научном формате. Вставляет E или e между числом и его экспонентом. Для E+ или e+отображается знак плюс (+) перед положительными экспонентами и знак минус (-) перед отрицательными экспонентами. Для E-или e-отображает знак минус (-) только при отрицательном экспоненте.  <br/> Например, FORMAT(12345.67,"#####e+#") отображает 123.5e+2.  <br/> |
|u или U  <br/> |Короткий задаток метки. Вставляет сокращенные метки единиц после каждого подразделения. Например: in., ft., deg. Местообладатель U вставляет метки смешанного корпуса, а у - метки нижнего регистра. Вставляет то же количество пробелов перед меткой, что и перед держателем.  <br/> Например, FORMAT (12 c 13 d, "#u") отображает 13c1.  <br/> |
|uu или UU  <br/> |Длинное местоодатель меток. Вставляет метки единиц после каждого подразделения. Например: дюймовые, футы, градусы, которые вставляет метки смешанного корпуса, в то время как самообладатель вставляет метки нижнего регистра. |uuu или UUU  <br/> |Универсальный держатель меток. Вставляет универсальную (внутреннюю Visio) форму меток единиц после каждого подразделения. Местообладатель U вставляет метки смешанного корпуса, а у - метки нижнего регистра. Вставляет то же количество пробелов перед меткой, что и перед держателем.  <br/> |
|/  <br/> |Местообладатель фракции. Отображает выражение как целое число с фракцией, если присутствует ведущий держатель цифр. В противном случае отображается только весь номер в числителье. Если число следует за местообназначителям цифры в знаменателе, округите фракцию до ближайшей фракции, чья цифра 1, и упростит ее. Если в знаменателе указывается число без значимого задатщика, объехайте ближайшую фракцию, но это не упрощает.  <br/> Например, FORMAT (12.43,"# #/4") отображает 12 2/4.  <br/> |
|Пробел  <br/> |Отображает символ пространства в отформатированном выходе. Чтобы отобразить другой символ, используйте backslash \) (символ.  <br/> |

## <a name="text-values"></a>Текстовые значения

|**Символ**|**Описание**|
|:-----|:-----|
|\  <br/> |Отображает следующий символ как есть. Чтобы отобразить символ backslash, введите \\ . См. также текст.  <br/> |
|"text" или "text"  <br/> |Отображает текст, заключенный в кавычках, как есть. См. также \ (backslash).  <br/> |
|@  <br/> |Текстовое местоодатель. Заменяет строку, если значение выражения — строка.  <br/> Например, FORMAT ("Hello", "'You entered ('@')'" ) приводит к тому, что "Вы вошли (Hello)".  <br/> |
|@+  <br/> |Верхний задатчик текста. Для значений строк заменит входной вход верхним шкафом.  <br/> Например, FORMAT ("Hello", "@ @+ @-" ) приводит к "Hello HELLO hello)".  <br/> |
|@-  <br/> |Текстовое местоодатель. Для значений строк заменит входной ввод нижним регистром.  <br/> Например, FORMAT ("Hello", "@ @+ @-" ) приводит к "Hello HELLO hello)".  <br/> |
