# <a name="about-format-pictures"></a>О формате изображений

Изображение формата используется для определения отображения значения. Например, можно контролировать число цифр, отображаемого справа или слева от десятичной точки, или текстовую строку в верхнем или нижнем регистре.
  
> [!NOTE]
> Чтобы определить рисунок формата даты или времени с помощью Microsoft Office системного форматирования, заключите изображение формата в двойные фигурные скобки, например "{m/d/yy}}". Если используется предопределение формата, например 201, заключите его в фигурные скобки и угловые скобки, например: "{ \< 201 \> }" 
  
В следующих разделах показаны символы, которые можно использовать для формата различных типов значений для отображения.
  
## <a name="string-and-numeric-values"></a>Строки и числимые значения

|**Символ**|**Описание**|
|:-----|:-----|
|#  <br/> |Заполнитель цифры. Отображает цифру или ничего. Нули в первой и в окну не отображаются. Если слева от десятичной затметели больше цифр, отображаются все цифры. Если справа от десятичной заглухи больше цифр, чем замесел, дробная часть округляется до числа замесел. Если в измерении замещение является самой левой цифрой, подмногие части, которые являются 0, не отображаются.  <br/> Например, FORMAT(0ft 11.25in,"#.##u") отображает 11,25in.  <br/> |
|0  <br/> |Местозамлятель цифры (ноль). Отображает цифру или ничего. Отображаются нули в окнах. Если слева от десятичной затметели больше цифр, отображаются все цифры. Если справа от десятичной заглухи больше цифр, чем замесел, дробная часть округляется до числа замесел. Для измерения отображаются подгруппы, которые имеют 0.  <br/> Например, format(2ft 11.33in,"0.## u") отображает 2 ft. 11.33 in.  <br/> |
|.  <br/> |Заполнитель десятичного разделителя. Определяет, сколько цифр отображается слева и справа от десятичной заголовки. В многоэтапной единице десятичной замещение используется в наименьшем (правом) подмосковном подразделении. Отображает десятичной знак, определенный для  системных параметров языка и региона (панель управления).  <br/> Например, format(250 cm,"0.000 u") отображает 250.000 cm.  <br/> |
|,  <br/> |Сепаратор тысяч. Если разделитель находится в окружении цифр (# или 0), разделитель отделяет тысячи от сотен внутри числа, которое имеет четыре или более цифр слева от десятичной заготовки. Отображает группу групп, определяемую для  системных параметров языка и региона (панель управления).  <br/> |
|E- E+ e- e+  <br/> |Экспоненциальный формат. Если формат содержит по крайней мере один замессник цифр справа от этих символов, число отображается в формате". Вставляет E или e между числом и его экспонентом. Для E+ или e+отображает знак плюса (+) перед положительными экспонентами и знаком минуса (-) перед отрицательными экспонентами. Для E- или e-отображает знак минуса (-) только в том случае, если показатель exponent является отрицательным.  <br/> Например, format(12345.67,"###.#e+#") отображает 123.5e+2.  <br/> |
|u или U  <br/> |Краткий заметель меток. Вставляет сокращенные метки единиц после каждого подразделения. Например: in., ft., deg. Замессировщик U вставляет метки смешанного регистра, а замессировщик u вставляет подписи нижнего регистра. Вставляет то же количество пробелов перед меткой, что и до замещего.  <br/> Например, format(12 c 13 d,"#u") отображает 13c1.  <br/> |
|uu или UU  <br/> |Длинный замесерв метки. Вставляет метки единиц после каждого подразделения. Например, в дюймах, в дюймах и градусах вставьте метки смешанного регистра, а вставьте подписи нижнего регистра. Вставляет то же количество пробелов перед меткой, что и до замещего.  <br/> Например, format(12.43in,"# #/4 UU") отображает 12 2/4 INCHES.  <br/> |
|uuu или UUU  <br/> |Универсальный заместитель меток. Вставляет универсальную (внутреннюю) форму меток единиц после каждого подразделения. Замессировщик U вставляет метки смешанного регистра, а замессировщик u вставляет подписи нижнего регистра. Вставляет то же количество пробелов перед меткой, что и до замещего.  <br/> |
|/  <br/> |Дробный замещик. Отображает выражение как целое число с дробью, если присутствует замессатель цифр в первой части. В противном случае отображается только целое число в числовом номере. Если число следует за заметором цифры в деномилиторе, дробь округлается до ближайшей дроби, у которой числитель 1, и упрощает его. Если в деномилиаторе указано число без замещего числа, оно округит до ближайшей дроби, но не упростит его.  <br/> Например, format(12.43,"# #/4") отображает 12 2/4.  <br/> |
|Пробел  <br/> |Отображает пробел в отформатированные выходные данные. Чтобы отобразить другой символ, используйте знак backslash \) (знак backslash).  <br/> |

## <a name="text-values"></a>Текстовые значения

|**Символ**|**Описание**|
|:-----|:-----|
|\  <br/> |Отображает следующий символ как есть. Чтобы отобразить знак задней черты, введите \\ . См. также "text".  <br/> |
|"text" или "text"  <br/> |Отображает текст, заключенный в кавычках, как есть. См. также \ (backslash).  <br/> |
|@  <br/> |Замесятель текста. Заменяет строку, если значением выражения является строка.  <br/> Например, format("Hello", "You entered ('@')'" ) приводит к "You entered (Hello)".  <br/> |
|@+  <br/> |Замесятель текста в верхнем регистре. Для строки значений замените входные данные верхним регистром.  <br/> Например, format("Hello", "@ @+ @-" ) приводит к "Hello HELLO hello)".  <br/> |
|@-  <br/> |Замесятель текста. Для строки значений замените входные данные строчная буква.  <br/> Например, format("Hello", "@ @+ @-" ) приводит к "Hello HELLO hello)".  <br/> |
