# <a name="about-format-pictures"></a>Сведения о форматах изображений

Изображение формата используется для определения способа отображения значения. Например, можно управлять числом отображаемых цифр справа или слева от десятичной точки, а также будет ли текстовая строка заглавной или строчной.
  
> [!NOTE]
> Чтобы определить формат даты или времени с помощью форматирования системы Microsoft Office, заключите его в двойные фигурные скобки, например "{{m/d/гг}}". Если вы используете предварительно определенный формат, например 201, заключите его в фигурные скобки и угловые скобки, как показано ниже: "\<{\>201}" 
  
В следующих разделах показаны символы, которые можно использовать для форматирования различных типов значений для отображения.
  
## <a name="string-and-numeric-values"></a>Строковые и числовые значения

|**Символ**|**Описание**|
|:-----|:-----|
|#  <br/> |Заполнитель цифры. Отображает либо цифру, либо значение Nothing. Начальные и конечные нули не отображаются. Если слева от десятичной запятой больше цифр, чем заполнители, отображаются все цифры. Если справа от десятичного разделителя больше цифр, дробная часть округляется до количества заполнителей. Для измерения, если заполнителем является крайняя левая цифра, подединицы измерения 0 не отображаются.  <br/> Например, FORMAT (0ft 11.25 in, "#. # #u") отображает 11.25 в.  <br/> |
|нуль  <br/> |Заполнитель цифр (ноль). Отображает либо цифру, либо значение Nothing. Отображаются начальные и конечные нули. Если слева от десятичной запятой больше цифр, чем заполнители, отображаются все цифры. Если справа от десятичного разделителя больше цифр, дробная часть округляется до количества заполнителей. Для измерения отображаются подединицы, равные 0.  <br/> Например, в формате (2ft 11.33 in, "0. # # u") отображается 2 ФТ. 11,33.  <br/> |
|.  <br/> |Заполнитель десятичного разделителя. Определяет, сколько цифр отображается слева и справа от десятичного разделителя. В составном блоке десятичное значение используется в наименьшем (крайнем правом) подразделении. Отображает десятичный символ, определенный для языковых **и региональных** параметров системы (панель управления).  <br/> Например, в формате (250 см, "0.000 u") отображается 250,000 cm.  <br/> |
|,  <br/> |Разделитель групп разрядов. Если они заключены в заполнители цифр (# или 0), разделитель отделяет тысячи от сотен в числе с четырьмя или более цифр слева от десятичного разделителя. Отображает разделитель групп разрядов, определенный для **языковых и региональных** параметров системы (панель управления).  <br/> |
|E – e + E – e +  <br/> |Экспоненциальный формат. Если формат содержит по крайней мере один заполнитель цифр справа от этих символов, число отображается в экспоненциальном формате. Вставляет E или e между числом и его экспонентой. Для E + или E + отображается знак плюс (+) до положительных Экспонент и знак минус (-) до отрицательных показателей экспоненты. Для E-или e-, символ "минус" (-) отображается только в том случае, если экспонента отрицательна.  <br/> Например, в формате (12345.67, "# # #. #e + #") отображается 123.5 e + 2.  <br/> |
|u или U  <br/> |Короткий заполнитель метки. Вставляет сокращенные метки единиц измерения после каждой подединицы. Пример: в., ft., град. Заполнитель U вставляет смешанные метки, а заполнитель u вставляет метки в нижнем регистре. Вставляет то же количество пробелов перед меткой, что и заполнитель.  <br/> Например, в формате (12 c 13 d, "#u") отображается 13c1.  <br/> |
|UU или UU  <br/> |Длинный заполнитель метки. Вставляет метки единиц после каждой подединицы. Например: дюймы, футы, градусы. заполнитель U вставляет смешанные метки, в то время как заполнитель U вставляет метки нижнего регистра. Вставляет то же количество пробелов перед меткой, что и заполнитель.  <br/> Например, в формате (12.43 in, "# #/4 UU") отображается 12 2/4 дюймов.  <br/> |
|УУУ или ууу  <br/> |Универсальный заполнитель метки. Вставляет универсальную (внутреннюю по Visio) форму меток единиц измерения после каждого подединицы. Заполнитель U вставляет смешанные метки, а заполнитель u вставляет метки в нижнем регистре. Вставляет то же количество пробелов перед меткой, что и заполнитель.  <br/> |
|/  <br/> |Заполнитель дроби. Отображает выражение в виде целого числа с дробью при наличии заполнителя в начале цифры. В противном случае отображает только целое число в числителе. Если число соответствует заполнительу цифр в знаменателе, округляется дробь до ближайшей дроби, числитель которого равен 1, и упрощает его. Если число указывается в знаменателе без заполнителя цифры, округляется до ближайшей дроби, но не упрощает его.  <br/> Например, в формате (12.43, "# #/4") отображается 12 2/4.  <br/> |
|Пробел  <br/> |Отображает символ пробела в отформатированных выходных данных. Чтобы отобразить другой символ, используйте обратную косую черту (\) символ.  <br/> |

## <a name="text-values"></a>Текстовые значения

|**Символ**|**Описание**|
|:-----|:-----|
|\  <br/> |Отображает следующий знак, как есть. Чтобы отобразить символ обратной косой черты, введите \\. См. также "текст".  <br/> |
|"текст" или "текст"  <br/> |Отображает текст, заключенный в кавычки, как есть. См. также \ (обратная косая черта).  <br/> |
|@  <br/> |Заполнитель текста. Заменяет строку, если значением выражения является строка.  <br/> Например, в формате ("Hello", "'" введен ("@") "") результатом будет "вы ввели" (Hello) ".  <br/> |
|@+  <br/> |Замещающий текст в верхнем регистре. Для строковых значений заменяет входные данные на прописные.  <br/> Например, результат "Hello HELLO Hello" ("Привет", "@ @ + @-") приведет к результату.  <br/> |
|@-  <br/> |Заполнитель текста. Для строковых значений заменяются на ввод строчными буквами.  <br/> Например, результат "Hello HELLO Hello" ("Привет", "@ @ + @-") приведет к результату.  <br/> |
