# <a name="about-functions"></a>Сведения о функциях

Функция выполняет один четко определенной задачи. Большинство функций вступили число аргументов в качестве входных данных. Несмотря на то, что тип и количество аргументов зависят от функции, все функции имеют один и тот же общий синтаксис:
  
 **Функции (** _аргумент1_, _аргумент2_,...  _argumentN_ [, _argumentA_ |  _аргумент_ **])**
  
|**Синтаксис элемента**|**Описание**|
|:-----|:-----|
| ( )  <br/> | Если функция не принимает аргументы, он должен следовать пустой набор скобки ().  <br/> |
| ,  <br/> | Аргументы, разделенных запятыми.  <br/> |
| ...  <br/> | Используется для нотацию только; не включайте в функции.  <br/> |
| [ ]  <br/> | Необязательный аргумент. Используется для нотацию только; не включайте в функции.  <br/> |
| |  <br/> | Выбор; может включать _argumentA_ или _аргумент_. Используется для нотацию только; не включайте в функции.  <br/> |
   
Многие функции, которые можно использовать в формулах, которые похожи видно в электронных таблицах: математических, такие как SUM или корень; тригонометрических, например SIN или COS; или логическим, например, если нет. Другие функции являются уникальными для Microsoft Office Visio, такие как защита, ВЕСА и RUNADDON.
  
Дополнительные сведения о конкретных функций видеть этот справочник по таблице свойств фигуры.
  
> [!NOTE]
>  Определенные функции отображаются в формулах, созданных функцией Visio, но не отображаются в диалоговом окне **Изменить формулу** или описанные в этой справке, так как они зарезервированы для внутреннего использования и не должен использоваться в других формулах. Ниже приведены примеры: ELLIPSE_THETA, _ELLIPSE_ECC, _UCON_C1 и _SHAPEMIN.
