# <a name="about-functions"></a>О функциях

Функция выполняет одну четко определяемую задачу. Большинство функций принимают в качестве входных данных несколько аргументов. Хотя тип и число аргументов зависят от функции, все функции имеют одинаковый общий синтаксис:
  
 **FUNCTION(** _argument1_,  _argument2_, ...  _argumentN_ [, _argumentA_  |   _argument_ **])**
  
|**Элемент Syntax**|**Описание**|
|:-----|:-----|
| ( )  <br/> | Если функция не принимает аргументы, за ней должен следовать пустой набор скобок ( ).  <br/> |
| ,  <br/> | Аргументы разделяются запятой.  <br/> |
| ...  <br/> | Используется только для нотации; не включаем в функцию.  <br/> |
| [ ]  <br/> | Необязательный аргумент. Используется только для нотации; не включаем в функцию.  <br/> |
| |  <br/> | Выбор; можно включить  _аргументA или_  _аргумент_. Используется только для нотации; не включаем в функцию.  <br/> |
   
Многие функции, которые можно использовать в формулах, похожи на функции, которые вы видели в программах для работы с электронными таблицами: математические, такие как СУММ или SQRT; тригонометрия, например SIN или COS; или логический, например IF или NOT. Многие другие функции уникальны для Microsoft Office Visio, например GUARD, GRAVITY и RUNADDON.
  
Дополнительные сведения о конкретных функциях см. в этом справочнике по таблицам фигур.
  
> [!NOTE]
>  Некоторые функции отображаются в формулах, созданных Visio, но не отображаются в диалоговом окне "Изменить формулу" или описаны в этой ссылке, так как они зарезервированы для внутреннего использования и не должны использоваться в других формулах.  Ниже примеры: ELLIPSE_THETA, _ELLIPSE_ECC, _UCON_C1 и _SHAPEMIN.
