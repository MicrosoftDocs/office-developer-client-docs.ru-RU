# <a name="foreign-image-info-section"></a>Foreign Image Info Section

Содержит ширину и высоту объекта из другой программы, используемой в документе Microsoft Visio, и отображается в окне ShapeSheet только для таких объектов. Он также указывает расстояние, на которое смещено изображение объекта в пределах.
  
## <a name="remarks"></a>Примечания

В версиях продуктов Visio более ранней версии 2000 этот раздел назывался **"Сведения об образе".**
