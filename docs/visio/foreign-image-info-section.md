# <a name="foreign-image-info-section"></a>Foreign Image Info Section

Содержит ширину и высоту объекта из другой программы, используемой в документе Microsoft Visio, и отображается в окне таблицы свойств фигуры только для таких объектов. Он также указывает расстояние смещения изображения объекта в пределах границ.
  
## <a name="remarks"></a>Примечания

В версиях продуктов Visio, предшествующих версии 2000, этот раздел назывался **сведениями об изображении**.
