# <a name="foreign-image-info-section"></a>Раздел "Сведения о внешнем изображении"

Содержит ширину и высоту объекта из другой программы, используемые в документе Microsoft Visio и будет отображаться в окне таблицы свойств фигуры только для таких объектов. Также указывается расстояние смещения изображения объекта в границах.
  
## <a name="remarks"></a>Замечания

В версиях продуктов Visio до версии 2000 в этом разделе было с именем **Рисунка**.
