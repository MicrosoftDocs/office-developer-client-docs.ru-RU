# <a name="readonly-cell-actions-section"></a>ReadOnly Cell (Actions Section)

Определяет, является ли действие для тега действия или контекстного меню предназначенным только для чтения. 
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Действие отображается в меню, но доступно только для чтения.  <br/> |
|FALSE  <br/> |Действие отображается в меню и может быть выбрано (по умолчанию).  <br/> |
   
## <a name="remarks"></a>Примечания

Если действие доступно только для чтения, оно отображается в теге действия или в контекстном меню, но его нельзя выбрать. Он не затенен, а отображается на цветном фоне, например в метке. Чтобы элемент меню отображался серым цветом, используйте отключенную ячейку. 
  
Чтобы получить ссылку на ячейку ReadOnly по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Действия. *Name (имя* ). Действия Реадонливхере.  *Name* — имя строки действий  <br/> |
   
Чтобы получить ссылку на ячейку ReadOnly по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**Виссектионактион** <br/> |
|Индекс строки:  <br/> |**висровактион** +  *i* , где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**Висактионреадонли** <br/> |
