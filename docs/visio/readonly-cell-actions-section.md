# <a name="readonly-cell-actions-section"></a>Ячейка ReadOnly (раздел "Действия")

Определяет, является ли действие на тег или контекстное меню действий только для чтения. 
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий, называются смарт-тегов. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Действие отображается в меню, но только для чтения.  <br/> |
|FALSE  <br/> |Действие отображается в меню и может быть установлен (по умолчанию).  <br/> |
   
## <a name="remarks"></a>Замечания

При действие, которое доступно только для чтения, отображается в действие тег или контекстное меню, но не могут установить его. Он не недоступна, но вместо отображается на цветном фоне, как метки. Чтобы сделать элемент меню недоступны для изменения, используйте ячейку отключено. 
  
Чтобы получить ссылку на ячейку только для чтения по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Действия. *имя* . ReadOnlywhere действия.  *имя* — это имя строки действий  <br/> |
   
Для получения ссылки на ячейки только для чтения по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionAction** <br/> |
|Индекс строки:  <br/> |**visRowAction** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visActionReadOnly** <br/> |
