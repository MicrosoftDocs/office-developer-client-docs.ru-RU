# <a name="variationstyleindex-cell-theme-properties-section"></a>Ячейка VariationStyleIndex (Theme Properties Section)

Определяет стиль индекс вариант активной темы на странице, как целое число.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **VariationStyleIndex** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | VariationStyleIndex  <br/> |
   
Для получения ссылки на ячейки **VariationStyleIndex** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowThemeProperties** <br/> |
| Индекс ячейки:  <br/> |**visVariationStyleIndex** <br/> |
