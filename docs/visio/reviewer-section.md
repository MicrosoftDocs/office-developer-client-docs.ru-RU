# <a name="reviewer-section"></a>Reviewer Section

Содержит идентифицирующие сведения о каждом рецензенте документов.
  
## <a name="remarks"></a>Примечания

Раздел рецензента содержит по одной строке для каждого рецензента, который добавил разметку в документ, даже если удаляется вся разметка определенного рецензента. 
  
Раздел рецензента содержится в таблице shapesheet документа. Чтобы просмотреть таблицу фигур документа, [](run-in-developer-mode-display-the-developer-tab.md) на вкладке "Разработчик" в группе **"Показать/скрыть"** выберите обозреватель **документов,** щелкните правой кнопкой мыши имя документа и выберите "Показать таблицу **фигур".** 
  
Дополнительные сведения о комментариях, добавляемом в документ в составе разметки, см. в **разделе "Аннотация".**
