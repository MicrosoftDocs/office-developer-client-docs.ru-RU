# <a name="reviewer-section"></a>Reviewer Section

Содержит сведения о каждом рецензенте документов.
  
## <a name="remarks"></a>Примечания

Раздел рецензент содержит по одной строке для каждого рецензента, который добавил разметку в документ, даже если все разметки определенного рецензента удалены. 
  
Раздел рецензента содержится в таблице ShapeSheet документа. Чтобы просмотреть таблицу ShapeSheet [](run-in-developer-mode-display-the-developer-tab.md) документа на вкладке Разработчик в группе **Show/Hide,** выберите Обозреватель рисования **,** щелкните правой кнопкой мыши имя документа, а затем нажмите **кнопку Показать ShapeSheet**. 
  
Дополнительные сведения о комментариях, добавленных в документ в качестве части разметки, см. в разделе **Аннотация.**
