# <a name="reviewer-section"></a>Раздел "Рецензент"

Идентификационные сведения об оценке для каждого документа.
  
## <a name="remarks"></a>Замечания

В разделе редактор содержит одну строку для каждого рецензента, который добавил разметки в документ даже в том случае, если все разметку определенного рецензента удаляется. 
  
В разделе редактор содержится в ShapeSheet документа. Для просмотра документа фигуры на вкладке [Разработчик](run-in-developer-mode-display-the-developer-tab.md) в группе **Показать/скрыть** установите **Проводника по документам**, щелкните правой кнопкой мыши имя документа и нажмите кнопку **Показать таблицу свойств фигуры**. 
  
Дополнительные сведения о комментарии, вставленные в документ как часть разметки обратитесь к разделу **заметки** .
