# <a name="about-error-values"></a>Сведения о значениях ошибок

Значения ошибок отображаются в ячейках с неправильными формулами в ячейке.
  
Если формула ссылается на ячейку, содержащую значение ошибки, эта формула также отображает значение ошибки. Для поиска значений ошибок можно использовать функцию ЕОШ, ISERRNA, ЕОШИБКА или ISERRVALUE.
  
**Значения ошибок**

||||
|:-----|:-----|:-----|
|**Если в ячейке отображается** <br/> |**Формула включает** <br/> |**Пример** <br/> |
| #ДЕЛ/0!  <br/> |Деление на 0  <br/> |10/0  <br/> |
| #ЗНАЧ!  <br/> | Аргумент или операнд неправильного типа  <br/> | 5 + "дом"  <br/> |
| #ССЫЛКА!  <br/> | Ссылка на ячейку, которая не существует  <br/> | Ячейка, указывающая на другую ячейку, которая больше не существует  <br/> |
| #ЧИСЛО!  <br/> | Недопустимое число  <br/> | Квадратный корень отрицательного числа  <br/> |
| #N/A!  <br/> | Недоступное значение  <br/> | Функция НД ()  <br/> |
| #DIM!  <br/> | Значение измерения, которое превышает диапазон измерения (допустимые степени — целые числа — 128 \<= n \<= 127)  <br/> Значение измерения, используемое с недопустимой операцией  <br/> |1in ^ 100 \* 1in ^ 100 (результат равен 1in ^ 200, который находится за пределами диапазона измерения)  <br/> 5.2 cm ^ 1.5 (не целое число)  <br/> |
