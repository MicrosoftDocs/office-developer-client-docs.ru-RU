# <a name="about-error-values"></a>Сведения о значениях ошибок

Ошибка значения отображаются в ячейках, которые имеют неправильный формулы для этой ячейки.
  
Если формула ссылается на ячейку, содержащую значение ошибки, что формулы также отображается значение. Функция ЕОШ, ISERRNA, ЕОШИБКА или ISERRVALUE можно использовать для поиска ошибок.
  
**Значения ошибок**

||||
|:-----|:-----|:-----|
|**Если в ячейке отображается** <br/> |**Содержит формулу** <br/> |**Пример** <br/> |
| #DIV/0!  <br/> |Деление на 0  <br/> |10/0  <br/> |
| #VALUE!  <br/> | Аргумент или операнд недопустимого типа  <br/> | 5 + «Дом»  <br/> |
| #REF!  <br/> | Ссылка на ячейку, не существует  <br/> | Ячейки, на который ссылается на ячейку, который больше не существует  <br/> |
| #NUM!  <br/> | Неверный номер  <br/> | Квадратный корень из отрицательное значение  <br/> |
| # Н/Д!  <br/> | Не доступное значение  <br/> | Функция НД)  <br/> |
| #DIM!  <br/> | Измерения значение, превышающее диапазон измерений (допустимый степени являются целые числа от -128 \<= n \<= 127)  <br/> Измерения значение, используемое в неподходящие операции  <br/> |1 в ^ 100 \* 1 в ^ 100 (результатом будет 1 в ^ 200, который находится за пределами диапазона аналитики)  <br/> 5.2 cm ^ 1,5 (не power целое число)  <br/> |
