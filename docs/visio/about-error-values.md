# <a name="about-error-values"></a>О значениях ошибок

Значения ошибок отображаются в ячейках с неправильными формулами для этой ячейки.
  
Если формула ссылается на ячейку, содержаную значение ошибки, эта формула также отображает значение ошибки. Чтобы найти значения ошибок, можно использовать функции ISERR, ISERRNA, ISERROR или ISERRVALUE.
  
**Значения ошибок**

||||
|:-----|:-----|:-----|
|**Если ячейка отображает** <br/> |**Формула включает в себя** <br/> |**Пример** <br/> |
| #ДЕЛ/0!  <br/> |Разделение на 0  <br/> |10/0  <br/> |
| #ЗНАЧ!  <br/> | Аргумент или операнд неправильного типа  <br/> | 5 + "Дом"  <br/> |
| #ССЫЛКА!  <br/> | Ссылка на ячейку, которая не существует  <br/> | Ячейка, которая ссылается на другую ячейку, которая больше не существует  <br/> |
| #ЧИСЛО!  <br/> | Недействительный номер  <br/> | Квадратный корневой корень отрицательного числа  <br/> |
| #N/A!  <br/> | Не доступное значение  <br/> | ФУНКЦИЯ NA()  <br/> |
| #DIM!  <br/> | Размерное значение, которое превышает диапазон измерений (допустимые силы — это ряды -128 \< = n \< = 127)  <br/> Размерное значение, используемое при ненадлежащей операции  <br/> |1in^100 \* 1in^100 (результат 1in^200, который находится за пределами диапазона измерений)  <br/> 5.2cm^1.5 (не вменая мощность)  <br/> |
