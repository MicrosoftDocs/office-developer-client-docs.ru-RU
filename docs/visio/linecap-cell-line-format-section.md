# <a name="linecap-cell-line-format-section"></a>Ячейка LineCap (раздел "Формат линий")

Указывает, будет ли строки скругленные, квадратные или расширенные отрезков.
  
|**Значение**|**Стиль конца строки**|
|:-----|:-----|
|0  <br/> |Округленный  <br/> |
|1  <br/> |Квадратный  <br/> |
|2  <br/> |Долго  <br/> |
   
## <a name="remarks"></a>Замечания

Также можно задать значение данной ячейки в диалоговом окне " **строка** " (на вкладке **Главная** в группе **фигуру** щелкните **строку**, выберите пункт **стрелки**и нажмите кнопку **Дополнительно стрелки**).
  
Чтобы получить ссылку на ячейку LineCap по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |LineCap  <br/> |
   
Для получения ссылки на ячейки LineCap по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowLine** <br/> |
|Индекс ячейки:  <br/> |**visLineEndCap** <br/> |
