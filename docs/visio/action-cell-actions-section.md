# <a name="action-cell-actions-section"></a>Action Cell (Actions Section)

Содержит формулу, которая должна выполняться, когда пользователь выбирает команду в контекстном меню или меню тегов действий.
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
## <a name="remarks"></a>Замечания

Ячейка Action оценивается только при возникновении действия, а не при вводе формулы.
  
Чтобы получить ссылку на ячейку Action по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Действия.  *Name (имя* ). Действие, в котором выполняются действия. *Name* — имя строки действий  <br/> |
   
Чтобы получить ссылку на ячейку действия Сесе по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**Виссектионактион** <br/> |
| Индекс строки:  <br/> |**висровактион** +  *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**Висактионактион** <br/> |
