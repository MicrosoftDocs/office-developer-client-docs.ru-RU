# <a name="aligncenter-cell-alignment-section"></a>AlignCenter Cell (Alignment Section)

Определяет горизонтальное положение по отношению к источнику родительского вертикального руководства или точки направляния, к которой выравнивается горизонтальный центр фигуры.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку AlignCenter по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | AlignCenter  <br/> |
   
Чтобы получить ссылку на ячейку AlignCenter по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignCenter** <br/> |
