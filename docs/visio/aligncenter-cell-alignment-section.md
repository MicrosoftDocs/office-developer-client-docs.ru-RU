# <a name="aligncenter-cell-alignment-section"></a>Ячейка AlignCenter (раздел "Выравнивание")

Определяет позицию по горизонтали, по отношению к источнику родительского объекта вертикальной направляющей или направляющей точки, относительно которого выровнен горизонтальный центр фигуры.
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку AlignCenter по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AlignCenter  <br/> |
   
Для получения ссылки на ячейки AlignCenter по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignCenter** <br/> |
