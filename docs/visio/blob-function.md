# <a name="blob-function"></a>Функция BLOB

Декодирует base64 и сохраняет его в виде двоичных данных. 
  
## <a name="syntax"></a>Синтаксис

Больших двоичных ОБЪЕКТОВ (** *bintext* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _bintext_ <br/> |Обязательный  <br/> |**Строка** <br/> | Строка в кодировке base64.  <br/> |
