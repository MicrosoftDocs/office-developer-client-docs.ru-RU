# <a name="theme-properties-section"></a>Theme Properties Section

Отображает значения индекса темы, применяемой в настоящее время к документу, включая значения цвета, шрифта, соединителя и эффектов. 
  
## <a name="remarks"></a>Примечания

Вы можете установить значения темы на вкладке **Design** или внести изменения в ячейки в разделе **Свойства** темы в ShapeSheet.
