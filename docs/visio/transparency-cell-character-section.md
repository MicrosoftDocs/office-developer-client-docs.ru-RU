# <a name="transparency-cell-character-section"></a>Ячейка Transparency (раздел "Символ")

Определяет уровень прозрачности для диапазона цвет текста фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 — 100  <br/> |Представляет собой процентную долю прозрачность. Значение по умолчанию — 0% (полностью непрозрачный).  <br/> |
   
## <a name="remarks"></a>Замечания

Значения округляются до ближайшего большего половины процентов. Значение 100% является полностью прозрачным. Хотя фигуры, полностью прозрачным текст отображается на странице документа как фигуры, которая не содержит текста, взаимодействия с другие объекты на странице таким же образом, как если бы его прозрачность 0%.
  
Также можно задать это значение с помощью ползунка на вкладке « **Шрифт** » в диалоговое окно " **Text** " (на вкладке **Главная** щелкните стрелку **шрифтов** ). 
  
Если раздел символов содержит несколько строк, ячейка прозрачность содержит сведения о форматировании, применяемые к поддиапазона текстом фигуры. В противном случае содержит сведения о форматировании для всех текста фигуры.
  
Чтобы получить ссылку на ячейку прозрачность по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Char.ColorTrans [ *i* ] где *i* = < 1 > 2, 3...  <br/> |
   
Для получения ссылки на ячейки прозрачность по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionCharacter** <br/> |
|Индекс строки:  <br/> |**visRowCharacter** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visCharacterColorTrans** <br/> |
