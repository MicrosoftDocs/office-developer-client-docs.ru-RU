# <a name="transparency-cell-character-section"></a>Transparency Cell (Character Section)

Определяет уровень прозрачности для диапазона цвета текста фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 - 100  <br/> |Представляет процент прозрачности. Значение по умолчанию — 0 % (полностью непрозрачной).  <br/> |
   
## <a name="remarks"></a>Примечания

Значения округлются до ближайшей половины процента. Значение 100% полностью прозрачно. Хотя фигура с полностью прозрачным текстом отображается на странице документа так же, как фигура без текста, она взаимодействует с другими объектами на странице так же, как если бы ее прозрачность была 0%.
  
Это значение также можно установить с помощью ползунок  на вкладке **"Шрифт"** в диалоговом окне "Текст" (на вкладке "Главная" щелкните стрелку **шрифта).**  
  
Если раздел "Символы" содержит несколько строк, ячейка Transparency содержит сведения о форматированиях, применяемые к поддиаплю текста фигуры. В противном случае он содержит сведения о формате для всего текста фигуры.
  
Чтобы получить ссылку на ячейку Transparency по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Char.ColorTrans[ *i*  ] где  *i*  = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку Transparency по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionCharacter** <br/> |
|Индекс строки:  <br/> |**visRowCharacter**  +   *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visCharacterColorTrans** <br/> |
