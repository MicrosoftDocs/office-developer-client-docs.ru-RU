# <a name="glue-cell-layers-section"></a>Glue Cell (Layers Section)

Указывает, можно ли склеить фигуры, принадлежащие слою.
  
|**Value**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Включена функция приклеивание.  <br/> |
|FALSE  <br/> |Приклеивание отключена.  <br/> |
   
## <a name="remarks"></a>Комментарии

Эта ячейка соответствует параметру "приКлеить" в диалоговом окне **Свойства слоя** (на вкладке **Главная** , в группе **Правка** щелкните **слои**, а затем выберите **Свойства слоя** ). **** 
  
Чтобы получить ссылку на ячейку с приКреплением по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Layers. приклеивание [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку с приКреплением по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**Виссектионлайер** <br/> |
|Индекс строки:  <br/> |**висровлайер** +  *i* , где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**Вислайерглуе** <br/> |
