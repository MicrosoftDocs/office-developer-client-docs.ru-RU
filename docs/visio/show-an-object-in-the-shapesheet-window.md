# <a name="show-an-object-in-the-shapesheet-window"></a>Показать объект в окне таблицы фигур

Объекты Microsoft Visio хранятся внутри в качестве набора формул. Вы можете просмотреть объект Visio в окне таблицы фигур, чтобы просмотреть и изменить формулы, определяющие его внешний вид и поведение.
  
Для получения дополнительных сведений о том, как отобразить определенный объект в окне ShapeSheet, щелкните элемент в следующем списке.
  
## <a name="show-a-shape-group-guide-guide-point-or-foreign-object"></a>Показать фигуру, группу, руководство, точку руководства или иной объект

1. Выберите объект.
    
2. На [вкладке "Разработчик"](run-in-developer-mode-display-the-developer-tab.md) **щелкните "Показать таблицу фигур".**
    
## <a name="show-a-master"></a>Показать хозяина

Вы можете получить доступ к окну ShapeSheet и редактировать формулы только для мастеров в настраиваемом или документе трафарета. Начиная с Microsoft Office Visio 2003, вы не можете редактировать вешки на трафаретах, которые погрузка с продуктом Visio.
  
1. В **окне "Фигуры"** щелкните **"Другие фигуры"** и выберите **"Открыть трафарет".**
    
2. Найдите и выберите трафарет, содержащий хозяина,  щелкните стрелку рядом с кнопкой "Открыть", выберите параметр открытия в выпадаемом списке и нажмите кнопку **"Открыть".** 
    
    Если трафарет уже открыт, щелкните правой кнопкой мыши его  заголовок и выберите пункт "Изменить трафарет" в ярлыковом меню. 
    
3. Щелкните правой кнопкой мыши хозяин в трафарете, выберите пункт **"Изменить** хозяин" в ярлыковом меню и выберите пункт **"Изменить фигуру хозяина".**
    
4. В окне редактирования хозяина щелкните правой кнопкой мыши хозяин и выберите **"Показать таблицу фигур".**
    
## <a name="show-a-page"></a>Показать страницу

1. Убедитесь, что на странице не выбрано ничего.
    
2. Щелкните страницу правой кнопкой мыши и выберите **"Показать таблицу фигур".**
    
## <a name="show-a-document"></a>Показать документ

1. На [вкладке "Разработчик"](run-in-developer-mode-display-the-developer-tab.md) в группе **"Показать/скрыть"** выберите **окно проводника.**
    
2. В **обозревателе документов** щелкните правой кнопкой мыши документ и выберите "Показать **таблицу фигур"** в ярлыковом меню. 
    
## <a name="show-a-style"></a>Показать стиль

1. На [вкладке "Разработчик"](run-in-developer-mode-display-the-developer-tab.md) в группе **"Показать/скрыть"** выберите **окно проводника.**
    
2. В **обозревателе рисунков** разоберем **папку "Стили".** 
    
3. Щелкните правой кнопкой мыши стиль и выберите пункт **"Показать таблицу фигур"** в ярлыковом меню.
