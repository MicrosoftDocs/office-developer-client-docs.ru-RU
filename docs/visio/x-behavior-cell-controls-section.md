# <a name="x-behavior-cell-controls-section"></a>Ячейка X Behavior (раздел "Элементы управления")

Управляет поведением *x* -координата дескриптора элемента управления будет отличаться от после перемещения. 
  
|**Значение**|**Поведение**|**Определение**|**Константа автоматизации**|
|:-----|:-----|:-----|:-----|
| 0  <br/> | Пропорциональный  <br/> | Можно переместить дескриптора элемента управления, а также перемещается в пропорции с фигурой при его растяжении.  <br/> |**visCtlProportional** <br/> |
| 1  <br/> | Пропорциональный заблокирован  <br/> | Дескриптор управления перемещается пропорции с фигурой, но нельзя перемещать самого дескриптора элемента управления.  <br/> |**visCtlLocked** <br/> |
| 2  <br/> | Смещение от левого края  <br/> | Смещение дескриптор управления константу расстояние от левого края фигуры.  <br/> |**visCtlOffsetMin** <br/> |
| 3  <br/> | Смещение из центра  <br/> | Смещение дескриптор управления константу расстояние от центра фигуры.  <br/> |**visCtlOffsetMid** <br/> |
| 4  <br/> | Смещение от правому краю  <br/> | Смещение дескриптор управления константу расстояние от правой части фигуры.  <br/> |**visCtlOffsetMax** <br/> |
| 5  <br/> | Пропорционально, скрытые  <br/> | То же, что 0, но дескриптора элемента управления не отображается.  <br/> |**visCtlProportionalHidden** <br/> |
| 6  <br/> | Пропорциональный блокируется, скрытые  <br/> | То же, что 1, но дескриптора элемента управления не отображается.  <br/> |**visCtlLockedHiddenv** <br/> |
| 7  <br/> | Смещение от левого края, скрытым  <br/> | То же, что 2, но дескриптора элемента управления не отображается.  <br/> |**visCtlOffsetMinHidden** <br/> |
| 8  <br/> | Смещение из центра, скрытым  <br/> | То же, что 3, но дескриптора элемента управления не отображается.  <br/> |**visCtlOffsetMidHidden** <br/> |
| 9  <br/> | Смещение от правого края, скрытым  <br/> | То же, что 4, но дескриптора элемента управления не отображается.  <br/> |**visCtlOffsetMaxHidden** <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки X поведение по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Элементы управления.  *имя* . Элементы управления XConwhere.  *имя* — это имя строки элементов управления.  <br/> |
   
Для получения ссылки на ячейки X поведение по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionControls** <br/> |
| Индекс строки:  <br/> |**visRowControl** +  *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visCtlXCon** <br/> |
