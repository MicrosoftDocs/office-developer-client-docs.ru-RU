# <a name="name-cell-reviewer-section"></a>Name Cell (Reviewer Section)

Содержит имя рецензента документов.
  
## <a name="remarks"></a>Примечания

 Это значение по умолчанию занося в имя, найденное в поле "Имя пользователя"  на вкладке "Общие" диалогового окна "Параметры **Visio"** (щелкните вкладку "Файл", выберите "Параметры" и выберите    **"Общие").** 
  
Чтобы получить ссылку на ячейку Name по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Reviewer.Name [  *i*  ] где  *i*  = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку Name по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionReviewer** <br/> |
| Индекс строки:  <br/> |**visRowReviewer**  +   *i,* *где i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visReviewerName** <br/> |
