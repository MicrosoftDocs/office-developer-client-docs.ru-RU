# <a name="shdwtype-cell-page-properties-section"></a>Ячейка ShdwType (раздел "Свойства страницы")

Указывает тип тени по умолчанию для страницы.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| 1  <br/> | Простое  <br/> |**visFSTSimple** <br/> |
| 2  <br/> | Наклон  <br/> |**visFSTOblique** <br/> |
|3  <br/> |Внутренний  <br/> |**visFSTInner** <br/> |
   
## <a name="remarks"></a>Замечания

 При использовании ShapeShdwType ячейки (тип тени для отдельных фигуры на странице) страница по умолчанию (**visFSTPageDefault** ) используется тип тени, описанных в этой ячейке. 
  
Простая тень типов описаны как смещения тени пользовательского интерфейса (UI). Простая тень дает влияние фигуры, теневую на параллельный плоскости, расположенной некоторые расстояние под ним. Наклон тени описанными наклон теней в пользовательском Интерфейсе и создания эффекта тени приведение на плоскости перпендикулярно фигуры. 
  
Список типов предварительно заданных простой и наклон тени, можно найти в списке **стиль** на вкладке **теней** диалогового окна **Параметры страницы** (на вкладки " **Конструктор** ", нажмите стрелку **Параметры страницы** ). 
  
Чтобы получить ссылку на ячейку ShdwType по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | ShdwType  <br/> |
   
Для получения ссылки на ячейки ShapeShdwOffsetX по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowPage** <br/> |
| Индекс ячейки:  <br/> |**visPageShdwType** <br/> |
