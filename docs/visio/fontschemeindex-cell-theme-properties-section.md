# <a name="fontschemeindex-cell-theme-properties-section"></a>Ячейка FontSchemeIndex (раздел "Свойства темы")

Определяет схему шрифтов темы, которая применяется к фигуре, как целое число. 
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на **FontSchemeIndex** cell по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | FontSchemeIndex  <br/> |
   
Чтобы получить ссылку на **FontSchemeIndex** cell по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowThemeProperties** <br/> |
| Индекс ячейки:  <br/> |**visFontSchemeIndex** <br/> |
