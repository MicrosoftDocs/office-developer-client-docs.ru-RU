# <a name="foreigndata-element-shapesheettype-complextype-visio-xml"></a>Элемент ForeignData (Шапешит_типе complexType) (' Visio XML ')

Содержит зашифрованный большой ДВОИЧный объект MIME (многоцелевые поЧтовые расширения) для графических данных, таких как метафайл Windows, точечный рисунок или данные OLE.

|:-----|:-----|
|**Тип элемента** <br/> |[Фореигндата_типе](foreigndata_type-complextypevisio-xml.md) <br/> |
|**Пространство имен** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Файл схемы** <br/> |VisioSchema15. xsd  <br/> |
|**Части документа** <br/> |страница #. XML, Master #. XML  <br/> |
   
## <a name="definition"></a>Определение

```XML
< xs:element name="ForeignData" type="ForeignData_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```
