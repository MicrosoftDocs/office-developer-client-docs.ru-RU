# <a name="character-section"></a>Раздел "Символ"

Отображает атрибуты форматирования для текста фигуры, такие как шрифт, цвет, стиль текста, случая, поместите относительно базового и размер шрифта.
  
## <a name="remarks"></a>Замечания

Каждый стиль знака в фигуры имеет соответствующая строка в разделе **символ** , как просматривать в окне таблицы свойств фигуры. Имя строки представляет число знаков, которое применяется стиль символов. 
  
Можно установить форматы текст в **текстовом** поле диалогового окна (нажмите кнопку вызова диалогового окна **Шрифт** группы на вкладке **Главная** ), путем применения стиля текста или путем создания записи в ячейку раздел **символ** .
