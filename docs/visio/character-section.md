# <a name="character-section"></a>Character Section

Отображает атрибуты форматирования для текста фигуры, такие как шрифт, цвет, стиль текста, случай, положение относительно базового и размера точки.
  
## <a name="remarks"></a>Примечания

Каждый стиль символов в фигуре  имеет соответствующую строку в разделе "Символы" в окне таблицы фигур. Имя строки представляет количество символов, к которых применяется стиль символов. 
  
Вы можете настроить форматы  текста в диалоговом окне "Текст" (щелкните окно запуска диалогового окна в группе шрифтов на вкладке "Главная"), применив текстовый стиль или сделав запись в ячейке раздела   **"Символ".**
