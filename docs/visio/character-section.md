# <a name="character-section"></a>Character Section

Показаны атрибуты форматирования текста фигуры, такие как шрифт, цвет, стиль текста, корпус, положение относительно базового и размера точеки.
  
## <a name="remarks"></a>Примечания

Каждый стиль символов в форме имеет соответствующую строку в разделе **Character,** как это просматривается в окне ShapeSheet. Имя строки представляет количество символов, к числу которых применяется стиль символов. 
  
Текстовые форматы можно установить в диалоговом окне **Text** (щелкните  диалоговое окно в группе **Шрифт** на вкладке Home), применяя текстовый стиль или внося запись в ячейку раздела **Character.**
