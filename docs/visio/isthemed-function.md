# <a name="isthemed-function"></a>Функция ISTHEMED

Возвращает логическое значение, указывающее, применена ли к фигуре тема. 
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2013
 
  
## <a name="syntax"></a>Синтаксис

 **** С учетом ()
  
## <a name="return-value"></a>Возвращаемое значение

Boolean
  
## <a name="remarks"></a>Примечания

> [!NOTE]
> Функция **** Целлиссемед в Visio 2013 заменяет функцию **** из предыдущих версий Visio. 
  
Функция **** "вопорции" позволяет назначать подходящие части форматирования темы фигуре, но сохранять возможность переопределения других частей форматирования темы с форматированием, примененным вручную. При повторном применении темы любое ручное форматирование переопределяется, а фигура принимает все форматирование темы. 
  
 **** ВОЗВРАЩАЕТ значение true, если ячейка [ColorSchemeIndex](colorschemeindex-cell-theme-properties-section.md) в фигуре больше 0. Если ячейка равна 0, то выдается значение FALSE. **** Тема DocumentSheet и PageSheet не влияет на значение функции, используемой **** в таблице свойств фигуры. Только в том **** случае, если функция PageSheet содержит тему страницы. 
  
## <a name="example"></a>Пример

||||
|:-----|:-----|:-----|
|Cell  <br/> |Формула  <br/> |Результат  <br/> |
|Char. Font  <br/> |Если (in (), THEMEVAL (), FONT ("Calibri"))  <br/> |Если к фигуре применены темы, текст фигуры принимает форматирование шрифта из темы. Если фигура не имеет темы, текст фигуры форматируется с использованием шрифта "Calibri".  <br/> |
|LineColor  <br/> |IF (IN, RGB (255, 0, 0), RGB (0, 255, 0))  <br/> |Если к фигуре применены темы, то цвет линии фигуры — красный. Если фигура не имеет темы, то цвет линии фигуры — зеленый.  <br/> |
