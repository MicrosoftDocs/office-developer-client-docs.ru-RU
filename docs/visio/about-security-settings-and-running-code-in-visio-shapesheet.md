# <a name="about-security-settings-and-running-code-in-visio-shapesheet"></a>О безопасности Параметры и запуск кода в Visio (ShapeSheet)

 Создание безопасных приложений является одной из основных проблем, с которыми сталкиваются разработчики решений. Пользователи, администраторы и разработчики все больше осознают возможность непреднабежаемого запуска кода, который может быть вредным для их компьютеров. Как никогда важно, чтобы вы помогли обеспечить целостность приложений. 
  
Все параметры безопасности Office и устанавливаются в  Центре доверия (щелкните вкладку **Файл,** щелкните Параметры **и** нажмите кнопку **Центр доверия).** Затронутые параметры включают следующие:
  
- Указание доверенных издателей
    
- Указание надежных местоположений
    
- Загрузка надстройок COM 
    
- Загрузка опубликованных надстройок и обнаруженных путей
    
- Загрузка макрос VBA
    
В предыдущих версиях Visio параметры были  сделаны в диалоговом окне Security и вкладке Безопасность диалогового окна **Options**  **(Меню инструментов).** По Office Visio 2007 г. эти диалоговые окна были устранены, а по Visio 2010 г. Visio панели инструментов и меню были заменены лентой. 
  
Дополнительные сведения о параметрах в центре Office см. в заметках по безопасности [для Microsoft Office разработчиков решений.](https://docs.microsoft.com/previous-versions/office/developer/office-2007/aa433259(v=office.12))
  
 Сведения о цифровом подписании кода и надежных источниках и издателях можно найти на веб-сайте MSDN, Microsoft Developer Network коде. 
  
Дополнительные сведения о методиках и технологиях разработки безопасности можно найти в msDN.
