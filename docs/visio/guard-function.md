# <a name="guard-function"></a>Функция GUARD

Защищает *выражение* от удаления и изменения по действиям, выполняемым в окне документа, например при перемещении, изменении размера, группировании или разгруппировании фигур. 
  
## <a name="syntax"></a>Синтаксис

GUARD (* * *Expression* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**String** <br/> |Сочетание констант, операторов, функций и ссылок на ячейки таблицы свойств фигуры, в результате чего получается значение.  <br/> |
   
## <a name="remarks"></a>Примечания

К ячейкам, которые чаще всего влияют функцией GUARD, относятся ширина, высота, PinX и PinY. 
  
Защита формулы в ячейках в любой ячейке не позволяет изменить значение этой ячейки с помощью действий в окне документа. Однако одно действие в окне документа может повлиять на несколько ячеек, и каждая из этих ячеек должна быть защищена, если необходимо предотвратить непредвиденные изменения внешнего вида фигуры. 
  
## <a name="example"></a>Пример

GUARD (TEXTWIDTH (TheText) + 0,5 in) 
  
Это выражение в ячейке "ширина" в разделе "Преобразование фигуры фигуры" запрещает замену выражения (TEXTWIDTH (TheText) + 0,5 in) на другое значение при изменении ширины фигуры в окне документа. TheText — ячейка, которая вызывается при изменении композиции текста связанной фигуры.
