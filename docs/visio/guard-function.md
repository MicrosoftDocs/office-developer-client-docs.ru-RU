# <a name="guard-function"></a>Функция GUARD

Защищает  *выражение*  от удаления и изменения действиями, выполняемыми в окне рисования, например, перемещением, размером, группировкой или разгруппировка фигур. 
  
## <a name="syntax"></a>Синтаксис

GUARD(** *выражение* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**String** <br/> |Сочетание констант, операторов, функций и ссылок на ячейки ShapeSheet, что приводит к значению.  <br/> |
   
## <a name="remarks"></a>Примечания

Чаще всего функцией GUARD страдают ячейки Ширина, Высота, PinX и PinY. 
  
Охрана формулы в любой ячейке не позволяет изменить значение этой ячейки действиями в окне рисования. Однако одно действие в окне рисования может повлиять на несколько ячеек, и каждая из этих ячеек должна быть охраняема, если вы хотите предотвратить неожиданные изменения внешнего вида фигуры. 
  
## <a name="example"></a>Пример

GUARD (TEXTWIDTH(TheText) + 0.5 in) 
  
Это выражение в ячейке Ширина раздела Преобразование формы формы предотвращает замену выражения (TEXTWIDTH (TheText) + 0.5 in на другое значение при смене ширины формы в окне рисования. TheText — это ячейка, которая запускается при смене текстовой композиции связанной фигуры.
