# <a name="centery-cell-print-properties-section"></a>Ячейка CenterY (раздел "Свойства печати")

Определяет, будет ли страницы рисунка центру по по вертикали на странице принтера. 
  
|**Значение**|**Описание**|
|:-----|:-----|
| TRUE  <br/> | Центр страницы рисунка по вертикали на странице принтера.  <br/> |
| FALSE  <br/> | Не центрирования страницы рисунка по вертикали на странице принтера (по умолчанию).  <br/> |
   
## <a name="remarks"></a>Замечания

По умолчанию страниц документа краю верхней и левой части страницы принтера. Установка ячейки CenterX и CenterY TRUE местах на странице документа в центре страницы принтера (или страниц при необходимости заполнения). 
  
Для получения ссылки на ячейки координат центра по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | CenterY  <br/> |
   
Для получения ссылки на ячейки координат центра по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowPrintProperties** <br/> |
| Индекс ячейки:  <br/> |**visPrintPropertiesCenterY** <br/> |
