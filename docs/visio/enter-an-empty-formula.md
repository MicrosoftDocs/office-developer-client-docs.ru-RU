# <a name="enter-an-empty-formula"></a>Ввод пустой формулы

1. Выберите ячейку, которая будет содержать формулу.
    
2. Нажмите КНОПКУ УДАЛИТЬ, а затем нажмите КНОПКУ ВВОД.
    
> [!NOTE]
> Попробуйте это в **ячейке Scratch** после вставки локальной формулы. Ячейка возвращается к "Нет формуле", так как нет формулы, которую можно наследовать от мастера.
