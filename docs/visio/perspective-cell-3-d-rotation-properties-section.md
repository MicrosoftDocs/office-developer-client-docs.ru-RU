# <a name="perspective-cell-3-d-rotation-properties-section"></a>Перспектива ячейки (поворот объема Properties Section)

Определяет угол перспективы для перспективы вращение, в градусов (0-359.9)
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **Перспектива** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Перспектива  <br/> |
   
Для получения ссылки на ячейки **Перспектива** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRow3DRotationProperties** <br/> |
|Индекс ячейки:  <br/> |**visPerspective** <br/> |
