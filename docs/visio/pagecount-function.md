# <a name="pagecount-function"></a>Функция PAGECOUNT

Возвращает количество страниц переднего плана в документе. 
  
## <a name="syntax"></a>Синтаксис

PAGECOUNT ()
  
### <a name="return-value"></a>Возвращаемое значение

Числовой
