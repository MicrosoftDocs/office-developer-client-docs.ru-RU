# <a name="atan-function"></a>Функция ATAN

Возвращает арктангенс числа, например угол, тангенс которого равен _числу_. Полученный угол находится в диапазоне-Пи/2 _Лт_ = Angle _Лт_ = PI/2 радиан (-90 _Лт_ = угол _Лт_ = 90 градусов). 
  
## <a name="syntax"></a>Синтаксис

ATAN (* * *Number* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Тангенс угла.  <br/> |
   
## <a name="example"></a>Пример

ATAN (1) 
  
Возвращает 45 град
