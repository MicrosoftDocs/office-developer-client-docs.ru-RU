# <a name="atan-function"></a>Функция ATAN

Возвращает arctangent номера, например, угол, касающееся которого  _является номером_. В результате угол находится в диапазоне -PI/2 <= угол <= радианы PI/2 (-90 <= угол <= 90 градусов). 
  
## <a name="syntax"></a>Синтаксис

ATAN(** *number* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательный  <br/> |**Числовой** <br/> |По касательной угла.  <br/> |
   
## <a name="example"></a>Пример

ATAN (1) 
  
Возвращает 45 дег
