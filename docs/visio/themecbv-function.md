# <a name="themecbv-function"></a>Функция THEMECBV

Возвращает значение RGB или целое число, представляющее индекс в документе цветовой палитры, в которых был изменен цвет (число), переданной в качестве аргумента по указанному значению tint или тень, хранящиеся в параметры градиента активной темы. 
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2013
 
  
## <a name="syntax"></a>Синтаксис

 **THEMECBV** ( _цвет_, _gradient_stop_number_)
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Цвет_ <br/> |Обязательный  <br/> |**Число** <br/> |Число, представляющее индекса в документе цветовой палитры.  <br/> |
| _gradient_stop_number_ <br/> |Обязательный  <br/> |**Число** <br/> |Градиента (tint или тень) хранятся в параметры градиента активной темы для применения к цвет.  <br/> |
   
## <a name="return-value"></a>������������ ��������

 **Число**
  
## <a name="remarks"></a>Замечания

> [!NOTE]
> Функция THEMECBV не выполняет никаких действий на цвет, переданное как аргумент Если QuickStyle, назначенная фигуры не имеет градиент. 
  
Параметры градиента в теме — это серии нумерованный градиента, которые соответствуют «осветляющего» (tint) или «затемняющего» (тень). Эти тени и оттенки применяются к основной цвет для создания эффекта градиента.
  
Функция **THEMECBV** принимает ввод цвета и выводит цвет после изменения с помощью tint или тень указанного градиента. Оттенки и тени поступают из определения темы, если текущий экспресс-стилей содержит градиентной заливки. Если active экспресс-стилей не определяет, что градиентной заливки или фигуры задано значение Нет темы, эту формулу просто возвращает цвета, которое было передано в качестве первого аргумента. 
  
## <a name="example"></a>Пример

 `THEMECBV(FillForegnd, 5)`
  
Возвращает цвет, с помощью tint или тень в пятый градиента градиента цвет, определенный в ячейке **FillForegnd** . 
  
 `THEMECBV(RGB(255,0,0), 2)`
  
Возвращает тень или tint красного, с помощью второй градиента основной цвет красный.
