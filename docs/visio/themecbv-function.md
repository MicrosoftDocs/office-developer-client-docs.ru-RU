# <a name="themecbv-function"></a>Функция THEMECBV

Возвращает значение RGB или целый ряд, который представляет индекс в цветовой палитре документа, где цвет (номер), переданный в качестве аргумента, был изменен указанным значением оттенка или тени, хранимых в параметрах градиента активной темы. 
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2013
 
  
## <a name="syntax"></a>Синтаксис

 **THEMECBV** _(цвет_, _gradient_stop_number)_
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color_ <br/> |Обязательный  <br/> |**Number** <br/> |Число, представляющее индекс в цветовой палитре документа.  <br/> |
| _gradient_stop_number_ <br/> |Обязательный  <br/> |**Number** <br/> |Стоп градиента (оттенок или оттенок), хранимый в параметрах градиента активной темы для применения к цвету.  <br/> |
   
## <a name="return-value"></a>Возвращаемое значение

 **Number**
  
## <a name="remarks"></a>Примечания

> [!NOTE]
> Функция THEMECBV ничего не делает с цветом, переданным в качестве аргумента, если quickStyle, который назначен фигуре, не имеет градиента. 
  
Параметры градиента в теме — это серия про номерных остановок градиента, соответствующих "осветления" (оттенок) или "затемнению" (оттенок). Эти оттенки и оттенки применяются к базовому цвету, чтобы создать эффект цвета градиента.
  
Функция **THEMECBV** принимает цветовой ввод и выводит цвет после изменения оттенка или тени указанной остановки градиента. Оттенки и оттенки приходят из определения темы, если текущий быстрый стиль содержит заполнение градиента. Если активный быстрый стиль не указывает заполнение градиента или форма настроена на No Theme, эта формула просто возвращает цвет, переданный для первого аргумента. 
  
## <a name="example"></a>Пример

 `THEMECBV(FillForegnd, 5)`
  
Возвращает цвет, созданный путем применения оттенка или оттенка в пятой остановке градиента градиента к цвету, указанному в ячейке **FillForegnd.** 
  
 `THEMECBV(RGB(255,0,0), 2)`
  
Возвращает оттенок или оттенок красного цвета, созданный путем применения второй остановки градиента к базовому цвету красного.
