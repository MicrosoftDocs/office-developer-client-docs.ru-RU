# <a name="themecbv-function"></a>Функция THEMECBV

Возвращает значение RGB или integer, которое представляет индекс в цветовой палитре документа, где цвет (число), переданный в качестве аргумента, был изменен указанным значением оттенка или затенения, сохраненным в параметрах градиента активной темы. 
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2013
 
  
## <a name="syntax"></a>Синтаксис

 **THEMECBV**( _цвет,_ _gradient_stop_number)_
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color_ <br/> |Обязательна  <br/> |**Number** <br/> |Число, представляющее индекс в цветовой палитре документа.  <br/> |
| _gradient_stop_number_ <br/> |Обязательна  <br/> |**Number** <br/> |Остановка градиента (оттенка или затенения), хранимая в параметрах градиента активной темы для применения к цвету.  <br/> |
   
## <a name="return-value"></a>Возвращаемое значение

 **Number**
  
## <a name="remarks"></a>Примечания

> [!NOTE]
> Функция THEMECBV не делает ничего для цвета, переданного в качестве аргумента, если quickStyle, который назначен фигуре, не имеет градиента. 
  
Параметры градиента в теме — это ряд проклиненных остановок градиента, соответствующих "осветлке" (оттенку) или "затемнению" (теню). Эти оттенки и оттенки применяются к базовому цвету для создания эффекта цвета градиента.
  
Функция **THEMECBV** принимает входные данные цвета и выводит цвет после того, как он был изменен оттенком или оттенком указанной остановки градиента. Оттенки и оттенки налагаются определением темы, если текущий быстрый стиль содержит заливку градиента. Если активный экспресс-стиль не указывает заливку градиента или фигура имеет вид "Нет темы", эта формула просто возвращает цвет, переданный для первого аргумента. 
  
## <a name="example"></a>Пример

 `THEMECBV(FillForegnd, 5)`
  
Возвращает цвет, созданный путем применения оттенка или затенения в пятой остановке градиента к цвету, указанному в ячейке **FillForegnd.** 
  
 `THEMECBV(RGB(255,0,0), 2)`
  
Возвращает оттенки красного цвета, созданные путем применения второй остановки градиента к базовому цвету красного.
