# <a name="themecbv-function"></a>Функция THEMECBV

Возвращает значение RGB или целое число, представляющее индекс в цветовой палитре документа, где цвет (число), переданный в качестве аргумента, был изменен указанным оттенком или значением затенения, хранящимся в параметрах градиента активной темы. 
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2013
 
  
## <a name="syntax"></a>Синтаксис

 **THEMECBV** ( _Color_, _градиент_стоп_нумбер_)
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color_ <br/> |Обязательный  <br/> |**Number** <br/> |Число, представляющее индекс в цветовой палитре документа.  <br/> |
| _градиент_стоп_нумбер_ <br/> |Обязательный  <br/> |**Number** <br/> |Точка градиента (оттенок или тень), хранящаяся в параметрах градиента активной темы, которые применяются к цвету.  <br/> |
   
## <a name="return-value"></a>Возвращаемое значение

 **Number**
  
## <a name="remarks"></a>Примечания

> [!NOTE]
> Функция THEMECBV ничего не относится к цвету, переданному в качестве аргумента, если Куиккстиле, назначенный фигуре, не имеет градиента. 
  
Параметры градиента в теме — это ряд пронумерованных точек градиента, соответствующих "осветлению" (оттенок) или "затемнение" (затенение). Эти тени и оттенки применяются к базовому цвету для создания цветового эффекты градиента.
  
Функция **THEMECBV** принимает в качестве входных данных цвет и выводит цвет после его изменения оттенокм или затенением указанного ограничителя градиента. Оттенок и тени берутся из определения темы, если текущий Экспресс же стиль содержит градиентную заливку. Если в активном быстром стиле не указана Градиентная заливка или для фигуры не задано значение "нет темы", то эта формула просто возвращает цвет, переданный для первого аргумента. 
  
## <a name="example"></a>Пример

 `THEMECBV(FillForegnd, 5)`
  
Возвращает цвет, созданный с помощью оттенка или оттенка в пятом элементе градиента градиента, к цвету, указанному в ячейке **FillForegnd** . 
  
 `THEMECBV(RGB(255,0,0), 2)`
  
Возвращает тень или оттенок красного цвета, созданный с применением второго ограничителя градиента к базовому цвету красного цвета.
