# <a name="tip-cell-controls-section"></a>Tip Cell (Controls Section)

Представляет текстовую строку описательной, которая появляется в виде подсказки инструмента, когда пользователь приостанавливирует указатель над ручкой управления фигурой. Приложение автоматически заключит строку подсказки в кавычках в ячейке, но кавычка не отображается в подсказке инструмента.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку Подсказка по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Controls.  *имя*  . Элементы управления Tipwhere.  *имя* — название строки элементов управления.  <br/> |
   
Чтобы получить ссылку на ячейку Подсказка по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionControls** <br/> |
| Индекс строки:  <br/> |**visRowControl** +  *i*, где *i* = 0, 1, 2…  <br/> |
| Индекс ячейки:  <br/> |**visCtlTip** <br/> |
