# <a name="tip-cell-controls-section"></a>Совет ячейки (раздел элементы управления)

Представляет строку описательный текст, который отображается в виде подсказки при наведении указателя мыши на элемент управления маркера фигуры. Приложение автоматически заключает строку подсказки в кавычки в ячейке, но кавычки не отображаются в подсказке.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки совет по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Элементы управления.  *имя* . Элементы управления Tipwhere.  *имя* — это имя строки элементов управления.  <br/> |
   
Для получения ссылки на ячейки совет по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionControls** <br/> |
| Индекс строки:  <br/> |**visRowControl** +  *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visCtlTip** <br/> |
