# <a name="tip-cell-controls-section"></a>Tip Cell (Controls Section)

Представляет строку описательного текста, которая отображается в виде подсказки, когда пользователь приостанавливает указатель над маркером управления фигуры. Приложение автоматически заключает строку подсказки в кавычки в ячейке, но кавычки не отображаются в подсказке.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку TIP по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Controls.  *Name (имя* ). Элементы управления Типвхере.  *имя* — название строки элементов управления.  <br/> |
   
Чтобы получить ссылку на ячейку TIP по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionControls** <br/> |
| Индекс строки:  <br/> |**visRowControl** +  *i*, где *i* = 0, 1, 2…  <br/> |
| Индекс ячейки:  <br/> |**Висктлтип** <br/> |
