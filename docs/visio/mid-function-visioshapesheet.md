# <a name="mid-function-visioshapesheet"></a>MID Function (VisioShapeSheet)

Возвращает определенное число символов из строки текста, начиная с указанной позиции, основанный на число знаков, которое можно указать.
  
## <a name="syntax"></a>Синтаксис

MID (** *текст* **, ** *Начальная позиция* **, ** *число_знаков* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _text_ <br/> |Обязательный  <br/> |**Строка** <br/> |Текстовая строка, содержащая знаки, которые вы хотите извлечь.  <br/> |
| _Начальная позиция_ <br/> |Обязательный  <br/> |**Число** <br/> |Положение первого знака, которые нужно извлечь. Первым символом в текстовой строки — позиции 1.  <br/> |
| _Количество_знаков_ <br/> |Обязательный  <br/> |**Число** <br/> |Число возвращаемых знаков.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Строка
  
## <a name="remarks"></a>Замечания

Если *Начальная позиция* : 
  
- Больше, чем длина *текста* , MID возвращает «» (пустая строка). 
    
- Меньше, чем длина *текста* , но *Начальная_позиция* плюс *Количество_знаков* превышает длину *текста* , MID возвращает символы до конца *текста* . 
    
- Меньше 1 MID возвращает #VALUE! значение ошибки. 
    
Если *число_знаков* отрицательно, MID возвращает #VALUE! значение ошибки. 
  
## <a name="example"></a>Пример

MID («SSN 999-99-9999», 5, 11) 
  
Возвращает 999-99-9999.
