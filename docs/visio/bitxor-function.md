# <a name="bitxor-function"></a>Функция BITXOR

Возвращает 16-разрядный двоичное число, в котором каждый бит задано значение 1, если соответствующий бит в либо, но не оба двоичные Число1 и число2 двоичные 1. В противном случае бит равен 0.
  
## <a name="syntax"></a>Синтаксис

BITXOR (** *двоичные Число1* **, ** *двоичные число2* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _двоичные Число1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первый номер двоичные 16-разрядной.  <br/> |
| _двоичные число2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второй 16-разрядное число двоичные.  <br/> |
   
### <a name="return-value"></a>������������ ��������

16-разрядный двоичный
  
## <a name="example"></a>Пример

BITXOR(12,6)
  
Возвращает значение 10. 12 = 0... 01100. 6 = 0... 00110. Таким образом, BITXOR(12,6) = 0... 01010.
