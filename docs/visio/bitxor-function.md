# <a name="bitxor-function"></a>Функция BITXOR

Возвращает 16-битный двоичный номер, в котором для каждого бита установлено 1, если соответствующий бит либо в двоичном номере 1, либо двоичном номере 1 и двоичном номере 1. В противном случае для бита установлено 0.
  
## <a name="syntax"></a>Синтаксис

BITXOR(** *двоичный номер1* **, ** *двоичный номер2* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _двоичный номер1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первый 16-битный двоичный номер.  <br/> |
| _двоичный номер2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второй 16-битный двоичный номер.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

16-битная двоичная
  
## <a name="example"></a>Пример

BITXOR (12,6)
  
Возвращает 10. 12 = 0...01100. 6 = 0...00110. Таким образом, BITXOR (12,6) = 0...01010.
