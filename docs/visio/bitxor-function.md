# <a name="bitxor-function"></a>Функция BITXOR

Возвращает 16-разрядное двоичное число, в котором для каждого бита задано значение 1, если соответствующий бит, но не оба двоичных файла Число1 и число2 имеют значение 1. В противном случае для бита задается значение 0.
  
## <a name="syntax"></a>Синтаксис

БИТ. ИСКЛИЛИ (* * *двоичные числа Число1* * *, * * *двоично-число2* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Двоичные Число1_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Первое 16 – разрядное двоичное число.  <br/> |
| _двоичные файлы число2_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Второй 16 – разрядный двоичный номер.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

16 – разрядный двоичный
  
## <a name="example"></a>Пример

БИТ. ИСКЛИЛИ (12, 6)
  
Возвращает 10. Значение 12 = 0... 01100. 6 = 0... 00110. Таким образом, бит. ИСКЛИЛИ (12, 6) = 0... 01010.
