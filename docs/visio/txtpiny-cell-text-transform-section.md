# <a name="txtpiny-cell-text-transform-section"></a>TxtPinY Cell (Text Transform Section)

Определяет координату *y* центра вращения блока текста относительно начала фигуры. По умолчанию используется следующая формула: 
  
= Высота \* 0,5
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку TxtPinY по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | TxtPinY  <br/> |
   
Чтобы получить ссылку на ячейку TxtPinY по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**Висровтекстксформ** <br/> |
| Индекс ячейки:  <br/> |**Висксформпини** <br/> |
