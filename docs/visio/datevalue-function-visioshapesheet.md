# <a name="datevalue-function-visioshapesheet"></a>DATEVALUE Function (VisioShapeSheet)

Возвращает значение типа date, представленное _даты и времени_ или _выражение_.
  
## <a name="syntax"></a>Синтаксис

Функция DATEVALUE ("** *даты и времени* **" | ** *выражение* ** [, ** *lcid* **]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _даты и времени_ <br/> |Обязательный  <br/> |**Строка** <br/> |Любая строка, часто распознается как даты и времени или ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**Строка** <br/> |Любое выражение, возвращающее даты и времени.  <br/> |
| _код языка_ <br/> |Optional  <br/> |**Число** <br/> |Задает идентификатор языкового стандарта для использования при оценке нелокальных datetime. Идентификатор языкового стандарта — это число, описанных в файлы заголовков системы.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Даты и времени
  
## <a name="remarks"></a>Замечания

Любой компонент времени в *даты и времени* или *выражение,* удаляются. 
  
Если *даты и времени* отсутствует или не может быть преобразована допустимый результат, DATEVALUE возвращает #VALUE! Ошибка. 
  
Возвращаемое значение отображается с использованием стилей краткий формат даты, задание с текущих региональных параметров компьютера. 
  
Функция DATEVALUE также принимает одно значение номеров для *выражения* , где целую часть результата представляет дней, прошедших с 30 декабря 1899. 
  
## <a name="example-1"></a>Пример 1

Функция DATEVALUE (СЕЙЧАС ()) + 5 ed.
  
Возвращает дату пять дней сейчас.
  
## <a name="example-2"></a>Пример 2

ФУНКЦИЯ DATEVALUE ("7/19/1995 12:30")
  
Возвращает дату.
  
## <a name="example-3"></a>Пример 3

Функция DATEVALUE ("мая 33, 1997 г.")
  
Возвращает #VALUE! Ошибка.
  
## <a name="example-4"></a>Пример 4

DATEVALUE(35580.6337)
  
Возвращает 5/30/1997 г.
