# <a name="rectsect-function"></a>Функция RECTSECT

Вычисляет сектор прямоугольника, связанный с *x* и *y* , и возвращает целое число от 0 до 4, обозначающее сектор. 
  
## <a name="syntax"></a>Синтаксис

RECTSECT (* * *Ширина* * *, * * *Высота* * *, * * *x* * *, * * *y* * *, * * *параметр* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _width_ <br/> |Обязательный  <br/> |**String** <br/> |Ширина прямоугольника.  <br/> |
| _height_ <br/> |Обязательный  <br/> |**String** <br/> |Высота прямоугольника.  <br/> |
| _x_ <br/> |Обязательный  <br/> |**String** <br/> |Координата x.  <br/> |
| _y_ <br/> |Обязательный  <br/> |**String** <br/> |Координата y.  <br/> |
| _)_ <br/> |Обязательный  <br/> |**Логический** <br/> |Указывает, как обрабатываются точки, которые попадают на диагонали. Задайте для параметра значение 0, чтобы использовать левые и правые сектора для точек по диагонали. Присвойте этому параметру значение 1, чтобы использовать верхние и нижние сектора для точек по диагонали.  <br/> |
   
## <a name="remarks"></a>Примечания

Рассмотрим прямоугольник с *шириной* и *высотой* , а также точку (*x, y*) от центральной точки прямоугольника. Нарисуйте диагональные линии через углы прямоугольника, чтобы разделить его на четыре сектора и центральную точку. Сектора от 0 до 4 представляют центральную, правую, верхнюю, левую и нижнюю границу соответственно. 
  
![Секторы от 0 до 4 представляют центральную, правую, верхнюю, левую и нижнюю соответственно](media/ShpSheetRef_CA_03_ZA07645862.gif)
  
## <a name="example"></a>Пример

RECTSECT (1 в, 2 в, 1 в,-7, 0) 
  
Возвращает 4.
