# <a name="rectsect-function"></a>Функция RECTSECT

Вычисляет сектора прямоугольника, связанный с *x* и *y* и возвращает целое число 0 до 4, указывающее сектора. 
  
## <a name="syntax"></a>Синтаксис

RECTSECT (** *ширины* **, ** *Высота* **, ** *x* **, ** *y* **, ** *параметр* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _width_ <br/> |Обязательный  <br/> |**Строка** <br/> |Ширина прямоугольника.  <br/> |
| _height_ <br/> |Обязательный  <br/> |**Строка** <br/> |Высота прямоугольника.  <br/> |
| _x_ <br/> |Обязательный  <br/> |**Строка** <br/> |X координата.  <br/> |
| _y_ <br/> |Обязательный  <br/> |**Строка** <br/> |Координата.  <br/> |
| _параметр_ <br/> |Обязательный  <br/> |**Boolean** <br/> |Указывает порядок обработки точек, которые могут быть разделены на диагоналей. Задайте значение 0, чтобы использовать секторов влево и вправо для указывает на вправо. Задайте значение 1 для использования верхних и нижних секторов для указывает на вправо.  <br/> |
   
## <a name="remarks"></a>Замечания

Рассмотрите возможность прямоугольник, который имеет *ширину* и *высоту* и точка (*x, y*) из центральной точки прямоугольника. Рисование диагональные линии через углов прямоугольника для отделения его в четырех секторов и центральной точки. Цифры от 0 до 4 секторов представляют центральной точки, справа, сверху, слева и вниз соответственно. 
  
![Секторов цифры от 0 до 4 представляют центральной точки, справа, сверху, слева и соответственно вниз](media/ShpSheetRef_CA_03_ZA07645862.gif)
  
## <a name="example"></a>Пример

RECTSECT (1, 2 в, 1 в, -7 в, 0) 
  
Возвращает 4.
