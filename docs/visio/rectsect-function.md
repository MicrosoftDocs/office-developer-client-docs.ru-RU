# <a name="rectsect-function"></a>Функция RECTSECT

Вычисляет сектор прямоугольника, связанного с  *x*  и  *y,*  и возвращает количество от 0 до 4, обозначающее сектор. 
  
## <a name="syntax"></a>Синтаксис

RECTSECT(width, ***height,*** ***x***, ***y***, ***option)*** 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _width_ <br/> |Обязательно  <br/> |**Строка** <br/> |Ширина прямоугольника.  <br/> |
| _height_ <br/> |Обязательно  <br/> |**Строка** <br/> |Высота прямоугольника.  <br/> |
| _x_ <br/> |Обязательно  <br/> |**Строка** <br/> |X-координата.  <br/> |
| _y_ <br/> |Обязательно  <br/> |**Строка** <br/> |Координата Y.  <br/> |
| _option_ <br/> |Обязательный  <br/> |**Логический** <br/> |Указывает, как обрабатываются точки, попадая на диагонали. Установите значение 0, чтобы использовать левый и правый секторы для точек по диагонали. Установите значение 1, чтобы использовать верхний и нижний секторы для точек по диагонали.  <br/> |
   
## <a name="remarks"></a>Примечания

Рассмотрим прямоугольник с шириной и высотой *и* точкой *(x,y)* от центральной точки прямоугольника.  Нарисуйте диагональные линии через углы прямоугольника, чтобы разделить его на четыре сектора и центр. Секторы от 0 до 4 представляют собой центр, правую, верхнюю, левую и нижнюю соответственно. 
  
![Секторы от 0 до 4 представляют собой центр, правую, верхнюю, левую и нижнюю соответственно](media/ShpSheetRef_CA_03_ZA07645862.gif)
  
## <a name="example"></a>Пример

RECTSECT(1 in, 2 in, 1 in, -7 in, 0) 
  
Возвращает 4.
