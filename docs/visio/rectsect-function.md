# <a name="rectsect-function"></a>Функция RECTSECT

Вычисляет сектор прямоугольника, связанный с  *x*  и  *y,*  и возвращает 0 на 4, указав сектор. 
  
## <a name="syntax"></a>Синтаксис

RECTSECT ***(ширина,*** ***высота***, ***x***, ***y***, ***вариант*** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _width_ <br/> |Обязательный  <br/> |**String** <br/> |Ширина прямоугольника.  <br/> |
| _height_ <br/> |Обязательный  <br/> |**String** <br/> |Высота прямоугольника.  <br/> |
| _x_ <br/> |Обязательный  <br/> |**String** <br/> |X-coordinate.  <br/> |
| _y_ <br/> |Обязательный  <br/> |**String** <br/> |y-coordinate.  <br/> |
| _параметр_ <br/> |Обязательный  <br/> |**Логический** <br/> |Указывает, как обрабатываются точки, которые попадают на диагонали. Установите значение 0 для использования левого и правого секторов для точек на диагонали. Установите значение 1 для использования верхних и нижних секторов для точек на диагонали.  <br/> |
   
## <a name="remarks"></a>Примечания

Рассмотрим прямоугольник, который  имеет ширину и высоту,  и точку *(x,y)* от центральной точки прямоугольника. Нарисуйте диагональные линии по углам прямоугольника, чтобы разделить его на четыре сектора и центрную точку. Сектора от 0 до 4 представляют центр- точку, правую, верхнюю, левую и нижнюю соответственно. 
  
![Секторы от 0 до 4 представляют центр, правую, верхнюю, левую и нижнюю области соответственно](media/ShpSheetRef_CA_03_ZA07645862.gif)
  
## <a name="example"></a>Пример

RECTSECT (1 в, 2 в, 1 в, -7 в, 0) 
  
Возвращает 4.
