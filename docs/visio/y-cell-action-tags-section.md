# <a name="y-cell-action-tags-section"></a>Ячейка Y (действие помечает раздел)

*Y* -координат в локальной системе координат фигуры вокруг которых размещается кнопка тега действия. 
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий, называются смарт-тегов. 
  
## <a name="remarks"></a>Замечания

Ячейки X и Y указывать точки в локальной системе координат фигуры и ячейки обоснование X и Y ширине поместить кнопки тега действия по отношению к точке. 
  
Чтобы получить ссылку на ячейку Y по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Смарт-тегов.  *имя* . Y где смарт-тегов. *имя* — это имя строки тега действия  <br/> |
   
Для получения ссылки на ячейки Y по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionSmartTag** <br/> |
| Индекс строки:  <br/> |**visRowSmartTag** +  *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visSmartTagY** <br/> |
