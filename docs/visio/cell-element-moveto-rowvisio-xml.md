# <a name="cell-element-moveto-row-visio-xml"></a>Элемент Cell (MoveTo Row) (Visio XML)

Содержит x или y-координаты первой вершины фигуры или представляет x- или y-координаты первой вершины после разрыва пути.
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|**Тип элемента** <br/> |[Cell_Type](cell_type-complextypevisio-xml.md) <br/> |
|**Пространство имен** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Файл схемы** <br/> |VisioSchema15.xsd  <br/> |
|**Части документов** <br/> |master#.xml, page#.xml  <br/> |
   
## <a name="definition"></a>Определение

```XML
< xs:element name="Cell" type="Cell_Type" minOccurs="0" maxOccurs="unbounded" >
</xs:element >
```

## <a name="remarks"></a>Примечания

Атрибут **N** этого элемента **Cell** должен быть одним из ограниченного набора значений, соответствующих ячейкам ShapeSheet. Обратитесь к приведенной ниже таблице, чтобы определить значения атрибута **N,** разрешенные для этого элемента **Cell.** 
  
|**Значение**|**Описание**|**Дополнительные сведения**|
|:-----|:-----|:-----|
|X  <br/> |Если **строка MoveTo** — это первая строка в разделе, **X-ячейка** представляет x-координату первого вершины фигуры. Если **строка MoveTo** отображается между двумя строками, **X-ячейка** представляет x-координату первой вершины после перерыва в пути.  <br/> |[MoveTo Row (Geometry Section)](moveto-row-geometry-section.md) <br/> |
|Да  <br/> |Если **строка MoveTo** — это первая строка в разделе, **ячейка Y** представляет y-координату первого вершины фигуры. Если **строка MoveTo** отображается между двумя строками, **ячейка Y** представляет y-координату первой вершины после перерыва в пути.  <br/> |[MoveTo Row (Geometry Section)](moveto-row-geometry-section.md) <br/> |
