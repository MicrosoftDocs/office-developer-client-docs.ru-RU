# <a name="bottommargin-cell-text-block-format-section"></a>Ячейка BottomMargin (раздел "Формат текстового блока")

Определяет расстояние между нижней границы блока текста и последней строки текста, которые он содержит. Значение по умолчанию — 0,1 дюйма. Это значение не зависит от масштаба документа. Если документа изменяется, нижнее остается неизменным.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки BottomMargin по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | BottomMargin  <br/> |
   
Для получения ссылки на ячейки BottomMargin по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowText** <br/> |
| Индекс ячейки:  <br/> |**visTxtBlkBottomMargin** <br/> |
