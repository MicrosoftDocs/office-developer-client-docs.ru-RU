# <a name="gradient-properties-section"></a>Раздел "Свойства градиента"

Отображает свойства для любого цвета эффектов градиентной заливки или строку, применяемой к фигуре. В этом разделе также включает в себя ячеек, которые управляют ли поворот градиента фигуры с формой или ли фигуры предполагается, применяемые градиентом, сгруппированный с другими фигурами. 
  
## <a name="remarks"></a>Замечания

В диалоговом окне **Формат фигуры** можно задать параметры градиента (**Домашняя страница** > **Стили фигур** > **заливки** > **Параметры автозаполнения** > **заливки** > **градиентной заливки**).
