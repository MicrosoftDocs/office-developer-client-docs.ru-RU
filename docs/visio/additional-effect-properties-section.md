# <a name="additional-effect-properties-section"></a>Additional Effect Properties Section

Показывает дополнительные эффекты, примененные к фигуре, включая эффекты отражения, свечения и мягких краев. В этом разделе также содержатся значения для эффекта наброска.
  
## <a name="remarks"></a>Примечания

Вы можете настроить эффекты отражения, свечения и мягких краев в диалоговом окне **"Формат** фигуры"**(эффекты стилей**  >  **домашней**  >  **фигуры).** Эффект наброска можно применить к фигуре с помощью одной из тем наброска (**Эффекты** конструктора тем) или внесения изменений в значения в соответствующих ячейках  >    >  ShapeSheet.
