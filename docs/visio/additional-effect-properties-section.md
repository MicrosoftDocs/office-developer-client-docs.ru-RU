# <a name="additional-effect-properties-section"></a>Раздел "Дополнительные свойства эффекта"

Отображает дополнительные эффекты, примененные к фигуры, включая эффектов сглаживание, свечения и отражения. В этом разделе также включаются значения для создания эффекта эскиз.
  
## <a name="remarks"></a>Замечания

Можно задать эффектов сглаживание, свечения и отражения в диалоговом окне **Формат фигуры** (**Домашняя страница** > **Стили фигур** > **эффекты**). Влияние черновика может применяться к фигуры, используя один из темы эскиз (**разработки** > **тем** > **эффекты**) или при внесении изменений в значения в соответствующей таблице свойств фигуры ячейки.
