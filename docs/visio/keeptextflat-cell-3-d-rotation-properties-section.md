# <a name="keeptextflat-cell-3-d-rotation-properties-section"></a>KeepTextFlat Cell (3-D Rotation Properties Section)

Указывает, будет ли текст фигуры игнорировать вращение фигуры в 3-D. Не применяется к 2-D ротации. 
  
****

|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Текст формы не вращается с геометрией фигуры.  <br/> |
|FALSE  <br/> |Текст формы преобразуется для поворота с геометрией фигуры.  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку **KeepTextFlat** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы с использованием свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |KeepTextFlat  <br/> |
   
Чтобы получить ссылку на ячейку **KeepTextFlat** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRow3DRotationProperties** <br/> |
|Индекс ячейки:  <br/> |**visKeepTextFlat** <br/> |
