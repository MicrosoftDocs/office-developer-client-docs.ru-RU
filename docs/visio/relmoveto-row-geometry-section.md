## <a name="remarks"></a><span data-ttu-id="d24f8-116">Примечания</span><span class="sxs-lookup"><span data-stu-id="d24f8-116">Remarks</span></span>

<span data-ttu-id="d24f8-117">Значения в строке **RelMoveTo** эквивалентны значениям в строке [MoveTo,](moveto-row-geometry-section.md) умноженным на ширину и высоту фигуры.</span><span class="sxs-lookup"><span data-stu-id="d24f8-117">Values in the **RelMoveTo** row are equivalent to values in a [MoveTo](moveto-row-geometry-section.md) row that are multiplied by the width and height of the shape.</span></span> <span data-ttu-id="d24f8-118">Например, строка **RelMoveTo,** где значение ячейки **X** равно 0, а ячейка **Y** — "0,5", может быть заменена строкой **MoveTo,** где значение ячейки **X** — это формула "Width 0", а ячейка ***Y***— формула "Height 0.5".</span><span class="sxs-lookup"><span data-stu-id="d24f8-118">For example: a **RelMoveTo** row where the value of the **X** cell is "0" and the value of the **Y** cell is "0.5" could be replaced with **MoveTo** row where the value of the **X** cell is the formula "Width *0" and the **Y** cell is the formula "Height* 0.5."</span></span> 
  
<span data-ttu-id="d24f8-119">Строка **RelMoveTo** содержит координаты  *x*  и  *y*  первой вершины фигуры, если строка MoveTo является первой строкой в разделе.</span><span class="sxs-lookup"><span data-stu-id="d24f8-119">The **RelMoveTo** row contains the  *x*  - and  *y*  -coordinates of the first vertex for the shape if the MoveTo row is the first row in the section.</span></span> <span data-ttu-id="d24f8-120">Обычно это первая вершина, размещенная при отрисовке фигуры, и она не обязательно соответствует началу точки 1-D фигуры.</span><span class="sxs-lookup"><span data-stu-id="d24f8-120">Usually this is the first vertex placed when the shape was drawn, and it does not necessarily correspond to the begin point of a 1-D shape.</span></span>
