# <a name="relmoveto-row-geometry-section"></a>RelMoveTo Row (Geometry Section)

Содержит  *координаты x*  и  *y*  первой вершины фигуры или  *координат*  x и  *Y*  первой вершины после разрыва пути относительно высоты и ширины фигуры. 
  
> [!NOTE]
> Строка **RelMoveTo** может сохраняться только в форматах VSDX, VSDM, VSTX, VSTM, VSSX и VSSM. При сохранении файла в форматах Visio 2003-2010 строка **RelMoveTo** преобразуется в [строку MoveTo.](moveto-row-geometry-section.md) 
  
Строка **RelMoveTo** содержит следующие ячейки. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |Если **строка RelMoveTo** является первой строкой в разделе, ячейка X представляет  *x-координату*  первой вершины фигуры относительно ширины фигуры. Если **строка RelMoveTo** отображается между двумя строками, ячейка X представляет  *x-координату*  первой вершины после разрыва пути.  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |Если **строка RelMoveTo** является первой строкой в разделе, ячейка Y представляет  *координату Y*  первой вершины фигуры. Если **строка RelMoveTo** отображается между двумя строками, ячейка Y представляет  *координату Y*  первой вершины после разрыва пути.  <br/> |
   
## <a name="remarks"></a>Примечания

Значения в строке **RelMoveTo** эквивалентны значениям в строке [MoveTo,](moveto-row-geometry-section.md) умноженным на ширину и высоту фигуры. Например, строка **RelMoveTo,** где значение ячейки **X** равно 0, а ячейка **Y** — "0,5", может быть заменена строкой **MoveTo,** где значение ячейки **X** — это формула "Width 0", а ячейка ***Y***— формула "Height 0.5". 
  
Строка **RelMoveTo** содержит координаты  *x*  и  *y*  первой вершины фигуры, если строка MoveTo является первой строкой в разделе. Обычно это первая вершина, размещенная при отрисовке фигуры, и она не обязательно соответствует началу точки 1-D фигуры. 
  
Раздел  "Геометрия" должен начинаться со **строки MoveTo** или **RelMoveTo,** но вы также можете использовать строки **RelMoveTo** и **MoveTo** для представления разрывов в простроке пути фигуры относительно ширины и высоты фигуры. Однако если путь используется для определения границы заполненной области, этот разрыв интерпретируется как сегмент прямой линии. Чтобы вставить такой разрыв, вставьте строку между двумя строками и измените тип строки на **RelMoveTo.** Если **строка RelMoveTo** находится между двумя строками, она содержит координаты  *x*  и  *y*  первой вершины линии после разрыва пути фигуры.
