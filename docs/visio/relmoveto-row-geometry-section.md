# <a name="relmoveto-row-geometry-section"></a>RelMoveTo Row (Geometry Section)

Содержит  *x*  - и  *y-координаты*  первого вершины формы или  *x*  - и  *y-координаты*  первого вершины после перерыва в пути, относительно высоты и ширины фигуры. 
  
> [!NOTE]
> Строка **RelMoveTo** может сохраняться только в форматах файлов .vsdx, .vsdm, .vstx, .vstm, .vssx и .vssm file. Если файл сохранен в форматах 2003-2010 Visio, строка **RelMoveTo** преобразуется в строку [MoveTo.](moveto-row-geometry-section.md) 
  
Строка **RelMoveTo** содержит следующие ячейки. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |Если **строка RelMoveTo** является первой строкой в разделе, X-ячейка представляет  *x-координату*  первого вершины фигуры относительно ширины фигуры. Если **строка RelMoveTo** отображается между двумя строками, X-ячейка представляет  *x-координату*  первой вершины после разрыва пути.  <br/> |
|[Да](y-cell-geometry-section.md) <br/> |Если **строка RelMoveTo** является первой строкой в разделе, ячейка Y представляет  *y-координату*  первой вершины фигуры. Если **строка RelMoveTo** отображается между двумя строками, ячейка Y представляет  *y-координату*  первой вершины после перерыва в пути.  <br/> |
   
## <a name="remarks"></a>Примечания

Значения в строке **RelMoveTo** эквивалентны значениям в строке [MoveTo,](moveto-row-geometry-section.md) умножаемой на ширину и высоту фигуры. Например: строка **RelMoveTo,** где  значение X-ячейки равно "0", а ячейка Y — "0.5", может  быть заменена строкой **MoveTo,** где значение X-ячейки — формула "Ширина *0", а **ячейка Y***— формула "Высота 0.5".  
  
Строка **RelMoveTo** содержит  *x*  и  *y-координаты*  первого вершины для фигуры, если строка MoveTo является первой строкой в разделе. Обычно это первый вершина, помещаемая при нарисовании фигуры, и она не обязательно соответствует точке начала формы 1-D. 
  
Раздел **Геометрия** должен начинаться с **строки MoveTo** или **RelMoveTo,** но вы также можете использовать строку **RelMoveTo** и **Строку MoveTo** для представления пробела в поглаживание пути фигуры относительно ширины и высоты фигуры. Однако, когда путь используется для определения границы заполненного региона, этот разрыв интерпретируется как сегмент прямой линии. Чтобы вставить такой пробел, вставьте строку между двумя строками и измените тип строки **на RelMoveTo.** Если строка **RelMoveTo** находится между двумя строками, она содержит x  *-*  и  *y-координаты*  первого вершины линии после перерыва в пути фигуры.
