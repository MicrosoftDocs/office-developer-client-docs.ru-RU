# <a name="angletopar-function"></a>Функция ANGLETOPAR

Возвращает преобразованный угол в родительской системе координат конечной фигуры. Преобразует угол из локальных координат исходной фигуры в родительские координаты конечной фигуры. 
  
## <a name="syntax"></a>Синтаксис

ANGLETOPAR (* * *сркангле* * *, * * *сркреф* * *, * * *дстреф* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _сркангле_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Угол в исходной системе координат.  <br/> |
| _сркреф_ <br/> |Обязательный  <br/> |**String** <br/> | Ссылка на ячейку в исходном объекте, например фигуру, группу, страницу и т. д.  <br/> |
| _дстреф_ <br/> |Обязательный  <br/> |**String** <br/> |Ссылка на ячейку в целевом объекте, например фигуру, группу, страницу и т. д.  <br/> |
   
## <a name="remarks"></a>Примечания

Эта функция работает, даже если исходная и конечная фигуры находятся в группах. Он также настраивается для вращения и отражения в промежуточном преобразовании.
  
Координаты источника и назначения должны находиться на одной странице.
  
Если объект Destination является страницей, не имеющей родителя, результат выражается в локальных координатах страницы.
