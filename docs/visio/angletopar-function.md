# <a name="angletopar-function"></a>Функция ANGLETOPAR

Возвращает преобразованный угол в родительской системе координат конечной фигуры. Преобразует угол из локальных координат исходной фигуры в родительские координаты конечной фигуры. 
  
## <a name="syntax"></a>Синтаксис

ANGLETOPAR (** *srcAngle* **, ** *srcRef* **, ** *dstRef* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _srcAngle_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Угол в исходной системы координат.  <br/> |
| _srcRef_ <br/> |Обязательный  <br/> |**Строка** <br/> | Ссылка на ячейку в объекте источника, например фигуры, группы, страницы и т. д.  <br/> |
| _dstRef_ <br/> |Обязательный  <br/> |**Строка** <br/> |Ссылка на ячейку в целевой объект, например фигуры, группы, страницы и т. д.  <br/> |
   
## <a name="remarks"></a>Замечания

Эта функция работает даже в том случае, если исходный и целевой фигур, присутствующих в группах. Также настраивает для поворот и зеркальное отражение промежуточных преобразования.
  
Координаты исходной и целевой должен находиться на той же странице.
  
Если destination является страницу, которая не имеет родительского, результат выражается в локальной системе координат страницы.
