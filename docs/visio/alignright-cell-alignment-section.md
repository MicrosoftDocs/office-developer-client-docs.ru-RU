# <a name="alignright-cell-alignment-section"></a>Ячейка AlignRight (раздел "Выравнивание")

Определяет позицию по горизонтали, по отношению к источнику родительского объекта вертикальной направляющей или направляющей точки, относительно которого выровнена правая граница фигуры.
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку AlignRight по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AlignRight  <br/> |
   
Для получения ссылки на ячейки AlignRight по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignRight** <br/> |
