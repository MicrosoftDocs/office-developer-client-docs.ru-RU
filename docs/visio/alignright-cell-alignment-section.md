# <a name="alignright-cell-alignment-section"></a>AlignRight Cell (Alignment Section)

Определяет горизонтальное положение, относительно происхождения родительского, вертикального руководства или указателя, к которому выровнена правая граница фигуры.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку AlignRight по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | AlignRight  <br/> |
   
Чтобы получить ссылку на ячейку AlignRight по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignRight** <br/> |
