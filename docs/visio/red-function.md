# <a name="red-function"></a>КРАСНЫЙ функции

Возвращает значение красного компонента цвета. 
  
## <a name="syntax"></a>Синтаксис

КРАСНЫЙ (** *выражение* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**Может отличаться** <br/> |Индекс цвета в таблице цветов документа, выражение, которое разрешается в настраиваемый цвет (например, RGB или HSL) или ссылку на ячейку, содержащую цвет индекса или цветовой результатов.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Число
  
## <a name="remarks"></a>Замечания

Возвращаемое значение — это число в диапазоне от 0 до 255 включительно, или ссылка на ячейку, которая разрешается в индекс. Если _выражение_ является недопустимым, эта функция возвращает 0 (черный цвет). 
  
## <a name="example-1"></a>Пример 1

RED(22)
  
Возвращает 51, если в документе используется по умолчанию Microsoft Office Visio цветовой палитры, в котором темно-серый цвет по индексу 22.
  
## <a name="example-2"></a>Пример 2

Red(char.Color)
  
Возвращает значение красного компонента текущий цвет шрифта.
  
## <a name="example-3"></a>Пример 3

КРАСНЫЙ (RGB (10, 20, 30))
  
Возвращает значение 10.
