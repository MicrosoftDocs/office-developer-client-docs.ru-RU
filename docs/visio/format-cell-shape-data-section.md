# <a name="format-cell-shape-data-section"></a>Format Cell (Shape Data Section)

Указывает форматирование элемента данных фигуры, который является строкой, фиксированным списком, номером, списком переменной, датой или временем, продолжительностью или валютой.
  
## <a name="remarks"></a>Примечания

|**Тип элемента фигуры данных**|**Значение**|**Формат содержимого ячейки**|
|:-----|:-----|:-----|
| String  <br/> | 0  <br/> | Форматная картинка, соответствующая типу данных.  <br/> |
| Исправленный список  <br/> | 1  <br/> | Элементы, которые отображаются в списке, разделены зайколонами.  <br/> |
| Номер  <br/> | 2  <br/> | Форматная картинка, соответствующая типу данных.  <br/> |
| Переменный список  <br/> | 4   <br/> | Элементы, которые отображаются в списке, разделены зайколонами.  <br/> |
| Дата или время  <br/> | 5   <br/> | Форматная картинка, соответствующая типу данных.  <br/> |
| Длительность  <br/> | 6   <br/> | Форматная картинка, соответствующая типу данных.  <br/> |
| Денежный  <br/> | 7   <br/> | Форматная картинка, соответствующая типу данных.  <br/> |
   
В качестве примера указания форматной картинки, соответствующей типу данных, форматная картинка "# #/4 UU" имеет номер 12.43. как 12 2/4 ДЮЙМА. Дополнительные сведения о том, как указать изображение формата, см. в [рисунках формата.](about-format-pictures.md)
  
Пример указания элементов для списка : "Инженерия; Кадры; Продажи; Маркетинг".
  
Значения даты (тип = 5) отображаются в кратком формате даты. Значения валюты (тип = 7) отображаются с помощью текущего параметра пользователя для **валюты** на вкладке **Региональные** параметры в элементе Региональные и **Языковые** параметры в **панели управления.**
  
Число (тип = 2) может представлять размер, scalar, угол, дату, время или валюту. Чтобы число ввода всегда оценивалось как дата, время или валюта, используйте функцию DATETIME или CY в ячейке Format вместо изображения формата.
  
Чтобы получить ссылку на ячейку Format по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Prop.  *имя*  . Формат, в котором prop.  *имя*  — это имя строки  <br/> |
   
Чтобы получить ссылку на ячейку Format по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionProp** <br/> |
| Индекс строки:  <br/> |**visRowProp**  +   *i,* *где i* = 0, 1, 2 ...  <br/> |
| Индекс ячейки:  <br/> |**visCustPropsFormat** <br/> |
