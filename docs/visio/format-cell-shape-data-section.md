# <a name="format-cell-shape-data-section"></a>Format Cell (Shape Data Section)

Указывает форматирование элемента данных фигуры, который является строкой, фиксированным списком, числом, списком переменных, датой или временем, продолжительностью или валютой.
  
## <a name="remarks"></a>Примечания

|**Тип элемента данных фигуры**|**Значение**|**Форматирование содержимого ячейки**|
|:-----|:-----|:-----|
| String  <br/> | 0  <br/> | Изображение формата, подходящее для типа данных.  <br/> |
| Фиксированный список  <br/> | 1   <br/> | Элементы, которые должны отображаться в списке, разделенные с зачетами.  <br/> |
| Числовой  <br/> | 2   <br/> | Изображение формата, подходящее для типа данных.  <br/> |
| Список переменных  <br/> | 4   <br/> | Элементы, которые должны отображаться в списке, разделенные с зачетами.  <br/> |
| Дата и время  <br/> | 5   <br/> | Изображение формата, подходящее для типа данных.  <br/> |
| Duration  <br/> | 6   <br/> | Изображение формата, подходящее для типа данных.  <br/> |
| Валюта  <br/> | 7   <br/> | Изображение формата, подходящее для типа данных.  <br/> |
   
В качестве примера указания изображения формата, соответствующего типу данных, изображение формата "# #/4 UU" форматирование числа 12,43 в. как 12 2/4 ДЮЙМА. Дополнительные сведения об указании формата рисунка см. в [сведениях о формате изображений.](about-format-pictures.md)
  
Примером указания элементов для списка является "Engineering; Кадры; Продажи; Маркетинг".
  
Значения даты (тип = 5) отображаются в кратком формате даты. Значения валюты (тип = 7) отображаются с использованием  текущего  параметра валюты  пользователя на вкладке "Региональные параметры" в элементе "Региональные и языковые параметры" панели **управления.**
  
Число (тип = 2) может представлять измерение, скалярное, угол, дату, время или валюту. Чтобы обеспечить, чтобы входной номер всегда оценивался как дата, время или валюта, используйте функцию DATETIME или CY в ячейке Format вместо изображения формата.
  
Чтобы получить ссылку на ячейку Format по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Реквизит.  *name*  . Формат, где реквизит.  *name*  — это имя строки  <br/> |
   
Чтобы получить ссылку на ячейку Format по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionProp** <br/> |
| Индекс строки:  <br/> |**visRowProp**  +   *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visCustPropsFormat** <br/> |
