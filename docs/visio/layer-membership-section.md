# <a name="layer-membership-section"></a>Layer Membership Section

Содержит одну строку с перечислением каждого слоя, которому назначена фигура.
  
## <a name="remarks"></a>Примечания

Назначение уровня отображается в качестве индекса для списка уровней на странице. Индекс слоя соответствует порядку уровней в диалоговом **окне** "Свойства слоя"  (на вкладке "Главная" в группе редактирования щелкните "Слои" и выберите "Свойства  **слоя").** Первое имя в диалоговом окне — уровень 0, второе — уровень 1 и так далее.
  
Если фигура назначена более чем одному из уровней, каждый индекс слоя отображается в ячейке "Членство на уровне", отделяемой друг от друга с помощью четной четной четности.
  
Чтобы сослаться на значение ячейки Layer Membership в формуле, используйте имя **LayerMember.**
