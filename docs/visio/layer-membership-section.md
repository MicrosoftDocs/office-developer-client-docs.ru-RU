# <a name="layer-membership-section"></a>Layer Membership Section

Содержит одну строку со списком каждого слоя, которому назначена фигура.
  
## <a name="remarks"></a>Примечания

Назначение слоя отображается в виде индекса в списке слоев на странице. Индекс слоя соответствует порядку слоев в диалоговом окне **Свойства слоя** (на вкладке **Главная** , в группе **Правка** щелкните **слои**, а затем выберите **Свойства слоя**). Первое имя в диалоговом окне — Layer 0, второй — Layer 1 и т. д.
  
Если фигура назначена нескольким слоям, каждый индекс слоя отображается в ячейке членства слоя, разделенной точкой с запятой.
  
Чтобы сослаться на значение ячейки уровня членства в формуле, используйте имя **лайермембер**.
