# <a name="type-cell-shape-data-section"></a>Type Cell (Shape Data Section)

Указывает тип данных для значения данных фигуры.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Строка. Этот параметр используется по умолчанию.  <br/> |**visPropTypeString** <br/> |
|1   <br/> |Фиксированный список. Отображает элементы списка в выпадаемом поле со списком в диалоговом окне **"Определение данных** фигуры". Укажите элементы списка в ячейке Format. Пользователи могут выбрать в списке только один элемент.  <br/> |**visPropTypeListFix** <br/> |
|2   <br/> |Номер. Включает значения даты, времени, длительности и валюты, а также скаляры, измерения и углы. Укажите формат изображения в ячейке Format.  <br/> |**visPropTypeNumber** <br/> |
|3   <br/> |Boolean. Отображает false и TRUE, когда пользователи могут выбирать элементы из выпадаемого списка в диалоговом окне **"Определение данных** фигуры".  <br/> |**visPropTypeBool** <br/> |
|4   <br/> |Список переменных. Отображает элементы списка в выпадаемом поле со списком в диалоговом окне **"Определение данных** фигуры". Укажите элементы списка в ячейке Format. Пользователи могут выбрать элемент списка или ввести новый элемент, добавленный в текущий список в ячейке Format.  <br/> |**visPropTypeListVar** <br/> |
|5   <br/> |Значение даты или времени. Отображает дни, месяцы и годы, секунды, минуты и часы или объединенную дату и время. Укажите формат изображения в ячейке Format.  <br/> |**visPropTypeDate** <br/> |
|6   <br/> |Значение длительности. Отображает время, за который у вас есть время. Укажите формат изображения в ячейке Format.  <br/> |**visPropTypeDuration** <br/> |
|7   <br/> |Значение валюты. Использует текущие региональные параметры системы. Укажите формат изображения в ячейке Format.  <br/> |**visPropTypeCurrency** <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку Type по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Prop. *Name*  . Введите , где prop.  *Имя*  — это имя строки  <br/> |
   
Чтобы получить ссылку на ячейку Type по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionProp** <br/> |
|Индекс строки:  <br/> |**visRowProp**  +   *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visCustPropsType** <br/> |
