# <a name="type-cell-shape-data-section"></a>Тип ячейки (раздел данных фигуры)

Указывает тип данных для значения данных фигуры.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Строка. Это значение по умолчанию.  <br/> |**visPropTypeString** <br/> |
|1  <br/> |Фиксированный список. Отображает элементы списка в поле со списком раскрывающемся поле в диалоговом окне **Определение данных фигуры** . Указание элементов списков в формат ячейки. Пользователи могут выбрать только один элемент из списка.  <br/> |**visPropTypeListFix** <br/> |
|2  <br/> |Номер. Включает в себя дату, значения времени, продолжительность и валюты, а также скаляры, измерений и углы. Укажите изображение формата в формат ячейки.  <br/> |**visPropTypeNumber** <br/> |
|3  <br/> |Логическое значение. Отображается FALSE и значение TRUE, как элементы пользователи могут выбрать в раскрывающемся списке в диалоговом окне **Определение данных фигуры** .  <br/> |**visPropTypeBool** <br/> |
|4  <br/> |Список переменных. Отображает элементы списка в поле со списком раскрывающемся поле в диалоговом окне **Определение данных фигуры** . Указание элементов списков в формат ячейки. Пользователей можно выбрать элемент списка или ввести новый элемент, который добавляется в текущий список в ячейке формат.  <br/> |**visPropTypeListVar** <br/> |
|5  <br/> |Значение даты или времени. Отображение дней, месяцев и лет или секунд, минут и часов, или объединены значения даты и времени. Укажите изображение формата в формат ячейки.  <br/> |**visPropTypeDate** <br/> |
|6  <br/> |Значения длительности. Отображение времени, затраченного. Укажите изображение формата в формат ячейки.  <br/> |**visPropTypeDuration** <br/> |
|7  <br/> |Значение типа currency. Использует текущих региональных параметров компьютера. Укажите изображение формата в формат ячейки.  <br/> |**visPropTypeCurrency** <br/> |
   
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку тип по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Свойства. *Имя* . Тип которых свойств.  *Имя* — это имя строки  <br/> |
   
Для получения ссылки на ячейки типа по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionProp** <br/> |
|Индекс строки:  <br/> |**visRowProp** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visCustPropsType** <br/> |
