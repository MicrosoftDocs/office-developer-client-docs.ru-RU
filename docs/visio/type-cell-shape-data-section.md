# <a name="type-cell-shape-data-section"></a>Type Cell (Shape Data Section)

Указывает тип данных для значения данных фигуры.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Строка. Этот параметр используется по умолчанию.  <br/> |**visPropTypeString** <br/> |
|1  <br/> |Исправленный список. Отображает элементы списка в выпадаемом комбо-окне в диалоговом окне **Определить форму** данных. Укажите элементы списка в ячейке Format. Пользователи могут выбрать только один элемент из списка.  <br/> |**visPropTypeListFix** <br/> |
|2  <br/> |Номер. Включает значения даты, времени, продолжительности и валюты, а также масштабы, размеры и углы. Укажите изображение формата в ячейке Format.  <br/> |**visPropTypeNumber** <br/> |
|3  <br/> |Boolean. Отображает FALSE и TRUE в виде элементов, которые пользователи могут выбрать из окна списка в диалоговом окне **Define Shape Data.**  <br/> |**visPropTypeBool** <br/> |
|4   <br/> |Переменный список. Отображает элементы списка в выпадаемом комбо-окне в диалоговом окне **Определить форму** данных. Укажите элементы списка в ячейке Format. Пользователи могут выбрать элемент списка или ввести новый элемент, добавленный в текущий список в ячейке Format.  <br/> |**visPropTypeListVar** <br/> |
|5   <br/> |Значение даты или времени. Отображает дни, месяцы и годы или секунды, минуты и часы или совокупное значение даты и времени. Укажите изображение формата в ячейке Format.  <br/> |**visPropTypeDate** <br/> |
|6   <br/> |Значение продолжительности. Отображает по и сего времени. Укажите изображение формата в ячейке Format.  <br/> |**visPropTypeDuration** <br/> |
|7   <br/> |Значение валюты. Использует текущую региональную Параметры. Укажите изображение формата в ячейке Format.  <br/> |**visPropTypeCurrency** <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку Type по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Prop. *Name*  . Введите, где Prop.  *Имя*  — это имя строки  <br/> |
   
Чтобы получить ссылку на ячейку Type по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionProp** <br/> |
|Индекс строки:  <br/> |**visRowProp**  +   *i,* *где i* = 0, 1, 2 ...  <br/> |
|Индекс ячейки:  <br/> |**visCustPropsType** <br/> |
