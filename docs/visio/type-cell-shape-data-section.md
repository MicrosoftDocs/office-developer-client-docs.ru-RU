# <a name="type-cell-shape-data-section"></a>Type Cell (Shape Data Section)

Указывает тип данных для значения данных фигуры.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|нуль  <br/> |Строка. Этот параметр используется по умолчанию.  <br/> |**виспроптипестринг** <br/> |
|1,1  <br/> |Фиксированный список. Отображает элементы списка в раскрывающемся поле со списком в диалоговом окне **Определение данных фигуры** . Укажите элементы списка в ячейке Format. Пользователи могут выбрать только один элемент из списка.  <br/> |**виспроптипелистфикс** <br/> |
|2  <br/> |Значение. Включает в себя даты, время, продолжительность и денежные значения, а также скаляры, измерения и углы. Укажите формат рисунка в ячейке Format (формат).  <br/> |**виспроптипенумбер** <br/> |
|4  <br/> |Логического. Отображает FALSE и TRUE в качестве элементов, которые пользователи могут выбирать из раскрывающегося списка в диалоговом окне **Определение данных фигуры** .  <br/> |**виспроптипебул** <br/> |
|4   <br/> |Список переменных. Отображает элементы списка в раскрывающемся поле со списком в диалоговом окне **Определение данных фигуры** . Укажите элементы списка в ячейке Format. Пользователи могут выбрать элемент списка или ввести новый элемент, который добавляется в текущий список в ячейке Format.  <br/> |**виспроптипелиствар** <br/> |
|5   <br/> |Значение даты или времени. Отображает дни, месяцы и годы, а также секунды, минуты и часы, а также объединенное значение даты и времени. Укажите формат рисунка в ячейке Format (формат).  <br/> |**виспроптипедате** <br/> |
|6   <br/> |Значение длительности. Отображает затраченное время. Укажите формат рисунка в ячейке Format (формат).  <br/> |**виспроптипедуратион** <br/> |
|7   <br/> |Денежное значение. Использует текущие региональные параметры системы. Укажите формат рисунка в ячейке Format (формат).  <br/> |**виспроптипекурренци** <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку Type по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Установите. *Name (имя* ). Введите, где prop.  *Name* — имя строки  <br/> |
   
Чтобы получить ссылку на ячейку Type по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**виссектионпроп** <br/> |
|Индекс строки:  <br/> |**висровпроп** +  *i* , где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**вискустпропстипе** <br/> |
