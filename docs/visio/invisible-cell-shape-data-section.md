# <a name="invisible-cell-shape-data-section"></a>Ячейка Invisible (раздел "Данные фигуры")

Указывает, видима ли данные фигуры в окне **Данных фигуры** . 
  
|**Значение**|**Описание**|
|:-----|:-----|
| TRUE  <br/> | Данные фигуры не отображается.  <br/> |
| FALSE  <br/> | Данные фигуры является видимым.  <br/> |
   
## <a name="remarks"></a>Замечания

Значение в этой ячейке соответствующий флажок **скрытый** в диалоговом окне **Определение данных фигуры** (щелкните правой кнопкой мыши фигуру, выберите пункт **данные**и выберите **Определение данных фигуры**).
  
Для получения ссылки на ячейки невидимой по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Свойства.  *имя* . Невидимое where свойств.  *имя* — это имя строки  <br/> |
   
Для получения ссылки на ячейки невидимой по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionProp** <br/> |
| Индекс строки:  <br/> |**visRowProp** +  *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visCustPropsInvis** <br/> |
