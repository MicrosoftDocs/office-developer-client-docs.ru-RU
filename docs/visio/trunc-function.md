# <a name="trunc-function"></a>Функция TRUNC

Возвращает число, которое усекается до указанного количества цифр.
  
## <a name="syntax"></a>Синтаксис

ОТБР (* * *число* * *, * * *нумберофдигитс* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Число, которое необходимо усечь.  <br/> |
| _нумберофдигитс_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Количество цифр, на которые усекается _число_.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовых.
  
## <a name="remarks"></a>Примечания

Если _нумберофдигитс_ больше 0, _число_ усекается до _нумберофдигитс_ справа от десятичного разделителя. Если _нумберофдигитс_ имеет значение 0, _число_ усекается до целого. Если _нумберофдигитс_ меньше 0, _число_ усекается до _нумберофдигитс_ слева от десятичного разделителя. 
  
## <a name="example-1"></a>Пример 1

ОТБР (123.654, 2)
  
Возвращает 123,65.
  
## <a name="example-2"></a>Пример 2

ОТБР (123.654, 0)
  
Возвращает 123.
  
## <a name="example-3"></a>Пример 3

ОТБР (123.654,-1)
  
Возвращает 120.
