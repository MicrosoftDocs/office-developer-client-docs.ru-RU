# <a name="sortkey-cell-shape-data-section"></a>SortKey Cell (Shape Data Section)

Оценивается в виде строки, которая влияет на порядок, в котором перечислены элементы в **окне "Данные** фигуры". 
  
## <a name="remarks"></a>Примечания

Вычисление, используемого для сравнения значений SortKey, нечувствительно к региональным и конкретным случаям. Если значения SortKey равны, данные фигуры перечислены в порядке строки. Данные фигуры без ключа сортировки перечислены после данных фигуры, содержащих ключ сортировки.
  
Ниже приводится пример использования ключей сортировки для отображения данных фигуры в окне **"Данные** фигуры" в следующем порядке: "Номер элемента", "Количество", "Цена". 
  
 *Строка, метка*  *и SortKey*  относятся к ячейкам в строке данных фигуры. В этом случае были названы строки данных фигуры. 
  
|**Row**|**Label**|**SortKey**|
|:-----|:-----|:-----|
| Prop.Item  <br/> | Номер элемента  <br/> | 1   <br/> |
| Prop.Price  <br/> | ЦЕНА  <br/> | 3   <br/> |
| Prop.Quan  <br/> | Количество  <br/> | 2   <br/> |
   
Чтобы получить ссылку на ячейку SortKey по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Реквизит.  *Name*  . SortKey where Prop.  *Имя*  — это имя строки настраиваемого свойства  <br/> |
   
Чтобы получить ссылку на ячейку SortKey по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionProp** <br/> |
| Индекс строки:  <br/> |**visRowProp**  +   *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visCustPropsSortKey** <br/> |
