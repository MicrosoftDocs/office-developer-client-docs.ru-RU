# <a name="sortkey-cell-shape-data-section"></a>Ячейка SortKey (раздел "Данные фигуры")

Представляет собой строку, которая влияет на порядок, в котором перечислены элементы в окне **Данных фигуры** . 
  
## <a name="remarks"></a>Замечания

Вычисления, используется для сравнения значения SortKey языкового стандарта и без учета регистра. Если значения SortKey равны, данные фигуры указано в порядок строк. Данные фигуры, которые имеют ключ не сортировки перечислены после данных фигуры, которая содержит ключ сортировки.
  
Ниже приведен пример использования ключей сортировки для отображения данных фигуры в окне **Данных фигуры** в порядке: номер товара, количество, цена. 
  
 *Строки, метки* и *SortKey* обращение к ячейкам в строке данных фигуры. В этом случае имя строк данных фигуры. 
  
|**Row**|**Подпись**|**SortKey**|
|:-----|:-----|:-----|
| Prop.Item  <br/> | Номер элемента  <br/> | 1  <br/> |
| Prop.Price  <br/> | Цены  <br/> | 3  <br/> |
| Prop.Quan  <br/> | "Quantity" (Количество)  <br/> | 2  <br/> |
   
Чтобы получить ссылку на ячейку SortKey по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Свойства.  *Имя* . SortKey где свойств.  *Имя* — это имя настраиваемого свойства строки  <br/> |
   
Для получения ссылки на ячейки SortKey по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionProp** <br/> |
| Индекс строки:  <br/> |**visRowProp** +  *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visCustPropsSortKey** <br/> |
