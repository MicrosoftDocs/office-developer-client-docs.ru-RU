# <a name="sortkey-cell-shape-data-section"></a>SortKey Cell (Shape Data Section)

Оценивается как строка, которая влияет на порядок, в котором отображаются элементы в окне " **данные фигуры** ". 
  
## <a name="remarks"></a>Комментарии

Вычисление, используемое для сравнения значений SortKey, зависит от языкового стандарта и без учета регистра. Если значения SortKey совпадают, данные фигуры отображаются в порядке строк. Данные фигуры, у которых нет ключа сортировки, указаны после данных фигуры, содержащих ключ сортировки.
  
Ниже приведен пример использования ключей сортировки для отображения данных фигуры в окне " **данные фигуры** " в порядке: "номер товара", "количество", "цена". 
  
 *Строка, подпись* и *SortKey* ссылаются на ячейки в строке данных фигуры. В этом случае были именованы строки данных фигуры. 
  
|**Row**|**Label**|**SortKey**|
|:-----|:-----|:-----|
| Prop. Item  <br/> | Номер элемента  <br/> | 1,1  <br/> |
| Prop. Price  <br/> | ЦЕНА  <br/> | 4  <br/> |
| Prop. Куан  <br/> | Количество  <br/> | 2  <br/> |
   
Чтобы получить ссылку на ячейку SortKey по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Установите.  *Name (имя* ). SortKey, где prop.  *Name* — имя строки настраиваемого свойства  <br/> |
   
Чтобы получить ссылку на ячейку SortKey по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**Виссектионпроп** <br/> |
| Индекс строки:  <br/> |**висровпроп** +  *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**Вискустпропссорткэй** <br/> |
