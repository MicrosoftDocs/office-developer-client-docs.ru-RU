# <a name="linetolinex-cell-page-layout-section"></a>Ячейки LineToLineX (раздел макет страницы)

Определяет горизонтальную свободное пространство между все соединители на странице документа.
  
## <a name="remarks"></a>Замечания

Также можно задать значение данной ячейки в диалоговом окне **Макет и интервал по маршрутизации** . (На вкладке **Конструктор** щелкните стрелку **Параметры страницы** , нажмите кнопку **Макет и маршрутизации**и нажмите кнопку **интервалы**.)
  
Для получения ссылки на ячейки LineToLineX по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |LineToLineX  <br/> |
   
Для получения ссылки на ячейки LineToLineX по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPageLayout** <br/> |
|Индекс ячейки:  <br/> |**visPLOLineToLineX** <br/> |
