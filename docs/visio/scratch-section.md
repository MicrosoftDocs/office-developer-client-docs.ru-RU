# <a name="scratch-section"></a>Раздел "Вспомогательный"

Рабочая область для ввода и проверки формул, которые могут использоваться в других ячеек.
  
## <a name="remarks"></a>Замечания

В этом разделе можно добавить с помощью диалогового окна **Вставить раздел** . Щелкните правой кнопкой мыши в окне таблицы свойств фигуры и нажмите кнопку **Вставить раздел**.
  
В разделе **рабочие** обычно используется для изолирования повторяющиеся сложные вычисления. Если решение должно четко определенные цели, приходить использовать ячейки в разделе **Пользовательские ячейки** для ясности, так как ячейки пользователь может с именем. 
  
Ячейки в разделе **рабочие** используют единиц измерения двумя способами. X и Y ячеек используйте рисунка единиц измерения; До D ячеек не используйте единиц измерения. (В жаргон программистов C ячейки X и Y «вводятся» и ячейки с A по D «void».) Ячейки **Вспомогательный X** и **Y рабочие** часто используются для извлечения координаты *x* и *y* , такие как **PinX** и **PinY**или X и Y ячеек, найденные в ячейке раздел **геометрии** . Вспомогательный буквы от A до D можно отобразить независимо от устройства, можно указать ячеек. 
  
Дальнейший разницы способ таких ячеек хранения значения точек. Точка в Visio — это пакет данных single для координаты ( *x, y*). Если формула возвращает значение точки, это значение интерпретируется в одном из трех следующих способов в зависимости от формула находится в ячейке таблицы свойств фигуры. Ячейки, которые относятся к *x* -координаты (например, **PinX**или ячеек в столбце X раздел **геометрии** ) извлекать только что *x* -координации значение точки. Ячейки, которые относятся к *y* -координаты извлекать только что *y* -координации значение точки. 
  
Например, для Visio извлекает формулу `PNT(3,4)` следующими тремя способами. 
  
|**Cell**|**Если ввести**|**Visio обрабатывает ее как**|**Результат**|
|:-----|:-----|:-----|:-----|
| X   <br/> | `PNT(3,4)` <br/> | `PNTX(PNT(3,4))` <br/> | 3.0000 in.  <br/> |
| Да  <br/> | `PNT(3,4)` <br/> | `PNTY(PNT(3,4))` <br/> | 4.0000 in.  <br/> |
| A-D  <br/> | `PNT(3,4)` <br/> | `PNT(3,4)` <br/> | Раздела PNT (3.0000 дюймов, 4.0000 in.)  <br/> |
