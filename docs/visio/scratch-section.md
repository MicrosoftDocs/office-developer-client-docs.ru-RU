# <a name="scratch-section"></a>Scratch Section

Область работы для ввода и тестирования формул, на которые можно ссылаться другими ячейками.
  
## <a name="remarks"></a>Примечания

Этот раздел можно добавить с помощью диалоговое окно **Insert Section.** Щелкните правой кнопкой мыши в окне ShapeSheet и нажмите **кнопку Вставить раздел**.
  
Раздел **Scratch** обычно используется для изоляции повторных сложных вычислений. Если ваше решение имеет четко определенное назначение, для ясности целесообразно использовать ячейку в разделе **Пользовательские** определенные ячейки, так как ячейки пользователей можно назвать. 
  
Ячейки в **разделе Scratch** используют единицы двумя способами. Ячейки X и Y используют единицы рисования; Ячейки через D не используют единицы. (В жаргоне программистов C ячейки X и Y "введите", а ячейки A через D являются "недействительными".) Ячейки **Scratch X** и Scratch Y часто используются для производных *x-* и *y-координат,* таких как **PinX** и **PinY,** или ячеек X и **Y,** найденных в ячейке раздела **Геометрия.** Scratch cells A through D can display whatever units you specify. 
  
Еще одно отличие состоит в том, как эти ячейки хранят значения точечные. Точкой в Visio является единый пакет данных для координат *(x,y).* Когда формула возвращает значение точки, это значение интерпретируется одним из трех способов, в зависимости от ячейки ShapeSheet, в которую находится формула. Ячейки,  которые относятся к x-координатам (например, **PinX** или ячейкам в X-столбце раздела **Геометрия),** извлекают только *x-coordinate* часть значения точки. Ячейки, которые относятся к y-координатам, извлекают только *часть y-coordinate* значения точки.  
  
Например, Visio извлекает формулу `PNT(3,4)` тремя способами. 
  
|**Cell**|**Если ввести**|**Visio относится к нему как к**|**Результат**|
|:-----|:-----|:-----|:-----|
| X  <br/> | `PNT(3,4)` <br/> | `PNTX(PNT(3,4))` <br/> | 3.0000 в.  <br/> |
| Да  <br/> | `PNT(3,4)` <br/> | `PNTY(PNT(3,4))` <br/> | 4.0000 в.  <br/> |
| A-D  <br/> | `PNT(3,4)` <br/> | `PNT(3,4)` <br/> | PNT (3.0000 in., 4.0000 in.)  <br/> |
