# <a name="scratch-section"></a>Scratch Section

Область работы для ввода и тестирования формул, на которые могут ссылиться другие ячейки.
  
## <a name="remarks"></a>Примечания

Этот раздел можно добавить в диалоговом окне **"Вставка** раздела". Щелкните правой кнопкой мыши в окне таблицы фигур и выберите **"Вставить раздел".**
  
Раздел **"С нуля"** обычно используется для изоляции повторяются сложные вычисления. Если ваше решение имеет четко определенное назначение, лучше использовать ячейку  в разделе "Пользовательские ячейки" для ясности, так как ячейки пользователей можно назвать. 
  
Ячейки в разделе **"С** нуля" используют единицы по-разному. Ячейки X и Y используют единицы рисования; Ячейки от A до D не используют единицы. (В Яргоне программистов C ячейки X и Y являются "типными", а ячейки A–D — "void".) Ячейки "С нуля **X"** и **"Y"** часто используются для производных координат *x и* *y,* таких как **PinX** и **PinY,** или ячеек X и Y, найденных в ячейке раздела "Геометрия".  Ячейки с нуля от A до D могут отображать любые заказать единицы. 
  
Еще одно различие заключается в том, как эти ячейки хранят значения точечные значения. Точка в Visio — это один пакет данных для координаты *x,y.* Когда формула возвращает значение точки, это значение интерпретируется одним из трех способов, в зависимости от ячейки ShapeSheet, в которых находится формула. Ячейки, которые относятся к *координатам x* (например, **PinX** или ячейкам в столбце X раздела "Геометрия"), извлекают только часть *координат x* значения точки.  Ячейки, которые относятся к  *координатам y,*  извлекают только  *часть координаты Y*  значения точки. 
  
Например, Visio извлекает формулу  `PNT(3,4)` тремя способами. 
  
|**Cell**|**При вводе**|**Visio обрабатывает его как**|**Result**|
|:-----|:-----|:-----|:-----|
| X  <br/> | `PNT(3,4)` <br/> | `PNTX(PNT(3,4))` <br/> | 3,0000 in.  <br/> |
| Да  <br/> | `PNT(3,4)` <br/> | `PNTY(PNT(3,4))` <br/> | 4.0000 in.  <br/> |
| A-D  <br/> | `PNT(3,4)` <br/> | `PNT(3,4)` <br/> | PNT(3.0000 in., 4.0000 in.)  <br/> |
