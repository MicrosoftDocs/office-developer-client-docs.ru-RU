# <a name="actions-row-actions-section"></a>Actions Row (Actions Section)

Содержит ячейки, указывающие действия, связанные с настраиваемыми командами в контекстном меню или меню тегов действий. Раздел действия содержит по одной строке действий для каждого действия.
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
Строки действий называются действиями. *имя* и содержит следующие ячейки. Более подробную информацию можно найти в разделах, посвященных определенным ячейкам. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[Действие](action-cell-actions-section.md) <br/> |Содержит формулу, которая должна выполняться, когда пользователь выбирает элемент в контекстном меню или меню тегов действий.  <br/> |
|[Меню](menu-cell-actions-section.md) <br/> |Определяет имя элемента меню, который отображается в теге действия или в контекстном меню.  <br/> |
|[TagName](tagname-cell-actions-section.md) <br/> |Логическое имя тега действия, в котором должно отображаться действие.  <br/> |
|[ButtonFace](buttonface-cell-actions-section.md) <br/> |Определяет значок, который появляется рядом с элементом в контекстном меню или меню тегов действий.  <br/> |
|[SortKey](sortkey-cell-actions-section.md) <br/> |Число, определяющее порядок пунктов меню в теге действия или в контекстном меню.  <br/> |
|[Checked](checked-cell-actions-section.md) <br/> |Указывает, выбран ли элемент меню в теге действия или в контекстном меню.  <br/> |
|[Disabled](disabled-cell-actions-section.md) <br/> |Указывает, отключен ли элемент меню в контекстном меню или меню тегов действий.  <br/> |
|[ReadOnly](readonly-cell-actions-section.md) <br/> |Указывает, доступен ли элемент меню только для чтения (невозможно щелкнуть).  <br/> |
|[Скрытым](invisible-cell-actions-section.md) <br/> |Указывает, виден ли элемент меню в теге действия или в контекстном меню.  <br/> |
|[BeginGroup](begingroup-cell-actions-section.md) <br/> |Указывает, следует ли вставить разделитель в меню над элементом меню.  <br/> |
   
## <a name="remarks"></a>Замечания

 Можно добавить любое количество действий.  ** укажите имена строк, назначьте им значимые имена и задайте значения ячеек. Чтобы добавить пользовательскую команду в раздел существующие действия, щелкните строку правой кнопкой мыши и выберите команду **Вставить строку** в контекстном меню. 
  
Вы можете ссылаться на эти ячейки по имени строки, которое отображается в окне таблицы свойств фигуры в виде красного текста. , Чтобы присвоить действием понятное имя. Строка *имени* щелкните строку, а затем введите имя (например, *Custom* ), например, чтобы создать имя строки Actions. Custom. Затем вы можете ссылаться на ячейку меню с помощью команды Actions. Custom. Menu. 
  
Введенное имя строки должно быть уникальным в пределах раздела.
