# <a name="actions-row-actions-section"></a>Строки действий (раздела "действия")

Содержит ячейки, которые определяют действия, связанные с пользовательской команды меню тега ярлык или действие. В разделе действия содержит одну строку действия для каждого действия.
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий, называются смарт-тегов. 
  
Строки действий с именем действия. *имя* и содержат следующие ячейки. Для получения дополнительных сведений см определенную ячейку. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[Действие](action-cell-actions-section.md) <br/> |Содержит формулу для выполнения, если пользователь выбирает пункт меню тега ярлык или действие.  <br/> |
|[Меню](menu-cell-actions-section.md) <br/> |Определяет имя элемента меню, которое отображается в действие тег или контекстное меню.  <br/> |
|[TagName](tagname-cell-actions-section.md) <br/> |Логическое имя тега действия, в которой должен отображаться это действие.  <br/> |
|[ButtonFace](buttonface-cell-actions-section.md) <br/> |Определяет значок, который отображается рядом с элементом меню тега ярлык или действие.  <br/> |
|[SortKey](sortkey-cell-actions-section.md) <br/> |Число, определяющее порядок в действие тег или контекстное меню.  <br/> |
|[Проверка](checked-cell-actions-section.md) <br/> |Указывает, установлен ли флажок действие тег или контекстное меню элемента меню.  <br/> |
|[Отключено](disabled-cell-actions-section.md) <br/> |Указывает, отключена ли элемент меню действия или контекстное меню тега.  <br/> |
|[Только для чтения](readonly-cell-actions-section.md) <br/> |Указывает, является ли элемент меню только для чтения (не может быть выбрана).  <br/> |
|[Невидимое](invisible-cell-actions-section.md) <br/> |Указывает, видима ли в тег или контекстное меню Действие элемента меню.  <br/> |
|[BeginGroup](begingroup-cell-actions-section.md) <br/> |Указывает, следует ли вставить разделитель в меню над элемента меню.  <br/> |
   
## <a name="remarks"></a>Замечания

 Можно добавить столько действий.  *имя* строки при необходимости назначить понятные имена строк и задайте значения ячейки. Чтобы добавить пользовательской команды в существующий раздел действий, щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить строку** в контекстном меню. 
  
Можно ссылаться на эти ячейки с их имя строки, который отображается в окне таблицы свойств фигуры текстом красного цвета. Чтобы назначить понятное имя действия. *имя* строки, щелкните строку и введите имя, например *Custom* , например для создания имя строки Actions.Custom. Затем можно ссылаться меню ячейки с помощью Actions.Custom.Menu. 
  
Имя строки при вводе должно быть уникальным в рамках раздела.
