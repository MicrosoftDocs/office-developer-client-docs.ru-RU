# <a name="gluetype-cell-glue-info-section"></a>GlueType Cell (Glue Info Section)

Определяет, использует ли однообразная фигура статический (точка-точка) или динамический (от фигуры к фигуре) при наклеии к другой фигуре.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| &amp;H0  <br/> | Значение, используемое по умолчанию. Разрешить динамическое приклеить только для динамического соединитела; в противном случае используйте статический приклейки.  <br/> |**visGlueTypeDefault** <br/> |
| &amp;H1  <br/> | Разрешить динамическое приклеить.  <br/> | Устаревшее в Visio 2002  <br/> |
| &amp;H2  <br/> | Разрешить динамическое приклеить.  <br/> |**visGlueTypeType** <br/> |
| &amp;H4  <br/> | Не разрешайте динамическое приклеить.  <br/> |**visGlueTypeNo Дейсинг** <br/> |
| &amp;H8  <br/> | Не разрешайте связывать эту двух d-фигуру с помощью динамического приклейки.  <br/> |**visGlueTypeNoTypeNoTypeingTo** <br/> |
   
## <a name="remarks"></a>Примечания

Если эта ячейка содержит значение 1, 2 или 3, динамический приклей будет установлен при любом из следующих значений:
  
- Фигуры динамически привяются в пользовательском интерфейсе.
    
- Фигуры прикрепиваются к ячейке PinX или PinY другой фигуры из программы.
    
Если фигуры прикрепиваются к ячейкам фигур, кроме PinX или PinY из программы, используется статический приклейки.
  
Если изменить это значение, не разрешив динамическое приклеить, существующий динамический приклей не изменится. Программы могут устанавливать динамический приклейки, даже если динамический приклейки отключен в ячейке GlueType.
  
Чтобы получить ссылку на ячейку GlueType по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | GlueType  <br/> |
   
Чтобы получить ссылку на ячейку GlueType по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowMisc** <br/> |
| Индекс ячейки:  <br/> |**visGlueType** <br/> |
