# <a name="gluetype-cell-glue-info-section"></a>GlueType Cell (Glue Info Section)

Определяет, использует ли Одномерная фигура статическую ("точка-точка") или динамическую (фигурную фигуру) приклеить к другой фигуре.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| &amp;H0  <br/> | Значение, используемое по умолчанию. Разрешить динамическое приклеивание только для динамического соединителя; в противном случае используйте статическую привязывание.  <br/> |**висглуетипедефаулт** <br/> |
| &amp;H1  <br/> | Разрешить динамическое привязывание.  <br/> | Устаревшие в Visio 2002  <br/> |
| &amp;H2  <br/> | Разрешить динамическое привязывание.  <br/> |**висглуетипевалкинг** <br/> |
| &amp;H4  <br/> | Не разрешать динамическое приклеивание.  <br/> |**висглуетипеновалкинг** <br/> |
| &amp;H8  <br/> | Не разрешать подключаться к этой двумерной фигуре с динамической привязкой.  <br/> |**висглуетипеновалкингто** <br/> |
   
## <a name="remarks"></a>Примечания

Если эта ячейка содержит значение 1, 2 или 3, динамическое приклеивание будет установлено в следующих случаях:
  
- Фигуры динамически связаны в пользовательском интерфейсе.
    
- Фигуры привязываются к ячейке PinX или PinY другой фигуры из программы.
    
Если фигуры привязываются к ячейкам фигур, отличным от PinX или PinY, в программе, то используется статическая приклеивание.
  
Изменение этого значения с "разрешить" без разрешения динамической приклеивание не является критичным или изменяет существующее динамическое привязывание. Программы могут устанавливать динамическую привязывание, даже если динамическое приклеивание отключено в ячейке GlueType.
  
Чтобы получить ссылку на ячейку GlueType по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | GlueType  <br/> |
   
Чтобы получить ссылку на ячейку GlueType по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**висровмиск** <br/> |
| Индекс ячейки:  <br/> |**висглуетипе** <br/> |
