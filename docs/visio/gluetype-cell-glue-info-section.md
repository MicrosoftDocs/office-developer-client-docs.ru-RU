# <a name="gluetype-cell-glue-info-section"></a>Ячейка GlueType (раздел "Сведения о приклеивании")

Определяет, используется ли одномерной фигуры static (точка-точка) или динамической связывающих (фигура к фигуре), когда он связан с другой фигурой.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| &amp;H0  <br/> | Значение, используемое по умолчанию. Разрешить динамические связывающих для динамического соединителя. в противном случае используйте статические связывающих.  <br/> |**visGlueTypeDefault** <br/> |
| &amp;H1  <br/> | Разрешить динамические связывающих.  <br/> | Устаревший атрибут в Visio 2002  <br/> |
| &amp;H2  <br/> | Разрешить динамические связывающих.  <br/> |**visGlueTypeWalking** <br/> |
| &amp;H4  <br/> | Не разрешать динамического связывающих.  <br/> |**visGlueTypeNoWalking** <br/> |
| &amp;H8  <br/> | Не разрешать в этом плоских фигур связаны с динамических связывающих.  <br/> |**visGlueTypeNoWalkingTo** <br/> |
   
## <a name="remarks"></a>Замечания

Если эта ячейка содержит значение 1, 2 или 3, динамический связывающих будет установлено когда происходит одно из следующих:
  
- Фигур динамически связан в пользовательском интерфейсе.
    
- Фигуры являются привязку PinX или PinY ячейки другую фигуру из программы.
    
Если фигуры являются привязку ячейки фигуры, отличный от PinX или PinY из программы, статические связывающих используется.
  
При изменении данного значения из разрешение не позволяет динамической связывающих не сервера или изменение существующей динамической связывающих. Программы можно установить динамических связывающих, даже если динамических связывающих отключена в ячейке приклеивания.
  
Для получения ссылки на ячейки приклеивания по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Приклеивания  <br/> |
   
Для получения ссылки на ячейки приклеивания по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowMisc** <br/> |
| Индекс ячейки:  <br/> |**visGlueType** <br/> |
