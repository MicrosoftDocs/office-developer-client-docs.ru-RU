# <a name="bitand-function"></a>Функция BITAND

Возвращает 16-битный двоичный номер, в котором каждый бит имеет 1, только если соответствующий бит в двоичном номере1 и двоичном номере2 имеет 1. В противном случае для бита устанавливается 0. 
  
## <a name="syntax"></a>Синтаксис

BITAND(***binarynumber1***, ***binarynumber2*** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _binary number1_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Первый 16-битный двоичный номер.  <br/> |
| _binary number2_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Второй 16-битный двоичный номер.  <br/> |
   
## <a name="remarks"></a>Примечания

Эту функцию можно использовать для проверки и изменения свойств фигуры, которые хранятся в виде битовых вешек, например для текстового формата фигуры.
  
## <a name="example"></a>Пример

BITAND(12,6)
  
Возвращает 4. 12 = 0...01100. 6 = 0...00110. Таким образом, BITAND(12,6) = 0...00100.
