# <a name="move-around-a-shapesheet-window"></a>Перемещение по окну ShapeSheet

Чтобы переместиться из ячейки в ячейку в окне ShapeSheet, используйте стрелку и клавиши TAB. Щелкните ячейку, чтобы активировать ее или прокрутите ячейки ShapeSheet, не изменяя активную ячейку.
  
Вы также можете использовать методы в следующей таблице.
  
|**Задача**|**Рекомендации**|
|:-----|:-----|
| Активация ячейки  <br/> | Щелкните ячейку указателем.  <br/> |
| Выберите про номерной ряд ячеек  <br/> | Щелкните номер строки.  <br/> |
| Перемещение левой ячейки  <br/> | Нажмите клавишу LEFT ARROW или SHIFT+TAB.  <br/> |
| Перемещение правой ячейки  <br/> | Нажмите клавишу RIGHT ARROW или TAB.  <br/> |
| Перемещение одной ячейки  <br/> | Нажмите клавишу UP ARROW.  <br/> |
| Перемещение одной ячейки  <br/> | Нажмите клавишу DOWN ARROW.  <br/> |
| Перемещение в последнюю ячейку строки  <br/> | Нажмите КНОПКУ КОНЕЦ.  <br/> |
| Перемещение в первую ячейку строки  <br/> | Нажмите КНОПКУ HOME.  <br/> |
| Прокрутите вниз один экран  <br/> | Нажмите PAGEDOWN.  <br/> |
| Прокрутите один экран  <br/> | Нажмите PAGEUP.  <br/> |
| Прокрутите правый один экран  <br/> | Нажмите кнопку CTRL+PAGEDOWN.  <br/> |
| Прокрутите левый экран  <br/> | Нажмите кнопку CTRL+PAGEUP.  <br/> |
| Прокрутите в нижний правый угол окна  <br/> | Нажмите кнопку CTRL+END.  <br/> |
| Прокрутите в верхний правый угол окна  <br/> | Нажмите кнопку CTRL+HOME.  <br/> |
