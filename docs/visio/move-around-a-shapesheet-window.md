# <a name="move-around-a-shapesheet-window"></a>Перемещение по окну таблицы свойств фигуры

Для перемещения между ячейками в окне таблицы свойств фигуры, используйте стрелки и клавишу TAB. Щелкните ячейку, чтобы включить его или прокрутите ячейки таблицы свойств фигуры, не изменяя активную ячейку.
  
Также можно использовать методы в следующей таблице.
  
|**Задача**|**Рекомендации**|
|:-----|:-----|
| Активация ячейки  <br/> | Щелкните ячейку с помощью указателя мыши.  <br/> |
| Выберите нумерованный строку ячеек  <br/> | Щелкните номер строки.  <br/> |
| Переход на одну ячейку влево  <br/> | Нажмите клавишу Стрелка влево или SHIFT + TAB.  <br/> |
| Перемещение одну ячейку вправо  <br/> | Нажмите клавишу Стрелка вправо или TAB.  <br/> |
| Вверх на одну ячейку  <br/> | Нажмите клавишу Стрелка вверх.  <br/> |
| Перемещение вниз на одну ячейку  <br/> | Нажмите клавишу Стрелка вниз.  <br/> |
| Переход к последней ячейке строки  <br/> | Нажмите клавишу END.  <br/> |
| Перемещение к первой ячейке в строке  <br/> | Нажмите клавишу HOME.  <br/> |
| Прокрутите список вниз на один экран  <br/> | Нажмите клавишу PAGEDOWN.  <br/> |
| Прокрутить вверх на один экран  <br/> | Нажмите клавишу PAGEUP.  <br/> |
| Прокрутки вправо на один экран  <br/> | Нажмите сочетание клавиш CTRL + PAGEDOWN.  <br/> |
| Прокрутите список один экран влево  <br/> | Нажмите сочетание клавиш CTRL + PAGEUP.  <br/> |
| Прокрутите список в правом нижнем углу окна  <br/> | Нажмите сочетание клавиш CTRL + END.  <br/> |
| Прокрутите список в правом верхнем углу окна  <br/> | Нажмите сочетание клавиш CTRL + HOME.  <br/> |
