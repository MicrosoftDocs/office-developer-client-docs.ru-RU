# <a name="softedgessize-cell-additional-effect-properties-section"></a>Ячейка SoftEdgesSize (раздел "Дополнительные свойства эффекта")

Определяет размер эффекта программных пограничного сервера в пунктах от 0,00 для 100,00. Если ячейка **SoftEdgesSize** имеет значение 0, фигура не имеет сглаживание. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **SoftEdgesSize** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | SoftEdgesSize  <br/> |
   
Для получения ссылки на ячейки **SoftEdgesSize** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowOtherEffectProperties** <br/> |
| Индекс ячейки:  <br/> |**visSoftEdgesSize** <br/> |
