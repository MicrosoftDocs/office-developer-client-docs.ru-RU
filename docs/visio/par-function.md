# <a name="par-function"></a>Функция PAR

Возвращает координаты _x и y_ точки в системе координат родительского элемента фигуры. 
  
## <a name="syntax"></a>Синтаксис

НОМИНАЛ (* * *Point* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Указывает_ <br/> |Обязательна  <br/> |**Число, число** <br/> |Координаты точки в системе координат текущей фигуры.  <br/> |
   
## <a name="remarks"></a>Примечания

В Microsoft Visio точка — это одно значение, ембодиес с двумя координатами *x* и *y* . Если фигура находится в группе, ее родитель — это группа. Если фигура не входит в группу, ее родителем является страница. 
  
## <a name="example"></a>Пример

НОМИНАЛ (PNT (PinX, PinY)) 
  
В этом выражении PNT преобразует две координаты в текущей фигуре в точку. Номинал затем преобразует точку в комбинацию координат относительно левого нижнего угла страницы или группы, в которой содержится текущая фигура.
