# <a name="min-function"></a>Функция MIN

Возвращает наименьшее число из списка. Наименьшее означает, что наиболее близкого к отрицательным infinity.
  
## <a name="syntax"></a>Синтаксис

MIN (** *Число1* **, ** *число2* **,..., ** *numberN* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Число1_ <br/> |Обязательный  <br/> |**Разные** <br/> |Первый номер в списке.  <br/> |
| _число2_ <br/> |Optional  <br/> |**Разные** <br/> | Второй номер в списке.  <br/> |
| _numberN_ <br/> |Optional  <br/> |**Разные** <br/> |N-й номер в списке.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Разные
  
## <a name="example"></a>Пример

MIN (13 в 1 ft 20 см) 
  
Возвращает 20 см.
