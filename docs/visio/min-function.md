# <a name="min-function"></a>Функция MIN

Возвращает наименьшее число из списка. Наименьшие средства ближе всего к отрицательной бесконечности.
  
## <a name="syntax"></a>Синтаксис

MIN(** *number1* **, ** *number2* **,..., ** *numberN* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число1_ <br/> |Обязательный  <br/> |**Разные** <br/> |Первый номер в списке.  <br/> |
| _число2_ <br/> |Необязательный  <br/> |**Разные** <br/> | Второй номер в списке.  <br/> |
| _numberN_ <br/> |Необязательный  <br/> |**Разные** <br/> |Номер nth в списке.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Разные
  
## <a name="example"></a>Пример

MIN (13 в,1 фута, 20 см) 
  
Возвращает 20 сантиметров.
