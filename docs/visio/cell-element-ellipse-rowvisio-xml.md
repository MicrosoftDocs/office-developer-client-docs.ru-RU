# <a name="cell-element-ellipse-row-visio-xml"></a>Элемент ячейки (эллипс строка) ('Visio XML»)

Содержит x - или координаты y центральной точки и два аспекта на эллипс эллипса.
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|**Тип элемента** <br/> |[Cell_Type](cell_type-complextypevisio-xml.md) <br/> |
|**Пространство имен** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Файл схемы** <br/> |VisioSchema15.xsd  <br/> |
|**Части документа** <br/> |главные # .xml, страницы # .xml  <br/> |
   
## <a name="definition"></a>Определение

```XML
< xs:element name="Cell" type="Cell_Type" minOccurs="0" maxOccurs="unbounded" >
</xs:element >
```

## <a name="remarks"></a>Замечания

Атрибут **N** этого элемента **ячейки** должен быть один из ограниченный набор значений, которые соответствуют ячейки таблицы свойств фигуры. Обратитесь к таблице ниже для определения значений атрибутов **N** , разрешенных для этого элемента **ячейки** . 
  
|**Значение**|**Описание**|**Дополнительные сведения**|
|:-----|:-----|:-----|
|X   <br/> |Координата x центральной точки.  <br/> |[Строка Ellipse (раздел "Геометрия")](ellipse-row-geometry-section.md) <br/> |
|Да  <br/> |Координата y центральной точки.  <br/> |[Строка Ellipse (раздел "Геометрия")](ellipse-row-geometry-section.md) <br/> |
|A  <br/> |X координата первой точкой на эллипс; в сочетании с представленной ячейки B по оси y.  <br/> |[Строка Ellipse (раздел "Геометрия")](ellipse-row-geometry-section.md) <br/> |
|B  <br/> |Координата первой точкой на эллипс; в сочетании с представленной ячейке по оси x.  <br/> |[Строка Ellipse (раздел "Геометрия")](ellipse-row-geometry-section.md) <br/> |
|C  <br/> |Координата x второй точки на эллипс; в сочетании с представленной ячейке D по оси y.  <br/> |[Строка Ellipse (раздел "Геометрия")](ellipse-row-geometry-section.md) <br/> |
|D  <br/> |Координата второй точки на эллипс; в сочетании с представленной ячейки C по оси y.  <br/> |[Строка Ellipse (раздел "Геометрия")](ellipse-row-geometry-section.md) <br/> |
