# <a name="shade-function"></a>Функция SHADE

Изменяет цвет, уменьшая его светоотступность на количество (положительное или отрицательное), указанное в _параметре int._ 
  
## <a name="syntax"></a>Синтаксис

SHADE(** *color* **, ** *int* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Индекс Visio microsoft или RGB-значение цвета.  <br/> |
| _int_ <br/> |Обязательный  <br/> |**Integer** <br/> |Количество, с помощью которого можно уменьшить светоосякость цвета. Может быть положительным или отрицательным.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 **RGB**
  
## <a name="remarks"></a>Примечания

Верхние и нижние границы светопоявтости 0 и 240 соответственно. Размер конечного параметра int не ограничивается,  но светоотступность никогда не превышает этих ограничений.
