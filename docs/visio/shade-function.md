# <a name="shade-function"></a>Функция SHADE

Изменяет цвет, уменьшая его яркость на величину (положительный или отрицательный), указанную в параметре _int_ . 
  
## <a name="syntax"></a>Синтаксис

Тень (* * *Цвет* * *, * * *int* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Цветовой индекс Microsoft Visio или значение RGB цвета.  <br/> |
| _int_ <br/> |Обязательный  <br/> |**Integer** <br/> |Величина, на которую уменьшается яркость цвета. Может быть положительным или отрицательным.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 **RGB**
  
## <a name="remarks"></a>Комментарии

Верхние и нижние пределы яркости — 0 и 240 соответственно. Не существует ограничения на размер целого числа, которое можно передать для параметра _int_ , но яркость не превышает эти ограничения.
