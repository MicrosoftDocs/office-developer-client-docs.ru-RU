# <a name="label-cell-shape-data-section"></a>Ячейка Label (раздел "Данные фигуры")

Метка, которая отображается для пользователей в окне **Данных фигуры** . Элемент label состоит из буквенно-цифровых символов, включая символ подчеркивания (_). 
  
## <a name="remarks"></a>Замечания

Приложение автоматически заключает строку подписи в кавычки в ячейке, но кавычки не отображаются в окне **Данных фигуры** . 
  
Если текст не метка не найден, Visio имя строки (Prop.Row) в окне **Данных фигуры** . 
  
Для получения ссылки на ячейки метки по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Свойства. *Имя* . Метка где свойств.  *Имя* — это имя строки  <br/> |
   
Для получения ссылки на ячейки метки по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionProp** <br/> |
|Индекс строки:  <br/> |**visRowProp** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visCustPropsLabel** <br/> |
