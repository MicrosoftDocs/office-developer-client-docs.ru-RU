# <a name="docmd-function"></a>Функция DOCMD

Выполняет указанную команду.
  
## <a name="syntax"></a>Синтаксис

 **DOCMD** ( _идентификатор команды_)
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Идентификатор команды_ <br/> |Обязательный  <br/> |**Число** <br/> | Чтобы выполнить команду.  <br/> |
   
## <a name="remarks"></a>Замечания

Чтобы получить список команды, которые поддерживают функцию DOCMD приведены в разделе «DoCmd/DOCMD команды» в справочник по автоматизации Microsoft Visio 2013. 
  
## <a name="example"></a>Пример

 `DOCMD (1312)`
  
Вызывает диалоговое окно " **Данные фигуры** " отображаться в пользовательском интерфейсе.
