# <a name="xrulerorigin-cell-ruler-amp-grid-section"></a>Ячейка XRulerOrigin (линейки &amp; сетка)

Задает координат на оси x линейке для страницы.
  
## <a name="remarks"></a>Замечания

В этой ячейке соответствует горизонтальной **линейки нулевой** параметр на странице **линейки &amp; сетки** диалоговое окно (на вкладке **Вид** щелкните стрелку **Показать** ). 
  
Чтобы получить ссылку на ячейку XRulerOrigin по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |XRulerOrigin  <br/> |
   
Для получения ссылки на ячейки XRulerOrigin по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowRulerGrid** <br/> |
|Индекс ячейки:  <br/> |**visXRulerOrigin** <br/> |
