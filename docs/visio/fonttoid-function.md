# <a name="fonttoid-function"></a>Функция FONTTOID

Возвращает идентификатор указанного шрифта.
  
> [!NOTE]
> Рекомендуется использовать функцию [FONT,](font-function.md) аналогичную функции **FONTTOID.** В Microsoft Visio 2013 г. функция **FONT** была представлена для замены **FONTTOID.** 
  
## <a name="syntax"></a>Синтаксис

FONTTOID(** *font_name* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _font_name_ <br/> |Обязательный  <br/> |**String** <br/> |Имя шрифта.  <br/> |
