# <a name="avoidpagebreaks-cell-page-layout-section"></a>Ячейка AvoidPageBreaks (раздел "Макет страницы")

Определяет, можно ли поместить фигур над разрывов страниц при фигур постепенного выравнивания, последовательно расположенные или оба.
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку AvoidPageBreaks по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте следующие. 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |AvoidPageBreaks  <br/> |
   
Для получения ссылки на ячейки AvoidPageBreaks по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPageLayout** <br/> |
|Индекс ячейки:  <br/> |**visPLOAvoidPageBreaks** <br/> |
