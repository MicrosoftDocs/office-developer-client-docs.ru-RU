# <a name="pagescale-cell-page-properties-section"></a>PageScale Cell (Page Properties Section)

Определяет значение единицы страницы в текущей шкале рисования. Шкала рисования для страницы — это отношение элемента страницы, показанного в ячейке PageScale, к единице рисования, показанной в ячейке DrawingScale.
  
## <a name="remarks"></a>Примечания

Вы также можете установить значение ячейки  PageScale на  вкладке "Шкала рисования" в диалоговом окне Настройка страницы (на вкладке **Дизайн** щелкните стрелку **установки** страницы). Значение ячейки является первым из двух чисел в заранее  определенном поле шкалы или настраиваемом окне масштабирования, в зависимости от параметра масштабирования рисования, выбранного в **масштабе Drawing**.  Например, если для рисунка выбрана архитектурная шкала, шкала рисования для страницы 3/32 " = 1'0". Значение в ячейке PageScale — 0.0938. (или 3/32), а значение в ячейке DrawingScale составляет 1 фут.
  
Чтобы получить ссылку на ячейку PageScale по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |PageScale  <br/> |
   
Чтобы получить ссылку на ячейку PageScale по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPage** <br/> |
|Индекс ячейки:  <br/> |**visPageScale** <br/> |
