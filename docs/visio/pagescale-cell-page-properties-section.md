# <a name="pagescale-cell-page-properties-section"></a>Ячейка PageScale (раздел "Свойства страницы")

Определяет значение единицы страницы в текущем масштабе документа. Масштаб документа для страницы представляет собой отношение единицы страницы в ячейке PageScale к единице документа в ячейке DrawingScale.
  
## <a name="remarks"></a>Замечания

Значение ячейки PageScale также можно настроить на вкладке **Масштаб документа** в диалоговом окне " **Параметры страницы** " (на вкладке " **Конструктор** ", щелкните стрелку **Параметры страницы** ). Значение ячейки это первая из двух чисел в поле **предопределенный масштаб** или **Настраиваемый масштаб** , в зависимости от того, размеры масштаба параметра, выбранного в разделе **Масштаб документа**. Например при выборе архитектурные масштаба рисунка масштаб документа для страницы — 3/32 "= 1'0". Значение в ячейке PageScale — 0.0938 in. (или 3/32"), а значение в ячейке DrawingScale — 1 ft.
  
Чтобы получить ссылку на ячейку PageScale по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |PageScale  <br/> |
   
Для получения ссылки на ячейки PageScale по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPage** <br/> |
|Индекс ячейки:  <br/> |**visPageScale** <br/> |
