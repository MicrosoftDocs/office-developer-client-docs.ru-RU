# <a name="opentextwin-function"></a>Функция OPENTEXTWIN

Открывает блок текста фигуры, поэтому можно редактировать текст.
  
## <a name="syntax"></a>Синтаксис

(OPENTEXTWIN)
  
## <a name="remarks"></a>Замечания

В более ранних версиях Visio эта функция отображается в виде _OPENTEXTWIN. Visio версии 4.0 и более поздних версий принимать любого из стилей.
