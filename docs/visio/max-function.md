# <a name="max-function"></a>Функция MAX

Возвращает наибольшее число из списка. Самый крупный — ближайший к плюс бесконечности.
  
## <a name="syntax"></a>Синтаксис

Max (* * *Число1* * *, * * *число2* * *,..., * * *нумберн* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число1_ <br/> |Обязательный  <br/> |**Разные** <br/> |Первый номер в списке.  <br/> |
| _число2_ <br/> |Необязательна  <br/> |**Разные** <br/> | Второй номер в списке.  <br/> |
| _нумберн_ <br/> |Необязательна  <br/> |**Разные** <br/> |Значение n в списке.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Разные
  
## <a name="example"></a>Пример

МАКСИМУМ (13 в, 1 ФТ, 20 cm) 
  
Возвращает 13 дюймов.
