# <a name="max-function"></a>Функция Макс

Возвращает наибольшее число из списка. Наибольшее означает, что наиболее близкого к положительным infinity.
  
## <a name="syntax"></a>Синтаксис

Максимальное число (** *Число1* **, ** *число2* **,..., ** *numberN* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Число1_ <br/> |Обязательный  <br/> |**Может отличаться** <br/> |Первый номер в списке.  <br/> |
| _число2_ <br/> |Optional  <br/> |**Может отличаться** <br/> | Второй номер в списке.  <br/> |
| _numberN_ <br/> |Optional  <br/> |**Может отличаться** <br/> |N-й номер в списке.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Разные
  
## <a name="example"></a>Пример

Максимальное число (13 в 1 ft 20 см) 
  
Возвращает 13 дюймов.
