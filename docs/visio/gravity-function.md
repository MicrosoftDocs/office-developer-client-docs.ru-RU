# <a name="gravity-function"></a>Функция GRAVITY

Вычисляет правильный угол поворота текстового блока для показанного поворота фигуры, чтобы не допустить поворота текста вниз.
  
## <a name="syntax"></a>Синтаксис

GRAVITY(** *angle* **,[ ** *limit1* ** ],[ ** *limit2* ** ]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |Обязательно  <br/> |**Строка** <br/> | Угол фигуры.  <br/> |
| _limit1_ <br/> |Необязательна  <br/> |**Строка** <br/> |Первое ограничение поворота. Значение по умолчанию — 90 градусов.  <br/> |
| _limit2_ <br/> |Необязательна  <br/> |**Строка** <br/> |Второе ограничение поворота. Значение по умолчанию — 270 градусов.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Функция GRAVITY обычно используется в ячейке TxtAngle. 
  
Возвращаемая величина составляет 180  _градусов,_ если угол находится между значениями, указанными  _в limit1_ и  _limit2;_ в противном случае возвращается значение 0 градусов.
  
Все аргументы автоматически нормализуются функцией от 0 до 360 градусов. Если аргумент не указывает единицы, предполагается, что радианы. 
  
## <a name="example-1"></a>Пример 1

GRAVITY(Angle)
  
Возвращает 180 градусов, если  *угол*  находится от 90 до 270 градусов; в противном случае возвращается 0 градусов. 
  
## <a name="example-2"></a>Пример 2

GRAVITY(2)
  
Возвращает 180 градусов, так как 2 радиана — от 90 до 270 градусов.
  
## <a name="example-3"></a>Пример 3

GRAVITY(100 deg, 110 deg, 290 deg)
  
Возвращает 0 градусов.
  
## <a name="example-4"></a>Пример 4

GRAVITY(100 deg, 290 deg, 110 deg)
  
Возвращает 0 градусов.
