# <a name="gravity-function"></a>Функция GRAVITY

Вычисляет правильный угол вращения текстового блока для вращения фигуры, чтобы не перевернуть текст с ног на голову.
  
## <a name="syntax"></a>Синтаксис

GRAVITY(** *angle* **,[** limit1**],[** *limit2* ** ])  
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _угол_ <br/> |Обязательный  <br/> |**String** <br/> | Угол фигуры.  <br/> |
| _limit1_ <br/> |Необязательный  <br/> |**String** <br/> |Первое ограничение вращения. Значение по умолчанию — 90 градусов.  <br/> |
| _limit2_ <br/> |Необязательный  <br/> |**String** <br/> |Второе ограничение вращения. Значение по умолчанию — 270 градусов.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Функция GRAVITY обычно используется в ячейке TxtAngle. 
  
Возвращено значение 180 градусов,  если угол между значениями, указанными _limit1_ и _limit2;_ в противном случае возвращенное значение — 0 градусов.
  
Все аргументы автоматически нормализуются функцией от 0 до 360 градусов. Если аргумент не указывает единицы, предполагается, что радианы. 
  
## <a name="example-1"></a>Пример 1

GRAVITY (Angle)
  
Возвращает 180 градусов, если  *угол*  между 90 и 270 градусов; в противном случае возвращается 0 градусов. 
  
## <a name="example-2"></a>Пример 2

GRAVITY (2)
  
Возвращается на 180 градусов, так как 2 радиана между 90 и 270 градусами.
  
## <a name="example-3"></a>Пример 3

GRAVITY (100 дег, 110 дег, 290 дег)
  
Возвращает 0 градусов.
  
## <a name="example-4"></a>Пример 4

GRAVITY (100 дег, 290 дег, 110 дег)
  
Возвращает 0 градусов.
