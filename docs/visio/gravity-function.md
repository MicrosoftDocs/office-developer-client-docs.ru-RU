# <a name="gravity-function"></a>Функция ВЕСА

Вычисляет блок текста правильный угол вращения для вращение указанного фигуры для предотвращения текст и включение сверху вниз.
  
## <a name="syntax"></a>Синтаксис

ВЕС (** *угол* **, [** *limit1* **], [** *limit2* **]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _угол_ <br/> |Обязательный  <br/> |**Строка** <br/> | Угол фигуры.  <br/> |
| _limit1_ <br/> |Optional  <br/> |**Строка** <br/> |Первое ограничение вращение. Значение по умолчанию — 90 градусов.  <br/> |
| _limit2_ <br/> |Optional  <br/> |**Строка** <br/> |Второй ограничение вращение. Значение по умолчанию — на 270 градусов.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Строка
  
## <a name="remarks"></a>Замечания

Функция ВЕСА обычно используются в ячейке TxtAngle. 
  
Возвращаемое значение 180 градусов при _угловая_ между значениями, указанными с _limit1_ и _limit2_; в противном случае — значение, возвращаемое — 0 градусов.
  
Все аргументы автоматически нормализации в диапазоне от 0 до 360 градусов с помощью функции. Если аргумент не указан единиц измерения, считаются радианах. 
  
## <a name="example-1"></a>Пример 1

GRAVITY(angle)
  
Возвращает 180 градусов, если *угол* между 90 и 270 градусов; в противном случае возвращает 0 градусов. 
  
## <a name="example-2"></a>Пример 2

GRAVITY(2)
  
Возвращает 180 градусов, так как 2 радианах находится в пределах 90 и 270 градусов.
  
## <a name="example-3"></a>Пример 3

ВЕС (100 градусов, 110 градусов, 290 градусов)
  
Возвращает 0 градусов.
  
## <a name="example-4"></a>Пример 4

ВЕС (100 градусов, 290 градусов, 110 градусов)
  
Возвращает 0 градусов.
