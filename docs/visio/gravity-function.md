# <a name="gravity-function"></a>Функция GRAVITY

Вычисляет правильный угол вращения блока текста для указанного поворота фигуры, чтобы предотвратить переворачивание текста.
  
## <a name="syntax"></a>Синтаксис

Сила притяжения (* * *угол* * *, [* * *ограничено1* * *], [* * *limit2* * *]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _градусов_ <br/> |Обязательный  <br/> |**String** <br/> | Угол фигуры.  <br/> |
| _ограничено1_ <br/> |Необязательный  <br/> |**String** <br/> |Первое предельное значение поворота. Значение по умолчанию — 90 градусов.  <br/> |
| _limit2_ <br/> |Необязательный  <br/> |**String** <br/> |Второй лимит вращения. Значение по умолчанию — 270 градусов.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Функция сила ПРИТЯЖЕНИя обычно используется в ячейке TxtAngle. 
  
Возвращаемое значение равно 180 градусов, если _угол_ между значениями, указанными в _ограничено1_ и _limit2_; в противном случае возвращаемое значение равно 0 градусов.
  
Все аргументы автоматически нормализованы в пределах от 0 до 360 градусов по функциям. Если аргумент не указывает единицы, предполагается, что используются радианы. 
  
## <a name="example-1"></a>Пример 1

СИЛА ПРИТЯЖЕНИя (угол)
  
Возвращает 180 градусов, если *угол* находится в пределах от 90 до 270 градусов; в противном случае возвращает 0 градусов. 
  
## <a name="example-2"></a>Пример 2

СИЛА ПРИТЯЖЕНИЯ (2)
  
Возвращает значение 180 градусов, так как 2 радианы находятся в диапазоне от 90 до 270 градусов.
  
## <a name="example-3"></a>Пример 3

СИЛА ПРИТЯЖЕНИя (100 градусов, 110 град, 290 градусов)
  
Возвращает 0 градусов.
  
## <a name="example-4"></a>Пример 4

СИЛА ПРИТЯЖЕНИя (100 градусов, 290 град, 110 градусов)
  
Возвращает 0 градусов.
