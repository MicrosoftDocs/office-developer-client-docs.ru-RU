# <a name="sortkey-cell-actions-section"></a>SortKey Cell (Actions Section)

Число, определяющее порядок действий, отображаемых в контекстном меню или меню тегов действий.
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами. 
  
## <a name="remarks"></a>Замечания

Действия, выполняемые с помощью тегов действий или контекстных меню, отображаются в меню, отсортированном от самого низкого до более высокого, с меньшими номерами в верхней части меню. Если две строки действий имеют одинаковое значение ячейки SortKey, порядок определяется физическим порядком строк. Значение по умолчанию — 0 (ноль).
  
Чтобы получить ссылку на ячейку SortKey по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Действия. *Name (имя* ). Действия Сорткэйвхере.  *Name* — имя строки действий  <br/> |
   
Чтобы получить ссылку на ячейку SortKey по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**Виссектионактион** <br/> |
|Индекс строки:  <br/> |**висровактион** +  *i* , где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**Висактионсорткэй** <br/> |
