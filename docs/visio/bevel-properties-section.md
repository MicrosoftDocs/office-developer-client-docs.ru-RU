# <a name="bevel-properties-section"></a>Bevel Properties Section

Показывает атрибуты для любых эффектов bevel, примененных к фигуре. 
  
## <a name="remarks"></a>Примечания

Вы можете применить к фигуре фигуру в области **Формы** формата (Стили домашней формы  >    >  **Bevel**  >  **3-D Options)** или внести изменения в ячейки в разделе **Bevel Properties** в ShapeSheet.
