# <a name="bevel-properties-section"></a>Bevel Properties Section

Показывает атрибуты для любых эффектов bevel, примененных к фигуре. 
  
## <a name="remarks"></a>Примечания

Вы можете применить багетную к  фигуре в области "Фигура формата" (параметры  >    >    >  **3D**  Bevel для стилей домашней фигуры) или внести изменения в ячейки в разделе "Свойства багетной области" таблицы свойств фигуры.
