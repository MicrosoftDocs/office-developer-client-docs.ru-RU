# <a name="doubleuline-cell-character-section"></a>DoubleULine Cell (Character Section)

Определяет, есть ли в диапазоне текста двойное подчеркивание.
  
|**Value**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |В тексте под ним находится двойное подчеркивание.  <br/> |
|FALSE  <br/> |Под текстом не может быть двойное подчеркивание.  <br/> |
   
## <a name="remarks"></a>Примечания

Ячейка DoubleULine содержит сведения о форматировании, примененные к поддиапазону текста фигуры, если раздел "символы" содержит несколько строк. В противном случае он содержит сведения о форматировании для всего текста фигуры.
  
Значение этой ячейки также можно задать с помощью диалогового окна **текст** (щелкните стрелку на вкладке **Главная** ). **** 
  
Чтобы получить ссылку на ячейку DoubleULine по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Char. Дблундерлине [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку DoubleULine по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**Виссектиончарактер** <br/> |
|Индекс строки:  <br/> |**висровчарактер** +  *i* , где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**Висчарактердблундерлине** <br/> |
