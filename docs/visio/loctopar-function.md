# <a name="loctopar-function"></a>Функция LOCTOPAR

Возвращает преобразованную точку в родительских координатах в конечной системе координат.
  
## <a name="syntax"></a>Синтаксис

LOCTOPAR (* * *сркпоинт* * *, * * *сркреф* * *, * * *дстреф* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Сркпоинт_ <br/> |Обязательный  <br/> |**String** <br/> | Точка в локальных координатах в исходной системе координат.  <br/> |
| _Сркреф_ <br/> |Обязательный  <br/> |**String** <br/> | Ссылка на ячейку в исходном объекте.  <br/> |
| _Дстреф_ <br/> |Обязательный  <br/> |**String** <br/> | Ссылка на ячейку в целевом объекте.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Преобразует точку из локальных координат исходной фигуры в родительские координаты конечной фигуры. Можно использовать функцию LOCTOPAR для установки родительских координат в ячейках, таких как PinX, PinY, BeginX и начало фигуры, с помощью другой точки из другой системы координат. 
  
Эта функция работает, даже если исходная и конечная фигуры находятся в группах. Он также настраивается для вращения и отражения в промежуточном преобразовании. 
  
Координаты источника и назначения должны находиться на одной странице. 
  
Если объект Destination является страницей, не имеющей родителя, результат выражается в локальных координатах страницы.
