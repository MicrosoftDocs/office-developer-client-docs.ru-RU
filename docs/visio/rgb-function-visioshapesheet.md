# <a name="rgb-function-visioshapesheet"></a>RGB Function (VisioShapeSheet)

Возвращает значение, представляющее индекса в документе цветовой палитры. Он указывает цвет по его красного, зеленого и синего компонентов, где каждый — число в диапазоне от 0 до 255 включительно, или выражение, которое оценивается как такое число. 
  
## <a name="syntax"></a>Синтаксис

RGB (** *красной* **, ** *зеленой* **, ** *Синий* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _красный_ <br/> |Обязательный  <br/> |**Число** <br/> |Красный компонент.  <br/> |
| _Зеленый_ <br/> |Обязательный  <br/> |**Число** <br/> |Зеленый компонент.  <br/> |
| _синий_ <br/> |Обязательный  <br/> |**Nmber** <br/> |Синий компонент.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Number
  
## <a name="remarks"></a>Замечания

Если цвет, возвращаемого функцией еще не включенное в текущем документе цветовой палитры, добавляется палитры.
  
В следующей таблице перечислены некоторые стандартные цвета и их значения красного, зеленого и синего.
  
|**Цвет**|**Значение красного**|**Значение зеленого**|**Значение синего**|
|:-----|:-----|:-----|:-----|
|Черный  <br/> |0  <br/> |0  <br/> |0  <br/> |
|Синий  <br/> |0  <br/> |0  <br/> |255  <br/> |
|Зеленый  <br/> |0  <br/> |255  <br/> |0  <br/> |
|Голубой  <br/> |0  <br/> |255  <br/> |255  <br/> |
|Красный  <br/> |255  <br/> |0  <br/> |0  <br/> |
|Пурпурный  <br/> |255  <br/> |0  <br/> |255  <br/> |
|Желтый  <br/> |255  <br/> |255  <br/> |0  <br/> |
|Белый  <br/> |255  <br/> |255  <br/> |255  <br/> |
   
## <a name="example-1"></a>Пример 1

RGB(0,0,255)
  
Возвращает индекс цвета синего.
  
## <a name="example-2"></a>Пример 2

RGB (красный (Sheet.1! FillForegnd) 120, 0)
  
Возвращает индекс для цвета, красный компонент отражает листе. 1 заливки переднего плана.
