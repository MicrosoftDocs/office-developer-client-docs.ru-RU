# <a name="rgb-function-visioshapesheet"></a>RGB Function (VisioShapeSheet)

Возвращает значение, представляющее индекс в цветовой палитре документа. Он указывает цвет по своим красным, зеленым и синим компонентам, где каждый из них имеет число в диапазоне от 0 до 255 включительно или выражение, которое оценивается до такого числа. 
  
## <a name="syntax"></a>Синтаксис

RGB(** *красный* **, ** *зеленый* **, ** *синий* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _красный_ <br/> |Обязательный  <br/> |**Number** <br/> |Красный компонент.  <br/> |
| _зеленый цвет_ <br/> |Обязательный  <br/> |**Number** <br/> |Зеленый компонент.  <br/> |
| _синий_ <br/> |Обязательный  <br/> |**Nmber** <br/> |Синий компонент.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Номер
  
## <a name="remarks"></a>Примечания

Если цвет, возвращаемый функцией, еще не существует в цветовой палитре текущего документа, он добавляется в палитру.
  
В следующей таблице перечислены некоторые стандартные цвета и их красные, зеленые и синие значения.
  
|**Color**|**Красное значение**|**Зеленое значение**|**Синее значение**|
|:-----|:-----|:-----|:-----|
|Черный  <br/> |0  <br/> |0  <br/> |0  <br/> |
|Синий  <br/> |0  <br/> |0  <br/> |255  <br/> |
|Зеленый  <br/> |0  <br/> |255  <br/> |0  <br/> |
|Cyan  <br/> |0  <br/> |255  <br/> |255  <br/> |
|Красный  <br/> |255  <br/> |0  <br/> |0  <br/> |
|Пурпурный  <br/> |255  <br/> |0  <br/> |255  <br/> |
|Желтый  <br/> |255  <br/> |255  <br/> |0  <br/> |
|Белый  <br/> |255  <br/> |255  <br/> |255  <br/> |
   
## <a name="example-1"></a>Пример 1

RGB (0,0,255)
  
Возвращает индекс для синего цвета.
  
## <a name="example-2"></a>Пример 2

RGB(RED(Sheet.1! FillForegnd),120,0)
  
Возвращает индекс для цвета, красный компонент которого отражает на переднем плане заполнения Sheet.1.
