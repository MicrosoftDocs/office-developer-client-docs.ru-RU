# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>Копирование и вставка формулы из одной ячейки в другую

1. Выберите ячейку, содержащую формулу.
    
2. Нажмите клавиши CTRL + C.
    
    Чтобы заменить всей формулы, выберите ячейку формула, которого вы хотите заменить.
    
    Чтобы вставить формулу в существующей формулы, дважды щелкните ячейку и нажмите кнопку точку вставки, где требуется вставить формулу.
    
3. Нажмите сочетание клавиш CTRL + V
    
4. Нажмите клавишу ВВОД, чтобы принять формулу.
