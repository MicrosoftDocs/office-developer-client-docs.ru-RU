# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>Копирование и Вставка формулы из одной ячейки в другую

1. Выберите ячейку, содержащую нужную формулу.
    
2. Нажмите клавиши CTRL + C.
    
    Чтобы заменить формулу целиком, выберите ячейку, формулу которой нужно заменить.
    
    Чтобы вставить формулу в существующую формулу, дважды щелкните ячейку, а затем щелкните точку вставки, где необходимо вставить формулу.
    
3. Нажмите клавиши CTRL + V
    
4. Нажмите клавишу ВВОД, чтобы принять формулу.
