# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>Копирование и вклейка формулы из одной ячейки в другую

1. Выберите ячейку, которая содержит формулу, которую вы хотите.
    
2. Нажмите кнопку CTRL+C.
    
    Чтобы заменить всю формулу, выберите ячейку, формулу которой необходимо заменить.
    
    Чтобы вставить формулу в существующую формулу, дважды щелкните ячейку, а затем щелкните точку вставки, где необходимо вставить формулу.
    
3. Нажмите кнопку CTRL+V
    
4. Нажмите КНОПКУ ВВОД, чтобы принять формулу.
