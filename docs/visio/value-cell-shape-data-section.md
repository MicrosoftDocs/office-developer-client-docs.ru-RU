# <a name="value-cell-shape-data-section"></a>Ячейка Value (раздел "Данные фигуры")

Содержит значение элемента данных фигуры, введенное в диалоговом окне " **Определение данных фигуры** ". 
  
## <a name="remarks"></a>Замечания

Содержащиеся в этой ячейке формулы заменяются на значения, введенные в диалоговом окне **Определение данных фигуры** . Это значение true, даже в том случае, если функция защиты для защиты формулу. 
  
Чтобы получить ссылку на значение ячейки по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Свойства.  *Имя* . Значение где свойств.  *Имя* — это имя строки  <br/> |
   
Для получения ссылки на ячейки значение по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionProp** <br/> |
| Индекс строки:  <br/> |**visRowProp** +  *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visCustPropsValue** <br/> |
