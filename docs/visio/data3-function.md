# <a name="data3-function"></a>Функция DATA3

Возвращает текст из поля " **данные 3** " в диалоговом окне " **имя фигуры** " для фигуры (на вкладке " **разработчик** " щелкните **имя фигуры** ) в виде строки. 
  
## <a name="syntax"></a>Синтаксис

DATA3 ()
  
## <a name="remarks"></a>Комментарии

Если поле пустое, функция возвращает пустую строку.
