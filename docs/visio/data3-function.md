# <a name="data3-function"></a>Функция DATA3

Возвращает текст из поля **данных 3** в диалоговом окне **Имени фигуры** фигуры (на вкладке **Разработчик** выберите **Имя фигуры** ), как строку. 
  
## <a name="syntax"></a>Синтаксис

(DATA3)
  
## <a name="remarks"></a>Замечания

Если поле пустое, функция возвращает пустую строку.
