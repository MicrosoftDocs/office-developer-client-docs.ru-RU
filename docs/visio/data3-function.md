# <a name="data3-function"></a>Функция DATA3

Возвращает текст из окна **"Данные 3"** в диалоговом окне **"Имя** фигуры" для фигуры (на вкладке "Разработчик" щелкните **"Имя** фигуры") в виде строки.  
  
## <a name="syntax"></a>Синтаксис

DATA3 ()
  
## <a name="remarks"></a>Примечания

Если поле пустое, функция возвращает пустую строку.
