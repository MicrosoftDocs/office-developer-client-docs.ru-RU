# <a name="cell-element-layer-section-visio-xml"></a>Элемент Cell (раздел "слой") ("Visio XML")

Задает одно свойство для слоя или его свойств для страницы.
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|**Тип элемента** <br/> |[Целл_типе](cell_type-complextypevisio-xml.md) <br/> |
|**Пространство имен** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Файл схемы** <br/> |VisioSchema15. xsd  <br/> |
|**Части документа** <br/> |файлы хозяев. XML, Pages. XML  <br/> |
   
## <a name="definition"></a>Определение

```XML
< xs:element name="Cell" type="Cell_Type" minOccurs="0" maxOccurs="unbounded" >
</xs:element >
```

## <a name="remarks"></a>Замечания

Атрибут **N** этого элемента **Cell** должен иметь ограниченный набор значений, соответствующих ячейкам таблицы свойств фигуры. Чтобы определить значения атрибута **N** , которые разрешено использовать для этого элемента **ячейки** , обратитесь к приведенной ниже таблице. 
  
|**Value**|**Описание**|**Дополнительные сведения**|
|:-----|:-----|:-----|
|Активное  <br/> |Указывает, активен ли слой.  <br/> |Нет.  <br/> |
|Цвет  <br/> |Указывает один из следующих параметров: индекс цвета в таблице цветов, используемый для отображения слоя, или значение RGB, задающее пользовательский цвет, которого нет в таблице цветов.  <br/> |Нет.  <br/> |
|Колортранс  <br/> |Определяет степень прозрачности цвета текста слоя или фигуры от 0 (полностью непрозрачна) до 1 (полностью прозрачно).  <br/> |Нет.  <br/> |
|Средство  <br/> |Указывает, могут ли фигуры, принадлежащие слою, приклеить к.  <br/> |Нет.  <br/> |
|Lock  <br/> |Указывает, заблокированы ли фигуры, принадлежащие слою, для выбора или изменения.  <br/> |Нет.  <br/> |
|Имя  <br/> |Имя слоя.  <br/> |Нет.  <br/> |
|Намеунив  <br/> |Задает универсальное имя слоя.  <br/> |Нет.  <br/> |
|Print  <br/> |Указывает, печатаются ли фигуры, принадлежащие слою, при печати документа.  <br/> |Нет.  <br/> |
|Доверие  <br/> |Указывает, могут ли другие фигуры привязываться к фигурам, назначенным для слоя.  <br/> |Нет.  <br/> |
|Состояние  <br/> |Указывает, является ли слой допустимым слоем для документа.  <br/> |Нет.  <br/> |
|Visible  <br/> |Указывает, видимы ли фигуры, принадлежащие слою, на странице документа.  <br/> |Нет.  <br/> |
