# <a name="data2-function"></a>Функция DATA2

Возвращает текст из поля **данных 2** в диалоговом окне **Имени фигуры** фигуры (на вкладке **Разработчик** выберите **Имя фигуры** ), как строку. 
  
## <a name="syntax"></a>Синтаксис

(БД2)
  
## <a name="remarks"></a>Замечания

Если поле пустое, функция возвращает пустую строку.
