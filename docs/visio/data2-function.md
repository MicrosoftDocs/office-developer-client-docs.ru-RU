# <a name="data2-function"></a>Функция DATA2

Возвращает текст из окна **Data 2** в диалоговом окне  **Имя** фигуры для фигуры (на вкладке Разработчик щелкните **Имя** фигуры) в виде строки. 
  
## <a name="syntax"></a>Синтаксис

DATA2 ()
  
## <a name="remarks"></a>Примечания

Если поле пусто, функция возвращает пустую строку.
