# <a name="data2-function"></a>Функция DATA2

Возвращает текст из поля " **данные 2** " в диалоговом окне " **имя фигуры** " для фигуры (на вкладке " **разработчик** " щелкните **имя фигуры** ) в виде строки. 
  
## <a name="syntax"></a>Синтаксис

ФАЙЛ2 ()
  
## <a name="remarks"></a>Комментарии

Если поле пустое, функция возвращает пустую строку.
