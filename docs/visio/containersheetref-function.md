# <a name="containersheetref-function"></a>Функция CONTAINERSHEETREF

Возвращает ссылку на лист для указанного контейнера, содержащего фигуру.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

CONTAINERSHEETREF (* * *index* * * * * *[, Category]* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _index_ <br/> |Обязательна  <br/> |**Целое число** <br/> |Индекс контейнера, основанный на 1. Дополнительные сведения см.  <br/> |
| _категории_ <br/> |Необязательна  <br/> |**String** <br/> |Категория контейнера. Дополнительные сведения см.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Ссылка на таблицу свойств фигуры
  
## <a name="remarks"></a>Примечания

Индекс контейнера вычисляется на основе z-порядка контейнеров от передней до заднего.
  
 *Категории* — это пользовательские строки, которые можно использовать для категоризации фигур. Можно определить категории в ячейке User. Мсвшапекатегориес таблицы свойств фигуры для фигуры. Можно определить несколько категорий для фигуры, разделяя категории точками с запятой. 
  
Если фигура не является членом контейнера или не имеет контейнера, соответствующего указанному номеру индекса и категории, CONTAINERSHEETREF возвращает #REF!.
  
## <a name="example"></a>Пример

CONTAINERSHEETREF (1)! Полноразмерные 
  
Возвращает значение в ячейке Height контейнера, которое является наиболее переадресованным на странице, к которой принадлежит фигура.
