# <a name="nosnap-cell-geometry-section"></a>NoSnap Cell (Geometry Section)

Определяет, прикреплены ли другие фигуры к пути.
  
|**Значение**|**Описание**|
|:-----|:-----|
| TRUE  <br/> | Не разрешайте другим фигурам прикрепление к этому пути.  <br/> |
| FALSE  <br/> | Разрешим другим фигурам прикрепление к этому пути.  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку NoSnap по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Геометрия  *i*  . NoSnap,  *где i*  = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку NoSnap по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionFirstComponent** +  *i*, где *i* = 0, 1, 2...  <br/> |
| Индекс строки:  <br/> |**visRowComponent** <br/> |
| Индекс ячейки:  <br/> |**visCompNoSnap** <br/> |
