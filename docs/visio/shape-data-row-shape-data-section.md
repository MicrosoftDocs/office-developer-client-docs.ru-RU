# <a name="shape-data-row-shape-data-section"></a>Shape Data Row (Shape Data Section)

Содержит сведения для элемента данных единой фигуры, связанного с фигурой. Фигура содержит одну строку Shape Data для каждого элемента данных фигуры. Строки Shape Data именуются Prop.name и содержат следующие ячейки. Дополнительные сведения см. в конкретных разделах ячейки.
  
|**Cell**|**Описание**|
|:-----|:-----|
|[Label](label-cell-shape-data-section.md) <br/> |Указывает метку, которая отображается пользователям в диалоговом окне Или окне **Shape Data.**  <br/> |
|[Prompt](prompt-cell-shape-data-section.md) <br/> |Указывает описательный или учебный текст, который отображается пользователям в диалоговом окне **Shape Data** или окне при выборе элемента.  <br/> |
|[Тип](type-cell-shape-data-section.md) <br/> |Указывает тип данных для значения элемента фигуры: строка (0), фиксированный список (1), число (2), значение Boolean (3), переменный список (4), дата или время (5), продолжительность (6) или валюта (7).  <br/> |
|[Format](format-cell-shape-data-section.md) <br/> |Указывает форматирование элемента данных фигуры.  <br/> |
|[Значение](value-cell-shape-data-section.md) <br/> |Содержит значение элемента в диалоговом окне Или окне **Shape Data.**  <br/> |
|[SortKey](sortkey-cell-shape-data-section.md) <br/> |Указывает ключ, с помощью которого перечислены элементы в диалоговом окне **Shape Data** или окне.  <br/> |
|[Невидимый](invisible-cell-shape-data-section.md) <br/> |Указывает, виден ли элемент данных фигуры в диалоговом окне Или окне **Shape** Data. TRUE = не отображается; FALSE = видимый.  <br/> |
|[Спросите](ask-cell-shape-data-section.md) <br/> |Определяет, запрашивается ли у пользователя ввод сведений о форме данных для фигуры при создания экземпляра или дублировании или копировании фигуры.  <br/> |
|[LangID](langid-cell-shape-data-section.md) <br/> |Указывает язык, на котором отображается значение элемента данных фигуры.  <br/> |
|[Calendar](calendar-cell-miscellaneous-section.md) <br/> |Указывает тип календаря, используемый при типе элемента данных фигуры Date.  <br/> |
   
## <a name="remarks"></a>Примечания

 Можно добавить как можно больше реквизита.  *Строки*  имен по мере необходимости назначьте значимые имена строкам и установите значения ячейки. Чтобы добавить элемент данных фигуры в существующий раздел "Данные фигуры", щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить** строку в меню ярлыка. 
  
Вы можете ссылаться на эти ячейки по имени строки, которое отображается в окне ShapeSheet в красном тексте. Чтобы назначить значимые имена строкам *Prop.*  Нажмите строку, а затем введите имя, например  *Price,*  чтобы создать имя строки Prop.Price. Затем вы можете ссылаться на ячейку Label с помощью Prop.Price.Label. 
  
Имя строки, которое вы вводите, должно быть уникальным в разделе.
