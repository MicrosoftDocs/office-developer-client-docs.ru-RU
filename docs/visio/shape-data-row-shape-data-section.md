# <a name="shape-data-row-shape-data-section"></a>Строка Shape Data (раздел "Данные фигуры")

Содержит данные об одной фигуры элемента данных, связанных с фигурой. Фигура содержит одну строку данных фигур для каждого элемента данных фигуры. Строк данных фигуры с именем Prop.name и содержит следующие ячейки. Для получения дополнительных сведений см определенную ячейку.
  
|**Cell**|**Описание**|
|:-----|:-----|
|[Подпись](label-cell-shape-data-section.md) <br/> |Метка, которая отображается для пользователей в окне или диалоговое окно " **Данные фигуры** ".  <br/> |
|[Запрос](prompt-cell-shape-data-section.md) <br/> |Задает описательный или управляющий текст, который отображается для пользователей в окне или диалоговое окно " **Данные фигуры** " при выборе элемента.  <br/> |
|[Тип](type-cell-shape-data-section.md) <br/> |Указывает тип данных для значения элемента данных фигуры: строки (0), основных списка (1), номер (2), логическое значение (3), список переменных (4), даты или времени (5), длительность (6) или currency (7).  <br/> |
|[Формат](format-cell-shape-data-section.md) <br/> |Задает форматирование элемента данных фигуры.  <br/> |
|[Значение](value-cell-shape-data-section.md) <br/> |Содержит значение элемента, введенное в окне или диалоговое окно " **Данные фигуры** ".  <br/> |
|[SortKey](sortkey-cell-shape-data-section.md) <br/> |Указывает, какие элементы в диалоговом окне **Данных фигуры** перечислены поля или окна ключ.  <br/> |
|[Невидимое](invisible-cell-shape-data-section.md) <br/> |Указывает, видима ли данные фигуры в окне или диалоговое окно " **Данные фигуры** ". ИСТИНА = не отображается; FALSE = видимым.  <br/> |
|[Попросите](ask-cell-shape-data-section.md) <br/> |Определяет, является ли пользователь получает запрос на введите сведения о данных фигуры для фигуры, при создании экземпляра или фигуры и копировании.  <br/> |
|[LangID](langid-cell-shape-data-section.md) <br/> |Задает язык, в котором будет отображаться фигуры значение элемента данных.  <br/> |
|[Календарь](calendar-cell-miscellaneous-section.md) <br/> |Указывает тип используется, если тип элемента данных фигуры даты календаря.  <br/> |
   
## <a name="remarks"></a>Замечания

 Можно добавить столько свойства.  *имя* строки при необходимости назначить понятные имена строк и задайте значения ячейки. Чтобы добавить элемент данных фигуры в существующий раздел данных фигуры, щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить строку** в контекстном меню. 
  
Можно ссылаться на эти ячейки с их имя строки, который отображается в окне таблицы свойств фигуры текстом красного цвета. Чтобы назначить понятные имена свойства. *имя* строки, щелкните строку и введите имя, например *цены* , например для создания имя строки Prop.Price. Затем можно ссылаться метки ячейки с помощью Prop.Price.Label. 
  
Имя строки при вводе должно быть уникальным в рамках раздела.
