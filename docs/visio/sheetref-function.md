# <a name="sheetref-function"></a>Функция SHEETREF

Возвращает ссылку на листе (фигуры), который указан в _sheetref_или, если квантор _sheetref_ в текущий лист. 
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

 *\<sheetref\>!*  SHEETREF()

Справочник по таблице свойств фигуры
  
## <a name="remarks"></a>Замечания

Эта функция в другие функции, которые принимают ссылку листа маркеров.
