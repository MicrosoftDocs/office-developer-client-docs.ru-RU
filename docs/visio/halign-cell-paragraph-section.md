# <a name="halign-cell-paragraph-section"></a>HAlign Cell (Paragraph Section)

Определяет горизонтальное выравнивание текста в блоке текста фигуры.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| нуль  <br/> | Выравнивание по левому краю  <br/> |**Вишорзлефт** <br/> |
| 1,1  <br/> | Center  <br/> |**Вишорзцентер** <br/> |
| 2  <br/> | Выравнивание по правому краю  <br/> |**Вишорзригхт** <br/> |
| 4  <br/> | Justify  <br/> |**Вишорзжустифи** <br/> |
| SP4  <br/> | Принудительное выравнивание  <br/> |**Вишорзфорце** <br/> |
   
## <a name="remarks"></a>Примечания

ВыРовнять по ширине пробелы между словами в каждой строке, кроме последней строки абзаца, чтобы выровнять левую и правую части текста по полям.
  
Принудительно выравнивает каждую строку в абзаце, в том числе последнюю.
  
Чтобы получить ссылку на ячейку HAlign по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Para. Хорзалигн [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку HAlign по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**Виссектионпараграф** <br/> |
| Индекс строки:  <br/> |**висровпараграф** +  *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**Вишорзалигн** <br/> |
