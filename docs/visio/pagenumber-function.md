# <a name="pagenumber-function"></a>Функция PAGENUMBER

Возвращает номер страницы, содержаной фигуру. 
  
## <a name="syntax"></a>Синтаксис

PAGENUMBER()
  
### <a name="return-value"></a>Возвращаемое значение

Номер
  
## <a name="remarks"></a>Примечания

Если текущая страница является фоновой страницей (неучленной) или если возникает ошибка, функция PAGENUMBER возвращает ноль.
