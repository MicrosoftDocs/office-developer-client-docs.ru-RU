# <a name="line-format-section"></a>Line Format Section

Содержит ячейки, которые контролируют атрибуты строки для фигуры, такие как шаблон, вес и цвет. Они определяют форматирование конечных строк (например, с наконечникой стрелки), размер конечных форматов строки, радиус круга округлиния, примененного к линии, и стиль крышки строки (круглый или квадратный).
  
## <a name="remarks"></a>Примечания

Вы можете настроить форматы строк с помощью области  **Формы** формата  (на вкладке Главная, в группе Стилей формы, щелкните **Строка,** щелкните Параметры строки), применяя стиль строки или введите формулу в ячейке формата строки.
