# <a name="line-format-section"></a>Раздел "Формат линии"

Содержит ячейки, которые управляют атрибуты строки для фигуры, такие как шаблон, вес и цвет. Они определяют ли совместите форматируются (например, с помощью стрелки), размер конца строки форматов, radius круга округления, применяемые к линии и стиль строки (круглый или квадратный).
  
## <a name="remarks"></a>Замечания

Можно задать линия форматы с помощью области **Формат фигуры** (на вкладке **Главная** в группе **Стили фигур** , щелкнув **строку**, нажмите кнопку **Параметры строки**), применив стиль линии или путем ввода формулы в ячейке формат строки.
