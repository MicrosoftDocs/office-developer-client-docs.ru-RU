# <a name="line-format-section"></a>Line Format Section

Содержит ячейки, которые контролируют атрибуты линии для фигуры, такие как шаблон, вес и цвет. Они определяют форматирование конца строки (например, с помощью наконечника), размер форматов окончания линии, радиус круга округления, применяемый к строке, и стиль линии (округлый или квадратный).
  
## <a name="remarks"></a>Примечания

Форматы строк можно настроить  с помощью области "Формат фигуры"  (на вкладке "Главная" в группе "Стили фигуры" щелкните "Строка" **и**"Параметры строки"), применив стиль строки или введите формулу в ячейке "Формат строки".
