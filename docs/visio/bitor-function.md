# <a name="bitor-function"></a>Функция BITOR

Возвращает 16-разрядный двоичное число, в котором каждый бит задано значение 1, если соответствующий бит в *двоичные Число1* или *двоичные число2* — 1. Только в том случае, если соответствующий бит равен 0 в *двоичные Число1* и *число2 двоичный* бит имеет значение 0. 
  
## <a name="syntax"></a>Синтаксис

BITOR (** *двоичные Число1* **, ** *двоичные число2* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _двоичные Число1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первый номер двоичные 16-разрядной.  <br/> |
| _двоичные число2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второй 16-разрядное число двоичные.  <br/> |
   
### <a name="return-value"></a>������������ ��������

16-разрядный двоичный
  
## <a name="example"></a>Пример

BITOR(12,6)
  
Возвращает 14. 12 = 0... 01100. 6 = 0... 00110. Таким образом, BITOR(12,6) = 0... 01110.
