# <a name="bitor-function"></a>Функция BITOR

Возвращает 16-разрядное двоичное число, в котором для каждого бита задано значение 1, если соответствующий бит в *двоичных* значениях Число1 или *число2* имеет значение 1. Для бита устанавливается значение 0, только если соответствующий бит равен 0 в *двоичных* значениях Число1 и *число2* . 
  
## <a name="syntax"></a>Синтаксис

БИТ (* * *двоичные числа Число1* * *, * * *двоично-число2* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Двоичные Число1_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Первое 16 – разрядное двоичное число.  <br/> |
| _двоичные файлы число2_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Второй 16 – разрядный двоичный номер.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

16 – разрядный двоичный
  
## <a name="example"></a>Пример

БИТ (12, 6)
  
Возвращает 14. Значение 12 = 0... 01100. 6 = 0... 00110. Таким образом, бит (12, 6) = 0... 01110.
