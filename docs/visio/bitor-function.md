# <a name="bitor-function"></a>Функция BITOR

Возвращает 16-битный двоичный номер, в котором для каждого бита установлено 1, если соответствующий бит в двоичном  *номере 1*  или  *двоичном номере 2*  имеет 1. Бит имеет 0, только если соответствующий бит имеет 0 в двоичном *номере 1* и *двоичном номере 2.* 
  
## <a name="syntax"></a>Синтаксис

BITOR(** *binary number1* **, ** *binary number2* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _binary number1_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Первый 16-битный двоичный номер.  <br/> |
| _binary number2_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Второй 16-битный двоичный номер.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

16-битный двоичный файл
  
## <a name="example"></a>Пример

BITOR(12,6)
  
Возвращает 14. 12 = 0...01100. 6 = 0...00110. Таким образом, BITOR(12,6) = 0...01110.
