# <a name="bitor-function"></a>Функция BITOR

Возвращает 16-битный двоичный номер, в котором для каждого бита установлено 1, если соответствующий бит в двоичном  *номере1*  или  *двоичном номере 2*  — 1. Бит установлен до 0 только в том случае, если соответствующий бит 0 в *двоичном номере1* и *двоичном номере 2.* 
  
## <a name="syntax"></a>Синтаксис

BITOR(** *двоичный номер1* **, ** *двоичный номер2* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _двоичный номер1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Первый 16-битный двоичный номер.  <br/> |
| _двоичный номер2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Второй 16-битный двоичный номер.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

16-битная двоичная
  
## <a name="example"></a>Пример

BITOR (12,6)
  
Возвращает 14. 12 = 0...01100. 6 = 0...00110. Таким образом, BITOR (12,6) = 0...01110.
