# <a name="glowsize-cell-additional-effect-properties-section"></a>Ячейка GlowSize (раздел "Дополнительные свойства эффекта")

Определяет размер внешнего свечения фигуры в точках. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **GlowSize** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | GlowSize  <br/> |
   
Для получения ссылки на ячейки **GlowSize** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowOtherEffectProperties** <br/> |
| Индекс ячейки:  <br/> |**visGlowSize** <br/> |
