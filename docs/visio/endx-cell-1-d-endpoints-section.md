# <a name="endx-cell-1-d-endpoints-section"></a>EndX Cell (1-D Endpoints Section)

Представляет  *x-координату*  конечной точки формы 1-D по отношению к происхождению ее родителя. 
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку EndX по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | EndX  <br/> |
   
Чтобы получить ссылку на ячейку EndX по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowXForm1D** <br/> |
| Индекс ячейки:  <br/> |**vis1DEndX** <br/> |
