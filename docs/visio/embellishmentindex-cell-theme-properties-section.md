# <a name="embellishmentindex-cell-theme-properties-section"></a>EmbellishmentIndex Cell (Theme Properties Section)

Изменяет внешний вид (украшение) вызовите, контейнеры, временные рамки и формы диаграммы организации.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку **ПриукрашиваниеIndex** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы с использованием свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ПриукрашиваниеIndex  <br/> |
   
Чтобы получить ссылку на ячейку **ПриукрашиваниеIndex** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowThemeProperties** <br/> |
| Индекс ячейки:  <br/> |**visEmbellishmentIndex** <br/> |
