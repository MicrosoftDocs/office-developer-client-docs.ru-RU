# <a name="clippingpath-cell-foreign-image-info-section"></a>ClippingPath Cell (Foreign Image Info Section)

Содержит ссылку на геометрию пути, который связан с изображением. 
  
## <a name="remarks"></a>Примечания

Если **ячейка ClippingPath** указывает на допустимый путь, изображение обрезается таким образом, чтобы изображение отрисовылось внутри пути. Если **ячейка ClippingPath** пуста или содержит недействительный вход, изображение будет отрисовывалось прямоугольным клипом с использованием значений масштабирования и смещения. 
  
> [!NOTE]
> Допустимы только пути, определенные разделом [Геометрия](geometry-section.md) в shapeSheet изображения, для ячейки **ClippingPath.** Ссылки на перекрестный лист нельзя использовать для определения пути обрезки изображений. 
  
Чтобы получить ссылку на ячейку **ClippingPath** по имени из другой формулы, по значению **атрибута N** элемента **Cell** или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ClippingPath  <br/> |
   
Чтобы получить ссылку на ячейку **ClippingPath** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowForeign** <br/> |
| Индекс ячейки:  <br/> |**visFrgnImgClippingPath** <br/> |
   
## <a name="example"></a>Пример

Вы можете изменить очертания изображения на овал, делая следующее:
  
- Вставьте изображение на рисунок холста.
    
- Щелкните правой кнопкой мыши изображение, а затем **выберите Show ShapeSheet**.
    
- Щелкните правой кнопкой мыши в любом месте таблицы ShapeSheet и выберите **раздел Вставки**.
    
- В **диалоговом окне Вставить** раздел выберите **геометрию** и нажмите **кнопку ОК**.
    
- В новом разделе Геометрия (например, "Geometry2") удалите все строки, кроме одной строки.
    
- Щелкните правой кнопкой мыши оставшиеся строки и нажмите **кнопку Изменить строку типа**.
    
- В **диалоговом окне Тип** строки изменения выберите **Ellipse** и нажмите **кнопку ОК**.
    
- В разделе **Foreign Image** установите формулу ячейки **ClippingPath** и  `="Geometry2.Path"` примите ее.
