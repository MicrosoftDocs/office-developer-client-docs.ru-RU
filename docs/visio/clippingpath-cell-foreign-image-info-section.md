# <a name="clippingpath-cell-foreign-image-info-section"></a>ClippingPath Cell (Foreign Image Info Section)

Содержит ссылку на геометрию пути, к которую привязано изображение. 
  
## <a name="remarks"></a>Примечания

Если **ячейка ClippingPath** указывает на допустимый путь, изображение обрезается, чтобы изображение отрисовылось внутри пути. Если **ячейка ClippingPath** пуста или содержит недействительные записи, изображение будет отрисовировано прямоугольным клипом, используя значения масштаба и смещения. 
  
> [!NOTE]
> Допустимы только пути, определенные разделом ["Геометрия"](geometry-section.md) в таблице фигур изображения, для ячейки **ClippingPath.** Ссылки на несколько листов нельзя использовать для определения пути обрезки изображения. 
  
Чтобы получить ссылку на ячейку **ClippingPath** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ClippingPath  <br/> |
   
Чтобы получить ссылку на ячейку **ClippingPath** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowForeign** <br/> |
| Индекс ячейки:  <br/> |**visFrgnImgClippingPath** <br/> |
   
## <a name="example"></a>Пример

Вы можете изменить ограниченную форму изображения на овал, выступая следующим образом:
  
- Вставьте рисунок на полотно.
    
- Щелкните рисунок правой кнопкой мыши и выберите **"Показать таблицу фигур".**
    
- Щелкните правой кнопкой мыши в любом месте таблицы фигур и выберите **"Вставить раздел".**
    
- В **диалоговом окне** "Вставка раздела" выберите **"Геометрия"** и нажмите кнопку **"ОК".**
    
- В новом разделе "Геометрия" (например, "Geometry2") удалите все строки, кроме одной.
    
- Щелкните правой кнопкой мыши оставшуюся строку и выберите **"Изменить тип строки".**
    
- В **диалоговом окне** "Изменение типа строки" выберите **"Эллипс"** и нажмите кнопку **"ОК".**
    
- В разделе **"Внешнее** изображение" установите для ячейки **ClippingPath** формулу и  `="Geometry2.Path"` примите ее.
