# <a name="page-properties-section"></a>Page Properties Section

Содержит ячейки, которые контролируют атрибуты страницы, такие как ширина страницы, высота и масштаб.
  
## <a name="remarks"></a>Примечания

Значения свойств страниц можно установить, сделав запись в  ячейке раздела Свойства страниц  или с помощью диалогового окна Установки страницы (на вкладке  **Дизайн** щелкните диалоговое окно в группе Установки страницы).
