# <a name="page-properties-section"></a>Page Properties Section

Содержит ячейки, которые контролируют атрибуты страницы, такие как ширина страницы, высота и масштаб.
  
## <a name="remarks"></a>Примечания

Значения свойств страницы можно установить, сделав запись в  ячейке раздела "Свойства страницы" или с помощью  диалогового окна "Настройка страницы" (на вкладке "Конструктор" щелкните окно запуска диалогового окна в группе "Настройка страницы").
