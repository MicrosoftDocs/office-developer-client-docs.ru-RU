# <a name="glowcolor-cell-additional-effect-properties-section"></a>Ячейка GlowColor (раздел "Дополнительные свойства эффекта")

Определяет цвет, используемый для штрих внешнего свечения, применяемые к фигуре, как значение RGB и темы.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **GlowColor** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | GlowColor  <br/> |
   
Для получения ссылки на ячейки **GlowColor** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowOtherEffectProperties** <br/> |
| Индекс ячейки:  <br/> |**visGlowColor** <br/> |
