# <a name="lineto-row-geometry-section"></a>Строка LineTo (раздел "Геометрия")

Содержит *x* - и *y* -координаты окончания вершины прямой сегмент. 
  
Строка LineTo содержит следующие ячейки.
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X* -координаты окончания вершины прямой сегмент.  <br/> |
|[Да](y-cell-geometry-section.md) <br/> |*Y* -координат окончания вершины прямой сегмент.  <br/> |
