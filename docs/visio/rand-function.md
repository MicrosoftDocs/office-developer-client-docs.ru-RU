# <a name="rand-function"></a>Функция RAND

Возвращает случайное число с плавающей запятой от 0,0 до 1,0. Он возвращает разное число всякий раз, когда вычисляется функция, которая выполняется один раз в минуту в соответствии с системными часами. 
  
## <a name="syntax"></a>Синтаксис

СЛЧИС ()
  
### <a name="return-value"></a>Возвращаемое значение

С плавающей запятой
  
## <a name="remarks"></a>Примечания

С помощью этой функции можно создавать анимационные эффекты, присвоив свойствам фигур случайные значения.
  
## <a name="example"></a>Пример

СЛЧИС () 
  
Возвращает десятичную дробь, например 0,3503.
