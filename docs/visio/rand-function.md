# <a name="rand-function"></a>Функция RAND

Возвращает случайное число с плавающей запятой от 0,0 до версии 1.0. Она возвращает другой номер каждый раз, когда вычисляется функция, которая выполняется один раз в минуту в соответствии с системным таймером. 
  
## <a name="syntax"></a>Синтаксис

(РЭНД)

Float
  
## <a name="remarks"></a>Замечания

Эта функция используется для создания эффектов анимации путем установки свойства фигур случайного значения.
  
## <a name="example"></a>Пример

(РЭНД) 
  
Возвращает дробной, такие как 0.3503.
