# <a name="shdwoffsety-cell-page-properties-section"></a>ShdwOffsetY Cell (Page Properties Section)

Определяет расстояние в единицах страницы, на которое тень фигуры смещается по вертикали от фигуры.
  
## <a name="remarks"></a>Примечания

Это значение задается в диалоговом окне " **Параметры страницы** " (на вкладке " **конструктор** " щелкните стрелку **Параметры страницы** ). Это значение не зависит от масштаба рисунка. Если масштаб документа изменяется, смещение тени остается прежним. 
  
Чтобы получить ссылку на ячейку ShdwOffsetY по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ShdwOffsetY  <br/> |
   
Чтобы получить ссылку на ячейку ShdwOffsetY по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**Висровпаже** <br/> |
| Индекс ячейки:  <br/> |**Виспажешдвоффсети** <br/> |
