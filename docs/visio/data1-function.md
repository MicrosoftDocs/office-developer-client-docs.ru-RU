# <a name="data1-function"></a>Функция DATA1

Возвращает текст из поля " **данные 1** " в диалоговом окне " **имя фигуры** " для фигуры (на вкладке " **разработчик** " щелкните **имя фигуры** ) в виде строки. 
  
## <a name="syntax"></a>Синтаксис

Файл1 ()
  
## <a name="remarks"></a>Комментарии

Если поле пустое, функция возвращает пустую строку.
