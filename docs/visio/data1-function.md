# <a name="data1-function"></a>Функция DATA1

Возвращает текст из поля **данных 1** в диалоговом окне **Имени фигуры** фигуры (на вкладке **Разработчик** выберите **Имя фигуры** ), как строку. 
  
## <a name="syntax"></a>Синтаксис

(БД1)
  
## <a name="remarks"></a>Замечания

Если поле пустое, функция возвращает пустую строку.
