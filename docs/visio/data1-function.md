# <a name="data1-function"></a>Функция DATA1

Возвращает текст из окна **Data 1** в диалоговом окне  **Имя** фигуры для фигуры (на вкладке Разработчик щелкните **Имя** фигуры) в виде строки. 
  
## <a name="syntax"></a>Синтаксис

DATA1 ()
  
## <a name="remarks"></a>Примечания

Если поле пусто, функция возвращает пустую строку.
