# <a name="shape-layout-section"></a>Shape Layout Section

Содержит ячейки, которые контролируют размещение фигуры и параметры маршрутивки соединителя.
  
## <a name="remarks"></a>Примечания

Вы также можете установить значения определенных фигур ячеек  в этом  разделе с помощью вкладки Размещение в диалоговом окне Поведение (на вкладке Разработчик, в группе **Shape Design,** щелкните [](run-in-developer-mode-display-the-developer-tab.md) **Поведение).** Значения ячеек, определенных соединителу, можно установить в этом разделе с помощью вкладки **Соединитель** в диалоговом окне **Поведение.** (Чтобы увидеть вкладку **Соединитель,** выберите соединитель в окне рисования и отобразите диалоговое окно **Поведение.)** 
  
Значения в разделе **Макет фигуры** можно использовать для переопределения значений по умолчанию, содержащихся в разделе **Макет** страницы.
