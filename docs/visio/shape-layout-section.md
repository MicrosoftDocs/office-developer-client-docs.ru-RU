# <a name="shape-layout-section"></a>Shape Layout Section

Содержит ячейки, которые контролируют размещение фигуры и параметры маршрутизации соединителей.
  
## <a name="remarks"></a>Замечания

Кроме того, можно задать значения для ячеек, относящихся к фигурам, в этом разделе, используя вкладку **Размещение** в диалоговом окне **поведение** (на вкладке [разработчик](run-in-developer-mode-display-the-developer-tab.md) в группе **Макет фигуры** щелкните **поведение**). В этом разделе можно задать значения для ячеек, относящихся к соединителю, с помощью вкладки **соединитель** в диалоговом окне **поведение** . (Чтобы открыть вкладку **соединитель** , выберите соединитель в окне рисунок, а затем откройте диалоговое окно **поведение** .) 
  
Значения в разделе **Макет фигуры** можно использовать для переопределения значений по умолчанию, которые хранятся в разделе **Макет страницы** .
