# <a name="nurbs-function"></a>Функция NURBS

Возвращает неоднородного рациональный B-сплайн (NURBS). Эта функция используется в ячейке E в строках геометрии NURBSTo.
  
## <a name="syntax"></a>Синтаксис

NURBS (* * *кнотласт* * *, * * *градусы* * *, * * *кстипе* * *, * * *итипе* * *, * ** *, * * * * ** * * * *, * * *knot1* * *, * * *weight1* * *,...) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Кнотласт_ <br/> |Обязательный  <br/> |**строка** <br/> | Последнее кнот.  <br/> |
| _угол_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Степень сплайна.  <br/> |
| _Кстипе_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Указывает способ интерпретации входных данных _x_ . Если _кстипе_ имеет значение 0, все входные данные _x_ интерпретируются как процент ширины. Если _кстипе_ имеет значение 1, все входные данные _x_ интерпретируются как локальные координаты.  <br/> |
| _Итипе_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Указывает способ интерпретации входных данных _y_ . Если _итипе_ имеет значение 0, все входные данные _y_ интерпретируются как процент от высоты. Если _итипе_ имеет значение 1, все входные данные _y_ интерпретируются как локальные координаты.  <br/> |
| _x1_ <br/> |Обязательный  <br/> |**String** <br/> |Координата x.  <br/> |
| _Y1_ <br/> |Обязательный  <br/> |**String** <br/> |Координата y.  <br/> |
| _knot1_ <br/> |Обязательный  <br/> |**String** <br/> |Объект кнот на B — сплайн.  <br/> |
| _weight1_ <br/> |Обязательный  <br/> |**String** <br/> |Вес для сбалансированного сплайна.  <br/> |
   
## <a name="remarks"></a>Комментарии

Для каждого аргумента *x* должен быть аргумент *y* ; в противном случае возвращается ошибка. 
  
Необходимо указать по крайней мере один аргумент *x*, *y*, *кнот* и *Weight* ; в противном случае Visio возвращает ошибку.
