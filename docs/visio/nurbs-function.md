# <a name="nurbs-function"></a>Функция NURBS

Возвращает несвязавный логический B-spline (NURBS). Эта функция используется в ячейке E в геометрических строках NURBSTo.
  
## <a name="syntax"></a>Синтаксис

NURBS(** *etcLast* **, ** *degree* **, ** *xType* **, ** *yType* **, ** *x1* **, ** *y1* **, **т1 **, ** *weight1* **, ...)  
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _last_ <br/> |Обязательна  <br/> |**строка** <br/> | Последняя ветвь.  <br/> |
| _degree_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Степень spline.  <br/> |
| _xType_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Указывает, как интерпретировать _входные данные x._ Если  _xType_ имеет 0, все входные  _данные x_ интерпретируются как процент ширины. Если  _xType_ имеет 1, все  _входные данные x_ интерпретируются как локальные координаты.  <br/> |
| _yType_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Указывает, как интерпретировать _входные данные y._ Если  _yType_ имеет 0, все входные  _данные Y_ интерпретируются как процент высоты. Если  _yType_ имеет 1, все входные  _данные Y_ интерпретируются как локальные координаты.  <br/> |
| _x1_ <br/> |Обязательно  <br/> |**Строка** <br/> |X-координата.  <br/> |
| _y1_ <br/> |Обязательно  <br/> |**Строка** <br/> |Координата Y.  <br/> |
| _1_ <br/> |Обязательно  <br/> |**Строка** <br/> |Заметь на B-spline.  <br/> |
| _weight1_ <br/> |Обязательно  <br/> |**Строка** <br/> |Вес на B-spline.  <br/> |
   
## <a name="remarks"></a>Примечания

Для каждого  *аргумента x*  должен быть аргумент  *y;*  в противном случае возвращается ошибка. 
  
Необходимо указать по крайней мере один  *аргумент x*, *y*, *argument и*  *weight;*  в противном случае Visio возвращает ошибку.
