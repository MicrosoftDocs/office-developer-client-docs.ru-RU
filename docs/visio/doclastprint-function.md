# <a name="doclastprint-function"></a>Функция DOCLASTPRINT

Возвращает дату и время последней печати документа, как значение последовательный.
  
## <a name="syntax"></a>Синтаксис

(DOCLASTPRINT)
  
## <a name="remarks"></a>Замечания

Функция FORMAT используется для форматирования значения.
