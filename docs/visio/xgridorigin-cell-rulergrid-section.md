# <a name="xgridorigin-cell-ruler-amp-grid-section"></a>Ячейка XGridOrigin (линейки &amp; сетка)

Задает горизонтальной оси начала сетки.
  
## <a name="remarks"></a>Замечания

В этой ячейке соответствует горизонтальной **начала сетки** параметра в **линейки &amp; сетки** диалоговое окно "" (на вкладке **Вид** щелкните стрелку **Показать** . 
  
Чтобы получить ссылку на ячейку XGridOrigin по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |XGridOrigin  <br/> |
   
Для получения ссылки на ячейки XGridOrigin по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowRulerGrid** <br/> |
|Индекс ячейки:  <br/> |**visXGridOrigin** <br/> |
