# <a name="cos-function-visioshapesheet"></a>COS Function (VisioShapeSheet)

Возвращает косинус угла. 
  
## <a name="syntax"></a>Синтаксис

*COS(angle)* 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Угол, с которого можно получить косинус.  <br/> |
   
## <a name="remarks"></a>Примечания

Функция COS использует радианы, если не указать другую единицу измерения в *углу.* 
  
## <a name="example"></a>Пример

COS(45 deg) 
  
Возвращает 0,7071.
