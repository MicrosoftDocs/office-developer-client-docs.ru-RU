# <a name="prompt-cell-actions-section"></a>Prompt Cell (Actions Section)

Начиная с Visio 2002, эта ячейка больше не используется.
  
## <a name="remarks"></a>Примечания

В более ранних версиях Visio в этой ячейке указан описательный запрос, который отображался в строке состояния при выборе соответствующей команды.
