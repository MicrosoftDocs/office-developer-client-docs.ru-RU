# <a name="opengroupwin-function"></a>Функция OPENGROUPWIN

Открывает группу в окне группы. Если объект не является группой, эта функция ничего не делает.
  
## <a name="syntax"></a>Синтаксис

OPENGROUPWIN( )
  
## <a name="remarks"></a>Примечания

В более ранних версиях Visio эта функция отображается как _OPENGROUPWIN. Visio версий 4.0 и более поздних версий принимают любой стиль.
