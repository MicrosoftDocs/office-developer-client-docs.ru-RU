# <a name="atan2-function"></a>Функция ATAN2

Возвращает угол между вектор, представленное *x, y* и направления *x* -оси. Результатом является число в текущей единицы измерения углов. 
  
## <a name="syntax"></a>Синтаксис

То функция ATAN2 (** *y* **, ** *x* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _y_ <br/> |Обязательный  <br/> |**Числовой** <br/> |_Y_— значение точки.  <br/> |
| _x_ <br/> |Обязательный  <br/> |**Числовой** <br/> |_X_— значение точки.  <br/> |
   
## <a name="remarks"></a>Замечания

Арктангенс — это угол отсчитывается против часовой стрелки от срабатывание *x* -ось в строке, которая с ним координат (0, 0) и точки, представленной *x* и *y* . В Microsoft Visio ATAN2(0,0) возвращает значение 0. Для принудительной результатов то функция ATAN2 в разных угловые измерения, используйте Градусов или RAD функции. 
  
Функция то функция ATAN2 является antifunction функция TAN. Функция то функция ATAN2 возвращает угол, чьи угол равно *y* , разделенные на *x* . Если то функция ATAN2 (*y, x*), представляющее угол в правом треугольник, *y* — это «положительно стороне» и *x* является «рядом с стороне», поэтому функции может быть записано в формате ATAN2(opposite,adjacent). 
  
## <a name="example-1"></a>Пример 1

ATAN2(1.25,2.25)
  
Возвращает 29.0456 градусов
  
## <a name="example-2"></a>Пример 2

ATAN2(1,SQRT(3))
  
Возвращает 30 градусов
  
## <a name="example-3"></a>Пример 3

ATAN2(1,1)
  
Возвращает 45 градусов
