# <a name="typedesc-function"></a>Функция TYPEDESC

Возвращает строку, описывающую внутренний тип объекта. 
  
## <a name="syntax"></a>Синтаксис

TYPEDESC ()
  
### <a name="return-value"></a>Возвращаемое значение

Строка
