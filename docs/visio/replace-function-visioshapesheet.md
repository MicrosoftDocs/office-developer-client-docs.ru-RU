# <a name="replace-function-visioshapesheet"></a>REPLACE Function (VisioShapeSheet)

Заменяет часть строки текста, на основе числа символов, с другой строкой текста.
  
## <a name="syntax"></a>Синтаксис

ЗАМЕНИТЕ (** *стар_текст* **, ** *Начальная позиция* **, ** *число_знаков* **, ** *нов_текст* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _стар_текст_ <br/> |Обязательный  <br/> |**Строка** <br/> |Текст, в котором вы хотите заменить некоторые знаки.  <br/> |
| _Начальная позиция_ <br/> |Обязательный  <br/> |**Число** <br/> |Позиция символа в _стар_текст_ , который вы хотите заменить на _нов_текст_. Первый символ в строке является позиции 1.  <br/> |
| _Количество_знаков_ <br/> |Обязательный  <br/> |**Число** <br/> |Число знаков в _тексте старый_текст_ , который вы хотите заменить  <br/> |
| _нов_текст_ <br/> |Обязательный  <br/> |**Строка** <br/> |Текст, который заменяет знаки в _тексте старый_текст_.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Строка
  
## <a name="remarks"></a>Замечания

Функция REPLACE для замены текста, что происходит в определенное расположение в текстовую строку. Если вы хотите заменить определенный текст в текстовой строке, используйте функцию ЗАМЕНЫ.
  
## <a name="example"></a>Пример

ЗАМЕНИТЕ («01/03/2002», 9,2, «03») 
  
Возвращает 01/03/2003.
