# <a name="replace-function-visioshapesheet"></a>REPLACE Function (VisioShapeSheet)

Заменяет часть текстовой строки на заданное число символов другой текстовой строкой.
  
## <a name="syntax"></a>Синтаксис

RePLACE (* * *стар_текст* * *, * * *Нач_позиция* * *, * * *Количество_знаков* * *, * * *новый_текст* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _стар_текст_ <br/> |Обязательный  <br/> |**String** <br/> |Текст, в котором нужно заменить некоторые символы.  <br/> |
| _равн_ <br/> |Обязательный  <br/> |**Number** <br/> |Позиция знака в _тексте старый_текст_ , которую нужно заменить на _новый_текст_. Первый символ в строке равен позиции 1.  <br/> |
| _Количество_знаков_ <br/> |Обязательный  <br/> |**Number** <br/> |Число знаков в тексте _старый_текст_ , которые необходимо заменить  <br/> |
| _новый_текст_ <br/> |Обязательный  <br/> |**String** <br/> |Текст, который заменит символы в _тексте старый_текст_.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Используйте функцию rePLACE, чтобы заменить текст, который находится в определенном месте в текстовой строке. Если вы хотите заменить определенный текст в текстовой строке, используйте функцию подСТАНОВКи.
  
## <a name="example"></a>Пример

REPLACE ("01/03/2002", 9, 2, "03") 
  
Возвращает 01/03/2003.
