# <a name="drawingresizetype-cell-page-properties-section"></a>Ячейка DrawingResizeType (раздел "Свойства страницы")

Определяет ли страницы рисунка изменяет размер автоматически в соответствии со на схеме. 
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку DrawingResizeType по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте следующие. 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |DrawingResizeType  <br/> |
   
Для получения ссылки на ячейки DrawingResizeType по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPage** <br/> |
|Индекс ячейки:  <br/> |**visPageDrawResizeType** <br/> |
