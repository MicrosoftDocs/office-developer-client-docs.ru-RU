# <a name="infiniteline-row-geometry-section"></a>Строка InfiniteLine (раздел "Геометрия")

Содержит *x* - и *y* -координаты двух точек на строку не ограничен. 
  
Строка InfiniteLine содержит следующие ячеек.
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X* -координаты точки на строке не ограничен. в сочетании с *y* -координата, представленный в ячейку Y.  <br/> |
|[Да](y-cell-geometry-section.md) <br/> |*Y* -координаты точки на строке не ограничен. в сочетании с *x* -координата, представленный в ячейку X.  <br/> |
|[A](a-cell-geometry-section.md) <br/> |*X* -координаты точки на строке не ограничен. в сочетании с *y* -координата, представленное B ячейки.  <br/> |
|[B](b-cell-geometry-section.md) <br/> |*Y* -координаты точки на неограниченное строке. в сочетании с *x* -координата, представленное ячейке.  <br/> |
   
## <a name="remarks"></a>Замечания

Раздел геометрии, в котором содержится эллипса или InfiniteLine строка не должен содержать других строк.
