# <a name="infiniteline-row-geometry-section"></a>InfiniteLine Row (Geometry Section)

Содержит  *x*  и  *y-координаты*  двух точек на бесконечной строке. 
  
Строка InfiniteLine содержит следующие ячейки.
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X-координата* точки на бесконечной строке; в паре *с y-координатами,* представленными ячейкой Y.  <br/> |
|[Да](y-cell-geometry-section.md) <br/> |y-координата точки на бесконечной строке;  в паре с x-координатами, представленными ячейкой X.   <br/> |
|[A](a-cell-geometry-section.md) <br/> |*X-координата* точки на бесконечной строке; в паре *с y-coordinate,* представленным ячейкой B.  <br/> |
|[B](b-cell-geometry-section.md) <br/> |y-координата точки на бесконечной строке;  в паре с x-координатами, представленными ячейкой A.   <br/> |
   
## <a name="remarks"></a>Примечания

Раздел геометрии, содержащий строку Ellipse или InfiniteLine, не должен содержать другие строки.
