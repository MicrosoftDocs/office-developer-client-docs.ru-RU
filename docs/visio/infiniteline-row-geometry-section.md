# <a name="infiniteline-row-geometry-section"></a>InfiniteLine Row (Geometry Section)

Содержит  *координаты x*  и  *y*  двух точек на бесконечной линии. 
  
Строка InfiniteLine содержит следующие ячейки.
  
|**Cell**|**Описание**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |X-координата точки на бесконечной линии;  сопряжено с *координатой y,* представленной ячейкой Y.  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |Y-координата точки на бесконечной линии;  сопряжено  с x-координатой, представленной ячейкой X.  <br/> |
|[A](a-cell-geometry-section.md) <br/> |X-координата точки на бесконечной линии;  в паре *с координатой y,* представленной ячейкой B.  <br/> |
|[B](b-cell-geometry-section.md) <br/> |Y-координата точки на бесконечной линии;  сопряжено  с x-координатой, представленной ячейкой A.  <br/> |
   
## <a name="remarks"></a>Примечания

Раздел геометрии, содержащий строку Ellipse или InfiniteLine, не должен содержать другие строки.
