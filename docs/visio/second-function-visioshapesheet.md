# <a name="second-function-visioshapesheet"></a>SECOND Function (VisioShapeSheet)

Возвращает компоненты от 0 до 59, которые представляют собой компонент секунд времени даты _или_ _выражения._
  
## <a name="syntax"></a>Синтаксис

SECOND(" ** *datetime* ** "| ** *выражение* ** [, ** *lcid* ** ]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _datetime_ <br/> |Обязательный  <br/> |**String** <br/> |Любая строка, распознаваемая как дата и время либо ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**String** <br/> | Любое выражение, возвращающее дату и время.  <br/> |
| _lcid_ <br/> |Необязательный  <br/> |**Числовой** <br/> |Идентификатор locale, который будет использоваться для оценки нелокального _времени даты._ Идентификатор языкового стандарта — это число, представленной в файлах системных заголовков.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Целое число
  
## <a name="remarks"></a>Замечания

Компонент даты в  _дате_ или  _выражении_ удаляется. 
  
Округление не выполняется. Если  _дата отсутствует_ или не может быть преобразована в допустимый результат, эта функция возвращает ошибку. 
  
Функция SECOND также принимает одно значение  номера для выражения, где десятичная часть результата представляет долю дня с полуночи. 
  
## <a name="example-1"></a>Пример 1

SECOND ("05/30/1997 13:45:32")
  
Возвращает 32.
  
## <a name="example-2"></a>Пример 2

SECOND (TIMEVALUE("30 мая 1996 г. 12:07:45") + 7 es.)
  
Возвращает 52.
  
## <a name="example-3"></a>Пример 3

SECOND (0.6337)
  
Возвращает 32.
