# <a name="document-properties-section"></a>Раздел "Свойства документа"

Содержит ячейки для документа, которые управляют качество предварительного просмотра, область и выходной формат.
  
## <a name="remarks"></a>Замечания

 Можно также задать эти значения с помощью диалогового окна **Свойства** (перейдите на вкладку **файл** , щелкните **информация**, выберите пункт **Свойства**и выберите команду **Дополнительные свойства**).
