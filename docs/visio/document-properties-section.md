# <a name="document-properties-section"></a>Document Properties Section

Содержит ячейки для документа, которые управляют качеством, областью и выходным форматом предварительного просмотра.
  
## <a name="remarks"></a>Примечания

 Вы также можете задать эти значения с помощью диалогового окна **Свойства** (перейдите на вкладку **файл** , щелкните **сведения**, выберите **Свойства**, а затем щелкните **Дополнительные свойства**).
