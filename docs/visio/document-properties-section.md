# <a name="document-properties-section"></a>Document Properties Section

Содержит ячейки для документа, который управляет качеством, областью и форматом вывода.
  
## <a name="remarks"></a>Примечания

 Эти значения можно также задать с помощью диалоговой окне **Свойства** (щелкните вкладку **Файл,** щелкните **Info,** щелкните Свойства **и** нажмите **кнопку Advanced Properties).**
