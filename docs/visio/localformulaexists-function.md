# <a name="localformulaexists-function"></a>Функция LOCALFORMULAEXISTS

Указывает, содержит ли указанная ячейка локализованную формулу. 
  
## <a name="syntax"></a>Синтаксис

LOCALFORMULAEXISTS (** *cellref* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _cellref_ <br/> |Обязательно  <br/> |**Строка** <br/> | Ячейка, которую необходимо проверить на наличие формулы.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Boolean
  
## <a name="remarks"></a>Примечания

Функция LOCALFORMULAEXISTS возвращает 1, если ячейка содержит локализованную формулу; если формула не существует или она наследуется, возвращается значение 0 (ноль).
