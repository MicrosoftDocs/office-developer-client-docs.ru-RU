# <a name="linecolortrans-cell-line-format-section"></a>Ячейка LineColorTrans (раздел "Формат линий")

Определяет уровень прозрачности цвета линии фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 — 100  <br/> |Представляет собой процентную долю прозрачность. Значение по умолчанию — 0% (полностью непрозрачный).  <br/> |
   
## <a name="remarks"></a>Замечания

Значения округляются до ближайшего большего половины процентов. Значение 100% является полностью прозрачным. Несмотря на то, что фигуры с цветом полностью прозрачным строки отображается же, как фигуры, без строк на странице документа, он будет взаимодействовать с другими объектами на странице же образом как в случае его прозрачность 0%. 
  
Также можно задать это значение с помощью элемента управления ползунок в диалоговом окне " **строка** " (на вкладке **Главная** в группе **фигуры** щелкните **строку**, выберите пункт **Вес**и нажмите кнопку **Другие линии**).
  
Чтобы получить ссылку на ячейку LineColorTrans по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |LineColorTrans  <br/> |
   
Для получения ссылки на ячейки LineColorTrans по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowLine** <br/> |
|Индекс ячейки:  <br/> |**visLineColorTrans** <br/> |
