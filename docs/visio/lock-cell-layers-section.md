# <a name="lock-cell-layers-section"></a>Lock Cell (Layers Section)

Указывает, заблокированы ли фигуры, принадлежащие слою, для выбора или изменения.
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Фигуры заблокированы.  <br/> |
|FALSE  <br/> |Фигуры не блокируются.  <br/> |
   
## <a name="remarks"></a>Примечания

Вы также можете задать это значение, выбрав пункт **Блокировка** в диалоговом окне **Свойства слоя** (на вкладке **Главная** , в группе **Правка** щелкните **слои**, а затем выберите **Свойства слоя**).
  
Чтобы получить ссылку на ячейку Lock по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |Слои. заблокировано [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку Lock по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**Виссектионлайер** <br/> |
|Индекс строки:  <br/> |**висровлайер** +  *i* , где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**Вислайерлокк** <br/> |
