# <a name="transparency-cell-layers-section"></a>Прозрачность ячейки (слои раздел)

Определяет уровень прозрачности цвета слоя.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 — 100  <br/> |Представляет собой процентную долю прозрачность. Значение по умолчанию — 0% (полностью непрозрачный).  <br/> |
   
## <a name="remarks"></a>Замечания

Значения округляются до ближайшего большего половины процентов. Значение 100% является полностью прозрачным. Несмотря на то, что уровень полностью прозрачным цвета появится на странице документа как уровень не цвета, взаимодействия с других объектов на странице так же, как как если бы его прозрачность 0%. Также можно задать это значение с помощью элемента управления ползунок в диалоговом окне **Свойства слоя** (на вкладке **Главная** в группе **Редактирование** щелкните **слои**и выберите пункт **Свойства Layer**).
  
Чтобы получить ссылку на ячейку прозрачность по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Layers.ColorTrans [ *i* ] где *i* = < 1 > 2, 3...  <br/> |
   
Для получения ссылки на ячейки прозрачность по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionLayer** <br/> |
|Индекс строки:  <br/> |**visRowLayer** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visLayerColorTrans** <br/> |
