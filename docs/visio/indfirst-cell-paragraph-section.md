# <a name="indfirst-cell-paragraph-section"></a>IndFirst Cell (Paragraph Section)

Представляет расстояние, на которое в текстовом блоке фигуры находится первая строка каждого абзаца, отступив от левого отступа абзаца. Это значение не зависит от масштаба рисунка. Если рисунок масштабироваться, отступ первой строки остается тем же.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку IndFirst по имени из другой формулы или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Para.IndFirst[  *i*  ] где  *i*  = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку IndFirst по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionParagraph** <br/> |
| Индекс строки:  <br/> |**visRowParagraph**  +   *i,* *где i* = 0, 1, 2 ...  <br/> |
| Индекс ячейки:  <br/> |**visIndentFirst** <br/> |
