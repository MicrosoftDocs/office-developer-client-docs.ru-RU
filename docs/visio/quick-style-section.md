# <a name="quick-style-section"></a>Раздел "Экспресс-стиль"

Отображает текущий экспресс-стилей форматирования значения, применяемые к фигуры, включая цвет линии и стиль, цвет заливки и стиль, цвет затенения, эффекты, шрифт и стиль. 
  
## <a name="remarks"></a>Замечания

Вы можете экспресс-стилей выбрав фигуры и затем выбрав новый экспресс-стилей (на вкладке **Главная** в группе **Стили фигуру** нажмите кнопку **Дополнительно**и выберите другой экспресс-стилей из коллекции).
