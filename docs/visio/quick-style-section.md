# <a name="quick-style-section"></a>Quick Style Section

Показывает текущие значения форматирования быстрых стилей, примененные к фигуре, в том числе цвет и стиль линии, цвет и стиль заливки, цвет тени, эффекты, шрифт и тип стиля. 
  
## <a name="remarks"></a>Примечания

Вы можете быстро выбрать фигуру, а затем выбрать новый экспресс-стиль (на вкладке **Главная** , в группе **Стили фигур** нажмите кнопку **Дополнительно**, а затем выбрать другой экспресс-стиль из коллекции).
