# <a name="quick-style-section"></a>Quick Style Section

Показывает текущие значения форматирования quick Style, примененные к фигуре, включая цвет и стиль линии, цвет заливки и стиль, цвет теней, эффекты, шрифт и тип стиля. 
  
## <a name="remarks"></a>Примечания

Вы можете быстро выбрать фигуру, а затем выбрать новый быстрый стиль  (на вкладке "Главная" в группе стилей фигур щелкните "Еще" и выберите другой стиль в коллекции).
