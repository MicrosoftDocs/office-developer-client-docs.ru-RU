# <a name="plowcode-cell-page-layout-section"></a>Ячейка PlowCode (раздел "Макет страницы")

Определяет, будет ли размещаемые фигуры перемещение (удаление) при удалении размещаемой фигуры рядом с другой размещаемой фигуры на странице документа.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Не перемещать фигур  <br/> |**visPLOPlowNone** <br/> |
|1  <br/> |Перемещение фигур  <br/> |**visPLOPlowAll** <br/> |
   
## <a name="remarks"></a>Замечания

Значение ячейки также можно настроить на вкладке **Макет и маршрутизации** в диалоговом окне " **Параметры страницы** " (на вкладке " **Конструктор** ", щелкните стрелку **Параметры страницы** ) с помощью флажок **Сдвигать другие фигуры при размещении** . 
  
Чтобы получить ссылку на ячейку PlowCode по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |PlowCode  <br/> |
   
Для получения ссылки на ячейки PlowCode по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPageLayout** <br/> |
|Индекс ячейки:  <br/> |**visPLOPlowCode** <br/> |
