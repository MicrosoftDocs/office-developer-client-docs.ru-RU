# <a name="shape-transform-section"></a>Shape Transform Section

Содержит общие сведения о размещении фигуры: например, ее ширину, высоту, угол и центр вращения (ПИН-код). Указывает, была ли фигура отражена; и как должна вести себя форма при изменении размеров в группе.
