# <a name="sketchseed-cell-additional-effect-properties-section"></a>Ячейка SketchSeed (раздел "Дополнительные свойства эффекта")

Представляет значение случайный выбор используется для определения геометрии эффекта эскиза как положительное целое число. Значение ячейки **SketchSeed** создается случайным образом при применении эффекта эскиза фигуры. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **SketchSeed** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | SketchSeed  <br/> |
   
Для получения ссылки на ячейки **SketchSeed** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowOtherEffectProperties** <br/> |
| Индекс ячейки:  <br/> |**visSketchSeed** <br/> |
