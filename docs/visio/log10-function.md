# <a name="log10-function"></a>Функция LOG10

Возвращает логарифм числа с основанием 10. Число должно быть положительным или LOG10 Возвращает значение ошибки #NUM.
  
## <a name="syntax"></a>Синтаксис

LOG10 (* * *Number* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательный  <br/> |**Числовой** <br/> | Число, десятичный логарифм которого требуется найти.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="example"></a>Пример

LOG10 (42) 
  
Возвращает 1,6232.
