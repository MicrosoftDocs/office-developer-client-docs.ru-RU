# <a name="pi-function"></a>Функция PI

Возвращает математическое постоянное число *Пи* , которое приблизительно равно 3.1415926535898. 
  
## <a name="syntax"></a>Синтаксис

PI ()
  
### <a name="return-value"></a>Возвращаемое значение

Число
  
## <a name="example"></a>Пример

(14 в/2) ^ 2 \* Пи () 
  
Возвращает 153,9380 дюймов, область круга, диаметр которого составляет 14 дюймов.
