# <a name="lineweight-cell-line-format-section"></a>Толщина линии ячейки (раздел формат строки)

Определяет Толщина линии фигуры. Задайте Толщина линии путем ввода цифр с допустимой единицы измерения.
  
## <a name="remarks"></a>Замечания

Можно также задать значение Толщина линии в диалоговом окне " **строка** " (на вкладке **Главная** в группе **фигуры** щелкните **строку**, выберите пункт **Вес**и нажмите кнопку **Другие линии**).
  
Если единицы измерения не введена, используется единицу измерения для текста, указанного в диалоговом окне **Параметры Visio** (перейдите на вкладку **файл** и выберите пункт **Параметры**). Толщина линии не зависит от масштаба документа. Если документа изменяется, толщина линии остается неизменным. 
  
Для получения ссылки на ячейки Толщина линии по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Толщина линии  <br/> |
   
Для получения ссылки на ячейки Толщина линии по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowLine** <br/> |
| Индекс ячейки:  <br/> |**visLineWeight** <br/> |
