# <a name="aligntop-cell-alignment-section"></a>Ячейка AlignTop (раздел "Выравнивание")

Определяет позицию по вертикали, по отношению к источнику родительского объекта по горизонтали, направляющей или направляющей точки, относительно которого выровнена верхняя граница фигуры.
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на ячейку AlignTop по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AlignTop  <br/> |
   
Для получения ссылки на ячейки AlignTop по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowAlign** <br/> |
| Индекс ячейки:  <br/> |**visAlignTop** <br/> |
