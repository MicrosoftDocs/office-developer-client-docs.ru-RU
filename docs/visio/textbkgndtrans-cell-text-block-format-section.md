# <a name="textbkgndtrans-cell-text-block-format-section"></a>Ячейка TextBkgndTrans (раздел формат блока текст)

Определяет уровень прозрачности цвета фона блока текста фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 — 100  <br/> |Представляет собой процентную долю прозрачность. Значение по умолчанию — 0% (полностью непрозрачный).  <br/> |
   
## <a name="remarks"></a>Замечания

Значения округляются до ближайшего большего половины процентов. Значение 100% является полностью прозрачным. Хотя фигуры, полностью прозрачным текст фон отображается на странице документа как фигуры, не содержит текст фона, взаимодействия с другие объекты на странице таким же образом, как если бы его прозрачность 0%.
  
Также можно задать это значение, с помощью элемента управления ползунок на вкладке **Шрифт** диалоговое окно " **Text** " (на вкладке **Главная** щелкните стрелку **шрифтов** ). 
  
Чтобы получить ссылку на ячейку TextBkgndTrans по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |TextBkgndTrans  <br/> |
   
Для получения ссылки на ячейки TextBkgndTrans по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowText** <br/> |
|Индекс ячейки:  <br/> |**visTxtBlkBkgndTrans** <br/> |
