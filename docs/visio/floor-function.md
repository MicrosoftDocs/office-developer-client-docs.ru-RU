# <a name="floor-function"></a>Функция FLOOR

Округляет число в сторону нуль (0), целого или до ближайшего из _нескольких_.
  
## <a name="syntax"></a>Синтаксис

ЭТАЖА (** *номер* **, ** *несколько* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Обязательный  <br/> |**Число** <br/> |Чтобы округлить число.  <br/> |
| _несколько_ <br/> |Обязательный  <br/> |**Число** <br/> |Множитель, к которому следует округлить.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Number
  
## <a name="remarks"></a>Замечания

Если _несколько_ не указан, число округляется к 0 до ближайшего целого числа. 
  
 _Номер_ и _нескольких_ должны иметь одинаковые знаки или #NUM! возвращается ошибка. Если _номер_ или _нескольких_ невозможно преобразовать в значение #VALUE! возвращается ошибка. Если _номер_ или _нескольких_ равно 0, результатом будет 0. 
  
## <a name="example-1"></a>Пример 1

FLOOR(3.7)
  
Возврат значения 3.
  
## <a name="example-2"></a>Пример 2

FLOOR(-3.7)
  
Возвращает -3.
  
## <a name="example-3"></a>Пример 3

ЭТАЖА (3,7, 0,5)
  
Возвращает 3.5.
