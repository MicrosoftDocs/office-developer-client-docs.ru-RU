# <a name="floor-function"></a>Функция FLOOR

Округлит число к 0 (ноль), к следующему экземпляру или к следующему экземпляру _из нескольких экземпляров._
  
## <a name="syntax"></a>Синтаксис

FLOOR(** *номер* **, ** *несколько* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательный  <br/> |**Number** <br/> |Число к округлу.  <br/> |
| _несколько_ <br/> |Обязательный  <br/> |**Number** <br/> |Множестве, к которому можно округлиться.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Номер
  
## <a name="remarks"></a>Примечания

Если  _несколько_ не указано, число округлится до 0 к следующему ряду. 
  
 _Число_ и  _несколько_ должны иметь одинаковые знаки или #NUM! возвращается ошибка. Если число  _или_  _несколько_ не могут быть преобразованы в значение, #VALUE! возвращается ошибка. Если число  _или_  _несколько_ — 0, результат — 0. 
  
## <a name="example-1"></a>Пример 1

FLOOR (3.7)
  
Возвращает 3.
  
## <a name="example-2"></a>Пример 2

FLOOR (-3.7)
  
Возвращает -3.
  
## <a name="example-3"></a>Пример 3

FLOOR (3.7, 0.5)
  
Возвращает 3.5.
