# <a name="controls-row-controls-section"></a>Строка Controls (раздел "Элементы управления")

Содержит ячейки, которые определяют *x* - и *y* -координаты и поведением маркера каждого элемента управления, определенных для фигуры. Фигура будет содержать одну строку элементов управления для каждого дескриптора элемента управления. 
  
Элементы управления строки имеют имена элементов управления. *имя* и содержат следующие ячейки. Для получения дополнительных сведений см определенную ячейку. 
  
|**Cell**|**Описание**|
|:-----|:-----|
|[x](x-cell-controls-section.md) <br/> |Представляет *x* -координата, который указывает расположение управляющего маркера фигуры в локальной системе координат.  <br/> |
|[y](y-cell-controls-section.md) <br/> |Представляет *y* -координата, который указывает расположение управляющего маркера фигуры в локальной системе координат.  <br/> |
|[X Dynamics](x-dynamics-cell-controls-section.md) <br/> |Представляет *x* -координаты точки привязки управляющего маркера в локальной системе координат.  <br/> |
|[Y Dynamics](y-dynamics-cell-controls-section.md) <br/> |Представляет *y* -координаты точки привязки управляющего маркера в локальной системе координат.  <br/> |
|[X поведение](x-behavior-cell-controls-section.md) <br/> |Управляет поведением *x* -координата дескриптора элемента управления будет отличаться от после перемещения.  <br/> |
|[Поведение Y](y-behavior-cell-controls-section.md) <br/> |Управляет поведением *y* -координата дескриптора элемента управления будет отличаться от после перемещения.  <br/> |
|[Можно связывания](can-glue-cell-controls-section.md) <br/> |Определяет, может ли быть связан дескриптор элемента управления на другие фигуры.  <br/> |
|[Совет](tip-cell-controls-section.md) <br/> |Представляет строку описательный текст, который отображается в виде подсказки при наведении указателя мыши на элемент управления маркера фигуры.  <br/> |
   
## <a name="remarks"></a>Замечания

 Можно добавить столько элементов управления.  *имя* строки при необходимости назначить понятные имена строк и задайте значения ячейки. Чтобы добавить маркеры элемента управления в существующий раздел элементы управления, щелкните правой кнопкой мыши строку и нажмите кнопку **Вставить строку** в контекстном меню. 
  
Можно ссылаться на эти ячейки с их имя строки, который отображается в окне таблицы свойств фигуры текстом красного цвета. Чтобы назначить понятные имена элементов управления. *имя* строки, щелкните строку, а затем введите *Custom* , например, для создания Controls.Custom имя строки. Затем можно ссылаться на ячейки X, с помощью Controls.Custom.X. 
  
Имя строки при вводе должно быть уникальным в рамках раздела.
