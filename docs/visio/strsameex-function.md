# <a name="strsameex-function"></a>Функция STRSAMEEX

Определяет, одинаковы ли две строки.
  
## <a name="syntax"></a>Синтаксис

STRSAMEEX (» ** *string1* ** «,» ** *string2* ** «, ** *localeID* **, ** *флаг* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _string1_ <br/> |Обязательный  <br/> |**Строка** <br/> |Первая строка для сравнения.  <br/> |
| _Строка string2_ <br/> |Обязательный  <br/> |**Строка** <br/> | Вторая строка для сравнения.  <br/> |
| _localeID_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Код идентификатор языкового стандарта.  <br/> |
| _флаг_ <br/> |Обязательный  <br/> |**Числовой** <br/> | Бит, указывающий тип сравнения.  <br/> |

Логический
  
## <a name="remarks"></a>Замечания

STRSAMEEX возвращает значение TRUE, если оба входных строк совпадают с и значение FALSE, если они не. Эта функция используется для сравнения строк из нескольких байтов или для выполнения сравнения, использующие делами правил для конкретного языка.
  
Сочетание любых из следующих флагов можно использовать с помощью функции STRSAMEEX.
  
|**Flag**|**Описание**|
|:-----|:-----|
|1  <br/> |Игнорируйте регистр.  <br/> |
|2  <br/> |Пропуск дополнительный символов.  <br/> |
|4  <br/> |Игнорируйте символы.  <br/> |
|4096  <br/> |Обрабатывать знаков препинания так же, как символы.  <br/> |
|65536  <br/> |Не различать хирагана и катакана.  <br/> |
|131072  <br/> |Не различать однобайтовых знаков и со знаком двухбайтовых знаков.  <br/> |
