# <a name="strsameex-function"></a>Функция STRSAMEEX

Определяет, совпадают ли две строки.
  
## <a name="syntax"></a>Синтаксис

STRSAMEEX ("* * *строка_поиска* * *", "* * *строка2* * *", * * *LocaleID* * *, * * *Flag* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _строка1_ <br/> |Обязательный  <br/> |**String** <br/> |Первая сравниваемая строка.  <br/> |
| _строка2_ <br/> |Обязательный  <br/> |**String** <br/> | Вторая сравниваемая строка.  <br/> |
| _Языка_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Код языка.  <br/> |
| _flag_ <br/> |Обязательный  <br/> |**Числовой** <br/> | Бит, указывающий тип сравнения.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Boolean
  
## <a name="remarks"></a>Примечания

STRSAMEEX возвращает значение TRUE, если входные строки одинаковы, и значение FALSE в противном случае. Эта функция используется для сравнения многобайтовых строк или для сравнения, использующих правила для определенного языкового стандарта.
  
С помощью функции STRSAMEEX можно использовать любой из следующих флагов.
  
|**Флаг**|**Описание**|
|:-----|:-----|
|1,1  <br/> |Не учитывать регистр.  <br/> |
|2  <br/> |ПроПускать символы, отличные от пробелов.  <br/> |
|SP4  <br/> |Игнорировать символы.  <br/> |
|4096  <br/> |Знаки препинания считаются символами.  <br/> |
|65536  <br/> |Не различаются символы хирагана и катакана.  <br/> |
|131072  <br/> |Не следует отличать однобайтовый символ и тот же символ как двухбайтовый символ.  <br/> |
