# <a name="strsameex-function"></a>Функция STRSAMEEX

Определяет, одинаковы ли две строки.
  
## <a name="syntax"></a>Синтаксис

STRSAMEEX (" ** *string1* ** ", " ** *string2* ** ", ** *localeID* **, ** *flag* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _строка1_ <br/> |Обязательно  <br/> |**Строка** <br/> |Первая строка для сравнения.  <br/> |
| _строка2_ <br/> |Обязательно  <br/> |**Строка** <br/> | Вторая строка для сравнения.  <br/> |
| _localeID_ <br/> |Обязательна  <br/> |**Числовой** <br/> |Код кода кода локализованного кода.  <br/> |
| _flag_ <br/> |Обязательна  <br/> |**Числовой** <br/> | Бит, который указывает тип сравнения.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Boolean
  
## <a name="remarks"></a>Примечания

STRSAMEEX возвращает true, если обе входные строки одинаковы, и FALSE, если они не являются. Используйте эту функцию для сравнения многофайтных строк или сравнения, которые используют правила дела для определенного локаля.
  
Можно использовать сочетание любого из следующих флагов с функцией STRSAMEEX.
  
|**Flag**|**Описание**|
|:-----|:-----|
|1  <br/> |Игнорируйте дело.  <br/> |
|2   <br/> |Игнорируйте символы без интервала.  <br/> |
|4   <br/> |Игнорируйте символы.  <br/> |
|4096  <br/> |Знаки препинания обрабатывают так же, как символы.  <br/> |
|65536  <br/> |Не различайте символы хираганы и Катаканы.  <br/> |
|131072  <br/> |Не различайте один и тот же символ, что и двухбайт.  <br/> |
