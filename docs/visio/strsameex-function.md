# <a name="strsameex-function"></a>Функция STRSAMEEX

Определяет, одинаковы ли две строки.
  
## <a name="syntax"></a>Синтаксис

STRSAMEEX (" ** *string1* ** ", "* *string2* ** ", ** *localeID* **, ** *flag* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _строка1_ <br/> |Обязательный  <br/> |**String** <br/> |Первая строка для сравнения.  <br/> |
| _строка2_ <br/> |Обязательный  <br/> |**String** <br/> | Вторая строка для сравнения.  <br/> |
| _localeID_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Код кода локального кода.  <br/> |
| _flag_ <br/> |Обязательный  <br/> |**Числовой** <br/> | Немного, который указывает тип сравнения.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Boolean
  
## <a name="remarks"></a>Примечания

STRSAMEEX возвращает TRUE, если обе строки ввода одинаковы, а FALSE — нет. Используйте эту функцию для сравнения строк с несколькими byte или сравнения, которые используют правила дела для определенного локального.
  
Вы можете использовать сочетание любого из следующих флагов с функцией STRSAMEEX.
  
|**Флаг**|**Описание**|
|:-----|:-----|
|1  <br/> |Игнорируйте случай.  <br/> |
|2  <br/> |Игнорируйте не интервальные символы.  <br/> |
|4   <br/> |Игнорировать символы.  <br/> |
|4096  <br/> |Относитесь к пунктуации так же, как к символам.  <br/> |
|65536  <br/> |Не различайте символы Hiragana и Katakana.  <br/> |
|131072  <br/> |Не дифференцируйте один и тот же символ, что и двухбайт.  <br/> |
