# <a name="or-function"></a>Функция OR

Возвращает значение истина (1), если какие-либо из логических выражений, переданных в качестве параметров, имеют значение TRUE.
  
## <a name="syntax"></a>Синтаксис

OR (* * *logicalexpression1* * *, * * *logicalexpression2* * *,..., * * *логикалекспрессионн* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _logicalexpression1_ <br/> |Обязательный  <br/> |**String** <br/> |Первое выражение, для которого необходимо оценить истинность.  <br/> |
| _logicalexpression2_ <br/> |Обязательный  <br/> |**String** <br/> |Второе выражение, для которого необходимо оценить истинность.  <br/> |
| _Логикалекспрессионн_ <br/> |Обязательный  <br/> |**String** <br/> |Выражение n, для которого нужно вычислить истинность.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Boolean
  
## <a name="remarks"></a>Примечания

Любое выражение, результатом которого является ненулевое значение, считается ИСТИНным. Если все логические выражения имеют значение ложь или равно 0, эта функция возвращает значение FALSE. 
  
## <a name="example"></a>Пример

ИЛИ (Height \> 1, PinX \> 1) 
  
Возвращает значение истина (1), если любое из выражений имеет значение TRUE. Возвращает значение FALSE (0) только в том случае, если оба выражения имеют значение FALSE.
