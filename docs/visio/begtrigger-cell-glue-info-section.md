# <a name="begtrigger-cell-glue-info-section"></a>BegTrigger Cell (Glue Info Section)

Содержит формулу триггера, созданную приложением, которая определяет, нужно ли перемещать начальную точку одноМерной фигуры для сохранения ее связи с другой фигурой.
  
## <a name="remarks"></a>Примечания

При приклейте одноМерной фигуры к другой фигуре с помощью динамической приклеивание приложение создает формулу, которая ссылается на ячейку EventXFMod другой фигуры. При изменении этой фигуры Visio пересчитывает любую формулу, ссылающуюся на ячейку EventXFMod, включая формулу в ячейке BegTrigger. Другие формулы для фигуры 1 — это ссылка на ячейку BegTrigger и перемещение начальной точки фигуры 1 – D или изменение фигуры по мере необходимости.
  
Чтобы получить ссылку на ячейку BegTrigger по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | BegTrigger  <br/> |
   
Чтобы получить ссылку на ячейку BegTrigger по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**Висровграуп** <br/> |
| Индекс ячейки:  <br/> |**Висбегтригжер** <br/> |
