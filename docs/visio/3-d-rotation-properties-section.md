# <a name="3-d-rotation-properties-section"></a>Раздел "Свойства поворота объемной фигуры"

Отображает текущие значения фигуры угол вращения по x, y и оси z.
  
> [!NOTE]
> Разделе трехмерной вращение свойства не включает ячеек, которые управляют рельефов эффекты фигур. Чтобы задать эффект рельефов, изменение значений ячеек в разделе [Свойства Багетная рамка](bevel-properties-section.md) . 
  
## <a name="remarks"></a>Замечания

Можно изменить значения поворот объема в диалоговом окне **Формат фигуры** (**Домашняя страница** > **Стили фигур** > **эффекты** > **поворот объема** или изменение в одну из ячеек в **трехмерной вращение свойства **раздел.
