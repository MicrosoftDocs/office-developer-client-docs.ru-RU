# <a name="lineadjustto-cell-page-layout-section"></a>LineAdjustTo Cell (Page Layout Section)

Определяет, какая из динамических соединителей находится сверху друг от друга.
  
|**Значение**|**Прав**|**Константа автоматизации**|
|:-----|:-----|:-----|
|нуль  <br/> |Стиль маршрутизации по умолчанию  <br/> |**Висплолинеаджусттодефаулт** <br/> |
|1,1  <br/> |Линии, близкие друг к другу  <br/> |**Висплолинеаджусттоалл** <br/> |
|2  <br/> |Нет строк  <br/> |**Висплолинеаджусттононе** <br/> |
|4  <br/> |Связанные линии  <br/> |**Висплолинеаджустторелатед** <br/> |
   
## <a name="remarks"></a>Примечания

Значение этой ячейки также можно задать на вкладке **Макет и маршрутизация** в диалоговом окне **Параметры страницы** (на вкладке **конструктор** щелкните стрелку **Параметры страницы** , а затем выберите **Макет и маршрутизация**).
  
Чтобы получить ссылку на ячейку LineAdjustTo по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |LineAdjustTo  <br/> |
   
Чтобы получить ссылку на ячейку LineAdjustTo по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровпажелайаут** <br/> |
|Индекс ячейки:  <br/> |**Висплолинеаджустто** <br/> |
