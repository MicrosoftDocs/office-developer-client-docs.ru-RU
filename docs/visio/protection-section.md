# <a name="protection-section"></a>Protection Section

Показывает текущие параметры блокировки, заданные с помощью команды **защиты** (вкладки [разработчика](run-in-developer-mode-display-the-developer-tab.md) ), а также несколько дополнительных блокировок, которые можно задать только в окне таблицы свойств фигуры. 
  
## <a name="remarks"></a>Замечания

Блокировка помогает предотвратить непреднамеренное изменение фигуры, но не предотвращает сброс значений в других обстоятельствах. Кроме того, он не защищает изменения, внесенные в окно таблицы свойств фигуры.
  
Чтобы защитить формулу от изменения, используйте функцию **Guard** .
