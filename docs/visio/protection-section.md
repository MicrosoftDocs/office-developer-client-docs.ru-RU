# <a name="protection-section"></a>Protection Section

Показывает текущие параметры блокировки с командой **Protection** [(вкладка](run-in-developer-mode-display-the-developer-tab.md) Разработчик) плюс несколько дополнительных замков, которые можно установить только в окне ShapeSheet. 
  
## <a name="remarks"></a>Примечания

Блокировка помогает предотвратить непреднамеренное изменение фигуры, но не препятствует сбросу значений в других обстоятельствах. Он также не защищает от изменений, внесенных в окне ShapeSheet.
  
Чтобы защитить формулу от изменения, используйте функцию **GUARD.**
