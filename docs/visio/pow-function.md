# <a name="pow-function"></a>Функция POW

Возвращает число, возведенное в степень экспоненты.
  
## <a name="syntax"></a>Синтаксис

POW (* * *число* * *, * * *экспонента* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _число_ <br/> |Обязательный  <br/> |**Number** <br/> |Число, которое возводится в степень возведения в степень.  <br/> |
| _exponent_ <br/> |Обязательный  <br/> |**Number** <br/> |Показатель степени.  <br/> |
   
## <a name="remarks"></a>Примечания

_Число_ и _экспонента_ могут быть не целыми числами и могут быть отрицательными. Если _число_ не равно 0 и __ значение экспоненты равно 0, эта функция возвращает 1. Если _число_ равно 0 и _экспонента_ отрицательна, эта функция возвращает 0,0. Если _число_ и _экспонента_ равны 0 или _число_ отрицательно, а _экспонента_ не является целым числом, эта функция возвращает 0,0. Если _число_ и _показатель степени_ отрицательны, эта функция возвращает значение – 1. #IND. 
  
## <a name="example"></a>Пример

POW (5, 2) 
  
Возвращает 25.
