# <a name="pow-function"></a>Функция POW

Возвращает числа, возведенного в степень экспоненты.
  
## <a name="syntax"></a>Синтаксис

POW (** *номер* **, ** *степени* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Обязательный  <br/> |**Число** <br/> |Номер для возведения в степень экспоненты.  <br/> |
| _показатель_ <br/> |Обязательный  <br/> |**Число** <br/> |Степень.  <br/> |
   
## <a name="remarks"></a>Замечания

_Номер_ и _степени_ может быть не целые числа и может быть отрицательным. Если _номер_ не 0 и _степени_ равно 0, эта функция возвращает 1. Если _число_ равняется 0 и _степени_ является отрицательным, эта функция возвращает 0,0. Если _номер_ и _степень_ , 0 или _число_ является отрицательным и _степени_ не является целым числом, эта функция возвращает 0,0. Если отрицательные _числа_ и _степень_ , эта функция возвращает значение -1. #IND. 
  
## <a name="example"></a>Пример

POW(5,2) 
  
Возвращает 25.
