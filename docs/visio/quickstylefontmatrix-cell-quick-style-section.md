# <a name="quickstylefontmatrix-cell-quick-style-section"></a>Ячейка QuickStyleFontMatrix (раздел "Экспресс-стиль")

Определяет стиль шрифта для каждого экспресс-стилей, как целое число от 1 до 6.
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **QuickStyleFontMatrix** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | QuickStyleFontMatrix  <br/> |
   
Для получения ссылки на ячейки **QuickStyleFontMatrix** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowQuickStyleProperties** <br/> |
| Индекс ячейки:  <br/> |**visQuickStyleFontMatrix** <br/> |
