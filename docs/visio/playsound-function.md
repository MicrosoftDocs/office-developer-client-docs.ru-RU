# <a name="playsound-function"></a>Функция PLAYSOUND

Воспроизводит звуковой файл или системный звук. 
  
## <a name="syntax"></a>Синтаксис

PLAYSOUND(" ** *filename* ** "|" ** *alias* ** ", ** *isAlias* **, ** *beep* **, ** *synch* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _filename_ <br/> |Обязательно  <br/> |**Строка** <br/> |Имя звукового файла, который вы хотите воспроизведения.  <br/> |
| _alias_ <br/> |Обязательно  <br/> |**Строка** <br/> | Системный звук, представленный псевдонимом.  <br/> |
| _isAlias_ <br/> |Обязательный  <br/> |**Логический** <br/> | Указывает, является ли предыдущее выражение псевдонимом или именем файла; для указания псевдонима используйте ненулевую величину.  <br/> |
| _beep_ <br/> |Обязательный  <br/> |**Логический** <br/> |Указывает, должен ли Microsoft Visio сигналить, когда не может быть установлен звук; используйте для звукового сигнала ненульное число.  <br/> |
| _synch_ <br/> |Обязательный  <br/> |**Логический** <br/> |Определяет, будут ли звуки играть асинхронно (0) или синхронно (1).  <br/> |
   
## <a name="remarks"></a>Примечания

Обычно звуки следует воспроизводить асинхронно, чтобы Visio можно было продолжать обработку во время воспроизведения звука. Чтобы совмедить несколько звуков, синхронно их можно использовать, или некоторые из них могут не играть. 
  
## <a name="example-1"></a>Пример 1

PLAYSOUND("т.wav", 0, 0, 0, 0)
  
Асинхронно воспроизводит звуковой файл wave,wav без предупреждения.
  
## <a name="example-2"></a>Пример 2

PLAYSOUND("SystemExclamation", 1, 0, 0)
  
Воспроизводит системный восклицательный звук асинхронно без предупреждения.
