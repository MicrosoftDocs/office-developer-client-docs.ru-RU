# <a name="playsound-function"></a>Функция PLAYSOUND

Воспроизведение звукового файла или системного звука. 
  
## <a name="syntax"></a>Синтаксис

PLAYSOUND(" ** *filename* ** "|" ** *alias* ** ", ** *isAlias* **, ** *beep* **, ** *synch* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _имя файла_ <br/> |Обязательный  <br/> |**String** <br/> |Имя нужного звукового файла.  <br/> |
| _alias_ <br/> |Обязательный  <br/> |**String** <br/> | Системный звук, представленный псевдонимом.  <br/> |
| _isAlias_ <br/> |Обязательный  <br/> |**Логический** <br/> | Указывает, является ли предыдущее выражение псевдонимом или именем файла; используйте значение non-zero для указания псевдонима.  <br/> |
| _звуковой сигнал_ <br/> |Обязательный  <br/> |**Логический** <br/> |Указывает, будет ли Visio сигналы, когда звук не может быть сыграно; используйте ненулевую номер для звукового сигнала.  <br/> |
| _synch_ <br/> |Обязательный  <br/> |**Логический** <br/> |Определяет, играются ли звуки асинхронно (0) или синхронно (1).  <br/> |
   
## <a name="remarks"></a>Примечания

Обычно следует асинхронно воспроизводить звуки, Visio можно продолжать обработку во время воспроизведения звука. Чтобы с строкой несколько звуков вместе, играть их синхронно, или некоторые из них могут не играть. 
  
## <a name="example-1"></a>Пример 1

PLAYSOUND ("chord.wav", 0, 0, 0)
  
Воспроизводит аудиофайл волны chord.wav асинхронно без звукового сигнала.
  
## <a name="example-2"></a>Пример 2

PLAYSOUND ("SystemExclamation", 1, 0, 0)
  
Асинхронно воспроизводит звук восклицания системы без звукового сигнала.
