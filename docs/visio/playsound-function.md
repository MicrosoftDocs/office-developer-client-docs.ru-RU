# <a name="playsound-function"></a>Функция PLAYSOUND

Воспроизводит звуковой файл или системный звук. 
  
## <a name="syntax"></a>Синтаксис

PLAYSOUND ("* * *имя_файла* * *" | "* * *Alias* * *", * *- *Alias* * *, * * *Beep* * *, * * *synch* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _задан_ <br/> |Обязательный  <br/> |**String** <br/> |Имя звукового файла, который необходимо прослушать.  <br/> |
| _alias_ <br/> |Обязательный  <br/> |**String** <br/> | Системный звук, представленный псевдонимом.  <br/> |
| _Псевдоним_ <br/> |Обязательный  <br/> |**Логический** <br/> | Указывает, является ли предыдущее выражение псевдонимом или именем файла; Используйте ненулевое значение, чтобы указать псевдоним.  <br/> |
| _гудок_ <br/> |Обязательный  <br/> |**Логический** <br/> |Указывает, следует ли включать звуковые сигналы при невозможности воспроизведения звукового сигнала в Microsoft Visio. Используйте для звукового сигнала ненулевое значение.  <br/> |
| _точный_ <br/> |Обязательный  <br/> |**Логический** <br/> |Определяет, воспроизводится ли звук асинхронно (0) или синхронно (1).  <br/> |
   
## <a name="remarks"></a>Примечания

Обычно воспроизводите звуки асинхронно, чтобы Visio мог продолжать обработку при воспроизведении звука. Чтобы объединить несколько звуков, воспроизводить их синхронно, или некоторые из них могут не воспроизводиться. 
  
## <a name="example-1"></a>Пример 1

PLAYSOUND ("аккорд. wav", 0, 0, 0)
  
Выполняет асинхронное воспроизведение звукового файла аккорд. WAV без сигнала предупреждения.
  
## <a name="example-2"></a>Пример 2

PLAYSOUND ("Системекскламатион", 1, 0, 0)
  
Выполняет асинхронный звук с желтым восклицательным сигналом без предупреждения.
