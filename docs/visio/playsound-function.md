# <a name="playsound-function"></a>Функция PLAYSOUND

Воспроизводит звуковой файл или системный звук. 
  
## <a name="syntax"></a>Синтаксис

PLAYSOUND ("** *filename* **" | "** *псевдоним* **", ** *isAlias* **, ** *Звуковые* **, ** *synch* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _Имя файла_ <br/> |Обязательный  <br/> |**Строка** <br/> |Имя звукового файла, которые необходимо воспроизвести.  <br/> |
| _alias_ <br/> |Обязательный  <br/> |**Строка** <br/> | Система звуковой представленный псевдоним.  <br/> |
| _isAlias_ <br/> |Обязательный  <br/> |**Boolean** <br/> | Указывает, является ли предыдущего выражения псевдоним или имя; Используйте ненулевое значение, чтобы указать псевдоним.  <br/> |
| _звуковой сигнал_ <br/> |Обязательный  <br/> |**Boolean** <br/> |Указывает, будет ли Microsoft Visio издает звуковые сигналы при не может воспроизводиться звук; Используйте ненулевое число звуковые сигналы.  <br/> |
| _Синхронизация_ <br/> |Обязательный  <br/> |**Boolean** <br/> |Определяет, будут ли асинхронно воспроизводиться звуки (0) или синхронно (1).  <br/> |
   
## <a name="remarks"></a>Замечания

Вы должны обычно воспроизведения звуков асинхронно, чтобы Visio можно продолжить обработку во время воспроизведения звука. Чтобы строка несколько звуков друг с другом, их воспроизведения синхронно или некоторые не удается воспроизвести. 
  
## <a name="example-1"></a>Пример 1

PLAYSOUND ("chord.wav", 0, 0, 0)
  
Воспроизводит chord.wav звукового файла звукозаписи асинхронно с не сигнал.
  
## <a name="example-2"></a>Пример 2

PLAYSOUND ("SystemExclamation", 1, 0, 0)
  
Воспроизведение звука асинхронно с не сигнал восклицательный знак системы.
