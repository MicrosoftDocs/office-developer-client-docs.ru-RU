# <a name="blue-function"></a>Функция BLUE

Возвращает синий компонент цвета. Возвращаемое значение — это целое число в диапазоне от 0 до 255 включительно. Функция возвращает значение 0 для недопустимых входных данных.
  
## <a name="syntax"></a>Синтаксис

BLUE (* * *Expression* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**String** <br/> |Индекс цвета в таблице цветов документа, выражение, которое разрешается в настраиваемый цвет (например, RGB или HSL), или ссылка на ячейку, содержащую цветовой индекс или цветовой результат.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Целое число
  
## <a name="example-1"></a>Пример 1

СИНИЙ (sheet. 4! FillForegnd
  
Возвращает синий компонент листа лист. 4's заливки цветом текста.
  
## <a name="example-2"></a>Пример 2

СИНИЙ (13)
  
Возвращает 128, если документ использует цветовую палитру Visio по умолчанию, где голубой — цвет с индексом 13.
  
## <a name="example-3"></a>Пример 3

BLUE (RGB (10, 20, 30))
  
Возвращает значение 30.
