# <a name="hsl-function"></a>Функция HSL

Возвращает значение, представляющее индекс в цветовой палитре документа. Он определяет цвет с помощью компонентов тона, насыщенности и яркости.
  
## <a name="syntax"></a>Синтаксис

HSL (* * *тон* * *, * * *насыщенность* * *, * * *яркость* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _тона_ <br/> |Обязательный  <br/> |**Number** <br/> |Оттенок цвета, выраженный как число в диапазоне от 0 до 239 включительно или выражение, результатом которого является такое число.  <br/> |
| _насыщенность_ <br/> |Обязательный  <br/> |**Number** <br/> |Насыщенность цвета, выраженная в виде числа в диапазоне от 0 до 240 включительно или выражение, результатом которого является такое число.  <br/> |
| _яркости_ <br/> |Обязательный  <br/> |**Number** <br/> | Яркость цвета, выраженная в виде числа в диапазоне от 0 до 240 включительно или выражение, результатом которого является такое число.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Число
  
## <a name="remarks"></a>Примечания

Если цвет, возвращенный функцией, еще не существует в цветовой палитре текущего документа, он добавляется в список доступных цветов документа. 
  
В приведенной ниже таблице перечислены некоторые стандартные цвета и значения тона, насыщенности и яркости. 
  
|**Цвет**|**Значение тона**|**Значение насыщенности**|**Значение яркости**|
|:-----|:-----|:-----|:-----|
|Черный  <br/> |нуль  <br/> |нуль  <br/> |открыт  <br/> |
|Синий  <br/> |160  <br/> |240  <br/> |120  <br/> |
|Зеленый  <br/> |80  <br/> |240  <br/> |120  <br/> |
|Cyan  <br/> |120  <br/> |240  <br/> |120  <br/> |
|Красный  <br/> |нуль  <br/> |240  <br/> |120  <br/> |
|Пурпурный  <br/> |200  <br/> |240  <br/> |120  <br/> |
|Желтый  <br/> |40  <br/> |240  <br/> |120  <br/> |
|Белый  <br/> |нуль  <br/> |нуль  <br/> |240  <br/> |
   
## <a name="example-1"></a>Пример 1

HSL (160240120)
  
Возвращает индекс для синего цвета.
  
## <a name="example-2"></a>Пример 2

HSL (отТЕНОК (FillForegnd), Кот (FillForegnd), MIN (яркость (FillForegnd) + 100240))
  
Возвращает индекс цвета, который отражает цвет переднего плана заливки с увеличенной яркостью.
