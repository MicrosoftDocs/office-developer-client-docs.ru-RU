# <a name="hsl-function"></a>Функция HSL

Возвращает значение, представляющее индекса в документе цветовой палитры. Он указывает цвет по его тона, насыщенность и яркость компонентов.
  
## <a name="syntax"></a>Синтаксис

HSL (** *тона* **, ** *насыщенность* **, ** *яркость* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _тона_ <br/> |Обязательный  <br/> |**Число** <br/> |Цветового тона, выраженное как число в диапазоне 0 для 239 включительно, или выражение, которое оценивается как такое число.  <br/> |
| _насыщенность_ <br/> |Обязательный  <br/> |**Число** <br/> |Насыщенность цвета, выраженное как число в диапазоне от 0 до 240 включительно, или выражение, которое оценивается как такое число.  <br/> |
| _«яркость»_ <br/> |Обязательный  <br/> |**Число** <br/> | Яркость цвета, выраженное как число в диапазоне от 0 до 240 включительно, или выражение, которое оценивается как такое число.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Number
  
## <a name="remarks"></a>Замечания

Если цвет, возвращаемого функцией еще не включенное в текущем документе цветовой палитры, добавляется документа списка доступных цветов. 
  
В следующей таблице перечислены некоторые стандартные цвета и тона, насыщенность и яркость значения. 
  
|**Цвет**|**Значение тона**|**Насыщенность значение**|**Значение яркости**|
|:-----|:-----|:-----|:-----|
|Черный  <br/> |0  <br/> |0  <br/> |24  <br/> |
|Синий  <br/> |160  <br/> |240  <br/> |120  <br/> |
|Зеленый  <br/> |80  <br/> |240  <br/> |120  <br/> |
|Голубой  <br/> |120  <br/> |240  <br/> |120  <br/> |
|Красный  <br/> |0  <br/> |240  <br/> |120  <br/> |
|Пурпурный  <br/> |200  <br/> |240  <br/> |120  <br/> |
|Желтый  <br/> |40  <br/> |240  <br/> |120  <br/> |
|Белый  <br/> |0  <br/> |0  <br/> |240  <br/> |
   
## <a name="example-1"></a>Пример 1

HSL(160,240,120)
  
Возвращает индекс цвета синего.
  
## <a name="example-2"></a>Пример 2

HSL(HUE(FillForegnd),sat(FillForegnd),Min(LUM(FillForegnd)+100,240))
  
Возвращает индекс для цвета, который отражает цвет заливки с повышенной «яркость».
