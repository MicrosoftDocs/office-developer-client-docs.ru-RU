# <a name="lookup-function"></a>Функция LOOKUP

Возвращает нулевой индекс, указывающий расположение подстройки в списке, или возвращает -1, если целевая строка содержит _делимитер._  
  
## <a name="syntax"></a>Синтаксис

LOOKUP(" ** *key* ** "," ** *list* ** "[," ** *delimiter* ** "]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _key_ <br/> |Обязательный  <br/> |**String** <br/> |Строка, которую нужно искать.  <br/> |
| _list_ <br/> |Обязательный  <br/> |**String** <br/> | Список, в котором необходимо искать.  <br/> |
| _delimiter_ <br/> |Необязательный  <br/> |**String** <br/> | Строка, используемая в качестве делимитера в _списке._ Строка  _делимитера_ может иметь несколько символов в длину и может включать многобайтные символы. По умолчанию это полуколон.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="remarks"></a>Примечания

В функции LOOKUP используется нечувствительный поиск. Если список начинается или заканчивается делимитером, предполагается, что строка null существует до или после списка. Последовательные делимитеры подразумевают нулевую строку между ними. 
  
Все аргументы должны быть строками или выражениями, которые можно преобразовать в строки. Если это не так, пустая строка заменяется аргументом обижающей. 
  
## <a name="example-1"></a>Пример 1

LOOKUP ("крыса", "cat;rat;; коза")
  
Возвращает 1.
  
## <a name="example-2"></a>Пример 2

LOOKUP ("",";cat;rat;; коза")
  
Возвращает 0.
  
## <a name="example-3"></a>Пример 3

LOOKUP ("t", "cat;rat;; коза", "a")
  
Возвращает 3.
