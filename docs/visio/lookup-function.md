# <a name="lookup-function"></a>Функция LOOKUP

Возвращает отсчитываемый от нуля индекс, указывающий расположение _ключа_ подстроки в _списке_, или значение "– 1", если целевая строка содержит _Разделитель_.
  
## <a name="syntax"></a>Синтаксис

Lookup ("* * *Key* * *", "* * *List* * *" [, "* * *Разделитель* * *"]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _key_ <br/> |Обязательный  <br/> |**String** <br/> |Строка, которую требуется найти.  <br/> |
| _list_ <br/> |Обязательный  <br/> |**String** <br/> | Список, в котором необходимо выполнить поиск.  <br/> |
| _разделитель_ <br/> |Необязательный  <br/> |**String** <br/> | Строка, используемая в качестве разделителя в _списке_. Строка _разделителя_ может иметь длину более одного символа и может содержать многобайтовые символы. Значение по умолчанию — точка с запятой.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="remarks"></a>Примечания

Функция поиска использует поиск без учета регистра. Если список начинается или заканчивается разделителем, считается, что пустая строка существует до или после списка. Последовательные разделители подразумевают пустую строку между. 
  
Все аргументы должны быть строками или выражениями, которые могут быть преобразованы в строки. В противном случае для аргумента, вызывающего ошибку, заменяется пустая строка. 
  
## <a name="example-1"></a>Пример 1

Поиск ("Rat", "Cat; Rat;"; Гоат ")
  
Возвращает 1.
  
## <a name="example-2"></a>Пример 2

Поиск ("", "; Cat; Rat;; Гоат ")
  
Возвращает 0.
  
## <a name="example-3"></a>Пример 3

Поиск ("t", "Cat; Rat;; Гоат "," a ")
  
Возвращает значение 3.
