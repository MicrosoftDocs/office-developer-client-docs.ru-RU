# <a name="lookup-function"></a>Функция LOOKUP

Возвращает индекс с нулем, который указывает расположение ключа подстроки в списке, или возвращает -1, если целевая строка содержит  замещение.  
  
## <a name="syntax"></a>Синтаксис

LOOKUP(" ** *key* ** "," ** *list* ** "[," ** *delimiter* ** "]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _key_ <br/> |Обязательно  <br/> |**Строка** <br/> |Строка, которую вы хотите найти.  <br/> |
| _списке_ <br/> |Обязательно  <br/> |**Строка** <br/> | Список, в котором нужно найти.  <br/> |
| _delimiter_ <br/> |Необязательна  <br/> |**Строка** <br/> | Строка, используемая в качестве делегировки в _списке._ Строка  _с делегировкой_ может иметь длину более одного символа и может включать многобайтные символы. Значение по умолчанию — 1 с зачетом.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="remarks"></a>Примечания

Функция LOOKUP использует поиск без чувствительность к делу. Если список начинается или заканчивается с помощью разбиений, предполагается, что строка null существует до или после списка. Последовательные заметивы подразумевают нулевую строку между ними. 
  
Все аргументы должны быть строками или выражениями, которые можно преобразовать в строки. Если это не так, пустая строка заменяется оскорбить аргументом. 
  
## <a name="example-1"></a>Пример 1

LOOKUP("рат","cat;cat;; ")
  
Возвращает 1.
  
## <a name="example-2"></a>Пример 2

LOOKUP("",";cat;cat;; ")
  
Возвращает 0.
  
## <a name="example-3"></a>Пример 3

LOOKUP("t","cat;рат;; ","a")
  
Возвращает 3.
