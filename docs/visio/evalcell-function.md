# <a name="evalcell-function"></a>Функция EVALCELL

Принимает ссылку на ячейку, которая содержит пользовательскую функцию, а также одну или несколько пар "имя-значение", чтобы передать ее в качестве аргументов (необязательно). Возвращает вычисляемый результат пользовательской функции с учетом указанных аргументов и значений.
  
## <a name="syntax"></a>Синтаксис

EVALCELL(** *cellRef* **,[ ** *arg1Name,arg1* ** ],[ ** *arg2Name,arg2* ** ],...) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _cellRef_ <br/> |Обязательно  <br/> |**Строка** <br/> |Ссылка на ячейку, содержаную настраиваемую функцию. Перекрестные ссылки разрешены.  <br/> |
| _arg1Name_ <br/> |Необязательна  <br/> |**Строка** <br/> |Имя первого аргумента, передаемого пользовательской функции. Пробелы разрешены.  <br/> |
| _arg1_ <br/> |Необязательна  <br/> |**Разные** <br/> |Значение _параметра arg1._  <br/> |
| _arg2Name_ <br/> |Необязательна  <br/> |**Строка** <br/> |Имя второго аргумента, передаемого в настраиваемую функцию. Пробелы разрешены.  <br/> |
| _arg2_ <br/> |Необязательна  <br/> |**Разные** <br/> |Значение _параметра arg2._  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="remarks"></a>Примечания

Вызываемая ячейка не должна указывать каждый аргумент, используемый настраиваемой функцией. 
  
## <a name="example"></a>Пример

В следующем примере показано, как использовать функцию EVALCELL в сочетании с функцией ARG для поиска среднего значения из набора из трех значений. 
  
В ячейке выражения раз поместите следующий код, который определяет пользовательскую функцию: 
  
```vb
User.MiddleValue = IF(ARG("A")>ARG("B"),IF(ARG("B")>ARG("C"),ARG("B"),IF(ARG("A")>ARG("C"),ARG("C"),ARG("A"))),IF(ARG("A")>ARG("C"),ARG("A"),IF(ARG("B")>ARG("C"),ARG("C"),ARG("B"))))
```

В ячейках вызова раз поместите следующий код, который вызывает пользовательскую функцию:
  
```vb
User.Middle1 = EVALCELL(User.MiddleValue,"A",3,"B",9,"C",5) 
User.Middle2 = EVALCELL(User.MiddleValue,"A",12,"B",0,"C",21) 

```
