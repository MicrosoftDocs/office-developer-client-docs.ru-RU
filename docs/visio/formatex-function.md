# <a name="formatex-function"></a>Функция FORMATEX

Возвращает результат выражения, вычисленное в srcUnit как строку отформатированное в соответствии с формату в dstUnit.
  
## <a name="syntax"></a>Синтаксис

FORMATEX (** *выражение* **, "** *Формат* **", [** *srcUnit* **], [** *dstUnit* **], [** *langID* **] [, ** *calID* **]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**Строка** <br/> |Комбинация константы, операторы, функции и ссылки на ячейки таблицы свойств фигуры, которая приводит к значение.  <br/> |
| _format_ <br/> |Обязательный  <br/> |**Строка** <br/> |Формат рисунка, который используется для форматирования строки. Дополнительные сведения о формате изображения содержатся в разделе [О формате изображения](about-format-pictures.md).  <br/> |
| _srcUnit_ <br/> |Optional  <br/> |**Строка** <br/> | Единицы, используемые для оценки выражения (в, cm и т. д.).  <br/> |
| _dstUnit_ <br/> |Optional  <br/> |**Строка** <br/> |Единицы измерения для результата выражения (в, cm и т. д.).  <br/> |
| _langID_ <br/> |Optional  <br/> |**Число** <br/> |Язык, используемый при форматировании изображений даты и времени системы Microsoft Office.  <br/> |
| _calID_ <br/> |Optional  <br/> |**Число** <br/> |Календарь, используемый при форматировании изображений даты и времени системы Microsoft Office.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Строка
  
## <a name="remarks"></a>Замечания

Тип выражения и тип которых указан формат рисунка управляют поведением Возвращенная строка. Формат значения соответствуют типу выражения.
  
Аргумент srcUnit используется для масштабирования результатов нетипизированного выражения, такие как 3 + 4. Если результат выражения явного типа, например 3 ft + 8 ft, то srcUnit игнорируется.
  
Аргумент dstUnit используется для указания единицы измерения, используемые для форматированного результата. Если dstUnit не указан, будут использоваться единиц измерения для результата выражения.
  
Возвращает ошибку, если результат выражения и типа в формате разного рода при наличии синтаксических ошибок в формате или не распознает единицы передается как srcUnit или dstUnit.
  
## <a name="example"></a>Пример

FORMATEX (5.5, "0,00 u», «cm», «ft") 
  
Возвращает 0,18 футов.
