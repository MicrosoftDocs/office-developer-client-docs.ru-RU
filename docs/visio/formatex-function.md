# <a name="formatex-function"></a>Функция FORMATEX

Возвращает результат выражения, оцениваемого в srcUnit как строка, отформатированная в соответствии с форматом, выраженным в dstUnit.
  
## <a name="syntax"></a>Синтаксис

FORMATEX(** *выражение* **," ** *формат* ** ",[** *srcUnit* ** ],[* *dstUnit* ** ],[* *langID* ** ][, ** *calID* ** ]] 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**String** <br/> |Сочетание констант, операторов, функций и ссылок на ячейки ShapeSheet, что приводит к значению.  <br/> |
| _format_ <br/> |Обязательный  <br/> |**String** <br/> |Изображение формата, используемая для формата строки. Дополнительные сведения о форматных изображениях см. в [рисунках формата.](about-format-pictures.md)  <br/> |
| _srcUnit_ <br/> |Необязательный  <br/> |**String** <br/> | Единицы, используемые для оценки выражений (в, см и так далее).  <br/> |
| _dstUnit_ <br/> |Необязательный  <br/> |**String** <br/> |Единицы, которые необходимо использовать для результатов выражения (в, см и так далее).  <br/> |
| _langID_ <br/> |Необязательный  <br/> |**Number** <br/> |Язык, используемый при форматировании Microsoft Office системных изображений даты и времени.  <br/> |
| _calID_ <br/> |Необязательный  <br/> |**Number** <br/> |Календарь, используемый при форматировании Microsoft Office системных изображений даты и времени.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Тип выражения и тип, указанный в изображении формата, регулируют поведение возвращенной строки. Формат должен быть подходящим для типа выражения.
  
Аргумент srcUnit используется для масштабирования результатов нетипиченного выражения, например 3 + 4. Если результат выражения имеет явный тип, например 3 фута + 8 футов, то srcUnit игнорируется.
  
Аргумент dstUnit используется для указания единиц, используемых для форматного результата. Если dstUnit не указан, используются единицы для результатов выражения.
  
Возвращает ошибку, если результат выражения и тип, ожидаемый в формате, отличаются от других, если имеются синтаксисные ошибки в формате или если они не распознают единицы, переданные как srcUnit или dstUnit.
  
## <a name="example"></a>Пример

FORMATEX (5.5, "0.00 u", "cm", "ft") 
  
Возвращает 0,18 футов.
