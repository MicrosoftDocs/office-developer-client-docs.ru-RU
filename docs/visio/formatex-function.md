# <a name="formatex-function"></a>Функция FORMATEX

Возвращает результат выражения, оцениваемого в srcUnit как строка, отформатированная в соответствии с форматом, выраженным в dstUnit.
  
## <a name="syntax"></a>Синтаксис

FORMATEX(** *expression* **," ** *format* ** ",[ ** *srcUnit* ** ],[ ** *dstUnit* ** ],[ ** *langID* ** ][, ** *calID* ** ]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательно  <br/> |**Строка** <br/> |Сочетание констант, операторов, функций и ссылок на ячейки ShapeSheet, которое приводит к значению.  <br/> |
| _format_ <br/> |Обязательно  <br/> |**Строка** <br/> |Изображение формата, используемая для формата строки. Дополнительные сведения о формате изображений см. в [сведениях о формате изображений.](about-format-pictures.md)  <br/> |
| _srcUnit_ <br/> |Необязательна  <br/> |**Строка** <br/> | Единицы, используемые для оценки выражения (в, cm и так далее).  <br/> |
| _dstUnit_ <br/> |Необязательна  <br/> |**Строка** <br/> |Единицы, которые будут использовать для результата выражения (в, cm и так далее).  <br/> |
| _langID_ <br/> |Необязательна  <br/> |**Number** <br/> |Язык, используемый при Microsoft Office системных рисунков даты и времени.  <br/> |
| _calID_ <br/> |Необязательна  <br/> |**Number** <br/> |Календарь, используемый при Microsoft Office системных рисунков даты и времени.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

Тип выражения и тип, указанные в изображении формата, управляют поведением возвращаемой строки. Формат должен быть соответствующим типу выражения.
  
Аргумент srcUnit используется для масштабирования результатов нетипизированных выражений, например 3 + 4. Если результат выражения имеет явный тип, например 3 ft + 8 ft, srcUnit игнорируется.
  
Аргумент dstUnit используется для указания единиц, используемых для отформатированный результат. Если dstUnit не указан, используются единицы для результата выражения.
  
Возвращает ошибку, если результат выражения и тип, ожидаемый в формате, имеют другой вид, существуют ошибки синтаксиса в формате или если он не распознает единицы, переданные как srcUnit или dstUnit.
  
## <a name="example"></a>Пример

FORMATEX(5.5, "0.00 u", "cm", "ft") 
  
Возвращает 0,18 метров.
