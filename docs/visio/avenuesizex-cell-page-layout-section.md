# <a name="avenuesizex-cell-page-layout-section"></a>Ячейка AvenueSizeX (раздел "Макет страницы")

Определяет горизонтальную расстояние между фигурами на странице документа при расположении фигур с помощью диалогового окна " **Настройка макета** " (на вкладке **Конструктор** в группе **Макет** нажмите кнопку **Изменить расположение**и нажмите кнопку ** более Параметры расположения **).
  
## <a name="remarks"></a>Замечания

Также можно задать это значение в диалоговом окне **Макет и маршрутизация интервалы** (на вкладку **Конструктор** , щелкните стрелку в группе **Параметры страницы** , перейдите на вкладку **Макет и маршрутизации** и нажмите кнопку **интервала**).
  
Динамические сетки используется в ячейке AvenueSizeX Если доступен только один фигуры для расчета интервал по горизонтали. Использование динамических сетки на вкладке **Вид** в группе **Вспомогательные элементы** выберите **Динамических сетки**.
  
Чтобы получить ссылку на ячейку AvenueSizeX по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | AvenueSizeY  <br/> |
   
Для получения ссылки на ячейки AvenueSizeX по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowPageLayout** <br/> |
| Индекс ячейки:  <br/> |**visPLOAvenueSizeX** <br/> |
