# <a name="distancefromground-cell-3-d-rotation-properties"></a>Ячейка DistanceFromGround (раздел "Свойства поворота объемной фигуры")

Определяет расстояние, на которое объект вызывается от нуля в точках при повороте в 3-D.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку **DistanceFromGround** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |DistanceFromGround  <br/> |
   
Чтобы получить ссылку на ячейку **DistanceFromGround** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRow3DRotationProperties** <br/> |
|Индекс ячейки:  <br/> |**visDistanceFromGround** <br/> |
