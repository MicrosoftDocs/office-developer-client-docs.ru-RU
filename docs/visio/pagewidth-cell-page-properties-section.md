# <a name="pagewidth-cell-page-properties-section"></a>PageWidth Cell (Page Properties Section)

Определяет ширину печатной страницы в единицах документа.
  
## <a name="remarks"></a>Замечания

Вы также можете задать ширину страницы на вкладке " **Размер страницы** " диалогового окна **Параметры страницы** (на вкладке **Макет** щелкните стрелку **настройки страницы** ) или вручную измените размер страницы с помощью мыши. Для этого перетащите край страницы, удерживая нажатой клавишу CTRL. 
  
Чтобы получить ссылку на ячейку PageWidth по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |PageWidth  <br/> |
   
Чтобы получить ссылку на ячейку PageWidth по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровпаже** <br/> |
|Индекс ячейки:  <br/> |**Виспажевидс** <br/> |
