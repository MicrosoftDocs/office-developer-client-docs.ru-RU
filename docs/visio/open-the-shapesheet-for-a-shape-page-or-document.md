# <a name="open-the-shapesheet-for-a-shape-page-or-document"></a>Откройте таблицу ShapeSheet для формы, страницы или документа

Окно ShapeSheet скрыто по умолчанию в Microsoft Visio. Чтобы получить доступ к ShapeSheet, необходимо выполнить Visio в режиме разработчика (отобразить вкладку разработчика).
  
### <a name="to-run-visio-in-developer-mode"></a>Запуск Visio в режиме разработчика

1. Щелкните **вкладку File** и нажмите кнопку **Параметры**.
    
2. В **диалоговом окне Visio Параметры** нажмите **кнопку Advanced,** а затем в **статье General** выберите **Выполнить в режиме разработчика.**
    
### <a name="to-view-the-shapesheet-for-a-shape"></a>Просмотр таблицы ShapeSheet для формы

- Щелкните правой кнопкой мыши фигуру на холсте рисования и нажмите **кнопку Показать ShapeSheet**.
    
- Выберите фигуру на рисунке, а затем на вкладке **Developer** в группе **Shape Design** нажмите **кнопку Показать ShapeSheet** и нажмите кнопку **Shape**.
    
### <a name="to-view-the-sheet-for-a-page"></a>Просмотр листа для страницы

- На **вкладке Разработчик** в группе **Shape Design** щелкните **Показать таблицу ShapeSheet** и нажмите **страницу**.
    
### <a name="to-view-the-sheet-for-a-document"></a>Просмотр листа для документа

- На **вкладке Разработчик** в группе **Shape Design** щелкните **Показать таблицу ShapeSheet** и нажмите кнопку **Документ**.
