# <a name="open-the-shapesheet-for-a-shape-page-or-document"></a>Открытие таблицы фигур для фигуры, страницы или документа

Окно ShapeSheet по умолчанию скрыто в Microsoft Visio. Чтобы получить доступ к таблице shapeSheet, необходимо запустить Visio в режиме разработчика (отобразить вкладку разработчика).
  
### <a name="to-run-visio-in-developer-mode"></a>Запуск Visio в режиме разработчика

1. Щелкните **вкладку "Файл"** и выберите **"Параметры".**
    
2. В **диалоговом окне "Параметры Visio"** нажмите кнопку **"Дополнительные",** а затем в области "Общие" выберите "Выполнить **в режиме разработчика".** 
    
### <a name="to-view-the-shapesheet-for-a-shape"></a>Просмотр таблицы фигур для фигуры

- Щелкните правой кнопкой мыши фигуру на полотне и выберите **"Показать таблицу фигур".**
    
- Выберите фигуру на полотне, а  затем на  вкладке "Разработчик" в группе "Конструктор фигур" щелкните "Показать **таблицу фигур"** и выберите **"Фигура".**
    
### <a name="to-view-the-sheet-for-a-page"></a>Просмотр листа для страницы

- На **вкладке "Разработчик"** в группе **"Конструктор фигур"** щелкните **"Показать таблицу фигур"** и выберите **"Страница".**
    
### <a name="to-view-the-sheet-for-a-document"></a>Просмотр листа для документа

- На **вкладке "Разработчик"** в группе **"Конструктор** фигур" щелкните **"Показать таблицу фигур"** и выберите **"Документ".**
