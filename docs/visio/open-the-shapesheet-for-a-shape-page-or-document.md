# <a name="open-the-shapesheet-for-a-shape-page-or-document"></a>Открытие таблицы свойств фигуры, страницы или документа

По умолчанию окно таблицы свойств фигуры скрыто в Microsoft Visio. Чтобы получить доступ к таблице свойств фигуры, необходимо запустить Visio в режиме разработчика (откройте вкладку "Разработчик").
  
### <a name="to-run-visio-in-developer-mode"></a>Запуск Visio в режиме разработчика

1. Перейдите на вкладку **файл** и нажмите кнопку **Параметры**.
    
2. В диалоговом окне **Параметры Visio** нажмите кнопку **Дополнительно**, а затем в разделе **Общие**выберите команду **запустить в режиме разработчика**.
    
### <a name="to-view-the-shapesheet-for-a-shape"></a>Просмотр таблицы свойств фигуры

- Щелкните фигуру на полотне правой кнопкой мыши и выберите пункт **Показать таблицу свойств фигуры**.
    
- Выберите фигуру на полотне, а затем на вкладке **разработчик** в группе **Макет фигуры** нажмите кнопку **Показать таблицу свойств**фигуры, а затем выберите пункт **фигура**.
    
### <a name="to-view-the-sheet-for-a-page"></a>Просмотр листа для страницы

- На вкладке **разработчик** в группе **Макет фигуры** нажмите кнопку **Показать таблицу свойств**фигуры, а затем выберите пункт **страница**.
    
### <a name="to-view-the-sheet-for-a-document"></a>Просмотр листа для документа

- На вкладке **разработчик** в группе **Макет фигуры** нажмите кнопку **Показать таблицу свойств**фигуры, а затем выберите пункт **документ**.
