# <a name="open-the-shapesheet-for-a-shape-page-or-document"></a>Откройте таблицы свойств фигуры для фигуры, страницы или документа

Окно таблицы свойств фигуры скрыт по умолчанию в Microsoft Visio. Для доступа к таблице свойств фигуры, необходимо запустить Visio в режиме разработчика (Отображение вкладки "Разработчик").
  
### <a name="to-run-visio-in-developer-mode"></a>Запуск Visio в режиме разработчика

1. Перейдите на вкладку **файл** и нажмите кнопку **Параметры**.
    
2. В диалоговом окне **Параметры Visio** нажмите кнопку **Дополнительно**и затем на вкладке **Общие**выберите **Запуск в режиме разработчика**.
    
### <a name="to-view-the-shapesheet-for-a-shape"></a>Чтобы просмотреть таблицы свойств фигуры для фигуры

- Щелкните правой кнопкой мыши фигуру на полотне и нажмите кнопку **Показать таблицу свойств фигуры**.
    
- Выберите фигуру на полотне и нажмите кнопку **Показать таблицу свойств фигуры**на вкладке **Разработчик** в группе **Разработки фигуры** и нажмите кнопку **фигуры**.
    
### <a name="to-view-the-sheet-for-a-page"></a>Чтобы просмотреть таблицы для страницы

- На вкладке **Разработчик** в группе **Создать фигуру** нажмите кнопку **Показать таблицу свойств фигуры**и нажмите кнопку **страницы**.
    
### <a name="to-view-the-sheet-for-a-document"></a>Чтобы просмотреть окно документа

- На вкладке **Разработчик** в группе **Создать фигуру** нажмите кнопку **Показать таблицу свойств фигуры**и нажмите кнопку **документа**.
