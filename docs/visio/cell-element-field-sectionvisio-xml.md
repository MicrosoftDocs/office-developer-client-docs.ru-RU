# <a name="cell-element-field-section-visio-xml"></a>Элемент ячейки (раздел поля) ('Visio XML»)

Отображает функций и формул вставлен в тексте фигуры с помощью диалогового окна поля.
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|**Тип элемента** <br/> |[Cell_Type](cell_type-complextypevisio-xml.md) <br/> |
|**Пространство имен** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Файл схемы** <br/> |VisioSchema15.xsd  <br/> |
|**Части документа** <br/> |главные # .xml, страницы # .xml  <br/> |
   
## <a name="definition"></a>Определение

```XML
< xs:element name="Cell" type="Cell_Type" minOccurs="0" maxOccurs="unbounded" >
</xs:element >
```

|ObjectKind  <br/> |Указывает тип текстового поля.  <br/> |[Ячейка ObjectKind (раздел "Текстовые поля")](objectkind-cell-text-fields-section.md) <br/> |
