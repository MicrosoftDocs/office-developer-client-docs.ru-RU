# <a name="tanh-function"></a>Функция TANH

Возвращает гиперболический тангент угла. 
  
## <a name="syntax"></a>Синтаксис

TANH(** *угол* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _угол_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Угол, под которым можно получить гипболический тангент.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
  
## <a name="remarks"></a>Примечания

Функция TANH использует радианы, если не указать другую единицу измерения в *угол.* 
  
## <a name="example"></a>Пример

TANH(-45) 
  
Возвращает -0.6558.
