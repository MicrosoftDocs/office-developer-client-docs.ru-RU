# <a name="glowcolortrans-cell-additional-effect-properties-section"></a>Ячейка GlowColorTrans (раздел "Дополнительные свойства эффекта")

Определяет уровень прозрачности цвета, используемого для штрих свечения фигуры в процентном выражении. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **GlowColorTrans** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | GlowColorTrans  <br/> |
   
Для получения ссылки на ячейки **GlowColorTrans** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowOtherEffectProperties** <br/> |
| Индекс ячейки:  <br/> |**visGlowColorTrans** <br/> |
