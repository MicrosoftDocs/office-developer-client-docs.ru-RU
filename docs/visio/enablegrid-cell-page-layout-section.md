# <a name="enablegrid-cell-page-layout-section"></a>EnableGrid Cell (Page Layout Section)

Определяет, формирует ли приложение фигуры на основе внутренней невидимой сетки страницы  при настройке макета в диалоговом окне "Настройка макета". (На **вкладке "Конструктор"** в группе **"Макет"** щелкните "Страница повторного макета" **и** выберите "Дополнительные **параметры макета".**
  
|**Значение**|**Описание**|
|:-----|:-----|
|TRUE  <br/> |Используйте внутреннюю сетку страниц.  <br/> |
|FALSE  <br/> |Не используйте внутреннюю сетку страниц.  <br/> |
   
## <a name="remarks"></a>Примечания

Сетка страницы создается с помощью пространства  между фигурами и  значений среднего размера фигур в диалоговом окне "Макет" и "Интервал маршрутов".  (На **вкладке "Конструктор"** щелкните **стрелку** "Настройка страницы", выберите "Макет" и "Маршрут" **и** щелкните **"Интервал".)** 
  
Когда вы включаете эту функцию, приложение выравнивает каждую центрную точку фигуры с центром блока во внутренней сетке страниц. 
  
Чтобы получить ссылку на ячейку EnableGrid по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |EnableGrid  <br/> |
   
Чтобы получить ссылку на ячейку EnableGrid по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPageLayout** <br/> |
|Индекс ячейки:  <br/> |**visPLOEnableGrid** <br/> |
