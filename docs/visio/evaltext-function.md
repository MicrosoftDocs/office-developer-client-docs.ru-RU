# <a name="evaltext-function"></a>Функция EVALTEXT

Оценивает текст в  _shapename_ как формулу и возвращает результат. 
  
## <a name="syntax"></a>Синтаксис

EVALTEXT(** *shapename!theText* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _shapename!theText_ <br/> |Обязательный  <br/> |**String** <br/> |Ячейка, которая запускается при смене текстовой композиции связанной фигуры.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

String
  
## <a name="remarks"></a>Примечания

 _shapename_ можно использовать для ссылки на текст фигуры, помимо текущей. 
  
Если текста нет, результат нулевой. Если текст не удается оценить, функция возвращает ошибку.
  
## <a name="example"></a>Пример

EVALTEXT(Line.2!theText) 
  
Оценивает текст, содержащийся в строке Line.2. Например, если Line.2 содержит "4 ft + 0.5 ft", возвращает значение 4.5 ft.
