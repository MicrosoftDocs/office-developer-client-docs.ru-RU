# <a name="usegroupgradient-cell-gradient-properties-section"></a>Ячейка UseGroupGradient (градиента Properties Section)

Определяет, принимает ли градиент фигуры при фигуры сгруппированные вместе с другими фигурами типа Boolean. Значение ячейки **UseGroupGradient** влияет только заливки фигуры. 
  
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **UseGroupGradient** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | UseGroupGradient  <br/> |
   
Для получения ссылки на ячейки **UseGroupGradient** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowGradientProperties** <br/> |
| Индекс ячейки:  <br/> |** visUseGroupGradient ** <br/> |
