# <a name="usegroupgradient-cell-gradient-properties-section"></a>UseGroupGradient Cell (Gradient Properties Section)

Определяет, принимает ли фигура градиент, если фигура сгруппирована вместе с другими фигурами в качестве boolean. Значение ячейки **UseGroupGradient** влияет только на заливку фигуры. 
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку **UseGroupGradient** по имени из другой формулы, по значению атрибута **N** элемента **Cell** или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | UseGroupGradient  <br/> |
   
Чтобы получить ссылку на ячейку **UseGroupGradient** по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowGradientProperties** <br/> |
| Индекс ячейки:  <br/> |**visUseGroupGradient ** <br/> |
