# <a name="replacelockshapedata-cell-change-shape-behavior-section"></a>Ячейка ReplaceLockShapeData (изменение фигуры поведение раздел)

Указывает ли значений указанного ячеек в основной фигуры перезаписи значений (в том числе локального значения) во время операции замены фигуры в процессе замены фигуры. **ReplaceLockShapeData** определяет, будет ли данные фигуры основной фигуры перезаписывает все данные фигуры фигура. 
  
|**Значение**|**Описание**|
|:-----|:-----|
|1 (TRUE)  <br/> |Все строки и значения в разделе **Данных фигуры** основной фигуры, копируются на замещающий фигуру и все локальные значения из старого фигура не учитываются.  <br/> |
|0 (FALSE)  <br/> |Фигуры замещения копируются строки и значения в разделе **Данных фигуры** основной фигуры. Все строки в разделе **Данных фигуры** старый фигуры на локальном значения переносятся фигуры замещения.  <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки **ReplaceLockShapeData** по имени из другой формулы, по значению атрибута **N** элемент **ячейки** и программы, с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | ReplaceLockShapeData  <br/> |
   
Для получения ссылки на ячейки **ReplaceLockShapeData** по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowReplaceBehaviors** <br/> |
| Индекс ячейки:  <br/> |**visReplaceLockShapeData** <br/> |
