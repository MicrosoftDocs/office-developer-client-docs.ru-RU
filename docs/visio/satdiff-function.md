# <a name="satdiff-function"></a>Функция SATDIFF

Возвращает разницу в насыщенности между _color1_ и _color2._
  
## <a name="syntax"></a>Синтаксис

SATDIFF(** *color1* **, ** *color2* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Индекс Visio microsoft или значение RGB первого цвета.  <br/> |
| _color2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Индекс Visio microsoft или RGB второго цвета.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Числовой
