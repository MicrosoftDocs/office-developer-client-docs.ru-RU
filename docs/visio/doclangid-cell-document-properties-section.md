# <a name="doclangid-cell-document-properties-section"></a>DocLangID Cell (Document Properties Section)

Указывает язык по умолчанию для документа.
  
## <a name="remarks"></a>Примечания

Язык документа основан на языке, определенном в диалоговом окне " **языковые параметры Microsoft Office** ". (Нажмите кнопку **Пуск**, **выберите все программы**, **Microsoft Office**, **средства Microsoft Office**, а затем выберите языковые **Параметры Microsoft Office**.)
  
Если вы хотите изменить язык документа, не изменяя эту ячейку напрямую, то лучше всего изменить это значение в диалоговом окне **Свойства** (перейдите на вкладку **файл** , щелкните **сведения**, выберите **Свойства**, а затем нажмите кнопку Дополнительно. ** Свойства**) или программным путем с помощью свойства **Language** объекта **Document** .

Чтобы получить ссылку на ячейку DocLangID по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |DocLangID  <br/> |
   
Чтобы получить ссылку на ячейку DocLangID по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровдок** <br/> |
|Индекс ячейки:  <br/> |**Висдоклангид** <br/> |
