# <a name="shapefixedcode-cell-shape-layout-section"></a>ShapeFixedCode Cell (Shape Layout Section)

Задает поведение при размещении размещаемой фигуры.
  
|**Значение**|**Режим выбора**|**Константа автоматизации**|
|:-----|:-----|:-----|
|&amp;H1  <br/> |Не перемещайте эту фигуру, когда фигуры размещаются с помощью диалогового окна " **Настройка макета** ".  <br/> |**Висслофикседплацемент** <br/> |
|&amp;H2  <br/> |Не перемещайте эту фигуру и не допускайте фигуры, которые плов помещаются поверх нее.  <br/> |**Висслофикседплов** <br/> |
|&amp;H4  <br/> |Не перемещайте эту фигуру и не разрешайте фигуры, которые плов помещаются поверх нее.  <br/> |**Висслофикседпермеаблеплов** <br/> |
|&amp;H20 (32)  <br/> |Игнорировать расположения точек подключений при маршрутизации.  <br/> |**Висслофикседконнптсигноре** <br/> |
|&amp;H40 (64)  <br/> |Разрешите маршрутизацию только сторонам с точками подключения.  <br/> |**Висслофикседконнптсонли** <br/> |
|&amp;H80 (128)  <br/> |Не приклейте на периметр этой фигуры. ПриКлейте к рамке выравнивания фигуры?  <br/> |**Висслофикседнофолдтошапе** <br/> |
   
## <a name="remarks"></a>Примечания

Вы также можете задать значение этой ячейки на вкладке " **Размещение** " в диалоговом окне **поведение** (с выбранной фигурой на вкладке " [разработчик](run-in-developer-mode-display-the-developer-tab.md) ", в группе " **Макет фигуры** " щелкните **поведение**, а затем перейдите на вкладку **Размещение** . ). 
  
Для этой ячейки можно задать любое сочетание этих значений. Например, можно ввести значение 3&amp;(H3), которое исключает перемещение при размещении фигур с помощью диалогового окна " **Настройка макета** " (на вкладке **конструктор** в группе **Макет** выберите пункт **изменить макет страницы**, а затем нажмите кнопку ** Дополнительные параметры макета** ), а также при размещении других размещаемых фигур или возле фигуры. 
  
В версиях, предшествующих Visio 2000, это поведение задается с помощью ячейки Обжинтеракт в разделе Разное. 
  
Чтобы получить ссылку на ячейку ShapeFixedCode по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |ShapeFixedCode  <br/> |
   
Чтобы получить ссылку на ячейку ShapeFixedCode по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровшапелайаут** <br/> |
|Индекс ячейки:  <br/> |**Висслофикседкоде** <br/> |
