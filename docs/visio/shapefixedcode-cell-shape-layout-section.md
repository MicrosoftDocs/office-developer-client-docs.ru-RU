# <a name="shapefixedcode-cell-shape-layout-section"></a>ShapeFixedCode Cell (Shape Layout Section)

Определяет поведение размещения для размещения фигуры.
  
|**Значение**|**Режим выделения**|**Константа автоматизации**|
|:-----|:-----|:-----|
|&amp;H1  <br/> |Не перемещайте эту фигуру, если фигуры выложены с помощью диалоговых окна "Настройка  макета".  <br/> |**visSLOFixedPlacement** <br/> |
|&amp;H2  <br/> |Не перемещайте эту фигуру и не разрешать размещать на ней фигуры, которые помещаются поверх него.  <br/> |**visSLOFixedPlow** <br/> |
|&amp;H4  <br/> |Не перемещайте эту фигуру и не разрешать помещать поверх нее фигуры, которые помещаются на plow.  <br/> |**visSLOFixedPermeablePlow** <br/> |
|&amp;H20 (32)  <br/> |Игнорировать расположения точек подключения при маршруте.  <br/> |**visSLOFixedConnPtsIgnore** <br/> |
|&amp;H40 (64)  <br/> |Разрешить только маршрутику на стороны с точками подключения.  <br/> |**visSLOFixedConnPtsOnly** <br/> |
|&amp;H80 (128)  <br/> |Не привяжйтесь к периметру этой фигуры. Вместо этого привяжйте к поле выравнивания фигуры.  <br/> |**visSLOFixedNoFoldToShape** <br/> |
   
## <a name="remarks"></a>Примечания

Вы также можете установить значение этой  ячейки  на вкладке "Размещение" в диалоговом окне  "Поведение" (с выбранной фигурой, на вкладке "Разработчик", в группе "Проектирование фигуры" щелкните "Поведение" и выберите вкладку **"Размещение").** [](run-in-developer-mode-display-the-developer-tab.md) 
  
Для этой ячейки можно настроить любое сочетание этих значений. Например, можно ввести значение 3 ( H3), которое исключает перемещение при размещении фигур с помощью диалоговых окна "Настройка макета" (на вкладке "Конструктор" в группе "Макет" щелкните "Страница повторного макета" и нажмите кнопку "Дополнительные параметры макета"), а также когда другие помещаемые фигуры размещаются на фигуре или рядом с &amp; фигурой.      
  
В версиях, более ранних, чем Visio 2000, такое поведение задано с помощью ячейки ObjInteract в разделе "Прочие". 
  
Чтобы получить ссылку на ячейку ShapeFixedCode по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |ShapeFixedCode  <br/> |
   
Чтобы получить ссылку на ячейку ShapeFixedCode по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowShapeLayout** <br/> |
|Индекс ячейки:  <br/> |**visSLOFixedCode** <br/> |
