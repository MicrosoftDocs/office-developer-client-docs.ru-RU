# <a name="shapefixedcode-cell-shape-layout-section"></a>Ячейка ShapeFixedCode (раздел "Макет фигуры")

Указывает поведение размещаемой фигуры.
  
|**Значение**|**Выбор режима**|**Константа автоматизации**|
|:-----|:-----|:-----|
|&amp;H1  <br/> |Не перемещать этой фигуры, когда расположения фигур с помощью диалогового окна " **Настройка макета** ".  <br/> |**visSLOFixedPlacement** <br/> |
|&amp;H2  <br/> |Не перемещать этой фигуры и не разрешать фигуры, которые сдвигать должна размещаться на его основе.  <br/> |**visSLOFixedPlow** <br/> |
|&amp;H4  <br/> |Не переместить фигуру и разрешить фигуры, которые сдвигать должна размещаться на его основе.  <br/> |**visSLOFixedPermeablePlow** <br/> |
|&amp;H20 (32)  <br/> |Когда поступает в игнорировать местоположения точек подключения.  <br/> |**visSLOFixedConnPtsIgnore** <br/> |
|&amp;H40 (64)  <br/> |Разрешать только маршрутизации и точек подключения.  <br/> |**visSLOFixedConnPtsOnly** <br/> |
|&amp;H80 (128)  <br/> |Не связывания для периметра этой фигуры. Вместо этого связывания фигуры выравнивание.  <br/> |**visSLOFixedNoFoldToShape** <br/> |
   
## <a name="remarks"></a>Замечания

Значение ячейки также можно настроить на вкладке **Размещение** в диалоговом окне **поведение** (с фигуры, выбранные на вкладке [Разработчик](run-in-developer-mode-display-the-developer-tab.md) в группе **Создать фигуру** нажмите кнопку **поведение**и затем перейдите на вкладку **Размещение** ). 
  
Можно задать любое сочетание следующих значений для этой ячейки. Например, можно ввести значение 3 (&amp;H3), которая позволяет устранить перемещения при расположении фигур с помощью диалогового окна " **Настройка макета** " (на вкладке **Конструктор** в группе **Макет** выберите пункт **Изменить расположение**и нажмите кнопку ** Дополнительные параметры макета** ) и помещения других размещаемые фигуры на или близок к фигуре. 
  
Более ранних чем Visio 2000 в версиях задайте это поведение, с помощью ObjInteract ячеек в разделе Разное. 
  
Чтобы получить ссылку на ячейку ShapeFixedCode по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |ShapeFixedCode  <br/> |
   
Для получения ссылки на ячейки ShapeFixedCode по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowShapeLayout** <br/> |
|Индекс ячейки:  <br/> |**visSLOFixedCode** <br/> |
