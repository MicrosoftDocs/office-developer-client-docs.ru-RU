# <a name="color-cell-reviewer-section"></a>Ячейка Color (раздел "Рецензент")

Значение RGB, представляющее цвет, назначенный разметки рецензент документа. 
  
## <a name="remarks"></a>Замечания

Цвета назначаются рецензентов в следующей последовательности: красный, синий, зеленый, фиолетовые, оранжевый, turquoise, серый. Эти цвета переключения еще раз для любого остальных рецензентов. 
  
Комментарии, введенные на странице исходного документа всегда желтый цвет, независимо от того, цвет, назначенный проверяющему в ячейке цвет. 
  
Для получения ссылки на ячейки цвет по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Reviewer.Color [ *i* ] где *i* = < 1 > 2, 3...  <br/> |
   
Для получения ссылки на ячейки цвет по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionReviewer** <br/> |
| Индекс строки:  <br/> |**visRowReviewer** +  *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visReviewerColor** <br/> |
