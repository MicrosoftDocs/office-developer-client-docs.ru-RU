# <a name="color-cell-reviewer-section"></a>Color Cell (Reviewer Section)

Значение RGB, представляющее цвет, назначенный разметке рецензента документов. 
  
## <a name="remarks"></a>Комментарии

Цвета назначаются проверяющим в следующей последовательности: Red, Blue, Green, фиолетовый, оранжевый, бирюзовый, серый. Эти цвета циклически передаются для всех оставшихся рецензентов. 
  
Комментарии, введенные на исходной странице документа, всегда окрашены желтым цветом, независимо от цвета, назначенного рецензенту в ячейке Color (цвет). 
  
Чтобы получить ссылку на ячейку Color по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Review. Color [ *i* ], где *i* = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку Color по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**Виссектионревиевер** <br/> |
| Индекс строки:  <br/> |**висровревиевер** +  *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**Висревиеверколор** <br/> |
