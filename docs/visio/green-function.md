# <a name="green-function"></a>Функция GREEN

Возвращает значение зеленого компонента цвета.
  
## <a name="syntax"></a>Синтаксис

Зеленый (** *выражение* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _expression_ <br/> |Обязательный  <br/> |**Разные** <br/> |Индекс цвета в таблице цветов документа, выражение, которое разрешается в настраиваемые цвета (например, RGB или HSL) или ссылку на ячейку, содержащую цвет индекса или цветовой результатов.  <br/> |
   
### <a name="return-value"></a>������������ ��������

Целое число
  
## <a name="remarks"></a>Замечания

Возвращаемое значение — это число в диапазоне от 0 до 255 включительно, или ссылка на ячейку, которая разрешается в индекс. Если *выражение* является недопустимым, он возвращает 0 (черный цвет). 
  
## <a name="example-1"></a>Пример 1

Зеленый (Sheet.4! FillForegnd)
  
Возвращает значение зеленого компонента цвет заливки Sheet. 4.
  
## <a name="example-2"></a>Пример 2

GREEN(11)
  
Возвращает 128, если в документе используется по умолчанию Visio цветовой палитры, в котором темно-желтый цвет по индексу 11.
  
## <a name="example-3"></a>Пример 3

ЗЕЛЕНЫЙ (RGB (10, 20, 30))
  
Возвращает 20.
