# <a name="diry--b-cell-connection-points-section"></a>Ячейка DirY / B (раздел "Точки соединения")

Определяет, *y* -компонент для вектор Обязательное выравнивание совпадающих точки подключения. Он также используется для сориентироваться вложенные ветвь динамических соединителя. В этой ячейке принимает число с плавающей запятой. 
  
## <a name="remarks"></a>Замечания

Чтобы получить ссылку на DirY / B ячейки по имени из другой формулы или из файла с помощью свойства **CellsU** , использования: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Connections.DirY [ *i* ] где *i* = < 1 > 2, 3...  <br/> |
   
Чтобы получить ссылку на DirY / B ячейки по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionConnectionPts** <br/> |
|Индекс строки:  <br/> |**visRowConnectionPts** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visCnnctDirY** (не только строк)           **visCnnctB** (расширенные строк)  <br/> |
   
Сведения о не расширены и расширенного строк, в разделе точек подключения строки.
