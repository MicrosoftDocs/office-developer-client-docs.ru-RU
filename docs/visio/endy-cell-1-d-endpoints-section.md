# <a name="endy-cell-1-d-endpoints-section"></a>EndY Cell (1-D Endpoints Section)

Представляет  *Y-координату*  конечной точки 1-D фигуры относительно начала родительской фигуры. 
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку EndY по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | EndY  <br/> |
   
Чтобы получить ссылку на ячейку EndY по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowXForm1D** <br/> |
| Индекс ячейки:  <br/> |**vis1DEndY** <br/> |
