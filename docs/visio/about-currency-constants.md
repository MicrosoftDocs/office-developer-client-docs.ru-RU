# <a name="about-currency-constants"></a>О константах валюты

Для формата числа в виде валюты можно использовать функцию CY и передавать необязательные константы, чтобы указать валюту страны или региона для использования. Константы валюты могут быть указаны в качестве номера ИД, соответствующего стране или региону, или в качестве строки (заключенной в кавычках), соответствующей трех символьной аббревиатуре ISO 4217.
  
Если вы показываете символы валют для нелокальных валют и система не знает символ для данной валюты, отображается символ доллара ($).
