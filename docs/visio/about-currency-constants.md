# <a name="about-currency-constants"></a>Сведения о константах денежных единиц

Чтобы отформатировать число как денежное, можно использовать функцию CY и передать необязательную константу, чтобы указать используемую валюту страны или региона. Константы валюты можно указать в качестве ИДЕНТИФИКАЦИОНного номера, соответствующего стране или региону, или в виде строки (заключенной в кавычки), которая соответствует аббревиатуре ISO 4217 3-символа.
  
Если вы показываете символы денежных единиц для нелокальных валют, и система не знает символ для данной валюты, отображается символ доллара ($).
