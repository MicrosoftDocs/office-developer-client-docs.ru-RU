# <a name="about-currency-constants"></a>О константах валюты

Чтобы форматизировать номер в качестве валюты, можно использовать функцию CY и передать необязательный констант, чтобы указать, какую валюту страны или региона использовать. Константы валюты могут быть указаны как номер ID, соответствующий стране или региону, или строка (заключенная в кавычках), соответствующая аббревиатуре ISO 4217 с тремя символами.
  
Если вы показываете символы валюты для нелокальных валют, а система не знает символ для данной валюты, отображается символ доллара ($).
