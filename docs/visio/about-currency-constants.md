# <a name="about-currency-constants"></a>Сведения о денежных константах

Форматирование числу, как валюта, можно использовать функцию кг и передать константой необязательно для указания страну или регион, в валюте для использования. Константы валюты можно указать как Идентификационный номер, соответствующий страны или региона или строка (заключены в кавычки), соответствующий сокращение ISO 4217 трех символов.
  
Если система не имеет символ для данной валюты Показать символы валюты для нелокальные валют, отображается символ доллара ($).
