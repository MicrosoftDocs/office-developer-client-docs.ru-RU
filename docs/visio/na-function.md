# <a name="na-function"></a>Функция NA

Возвращает значение ошибки #NA!. 
  
Используйте функцию НД в ячейке таблицы свойств фигуры, в которой отсутствуют сведения, чтобы значение по умолчанию для ячейки не использовалось в вычислениях.
  
## <a name="syntax"></a>Синтаксис

НД ()
  
### <a name="return-value"></a>Возвращаемое значение

Строка
  
## <a name="remarks"></a>Комментарии

НД — это значение ошибки, означающее "нет доступных значений".
