# <a name="disttopath-function"></a>Функция DISTTOPATH

Возвращает самое короткое расстояние от точки, представленной указанными координатами, до точки на пути.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

Раздел DISTTOPATH(**  **, ** *x* **, ** *y* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _section_ <br/> |Обязательно  <br/> |**Строка** <br/> |Раздел "Геометрия", который представляет путь, заданный ссылкой на его ячейку Path (например, Geometry1.Path).  <br/> |
| _x_ <br/> |Обязательна  <br/> |64-разрядное число с плавающей запятой двойной точности. <br/> |X-координата точки.   <br/> |
| _y_ <br/> |Обязательна  <br/> |64-разрядное число с плавающей запятой двойной точности. <br/> |Y-координата точки.   <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 64-разрядное число с плавающей запятой двойной точности.
  
## <a name="remarks"></a>Примечания

Microsoft Visio возвращает #REF! если  _раздел_ не существует. 
  
Возвращаемая величина является положительной, если точка находится слева от направления перемещения; это отрицательно, если точка находится справа от направления перемещения.
