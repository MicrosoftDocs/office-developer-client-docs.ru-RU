# <a name="minute-function-visioshapesheet"></a>MINUTE Function (VisioShapeSheet)

Возвращает целое число от 0 до 59, представляющее компонент минут в *значении DateTime* или *Expression* . 
  
## <a name="syntax"></a>Синтаксис

МИНУТа (" *DateTime* " |  *Expression (выражение* )  [, *LCID* ]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _datetime_ <br/> |Обязательный  <br/> |**String** <br/> |Любая строка, распознаваемая как дата и время либо ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**String** <br/> | Любое выражение, возвращающее дату и время.  <br/> |
| _lcid_ <br/> |Необязательный  <br/> |**Number** <br/> |Идентификатор языкового стандарта, используемый при оценке нелокальных даты и времени. Идентификатор языкового стандарта — это число, представленной в файлах системных заголовков.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Целое число
  
## <a name="remarks"></a>Замечания

Компонент даты в _DateTime_ и _выражении_ отбрасывается. 
  
Округление не выполняется. Если _DateTime_ отсутствует или не может быть преобразован в допустимый результат, функция возвращает сообщение об ошибке. 
  
Возвращаемое значение форматируется в соответствии со стилем времени, установленным текущими региональными параметрами системы.
  
Функция минуты также принимает одно числовое значение для _выражения_ , где дробная часть представляет собой долю дня после полуночи. 
  
## <a name="example-1"></a>Пример 1

МИНУТА ("7/7/1999 13:45:24")
  
Возвращает 45.
  
## <a name="example-2"></a>Пример 2

МИНУТа (TIMEVALUE ("Янв. 25, 1999 12:07:45") + 6 EM.)
  
Возвращает 13.
  
## <a name="example-3"></a>Пример 3

МИНУТА (0.575)
  
Возвращает 48.
