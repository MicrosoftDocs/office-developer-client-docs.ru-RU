# <a name="minute-function-visioshapesheet"></a>MINUTE Function (VisioShapeSheet)

Возвращает компоненты от 0 до 59, которые представляют собой компонент минут *даты* или *выражения.* 
  
## <a name="syntax"></a>Синтаксис

MINUTE(" *дата* "|  *выражение* [, *lcid])* 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _datetime_ <br/> |Обязательный  <br/> |**String** <br/> |Любая строка, распознаваемая как дата и время либо ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**String** <br/> | Любое выражение, возвращающее дату и время.  <br/> |
| _lcid_ <br/> |Необязательный  <br/> |**Number** <br/> |Идентификатор языкового стандарта, используемый при оценке нелокальных даты и времени. Идентификатор языкового стандарта — это число, представленной в файлах системных заголовков.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

Целое число
  
## <a name="remarks"></a>Замечания

Компонент даты в  _дате_ и  _выражении_ удаляется. 
  
Округление не выполняется. Если  _дата отсутствует_ или не может быть преобразована в допустимый результат, функция возвращает ошибку. 
  
Возвращенное значение отформатировано в соответствии со стилем времени, установленным текущим региональным Параметры.
  
Функция MINUTE также принимает одно значение  номера для выражения, где десятичная часть представляет долю дня с полуночи. 
  
## <a name="example-1"></a>Пример 1

MINUTE ("7/7/1999 13:45:24")
  
Возвращает 45.
  
## <a name="example-2"></a>Пример 2

MINUTE (TIMEVALUE("25 января 1999 г. 12:07:45")+6 em.)
  
Возвращает 13.
  
## <a name="example-3"></a>Пример 3

MINUTE (0.575)
  
Возвращает 48.
