# <a name="actions-section"></a>Раздел "Действия"

Содержит строки, описывающие пунктов меню на действие или контекстное меню тега фигуры или страницы. В предыдущих версиях Microsoft Visio теги действий, называются смарт-тегов. Ячейка **TagName** служит для связывания действия со строкой в разделе **Тег действия** , где определены теги действий. 
  
> [!NOTE]
> В предыдущих версиях Microsoft Visio теги действий, называются смарт-тегов. 
  
## <a name="remarks"></a>Замечания

Ячейка **TagName** служит для связывания действия со строкой в разделе **Тег действия** , где определены теги действий.
