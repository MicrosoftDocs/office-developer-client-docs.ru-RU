# <a name="pointalongpath-function"></a>Функция POINTALONGPATH

Возвращает координаты точки на пути или смещения с нее.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

POINTALONGPATH(** *раздел* **, **** путешествия ** *** [,смещение]* ** *[,сегмент]* ** * ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _section_ <br/> |Обязательный  <br/> |**String** <br/> |Раздел Геометрия, который представляет путь, заданный ссылкой на его ячейку Path (например, Geometry1.Path).  <br/> |
| _путешествия_ <br/> |Обязательный  <br/> |**Double** <br/> |Процент пройденного пути от точки начала до конечной точки, определяемой точкой. Должно быть от 0 до 1.  <br/> |
| _смещение_ <br/> |Необязательный  <br/> |**Double** <br/> |Расстояние, которое точка смещена с пути. Дополнительные сведения см. в комментарии.  <br/> |
| _segment_ <br/> |Необязательный  <br/> |**Integer** <br/> |1-ый сегмент пути, в котором вычисляются координаты.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 **Point**
  
## <a name="remarks"></a>Примечания

Если _раздел_ или _сегмент_ не существует, корпорация Майкрософт Visio возвращает #REF!. 
  
Положительные  *значения*  смещения указывают точки слева от направления перемещения. 
  
*Отрицательные* значения смещения указывают точки справа от направления перемещения. 
  
Точка **представляет** упорядоченную пару геометрических координат *(x,y)* как единое значение.
