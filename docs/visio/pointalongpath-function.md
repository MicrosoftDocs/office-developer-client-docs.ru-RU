# <a name="pointalongpath-function"></a>Функция POINTALONGPATH

Возвращает координаты точки пути или смещения от нее.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

POINTALONGPATH(** *section* **, ** *travel* ** *[,offset]* ** *** [,segment]* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _section_ <br/> |Обязательно  <br/> |**Строка** <br/> |Раздел "Геометрия", который представляет путь, заданный ссылкой на его ячейку Path (например, Geometry1.Path).  <br/> |
| _travel_ <br/> |Обязательна  <br/> |64-разрядное число с плавающей запятой двойной точности. <br/> |Процент прохода пути от точки начала до конечной точки, определяемой точкой. Должно быть от 0 до 1.  <br/> |
| _offset_ <br/> |Необязательна  <br/> |64-разрядное число с плавающей запятой двойной точности. <br/> |Расстояние, на которое точка смещена от пути. Дополнительные сведения см. в примечаиях.  <br/> |
| _segment_ <br/> |Необязательна  <br/> |64-разрядное целое число. <br/> |Сегмент пути на основе 1, в котором вычисляются координаты.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 **Point**
  
## <a name="remarks"></a>Примечания

Если  _раздел_ или  _сегмент_ не существует, Microsoft Visio возвращает #REF!. 
  
Положительные  *значения*  смещения указывают точки слева от направления перемещения. 
  
*Отрицательные* значения смещения указывают справа от направления перемещения. 
  
Точка **представляет** упорядоченную пару геометрических координат *(x,y)* в виде одного значения.
