# <a name="pointalongpath-function"></a>Функция POINTALONGPATH

Возвращает координаты точки на или смещение из пути.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

POINTALONGPATH (** *раздел* **, ** *путешествий* ** ** *[, смещение]* ** ** *[, сегмент]* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _section_ <br/> |Обязательный  <br/> |**Строка** <br/> |Раздел геометрии, представляющий путь, указанный с помощью ссылки на его ячейку Path (например, Geometry1.Path).  <br/> |
| _поездки_ <br/> |Обязательный  <br/> |**Double** <br/> |Процент путь, начиная с начальной точки в конечную точку, которая определяет точку. Должен быть в диапазоне от 0 до 1.  <br/> |
| _Смещение_ <br/> |Optional  <br/> |**Double** <br/> |Расстояние смещения, что точка из пути. Для получения дополнительных сведений см.  <br/> |
| _сегмента_ <br/> |Optional  <br/> |**Integer** <br/> |Сегмент путь для вычисления координат, основанный на 1.  <br/> |
   
### <a name="return-value"></a>������������ ��������

 **Точка**
  
## <a name="remarks"></a>Замечания

Если _раздел_ или _Сегмент_ не существует, Microsoft Visio возвращает #REF!. 
  
Положительное *смещение* определяют точки слева от направление командировок. 
  
Отрицательное *смещение* определяют точек справа от направление командировок. 
  
**Точка** представляет упорядоченную пару геометрические координат (*x, y*) как одно значение.
