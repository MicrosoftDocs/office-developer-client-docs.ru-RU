# <a name="pointalongpath-function"></a>Функция POINTALONGPATH

Возвращает координаты точки, а также смещение относительно пути.
  
## <a name="version-information"></a>Сведения о версии

Добавлена версия: Visio 2010
 
  
## <a name="syntax"></a>Синтаксис

POINTALONGPATH (* * *раздел* * *, * * *путешествие* * * * * *[, offset]* * * * * *[, сегмент]* * *) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _section_ <br/> |Обязательный  <br/> |**String** <br/> |Раздел геометрии, представляющий путь, заданный ссылкой на ячейку пути (например, Geometry1. Path).  <br/> |
| _дающих_ <br/> |Обязательна  <br/> |**Double** <br/> |Процент прохождения пути от точки Begin до конечной точки, определяющей точку. Значение должно находиться в пределах от 0 до 1.  <br/> |
| _корреспондирующей_ <br/> |Необязательна  <br/> |**Double** <br/> |Расстояние, на которое точка смещается относительно пути. Дополнительные сведения см.  <br/> |
| _segment_ <br/> |Необязательна  <br/> |**Целое число** <br/> |Сегмент на основе 1 пути, в котором вычисляется координат.  <br/> |
   
### <a name="return-value"></a>Возвращаемое значение

 **Point**
  
## <a name="remarks"></a>Примечания

Если _раздел_ или _сегмент_ не существует, Microsoft Visio возвращает #REF!. 
  
Положительные значения *смещения* задают точки слева от направления поездки. 
  
Отрицательные значения *смещения* задают точки справа от направления поездки. 
  
**Точка** представляет собой упорядоченную комбинацию геометрических координат (*x, y*) в виде одного значения.
