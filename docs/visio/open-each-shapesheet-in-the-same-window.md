# <a name="open-each-shapesheet-in-the-same-window"></a>Откройте каждую таблицу shapeSheet в одном окне

1. Щелкните **вкладку Файл,** щелкните **Параметры** и нажмите **кнопку Advanced**.
    
2. В **general** выберите **Откройте каждую таблицу ShapeSheet в одном окне.**
    
3. Нажмите кнопку **ОК**.
