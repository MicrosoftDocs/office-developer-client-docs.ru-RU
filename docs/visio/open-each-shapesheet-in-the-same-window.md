# <a name="open-each-shapesheet-in-the-same-window"></a>Открытие таблицы фигур в одном окне

1. Щелкните **вкладку** "Файл", выберите **"Параметры"** и нажмите кнопку **"Дополнительные".**
    
2. В **области "Общие"** выберите "Открыть все таблицы **фигур в одном окне".**
    
3. Нажмите кнопку **ОК**.
