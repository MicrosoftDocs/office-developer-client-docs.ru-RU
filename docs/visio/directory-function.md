# <a name="directory-function"></a>Функция DIRECTORY

Возвращает полный путь к каталогу (папке), в котором сохранен документ, в виде строки. Возвращает пустую строку, если документ не был сохранен.
  
## <a name="syntax"></a>Синтаксис

DIRECTORY ()
