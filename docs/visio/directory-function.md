# <a name="directory-function"></a>Функция DIRECTORY

Возвращает полный путь каталога (папки), в котором сохранен документ, в качестве строки. Возвращает пустую строку, если документ не был сохранен.
  
## <a name="syntax"></a>Синтаксис

DIRECTORY ()
