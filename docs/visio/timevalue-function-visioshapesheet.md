# <a name="timevalue-function-visioshapesheet"></a>TIMEVALUE Function (VisioShapeSheet)

Возвращает значение времени, представляемого  _датой или_  _выражением,_ на основе параметров языка и региона системы.
  
## <a name="syntax"></a>Синтаксис

TIMEVALUE(" ** *datetime* ** "| ** *expression* ** [, ** *lcid* ** ]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _datetime_ <br/> |Обязательно  <br/> |**Строка** <br/> | Любая строка, распознаваемая как дата и время либо ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**Разные** <br/> | Любое выражение, возвращающее дату и время.  <br/> |
| _lcid_ <br/> |Необязательный  <br/> |**Number** <br/> |Идентификатор языкового стандарта, используемый при оценке нелокальных даты и времени. Идентификатор языкового стандарта — это число, представленной в файлах системных заголовков.  <br/> |
   
## <a name="remarks"></a>Примечания

Любой компонент даты  _в дате или_  _выражении_ удаляется. 
  
Если  _дата-время_ отсутствует или не удается преобразовать в допустимый результат, эта функция возвращает #VALUE! error. 
  
Функция TIMEVALUE также принимает одно числовые значения для выражения, где десятичная часть результата представляет часть дня с полуночи.  
  
## <a name="example-1"></a>Пример 1

TIMEVALUE("6:00 AM")
  
Возвращает значение, представляющее 6:00.
  
## <a name="example-2"></a>Пример 2

TIMEVALUE("14:30")+4 eh.+30 em.
  
Возвращает значение, представляющее 19:00:00.
  
## <a name="example-3"></a>Пример 3

TIMEVALUE("11:00, 1 июля 1997 г.")
  
Возвращает значение, представляющее 11:00.
  
## <a name="example-4"></a>Пример 4

TIMEVALUE(0.6337)
  
Возвращает значение, представляющее 15:12:32.
  
## <a name="example-5"></a>Пример 5

TIMEVALUE("7:89")
  
Возвращает #VALUE! error.
