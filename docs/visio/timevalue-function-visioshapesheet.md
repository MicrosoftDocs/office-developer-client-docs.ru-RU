# <a name="timevalue-function-visioshapesheet"></a>Функция TIMEVALUE (VisioShapeSheet)

Возвращает значение времени, представленного _даты и времени_ или _выражения_на основании системные язык и региональные параметры.
  
## <a name="syntax"></a>Синтаксис

Функция TIMEVALUE ("** *даты и времени* **" | ** *выражение* ** [, ** *lcid* **]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _даты и времени_ <br/> |Обязательный  <br/> |**Строка** <br/> | Любая строка, часто распознается как даты и времени или ссылка на ячейку, содержащую дату и время.  <br/> |
| _expression_ <br/> |Обязательный  <br/> |**Может отличаться** <br/> | Любое выражение, возвращающее даты и времени.  <br/> |
| _код языка_ <br/> |Optional  <br/> |**Число** <br/> |Идентификатор языкового стандарта, который следует использовать для определения нелокальные datetime. Идентификатор языкового стандарта — это число, описанных в файлы заголовков системы.  <br/> |
   
## <a name="remarks"></a>Замечания

Любой компонент даты в _даты и времени_ или _выражение,_ удаляются. 
  
Если _даты и времени_ , отсутствует или не может быть преобразована допустимый результат, эта функция возвращает #VALUE! Ошибка. 
  
Функция TIMEVALUE также принимает одно значение номеров для _выражения_ , где целой и дробной части результат представляет долю день после полночи. 
  
## <a name="example-1"></a>Пример 1

ФУНКЦИЯ TIMEVALUE ("6:00:00")
  
Возвращает значение, представляющее 6:00:00.
  
## <a name="example-2"></a>Пример 2

Функция TIMEVALUE ("14:30") + 4 а. длинное + 30.
  
Возвращает значение, представляющее 19:00:00.
  
## <a name="example-3"></a>Пример 3

Функция TIMEVALUE ("11: 00, 1 июля 1997 г.")
  
Возвращает значение, представляющее 11:00:00.
  
## <a name="example-4"></a>Пример 4

TIMEVALUE(0.6337)
  
Возвращает значение, представляющее 15:12:32.
  
## <a name="example-5"></a>Пример 5

TIMEVALUE("7:89")
  
Возвращает #VALUE! Ошибка.
