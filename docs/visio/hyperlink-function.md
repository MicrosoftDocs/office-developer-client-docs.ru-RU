# <a name="hyperlink-function"></a>Функция ГИПЕРССЫЛКИ

Переходит по указанному адресу, который может быть файл, UNC или URL-адрес путь.
  
## <a name="syntax"></a>Синтаксис

Гиперссылка ("** *адрес* **" [,"** *дополнительный адрес* **","** *extrainfo* **", ** *окно* **, "** *frame* **"]) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _адрес_ <br/> |Обязательный  <br/> |**Строка** <br/> |Полный путь или относительный путь.  <br/> |
| _Дополнительный адрес_ <br/> |Optional  <br/> |**Строка** <br/> |Указывает расположение в адрес, который необходимо связать с. Например если адрес файл Microsoft Visio, дополнительный адрес может быть имя страницы; Если файл Microsoft Excel, дополнительный адрес может быть электронной таблицы или диапазон в электронной таблице; Если URL-адрес страницы HTML, дополнительный адрес может быть привязки.  <br/> |
| _ExtraInfo_ <br/> |Optional  <br/> |**Строка** <br/> |Передает данные, используемые в разрешении URL-адрес, например координаты гиперкарты.  <br/> |
| _окно_ <br/> |Optional  <br/> |**Boolean** <br/> |Указывает, будет ли гиперссылка открывается в новом окне. Значение по умолчанию — FALSE.  <br/> |
| _кадров_ <br/> |Optional  <br/> |**Строка** <br/> | Указывает имя рамки для конечного при открытом как активный документ в браузере ActiveX, такие как Microsoft Internet Explorer 3.0 или более поздней версии Visio. По умолчанию используется пустая строка.  <br/> |
   
## <a name="remarks"></a>Замечания

Если в документе есть путь не базового, Visio переходит в соответствии с путь к документу. Если документ не был сохранен, гиперссылки не определено. 
  
Относительные пути основаны на поле **базового гиперссылки** , указанных в диалоговом окне **Свойства Visio** . 
  
Функция НАСТРАНИЦУ перейдите к страницам документа. 
  
## <a name="example-1"></a>Пример 1

 `HYPERLINK("C:\My Documents\Drawing1.vsdx")`
  
## <a name="example-2"></a>Пример 2

 `HYPERLINK("\\Server\Share\Drawing1.vsdx")`
  
## <a name="example-3"></a>Пример 3

 `HYPERLINK("http://www.microsoft.com")`
  
## <a name="example-4"></a>Пример 4

 `HYPERLINK("..\data.xlsx","sheet1!A1")`
