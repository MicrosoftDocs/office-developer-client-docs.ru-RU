# <a name="eventxfmod-cell-events-section"></a>EventXFMod Cell (Events Section)

Ячейка события, которая оценивается при преобразовании положения или ориентации фигуры на странице ("XF").
  
## <a name="remarks"></a>Примечания

Ячейки событий оцениваются только при событии, а не при вводе формулы.
  
Чтобы получить ссылку на ячейку EventXFMod по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | EventXFMod  <br/> |
   
Чтобы получить ссылку на ячейку EventXFMod по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowEvent** <br/> |
| Индекс ячейки:  <br/> |**visEvtCellXFMod** <br/> |
