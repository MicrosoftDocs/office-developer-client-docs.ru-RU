# <a name="linecolor-cell-line-format-section"></a>Ячейка LineColor (раздел "Формат линий")

Определяет цвет линии фигуры.
  
## <a name="remarks"></a>Замечания

Чтобы задать цвет линии, введите число от 0 до 23, которого является индексом в коллекцию цвета строк. Можно просмотреть семейства цвет строки в поле **строка** (на вкладке **Главная** в группе **фигуры** щелкните **строку**, выберите пункт **Вес**и нажмите кнопку **Другие линии**). Также можно задать значение цвет линии в диалоговом окне **строки** . 
  
Чтобы ввести другой цвет, используйте функцию RGB или HSL. Настраиваемые цвета значение цвета RGB, а RGB ( *r, g, b*), а не числа, будут отображаться в окне таблицы свойств фигуры. При использовании в операции, настраиваемые цвета имеют значения из 24 и выше. 
  
Можно задать прозрачность цвета строки в ячейке LineColorTrans.
  
Чтобы получить ссылку на ячейку цвет линии по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Цвет линии  <br/> |
   
Для получения ссылки на ячейки цвет линии по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowLine** <br/> |
|Индекс ячейки:  <br/> |**visLineColor** <br/> |
