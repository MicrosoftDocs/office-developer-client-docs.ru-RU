# <a name="linejumpcode-cell-page-layout-section"></a>LineJumpCode Cell (Page Layout Section)

Определяет соединители, к которым вы хотите добавить переходы.
  
|**Значение**|**Соединители, к которым вы хотите добавить переходы**|**Константа автоматизации**|
|:-----|:-----|:-----|
|нуль  <br/> |Нет  <br/> |**Виспложумпноне** <br/> |
|1,1  <br/> |Горизонтальные линии  <br/> |**Виспложумфоризонтал** <br/> |
|2  <br/> |Вертикальные линии  <br/> |**Виспложумпвертикал** <br/> |
|4  <br/> |Последняя строка маршрута  <br/> |**Виспложумпластраутед** <br/> |
|SP4  <br/> |Последняя отображаемая строка (верхняя фигура в *z* -порядке)  <br/> |**Виспложумпдисплайордер** <br/> |
|17:00  <br/> |Первая отображаемая строка (фигура в нижней части *z* -порядка)  <br/> |**Виспложумпреверседисплайордер** <br/> |
   
## <a name="remarks"></a>Примечания

Значение этой ячейки также можно задать на вкладке **Макет и маршрутизация** в диалоговом окне **Параметры страницы** (на вкладке **конструктор** щелкните стрелку **Параметры страницы** , а затем выберите **Макет и маршрутизация**).
  
Чтобы получить ссылку на ячейку LineJumpCode по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |LineJumpCode  <br/> |
   
Чтобы получить ссылку на ячейку LineJumpCode по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровпажелайаут** <br/> |
|Индекс ячейки:  <br/> |**Виспложумпкоде** <br/> |
