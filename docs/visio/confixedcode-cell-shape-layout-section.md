# <a name="confixedcode-cell-shape-layout-section"></a>ConFixedCode Cell (Shape Layout Section)

Определяет, когда соединитель пересылается.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|нуль  <br/> |Перенаправление свободно  <br/> |**Висслоконфикседрераутефрили** <br/> |
|1,1  <br/> |Перенаправление при необходимости (перенаправление вручную)  <br/> |**Висслоконфикседрераутеаснидед** <br/> |
|2  <br/> |Никогда не перенаправлять  <br/> |**Висслоконфикседрераутеневер** <br/> |
|4  <br/> |Перенаправление при пересечении  <br/> |**Висслоконфикседрераутеонкроссовер** <br/> |
|SP4  <br/> |Только для внутреннего использования  <br/> |**Висслоконфикседбялгфром** <br/> |
|17:00  <br/> |Только для внутреннего использования  <br/> |**Висслоконфикседбялгто** <br/> |
|ICMPv6  <br/> |Только для внутреннего использования  <br/> |**Висслоконфикседбялгфромто** <br/> |
   
## <a name="remarks"></a>Примечания

Можно также задать значение этой ячейки, выбрав динамический соединитель, нажав кнопку **поведение** в группе **Макет фигуры** на вкладке [разработчик](run-in-developer-mode-display-the-developer-tab.md) , а затем щелкнув вкладку **соединитель** . 
  
Чтобы получить ссылку на ячейку ConFixedCode по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующее: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |ConFixedCode  <br/> |
   
Чтобы получить ссылку на ячейку ConFixedCode по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**Висровшапелайаут** <br/> |
|Индекс ячейки:  <br/> |**Висслоконфикседкоде** <br/> |
