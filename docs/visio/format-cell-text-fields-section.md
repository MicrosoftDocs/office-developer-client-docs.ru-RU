# <a name="format-cell-text-fields-section"></a>Format Cell (Text Fields Section)

Указывает форматирование текстового поля, которое является строкой, числом, датой или временем, продолжительностью или валютой.
  
## <a name="remarks"></a>Примечания

Если ячейка Type имеет значение 0, 2, 5, 6 или 7 (строка, число, дата, время, длительность или валюта соответственно), укажите формат, соответствующий типу данных. Например, изображение формата "# #/4 UU" форматирование числа 12,43 в. как 12 2/4 ДЮЙМА. Дополнительные сведения об указании формата рисунка см. в [сведениях о формате изображений.](about-format-pictures.md)
  
Число (тип = 2) может представлять измерение, скалярное, угол, дату, время или валюту. Чтобы обеспечить, чтобы входной номер всегда оценивался как дата, время или валюта, используйте функцию DATETIME или CY в ячейке Format вместо изображения формата.
  
Чтобы получить ссылку на ячейку Format по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Fields.Format[  *i*  ] где  *i*  = <1>, 2, 3...  <br/> |
   
Чтобы получить ссылку на ячейку Format по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionTextField** <br/> |
| Индекс строки:  <br/> |**visRowField**  +   *i* где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visFieldFormat** <br/> |
