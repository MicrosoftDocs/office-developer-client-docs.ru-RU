# <a name="case-cell-character-section"></a>Ячейка Case (раздел "Символ")

Определяет регистр текста формы. Все инвестиционные регистре (1) и прописными буквами (2) не изменяйте внешний вид текста, который был введен прописными буквами. Текст должен указываться в строчные буквы для этих параметров для отображения эффекта.
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
| 0  <br/> | Обычный case  <br/> |**visCaseNormal** <br/> |
| 1  <br/> | Все инвестиционные регистре  <br/> |**visCaseAllCaps** <br/> |
| 2  <br/> | Прописными буквами только  <br/> |**visCaseInitialCaps** <br/> |
   
## <a name="remarks"></a>Замечания

Для получения ссылки на ячейки вариантов по имени, из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки.  <br/> | Char.Case [ *i* ] где *i* = < 1 > 2, 3,...  <br/> |
   
Для получения ссылки на ячейки вариантов по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionCharacter** <br/> |
| Индекс строки:  <br/> |**visRowCharacter** +  *i* где *i* = 0, 1, 2,...  <br/> |
| Индекс ячейки:  <br/> |**visCharacterCase** <br/> |
