# <a name="acos-function"></a>Функция ACOS

Возвращает arccosine числа, например, угол, cosine которого является  *номером*  . 
  
## <a name="syntax"></a>Синтаксис

ACOS *(номер)* 
  
## <a name="remarks"></a>Примечания

Значение ввода должно быть в диапазоне -1 \< = номер \< = 1 или #NUM! возвращается ошибка. В результате угол находится в диапазоне 0 \< = угол \< = радианы PI (0 \< = угол = \< 180 градусов).
  
## <a name="example"></a>Пример

ACOS (0) 
  
Возвращает 90 дег
