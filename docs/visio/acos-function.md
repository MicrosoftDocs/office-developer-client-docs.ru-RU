# <a name="acos-function"></a>Функция ACOS

Возвращает арккосинус числа, например угол, косинус которого *является числом.* 
  
## <a name="syntax"></a>Синтаксис

*ACOS(number)* 
  
## <a name="remarks"></a>Примечания

Входное значение должно быть в диапазоне -1 \< = число \< = 1, или значение #NUM! возвращается ошибка. В результате угол находится в диапазоне 0 \< = угол \< = радианы PI (0 \< = угол = \< 180 градусов).
  
## <a name="example"></a>Пример

ACOS(0) 
  
Возвращает 90 deg
