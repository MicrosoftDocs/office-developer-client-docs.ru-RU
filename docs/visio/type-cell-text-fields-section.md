# <a name="type-cell-text-fields-section"></a>Ячейка Type (раздел "Текстовые поля")

Указывает тип данных для значения поля текст.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0  <br/> |Строка.  <br/> |
|2  <br/> |Номер. Включает в себя дату, значения времени, продолжительность и валюты, а также скаляры, измерений и углы. Укажите изображение формата в формат ячейки.  <br/> |
|5  <br/> |Значение даты или времени. Отображение дней, месяцев и лет или секунд, минут и часов, или объединены значения даты и времени. Укажите изображение формата в формат ячейки.  <br/> |
|6  <br/> |Значения длительности. Отображение времени, затраченного. Укажите изображение формата в формат ячейки.  <br/> |
|7  <br/> |Значение типа currency. Использует текущих региональных параметров компьютера. Укажите изображение формата в формат ячейки.  <br/> |
   
## <a name="remarks"></a>Замечания

Также можно задать значение ячейки с помощью диалогового окна **поля** (фигуры, выбранной на вкладке **Вставка** в группе **текст** щелкните **поле** ). 
  
Чтобы получить ссылку на ячейку тип по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |Fields.Type [ *i* ] где *i* = < 1 > 2, 3...  <br/> |
   
Для получения ссылки на ячейки типа по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionTextField** <br/> |
|Индекс строки:  <br/> |**visRowField** +  *i* где *i* = 0, 1, 2...  <br/> |
|Индекс ячейки:  <br/> |**visFieldType** <br/> |
