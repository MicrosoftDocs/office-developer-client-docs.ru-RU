# <a name="uivisibility-cell-page-properties-section"></a>Ячейка UIVisibility (раздел "Свойства страницы")

Определяет, является ли имя страницы, открытой в пользовательского интерфейса (UI).
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Отображаемое имя страницы в пользовательском Интерфейсе (по умолчанию).  <br/> |**visUIVNormal** <br/> |
|1  <br/> |Имя страницы не отображаются в пользовательском Интерфейсе.  <br/> |**visUIVHidden** <br/> |
   
## <a name="remarks"></a>Замечания

Установка для ячейки UIVisibility **visUIVHidden** запрещает странице Отображение в любом месте в пользовательском Интерфейсе, где отображается строка, содержащая имя страницы. Например страницы не могут быть указаны как параметр в **Обозревателе документа** или на вкладки. Страницы остается доступным, тем не менее, если вы используете автоматизации или пользовательского интерфейса путей, которые не включать имя страницы, например, команда **печати** . 
  
 В этой ячейке предназначен для использования со страницы документа; он не предназначен для использования с разметки перекрытия страниц, которые имеют UIVisibility ячейки, значение **visUIVHidden** по умолчанию и не должно изменяться. 
  
> [!NOTE]
> Чтобы скрыть страницы из команды **Печать** документа, сделать его на страницу фон (**Тип** свойства — **visTypeBackground** ), который не используется как фон любую страницу (фигуры на фоне печати страниц при страницы, используя в качестве фона При выводе на печать). Команда **Печать** документов работает только с индексированных страниц и фона страниц, не индексируются. 
  
Чтобы получить ссылку на ячейку UIVisibility по имени из другой формулы или из файла с помощью свойства **CellsU** , используйте: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |UIVisibility  <br/> |
   
Для получения ссылки на ячейки UIVisibility по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPage** <br/> |
|Индекс ячейки:  <br/> |**visPageUIVisibility** <br/> |
