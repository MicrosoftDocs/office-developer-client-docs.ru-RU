# <a name="uivisibility-cell-page-properties-section"></a>UIVisibility Cell (Page Properties Section)

Определяет, выставлено ли имя страницы в пользовательском интерфейсе (пользовательском интерфейсе).
  
|**Значение**|**Описание**|**Константа автоматизации**|
|:-----|:-----|:-----|
|0  <br/> |Отображение имени страницы в пользовательском интерфейсе (по умолчанию).  <br/> |**visUIVNormal** <br/> |
|1  <br/> |Не отображать имя страницы в пользовательском интерфейсе.  <br/> |**visUIVHidden** <br/> |
   
## <a name="remarks"></a>Примечания

Настройка ячейки UIVisibility **для visUIVHidden** не позволяет странице отображаться в любом месте пользовательского интерфейса, где отображается строка, содержащая имя страницы. Например, страница не будет указана в качестве параметра в обозревателе **рисования** или на вкладке страницы. Однако страница остается доступной, если вы используете пути автоматизации или пользовательского интерфейса, которые не включают имя страницы, например команду **Print.** 
  
 Эта ячейка предназначена для использования со страницами документов; она не предназначена для использования со страницами наложения разметки, которые имеют ячейку UIVisibility, задаваемую **visUIVHidden** по умолчанию и не должны быть изменены. 
  
> [!NOTE]
> Чтобы скрыть страницу из  команды Печать документа, сделайте ее фоновой **страницей** (Тип свойства **visTypeBackground),** которая не используется в качестве фона для любой страницы (фигуры на фоновых страницах печатаются при печати страницы с ее использованием в качестве фона). Команда Печати **документа** работает только с индексными страницами, а фоновые страницы не индексироваться. 
  
Чтобы получить ссылку на ячейку UIVisibility по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
|Имя ячейки:  <br/> |UIVisibility  <br/> |
   
Чтобы получить ссылку на ячейку UIVisibility по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowPage** <br/> |
|Индекс ячейки:  <br/> |**visPageUIVisibility** <br/> |
