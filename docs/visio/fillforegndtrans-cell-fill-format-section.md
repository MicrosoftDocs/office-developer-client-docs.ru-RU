# <a name="fillforegndtrans-cell-fill-format-section"></a>Ячейка FillForegndTrans (раздел "Формат заливки")

Определяет уровень прозрачности цвет переднего плана (заливки) узора заливки фигуры.
  
|**Значение**|**Описание**|
|:-----|:-----|
|0 — 100  <br/> |Представляет собой процентную долю прозрачность. Значение по умолчанию — 0% (полностью непрозрачный).  <br/> |
   
## <a name="remarks"></a>Замечания

Значения округляются до ближайшего большего половины процентов. Значение 100% является полностью прозрачным. Несмотря на то, что фигуры с полностью прозрачной заливкой отображается же, как фигуры без заливки на странице документа, он будет взаимодействовать с другими объектами на странице же образом как в случае его прозрачность 0%.
  
Также можно задать это значение, с помощью элемента управления ползунок в диалоговом окне **заполнить** (на вкладке **Главная** в группе **фигуру** щелкните **заполните поля**и нажмите кнопку **Параметры заливки**). Это значение определяет значение Прозрачность заливки фона и переднего плана. Чтобы задать эти значения независимо друг от друга, необходимо ввести их в окне таблицы свойств фигуры.
  
Чтобы получить ссылку на ячейку FillForegndTrans по имени из другой формулы, и программы, с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
|Имя ячейки.  <br/> |FillForegndTrans  <br/> |
   
Для получения ссылки на ячейки FillForegndTrans по индексу из программы, используйте свойство **CellsSRC** с следующие аргументы: 
  
|||
|:-----|:-----|
|Индекс раздела:  <br/> |**visSectionObject** <br/> |
|Индекс строки:  <br/> |**visRowFill** <br/> |
|Индекс ячейки:  <br/> |**visFillForegndTrans** <br/> |
