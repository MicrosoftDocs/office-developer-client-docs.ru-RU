# <a name="cy-function"></a>Функция CY

Возвращает значение валюты.
  
## <a name="syntax"></a>Синтаксис

CY(** *value* **, ** *cyID* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _value_ <br/> |Необязательна  <br/> |**Число или строка** <br/> |Число или строка, включаемая форматирование для определенной валюты. Если значение не указано, значение валюты отформатировано в соответствии со стилем валюты в параметрах языка и региона системы.  <br/> |
| _cyID_ <br/> |Необязательна  <br/> |**Number** <br/> |Числовая валюта или трех символьная строка в кавычках для аббревиатуры ISO 4217.  <br/> |
   
## <a name="remarks"></a>Примечания

Чтобы указать другую валюту, необходимо указать допустимый _cyID._ Список см. в [списке "О константах валюты".](about-currency-constants.md)
  
Если  _значение_ несовместимо с указанным типом валюты или указан недопустимый аргумент, например "не число", #VALUE! возвращается ошибка. Если  значение больше 922 337 203 685 477,5807 или меньше -922 337 203 685 477,5808, #VALUE! возвращается ошибка. 
  
Для более точной точности с очень большими значениями валюты, которые включают дробные части единицы, например 3,6- и 3,6-
  
При указании  _недопустимого cyID_ возвращается ошибка. 
  
## <a name="example-1"></a>Пример 1

Если в параметрах языка и региона пользователя указаны доллара США:
  
CY(999998.993)
  
Возвращает 999 998,99 долларов США
  
## <a name="example-2"></a>Пример 2

CY(12345678.993, "USD")
  
Возвращает 12 345 678,99 долларов США
  
## <a name="example-3"></a>Пример 3

CY(12345678.993, "DEM")
  
Возвращает 12 345 678,99 DEM
  
## <a name="example-4"></a>Пример 4

CY(12345678.7832, "XXX")
  
Возвращает 12 345 678,78
