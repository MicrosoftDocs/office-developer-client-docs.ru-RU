# <a name="cy-function"></a>Функция CY

Возвращает значение типа currency.
  
## <a name="syntax"></a>Синтаксис

КГ (** *значение* **, ** *cyID* **) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _value_ <br/> |Optional  <br/> |**Число или строка** <br/> |Номер или строку, которая включает в себя форматирование конкретной валюте. Если не указан, к денежному значению отформатированное в соответствии с денежный в язык и региональные параметры системы.  <br/> |
| _cyID_ <br/> |Optional  <br/> |**Число** <br/> |Идентификатор числовое валюты или трехзначный строка в кавычках для аббревиатура ISO 4217.  <br/> |
   
## <a name="remarks"></a>Замечания

Чтобы указать другую валюту, необходимо включить действительный _cyID_. Полный список представлен [о константы валюты](about-currency-constants.md).
  
Если _значение_ несовместимо с типом назначенного валюты, или если указан недопустимый аргумент, например «не число» будет указано, #VALUE! возвращается ошибка. Если _значение_ больше, чем 922337203685477,5807 или меньше, чем -922,337,203,685,477.5808, #VALUE! возвращается ошибка. 
  
Для улучшения точности при работе с очень большой денежными значениями, включая долей единиц, таких как триллион 3.6 используйте строковые аргументы для _значения_.
  
Указание недопустимый _cyID_ возвращает ошибку. 
  
## <a name="example-1"></a>Пример 1

Если язык и региональные параметры пользователя укажите долларов США.
  
CY(999998.993)
  
Возвращает $999,998.99
  
## <a name="example-2"></a>Пример 2

КГ (12345678.993 "ДОЛЛАРОВ США")
  
Возвращает $12,345,678.99
  
## <a name="example-3"></a>Пример 3

КГ (12345678.993 "ДЕМОКРАТИЧЕСКАЯ")
  
Возвращает 12,345,678.99 Демократическая
  
## <a name="example-4"></a>Пример 4

КГ (12345678.7832 "XXX")
  
Возвращает 12,345,678.78
