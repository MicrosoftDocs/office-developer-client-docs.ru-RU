# <a name="shapeshdwobliqueangle-cell-fill-format-section"></a>ShapeShdwObliqueAngle Cell (Fill Format Section)

Указывает угол наклонного направления тени фигуры.
  
## <a name="remarks"></a>Примечания

Значение 0 (0) в этой ячейке указывает, что направление угла прямо вверх и измеряется перемещением по часовой стрелке.
  
Это значение соответствует значению параметра "Направление" в диалоговом окне  "Тени" (на вкладке "Главная" в группе фигур щелкните "Тени" и выберите **"Параметры теней").**   
  
Чтобы получить ссылку на ячейку ShapeShdwObliqueAngle по имени из другой формулы или из программы, использующей свойство **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | ShapeShdwObliqueAngle  <br/> |
   
Чтобы получить ссылку на ячейку ShapeShdwObliqueAngle по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionObject** <br/> |
| Индекс строки:  <br/> |**visRowFill** <br/> |
| Индекс ячейки:  <br/> |**visFillShdwObliqueAngle** <br/> |
