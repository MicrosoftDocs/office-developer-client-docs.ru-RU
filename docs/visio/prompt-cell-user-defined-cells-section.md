# <a name="prompt-cell-user-defined-cells-section"></a>Prompt Cell (User-Defined Cells Section)

Задает описательное приглашение или комментарий для пользовательской ячейки. Приложение автоматически заключает текст подсказки в кавычки (""), чтобы указать, что это текстовая строка. Если ввести знак равенства (=) и опустить кавычки, в этой ячейке можно ввести формулу, оцениваемую приложением.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку приглашения по имени из другой формулы или из программы с помощью свойства **CellsU** , используйте следующую команду: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Пользователем.  *Name (имя* ). Подсказка для пользователя.  *Name* — имя строки  <br/> |
   
Чтобы получить ссылку на ячейку приглашения по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**виссектионусер** <br/> |
| Индекс строки:  <br/> |**висровусер +** *i* , где *i* = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**висусерпромпт** <br/> |
