# <a name="prompt-cell-user-defined-cells-section"></a>Prompt Cell (User-Defined Cells Section)

Указывает описательный запрос или комментарий для пользовательской ячейки. Приложение автоматически заключит текст запроса в кавычках (" "), чтобы указать, что это текстовая строка. Если ввести знак равного (=) и опустить кавычка, можно ввести формулу в этой ячейке, которую оценивает приложение.
  
## <a name="remarks"></a>Примечания

Чтобы получить ссылку на ячейку Prompt по имени из другой формулы или из программы с помощью свойства **CellsU,** используйте: 
  
|||
|:-----|:-----|
| Имя ячейки:  <br/> | Пользователь.  *Имя*  . Запрос, в котором находится пользователь.  *Имя*  — это имя строки  <br/> |
   
Чтобы получить ссылку на ячейку Prompt по индексу из программы, используйте свойство **CellsSRC** со следующими аргументами: 
  
|||
|:-----|:-----|
| Индекс раздела:  <br/> |**visSectionUser** <br/> |
| Индекс строки:  <br/> |**visRowUser +** *i*            где  *i*  = 0, 1, 2...  <br/> |
| Индекс ячейки:  <br/> |**visUserPrompt** <br/> |
