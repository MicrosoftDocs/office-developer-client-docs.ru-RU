# <a name="lumdiff-function"></a>Функция LUMDIFF

Возвращает разницу в светоносности *между color1* и *color2.* 
  
## <a name="syntax"></a>Синтаксис

LUMDIFF(** *color1* **, ** *color2* ** ) 
  
### <a name="parameters"></a>Параметры

|**Имя**|**Необходимость**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
| _color1_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Индекс Visio microsoft или значение RGB первого цвета.  <br/> |
| _color2_ <br/> |Обязательный  <br/> |**Числовой** <br/> |Индекс Visio microsoft или RGB второго цвета.  <br/> |
