# <a name="cell-element-actions-row-visio-xml"></a>Элемент ячейки (строки действий) ('Visio XML»)

Указывает одно свойство действие, связанное с пользовательской команды меню тега ярлык или действие.
  
## <a name="element-information"></a>Сведения об элементе

|||
|:-----|:-----|
|**Тип элемента** <br/> |[Cell_Type](cell_type-complextypevisio-xml.md) <br/> |
|**Пространство имен** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Файл схемы** <br/> |VisioSchema15.xsd  <br/> |
|**Части документа** <br/> |Masters.XML, главные # .xml, pages.xml, страницы # .xml  <br/> |
   
## <a name="definition"></a>Определение

```XML
< xs:element name="Cell" type="Cell_Type" minOccurs="0" maxOccurs="unbounded" >
</xs:element >
```

## <a name="remarks"></a>Замечания

Атрибут **N** этого элемента **ячейки** должен быть один из ограниченный набор значений, которые соответствуют ячейки таблицы свойств фигуры. Обратитесь к таблице ниже для определения значений атрибутов **N** , разрешенных для этого элемента **ячейки** . 
  
|**Значение**|**Описание**|**Дополнительные сведения**|
|:-----|:-----|:-----|
|Действие  <br/> |Содержит формулу для выполнения, если пользователь выбирает команду меню тега ярлык или действие.  <br/> |[Ячейка Action (раздел "Действия")](action-cell-actions-section.md) <br/> |
|BeginGroup  <br/> |Указывает, будет ли разделитель вставлен в меню над это действие.  <br/> |[Ячейка BeginGroup (раздел "Действия")](begingroup-cell-actions-section.md) <br/> |
|ButtonFace  <br/> |Определяет значок, который отображается рядом с элементом меню тега ярлык или действие.  <br/> |[Ячейка ButtonFace (раздел "Действия")](buttonface-cell-actions-section.md) <br/> |
|Проверка  <br/> |Указывает, установлен ли флажок элемента в меню тега ярлык или действие.  <br/> |[Ячейка Checked (раздел "Действия")](checked-cell-actions-section.md) <br/> |
|Отключена  <br/> |Указывает, отключена ли пункт меню тега ярлык или действие.  <br/> |[Ячейка Disabled (раздел "Действия")](disabled-cell-actions-section.md) <br/> |
|С параметром FlyoutChild  <br/> |Определяет, является ли строку всплывающего меню дочерние последней строки над текстом, который не является дочерним всплывающее меню.  <br/> |[Ячейка FlyoutChild (раздел "Действия")](flyoutchild-cell-actions-section.md) <br/> |
|Невидимое  <br/> |Указывает, видима ли действие в меню Действие тег или ярлык.  <br/> |[Ячейка Invisible (раздел "Действия")](invisible-cell-actions-section.md) <br/> |

|ReadOnly  <br/> |Определяет, является ли действие на тег или контекстное меню действий только для чтения.  <br/> |[Ячейка ReadOnly (раздел "Действия")](readonly-cell-actions-section.md) <br/> |
|SortKey  <br/> |Число, определяющее порядок действий, которые отображаются на действие или контекстное меню тега.  <br/> |[SortKey (раздела "действия") SortKey ячейке (раздела "действия")](sortkey-cell-actions-section.md) <br/> |
|TagName  <br/> |Содержит имя тега действия, с которым связана это действие.  <br/> |[Ячейка TagName (раздел "Действия")](tagname-cell-actions-section.md) <br/> |
