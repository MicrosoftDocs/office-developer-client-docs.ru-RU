# <a name="opensession"></a>OpenSession

**Область применения:** Excel 2013 | Office 2013 | Visual Studio 
  
Создает сеанс, в котором можно выполнять определенные пользователем функции.
  
```cpp
int OpenSession(WCHAR *Params)
```

## <a name="parameters"></a>Parameters

_Параметры_
  
> Указатель на строку параметров UNICODE с полуколоном для сеанса. Excel не использует этот аргумент.
    
## <a name="return-value"></a>Возвращаемое значение

ID сеанса, который можно использовать в других вызовах соединитетеля кластера, если сеанс был успешно создан; в **противном случае xlHpcRetCallFailed**.
