## <a name="example"></a><span data-ttu-id="33e5e-118">Пример</span><span class="sxs-lookup"><span data-stu-id="33e5e-118">Example</span></span>

<span data-ttu-id="33e5e-119">В следующем примере сравнивается экземпляр последней копии Excel, вызвавшей ее с текущей копией Excel.</span><span class="sxs-lookup"><span data-stu-id="33e5e-119">The following example compares the instance of the last copy of Excel that called it to the current copy of Excel that called it.</span></span> <span data-ttu-id="33e5e-120">Если они одинаковы, возвращается 1; в противном случае возвращается значение 0; Если функция завершается со сбоем, возвращается значение "– 1".</span><span class="sxs-lookup"><span data-stu-id="33e5e-120">If they are the same, it returns 1; if not, it returns 0; if the function fails, it returns -1.</span></span>
  
 `\SAMPLES\EXAMPLE\EXAMPLE.C`
  
```cs
short WINAPI xlGetInstExample(void)
{
    XLOPER12 xRes;
    static HANDLE hOld = 0;
    short iRet;
    if (Excel12(xlGetInst, &xRes, 0) != xlretSuccess)
        iRet = -1;
    else
    {
    HANDLE hNew;
    hNew = (HANDLE)xRes.val.w;
    if (hNew != hOld)
            iRet = 0;
    else
            iRet = 1;
    hOld = hNew;
    }
    return iRet;
}
```
