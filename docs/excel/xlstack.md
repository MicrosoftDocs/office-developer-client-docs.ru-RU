## <a name="example"></a><span data-ttu-id="e3a67-120">Пример</span><span class="sxs-lookup"><span data-stu-id="e3a67-120">Example</span></span>

<span data-ttu-id="e3a67-121">В первом примере отображается оповещение, содержащее объем стека слева и содержащийся в `\SAMPLES\EXAMPLE\EXAMPLE.C`.</span><span class="sxs-lookup"><span data-stu-id="e3a67-121">The first example displays an alert message containing the amount of stack space left and is contained in  `\SAMPLES\EXAMPLE\EXAMPLE.C`.</span></span> <span data-ttu-id="e3a67-122">Второй пример выполняет ту же операцию, работая с **XLOPER**s и не входит в пример кода SDK.</span><span class="sxs-lookup"><span data-stu-id="e3a67-122">The second example does the same thing, working with **XLOPER**s and is not contained in the SDK example code.</span></span>
  
```cs
short WINAPI xlStackExample(void)
{
   XLOPER12 xRes;
   Excel12(xlStack, &xRes, 0);
   Excel12(xlcAlert, 0, 1, (LPXLOPER12)&xRes);
   return 1;
} 
short int WINAPI xlStackExample_XLOPER(void)
{
    XLOPER xRes;
    Excel4(xlStack, (LPXLOPER)&xRes, 0);
    xRes.xltype = xltypeNum; // Change the type to double
    // Cast to an unsigned short to get rid of the overflow problem
    xRes.val.num = (double)(unsigned short) xRes.val.w;
    Excel4(xlcAlert, 0, 1, (LPXLOPER)& xRes);
    return 1;
}
```
