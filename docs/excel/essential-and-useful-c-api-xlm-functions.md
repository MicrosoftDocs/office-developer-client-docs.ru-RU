# <a name="essential-and-useful-c-api-xlm-functions"></a>Основные и полезные функции XLM API C

 **Область применения:** Excel 2013 | Office 2013 | Visual Studio 
  
Описанные в этом разделе функции Microsoft Excel вызова, которые особенно полезны разработчикам DLL и XLL. Из них функция **xlfRegister** имеет важное значение для XLLs и DLLs, которые хотят зарегистрировать свои функции и команды, чтобы они могли быть вызваны непосредственно из Excel. Функции **xlfUnregister** и **xlfSetName** используются в сочетании с незарегистрированными функциями и командами DLL и XLL. 
  
Многие другие функции Excel через API C, которые полезны при разработке XLLs. Они соответствуют функциям Excel, функциям и командам, доступным из макрос XLM.
