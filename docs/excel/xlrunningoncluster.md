# <a name="xlrunningoncluster"></a>xlRunningOnCluster

**Область применения:** Excel 2013 | Office 2013 | Visual Studio 
  
Возвращает значение, которое указывает, запущена ли пользовательской функции в кластере. 
  
```cpp
Excel12(xlRunningOnCluster, LPXLOPER12 pxRes, 0);
```

## <a name="parameters"></a>Параметры

Эта функция не имеет аргументов.
  
## <a name="return-value"></a>Возвращаемое значение

Если функция работает в процессе Excel, возвращает 0 в **XLOPER12** типа **xlTypeInt.** Если функция работает в кластере, тип возврата и значение определяются поставщиком соединители кластера.
  
## <a name="requirements"></a>Требования

Эта функция определена в файле загона Xlcall.h.
