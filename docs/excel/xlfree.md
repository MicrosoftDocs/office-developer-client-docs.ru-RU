## <a name="example"></a><span data-ttu-id="34869-119">Пример</span><span class="sxs-lookup"><span data-stu-id="34869-119">Example</span></span>

<span data-ttu-id="34869-120">В этом примере вызывается **Get. WORKSPACE (1)** для возврата платформы, на которой в данный момент работает Excel в виде строки.</span><span class="sxs-lookup"><span data-stu-id="34869-120">This example calls **GET.WORKSPACE(1)** to return the platform on which Excel is currently running as a string.</span></span> <span data-ttu-id="34869-121">Код копирует возвращаемую строку в буфер для последующего использования.</span><span class="sxs-lookup"><span data-stu-id="34869-121">The code copies this returned string into a buffer for later use.</span></span> <span data-ttu-id="34869-122">Код поместит буфер обратно в **XLOPER12** для последующего использования с функцией Excel.</span><span class="sxs-lookup"><span data-stu-id="34869-122">The code places the buffer back into the **XLOPER12** for later use with the Excel function.</span></span> <span data-ttu-id="34869-123">И, наконец, код отображает строку в окне оповещения.</span><span class="sxs-lookup"><span data-stu-id="34869-123">Finally, the code displays the string in an alert box.</span></span> 
  
 `\SAMPLES\EXAMPLE\EXAMPLE.C`
  
```cs
short WINAPI xlFreeExample(void)
{
   XLOPER12 xRes, xInt;
   XCHAR buffer[cchMaxStz];
   int i,len;
   // Create an XLOPER12 for the argument to Getworkspace.
   xInt.xltype = xltypeInt;
   xInt.val.w = 1;
   // Call GetWorkspace.
   Excel12f(xlfGetWorkspace, &xRes, 1, (LPXLOPER12)&xInt);
   
   // Get the length of the returned string
   len = (int)xRes.val.str[0];
   //Take into account 1st char, which contains the length
   //and the null terminator. Truncate if necessary to fit
   //buffer.
   if (len > cchMaxStz - 2)
      len = cchMaxStz - 2;
   // Copy to buffer.
   for(i = 1; i <= len; i++)
      buffer[i] = xRes.val.str[i];
   // Null terminate, Not necessary but a good idea.
   buffer[len] = '\0';
   buffer[0] = len;
   // Free the string returned from Excel.
   Excel12f(xlFree, 0, 1, &xRes);
   // Create a new string XLOPER12 for the alert.
   xRes.xltype = xltypeStr;
   xRes.val.str = buffer;
   // Show the alert.
   Excel12f(xlcAlert, 0, 1, (LPXLOPER12)&xRes);
   return 1;
}
```
