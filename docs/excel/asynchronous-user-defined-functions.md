## <a name="when-to-use-asynchronous-user-defined-functions"></a><span data-ttu-id="36101-108">Когда использовать асинхронные функции, определенные пользователем</span><span class="sxs-lookup"><span data-stu-id="36101-108">When to use asynchronous user-defined functions</span></span>
