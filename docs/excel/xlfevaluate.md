## <a name="example"></a><span data-ttu-id="cb835-122">Пример</span><span class="sxs-lookup"><span data-stu-id="cb835-122">Example</span></span>

<span data-ttu-id="cb835-123">В этом примере используется **xlfEvaluate** для приведения текста "! B38 "к содержимому ячейки B38.</span><span class="sxs-lookup"><span data-stu-id="cb835-123">This example uses **xlfEvaluate** to coerce the text "!B38" to the contents of cell B38.</span></span> 
  
 <span data-ttu-id="cb835-124">`\SAMPLES\EXAMPLE\EXAMPLE.C`.</span><span class="sxs-lookup"><span data-stu-id="cb835-124"></span></span> <span data-ttu-id="cb835-125">Эта функция вызывает макрос Command (**кслкалерт**) и правильно работает только при вызове из листа макросов или команды макроса.</span><span class="sxs-lookup"><span data-stu-id="cb835-125">This function calls a command macro (**xlcAlert**) and will work correctly only when called from a macro sheet or as a macro command.</span></span>
  
```cs
short WINAPI EvaluateExample(void)
{
    XLOPER12 xFormulaText, xRes, xRes2, xInt;
    xFormulaText.xltype = xltypeStr;
    xFormulaText.val.str = L"\004!B38";
    Excel12(xlfEvaluate, &xRes, 1, (LPXLOPER12)&xFormulaText);
    xInt.xltype = xltypeInt;
    xInt.val.w = 2;
    Excel12(xlcAlert, &xRes2, 2, (LPXLOPER12)&xRes, (LPXLOPER12)&xInt);
    Excel12(xlFree, 0, 1, (LPXLOPER12)&xRes);
    Excel12(xlFree, 0, 1, (LPXLOPER12)&xRes2);
    return 1;
}
```
