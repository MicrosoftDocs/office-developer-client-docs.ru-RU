# <a name="c-api-callback-functions-excel4-excel12"></a>Функции вызова API C Excel4, Excel12

**Область применения:** Excel 2013 | Office 2013 | Visual Studio 
  
Функции **Excel4** и **Excel12** предоставляются для того, чтобы DLLs могли вызывать внутреннюю Microsoft Excel лист, функцию макроса или команду или специальную функцию или команду только для XLL. Все последние версии Excel поддерживают функцию **Excel4.** Начиная с Excel 2007 г. поддерживается функция **Excel12.** Обе функции предоставляются в двух формах: 
  
- Форма списка аргументов переменной длины **(Excel4/Excel12)**
    
- Форма массива аргументов **(Excel4v/Excel12v)**
    
За исключением способа, в котором аргументы передаются этим вызовам, эти две формы являются функционально эквивалентными. Основные понятия для обеих форм полностью описаны в [Excel4/Excel12](excel4-excel12.md). [Excel4v/Excel12v](excel4v-excel12v.md) освещает другие проблемы, которые могут быть у этой формы.
