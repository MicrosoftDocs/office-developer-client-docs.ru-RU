# <a name="developing-excel-cluster-connectors"></a>Разработка соединители кластера Excel

**Область применения:** Excel 2013 | Office 2013 | Visual Studio 
  
Соединитель кластера Excel предоставляет средства для автоматической разгрузки кластерно-безопасных вызовов пользовательских функций в XLL на кластерный сервер. Описание функций, безопасных для кластера, см. в описании [безопасных функций кластера.](cluster-safe-functions.md) Такая разгрузка может повысить производительность, позволяя использовать больше вычислительных ресурсов. Соединители кластера обычно разрабатываются поставщиком кластера высокой производительности.
  
## <a name="cluster-connectors"></a>Соединители кластера

Соединитель кластера — это DLL,который предоставляет определенные точки входа, которые Excel использует для координации вызовов пользовательских функций, безопасных для кластера. Он служит интерфейсом между Excel и высокоскоростным вычислительным кластером для управления сеансами, для выполнения вызовов функций (путем передачи полного имени функции и фактических аргументов вызова) и для возврата результатов вызова в Excel с помощью механизма обратного вызова.
  
Чтобы создать соединитель кластера, создайте DLL-запись, которая предоставляет точки входа, перечисленные в функциях [соединители кластера Excel.](excel-cluster-connector-functions.md)
  
## <a name="installing-a-cluster-connector"></a>Установка кластерного соединителя

Чтобы сделать соединитель кластера доступным в Excel, код установки соединителя должен установить DLL соединителя на компьютер, на котором установлен Excel. Кроме того, код установки соединителю должен добавить запись для соединителю в следующем ключе реестра:
  
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\Excel\Excel Cluster Connectors\
  
Добавьте узел в этот ключ для соединителю кластера, который указывает следующие строки:
  
-  `Name`— имя, которое будет отображаться в списке соединитений кластера в Excel.
    
-  `Filename`— полный путь для DLL.
