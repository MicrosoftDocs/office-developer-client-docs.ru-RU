## <a name="internal-and-external-worksheet-references"></a><span data-ttu-id="81bb0-125">Лист внутренних и внешних ссылок</span><span class="sxs-lookup"><span data-stu-id="81bb0-125">Internal and External Worksheet References</span></span>

<span data-ttu-id="81bb0-126">Важных различий между внутренними и внешними ссылки — это, что тип данных внешняя ссылка содержит идентификатор для листа, а также описание, которые называются ячеек.</span><span class="sxs-lookup"><span data-stu-id="81bb0-126">The key difference between internal and external references is that the external reference data type contains an ID for the worksheet and also a description of which cells are referred to.</span></span> <span data-ttu-id="81bb0-127">Внутренняя ссылка содержит ссылка на лист — оно подразумевается, что лист является текущий лист.</span><span class="sxs-lookup"><span data-stu-id="81bb0-127">An internal reference contains no reference to the sheet—it is implicit that the sheet is the current sheet.</span></span>

return references or take reference arguments.</span></span> <span data-ttu-id="81bb0-129">Все функции интерфейса API для C, ссылки, что принимает аргументы принимает внутренних или внешних ссылок, функции интерфейса API для C [xlfCaller](xlfcaller.md) возвращает ссылку на вызывающий ячеек по определению на текущем листе.</span><span class="sxs-lookup"><span data-stu-id="81bb0-131">For example,
