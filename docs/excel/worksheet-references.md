<span data-ttu-id="d9cef-106">Ссылка в Microsoft Excel — это тип данных, который ссылается на прямоугольный блок ячеек (который может быть только одной ячейкой) или, в некоторых случаях, на несколько несоотдиняемых блоков ячеек.</span><span class="sxs-lookup"><span data-stu-id="d9cef-106">A reference in Microsoft Excel is a data type that refers to a rectangular block of cells (which can be just one cell), or in some cases, a number of disjoint blocks of cells.</span></span> <span data-ttu-id="d9cef-107">Для ячеек на текущем листе Excel использует один тип ссылки, который называется внутренней ссылкой.</span><span class="sxs-lookup"><span data-stu-id="d9cef-107">Internally, Excel uses one reference type for cells on the current sheet, known as an internal reference.</span></span> <span data-ttu-id="d9cef-108">Любая ячейка, которая не находится на текущем листе, описывается другим типом ссылки, который называется внешней ссылкой.</span><span class="sxs-lookup"><span data-stu-id="d9cef-108">Any cell that is not on the current sheet is described by another type of reference known as an external reference.</span></span> <span data-ttu-id="d9cef-109">Определение активного и текущего см. в следующем разделе.</span><span class="sxs-lookup"><span data-stu-id="d9cef-109">See the next section for the definition of active and current.</span></span>
  
## <a name="active-vs-current"></a><span data-ttu-id="d9cef-110">Активный или текущий</span><span class="sxs-lookup"><span data-stu-id="d9cef-110">Active vs. Current</span></span>

<span data-ttu-id="d9cef-111">В Excel термин "активный" обозначает то, что просматривает пользователь.</span><span class="sxs-lookup"><span data-stu-id="d9cef-111">In Excel, the term active refers to what the user is viewing.</span></span> <span data-ttu-id="d9cef-112">Активная книга и таблица — это книги, на которые в данный момент смотрит пользователь, или, если Excel потеряет фокус на другом приложении, смотрит на время последнего фокуса Excel.</span><span class="sxs-lookup"><span data-stu-id="d9cef-112">The active workbook and worksheet are those that the user is currently looking at, or, if Excel has lost focus to another application, was looking at when Excel last had focus.</span></span> <span data-ttu-id="d9cef-113">Активный лист всегда находится в активной книге.</span><span class="sxs-lookup"><span data-stu-id="d9cef-113">The active sheet is always in the active workbook.</span></span> <span data-ttu-id="d9cef-114">Одна или несколько ячеек, выбранных на активном листе, называются активными ячейками.</span><span class="sxs-lookup"><span data-stu-id="d9cef-114">The one or more cells that are selected in the active sheet are known as the active cells.</span></span> <span data-ttu-id="d9cef-115">Если внедренный объект имеет фокус, последние выбранные ячейки по-прежнему активны.</span><span class="sxs-lookup"><span data-stu-id="d9cef-115">If an embedded object has focus, the last-selected cells are still active.</span></span> 
  
<span data-ttu-id="d9cef-116">Текущий термин относится к пересчету в Excel.</span><span class="sxs-lookup"><span data-stu-id="d9cef-116">The term current refers to what Excel is recalculating.</span></span> <span data-ttu-id="d9cef-117">Текущая книга и таблица — это книги, которые в настоящее время пересчитываются.</span><span class="sxs-lookup"><span data-stu-id="d9cef-117">The current workbook and worksheet are those that are currently being recalculated.</span></span> <span data-ttu-id="d9cef-118">Текущий лист всегда находится в текущей книге.</span><span class="sxs-lookup"><span data-stu-id="d9cef-118">The current sheet is always in the current workbook.</span></span> <span data-ttu-id="d9cef-119">Пересчитаемая ячейка называется текущей ячейкой или, в случае пересчета формулы массива, текущими ячейками.</span><span class="sxs-lookup"><span data-stu-id="d9cef-119">The cell being recalculated is known as the current cell, or, in the case of an array formula being recalculated, the current cells.</span></span> 
  
<span data-ttu-id="d9cef-120">Следует помнить о таких важных моментах:</span><span class="sxs-lookup"><span data-stu-id="d9cef-120">The important points to remember are as follows:</span></span>
  
- <span data-ttu-id="d9cef-121">Активная книга, таблица или ячейка обычно не является текущей, хотя это возможно.</span><span class="sxs-lookup"><span data-stu-id="d9cef-121">The active workbook/worksheet/cell is not generally the current one, although it can be.</span></span>
    
- <span data-ttu-id="d9cef-122">Функция надстройки в модуле Visual Basic для приложений (VBA), DLL или XLL всегда будет вызвана из текущей ячейки на текущем листе или одна из них в случае многопоточных пересчетов (MTR).</span><span class="sxs-lookup"><span data-stu-id="d9cef-122">An add-in function, whether in a Visual Basic for Applications (VBA) module or a DLL or XLL, is always called from the current cell on the current sheet, or one of them in the case of multithreaded recalculation (MTR).</span></span>
    
<span data-ttu-id="d9cef-123">Многие функции Excel, которые предоставляют сведения о ячейке, диапазоне ячеек или листе в книге, отличают активную книгу, лист или ячейку от текущей книги, листа или ячейки.</span><span class="sxs-lookup"><span data-stu-id="d9cef-123">Many Excel functions that provide information about a cell, a range of cells, or a sheet in a workbook distinguish between the active workbook, sheet, or cell and the current workbook, sheet, or cell.</span></span> <span data-ttu-id="d9cef-124">Это различие отражается в типах данных, используемых для описания ссылок на блоки ячеек, как описано в следующем разделе.</span><span class="sxs-lookup"><span data-stu-id="d9cef-124">This difference is reflected in the data types used to describe references to blocks of cells, as described in the following section.</span></span>
  
## <a name="internal-and-external-worksheet-references"></a><span data-ttu-id="d9cef-125">Внутренние и внешние ссылки на таблицы</span><span class="sxs-lookup"><span data-stu-id="d9cef-125">Internal and External Worksheet References</span></span>

<span data-ttu-id="d9cef-126">Ключевое различие между внутренними и внешними ссылками состоит в том, что тип данных внешней ссылки содержит ИД для таблицы, а также описание ячеек, на которые ссылаются.</span><span class="sxs-lookup"><span data-stu-id="d9cef-126">The key difference between internal and external references is that the external reference data type contains an ID for the worksheet and also a description of which cells are referred to.</span></span> <span data-ttu-id="d9cef-127">Внутренняя ссылка не содержит ссылки на лист — неявно, что лист является текущим листом.</span><span class="sxs-lookup"><span data-stu-id="d9cef-127">An internal reference contains no reference to the sheet—it is implicit that the sheet is the current sheet.</span></span> 
  
<span data-ttu-id="d9cef-128">Многие функции API C возвращают ссылки или принимают аргументы ссылок.</span><span class="sxs-lookup"><span data-stu-id="d9cef-128">Many C API functions return references or take reference arguments.</span></span> <span data-ttu-id="d9cef-129">Любая функция API C, которая принимает опорные аргументы, принимает внутренние или внешние ссылки, за исключением функции **xlSheetNm,** для которой требуется внешняя ссылка.</span><span class="sxs-lookup"><span data-stu-id="d9cef-129">Any C API function that takes reference arguments accepts either internal or external references, except the **xlSheetNm** function, which requires an external reference.</span></span> <span data-ttu-id="d9cef-130">Некоторые функции возвращают только внутренние или внешние ссылки.</span><span class="sxs-lookup"><span data-stu-id="d9cef-130">Some functions only return either internal or external references.</span></span> <span data-ttu-id="d9cef-131">Например, функция [XlfCaller](xlfcaller.md) API C возвращает ссылку на вызывающая ячейка по определению на текущем листе.</span><span class="sxs-lookup"><span data-stu-id="d9cef-131">For example, the C API function [xlfCaller](xlfcaller.md) returns a reference to the calling cells, by definition, on the current sheet.</span></span> <span data-ttu-id="d9cef-132">Возвращаемая ссылка всегда является внутренней ссылкой, хотя функция может возвращать не справочные типы, в которых функция не вызвана из ячейки таблицы.</span><span class="sxs-lookup"><span data-stu-id="d9cef-132">The returned reference is always an internal reference, although the function can return non-reference types where the function is not called from a worksheet cell.</span></span> <span data-ttu-id="d9cef-133">Функция API [C xlSheetId](xlsheetid.md) всегда возвращает ИД таблицы, содержаной во внешнем типе ссылок.</span><span class="sxs-lookup"><span data-stu-id="d9cef-133">The C API function [xlSheetId](xlsheetid.md) always returns the ID of a worksheet contained within an external reference data type.</span></span> 
  
<span data-ttu-id="d9cef-134">Другое ключевое различие между внутренними и внешними эталонными типами состоит в том, что тип данных внешней ссылки может описывать несколько несоединяемых блоков ячеек на одном листе.</span><span class="sxs-lookup"><span data-stu-id="d9cef-134">The other key difference between the internal and external reference types is that the external reference data type can describe multiple disjoint blocks of cells on the same sheet.</span></span> <span data-ttu-id="d9cef-135">Внутренние ссылки могут описывать только один блок на текущем листе.</span><span class="sxs-lookup"><span data-stu-id="d9cef-135">Internal references can describe only a single block on the current sheet.</span></span> <span data-ttu-id="d9cef-136">Несоединяемая ссылка может передаваться любой функции, которая принимает аргумент диапазона.</span><span class="sxs-lookup"><span data-stu-id="d9cef-136">Disjoint references can be passed to any function that takes a range argument.</span></span>
