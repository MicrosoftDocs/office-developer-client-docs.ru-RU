# <a name="tempstr"></a>TempStr

 **Область применения:** Excel 2013 | Office 2013 | Visual Studio 
  
Устаревшая функция библиотеки Framework, которая **создает временную** структуру, содержащую строку байта **кслтипестр** . В качестве входных данных принимается строка источника, заканчивающаяся нулем. Он пытается перезаписать первый символ предоставленной строки с длиной следующей строки. Это не всегда является безопасным действием: Microsoft Excel может выполнить сбой, если передать строку, доступную только для чтения. 
  
```cs
LPXLOPER TempStr(LPSTR str);
```

## <a name="parameters"></a>Параметры

 _str_
  
Указатель на строку источника, заканчивающуюся нулем. **Темпстр** усекает строки, размер которых превышает 255 байт. 
  
## <a name="return-value"></a>Возвращаемое значение

Возвращает строку **кслтипестр** , содержащую указатель на переданный буфер строк. 
  
## <a name="remarks"></a>Примечания

Такой способ создания временных строк теперь является нерекомендуемым в пользу того, как работают как в [темпстрконст, так и в TempStr12](tempstrconst-tempstr12.md) . Эти функции выделяют новый буфер памяти и копируют в него переданную строку. Входные строки для **темпстрконст** и **TempStr12** не изменяются, поэтому объявляются как **const**. В отличие от того, входная строка в **темпстр** изменяется и не может быть объявлена **константой**. Первый символ входной строки считается пространством для знака длины и перезаписывается этой функцией.
